Ruby is installed on all DreamHost servers. However, the version is out of date. As of May 2018, all servers are running version 1.9.3.

Ruby on Shared hosting

Due to the amount of memory Ruby on Rails may use, it's not recommended to run an app on Shared hosting. You will most likely be hit the Shared server's memory limits.

Installing a current version of Ruby

Running a Ruby app is only recommended on a VPS or Dedicated server. On those servers, you can install any version you like with Ruby Version Manager.
