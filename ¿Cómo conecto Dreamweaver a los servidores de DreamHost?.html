Generalidades

Adobe Dreamweaver es una poderosa herramienta de desarrollo web que te ayuda a crear un sitio web personalizado desde cero. Combina un editor WYSIWYG fácil de usar, plantillas básicas, la capacidad de codificar (y ver) tu sitio web para varios tamaños de pantalla (es decir, móvil y tableta) y una interfaz FTP/SFTP.

El siguiente artículo proporciona instrucciones sobre cómo utilizar Dreamweaver para conectarte a los servidores de DreamHost a través de FTP. Para obtener más información sobre el uso de Dreamweaver para crear un sitio web, visita Adobe.

Configurando tu sitio en Adobe Dreamweaver CC

Cuando abres Dreamweaver, verás varias opciones en la barra de herramientas superior.

Para configurar un nuevo sitio en Dreamweaver:

  1. Abre Dreamweaver.
  2. Haz clic en la pestaña Site en la barra superior de herramientas, y selecciona la opción New Site....
    La caja  Site Setup se abre:
  3. Ingresa lo siguiente:
    • Site Name: Puedes usar cualquier nombre que desees para el nombre de tu sitio.
    • Local Site Folder: Ingresa la ruta en tu computadora local donde guardarás los archivos para tu sitio.

Añadiendo una conexión de servidor

Para agregar una nueva conexión de servidor en Dreamweaver:

  1. Abre la caja Site Setup.
  2. En la derecha superior de la caja Site Setup, selecciona la pestaña Servers.
  3. Haz clic en el signo más (+) que aparece directamente debajo de ‘Name' field.
    La caja de configuraciones de servidor se abre:
  4. Ingresa la configuración del servidor DreamHost en los siguientes campos. Esta información se utiliza para conectar Dreamweaver a tu servidor web DreamHost:
    • Server Name: Nombre del sitio sin la parte http://. Por ejemplo, si tu sitio es https://www.example.com, ingresa example.com.
    • Connect Using: Dependiendo en cómo el usuario está configurado, ingresa FTP o SFTP. Ve el artículo Crear un usuario con acceso Shell (SSH) para más detalles sobre cómo escoger un tipo de usuario.
    • FTP Address: Nombre del servidor de DreamHost o el nombre del sitio en sí. Ve el artículo Encontrar tu nombre de host FTP para detalles sobre cómo obtener tu nombre de servidor.
    • Username: Nombre del usuario FTP con el que el dominio ha sido configurado. Puedes encontrar el nombre de usuario en la página Manage Domains. Está a la derecha del dominio bajo la columna Web Hosting.
    • Password: La contraseña que ingresaste cuando configuraste el dominio con el usuario. Puedes cambiar la contraseña en la página Manage Users.
    • Root Directory: Esto también es conocido como el Directorio Web. Probablemente sea el nombre de tu sitio con un slash (/) al final. Puedes confirmar tu Directorio web en la página Manage Domains. Haz clic en el botón Edit a la derecha de tu dominio y luego puedes verlo en la primera sección.
    • Web URL: Este es el nombre del sitio. Ingrese la URL completa aquí, incluido http:// o https:// si estás utilizando un certificado SSL.
    • Port: Para un usuario de FTP, ingresa el puerto 21. Para un usuario de SFTP o Shell, ingresa el puerto 22.
  5. Después de que ingreses la configuración de servidor, haz clic en el botón Test para confirmar la conexión.
    La siguiente caja aparece mostrando una conexión exitosa:
  6. Haz clic en el botón Save para guardar estos cambios.

Subir tus archivos a DreamHost

Una vez que hayas configurado y guardado una conexión exitosa a tu servidor, puedes usar Dreamweaver para cargar los archivos de tu sitio en cualquier momento.

Sigue estos pasos para cargar tus archivos desde tu carpeta local al servidor DreamHost remoto:

  1. Expande el panel de la derecha (CSS Designer, Files, Assets, y Insert tabs) que contiene toda la información de tus sitios.
  2. Selecciona la pestaña Files. (Asegúrate de no confundirla con la pestaña File en la barra de herramientas superior).
  3. Selecciona el sitio que deseas cargar en el primer menú desplegable. Este es el Site Name que creaste durante el proceso de configuración.
  4. Selecciona Local view en el segundo menú desplegable que aparece junto a tu sitio. Esto te permite ver y cargar los archivos del sitio que guardaste localmente en tu computadora.
  5. Haz clic en la flecha 'hacia arriba'. Cuando te desplazas sobre la flecha hacia arriba, dice: Coloque los archivos en "Remote Server example.com", que es el sitio alojado en los servidores de DreamHost.
  6. Espera hasta que los archivos terminen de cargarse. Aparece un cuadro de diálogo que muestra el progreso.
  7. Haz clic en el segundo menú desplegable y selecciona Servidor remoto para ver los archivos en tu servidor DreamHost.
    A continuación se muestra que el archivo 'DH Test Pic.jpg' se cargó correctamente en la carpeta del servidor remoto. Puedes arrastrarlo y soltarlo en los directorios de Dreamweaver según sea necesario:

Una vez que aprendas a cargar tus archivos, simplemente haz clic enk Site > Put en la barra de herramientas superior (o, Ctrl + Shift + U) para cargar rápidamente tus archivos en el servidor. Esta es una forma más fluida de mover tus archivos y es especialmente útil cuando está haciendo muchas ediciones en tu sitio web y no tienes abierto el panel de la derecha.
