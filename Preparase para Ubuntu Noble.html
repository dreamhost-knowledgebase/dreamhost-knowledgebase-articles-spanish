Generalidades

Este artículo proporciona información para ayudarte a preparar tu sitio para la actualización a Ubuntu 24.04.2 LTS (también conocida como Noble).

¿Qué esperar?

DreamHost te contactará por correo electrónico antes de la actualización.

  • El horario garantiza que habrá varios ingenieros altamente capacitados disponibles en caso de que surjan problemas importantes.
  • La actualización en sí no debería tardar más de una hora en completarse.
  • Cualquier servicio que dependa de la disponibilidad de tu servidor MySQL (WordPress u otros sitios web y aplicaciones que utilicen bases de datos, etc.) no estará disponible hasta que se complete la actualización. Esto NO afectará los servicios de nube o correo electrónico.

Solución de problemas

A continuación se proporciona más información sobre cómo solucionar algunos problemas conocidos.

ssh-rsa

El cifrado de clave ssh ssh-rsa ya no está permitido. Deberás regenerar tu llave de ssh si estás utilizando ssh-rsa. Revisa las siguientes guías para obtener instrucciones sobre cómo crear nuevas llaves de emparejamiento:

  • Cómo configurar el inicio de sesión sin contraseña en Mac OS y Linux
  • Configurar el inicio de sesión sin contraseña en PuTTY

Entorno virtual de Python

Si tu sitio web está utilizando un entorno virtual de Python y notas algún error después de la actualización, reinstala tu entorno virtual y todos los módulos. Esto ayuda a asegurar que todo sea compatible con las nuevas bibliotecas en el servidor.

sqlite

Ubuntu Noble ya no proporcionará soporte para SQLite versión 2. Deberás actualizar todas las referencias de SQLite Versión 2 a SQLite Versión 3.

¿Dónde puedo encontrar soporte?

Si tienes alguna pregunta o inquietud, por favor Contactar a Soporte.
