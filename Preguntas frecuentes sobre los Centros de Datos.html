¿Qué es un centro de datos?

Un centro de datos es una instalación que alberga diferentes tipos de servidores, como servidores de sitios web, servidores de bases de datos, servidores de correo y servidores en la nube. Si has comprado servicios de alojamiento de DreamHost, los datos de tu sitio web se almacenan y protegen dentro de un centro de datos.

¿Dónde se encuentran ubicados los centros de datos de DreamHost?

DreamHost tiene centros de datos en Ashburn Virginia y Hillsboro Oregon.

¿Dónde está mi centro de datos?

Puedes ver en qué centro de datos están ubicados tus servicios dentro de su panel. Navega a la página Centro de Datos. Aquí puedes ver todos tus servicios y el centro de datos correspondiente en el que se encuentran.

¿La ubicación de mi centro de datos es importante?

Esto solo es un problema si tu sitio web utiliza una base de datos. Esto se debe a que tus servidores web y de base de datos siempre deben estar en el mismo centro de datos.

Ocasionalmente, el servidor web y el servidor de base de datos pueden colocarse en diferentes centros de datos. Esto podría provocar lentitud en el sitio ya que toma un poco más de tiempo recuperar la información del otro centro de datos.

Si notas esto, contacta a soporte para que puedan mover tus servidores a la misma ubicación.

¿Puedo cambiar mi centro de datos?

Si. Contacta a soporte si prefieres que tu información sea almacenada en un centro de datos diferente.
