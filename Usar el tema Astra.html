Generalidades

Astra es un tema de WordPress totalmente personalizable adecuado para blogs, portafolios personales, sitios web de negocios. y sitios eCommerce (WooCommerce). El tema Astra viene preconfigurado con cualquier instalación de WordPress y puedes usar cualquiera de las plantillas de inicio gratuitas y listas para usar diseñadas por profesionales.

¿Qué es un constructor web de WordPress?

Un creador de sitios web de WordPress te permite crear rápidamente un sitio web. Hay varios tipos de creadores de sitios web, como Gutenberg, Elementor y Beaver Builder.

El plugin Starter Templates

El plugin Starter Templates se instala automáticamente cuando instalas WordPress en el panel. Este plugin te da acceso a plantillas predeterminadas cuano usas cualquiera de los constructores web mencionados abajo.

Los tres constructores web

Los siguientes son los tres constructores web que puedes usar con el tema Astra y el plugin Starter Templates.

Gutenberg

También conocido como Block Editor (o Editor de WordPress predeterminado), Gutenberg es un generador fácil de usar que te permite diseñar tus publicaciones y páginas con mayor facilidad. Reemplazó al Editor clásico de WordPress para ofrecer una experiencia de creación web más intuitiva.

El plugin Spectra

El plugin Spectra activa el editor Gutenberg con bloques avanzados que te ayudan a acelerar el proceso de creación de sitios web con configuraciones y personalizaciones fáciles de usar para un estilo visual intuitivo. Para saber más sobre Spectra y Astra, revisa los siguientes enlaces:

  • Página principal de Spectra
  • Página de anuncios de Astra

Elementor

Elementor es un generador de páginas de arrastrar y soltar para WordPress. Este plugin te ayuda a crear hermosas páginas usando un editor visual y está diseñado para crear sitios web dinámicos rápidamente.

Beaver Builder

Beaver Builder es un generador de páginas flexible de arrastrar y soltar que funciona en la interfaz de tu sitio web de WordPress. Este te permite crear fácilmente sitios web y páginas de destino sin escribir ningún código.

¿Cuál constructor web debería usar?

Si tu objetivo es el rendimiento, Gutenberg es tu mejor opción.

Si tu objetivo es tener una funcionalidad más amplia y facilidad de uso, entonces Elementor o Beaver Builder son una mejor opción.

Si no sabes cómo crear un sitio con ninguno de estos constructores, puedes comenzar a usar una plantilla de inicio, que se trata en la siguiente sección.

Usar una plantilla para construir un sitio web

Para crear una nueva plantilla:

  1. Da clic en el botón Get Started en la página del panel de WordPress.
  2. Da clic en el botón Build Your Website Now que aparece en la siguiente página.
    La página Page Builder se abre:
  3. Selecciona una Página de Construcción.

    DreamHost recomienda usar Block Editor (Gutenberg) ya que todas las plantillas disponibles son gratuitas y existen numerosos tipos entre los que puedes elegir.

    Después de seleccionar el Block Editor, aparece una lista de plantillas disponibles.
  4. Selecciona una plantilla que desees usar. Entonces puedes editar las siguientes opciones antes de lanzar tu nuevo diseño:
    • Subir un logotipo (Dimensiones sugeridas: 180x60 pixeles)
    • Editar el ancho del logotipo
    • Cambiar los colores
    • Cambiar las fuentes
  5. Ingresa los detalles adiciones sobre tu negocio cuando se te indique.

    DreamHost recomienda dejar seleccionadas todas las opciones debajo de Advanced Options (excepto Shared Non-Sensitive Data, el cual puedes desmarcar si deseas).

  6. Haz clic en el botón Submit & Build My Website.

El proceso de instalación no debería tardar más de unos minutos en completarse. Una vez que termine, haz clic en el botón View Your Website para navegar a tu sitio web. Luego puedes editar el contenido de tu sitio web como desees.
