Generalidades

El registro de dominio es el nombre de tu sitio, que debe comprarse primero para que el dominio se muestre en línea. Hay una variedad de nombres diferentes y Dominios de nivel superior (TLD) que puedes elegir para registrar. Este artículo proporciona información general sobre los registros de dominio con DreamHost.

Lo básico

En términos generales, hay dos pasos principales para obtener tu sitio en línea:

  1. Compra el registro de dominio (el nombre de tu sitio).
  2. Compra un plan de hosting y agrega tu sitio.
    • ¿Qué tipos de plan de alojamiento hay disponibles?
    • ¿Cómo agregar un dominio?

Un registro de dominio (o TLD - Dominio de nivel superior) es la primera mitad de un proceso de dos pasos para obtener un sitio en Internet. La compra de un registro de dominio proporciona información de contacto para el registrante (el propietario del nombre de dominio), y asigna los nameservers que controlan la configuración de DNS del dominio.

La información de registro para cualquier nombre de dominio se puede consultar con una búsqueda de WHOIS. Puedes consultar el siguiente artículo para obtener más detalles sobre la información de WHOIS y cómo puedes obtenerla para tu nombre de dominio:

  • Generalidades de WHOIS

Los comunes .COM/.NET/.ORG/ y .INFO (así como otros TLD) ofrecen privacidad para la información de contacto que proporcionas cuando se registra el dominio. Sin embargo, otros TLD pueden no ofrecer ninguna protección de privacidad. Visita el siguiente artículo para obtener una lista de registros que NO admiten la privacidad en DreamHost:

  • Privacidad de registro de dominio

Registros gratuitos

Puedes utilizar tu crédito de registro gratuito para los siguientes TLDs:

  • .art
  • .biz
  • .blog
  • .club
  • .co
  • .com
  • .design
  • .info
  • .me
  • .net
  • .online
  • .org
  • .shop
  • .site
  • .store
  • .tech
  • .us
  • .work
  • .xzy

Para calificar para un crédito de registro de dominio gratuito para el primer año completo, debes inscribirte en un plan anual de 1 año o de 3 años en un plan Shared Starter, Shared Unlimited, o cualquier plan anual de DreamPress (mensual compartido o los planes mensuales de DreamPress no aplican). Este crédito gratis NO está disponible bajo ningún otro plan de alojamiento. Por favor, visita las siguientes páginas para ver más detalles:

Transferencias

Debido a la política de transferencia de ICANN, cualquier cambio realizado en el nombre, la organización o la dirección de correo del registrante pondrá al dominio en un bloqueo de transferencia de 60 días; también activará correos de notificación que se envían al antiguo y al nuevo registrante notificándoles el cambio.

Por favor visita el siguiente artículo para más detalles:

Cambiar la ortografía de un nuevo registro

Cada dominio adquirido es único en Internet. Incluso si hay una sola letra diferente en la ortografía de un dominio, todavía se considera un nombre de dominio completamente diferente.

Esto significa que si escribiste mal un dominio cuando lo compraste, no hay forma de ajustar su ortografía. La única opción sería comprar un nuevo dominio con una ortografía diferente.

Reembolsar un dominio mal escrito

Los registros de dominio terminan en diferentes TLD como .com, .net, .io, .life. Cada uno de estos tiene reglas diferentes sobre cuándo o si puede cancelar el registro después de la compra. Comunícate con el soporte de inmediato si necesitas cancelar un registro nuevo.

Ten en cuenta que no es posible cancelar muchos tipos de TLD.

Adicionalmente, los pocos que permiten la cancelación del registro tienen un período de tiempo muy corto en cuanto a cuándo se puede hacer, generalmente no más de unos pocos días. Asegúrate de ponerte en contacto con el soporte técnico de inmediato si necesitas cancelar el registro.
