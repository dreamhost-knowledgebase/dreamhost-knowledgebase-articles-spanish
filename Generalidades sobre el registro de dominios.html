Generalidades

El registro de dominio es el nombre de tu sitio, que debe comprarse primero para que el dominio se muestre en línea. Hay una variedad de nombres diferentes y Dominios de nivel superior (TLD) que puede elegir para registrarse. Este artículo proporciona información general sobre los registros de dominio con DreamHost.

Lo basico

En términos generales, hay dos pasos principales para obtener tu sitio en línea:

  1. Compra el registro de dominio (el nombre de tu sitio).
  2. Compra un plan de hosting y agrega tu sitio.
    • ¿Qué tipos de plan de alojamiento hay dispnibles?
    • ¿Cómo agregar un dominio?

Un registro de dominio (o TLD - Dominio de nivel superior) es la primera mitad de un proceso de dos pasos para obtener un sitio en Internet. La compra de un registro de dominio proporciona información de contacto para el registrante (el propietario del nombre de dominio), y asigna los servidores de nombres que controlan la configuración de DNS del dominio.

La información de registro para cualquier nombre de dominio se puede consultar con una búsqueda de WHOIS. Puedes consultar el siguiente artículo para obtener más detalles sobre la información de WHOIS y cómo puedes obtenerla para tu nombre de dominio:

  • Generalidades de WHOIS

Los comunes .COM/.NET /.ORG / y .INFO (así como otros TLD) ofrecen privacidad para la información de contacto que proporcionas cuando se registra el dominio. Sin embargo, otros TLD pueden no ofrecer ninguna protección de privacidad. Visita el siguiente artículo para obtener una lista de registros que NO admiten la privacidad en DreamHost:

  • Privacidad de registro de dominio

Registros gratuitos

Puedes utilizar tu crédito de registro gratuito para los siguientes TLDs:

  • .art
  • .blog
  • .club
  • .com
  • .design
  • .info
  • .me
  • .net
  • .online
  • .org
  • .shop
  • .site
  • .store
  • .tech
  • .xzy

Para calificar para un crédito de registro de dominio gratuito para el primer año completo, debes inscribirte en un plan anual de 1 año o de 3 años en un plan Starter Shared, Unlimited Shared/Happy Hosting, o cualquier plan anual de DreamPress (mensual compartido o los planes mensuales de DreamPress no aplican). Este crédito gratis NO está disponible bajo ningún otro plan de alojamiento. Por favor, visita las siguientes páginas para ver más detalles:

  • Terminos de registro de dominios
  • Registrarse para un plan Shared Hosting

Puedes leer más sobre el registro gratuito ofrecido con los planes de Shared Hosting y DreamPress en el siguiente artículo:

  • Generalidades de los registros gratuitos

Transferencias

Debido a la política de transferencia de ICANN, cualquier cambio realizado en el nombre, la organización o la dirección de correo electrónico del registrante pondrá al dominio en un bloqueo de transferencia de 60 días; también activará correos electrónicos de notificación que se envían al antiguo y al nuevo registrante notificándoles el cambio.

Por favor visita el siguiente artículo para más detalles:

  • Politica de transferencia ICANN
