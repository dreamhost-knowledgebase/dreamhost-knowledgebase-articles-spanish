Generalidades

Después de que se hayas registrado para una nueva cuenta compartida de DreamHost y hayas iniciado sesión en tu panel, serás recibido con la página de Home del panel. En esta página, puedes administrar las principales áreas de tu cuenta.

La página de inicio que se muestra en este artículo solo está disponible para el propietario de la cuenta principal. Subcuentas que ha agregado usando Privilegios de cuenta no verá esta página.

La pestaña 'Build Website'

La pestaña Build Website muestra varios cuadros. La imagen de arriba asume que elegiste instalar WordPress en tu sitio al registrarte.

Inicia sesión en WordPress

Da click en Open WordPress Editor botón para iniciar sesión directamente en tu panel de administración de WordPress.

File Manager

Dando click en el botón Manage Files abre el programa WebFTP. Esto te permite editar directamente los archivos de tu servidor web en tu panel DreamHost.

Haz clic en 'Show Credentials' para ver tus credenciales FTP. Puede usarlos para iniciar sesión en tu servidor web utilizando un cliente FTP.

  • Generalidades sobre FTP y credenciales

Database (MySQL)

Dando click en el botón Manage Database para iniciar sesión directamente phpMyAdmin. Aquí puedes editar tu base de datos.

Haz clic en 'Show Credentials' para ver las credenciales de inicio de sesión de tu base de datos. Puedes usarlos para iniciar sesión en tu base de datos utilizando SSH o un cliente de terceros.

  • ¿Cómo me conecto a mi base de datos a través de SSH?
  • ¿Cómo me conecto a mi base de datos usando un programa de terceros?

Secure Shell (SSH)

El usuario original del sitio web creado para usted es un usuario de SFTP. Puede usar los datos del 'File Manager' de arriba para usar este usuario para iniciar sesión en tu servidor.

Sin embargo, dependiendo de tus necesidades, es posible que deba ejecutar comandos específicos de Shell. Para ejecutar un comando de Shell en el servidor, tu nombre de usuario debe cambiarse a un usuario SSH.

Haz clic en el botón 'Enable SSH'. Tu usuario se cambia a un usuario SSH. Luego puede iniciar sesión con un terminal SSH.

  • Generalidades de SSH

La pestaña 'Migration Service'

Esta pestaña muestra varias opciones para migrar y actualizar tu sitio web de WordPress.

Servicio de migración pagado

El Servicio de migración pagado es un servicio que DreamHost ofrece para migrar todo tu sitio de WordPress a tu servidor DreamHost.

Actualizar a WordPress administrado

Esta opción actualiza tu sitio de WordPress de un servidor web compartido estándar a un servidor privado de DreamPress. Este plan es adecuado para sitios de WordPress que han superado el alojamiento compartido y necesitan más recursos.
