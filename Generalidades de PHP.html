¿Qué es PHP?

PHP es un lenguaje de programación gratuito que suele combinarse con HTML para crear sitios web y software dinámicos e interactivos como WordPress. PHP puede añadir las siguientes funciones a un sitio web:

  • Crear y manipular archivos en el servidor
  • Guardar información de formularios de contacto
  • Conectar un sitio web a una base de datos
  • Encriptar datos

Actualizaciones

PHP es actualizado continuamente por sus desarrolladores. En cualquier momento dado, hay versiones que son soportadas oficialmente por el equipo de desarrollo de PHP y versiones que ya no se mantienen.

Es muy importante asegurarse de que la versión que usa tu sitio sea compatible actualmente con los desarrolladores de PHP. Los beneficios de mantenerse actualizado incluyen:

  • Nuevas características/funcionalidades
  • Mejoras de seguridad
  • Estabilidad y mejoras en rapidez
  • Arreglos de bugs

Soportados/Versiones desactualizadas

A medida que se lanzan versiones más nuevas, las versiones anteriores se retiran. Cuando una versión anterior de PHP ya no es compatible con sus desarrolladores, ha llegado a su estado de Fin de Vida Útil o (EOL). Puedes encontrar una lista de las versiones actuales de PHP y su calendario EOL aquí.

Plan de Soporte Extendido de DreamHost

DreamHost requiere el Soporte Extendido de PHP para continuar usando las versiones antiguas de PHP que han se encuentran en el estado de Fin de Vida Útil por php.net. Esto te da más tiempo para actualizar la versión de PHP después de que ya no sea oficialmente soportada. Revisa este artículo para más detalles.

Configuraciones de PHP de DreamHost

Configuraciones de PHP del servidor

DreamHost ejecuta PHP como una aplicación FastCGI.

mod_php no es soportada en los planes de alojamiento administrados. Estos incluyen:

  • Shared
  • VPS
  • DreamPress
  • Dedicados (sin un usuario admin)

Sin embargo, tu puedes instalarlo manualmente en un Servidor Dedicado o DreamCompute  pues estos planes permiten el acceso a sudo.

Ajustar las configuraciones predeterminadas de PHP

PHP se instala con configuraciones predeterminadas. Tu puedes ajustar muchas de estas creando o editando un archivo llamado php.ini. Cualquier configuración que agregues a este archivo anula las configuraciones por defecto de PHP del servidor.

El término estándar para este archivo es php.ini, sin embargo, DreamHost usa el término archivo phprc en vez. Esto actúa de la misma manera – solo el nombre es diferente. Revisa este artículo para obtener más información.
