PHP es un lenguaje de secuencias de comandos gratuito que generalmente se combina con HTML para crear sitios web dinámicos e interactivos. PHP puede agregar la siguiente funcionalidad a un sitio web:

  • Crear y manipular archivos en el servidor
  • Guardar información de formularios de contacto
  • Conectar un sitio web a una base de datos
  • Encriptar datos

Este artículo explica como PHP es configurado en DreamHost junto con detalles del plan de soporte extendido disponible.

PHP requiere un servidor

Donde HTML puede ejecutarse en cualquier computador, PHP necesita un servidor web para ejecutarse. Esto significa que debes comprar un plan de alojamiento y cargar tu código PHP en el servidor, o crear un servidor local en el computador de tu hogar para ejecutarlo.

PHP está instalado en todos los servidores de DreamHost por defecto.

Un lenguaje popular para constructores web

Muchas aplicaciones de constructores web son escritos en PHP. Por ejemplo:

  • WordPress
  • MediaWiki
  • Joomla
  • Drupal

Actualizaciones

PHP está siendo actualizado continuamente por sus desarrolladores. En cualquier momento, hay versiones que son soportadas oficialmente por el equipo de desarrollo de PHP y versiones que ya no se mantienen.

Es muy importante asegurarse de que la versión que usa tu sitio sea compatible actualmente con los desarrolladores de PHP. Los beneficios de mantenerse actualizado incluyen:

  • Nuevas características/funcionalidades
  • Mejoras de seguridad
  • Estabilidad y mejoras en rapidez
  • Arreglos de bugs

Soportados / Versiones antiguas

A medida que se lanzan versiones más nuevas, las versiones anteriores se retiran. Cuando una versión anterior de PHP ya no es compatible con sus desarrolladores, ha llegado a su estado de Fin de Vida Útil o (EOL). Puedes encontrar una lista de las versiones actuales de PHP y su calendario EOL en el siguiente enlace:

  • https://www.php.net/supported-versions.php  

Plan de soporte extendido de DreamHost

DreamHost ofrece un plan que soporta las versiones antiguas de PHP que han se encuentran en el estado de Fin de Vida Útil por php.net. Esto te da más tiempo para actualizar la versión de PHP después de que ya no sea oficialmente soportada. Revisa el siguiente artículo para obtener más detalles:

  • Soporte Extendido de PHP

Configuraciones de PHP de DreamHost

Configuración de servidor de PHP

Dreamhost en este momento permite la siguiente configuración de PHP:

  • PHP 8 corriendo como una aplicación de FastCGI application

mod_php no es soportada en los planes de alojamiento administrados. Estos incluyen:

  • Shared
  • VPS
  • DreamPress
  • Dedicados (sin un usuario admin)

Sin embargo, tu puedes instalarlo manualmente en un Servidor Dedicado o DreamCompute  pues estos planes permiten el acceso a sudo.

Ajustar las configuraciones predeterminadas de PHP

PHP se instala con configuraciones predeterminadas. Tu puedes ajustar muchas de estas creando o editando un archivo llamado php.ini. Cualquier configuración que agregues a este archivo anula las configuraciones por defecto de PHP del servidor.

El término estándar para este archivo es php.ini, sin embargo, DreamHost usa el término archivo phprc en vez. Esto actúa de la misma manera – solo el nombre es diferente. Revisa el siguiente artículo para más información:

  • Generalidades de php.ini
