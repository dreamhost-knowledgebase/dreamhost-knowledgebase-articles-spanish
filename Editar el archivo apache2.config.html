Generalidades

Este artículo explica cómo editar el archivo apache2.conf en un Servidor Dedicado. Este archivo es el archivo de configuración principal para los servidores Apache que le permite controlar la configuración global.

Para editar el archivo apache2.config, debes tener un Servidor Dedicado con un usuario admin. El acceso de root/sudo no está disponible en las maquinas VPS.

Editar el archivo apache2.conf

To edit this file:

  1. Inicia sesión en tu servidor a través de SSH.
  2. Navega al siguiente directorio. En este ejemplo, dh123456 es el nombre de tu servidor Dedicado. 
    [server]$ cd /etc/apache2-apache2-ds123456/apache2.conf
  3. Edita este archivo para hacer los cambios que necesites.
