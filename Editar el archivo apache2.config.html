Generalidades

Para editar el archivo apache2.config, debes tener un Servidor Dedicado con un usuario admin.

El acceso de root/sudo no está disponible en las maquinas VPS.

Ubicación del archivo apache2.conf

Si tu Servidor Dedicado ejecuta Ubuntu 22 (Jammy) o Debian Buster, puedes encontrar este archivo en la siguiente ubicación:

/etc/apache2-apache2-INSTANCE/apache2.conf
