Opciones de red privada

Una vez que el soporte técnico de DreamHost haya habilitado las redes privadas, puedes agregar una red privada a tu cuenta. Por favor revisa las siguientes opciones para determinar la configuración de la red privada.

Bloque de red

Hay varios bloques de red privada que están disponibles para su uso con redes privadas que se especifican en formato CIDR Ejemplos comunes de esto son 10.0.0.0/24 o 192.168.0.0/24, sin embargo, también hay otras redes para escoger también. En el siguiente ejemplo, se utiliza la dirección 10.0.0.0/24.

DHCP

Cuando se define una subred, DHCP se puede configurar como habilitado o deshabilitado. También se puede cambiar más tarde si se desea. Cuando DHCP está habilitado, las instancias recién creadas corren cloud-init al inicio y lo detectan, por lo que se determina que no es necesario codificar la configuración de red en el sistema operativo.

Si está deshabilitado, estos ajustes están codificados. Tener DHCP habilitado puede ayudar a crear capturas y nuevas instancias a partir de esas capturas, ya que las capturas no tendrán configuraciones de red codificadas.

Comprobando si se han habilitado las redes privadas

Después de haber solicitado que las Redes Privadas estén habilitadas en tu cuenta, revisa tu panel de DreamCompute para confirmar que está habilitado.

En el panel izquierdo, haz clic en las pestañas Project > Network > Networks. A la derecha verás un botón Crear Red. Si el botón está gris y dice 'Cuota excedida', la red privada aún no se ha habilitado en tu proyecto. Una vez que se haya habilitado, podrás presionar este botón para agregar una red privada.

 

Agregar la red privada

  1. Iniciar sesión en tu panel de DreamCompute.
  2. En el panel de la izquierda, da clic en la pestaña Project > Network > Networks.
  3. Da clic en el botón Create Network. Una nueva ventana aparece:
  4. Ingresa un nombre para la red como "red privada".
  5. Deja las casillas “Habilitar Estado de Administrador” y “Crear Subred” seleccionadas.
  6. Da clic en el botón Next para continuar en la pestaña Subnet.
  7. Un nombre opcional para la Subred puede ser especificado si se desea.
  8. En la pestaña Subnet ingresa el rango CIDR que decidiste arriba para la red privada en el campo “Dirección de Red”.
  9. Cuando termines, da clic en el botón Next para continuar en la pestaña Subnet Details.
  10. Selecciona o deselecciona Enable DHCP dependiendo de la decisión tomada en la sección DHCP anterior.
  11. En el campo de "DNS Name Servers" ingresa los valores 8.8.8.8 y 8.8.4.4 en sus propias líneas.
  12. Da clic en el botón Create.

Crear un enrutador

  1. Navega hasta la pestaña Project > Network > Routers.
  2. Da clic en el botón + Create Router en la parte superior derecha.
  3. Una nueva pestaña aparece. Ingresa un nombre para el enrutador como “enrutador-privado”.
  4. Deja la casilla 'Enable Admin State' seleccionado.
  5. Selecciona “public” del External Network en en menú desplegable.
  6. Selecciona el botón Create Router.
  7. Una vez el router se muestre, da clic en el nombre del enrutador para navegar a la página de detalles del enrutador.
  8. Da clic en la pestaña Interfaces que se muestra en la parte superior izquierda.
  9. Da clic en el botón + Add Interface en la parte superior derecha.
  10. En el menú desplegable Subnet selecciona la red privada creada en los pasos #1-4 de arriba.
  11. Haz clic en el botón Submit.
 

Usar la red privada

Esto completa el proceso de añadir una red privada a la cuenta. Para seleccionar la red privada y agregar una dirección IP flotante, sigue los pasos adicionales de abajo.

  1. Navega a la pestaña Project > Compute > Instances.
  2. Haz clic en el botón Launch Instance en la parte superior derecha.
  3. Completa las secciones Details, Source, Flavor, y Key Pair normalmente.
  4. En la pestaña Networks da clic en la flecha hacia arriba "⇑" para agregar una red privada que has creado en esta instancia.
  5. Haz clic en el botón Launch Instance para lanzar la instancia.

Asociar la IP flotante

  1. Navega a la pestaña Project > Compute > Instances.
  2. A la derecha de la instancia hay un menú desplegable. Haz clic en la flecha hacia abajo para expandirla y selecciona Associate Floating IP.
  3. Proporciona una dirección IP flotante si es necesario.
  4. Si aún no se ha aprovisionado una IP flotante, haz clic en el botón "+" para hacerlo. La ventana de provisión tiene solo un "Grupo" disponible llamado "Público" para seleccionar, y un botón "Asignar IP" para completar el proceso.
  5. Selecciona una IP flotante disponible en el menú desplegable IP Address y la dirección IP privada de la instancia anterior en el menú desplegable Port to be associated.
  6. Haz clic en el botón Associate.
  7. Verifica la asignación de la IP flotante. La dirección IP flotante asignada aparece en la página Project > Compute > Instances en la columna IP Address.
