Crear una Llave de Emparejamiento

Cuando lanzas una nueva instancia, las claves SSH son el único método para conectarte a la nueva instancia la primera vez. Puedes crear e importar una clave SSH en tu proyecto DreamCompute utilizando el siguiente artículo de DreamHost.

  • Generalidades de la creación de una Llave de Emparejamiento

Crea una instancia

Una vez que hayas creado e importado tu clave SSH en el panel de DreamCompute, puedes continuar y crear una instancia utilizando uno de los tutoriales a continuación.

  • Cómo lanzar una instancia usando el panel de DreamCompute
  • Cómo lanzar una instancia usando la CLI de OpenStack

Al crear la instancia, asegúrate de seleccionar la llave de emparejamiento que deseas utilizar en la pestaña Acceso y Seguridad.

Iniciar sesión en tu instancia

Para iniciar sesión, necesitarás el usuario predeterminado y la dirección IP de la instancia.

  • Cómo encontrar el usuario predeterminado de una imagen
  • Ubicación de la dirección IP de tu instancia

Finalmente, selecciona el artículo a continuación que corresponda a tu sistema operativo para obtener instrucciones sobre cómo iniciar sesión con tu Llave de Emparejamiento.

  • Conéctate a tu instancia DreamCompute con claves SSH en Windows
  • Conéctate a tu instancia DreamCompute con claves SSH en Mac o Linux
