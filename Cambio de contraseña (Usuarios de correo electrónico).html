Generalidades

Hay dos lugares para cambiar tu contraseña de correo electrónico:

  • En el panel de tu cuenta de DreamHost
  • En el panel de 'Mailboxes'

Este artículo también explica cómo cambiar la contraseña de una Lista de Discusión.

Usando el panel DreamHost

Para cambiarlo en el panel, continúa con las siguientes instrucciones:

  1. Navega a la página Manage Email.
  2. Haz clic en el botón Edit a la derecha de tu dirección de correo electrónico.
  3. Tienes dos opciones en esta página para crear una nueva contraseña:
    • Haz que el sistema elija una nueva contraseña para tí marcando la casilla 'Pick a new password for me'.
    • Elige tu propia contraseña e ingrésala dos veces.
  4. Haz clic en el botón Save Changes para guardar tu nueva contraseña.

Si eliges que el panel cree una contraseña para tí seleccionando la casilla 'Elegir una nueva contraseña para mí', la nueva contraseña generada por el sistema se mostrará en la página siguiente.

Usar el panel de Mailboxes

Consulta el siguiente artículo para obtener instrucciones sobre cómo iniciar sesión en la interfaz de Mailboxes:

  • Interfaz de Mailboxes

No hay forma de actualizar la contraseña dentro de Webmail. Debes iniciar sesión en Panel de DreamHost o la Interfaz de Mailboxes. Si necesitas dar acceso a tus usuarios de correo electrónico para cambiar su propia contraseña, puedes asignarles privilegios individuales en el panel. Visita el artículo Privilegios de cuenta para más detalles.

Usuario de lista de discusión

Si no tienes acceso a la lista, el soporte de DreamHost debe restablecerlo.

Crea un tiquete en tu panel en la página Contact Support. Asegúrate de incluir la lista para la que necesitas restablecer la contraseña de administrador.

También debes enviar la solicitud de soporte desde tu correo electrónico principal vinculado a tu cuenta. Solo entonces el soporte podrá actualizar la contraseña por tí.
