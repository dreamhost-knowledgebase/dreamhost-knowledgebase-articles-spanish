Generalidades

Si has olvidado o perdido tu contraseña de correo, debes iniciar sesión en el panel de DreamHost (ya sea como propietario principal de la cuenta o como usuario privilegiado) para crear una nueva. Si no tienes acceso al panel DreamHost, comunícate con el propietario de la cuenta para obtener ayuda

Hay dos lugares para cambiar tu contraseña de correo:

  • En el panel de tu cuenta de DreamHost.
  • En el panel de 'Mailboxes'.

Este artículo también explica cómo cambiar la contraseña de una Lista de Discusión.

Usar el panel DreamHost

Antes de empezar

La siguiente sección está también disponible como video instructivo:

Para cambiarlo en el panel, continúa con las siguientes instrucciones:

  1. Navega a la página Administrar Correos.
  2. Haz clic en el botón Edit a la derecha de tu dirección de correo.
  3. Tienes dos opciones en esta página para crear una nueva contraseña:
    • Haz que el sistema elija una nueva contraseña para tí marcando la casilla Pick a new password for me.
    • Elige tu propia contraseña e ingrésala dos veces.
  4. Haz clic en el botón Save Changes para guardar tu nueva contraseña. La nueva contraseña tendrá efecto en unos minutos.

Si eliges que el panel cree una contraseña para tí seleccionando la casilla 'Elegir una nueva contraseña para mí', la nueva contraseña generada por el sistema se mostrará en la página siguiente.

Usar el panel de Mailboxes

Ya debes conocer tu contraseña para usar esta opción. Luego puedes iniciar sesión en el panel de buzones para actualizarla a otra cosa.

Si no recuerdas tu contraseña, la única opción es actualizarla en el panel como se muestra arriba.

Mira el siguiente artículo para obtener instrucciones sobre cómo iniciar sesión en la interfaz de Mailboxes:

  • Interfaz de Mailboxes

No hay forma de actualizar la contraseña dentro de Webmail. Debes iniciar sesión en Panel de DreamHost o la Interfaz de Mailboxes. Si necesitas dar acceso a tus usuarios de correo para cambiar su propia contraseña, puedes asignarles privilegios individuales en el panel. Visita el artículo Privilegios de Cuenta para más detalles.

Usuario de lista de discusión

Si no tienes acceso a la lista, el soporte de DreamHost debe restablecerlo.

Crea un ticket en tu panel en la página Contacta a Soporte. Asegúrate de incluir la lista para la que necesitas restablecer la contraseña de administrador.

También debes enviar la solicitud de soporte desde tu correo principal vinculado a tu cuenta. Solo entonces el soporte podrá actualizar la contraseña por tí.
