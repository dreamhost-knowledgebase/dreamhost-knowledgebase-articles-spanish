Generalidades

Articulo relacionado: Página del panel websites

El botón Manage aparece en la página Websites:

  • Cuando te desplaxas sobre la vista Grid de tu sitio web, o
  • en el extremo derecho en cada linea de la vista List.

Haz clic en el botón azul Manage para abrir la página Manage Websites (excepto para planes DreamPress y Remixer, que en su lugar enlazan a su propia página de administración).

Los siguientes temas se describen en este artículo.:

  • Pantalla superior
  • Pestaña Website
  • Pestaña Hosting

Pantalla superior

La pantalla superior proporciona información de primera línea sobre tu sitio web, herramientas de navegación y dos pestañas (Sitio web y alojamiento) que proporcionan información más detallada.

Regresar a la página Websites (migas de pan)

Haz clic en el enlace Websites en la parte superior para volver a la página de Websites:

Información del plan de alojamiento

La URL de tu sitio web, así como el tipo de plan de alojamiento, se encuentra en la parte superior:

Desplázate sobre la URL de tu sitio web (un iconoaparece al lado) y haz clic para abrir el sitio web seleccionado. La URL del sitio web se vuelve azul al desplazarte sobre el.

Pestañas Website y Hosting

La página Manage Websites tambinén incluye las pestañas Website y Hosting, donde puedes hacer más ajustes a tu sitio web:

Haz clic en cualquiera de ellos para ver configuraciones y características específicas, que se describen en las siguientes secciones.

Pestaña Website

Dependiendo del tipo de plan de alojamiento que tengas y las características que están habilitadas (o deshabilitadas) en él, algunas de las siguientes características pueden no aparecer en tu panel.

Los siguientes elementos aparecen en la pestaña Websites de tu sitio web.

Add Hosting

Si tu sitio web no tiene un plan de alojamiento, aparece lo siguiente en la parte superior:

Haz clic en el botón Add Hosting, el cual abre la página Manage Domains de tu sitio.

Manage Your Site

El cuadro Manage Your Site incluye información sobre cómo acceder a los archivos de tu servidor utilizando tu propio cliente FTP de terceros o el administrador de archivos de DreamHost:

  • Haz clic en el enlace Show Login Info para ver la configuración de tu servidor, incluido el nombre de host, el nombre de usuario y la contraseña, así como un enlace a la página Manage Users.
  • Haz clic en el botón Manage para abrir el file manager.

WordPress

Si WordPress está instalado en tu sitio web, aparece el siguiente cuadro: 

  • Haz clic en el botón Manage para abrir la página wp-admin de tu sitio WordPress.
  • Haz clic en enlace Uninstall WordPress para desinstalar WordPress.
  • Haz clic en el botón Get Key para obtener una llave de migración que te permite usar el plugin de migración automatica de DreamHost

El siguiente cuadro aparece si WordPress aún no está instalado en tu sitio web, lo que te permite instalar WordPress rápidamente haciendo clic en el botón Install:
