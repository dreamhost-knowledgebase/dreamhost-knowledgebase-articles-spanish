Generalidades

Este artículo enumera algunas alternativas para el servicio de correo web de DreamHost.

Alojamiento sin correo de DreamHost

DreamHost ahora ofrece Google Workspace.

Google Workspace es un conjunto de aplicaciones para ayudar a organizar y simplificar el flujo de trabajo de tu organización e incluye un servicio de correo (Gmail). Para obtener más información sobre el uso de Google Workspace en DreamHost, consulta el siguiente artículo:

  • Generalidades sobre Google Workspace

Usar Gmail como cliente de correo

El cliente de correo POP3 Mail Fetcher de Gmail puede conectarse a tus cuentas de correo y enviar los correos del servidor DreamHost a tu cuenta de Gmail. Gmail también puede usar SMTP para enviar correos desde tu cuenta de correo.

Instalar tu propio cliente de correo

La mayoría del software de correo web requiere acceso a sudo, que solo está disponible a través de DreamCompute o un Servidor Dedicado.

El equipo de soporte de DreamHost no puede ayudar con ninguna instalación personalizada ni puede solucionar los errores que puedan ocurrir como resultado de una instalación personalizada.

También es posible configurar cualquier cliente de correo IMAP basado en la web como reemplazo de la interfaz predeterminada de Webmail después de deshabilitar DreamHost webmail. Esto desligará el subdominio de Webmail para que puedas usarlo con tu propio cliente de Webmail.

  1. Naviega a la página Webmail.
  2. Hacia el lado derecho de tu dominio, haz clic en el botón Edit. 
  3. Marca la casilla titulada 'Disable Webmail'.
  4. Haz clic en el botón Save Changes.

Luego puedes instalar tu propia versión de aplicaciones alternativas como SquirrelMail, Mailpile, o Open Webmail.
