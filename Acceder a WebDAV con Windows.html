Generalidades

La conexión a tu directorio WebDAV se puede realizar a través del Explorador de Windows de Microsoft. Esto monta la carpeta WebDAV como una unidad asignada y te permite ver, editar y eliminar archivos del servidor web remoto desde tu computadora.

Windows 7

  1. Haz clic el botón de Inicio.
  2. Selecciona 'Equipo'.
  3. Haz clic en Asignar unidad de red.
    • A partir de este momento, los pasos son los mismos que los de Windows 8.
  4. Mira las instrucciones a continuación (a partir del paso #4) para conectarse a WebDAV.

Windows 8 & 10

  1. Abre el Explorador de archivos de Windows.
    En Windows 8, 'Equipo' se llama 'Este PC':
  2. Haz clic derecho en 'Este PC'.
  3. En el menú desplegable, selecciona Asignar unidad de red ....
  4. Introduce lo siguiente:
    • Unidad: Selecciona la letra de unidad a la que deseas asignar tu conexión WebDAV.
    • Carpeta: Esta es la carpeta en tu computadora local donde agregarás/editarás archivos WebDAV.
  5. Selecciona las cajas:
    • Reconnect at sign-in – Conectar de nuevo al iniciar sesión: Esto permite que la conexión a tu WebDAV se restaure si reinicias tu computadora.
    • Conectar usando diferentes credenciales.
  6. Haz clic en el enlace Connect to a Web site that you can use to store your documents and pictures.
  7. Haz clic en el botón Siguiente.
  8. Haz clic en Elegir una ubicación de red personalizada y luego haz clic en el botón Siguiente.
  9. En el campo 'Dirección de Internet o de red', ingresa la URL de tu dominio con el directorio WebDAV al final.
  10. Ingresa tu nombre de usuario y contraseña de WebDAV en el cuadro emergente, y luego presiona el botón OK.
    El cuadro titulado 'Escriba un nombre para esta ubicación de red' ya está lleno y puedes cambiarlo si lo deseas:
  11. Haz clic en el botón Siguiente.
  12. Haz clic en el botón Finalizar.

Yu carpeta WebDAV aparece como una unidad asignada en el Explorador de Windows, y puedes agregar, editar o eliminar archivos y directorios inmediatamente usando el Explorador de archivos de Windows.

Solución de problemas con Windows

Si aún tienes problemas para conectarte a tu directorio WebDAV, es posible que debas actualizar el Nivel de Autenticación Básico en el registro. Visita la siguiente página para obtener más información sobre cómo acceder al registro:

  • Registro de Windows

Para cambiar el Nivel de Autenticación Básico:

  1. Localiza el siguiente directorio:
    HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\WebClient\Parameters
  2. Localiza el valor BasicAuthLevel.
    • El Valor en la caja de datos se debe establecer a 2.
  3. Si el valor no es 2, haz clic con el botón derecho y luego selecciona 'Modificar'.
  4. Cambiarlo a 2.
