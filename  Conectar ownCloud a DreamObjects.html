Generalidades

ownCloud es una aplicación de alojamiento de archivos de código abierto y gratuita que proporciona acceso universal a archivos, sincronización y uso compartido de tus datos (utilizando una única interfaz), independientemente de dónde se encuentren los datos. Puedes acceder, sincronizar y compartir casi cualquier cosa, como contactos, calendarios, marcadores y copias de seguridad dentro de ownCloud a través de la web, tu computadora o dispositivos móviles.

ownCloud te permite usar DreamObjects como una opción de almacenamiento externo, lo que te da total libertad y control sobre tus datos.

Esta guía describe cómo configurar ownCloud para conectarse al servicio de almacenamiento en la nube DreamObjects de DreamHost.

Cómo configurar ownCloud para funcionar con DreamObjects

Antes de que puedas comenzar a usar ownCloud con DreamObjects, debes instalar ownCloud en tu servidor DreamHost siguiendo esta guía paso a paso.

  • Cómo instalar ownCloud

También debes habilitar DreamObjects en tu cuenta DreamHost. Si no has habilitado DreamObjects ni has creado las claves y buckets necesarios, consulta los siguientes artículos:

  • Crear un usuario de DreamObjects
  • Crear Llaves de DreamObjects

Configurar ownCloud

Después de instalar ownCloud, puedes configurarlo para que se sincronice con tus buckets de DreamObjects.

Para configurar:

  1. Abre ownCloud.
  2. Inicia sesión en tu instalación de ownCloud utilizando tu nombre de usuario y contraseña de administrador de ownCloud.
  3. En la izquierda superior, haz clic en el icono y luego selecciona Market.
  4. En la página Market, selecciona Storage del menú de la barra izquierda.
  5. Busca la opción External Storage: S3 en la siguiente página y haz clic en ella.
  6. Desplázate hacia abajo y haz clic en el botón Install.
  7. En la derecha arriba, haz clic en tu nombre de usuario y luego selecciona Settings.
  8. En la página Configuración, realiza las siguientes selecciones:
    • En el panel izquierdo, selecciona Storage.
    • Selecciona la casilla llamada Enable external storage para conectarte a DreamObjects.
  9. Haz clic en el menú desplegable llamado Add storage y selecciona Amazon S3.
  10. Configura tus ajustes de almacenamiento en las siguientes secciones de configuración:
    • Folder name.
    • Configuration.
    • Available for.

    Folder name

    • Bajo Folder name, ingresa el nombre de tu carpeta ownCloud. Este será el nombre de la carpeta en el que vas a guardar tus datos y puede tener el nombre que desees.

    Configuration

    • Bucket name — El nombre de tu bucket de DreamObjects.

      El nombre de tu bucket no puede contener un guion bajo, ya que ownCloud fallará al conectarse.

    • Hostname — objects-us-east-1.dream.io
    • Port — Déjalo en blanco.
    • Region — us-east-1
    • Enable SSL — Selecciona esta casilla.
    • Enable Path Style — Selecciona esta casilla.
    • Access key —  Tu Llave de acceso de DreamObjects .
    • Secret key — Tu Llave secreta de DreamObjects .

Available for

  • Si tienes multiples usuarios de tu instalación ownCloud, puedes limitar quien puede acceder la carpeta al escribir users o groups en la caja Available fora la derecha de la información de configuración.

Si ingresaste todo correctamente, aparecerá un círculo verde a la izquierda del nombre de tu carpeta.

Ver tu carpeta y objetos

Para ver tus archivos, haz clic en Settings a la derecha arriba, luego haz clic en Files.

En el panel central, verás el nombre de la carpeta que creaste. Todo lo que guardes, cargues o crees allí se almacena en DreamObjects. Eliminar un objeto dentro de ownCloud elimina el objeto de DreamObjects.

Si deseas conectarte a buckets adicionales, simplemente crea una nueva carpeta y conecta este bucket a la carpeta a través de la sección Admin > External Storage del panel ownCloud. Puedes conectar tantos buckets como gustes con ownCloud.

Cargar archivos grandes dentro de ownCloud a DreamObjects

Si planeas cargar archivos grandes a DreamObjects usando ownCloud, deberás tener en cuenta algunas limitaciones y realizar algunos cambios de configuración, ya que cargar archivos grandes puede consumir mucha memoria.

Los objetos de DreamObjects tienen un tamaño limitado de 10 TB, pero deben cargarse en fragmentos de 5 GB. ownCloud no admite cargas de archivos de varias partes, por lo que está limitado a cargas de archivos de 5 GB a través de ownCloud.

Personalizar las configuraciones de PHP

ownCloud usa diferentes archivos de configuración de PHP dependiendo de dónde se ejecute el comando PHP.

  • phprc - El arcivo phprc es el archivo DreamHost estándar que se usa para personalizar la configuración de PHP. Este archivo ajusta la configuración de PHP para los comandos de Shell que se ejecutan en el servidor. Esto incluiría servicios como cron jobs. Mira los siguientes artículos para obtener instrucciones sobre cómo editar este archivo:
    • Crear un archivo phprc vía FTP (más fácil para principiantes)
    • Crear un archivo phprc vía SSH (para usuarios avanzados)
  • .user.ini - El archivo .user.ini es un archivo ownCloud que controla la configuración de PHP usada dentro del panel de control de tu sitio web. Esto incluiría la carga de archivos y los límites de tiempo de ejecución al interactuar con tu sitio. Este archivo se encuentra en la carpeta owncloud en el servidor y configura los siguiente paraámetros de PHP: 
    upload_max_filesize=513M
    post_max_size=513M
    memory_limit=512M
    mbstring.func_overload=0
    always_populate_raw_post_data=-1
    default_charset='UTF-8'
    output_buffering=0
