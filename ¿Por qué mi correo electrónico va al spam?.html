Generalidades

Si observas que los correos electrónicos enviados o recibidos, especialmente dentro del mismo dominio, están llegando a una carpeta de correo no deseado, puede deberse a la dirección IP de tu proveedor de servicios de Internet (ISP) que está en la lista negra. Los filtros antispam de DreamHost están diseñados para marcar el correo electrónico como spam cuando se originan en una dirección IP en la lista negra.

A continuación se proporcionan los pasos que puedes seguir para determinar si tu dirección IP está en una lista negra.

Buscando tu dirección IP

  1. Visita whatismyipaddress.com para buscar la dirección IP actual de tu hogar.
  2. Usa los siguientes sitios para verificar si tu dirección IP está en una lista negra:
    • multirbl.valli.org
    • zy0.de
  3. Si tu dirección IP está en una lista negra, comunícate con tu ISP (la compañía que te proporciona acceso a Internet) para determinar los pasos que puedes seguir para removerla.

¿Qué sucede si mi dirección IP no está en una lista negra?

Si tu dirección IP no está en la lista negra, copia los encabezados del mensaje que terminó en tu carpeta de spam y pégalo en un tiquete de soporte en tu panel DreamHost. Para obtener más información sobre cómo ver encabezados de correo electrónico, consulta el siguiente artículo:

  • ¿Cómo veo los encabezados de correo?

Asegúrate de explicar el problema que está experimentando para que el soporte pueda ayudarte más.
