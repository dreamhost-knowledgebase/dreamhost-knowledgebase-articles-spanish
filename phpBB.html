Generalidades

phpBB (PHP Bulletin Board) es un programa de foro que te permite crear un espacio en tu sitio web para que los miembros hablen sobre diferentes temas e ideas o cualquier cosa que desees. Puedes visitar el sitio web de phpBB para obtener más detalles y ver ejemplos de su uso.

  • Sitio oficial de phpBB

Instalando

phpBB está disponible para los usuarios de DreamHost como Instalación de un solo clic.

Una vez instalado, se envía un correo electrónico a la dirección de correo electrónico principal que figura en tu cuenta DreamHost. Este correo electrónico contiene toda la información de tu base de datos e instrucciones completas sobre cómo instalar. Puedes comenzar visitando tu sitio phpBB en:

  • example.com/install 

Después de ingresar la información de tu base de datos y hacer clic en Proceed to next step, puede que veas el siguiente error:

Esto solo significa que el nombre de host de tu base de datos se acaba de crear y necesita unas horas para que el DNS se actualice en línea. Puedes esperar unas 4 horas hasta que se actualice, o puedes usar otro nombre de host en tu cuenta. Consulta la página MySQL Databases para obtener una lista de tus nombres de host existentes.

Todos son intercambiables, por lo que cualquiera de ellos funcionará para cualquier base de datos en tu cuenta.

Actualizando

Antes de realizar cualquier ajuste en tu sitio, siempre es una buena idea hacer una copia de seguridad de tus datos. Ve el siguiente artículo para hacer una copia de seguridad de tus bases de datos y archivos de tu sitio:

  • Copia de Seguridad

Si tu instalación de phpBB es una instalación de un solo clic, puedes actualizarlo en tu panel. Visita el artículo Cómo actualizar una instalación de un solo clic para más de talles.

Si estás ejecutando una instalación manual, consulta la guía de actualización de phpBB para instrucciones detalladas.
