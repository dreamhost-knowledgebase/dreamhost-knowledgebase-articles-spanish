Generalidades

DreamHost proporciona varios servicios para realizar cambios en las configuraciones que normalmente serían manejados por un desarrollador o mantenedor de un sitio web. Si te encuentras en una situación que requiere cambios en tu sitio y tu desarrollador no está disponible para ayudar, el equipo de servicios profesionales de DreamHost puede ayudarte.

Para pedir uno de los servicios listados abajo, por favor genera un tiquete en la página Contact Support indicando los detalles de tu solicitud, incluyendo el dominio y el servicio profesional en el que estás interesado.

Puedes encontrar todos los servicios profesionales y las tarifas asociadas enumerados a continuación.

WordPress

Cambiar las URLs de un sitio WordPress

Precio: $19.99

Si has cambiado tu nombre de dominio o estás cambiando a un sitio habilitado para https, el equipo de soporte de DreamHost puede cambiar la URL de tu sitio de WordPress por ti.

Para enviar una solicitud de cambio de URL de WordPress, contacta el equipo de soporte. Por favor, incluye el dominio/URL desde el que estás cambiando y el nuevo dominio/URL que deseas utilizar. El equipo de soporte de DreamHost responderá confirmando el cargo de $19.99 y el trabajo realizado.

Actualizar WordPress

Precio: $19.99

Si estás utilizando una versión anterior de WordPress y te preocupa tener problemas al actualizar a una versión más nueva, el soporte de DreamHost puede actualizarlo por ti.

El soporte de DreamHost puede actualizar todos los temas y plugins que forman parte de los repositorios oficiales de temas y plugins de WordPress.org.

Para enviar una solicitud de actualización de WordPress, contacta el equipo de soporte. Por favor Incluye el sitio que deseas actualizar. El equipo de soporte de DreamHost revisará tu sitio para asegurarse de que cumpla con los requisitos de elegibilidad y responderá confirmando el cargo de $19.99 y el trabajo que se está realizando.

Optimización de Base de datos de WordPress

Precio: $19.99

Con el tiempo, una base de datos de WordPress puede acumular gastos generales innecesarios que pueden ralentizar tu sitio o impactar el rendimiento del sitio. Si esto ocurre, el soporte de DreamHost puede optimizar tu base de datos de WordPress por ti.

Para enviar una solicitud para optimizar la base de datos de tu sitio de WordPress, contacta el equipo de soporte. Por favor, incluye el sitio y, si es posible, la base de datos que deseas optimizar. El equipo de soporte de DreamHost revisará tu sitio y base de datos y confirmará el cargo de $19.99 por el servicio.

Importar un WP XML / demo data

Precio: $19.99

Si tienes una copia de seguridad XML de los datos de tu sitio (de wordpress.com o similar), DreamHost puede importar los datos XML por ti.

Cuando creas una copia de seguridad de tu sitio web alojado en Wordpress.com, el sitio exportado está en formato XML. Solo se pueden exportar publicaciones, páginas, comentarios, categorías y etiquetas. Los temas y widgets no se pueden exportar. Si necesitas exportar tu base de datos, debe exportarse por separado. Revisa el siguiente artículo para obtener más detalles:

  • Exportar una base de datos de WordPress.com

PHP y MySQL

Valores personalizados de php.ini

Precio: $29.99

Las diferentes aplicaciones basadas en PHP pueden tener valores específicos de php.ini que deben personalizarse en el archivo phprc de tu sitio. Si nunca has realizado estos cambios, esto puede ser un poco confuso y volverse difícil de hacer por tu cuenta. 

Contacta el equipo de soporte para enviar una solicitud para modificar tus valores de php.ini. Incluye el sitio y los cambios exactos que necesitas, o un enlace a los requisitos del sistema para la aplicación que debe modificarse. El equipo de soporte de DreamHost revisará tu solicitud, el trabajo necesario y luego confirmará el cargo de $29.99 por el servicio.

Importar una Base de datos MySQL

Precio: $19.99

Si estás importando datos de un sitio anterior o una empresa de alojamiento, es posible que necesites una base de datos MySQL importada a tu cuenta DreamHost.

Para enviar una solicitud para importar una base de datos MySQL, contacta el equipo de soporte. Por favor Incluye la ubicación de tu archivo .sql exportado y cualquier información que consideres que el equipo pueda necesitar al importar la base de datos. El equipo de soporte de DreamHost revisará tu solicitud y confirmará el cargo de $19.99 por el servicio.

Instalando aplicaciones de terceros

Precio: $29.99

Si deseas utilizar una aplicación web de terceros y no te sientes cómodo utilizando la línea de comandos (SSH), los Servicios profesionales de DreamHost pueden ayudarte a instalarla.

DNS

Añadir registros personalizados de DNS 

Precio: $19.99

Si necesitas ayuda para agregar un registro DNS en tu cuenta DreamHost para dirigir tu dominio a otro host o servicio, el soporte de DreamHost puede hacerlo por ti.

Para enviar una solicitud para agregar registros DNS personalizados a tu cuenta DreamHost para utilizar otros servicios, contacta el equipo de soporte. Por favor incluye el dominio y los registros DNS que deseas cambiar. El equipo de soporte de DreamHost revisará tu solicitud y confirmará el cargo de $19.99.

SSL

Importar SSLs externos por un cliente

Precio: $19.99

Si tienes un certificado SSL externo y no puedes encontrar la manera de pegarlo correctamente en el Panel Web DreamHost, el equipo de Servicios Profesionales de DreamHost puede ayudarte con eso.
