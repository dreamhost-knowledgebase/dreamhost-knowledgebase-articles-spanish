Introducción

Las instancias de DreamCompute se crean con una clave SSH específica que se utiliza para obtener acceso inicial. Esta guía cubre algunos pasos básicos para intentar recuperar los datos y obtener acceso nuevamente si lo deseas.

Cosas para tener en cuenta

  • Las direcciones IP cambiarán

    Para volver a una instancia es necesario volver a crearla. La nueva instancia recibe nuevas direcciones IPv4 e IPv6.

  • Conoce el tipo de tu instancia

    Por lo general, se puede averiguar si una instancia utiliza un disco de arranque efímero o de volumen mirando la página 'Instances' del panel de control y viendo si el nombre de una imagen personalizada o un sistema operativo aparece en la columna Nombre de la Imagen. Si aparece uno, la instancia es efímera. Si no lo es, usa un disco de arranque por volumen.

Recuperar el acceso a una instancia efímera

  1. Navega a la página Instancias del panel de DreamCompute.
  2. A la derecha de la instancia, haz clic en el botón Crear Captura y asigna un nombre a la captura. Esto crea una imagen del sistema operativo de tu instancia en ejecución. Espera a que se complete este proceso antes de continuar con el siguiente paso (por lo general, no toma más de unos minutos).
  3. Toma nota del flavor de la instancia, el nombre de la instancia y cualquier otra característica que deba conservarse en la nueva instancia.
  4. Haz clic en el botón Lanzar Instancia para crear una nueva instancia. En la pestaña Fuente, pon Seleccionar Fuente de Inicio en Captura de Instancia y haz clic en el símbolo de flecha arriba junto al nombre de la capturacreado en el paso anterior. Ten en cuenta que no puedes crear un nuevo volumen a partir de una captura de instancia en DreamCompute, así que asegúrate de establecer Crear Nuevo Volumen como No.
  5. En la pestaña Llave de Emparejamiento, selecciona la llave de emparejamiento para usar con el usuario por defecto. Si no hay claves válidas disponibles, puedes usar los botones Crear Llave de Emparejamiento o Importar Llave de Emparejamiento para asociar un nuevo par de claves con tu proyecto (y esta instancia). Cuando hayas terminado de personalizar la instancia, haz clic en el botón Lanza en la parte inferior derecha.
  6. Una vez que se haya iniciado la nueva instancia, intenta iniciar sesión con el nombre de usuario por defecto y la nueva dirección IP.
  7. Si todo parece correcto y se ha restaurado el acceso, ahora se puede eliminar la instancia original.
