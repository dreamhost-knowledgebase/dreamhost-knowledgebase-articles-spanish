Articulo relacionado: Usando la nueva página Manage Websites

La página del panel Websites La página del panel proporciona una presentación gráfica mejorada de todos tus dominios, ofreciéndole una vista simplificada donde puedes realizar fácilmente ajustes de funciones y ajustes a cualquiera de tus dominios en DreamHost.

Los siguientes temas se describen en este artículo:

  • Abriendo la página Websites
  • La barra de navegación superior
  • Acciones que puedes realizar en vista Grid o List

Abriendo la página Websites

Navega a la nueva página Websites en tu panel:

La página Websites se abre en tu panel, mostrándote todos tus dominios en una pantalla gráfica.

La barra de navegación superior

Lo siguiente describe cómo usar cada uno de los artículos mostrados en la barra de navegación superior.

Búsqueda

El campo de búsqueda te permite buscar cualquier sitio web en tu cuenta:

  • Haz clic en el icono de lupa y la barra de búsqueda se expande.
  • Empieza a escribir y cualquier nombre de sitio web que coincida se muestran inmediatamente.

Vista Grid o List

Puedes ver tus sitios web en cualquier vista Grid o List. Lo siguiente describe cada configuración de pantalla.

Vista Grid

La vista de cuadrícula muestra información básica para todos tus dominios en un formato de cuadrícula gráfica (la captura de pantalla de tu(s) sitio(s)) se actualiza cada 24 horas):

Vista List

  • Haz clic en el botón + Add Website para abrir la página Manage Domains donde puedes añadir alojamiento a un dominio.

Acciones que puedes realizar en cualquier vista grid o list

Dependiendo de la configuración de tu sitio web, puedes hacer varios ajustes en la vista de Grid o List. A continuación se describen todas las opciones disponibles en cada vista.

Grid

Descripción de artículos de Grid

  • Enlace a tu sitio web — Desplázate sobre la URL de tu sitio web que se muestra como exampledreamhost.com y example.com en los ejemplos anteriores. Unicono aparece junto a el. Haz clic para abrir el sitio web seleccionado. El enlace del sitio web también se vuelve azul al desplazarte sobre el.
  • o  (certificado SSL) — Muestra ya sea un () candado cerrado o un () candado sin cerrar. Un candado (verde) cerrado indica que el sitio web tiene un certificado SSL instalado y es seguro, y un candado (rojo) sin cerrar indica que no hay un certificado instalado y que no es seguro. Cuando haces clic en el candado desbloqueado, se abre un cuadro de diálogo con un enlace a la página SSL/TLS Certificates.
  • Plan de alojamiento de sitio web (Mostrado como DreamPress Advanced y Shared Unlimited en los ejemplos arriba) — El tipo de plan de alojamiento aparece debajo de la URL del sitio web. Por ejemplo: Shared Unlimited, VPS, DreamPress o Remixer (por nombrar algunos). También puede mostrar la configuración del dominio para el sitio web, como Mirror, DNS Only o Redirect.
  •  Si se trata de un sitio de WordPress, el logotipo   aparece en la esquina superior derecha del cuadro de cuadrícula. Haz clic en el botón WordPress para abrir la página wp-admin para tu sitio de WordPress.
  • (Email) — El icono  muestra si el correo electrónico está configurado en el sitio web o no, y si es así, cuántas direcciones de correo electrónico se le agregan.
  •  (DreamShield) — El logotipo   aparece si DreamShield Malware Remover está instalado.
  • Manage Desplázate sobre tu sitio web, y el botón Manage aparecerá. Abre la página Manage Websites de tu sitio web o la página Managed WordPress (si es un sitio DreamPress).
  • (Files) — Abre el administrador de archivos para el sitio web.
  • Edit Website (No se muestra en el ejemplo anterior) — Si es un sitio Remixer, el botón Edit Website aparece, el cual abre el panel de Remixer.
  • (No se muestra en el ejemplo anterior) — Si no hay un plan de alojamient en tu sitio, el botón Add Hosting aparece, el cual abre la página Manage Domains donde puedes agregar alojamiento a tu dominio.
  • (más información) — Haz clic en el icono de tres puntos () para abrir un menú que provee enlaces a varias áreas del panel web de DreamHost, incluyendo Registro de Dominio, Configuración DNS, Plan de alojamiento, Administrador de correo y más.

List

Descripción de elementos de la columna List

  • Domain Name — El enlace a tu sitio web. Desplazate sobre la URL de tu sitio web (un icono   aparece junto a el) y haz clic en el para abrir el sitio web seleccionado. El enlace del sitio web también se vuelve azul cuando te desplazas sobre el.
  • Registration — La fecha de expiración del dominio de tu sitio web.
  • Hosting Plan — El tipo de plan de alojamiento para tu sitio web. Por ejemplo: Shared Unlimited, VPS, DreamPress, o Remixer (para nombrar algunos). También puede mostrar la configuración del dominio para el sitio web, como Mirror, DNS Only o Redirect.
  • PHP — La versión PHP actualmente configurada en tu sitio.
  • SSL — Muestra un () candado cerrado o un () candado SIN cerrar. Cerrado (verde) indica que el sitio web tiene un certificado SSL instalado y es seguro, y sin cerrar (rojo) indica que no hay ningún certificado instalado y que no es seguro. Cuando haces clic en el candado sin cerrar, se abre un cuadro de diálogo con un enlace a la página SSL/TLS Certificates.
  • DreamShield — Indica si DreamShield Malware Remover está instalado () o no instalado (). Si DreamShield no está disponible en un dominio (p. Ej., Es solo DNS, Redirigir o no tiene hosting), aparece una marca '' en esta columna.
  • Email — Indica el número de direcciones de correo electrónico configuradas en el dominio. Un '0' indica que todavía no hay direcciones de correo electrónico configuradas.
  • Links — Hay varios tipos de iconos que aparecen en la columna Links:
    • — Abre la página Manage Domains dónde puedes añadir un plan de alojamiento.
    • — Abre la página DNS donde puedes hacer ajustes a los registros DNS de tu dominio.
    • — Abre la página administrador de archivos para tu sitio web, donde puedes conectarte a tu servidor web para ver, subir, descargar archivos y más.
    • — Abre Remixer.
    • — Abre la página wp-admin de tu sitio WordPress.
  • Columns — Puedes solo ajustar qué elementos de la columna mostrar al hacer clic en el icono  al extremo derecho.
  • Manage — Abre la página Manage Websites de tu sitio o la página Managed WordPress (si es un sitio DreamPress).
  • More information () — Haz clic en el icono de tres puntos () en el extremo derecho, el cual abre un menu que provee más enlaces para varias áreas del panel web de DreamHost, incluidos el Registro de Dominio, la configuración de DNS, el Plan de Alojamiento, la Administración de correo electrónico y más (las opciones disponibles se basan en el plan de alojamiento actual).
