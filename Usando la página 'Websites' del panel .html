Generalidades

La página del panel 'websites' proporciona una presentación gráfica mejorada de todos tus dominios, ofreciéndote una vista simplificada donde puedes realizar fácilmente ajustes de funciones y ajustes a cualquiera de tus dominios en DreamHost.

Los siguientes temas se describen en este artículo.

  • Abriendo la página 'Websites'
  • La barra de navegación superior
  • Ver tus sitios web usando la vista de cuadrícula o lista
  • Detalles del sitio web
  • Acciones que puedes realizar en la cuadrícula o en la vista de lista

Abriendo la página 'websites'

Navegue a la nueva página Websites  en tu panel.

  • Cuentas nuevas no verán el enlace 'Manage Domains' en el panel. Si necesitas acceder a esa página, haz clic en Manage Domains. Las cuentas nuevas también pueden acceder a la página 'Manage Domains' desde numerosos enlaces en la página del panel 'websites'.
  • Cuentas antiguas verás tanto el enlace Manage Domains como el nuevo enlace Websites (como se ve en la siguiente imagen). Puedes usar cualquiera para administrar tus dominios.

La barra de navegación superior

A continuación se describe cómo usar cada uno de los elementos que se muestran en la barra de navegación superior:

Usando la búsqueda

El campo de búsqueda te permite buscar cualquier sitio web en tu cuenta.

  • Haz clic en el icono de lupa y la barra de búsqueda se expande.
  • Comienza a escribir y cualquier nombre de sitio web coincidente se mostrará de inmediato

Acciones que puede realizar en la cuadrícula o en la vista de lista

Dependiendo de la configuración de tu sitio web, las siguientes opciones están disponibles que te permiten realizar más ajustes a un sitio web:

  • Enlace a tu sitio web — Desplázate sobre la URL de tu sitio web (unicono aparece al lado) o en cualquier parte del panel de vista previa (en la vista de cuadrícula) y haz clic para abrir el sitio web seleccionado. La URL del sitio web también se vuelve azul al pasar el mouse por encima.
  • Plan de alojamiento de sitios web — El tipo de plan de alojamiento aparece junto a la URL del sitio web. Por ejemplo: Shared Unlimited, VPS, DreamPress o Remixer (por nombrar algunos). También puede mostrar la configuración del dominio para el sitio web, como Mirror, DNS Only, Redirect o Cloaked (por nombrar algunos).
  • WordPress — Abre la página wp-admin para tu sitio de WordPress (ellogotipo aparece en la esquina superior derecha del cuadro de cuadrícula si es un sitio web de WordPress).
  • Manage — Abre la página Manage Domains para tu sitio web o la página Managed WordPress (si es un sitio de DreamPress).
  • My Files — Abre WebFTP para tu sitio.
  • Edit Website — Abre Remixer para permitirte editar el sitio (si es un sitio web Remixer).
  • Add Hosting — Abre la página Manage Domains page, donde puedes agregar hosting a un dominio (si no existe ninguno).

Agregar un sitio web

  • Haz clic en el botón + Add Website para abrir la página Manage Domains, donde puedes agregar alojamiento a un dominio.

Detalles del sitio web

Hay tres formas de ver detalles adicionales sobre un sitio web:

  • Al hacer clic en el icono  en la parte superior, que muestra más información sobre todos tus sitios web. Haz clic en el mismo icono para ocultar los detalles de todos tus sitios web.
  • En la vista de cuadrícula: haciendo clic en el enlace '' dentro de cada cuadro del sitio web, which shows more information about that specific website. que muestra más información sobre ese sitio web específico. Vuelve a hacer clic en el enlace para ocultar los detalles del sitio web.
  • En la vista de lista: haciendo clic en el ícono en el extremo derecho de la lista de sitios web, que muestra más información sobre ese sitio web específico. Vuelve a hacer clic en el icono para ocultar los detalles del sitio web.

Iconos que aparecen junto a la URL de tu sitio web en la vista de lista

Varios iconos de colores aparecen a la izquierda de la URL de tu sitio web, lo que indica que el dominio está:

  • — Fully hosted
  • — Mirrored
  • — Redirected
  • — Cloaked
  • — Parked
  • — DNS only

Acciones que puedes realizar en la cuadrícula o en la vista de lista

Puedes hacer más ajustes a tu sitio web haciendo clic en los siguientes enlaces de configuración (dependiendo del plan de alojamiento de tu sitio web):

  • Change Plan — Abre la página del plan de alojamiento actual (por ejemplo, DreamPress, VPS, o Manage Account) donde puede hacer ajustes a tu plan actual.
  • View DNS Records (el enlace se vuelve azul al pasar el mouse) — Abre la página Manage Domains donde puedes ajustar los registros DNS para el dominio del sitio web.
  • PHP Version — Muestra la versión actual de PHP para el sitio web.
  • SSL Certificate — Muestra Secure (en verde) o Not Secure (en rojo). Si Not Secure, haz clic en el enlace 'Add', que abre la página Add Certificate.
  • Malware Remover — Muestra Active (en verde) o Not Active (en gris). Haz clic en 'Manage' (para administrar tu plan actual) o 'Add' para habilitar Malware Remover.
  • Domain — Muestra la fecha de renovación del dominio del sitio web. Alternativamente, puede mostrar que está 'Expired' (en rojo) o 'Expires soon' (en naranja). Haz clic en el enlace 'Manage', 'Renew', o 'Transfer', que abre la página Registrations o Reg. Transfer  (según el estado del dominio).
  • Email — Muestra el número de direcciones de correo electrónico que están configuradas actualmente en el dominio de tu sitio web. Haz clic en el enlace 'Obtener correo electrónico', 'Agregar correo electrónico' o 'Administrar' (según el tipo de plan de correo electrónico y el estado), que abre la página Manage Email.
