Generalidades

Este artículo describe los diferentes planes de alojamiento disponibles en DreamHost.

Preguntas Frecuentes de Alojamiento

¿Qué es un plan de alojamiento

Un plan de alojamiento es un producto que debes comprar para que tu sitio web se muestre en línea. Este plan de alojamiento te proporciona un servidor en el que luego puedes subir tu sitio web. Si los registros DNS de tu sitio web están apuntados a DreamHost, tu sitio se cargará en línea desde tu nuevo servidor de DreamHost.

¿Cuál es la diferencia entre estos planes de alojamiento?

La principal diferencia es qué recursos proporciona el servidor y a qué personalizaciones necesitas acceder. La mayoría de los sitios funcionan muy bien en el plan básico de Alojamiento Shared. Sin embargo, si tu sitio recibe una gran cantidad de tráfico o si necesitas realizar personalizaciones que no están disponibles para los usuarios compartidos, es posible que necesites actualizar a un plan VPS, Dedicado, o DreamPress.

¿Cómo se mide el uso del ancho de banda?

El uso del ancho de banda en DreamHost es ilimitado en todos los planes de alojamiento, excepto para los servicios en la nube.

¿Qué pasa si mi dominio está registrado en otra compañía?

Tienes dos opciones:

  • Apuntar tu DNS a DreamHost en tu compañía actual
  • Transferir el registro a tu cuenta de DreamHost

¿Cuál es la política de reembolso?

Revisa el artículo de Reembolsos y Créditos de Cuenta para obtener más detalles.

Planes de alojamiento de sitios Web

Los siguientes planes de alojamiento te permiten subir tu sitio web para que aparezca en línea.

Alojamiento Shared Starter

Este es un plan de Alojamiento Shared introductorio dirigido a nuevos usuarios de alojamiento web. También es una solución perfecta para un sitio web personal o empresarial que muestra información simple sobre ti o los servicios que ofreces.

Limitaciones: Dado que este es un plan inicial, existen las siguientes limitaciones:

Estas NO se aplican al plan Shared regular.

  • 1 dominio que puedes usar para tu sitio web (e.g., example.com)
  • 5 subdominios de tu único nombre de dominio (e.g., blog.example.com)
  • El correo electrónico no está incluido (un plan de correo electrónico de pago está disponible como complemento)
  • El almacenamiento del sitio está limitado a 50GB
  • Un máximo de 6 usuarios SFTP o Shell
  • Un máximo de 6 bases de datos MySQL

Precios: Revisa la página Alojamiento Shared para ver los precios actuales.

Agregar este plan: Revisa el artículo Agregar un plan de alojamiento Shared Starter para obtener instrucciones sobre cómo agregar este plan.

Alojamiento Shared

El Alojamiento Shared es una excelente opción para la mayoría de los sitios web. Este plan es perfecto para blogs, portafolios, sitios personales/empresariales y sitios impulsados por bases de datos.

Limitaciones: En un servidor compartido, tu sitio web "compartirá" los recursos del servidor con otros clientes. Esto significa que si otro sitio web en el mismo servidor recibe una cantidad inusualmente grande de tráfico y usa una gran cantidad de recursos de CPU, tu sitio podría potencialmente funcionar más lento como resultado. Aunque DreamHost trabaja arduamente para garantizar que los sitios de los clientes nunca se vean interrumpidos, esto siempre es una posibilidad debido a la naturaleza del Alojamiento Shared.

Descuento para Asociaciones sin Ánimo de Lucro: Los planes de Alojamiento Shared ofrecen un descuento para organizaciones sin fines de lucro gratuito para organizaciones benéficas sin fines de lucro registradas en los Estados Unidos que cumplan con los requisitos.

Precios: Revisa la página Alojamiento Shared para ver los precios actuales.

Alojamiento VPS

Un Servidor Privado Virtual (VPS) es un plan de alojamiento premium disponible como complemento de un plan de alojamiento existente o como un servicio independiente. Un VPS es ideal para sitios web que requieren más recursos de los que un servidor compartido puede proporcionar, pero que no necesitan la potencia completa de un Servidor Dedicado. Es ideal para negocios, eCommerce, diseñadores y desarrolladores.

Comparación con el alojamiento Shared: En un servidor compartido, los recursos son compartidos por otros usuarios. En un VPS, los recursos de RAM de un usuario están protegidos de otros usuarios en la misma máquina física, lo que mejora el rendimiento y ayuda a mantener la estabilidad. Además, tienes la capacidad de controlar cuánta RAM está disponible para que la use tu servidor.

¿Está disponible el acceso root/sudo?: el acceso root no está disponible en los planes VPS.

Precios: Revisa la página Alojamiento VPS para ver las opciones actuales del plan.

Agregar este: Revisa el artículo Agregar un VPS para obtener instrucciones sobre cómo agregar este plan.

Dedicados

Los Servidores Dedicados son servidores de clase empresarial totalmente gestionados, perfectos para aplicaciones que consumen muchos recursos y sitios web grandes. Están completamente dedicados exclusivamente a tus datos, ya que ningún otro cliente de DreamHost comparte recursos con tus sitios en ese servidor.

¿Está disponible el acceso root/sudo?: el acceso root está disponible mediante el comando sudo.

Bases de datos MySQL:  Todas las bases de datos se alojan en el mismo Servidor Dedicado, lo que permite un acceso más rápido. Esto es diferente a todos los demás planes, donde tus bases de datos están en un servidor de bases de datos separado.

Precios: Revisa la página Alojamiento Dedicado para ver las opciones actuales del plan.

Agregar este plan: Revisa el artículo Agregar un servidor Dedicado para obtener instrucciones sobre cómo agregar este plan.

DreamPress

DreamPress es un plan de alojamiento administrado para WordPress. Es un servicio escalable basado en la nube donde tanto los archivos del sitio web como la base de datos se alojan en la misma máquina, ofreciendo los beneficios adicionales de robustez del servidor, caché, y memoria.

Limitaciones: Estos están configurados para un solo sitio web. Si deseas múltiples dominios en DreamPress, cada uno debe tener su propio plan de alojamiento DreamPress activado por separado.

Precios Revisa el artículo registro de una cuenta DreamPress para ver las opciones actuales del plan.

Planes adicionales de alojamiento

Los siguientes son algunos planes adicionales que DreamHost ofrece.

Alojamiento de correo

Se puede agregar un plan de correo pago como un plan independiente si deseas alojar tu correo en DreamHost pero tu sitio web en otra compañía. También se puede adquirir como un plan adicional a los siguientes:

  • un plan Shared Starter
  • una cuenta solo de registro de dominio
  • un plan solo de DreamCompute

Todos los planes de Alojamiento Shared, VPS, DreamPress, y Servidores Dedicados incluyen el servicio de correo por defecto.

Alojamiento y VPS MySQL

Los planes MySQL VPS se pueden agregar a los planes Shared y VPS como una mejora a los servidores de bases de datos compartidos predeterminados. Este plan es similar a los planes VPS mencionados anteriormente, ya que los recursos de RAM están protegidos de otros usuarios en la misma máquina física, lo que mejora el rendimiento y ayuda a mantener la estabilidad.

¿Puedo elegir qué bases de datos mover?

No. Si compras este plan, todas tus bases de datos se moverán al MySQL VPS. No es posible dividir las bases de datos entre servidores Shared y VPS MySQL.

DreamCompute

DreamHost ofrece un servicio de computación en la nube llamado DreamCompute que proporciona recursos de computación escalables para desarrolladores y emprendedores. Revisa el artículo ¿Qué es DreamCompute? para más detalles.

Remixer

Remixer es un creador de sitios web 'en la nube' con clics para editar, que no requiere experiencia en codificación. Construyes todo tu sitio dentro del panel de Remixer, por lo que no hay archivos HTML/CSS/JavaScript a subir.

Apache y Nginx

Si has actualizado a un VPS o a un Servidor Dedicado, tienes la opción de ejecutar Nginx como tu servidor web. El alojamiento Shared y DreamPress deben usar Apache. Revisa la Comparación de rendimiento de servidores web para ver una comparación detallada.

Alojamiento seguro

El alojamiento seguro no es un plan de alojamiento. Es cuando agregas un certificado SSL a tu sitio web para que pueda ser accedido de manera segura. Revisa el artículo Generalidades de los certificados SSL para más detalles.
