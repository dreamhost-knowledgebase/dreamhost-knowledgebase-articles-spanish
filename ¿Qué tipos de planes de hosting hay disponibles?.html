Generalidades

Un plan de alojamiento es un producto que debes comprar para poder mostrar tu sitio web en línea. Cuando compras un plan de alojamiento, se te asigna un servidor. Entonces ahí vas a poder subir tu sitio web al servidor. Si los registros DNS de tu sitio web están apuntando a DreamHost, tu sitio podrá entonces cargar en línea desde tu nuevo servidor de DreamHost.

Planes de alojamiento de DreamHost

DreamHost ofrece varios tipos diferentes de planes de alojamiento que se ejecutan en el sistema operativo Ubuntu. La mayoría de los sitios funcionan bien en el plan de alojamiento compartido básico, sin embargo, si tu sitio está recibiendo una gran cantidad de tráfico o si necesitas hacer personalizaciones que no estén disponibles para los usuarios de alojamiento compartido, es posible que debas actualizar a un plan VPS, Dedicado o DreamPress.

Puedes encontrar más detalles en Alojamiento de DreamHost que describe las diversas opciones disponibles para ti como cliente de DreamHost.

Uso de ancho de banda

El uso de ancho de banda en Dreamhost no es medido en ningún plan de alojamiento excepto por los servicios de cloud.

¿Qué pasa si mi dominio está registrado en otra empresa?

Si compraste el registro del nombre de tu dominio a través de otro proveedor, no hay problema, puedes aún así alojar tu dominio con DreamHost. Solo necesitas apuntar tus DNS a DreamHost, o también podrías transferir el registro del nombre de dominio a tu cuenta de DreamHost si deseas. Revisa los siguientes artículos para más detalles:

Alojamiento Shared Starter

Shared Starter es el plan introductorio de alojamiento compartido de DreamHost pensado para usuarios que son nuevos en el área de alojamiento web. También es la solución perfecta para sitios personales o de negocios que muestran información simple acerca de ti o de los servicios que ofreces.

¿Qué incluye?

Como es un plan básico o inicial, tiene algunas limitaciones que no existen en un plan de alojamiento compartido regular. Por ejemplo, el plan Shared Starter ofrece lo siguiente:

  • 1 dominio que puedes usar para tu sitio (example.com).
  • 5 subdominios para tu sitio (blog.example.com).
  • Correo electrónico no está incluido — el plan de correo está disponible por un costo adicional.
  • Almacenamiento limitado a 50GB.
  • Máximo 6 usuarios de SFTP o Shell.
  • Máximo 6 bases de datos MYSQL.

Política de Reembolso

Los reembolsos en el plan Shared Starter siguen la misma política del plan de Alojamiento Shared.

Alojamiento Shared

En este plan de alojamiento compartido tu sitio está en un servidor donde el espacio se comparte entre varios clientes. Aunque ningún otro cliente tiene acceso a tus archivos, tu y otros clientes comparten los recursos del servidor. Esto significa que si otro sitio en el mismo servidor está usando gran cantidad de recursos de CPU o está recibiendo mucho tráfico, es posible que tu sitio también tenga un desempeño menor como resultado.

Aún cuando DreamHost trabaja arduamente para asegurarse de que todos los sitios en un Servidor compartido nunca afecten otros sitios, siempre existe una posibilidad de que esto pase debido a la naturaleza del alojamiento compartido.

Alojamiento VPS

El Servidor Privado Virtual de DreamHost (VPS) es un plan premium disponible como adición a un plan de DreamHost o como servicio independiente.

Este plan usa -Containers (LXC) para crear un servidor "virtual" que protege la CPU de un usuario, y los recursos de la RAM, de otros usuarios en la misma máquina física. Mediante aislamiento a nivel kernel se logran alojar en un mismo servidor físico múltiples unidades virtuales al mismo tiempo. Esas unidades están lo suficientemente aisladas para garantizar la seguridad necesaria, utilizando los recursos disponibles eficientemente, ya que se ejecutan en el mismo kernel.

Comparación entre los planes de alojamiento Shared y VPS

En un Servidor compartido, los recursos se comparten entre varios usuarios. En un VPS, estos recursos están más aislados, lo que mejora el desempeño y ayuda a mantener una estabilidad.

También tendrás la posibilidad de controlar qué tanto RAM usa tu servidor. En el plan Shared Hosting, esto se limita a una cantidad en específico; pero en un VPS, puedes decidir la cantidad de RAM que hay disponible para usar.

¿Qué puedo alojar en el plan de VPS?

Un VPS es ideal para sitios web que requieren más recursos de los que un plan Shared Hosting puede ofrecer, sin necesidad del poder total de un Servidor Dedicado. Es ideal para sitios empresariales, negocios en línea, diseñadores, y desarrolladores.

Dedicados

Un Servidor Dedicado es un servidor que está sola y completamente dedicado a tus datos. Esto significa que ningún otro cliente de DreamHost tiene acceso ni comparte ningún recurso con tus sitios en ese servidor.

Tienes acceso completo a la raíz para personalizar el servidor como desees, pero por favor ten en cuenta que el área de soporte de DreamHost no podrá ayudarte si surgen errores como resultado de la personalización del servidor.

Otra característica es que el servidor que aloja tus bases de datos MySQL, está en el mismo Servidor Dedicado permitiendo un acceso más rápido, a diferencia de los otros planes de alojamiento donde tus bases de datos MySQL están en un servidor separado ya sea compartido o “privado”.

DreamPress

DreamPress es el plan de alojamiento de DreamHost especializado en WordPress. Este es un servicio escalable almacenado en la nube donde los archivos y la base de datos se alojan en la misma máquina, ofreciendo precios accesibles para los beneficios de robustez del servidor, almacenamiento en caché, y memoria.

Cada plan de DreamPress se configura individualmente para cada sitio web. Si quieres tener varios dominios en DreamPress, cada dominio debe tener su propio plan DreamPress por separado.

Alojamiento de MySQL

En cualquier plan de Alojamiento Compartido o VPS, también tienes acceso gratuito a un Servidor compartido de bases de datos MySQL. Este almacena las bases de datos de todos tus sitios web alojados en los planes mencionados. Debido a que es un Servidor compartido, los recursos se comparten con otros clientes de la misma manera que en un Servidor compartido como en los planes de Shared Unlimited, así como se explico en dicha sección de esta guía.

Si necesitas más recursos para tus bases de datos, puedes comprar un MYSQL VPS:

  • Generalidades del servidor privado de MySQL

Si decides comprar este plan, todas tus bases de datos se moverán al  MYSQL VPS. No es posible dividir las bases de datos entre el Servidor compartido y MYSQL VPS.
