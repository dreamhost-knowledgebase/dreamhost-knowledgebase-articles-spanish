Generalidades

WordPress es un sistema de gestión de contenidos (CMS) y una herramienta de blogs. Se puede utilizar para crear sitios web y blogs personales.

Para obtener más información sobre qué es WordPress y qué puedes hacer con él, visita el sitio web de WordPress en:

  • http://wordpress.org/

Versión de WordPress en DreamHost

DreamHost trabaja en estrecha colaboración con WordPress para ofrecerle las funciones más actualizadas. Como tal, la versión más nueva de WordPress siempre está disponible en su panel DreamHost.

What you need before installing

Si encuentras Remixer o Wordpress regular demasiado limitado para tus necesidades, o deseas utilizar una alternativa a la función de exportación de WordPress desde Remixer, el soporte de DreamHost puede proporcionarte acceso gratuito al nuevo Creador de sitios web de WordPress (BoldGrid).

Para obtener más información sobre cómo instalar BoldGrid en tu sitio actual de Remixer o WordPress, comunícate con el equipo de soporte de DreamHost a través de Live Chat o enviando un correo electrónico a support@dreamhost.com. Un miembro del equipo de soporte de DreamHost puede ayudarte aún más con la configuración del nuevo Creador de sitios web de WP para tu sitio.

Consulta los siguientes artículos para obtener más información sobre BoldGrid:

  • Generalidades sobre BoldGrid
  • Centro de Soporte de BoldGrid

Antes de instalar WordPress, primero debes tener un plan de alojamiento activo como un plan de alojamiento compartido, VPS o dedicado.

A continuación, deberás haber comprado un registro de dominio. Si ya has comprado un registro de dominio en otra compañía, deberás apuntar tus nameservers o registros A a DreamHost. Si compraste el registro en DreamHost, no necesitas ajustar los DNS. Instalar DreamPress

DreamHost ofrece alojamiento de WordPress administrado personalizado llamado DreamPress. Es perfecto para usuarios intermedios que necesitan más rendimiento que el alojamiento compartido, pero que no están listos para actualizar a un VPS para administrar personalmente su servidor y sitio web.
Puedes leer más sobre este servicio actualizado aquí:

  • DreamPress

Usando las actualizaciones automáticas de WordPress

A partir de la versión 3.7, WordPress ahora presenta actualizaciones automáticas. Estas son solo para actualizaciones menores de revisión (por ejemplo, 4.1 a 4.1.1).

DreamHost recomienda encarecidamente que permitas que estas actualizaciones se realicen automáticamente, ya que generalmente incluyen actualizaciones de seguridad importantes o correcciones de errores.

Puedes leer más sobre esta función y cómo deshabilitarla a través del siguiente enlace:

  • http://codex.wordpress.org/Configuring_Automatic_Background_Updates

Importar

Después de instalar y configurar WordPress, puedes importar contenido externo. Puedes hacerlo iniciando sesión en tu sitio de WordPress y yendo a 'Herramientas' > 'Importar'.

Importar contenido puede requerir complementos adicionales dependiendo de la plataforma que estés importando.
