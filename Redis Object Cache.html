Generalidades

REmote DIctionary Server (Redis) es un servidor de estructura de datos de código abierto que se usa como base de datos, caché, motor de transmisión y agente de mensajes. Permite que los datos se almacenen en la memoria en lugar de en el disco, brindando un alto rendimiento y escalabilidad, y a menudo se usa para almacenar en caché las páginas web.

Redis en DreamHost

Redis Object Cache está disponible en todos los planes de DreamPress Pro, complementando el almacenamiento en caché del servidor incorporado de DreamPress. Este mejora el rendimiento de WordPress almacenando objetos de WordPress en la memoria para reducir el número de requerimientos de la base de datos.

La integración de Redis de DreamPress Pro activa el plugin Redis Object Cache de WordPress para habilitar el caché para cada objeto posible.

Make sure to deactivate any other caching plugins as they will conflict with the Redis Object Cache plugin's performance.

Usar el plugin Redis Object Cache

Para verificar el estado del caché de tu sitio WordPress, inicia sesión en tu sitio DreamPress y luego selecciona Object Cache > Settings en la barra superior.

La pestaña Overview

La pestaña Overview se muestra si:

  • el plugin está conectado
  • el sistema se puede escribir, y
  • Redis es accesible

La pestaña Metrics

La pestaña Metrics presenta un gráfico que muestra las métricas registradas a lo largo del tiempo con información sobre el consumo de Bytes, el porcentaje de la proporción de Aciertos frente a Fallas y la cantidad de Llamadas para comparar visualmente qué tan rápido funciona el sitio como resultado del almacenamiento en caché de objetos.

Purgar manualmente el objeto del caché a través del plugin

Para purgar manualmente el caché del objeto, inicia sesión en tu sitio DreamPress y luego selecciona Object Cache > Flush Cache en la barra superior.

Deshabilitar temporalmente Redis

Las siguientes instrucciones explican cómo deshabilitar temporalmente Redis. Si necesitas deshabilitarlo manualmente, por favor contacta a soporte.

  1. Abre la página WP-Admin de tu sitio DreamPress (ejemplo, example.com/wp-admin).
  2. Da clic en el botón Object Cache ubicado en la barra superior de WP-Admin.
  3. Selecciona la opción Settings del submenú.
  4. Da clic en el botón Disable Object Cache en la parte inferior y confirma los cambios de estado de Connected a Not Enabled en la pestaña Overview.
