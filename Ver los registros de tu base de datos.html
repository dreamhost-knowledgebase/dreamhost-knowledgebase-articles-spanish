Generalidades

Los usuarios de DreamHost no tienen acceso a los registros del servidor MySQL. Sin embargo, dependiendo del tipo de servidor MySQL en el que se ejecuten tus bases de datos, puede ser posible que el equipo de soporte técnico las recupere por ti.

Servidores MySQL compartidos

Visita la página Bases de Datos MySQL. Verás una lista de todos tus nombres de host y bases de datos. En la parte superior de la sección, aparece el nombre de tu base de datos. Si es un nombre específico, esto significa que estás en un servidor MySQL compartido. Por ejemplo:

Desafortunadamente, los registros no están disponibles en servidores MySQL compartidos. Esto significa que incluso el equipo de soporte técnico no podría recuperarlos por ti.

Servidores privados de MySQL

Visita la página Bases de Datos MySQL. Verás una lista de todos tus nombres de host y bases de datos. En la parte superior de la sección, aparece el nombre de tu base de datos. Si el nombre comienza con psmysql y va seguido de un número, esto significa que estás utilizando un servidor MySQL privado. Por ejemplo:

Los registros en un servidor MySQL privado están disponibles solo si contactas al área de soporte técnico. Asegúrate de especificar el nombre de la base de datos y el marco de tiempo del que deseas recuperar los registros.

Servidores Dedicados y DreamCompute

En un servidor dedicado, las bases de datos MySQL están en la misma máquina. Esto significa que puedes recuperarlos iniciando sesión en el servidor a través de SSH.

DreamCompute está completamente configurado manualmente, por lo que cualquier configuración depende completamente de ti.
