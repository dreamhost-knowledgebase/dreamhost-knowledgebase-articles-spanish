Generalidades

Las direcciones de conveniencia son direcciones de correo que se pueden crear para que los usuarios de su lista puedan usar listname@example.com para la lista, en lugar de listname@lists.example.com. Por lo tanto, el subdominio lists se elimina en estas direcciones de conveniencia.

Cada listname-action@lists.example.com tiene dos direcciones de conveniencia de solo reenvío:

  • listname-action@example.com
  • listname-example.com-action@lists.example.com

Las direcciones de conveniencia se enumeran como direcciones de solo reenvío en la página del panel Administrar Correo. Sin embargo, solo debes crearlos o eliminarlos mediante la casilla de verificación Agregar/Eliminar direcciones de correo de conveniencia configuradas para esta lista en la página del panel Listas de Discusión. Editar las direcciones de conveniencia mediante la página Administrar Correo puede causar problemas que solo el soporte de DreamHost puede corregir.
