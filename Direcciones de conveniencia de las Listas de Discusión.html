Generalidades

Las Direcciones de Conveniencia son direcciones de correo que se pueden crear para que los usuarios de tu lista puedan usar nombredelista@example.com para la lista, en vez de nombredelista@lists.example.com. De esta manera, la porción listas se elimina en las direcciones de conveniencia.

Ejemplos de las direcciones de conveniencia

Las siguientes direcciones de conveniencia se crean cuando cuando se agrega una Lista de Discusión llamada mylist al dominio example.com.

Todas las direcciones en la primera columna reenvían al correo correspondiente de la Lista de Discusión:

Dirección de Conveniencia Dirección de Lista de Discusión
mylist-admin@example.com mylist-admin@lists.example.com
mylist-bounces mylist-bounces@lists.example.com
mylist-confirm@example.com mylist-confirm@lists.example.com
