Generalidades

A continuación se describe cómo administrar usuarios de MySQL, hosts permitidos, y privilegios en el panel DreamHost.

Administrar las configuraciones del usuario

Navega a la página Bases de Datos MySQL y da clic en el nombre de usuario a la derecha de la base de datos para ver su configuración.

Se abre la página de configuración de usuario donde puedes administrar lo siguiente:

Eliminar un usuario

Da clic en el botón Eliminar en la parte superior para quitarle a este usuario MySQL el acceso a la base de datos. Cuando se elimina un usuario MySQL de todas las bases de datos, el usuario deja de existir y se considera eliminado por completo.

¿Qué puede hacer <USUARIO> con las tablas de estas bases de datos?

Esta sección te permite agregar o quitar privilegios de usuario específicos, lo que afecta su acceso para leer, escribirn o cambiar datos, así como para administrar el contenido de la base de datos. Marca cada casilla para otorgar la capacidad para lo siguiente:

  • Seleccionar — lee filas de tablas.
  • Insertar — agrega nuevas filas a las tablas.
  • Actualizar — modifica las filas existentes en las tablas.
  • Eliminar — eliminar filas en las tablas.
  • Crear — crea nuevas tablas. Esto not permite la creación de nuevas bases de datos en la línea de comando.
  • Soltar — elimina tablas y vistas.
  • Índice — agrega o elimina índices de las tablas existentes.
  • Alterar — cambia la estructura de las tablas.

Editar el campo de Hosts Permitidos

El campo Hosts permitidos muestra una lista de hosts a los que se les permite a este usuario MySQL iniciar sesión.

  • Por razones de seguridad, el valor predeterminado es %.dreamhost.com, que restringe las conexiones fuera de la red DreamHost, incluso si se proporciona un usuario y una contraseña correctos.
  • Se pueden agregar direcciones IP adicionales o wildcards en nuevas líneas para permitir el acceso a una red externa.
  • Modificar o eliminar %.dreamhost.com puede hacer que MySQL sea inaccesible para los sitios web alojados en DreamHost.

Ver y cambiar la contraseña de un usuario

Bajo la sección ¿Necesita saber la contraseña de <USUARIO>?, da clic en el botón Mostrar para mostrar la contraseña de este usuario de MySQL.

Para cambiarla, ingresa una nueva contraseña y haz clic en el botón Modificar exampleuser ahora para guardarla. La actualización puede demorar hasta 5 minutos.

Por favot ten en cuenta que todos los sitios que utilicen este usuario MySQL dejarán de funcionar inmediatamente hasta que todos los archivos de configuración se actualicen con la nueva contraseña.

Revisa la documentación de ayuda de tu software para obtener información sobre este archivo.
