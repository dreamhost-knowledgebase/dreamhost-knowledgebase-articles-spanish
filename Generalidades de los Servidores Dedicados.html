Generalidades

A continuación se describe cómo comenzar con un Servidor Dedicado en DreamHost.

Preguntas Frecuentes

¿Qué es un Servidor Dedicado?

Un Servidor Dedicado es un servidor que está completamente dedicado exclusivamente a tus datos. Esto significa que ningún otro cliente de DreamHost tiene acceso ni comparte recursos con tus sitios o bases de datos en ese servidor.

¿Para qué se utiliza un Servidor Dedicado?

Los Servidores Dedicados son máquinas premium ideales para sitios web de alto rendimiento que requieren recursos dedicados para funcionar de manera eficiente.

¿Cuánto cuesta un Servidor Dedicado?

Para obtener más información sobre planes y precios, por favor visita la página de Alojamiento Dedicado en el sitio web principal.

¿Está disponible el acceso root?

Los Servidores Dedicados ofrecen la posibilidad de añadir un usuario administrador, que son usuarios especiales que tienen acceso completo al servidor. Este usuario administrador tiene acceso root completo al Servidor Dedicado a través del comando sudo.

Ten cuidado cuando uses un usuario administrador ya que tendrás la capacidad de alterar la configuración del sistema, lo cual puede desactivar tus sitios o servidor.

Re-imagen de tu servidor

Hay una tarifa de $100 para re-imagen de tu Servidor Dedicado. Si deseas proceder con esto, contacta al soporte técnico por correo.
