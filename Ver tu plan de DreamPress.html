Generalidades

Este artículo explica cómo ver el plan actual de DreamPress que estás usando.

Ver tu plan

Una vez que tengas al menos un dominio en DreamPress, ve a la página de WordPress Administrado.

A la derecha de tu dominio, da clic en el botón Administrar. Aparece un panel que te muestra una descripción general de tu servicio DreamPress. A continuación se muestra una descripción de cada campo:

Pestaña Detalles - Sección superior

  • Administrar WordPress — Haz clic aquí para iniciar sesión directamente en tu sitio DreamPress.
  • Jetpack Professional — Muestra si has habilitado Jetpack Professional de tu sitio DreamPress.

Sección central

  • Plan — Se muestra si tienes un plan anual o mensual y su fecha de renovación.
  • Almacenamiento Usado — Muestra la cantidad de almacenamiento de tu plan actual que estás utilizando. También puedes hacer clic en 'Redimensionar Almacenamiento' para cambiar la cantidad de almacenamiento que usa tu plan.

Sección inferior

  • Subir Archivos — haz clic en el botón 'Show Credentials' para ver las credenciales de inicio de sesión de FTP/SSH de tu usuario. Luego puedes usar un cliente FTP o SSH para iniciar sesión en stu servidor.
  • Base De Datos (MySQL) — haz clic en el botón 'Show Credentials' para ver las credenciales de inicio de sesión de la base de datos de tu sitio web. Luego puedes usar phpMyAdmin para iniciar sesión en tu base de datos.

Pestaña Dominio - Sección inferior

Cuando haces clic en la pestaña Domain, tienes la opción de cambiar tu nombre de dominio de DreamPress. A la derecha también está la opción de importar un sitio de WordPress existente.
