Generalidades

Cada usuario tiene al menos un conjunto de Llaves de Emparejamiento que consta de lo siguiente:

  • Una llave de acceso de 20 caracteres (similar a un nombre de usuario para un bucket) y
  • Una llave secreta de 40 caracteres (similar a una contraseña para un bucket y generada automáticamente).

El panel utiliza estas credenciales para acceder a tus buckets y objetos.

Ver tus llaves

Para ver tus llaves, haz clic en tu nombre de usuario:

La llave secreta no se muestra por defecto.

Haz clic en el enlace Show Secret Key para ver la llave secreta.

  • La mayoría de los clientes de terceros como Cyberduck, así como las API y herramientas que se escriben usando la API, también usan estas credenciales para obtener acceso al clúster programático DreamObjects.
  • Un usuario puede tener varias claves para proporcionar acceso de grano fino.
  • Eliminar una Llave de Emparejamiento elimina el acceso para cualquier aplicación que las use.

Copiar tu Llave de Acceso

Para tu comodidad, cuando haces clic en una llave de acceso o clave secreta, el texto se selecciona (aunque no se copia en el portapapeles). Una vez que la llave está resaltada, puedes copiarla.

Windows

Ctrl + C para copiar.

Ctrl + V  para pegar.

Mac

(Command key) + C para copiar.

(Command key) + V para pegar.

Agregar una Llave de Emparejamiento

  1. Contrae el Usuario Object o desplázate hasta la parte inferior del Usuario object expandido para buscar los Controles del usuario.

  2. Haz clic en el botón Add Key.

    Una llave S3 es añadida inmediatamente a tu usuario.

Eliminar una Llave de Emparejamiento

  1. Navega hasta la Llave de Emparejamiento que deseas eliminar y ubica el botón Remove Key en el lado derecho de esa fila.

  2. Haz clic en el botón Remove Key.

    Una caja de diálogo de confirmación aparece:

Eliminar la Llave de Emparejamiento dará como resultado una pérdida de acceso para todos los clientes y scripts que hayan estado usando esta Llave de Emparejamiento.

  1. Haz clic en el botón Remove.

    La Llave de Emparejamientoes eliminada.
