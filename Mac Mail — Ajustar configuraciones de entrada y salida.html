Este artículo explica cómo ajustar la configuración de correo para una dirección de correo que ya has configurado. Estas instrucciones explican cómo ajustar lo siguiente:

  • Nombre de Host
  • Número de puerto
  • TLS/SSL (encendico/apagado)
  • Método de autenticación

Este artículo usa ejemplos con IMAP. Revisael siguiente artículo para ver las alternativas de POP:

  • Números de puerto y protocolos de los Clientes de Correo

Sonoma

  1. Navega a Correo > Configuraciones.
  2. Haz clic en la opción @ Cuentas en la barra superior.
  3. Haz clic en la pestaña Configuración del servidor.
  4. Edita los campos de Nombre de host para que coincidan con el nombre de tu servidor de correo DreamHost.
  5. Desmarca Administrar automáticamente la configuración de conexión. El campo Puerto y la casilla Usar TLS/SSL aparecen luego.
  6. Edita los puertos de Entrada/Salida como necesites. Se recomienda usar los valores de entrada como el puerto 993 para IMAP y 995 para POP.
  7. Haz clic en Guardar cuando termines.
