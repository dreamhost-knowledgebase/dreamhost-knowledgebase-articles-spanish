Generalidades

Este artículo te da unas cuantas sugerencias sobre cómo identificar inconvenientes de memoria y optimizar tu VPS de DreamHost. Esto ayudará a usar menos memoria lo que te ahorra dinero mientras aceleras tus sitios web.

Bloquear bots abusivos

Bloqueo de acceso abusivo

Un problema común para los sitios web es causado por los bots de indexación de los motores de búsqueda que repetidamente acceden a tu sitio web.

Crear un archivo robots.txt

Para minimizar los inconvenientes de bot abusivos, debes configurar un archivo robots.txt. Este archivo ayuda a controlar que bots pueden buscar y cada cuanto.

  • Controlar los bots, spiders, y rastreadores

Ver el archivo access.log

Encontrar los bots abusivos puede ser más un arte que una ciencia, pero puedes encontrar información sobre esto analizando el archivo access.log de tus sitios web. Revisa el siguiente artículo para ver información detallada sobre cómo revisar el archivo access.log y ver de dónde está viniendo más tráfico:

  • Ver y examinar tu access.log a través de SSH

Prevención de enlaces directos de imágenes o hotlinking

Hotlinking es cuando otro sitio web usa una imagen de tu sitio. Para ello, usan la URL de la imagen completa del tuyo en su propio sitio web. Esto puede causar problemas masivos en tu sitio, ya que incluso uno con un tráfico relativamente bajo puede causar un rendimiento deficiente del servidor.

Revisa el artículo Prevenir hotlinking de imágenes para más información sobre cómo prevenir que esto suceda.

Almacenamiento en caché en tu sitio

El almacenamiento en caché ayuda a almacenar archivos de sitios web, lo que hace que tus visitantes accedan a ellos más rápidamente. Asegúrate de que el software de tu sitio web use alguna funcionalidad de almacenamiento en caché para evitar problemas de memoria y mejorar el rendimiento general.

Procesos de PHP

Los procesos de PHP pueden ocupar una cantidad significativa de memoria. Si tu sitio recibe mucho tráfico, habilitar el almacenamiento en caché ayudará a reducir la cantidad de procesos PHP que se generan.

El almacenamiento en caché representa un script de PHP y luego "almacena en caché" esa salida HTML como un archivo. La próxima vez que se acceda a esa página, el servidor generará la página HTML estática en lugar de volver a procesar el script PHP.

Cloudflare

También puedes habilitar un servicio de terceros como Cloudflare, el cual te da opciones adicionales de almacenamiento en caché para tus sitios web si esperas un gran volumen de visitantes.

Usar OPcache con sitios web de PHP

OPcache es un motor de almacenamiento en caché construido en PHP que, cuando se habilita, incrementa dramáticamente el funcionamiento de los sitios web basados en PHP. Revisa el siguiente artículo para obtener detalles sobre esta función:

  • Generalidades de OPcache

Optimizaciones específicas de WordPress

Si estás ejecutando varias instancias del software de WordPress en VPS, revisa la categoría Optimización de WordPress para obtener artículos detallados.

Asegúrate de hacer lo siguiente con cada instalación:

  • Ejecuta siempre la versión más reciente de WordPress.
  • Instala sólo plugins esenciales.
  • Mantén actualizados tus plugins.
  • Elimina plugins que no utilices en lugar de simplemente desactivarlos.
  • Confirma que el tema que está activo no esté generando ningún error en tu archivo error.log (ubicado en el mismo directorio que access.log).
  • Revisa la página Optimizing Page Load Time (optimización del tiempo de carga de la página) y asegúrate de tener un plugin de almacenamiento en caché de WordPress instalado y configurado. Por ejemplo, W3TC o WP Super Cache.
  • Finalmente, comprimir tus archivos de imagen mejora el tiempo de carga y reduce el uso general de memoria.

Configurar las Tablas de IPs

No es posible personalizas las tablas de IPs en un VPS de DreamHost. Esto es por que los VPS se ejecutan en una sola máquina donde las tablas de IPs están configuradas, pero no hay acceso a la misma.

Si necesitas ejecutar tus propias tablas de IPs, la única opción sería actualizar a un Servidor Dedicado o DreamCompute e instalar este software manualmente.
