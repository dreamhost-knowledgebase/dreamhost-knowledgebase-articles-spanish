Generalidades

El panel DreamHost te permite realizar pagos anticipados en tu cuenta, sin embargo, el pago anticipado no se aplica a los cargos o servicios hasta que esté listo para renovar. Esto significa que el prepago que envíes permanecerá en tu cuenta como un saldo de crédito positivo hasta que se venza un cargo/servicio.

Tampoco puedes definir a qué cargo o servicio se aplicará el prepago; el próximo cargo en tu cuenta se retira automáticamente del saldo de crédito existente.

Uso del panel DreamHost para realizar un prepago

Los pagos deben realizarse a través de tu panel DreamHost, que ofrece una forma segura de transmitir datos de pago

Para realizar un prepago, navegue a la página Manage Payments:

¿Qué opciones de pago están disponibles?

Están disponibles las siguientes opciones de pago, que se describen a continuación:

  • Tarjeta crédito/debito (Visa, MasterCard, American Express, Discover, y Diners Club)
  • Cheque Electrónico/ACH
  • PayPal

Usando PayPal o una tarjeta de crédito

Consulta las instrucciones en el siguiente artículo sobre cómo agregar una tarjeta de crédito para pagar por adelantado los servicios de DreamHost:

  • Configurar Autopay
  • Realizar un solo pago con tu tarjeta de crédito o débito

Usando otro método de pago

Consulta las instrucciones en el siguiente artículo sobre cómo usar otra forma de pago para pagar por adelantado los servicios de DreamHost:

  • Usando otro método de pago

¿Qué sucede después de hacer un prepago?

Una vez que realices un prepago utilizando cualquiera de las opciones de pago anteriores, tu cuenta muestra un saldo positivo en la página Manage Payments con el monto que pagaste.

El cargo de DreamHost aparece como "DreamHost dh-fee.com" en tu extracto bancario.
