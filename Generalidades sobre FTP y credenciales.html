Generalidades

File Transfer Protocol (FTP) es un protocolo de red que se usa para cargar datos de sitios web desde tu computador local a tu servidor web para que aparezcan en línea.

Este artículo explica cómo ubicar tus credenciales e iniciar sesión en tu servidor usando un cliente SFTP.

Sitios DreamPress

En su lugar, mira el siguiente artículo si necesitas iniciar sesión en tu sitio DreamPress.

  • Iniciar sesión en tu sitio DreamPress vía FTP

Diferencias entre FTP y SFTP

FTP es un protocolo antiguo que NO ES SEGURO. Usa el puerto #21.

SFTP es un protocolo nuevo que SEGURO. Usa el puerto #22.

El término FTP

FTP es un término antiguo que todavía se usa comúnmente para hacer referencia a la transferencia de datos a tu servidor. Sin embargo, dado que no es seguro, nunca debe usarse. Solo deben usarse conexiones SFTP cuando te conectes a tu servidor.

Los nuevos usuarios creados en DreamHost se crean automáticamente como usuarios de SFTP, por lo que no es necesario ajustar nada. Deberías entender que cuando ves el término FTP, generalmente hace referencia a SFTP (que es la opción segura).

Usa solo el puerto 22 cuando te conectes. Esto asegura que estás utilizando una conexión SFTP segura.

Prerequisitos

Agregar alojamiento

Asegúrate de tener agregado un alojamiento a tu dominio en tu panel DreamHost. Esto te asigna las credenciales que necesitas para iniciar sesión en tu servidor.

Escoger un cliente FTP

Este artículo usa el cliente de FTP Filezilla para ilustrar los ejemplos. Tu puedes usar cualquier cliente que desees para iniciar sesión en tu servidor ya que la configuración es la misma.

Ubicar tus credenciales de inicio de sesión de FTP

Necesitarás la siguiente información para iniciar sesión en tu servidor web:

  • Servidor/Hostname
  • Usuario
  • Contraseña
  • Puerto

Las siguientes secciones explican cómo obtener cada uno.

Encontrar el hostname

Hay dos nombres de host que puede usar para iniciar sesión en su servidor DreamHost.

  • El nombre del sitio — Puedes usar el nombre de tu sitio para iniciar sesión en tu servidor, pero solo si el DNS ya está apuntando a DreamHost
  • El nombre del servidor — Siempre puedes usar el nombre del servidor para iniciar sesión en tu servidor, incluso si el DNS aún no apunta a DreamHost.

Opción 1 — Usa el nombre de tu sitio web

Si has agregado un alojamiento a tu dominio y el DNS de tu dominio está configurado para apuntar a tu hosting en DreamHost, puedes usar el nombre de tu dominio como el hostname SFTP. Por ejemplo, si tu sitio web se llama https://www.example.com solo usa example.com.

No agregues https://www en frente del dominio pues puede causar que no conecte.

Opción 2 — Usar el nombre del servidor

Si tienes agregado alojamiento a tu sitio, pero aún no has apuntado tu DNS a DreamHost, aún puedes conectarte al servidor usando el hostname del servidor.

Para saber en cual servidor estás:

  1. Inicia sesión en tu panel de DreamHost, y navega a Administrar Cuenta.
  2. Baja hasya la sección llamada Dominios Alojados. Ahí verás tu nombre de dominio.
  3. A la derecha del dominio, verás tu usuario y el servidor en el que está configurado:
  4. En la captura de arriba, el servidor se llama gervais.
  5. Necesitarás agregar .dreamhost.com después del nombre del servidor para completar el hostname.

En ete ejemplo, el hostname completo del servidor sería gervais.dreamhost.com. Así es como aparece en Filezilla.

Hostnames de VPS, Dedicados, y DreamPress hostnames

Si estás alojando tu sitio en un servidor VPS, Dedicados, o DreamPress, el hosrname aparece diferente en la columna Tipo de Alojamiento. Esto cambiaría el texto que agregas después del nombre del servidor. Por ejemplo:

Tipo de servidor Como aparece Hostname a usar
VPS ps123456 ps123456.dreamhostps.com
Dedicated ds123456 ds123456.dreamservers.com
DreamPress dp-1234567890 dp-1234567890.dreamhostps.com

Ubicar tu usuario

  1. Navega a la página Administrar Sitios Web.
  2. Da clic en el botón Administrar de tu dominio.
    • Si estás en la vista de cuadrícula, pasa sobre tu dominio y da clic en el botón Administrar.
    • Si estás en la vista de Lista, da clic al botón Administrar a la derecha de tu dominio.
  3. Da clic en Show Login Info debajo de la sección Administrar Archivos para ver el campo de Username.

Ubicar la contraseña del usuario

La contraseña se creó originalmente cuando se creó el nombre de usuario. Si no recuerdas esta contraseña, puedes restablecerla.

Para cambiar la contraseña del usuario en el panel:

  1. Navega a la página Administrar Sitios Web.
  2. Da clic en el botón Administrar de tu dominio.
    • Si estás en la vista de cuadrícula, pasa sobre tu dominio y da clic en el botón Administrar.
    • Si estás en la vista de Lista, da clic al botón Administrar a la derecha de tu dominio.
  3. Da clic en Show Login Info debajo de la sección Administrar Archivos para ver el campo de las configuraciones de usuario.
  4. Da clic en el enlace Reset Password.

    La siguiente barra de progreso aparece:
    Cuando la contraseña se actualiza correctamente, aparece el siguiente mensaje:
  7. Ingresa la contraseña en tu cliente FTP.

Choosing the port #

Como se explica arriba, el puerto #22 es el único número de puerto que debes usar para conectarte puesusa SFTP.

Si tieneS un usuario antiguo de FTP en tu panel, asegúrate de actualizarlo inmediatamente a un usuario de SFTP para que pueda usar el puerto # 22 para conectarte de forma segura.

Ingresa el puerto 22 en tu cliente de FTP.

Finalmente, da clic en Quickconnect para iniciar sesión en tu servidor:

Cuando inicies sesión originalmente en un servidor DreamHost, es posible que veas la siguiente advertencia:

The server's host key is unknown. You have no guarantee that the server is the computer you think it is.

Bloquos temporales

Múltiples intentos fallidos de conexión FTP o SSH pueden resultar en que el servidor bloquee temporalmente tu IP. Este bloqueo eventualmente se levantará automáticamente, pero si lo necesitas antes, por favor contacta a soporte.
