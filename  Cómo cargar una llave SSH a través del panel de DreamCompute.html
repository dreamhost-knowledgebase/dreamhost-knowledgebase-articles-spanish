Generalidades

Este artículo explica cómo cargar una llave SSH en tu panel de DreamCompute. Luego puedes usarlo al crear una nueva instancia.

Cargar una nueva llave

  1. Busca el archivo de llave pública y ábrelo para copiar tu contenido.
  2. Navega a la página de DreamCompute.
  3. Haz clic en el botón View Dashboard.
  4. A la izquierda, haz clic en Compute.
  5. Haz clic en Key Pairs.
  6. A la derecha, haz clic en el botón Import Key Pair.
  7. Dale un nombre y pega el contenido de tu llave pública.

    Si solo tienes una llave privada (sin extensión de archivo, .pem o .ppk son algunos ejemplos de llaves privadas) o la llave pública no está en un formato que DreamCompute pueda aceptar, deberás cargar la clave privada en una herramienta como PuTTY Key Generator o la utilidad de línea de comandos ssh-keygen con el parámetro -y para obtener una clave pública formateada correctamente.

  8. Haz clic en el botón Import Key Pair para guardarlo.
  9. Ahora puedes verlo en el panel.
