Paso 1 — Usar el panel de DreamPress

Si eres nuevo en WordPress o DreamPress, DreamHost quiere asegurarse de que todo sea fácil de encontrar. Hay dos ubicaciones separadas para administrar tu sitio web y la experiencia de alojamiento de DreamPress:

  • El panel de administración de WordPress.
  • El panel de DreamPress.

El panel de administración de WordPress

El Panel de administración de WordPress (wp-admin) es donde puedes agregar y editar contenido y funciones en tu sitio web de WordPress.

  • ¿Cómo iniciar sesión en tu panel de Administración de WordPress?

El panel de DreamPress

El panel de control de DreamPress es donde puedes administrar los detalles de alojamiento de tu sitio web, incluidas las nuevas funciones como copias de seguridad a pedido y sitio provisional. También puedes acceder a tu cuenta DreamHost para cosas como registros de dominio (compra de nuevos nombres de dominio), facturación y servicios adicionales si deseas más espacio o sitios web.

Puedes iniciar sesión en tu panel de DreamPress directamente haciendo clic en este botón de WordPress Administrado. Para obtener más información sobre el panel de DreamPress, consulta el siguiente artículo. 

  • Introducción al panel de DreamPress

Paso 2 — Crear un nuevo sitio en WordPress

¿Estás construyendo un nuevo sitio desde cero en DreamPress? ¿Eres nuevo en WordPress? Aquí hay algunos recursos para ayudarte a comenzar:

Creador de Sitios Web de WordPress (BoldGrid)

Sólo por ser un cliente de DreamPress, puedes construir tu sitio rápido y fácil usando el Creador de Sitios Web de WordPress('WP Website Builder'), patrocinado por una licencia GRATUITA Premium de BoldGrid.

Si seleccionaste la opción WP Website Builder cuando creaste inicialmente tu cuenta de DreamHost, WordPress se instalará automáticamnete con BoldGrid. Cuando accedas a tu panel de WordPress, verás la página BoldGrid Inspirations. Si necesitas ayuda, puedes seguir la guía paso a paso en el siguiente artículo Cómo crear tu sitio con BoldGrid.

Es altamente recomendable que empieces a usar BoldGrid en una instalación completamente nueva de WordPress. Si ya tienes un sitio existente diseñado, puedes solicitar un reinicio de tu sitio DreamPress a una instalación en blanco usando BoldGrid.

Contacta al área de Soporte para solicitar una instalación gratuita y licencia premium.

Paso 4 — Configurar tu dominio

Si adquiriste un nuevo dominio cuando te registraste, tu nombre de dominio ya está configurado y funcionando.

Si ya posees tu nombre de dominio, estas son las formas de configurarlo en DreamPress:

  • Transfiere el registro de tu dominio a DreamHost
  • Apunta tu dominio a DreamHost sin transferir el registro
  • Cambia un dominio temporal en DreamPress
  • ¿El dominio todavía no se ve? DNS: propagación explicada!

¿Todavía necesitas un nombre de dominio para tu sitio? Prueba la función de Búsqueda de Nombres de Dominio de DreamHost en tu panel.
