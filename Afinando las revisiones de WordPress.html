Generalidades

Post Revisions es una característica de WordPress que ha demostrado ser útil ya que el sistema de revisiones almacena un registro de cada borrador guardado o actualización publicada. Si bien estos son muy agradables, algunas bases de datos pueden aumentar si eres propenso a hacer muchos cambios o pasar mucho tiempo creando una publicación. Para ayudar a mantener el tamaño de tu base de datos bajo, lo que mantiene tu instalación rápida, es posible que desees ajustar la cantidad de revisiones que guarda tu sitio.

Ejecutando una consulta MySQL

Si no deseas agregar otro plugin a tu instalación de WordPress, puedes ejecutar una consulta SQL simple para eliminar las revisiones.

Para configurar una consulta SQL:

  1. Abre tu panel y navega a la página MySQL Databases.
  2. Inicia sesión en phpMyAdmin. Verás tus tablas listadas a la izquierda.
  3. Haz clic en el nombre de tu base de datos.
  4. Toma nota del prefijo de la tabla.

    El prefijo de la tabla puede aparecer como "wp_" pero es muy posible tener una cadena aleatoria de letras como "wp_qch9en_".

  5. En la parte inferior de phpMyAdmin, haz clic en el icono 'Console' para abrir la ventana de comandos SQL.
    Se abre la página de comandos SQL:
  6. Pega el siguiente comando SQL en la ventana SQL: 
    DELETE FROM wp_posts WHERE post_type = "revision";
  7. Para ejecutar el comando, haz clic en CTRL + ENTER.

Esto eliminará todas las revisiones almacenadas actualmente en la base de datos. Solo asegúrate de ingresar el prefijo de tabla correcto en la consulta. En el ejemplo anterior, el prefijo es solo wp_.

Desactivar revisiones

Para evitar que se creen y almacenen revisiones en primer lugar, agrega esta línea a tu archivo wp-config.php después de la información de la base de datos:

define('WP_POST_REVISIONS', false);

Puedes encontrar el archivo wp-config.php en tu cuenta FTP. Estará en la carpeta principal de tu dominio.

Si aún deseas guardar algunas revisiones, puedes usar el siguiente código en lugar del anterior:

define('AUTOSAVE_INTERVAL', 120 ); // Default value is 60 seconds. define('WP_POST_REVISIONS', 3); // Number of revisions to save.

Eso limita el número de revisiones a 3 y cambia el valor de guardado automático a cada 2 minutos. Puedes cambiar esos valores a algo más alto (o más bajo).
