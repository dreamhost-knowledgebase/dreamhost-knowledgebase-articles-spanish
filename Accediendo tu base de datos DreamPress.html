Generalidades

A continuación se describe cómo realizar cambios rápidamente en tu base de datos MySQL utilizando el panel de control de DreamPress.

Opción 1 — Iniciar sesión automáticamente en phpMyAdmin

  1. Navega a la página Managed WordPress.
  2. Haz clic en el botón MANAGE.
    La página 'Manage DreamPress' se abre:
  3. En la parte inferior derecha debajo 'DATABASE (MYSQL)', haz clic en el botón Manage Database.
    La siguiente página se abre.

Puedes realizar cambios en tu base de datos MySQL de DreamPress directamente en phpMyAdmin, independientemente de si el DNS del dominio apunta a DreamHost. phpMyAdmin es particularmente útil para realizar copias de seguridad, importar, exportar y restaurar una base de datos.

Para obtener más información sobre phpMyAdmin, consulta los enlaces en la siguiente sección.

Opción 2 — Ubicando tus credenciales

Tu otra opción es usar un cliente FTP para conectarte. La siguiente página enumera los clientes FTP de terceros populares que puedes usar para conectarte a tu servidor DreamPress.

  • Lista de software de clientes FTP

Una vez que estés conectado, puedes usar el siguiente artículo para crear y editar archivos dentro de tu cliente.

  • Creando y editando archivos a tráves de FTP

Ubicar las credenciales de inicio de sesión de tu base de datos

También puedes usar un programa de terceros o SSH para iniciar sesión en tu base de datos. Para hacerlo, usa las credenciales que se muestran en tu página de DreamPress.

  1. Navega a la página Managed WordPress.
  2. A la derecha de tu dominio, haz clic en el botón Manage.
  3. En la parte inferior derecha, verás una sección DATABASE (MYSQL). Alterna Show Credentials para ver las credenciales de tu base de datos.
  4. Haz clic en el icono de copia a la derecha de cada elemento y pega las credenciales en tu cliente MySQL elegido.
