Generalidades

Puedes notar el siguiente error cuando visites tu sitio o iniciando sesión en él:

Warning: Cannot modify header information - headers already sent by (output started at...'''

Esto esencialmente es un inconveniente con como tu sitio está codificado. Muy a menudo esto simplemente significa que que fué enviado al buscador antes de la función PHP header() fuera requerida. Técnicamente hablando, una vez que el bloque del encabezado haya sido enviado, no puedes agregar más líneas del mismo.

Corregir el error dentro del código de PHP (avanzado)

Esto es normalmente causado por tu código enviando información al buscador antes de la función del header(). Asegúrate que la función del header() esté en tu código antes de cualquier otro código. Esto podría requerir a un desarrollador para editar el código existente en tu sitio.

Corregir el error dentro del archivo phprc (fácil)

Si no quieres hacer ningún ajuste a tu código, puedes crear un archivo phprc y agregar unas líneas a éste para resolver este inconveniente.

Mira los siguientes artículos para ver instrucciones de como crear un archivo phprc.

  • ¿Como crear un archivo phprc vía FTP?
  • ¿Cómo crear un archivo phprc vía SSH?

Si el archivo ya existe, vé los siguientes artículos para ver instrucciones de cómo actualizar (dependiendo si usas FTP o SSH):

  • Crear y editar un archivo vía FTP
  • Crear y editar un archivo vía SSH

Una vez creado, agrega la siguiente línea en el archivo phprc:

output_buffering = 4096
