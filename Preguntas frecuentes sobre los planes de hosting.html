General

¿Qué tipo de planes de hosting ofrecen?

DreamHost ofrece Servidores Shared, VPS y Dedicados para alojar tus sitios. También hay un servicio dedicado de WordPress llamado DreamPress disponible. Visita el siguiente artículo para más información:

  • ¿Qué tipo de planes de alojamiento están disponibles?

¿cPanel está disponible en DreamHost?

No. El panel de DreamHost fue construido internamente y actualmente es el único panel disponible. Consulta el siguiente artículo para obtener más información:

  • ¿Dónde puedo encontrar cPanel?

¿Cuántos sitios puedo configurar un plan de hosting?

Depende del plan. Para Alojamientos compartidos, VPS y Dedicado, puedes agregar tantos dominios como desees.

Para DreamPress, cada plan solo está configurado para un dominio único, por lo que si deseas varios dominios en DreamPress, debes comprar un plan DreamPress para cada dominio.

Copias de seguridad

¿Cómo hacer una copia de seguridad de mi dominio?

Hay varias formas de hacer una copia de seguridad de tus datos. Visita el siguiente artículo para más detalles:

  • Respaldar una cuenta en el panel

¿DreamHost proporciona copias de seguridad a largo plazo?

DreamHost mantiene en promedio 2 semanas de copias de seguridad de cada sitio web y 5 días de copias de seguridad de bases de datos. Las copias de seguridad a largo plazo no están incluidas en ningún plan de alojamiento. Sin embargo, DreamHost ofrece un servicio de almacenamiento en la nube llamado DreamObjects. Este servicio te permite hacer una copia de seguridad de los datos que desees a un precio asequible.

Información de servidor

¿Qué sistema operativo se utiliza en los servidores web?

Los servidores compartidos corren Apache.

Los Servidores VPS y Dedicados pueden correr Apache o Nginx.

¿Tengo acceso root a mi servidor web?

El acceso root no está disponible en un plan de alojamiento compartido, VPS o DreamPress. Sin embargo, puedes crear un usuario administrador en un Servidor Dedicado que tenga acceso a sudo/root.

  • Agregar usuarios a un Servidor Dedicado

¿DreamHost admite las extensiones de Windows ASP, .NET o FrontPage?

No. DreamHost es completamente un entorno Linux.
