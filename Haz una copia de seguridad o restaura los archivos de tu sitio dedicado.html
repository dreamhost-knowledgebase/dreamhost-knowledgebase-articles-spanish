Generalidades

Los servidores dedicados no tienen la opción de que los clientes restauren una copia de seguridad del sitio web desde el panel de control. Todas las restauraciones deben realizarse con un usuario administrador, utilizando las técnicas siguientes.

Los siguientes comandos requieren acceso SSH usando un usuario Administrador. Debes cambiar al usuario root usando sudo -s o sudo -i antes de poder ejecutar los siguientes comandos.

Restaurar copias de seguridad

Se puede acceder a las copias de seguridad de los servidores dedicados desde el directorio /mnt/backup del servidor.

[server]$ cd /mnt/backup/home/USERNAME
  1. Para restaurar algo de la copia de seguridad más reciente, cambia el nombre del directorio activo del sitio web que deseas restaurar. Por ejemplo, si tu sitio es example.com, cámbiale el nombre a example.com.old. Este directorio se encuentra en /home/username/.
  2. Crea una nueva carpeta de sitio web vacía. En este ejemplo, se utiliza una carpeta en blanco denominada example.com.
  3. Ejecuta un rsync para copiar los archivos a esta nueva carpeta en blanco: 
    [server]$ rsync -av /mnt/backup/home/USERNAME/example.com/ /home/USERNAME/example.com/

Restaurar copias de seguridad antiguas

Si necesitas copias de seguridad anteriores, las instantáneas se pueden encontrar en el directorio.zfs y de instantáneas:

[server]$ /mnt/backup/.zfs/snapshot# ls
zfs-auto-snap_daily-2016-03-11-16h18  zfs-auto-snap_daily-2016-03-15-16h18  zfs-auto-snap_daily-2016-03-18-16h18
zfs-auto-snap_daily-2016-03-12-16h19  zfs-auto-snap_daily-2016-03-16-16h18  zfs-auto-snap_weekly-2016-03-06-16h18
zfs-auto-snap_daily-2016-03-14-16h18  zfs-auto-snap_daily-2016-03-17-16h18  zfs-auto-snap_weekly-2016-03-13-16h18

Para restaurar algo de una instantánea, cambia el nombre del directorio del sitio web que deseas restaurar como se mencionó anteriormente, crea un nuevo directorio en blanco y luego ejecuta rsync:

[server]$ rsync -av /mnt/backup/.zfs/snapshot/SNAPSHOTDIR/home/USERNAME/example.com/   /home/USERNAME/example.com/
