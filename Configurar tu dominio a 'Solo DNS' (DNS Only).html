Generalidades

Este artículo explica cómo eliminar los registros DNS no editables predeterminados para tu dominio.

Cuando tu dominio está como 'Dominio Alojado', este tiene varias direcciones IP de DreamHost (registros A) asignadas que controlan dónde está alojado el sitio. Si los servidores de nombres de tu sitio apuntan a DreamHost y tu sitio está como "Dominio Alojado", este está alojado en los servidores de DreamHost.

Si los servidores de nombres de tu sitio apuntan a DreamHost pero NO deseas alojar el sitio en DreamHost, debes eliminar el alojamiento al cual elimina estas direcciones IP. Configurar el sitio como 'DNS only' elimina el alojamiento y los registros A de tu sitio. Luego de esto, podrás agregar registros DNS personalizados para apuntar tu sitio a otra empresa de alojamiento.

Esto solo funciona con el nombre de dominio principal como 'example.com'. Esto NO funciona con subdominios como 'blog.example.com'. Para los subdominios, debes agregar registros DNS personalizados al dominio principal como se explica a continuación.

Agregar un nuevo dominio como 'DNS Only'

  1. Abre la página de Manage Domains.
  2. Haz clic en el enlace Add Hosting to a Domain / Sub-Domain.
  3. Cuando se abra la página siguiente, desplázate hacia abajo hasta 'DNS Only'.
  4. Ingresa el dominio que deseas establecer como 'DNS Only' y haz clic en el botón Host DNS Only.

Cambiar un dominio alojado a 'DNS Only'

  1. Navega a la página de Manage Domains.
  2. A la derecha del dominio debajo de la columna 'Alojamiento Web , haz clic en el botón Remove. Esto configura el dominio como 'DNS Only' (si no tienes un certificado SSL habilitado en el dominio).

Si tienes un certificado SSL enabled on the domain, la página 'Manage Domains' solo muestra 'dns' a la derecha de tu dominio cuando debería decir 'DNS Only'. Esto se debe al certificado SSL activo.

Debes eliminar el certificado SSL del dominio para poder eliminar todos los registros DNS y configurar el dominio como 'DNS Only'. Visita la siguiente sección para obtener instrucciones.

Eliminar alojamiento seguro

Si has agregado un certificado SSL  a tu dominio, debes eliminarlo para que se eliminen tus registros DNS.

  1. Navega a la página de Manage Domains.
  2. A la derecha del dominio puedes ver 'https On'. Haz clic en este enlace.
  3. Te llevará a la página de alojamiento seguro para este dominio. En esa página, haz clic en el botón 'Remove https for example.com'.
  4. Navega de vuelta a la página de Manage Domains.
  5. A la derecha de tu dominio ahora verás 'DNS Only'. También puedes hacer clic en el enlace 'DNS' debajo de tu dominio para confirmar que se han eliminado los registros.

¿Qué registros DNS se eliminan?

Cuando configuras un dominio como 'DNS only', eliminas los siguientes registros DNS A:

  • registro A no www
  • Registro A www
  • Registro A ssh
  • Registro A ftp

A continuación hay algunas cosas a tener en cuenta con respecto a estos cambios:

  • No presione el botón Delete para el dominio en la página Manage Domains. Esto elimina el dominio de la cuenta junto con los contenidos que anteriormente estaban alojados en este.
  • Establecer el dominio en ‘DNS Only’ no elimina los archivos alojados. Puedes hacer clic en el botón Edit para establecer el dominio como ‘Fully Hosted’. Esto crea de nuevo los registros DNS para tu dominio.
  • Ten en cuenta que esto solo afecta tu servicio de alojamiento web y deja intactos otros servicios como el correo electrónico (suponiendo que ya tienes un plan de alojamiento activo en tu cuenta). Si has eliminado tu plan de alojamiento pero deseas continuar alojando tu correo electrónico con DreamHost, puedes comprar un Plan de correo electrónico pagodo.

Además, estos cambios solo son posibles para el dominio principal: los subdominios no se pueden establecer como DNS Only.

Si este es tu único dominio en la cuenta y lo estás configurando para DNS Only, NO necesitas un plan de alojamiento. Un plan de alojamiento solo es necesario si deseas alojar un dominio y su contenido en los servidores DreamHost.

¿Qué hacer después de eliminar los registros DNS?

Una vez que el dominio se establece en 'DNS Only', puedes agregar registros DNS personalizados para apuntar tu sitio a donde quieras.
