Generalidades

Un cliente SSH es una aplicación en tu computador que se usa para ejecutar comandos de Shell. Estos comandos te permiten hacer cosas como iniciar sesión en un servidor, descargar archivos y administrar archivos en tu computadora o en un servidor web.

Hay muchos tipos diferentes de clientes SSH entre los que puedes elegir, dependiendo de tu sistema operativo. Este artículo te brinda una descripción general de algunas opciones populares.

Revisa el siguiente artículo para obtener más información sobre diferentes clientes SSH:

  • Comparación de clientes SSH

¿Qué información necesito para conectarme?

Para conectarse a tu servidor necesitarás la siguiente información:

  • Usuario.
  • Contraseña.
  • Nombre del servidor o nombre del sitio web.

Unix/Linux

La mayoría de las distribuciones de Unix/Linux vienen con el paquete OpenSSH. Cualquier aplicación de terminal (xterm, Konsole, etc.) se puede usar para iniciar un shell seguro.

Para abrir tu terminal, presiona Ctrl + Alt + T. Esto debería abrir automáticamente tu terminal donde puedes comenzar a ejecutar comandos. También puedes buscar la palabra 'terminal' en tu panel de distribución. Esto varía según la distribución de Linux que estés utilizando.

Windows

Windows 10 viene con un Shell incorporado. Puedes usar esto como un cliente SSH usando el símbolo del sistema de Windows o PowerShell. Mira el siguiente artículo para más detalles:

  • Usar Shell en Windows 10

También hay clientes de terceros que puedes usar con Windows. Cada cliente SSH de Windows tiene su forma de instalarse y ejecutarse, pero una vez que lo hayas configurado, no hay diferencia en otras operaciones. Sin embargo, cada cliente se adapta a necesidades específicas (o generales). Por lo tanto, es posible que desees probar más de uno para ver qué funciona mejor para ti. Aquí hay algunas opciones:

  • PuTTY
    Este es un cliente gratuito y muy popular para usuarios de Windows.
  • WinSCP
    Este es un cliente gratuito y popular también. Sin embargo, no es un cliente que funcione completamente en comparación con PuTTY.
  • SmartFTP
  • SecureCRT
  • ZOC Terminal
    También disponible para OS X.

Chrome

Secure Shell es un emulador de terminal y un cliente ssh independiente para el navegador web Chrome.

iPhone

Puede ubicar varias terminales en la App Store. Simplemente busca 'terminal ssh'. Una aplicación paga popular es Termius. Pero hay muchas otras opciones disponibles.

Android

Hay varios clientes que puede descargar de la tienda Google Play. Dos opciones son ConnectBot y JuiceSSH.
