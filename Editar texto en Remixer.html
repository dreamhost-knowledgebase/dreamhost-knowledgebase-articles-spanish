Generalidades

Cuándo estás editando una página, puedes editar y modificar el estilo del texto como desees. El editor incorporado te permite hacer varias cosas como:

  • Alinear texto.
  • Escoger fuente.
  • Cambiar tamaño.
  • Negrita/Cursiva.
  • Enlazar texto.
  • Agregar listas.

Solo puedes editar/agregar texto dentro de una sección de campo de texto predefinida de una 'Web Section'. No es posible agregar texto fuera de una 'Web Section' o eliminar un cuadro de texto de una 'Web Section'.

Cómo editar texto

  1. Asegúrate de estar en editar página .
  2. Busca el texto que vas a actualizar.
  3. Haz clic en el texto. Este se resalta y aparece la barra de herramientas de edición.
  4. Edita (o elimina) el texto según sea necesario. Por ejemplo, puedes cambiar el tamaño de fuente, el estilo de fuente, la alineación o color.
  5. Cuando termines de editar, haz clic fuera de la sección para guardar automáticamente.
