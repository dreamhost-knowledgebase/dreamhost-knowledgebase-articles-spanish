Generalidades

Lo siguiente explica algunas preguntas frecuentes sobre PHP.

Preguntas frecuentes

¿Qué versiones de PHP soporta DreamHost?

Revisa este artículo para obtener una lista de las versiones actuales de PHP compatibles con DreamHost. Por favor ten en cuenta que las versiones de PHP se actualizan continuamente. Visita la siguiente página para ver una línea de tiempo de las versiones de PHP:

  • php.net/supported-versions.php

¿Puedo configurar un archivo personalizado php.ini?

Sí. Revisa este artículo para obtener detalles sobre cómo configurarlo en tu sitio.

¿Dónde puedo ver la configuración predeterminada de PHP de DreamHost?

Puedes crear un archivo llamado phpinfo.php y subirlo a tu sitio para ver tu configuración de PHP. Para ver las instrucciones, revisa este artículo.

¿Ofrecen el registro de errores de PHP?

Sí. Sin embargo, por defecto, los errores de PHP no se registran. Puedes habilitar y establecer la ruta al registro de PHP con el archivo phprc. Visita el artículo registro de errores de PHP para más detalles.
