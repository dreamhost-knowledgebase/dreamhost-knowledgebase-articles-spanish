Generalidades

Hay dos maneras de acceder a tu servidor DreamPress.

  • WebFTP dentro del panel
  • Cliente FTP de terceros

Cualquier opción te permite acceder a tu servidor de DreamPress para trabajar en los archivos de tu sitio.

Opción 1 — WebFTP

WebFTP es una opción integrada en tu panel de DreamPress.

  1. Navega a la página Managed WordPress.
  2. Hacia la derecha de tu dominio, haz clic en el botón Manage.
  3. Haz clic en el botón Launch WebFTP en la parte inferior izquierda.

Esto abre automáticamente una nueva pestaña en tu navegador que inicia sesión en tu servidor DreamPress. Consulta el siguiente artículo para obtener información sobre cómo usar la interfaz WebFTP para administrar tus archivos.

  • WebFTP

Opción 2 — Cliente FTP de terceros

La otra opción es usar un cliente FTP para conectarse. La siguiente página enumera los populares clientes FTP de terceros que puedes usar para conectarte a tu servidor DreamPress.

  • Lista de programas de clientes FTP

Una vez que estés conectado, puedes usar el siguiente artículo para crear y editar archivos dentro de tu cliente FTP.

  • Crear y editar archivos via FTP

Ubicar tus credenciales de inicio de sesión FTP

  1. Navega a la página Managed WordPress.
  2. Hacia la derecha de tu dominio, haz clic en el botón Manage.
  3. En la parte inferior izquierda verás la sección 'File Upload'. Bajo el botón WebFTP, haz clic en la opción al lado de 'Show Credentials'. Tus credenciales FTP para DreamPress se mostrarán.
  4. Haz clic en el icono a la derecha de cada ítem para copiar y pega los credenciales en el cliente FTP de tu elección.

Si necesitas cambiar la contraseña de tu usuario, puedes hacerlo desde la página Manage Users. Revisa el siguiente artículo para mayores detalles:

  • Cambiar contraseña (usuarios Web)

Advertencia: la clave de host del servidor es desconocida

Cuando inicias sesión originalmente en un servidor DreamHost, puedes ver la siguiente advertencia:

The server's host key is unknown. You have no guarantee that the server is the computer you think it is.

(La clave de host del servidor es desconocida. No tiene garantía de que el servidor sea la computadora que cree que es.)
