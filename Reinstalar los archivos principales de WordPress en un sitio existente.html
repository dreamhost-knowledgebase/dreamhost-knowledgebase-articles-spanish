Generalidades

Las instrucciones de este artículo reinstalan los archivos principales de WordPress en tu sitio existente de WordPress. Estos pasos asumen que tus archivos originales de WordPress todavía están en el directorio de tu sitio web actual.

¿Qué hacen esos pasos?

Los siguientes métodos solo reinstalan los archivos principales de WordPress. Esto significa que tus temas, plugins, base de datos y carpeta wp-content no se ven afectados.

  • Opción 1 — Reinstalar en el panel de WordPress
  • Opción 2 — Reinstalar usando FTP
  • Opción 3 — Reinstalar usando wp-cli

Si necesitas instalar una nueva copia de WordPress en un sitio web vacío, consulta los siguientes artículos.

  • Cómo instalar WordPress usando el Instalador de Un Clic
  • Instalación de WordPress usando wp-cli y SSH

Opción 1 — Reinstalar en el panel de WordPress

Puedes reinstalar tus archivos principales en el panel de WordPress si aún puedes iniciar sesión.

  1. Inicia sesión en tu sitio de WordPress en example.com/wp-login.php.
  2. Navega hasta Escritorio > Actualizaciones.
  3. En la página siguiente, haz clic en el botón Reinstalar la version x.x.x.

Opción 2 — Reinstalar usando FTP

Si no tienes acceso a tu panel de WordPress, puedes cargar y sobrescribir los archivos usando un cliente FTP.

  1. Visita https://wordpress.org/download para descargar el archivo .zip de WordPress a tu computador.
  2. Descomprime el archivo en tu computador. Esto descomprime todos los archivos y directorios de WordPress.
  3. Dentro de estos archivos, elimina el directorio wp-content.
  4. Inicia sesión en tu sitio usando un cliente FTP.
  5. Carga todos los archivos y directorios que acabas de descargar en tu sitio actual. Cuando se te solicite, elige sobrescribir los archivos en el servidor.

Opción 3 — Reinstalar usando wp-cli

wp-cli es una herramienta de línea de comandos que puede utilizar para administrar su sitio de WordPress. El siguiente ejemplo usa la opción --skip-content opara descargar WordPress sin los temas y plugins predeterminados. Puede leer más sobre este comando aquí.

  1. Inicia sesión en tu sitio vía SSH.
  2. Navega al directorio principal de tu sitio. Asegúrate de cambiar username a tu usuario Shell. 
    [server]$ cd /home/username/example.com
  3. Corre el siguiente comando para descargar tus archivos principales. 
    [server]$ wp core download --skip-content --force
