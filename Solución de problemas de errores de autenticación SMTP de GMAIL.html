Muchos sitios web usan PHP para enviar correos a través de SMTP. Por ejemplo, puedes crear un formulario de correo personalizado o usar un plugin de SMTP de WordPress.

  • Enviar correo PHP vía SMTP

Un problema con el que te encontrarás es que la configuración de seguridad de Google no te permite enviar a través de SMTP a través de tus servidores a menos que lo permitas explícitamente.

Habilitar el protocolo OAuth 2.0 para autenticar conexiones

La solución es habilitar la autenticación OAuth 2.0 de tu aplicación.

  • https://developers.google.com/gmail/api/auth/about-auth
  • Habilitar el API OAuth 2 para usar con tu plugin Mail SMTP de WordPress

Por favor ten en cuenta que el soporte no puede ayudarte con esta configuración personalizada.
