Generalidades

Un error común que sucede en los sitios WordPress es el error 'redirección infinita'. Esto puede aparecer de diferentes formas dependiendo de tu buscador. Por ejemplo:

Firefox

"La página no está redirigiendo correctamente"
Chrome
"La página no está funcionando"

Causas del error

Este error error usualmente es causado por que el HOME y SITE URL dentro de WordPress no están configurados correctamente. Por ejemplo, si tu sitio es example.com, el Home y Site URL en tu base de datos deberán apuntar a:

  • https://example.com. 

Si Home y Site URL agregan www a la URL, puede aparecer así en tu base de datos:

  • https://www.example.com

Esto puede causar un redirección infinita.

Otras causas

Hay unas varias opciones que tu sitio pueda redirigir.

Plugins

Es posible que un plugin que hayas agregado a tu sitio cause una redirección. Trata deshabilitando tus plugins para confirmar que esta es la causa.

Archivo .htaccess

Tu archivo .htaccess puede tener una redirección configurada en algún lugar. Abre el archivo de .htaccess y mira sus contenidos. Busca por alguna redirección de tu sitio a una URL diferente.

Configuración de Cloudflare

Si has agregado un certificado SSL a tu sitio web y también estás usando Cloudflare, debes configurar algunas cosas en Cloudflare. Hay una configuración en el panel de Cloudflare para para configurar tu certificado SSL a usar un tipo específico. Este debe ser 'Full (strict)'.
