Administrar Registros de Dominio

La página Administrar Registros de Dominio es donde puedes registrar un nuevo dominio y administrar los registros de dominio actuales que tienes con DreamHost. Las modificaciones de dominio WHOIS (incluyendo nameservers) se pueden hacer desde aquí, así como la configuración de renovación de registro y el bloqueo de dominio.

Revisa el siguiente artículo para más detalles:

  • Registro de Dominios

Transferir Registros

La página Transferir Registros de Dominios es donde puedes transferir un registro de dominio hacia o desde DreamHost, así como obtener el código de autorización del dominio.

Revisa los siguientes artículos para más detalles:

  • Transferir tu registro de dominio a DreamHost
