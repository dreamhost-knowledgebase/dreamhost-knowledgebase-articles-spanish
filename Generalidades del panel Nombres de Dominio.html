Administrar Dominios

La página Administrar Dominios te permite administrar las siguientes funciones para tus dominios.

La página 'Administrar Dominios' te permite administrar el alojamiento de tu dominio. Aquí puedes agregar alojamiento y ajustar la configuración del dominio, y puedes restaurar los archivos web respaldados.

Mira el siguiente artículo para más detalles:

  • Generalidades sobre el panel Dominios

Registros de Dominios

La página Registros de Dominios es donde puedes registrar un nuevo dominio y administrar los registros de dominio actuales que tienes con DreamHost. Las modificaciones de dominio WHOIS (incluyendo nameservers) se pueden hacer desde aquí, así como la configuración de renovación de registro y el bloqueo de dominio.

Mira el siguiente artículo para más detalles:

  • Registro de Dominios

Transferencia de Registros

La página Transferencia de registros de dominios es donde puedes transferir un registro de dominio hacia o desde DreamHost, así como obtener el código de autorización del dominio.

Revisa los siguientes artículos para más detalles:

  • Transferir tu registro de dominio afuera de DreamHost
  • Transferir tu registro de dominio a DreamHost
