Generalidades

Este artículo explica cómo lanzar una nueva instancia en DreamCompute dentro de tu panel.

Están disponibles las siguientes distribuciones:

  • Ubuntu
  • Debian
  • Fedora

Lanzamiento de una instancia

  1. Navega hasta la página DreamCompute e inicia sesión en tu panel de DreamCompute.
  2. En la parte superior izquierda, haz clic en la pestaña Compute.
  3. En la lista expandida, haz clic en 'Instances'.

    El panel muestra las instancias existentes con tu información.

  4. Haz clic en el botón 'Launch Instance' en la parte superior derecha.
    Aparecerá un cuadro emergente de 'Launch Instance'.
  5. En el cuadro 'Launch Instance', haz clic en varias pestañas para especificar qué tipo de instancia deseas crear.

Pestaña de detalles — Información general

Nombre de instancia

Asigna un nombre a la máquina virtual.

El nombre que asignes se convierte en el nombre de host inicial de la instancia. Si el nombre tiene más de 63 caracteres, el servicio Compute lo trunca automáticamente para garantizar que dnsmasq funcione correctamente.

Si cambias el nombre de host directamente después de crear la instancia, el nombre no se actualiza en el panel.

No se garantiza que los nombres de instancia sean únicos cuando se crean, por lo que es posible que existan dos instancias con el mismo nombre de host.

Zona de disponibilidad

Actualmente, la única zona en la que se pueden lanzar instancias de DreamCompute es iad-2, que se encuentra en el centro de datos de DreamHost en Ashburn, Virginia, EE. UU.

Contar
Para lanzar varias instancias, ingresa un valor mayor que el valor predeterminado de 1.

Pestaña fuente — seleccionar la plantilla desde la que crear tu instancia

Seleccione Boot Source (4 opciones)

Hay cuatro opciones para la fuente de arranque:

  • Imagen
  • Captura de la Instancia
  • Volumen
  • Captura del Volumen

Imagen (selecciona para crear una nueva instancia)

Selecciona esta opción si deseas utilizar una imagen seleccionada por DreamHost o si has subido tu propia imagen. Una vez seleccionado, aparece una lista de imágenes disponibles en la parte inferior que puedes usar para crear tu nueva instancia. (Por ejemplo, Debian, Fedora, Ubuntu, CentOS, etc.)

Da clic a botón a la derecha de la imagen para seleccionarla. Si eliges 'Crear nuevo volumen' (que es la opción predeterminada), asegúrate de también:

  • Ingresar un 'Tamaño de volumen' (de lo contrario, tu volumen se establecerá por defecto en el tamaño mínimo requerido de la imagen que seleccione).
  • Elige si deseas que se elimine el volumen cuando elimine la instancia.

Capturar la Instancia

Selecciona esta opción si tienes una 'Captura de instancia' existente de una instancia existente o anterior. Una vez seleccionada, aparece una lista de instantáneas disponibles en la parte inferior que puedes usar para crear una nueva instancia.

Da clic al botón a la derecha de la imagen para seleccionarla. Si eliges 'Crear nuevo volumen' (que es la opción predeterminada), asegúrate de también:

  1. Ingresar un 'Tamaño de volumen' que coincida con el tamaño de tu 'Captura de instancia'.
  2. Elige si deseas que se elimine el volumen cuando elimines la instancia.

Volumen

Selecciona esta opción si tienes un volumen existente y te gustaría crear un formulario de instancia. Una vez seleccionado, se muestra una lista de volúmenes disponibles. Haz clic en el botón a la derecha del volumen para seleccionarlo. Asegúrate de elegir también si deseas que se elimine el volumen cuando elimines la instancia.

Captura del Volumen

Selecciona esta opción si tienes una "Captura del Volumen" existente desde la que te gustaría crear una instancia. Una vez seleccionado, se muestra una lista de Capturas de volumen disponibles. Haz clic en el botón a la derecha de la instantánea de volumen para seleccionarla. Asegúrate de elegir también si deseas que se elimine el volumen cuando elimines la instancia.

3 opciones de configuración

Según cuál de estas opciones desplegables selecciones, puedes ver hasta tres opciones adicionales para administrar la fuente de arranque de tu instancia:

  • Crear Nuevo Volumen
  • Tamaño del Volumen (GB)
  • Eliminar Volumen en Instancia Eliminar

Crear Nuevo Volumen

La opción "Crear Nuevo Volumen" determina si tu instancia utiliza un disco de arranque efímero de 80 GB o un disco de arranque de volumen del tamaño elegido. Consulta el siguiente artículo para obtener más detalles.

  • ¿Cuál es la diferencia entre los discos de arranque efímeros y por volumen?

Tamaño del Volumen

La opción "Tamaño de volumen (GB)" determina el tamaño de tu disco de arranque de volumen.

Eliminar Volumen en Instancia Eliminar

La opción "Eliminar volumen al eliminar la instancia" determina si el disco de arranque del volumen se elimina o no al eliminar la instancia. De forma predeterminada, se establece en "No" (que se recomienda). Esto te permite eliminar la instancia y usar el volumen existente como fuente de arranque de una nueva instancia.

Pestaña de Flavor — asignar recursos de CPU virtual y RAM a tu instancia

Especifica el tamaño de la instancia para lanzar. Haz clic en el símbolo a la derecha para agregarlo.

El valor de 'Total Disk' de 80 GB solo se aplica si no creas un volumen nuevo.

Puedes cambiar el tipo de una instancia después de lanzarla para aumentar o disminuir los recursos disponibles para la instancia. Para obtener más información, consulta el siguiente artículo:

  • Cómo cambiar los límites de recursos de una instancia de DreamCompute

Pestaña redes — cómo se conecta tu instancia a Internet

Si has habilitado 'Private Networking', debes especificar a qué red deseas asociar la instancia. Si no has habilitado la red privada, tu instancia usa la red pública de forma predeterminada y esta pestaña puede ignorarse.

Pestaña Grupos de Seguridad — que puertos están abiertos

El grupo de seguridad predeterminado se asigna automáticamente a la instancia, pero si deseas eliminar el grupo de seguridad predeterminado y agregar otros grupos, puedes hacerlo utilizando los símbolos de flecha hacia arriba y hacia abajo.

Puedes cambiar qué grupos de seguridad están asociados con una instancia después de lanzarla para modificar la accesibilidad de la red a la instancia. Consulta el siguiente artículo para obtener más detalles:

  • Configurar grupos de seguridad

Pestaña de Llaves de Emparejamiento — Autenticación SSH para el usuario predeterminado de tu instancia

A cada instancia solo se le puede asignar una única Llave de Emparejamiento. Cuando lanzas originalmente una instancia, puedes seleccionar el Llave de Emparejamiento que deseas utilizar con la nueva instancia.

No es posible dentro del panel DreamCompute cambiar las Llaves de Emparejamiento asignado a una instancia después de que se crea la instancia. Asegúrate de asignar las Llaves de Emparejamiento correctas cuando crees originalmente la instancia.

Si ya creaste y agregaste tu llave de SSH pública, aparece aquí. Haz clic en la flecha ARRIBA para mover la clave existente debajo de la sección Allocated.
Si aún no se ha creado una llave, puedes elegir'Create Key Pair' o 'Import Key Pair'. Se puede acceder a cada instancia que inicies a través de SSH mediante autenticación basada en llaves. Una vez que cargue tu clave pública, podrás usar inicios de sesión sin contraseña con el usuario predeterminado para tu imagen.

Si aún no tienes una clave SSH, revisa el siguiente artículo para obtener ayuda para crear/importar una:

  • Cómo crear una Llave de Emparejamiento

Lanzar una nueva instancia

Cuando termines de ingresar todas tus configuraciones, haz clic en el botón Launch Instance en la parte inferior derecha.

Si alguna de las pestañas mencionadas anteriormente todavía contiene un símbolo * junto a ellas, no se puede hacer clic en el botón "Launch Instance".

Conectarte a tu instancia

Cuando se crea la instancia, puedes conectarte a ella mediante tu Llave de Emparejamiento de SSH. Consulta el siguiente artículo para obtener instrucciones sobre cómo conectarte:

  • Descripción general sobre la conexión a tu instancia con llaves SSH
