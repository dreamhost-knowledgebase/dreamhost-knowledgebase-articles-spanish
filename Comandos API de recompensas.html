Advertencia de fin de funcionamiento

Este comando de API será eliminado y no va a funcionar en Noviembre 2 de 2021.

Ya no es posible agregar códigos de promoción como afiliado. Mira el artículo Programa de recompensas Refer and Earn para más detalles.

El módulo rewards de API contiene comandos para administrar códigos de promoción y seguir referidos.

Comandos

rewards-remove_promo_code

Elimina un código de promoción (Solo es permitido si el código nunca ha sido usado).

Comando rewards-remove_promo_code
Valores

code
Resultado

success
Posibles errores

invalid_code
code_used

rewards-enable_promo_code

Habilita un código de promoción para que pueda volver a utilizarse (debe haber sido desactivado).

Comando rewards-enable_promo_code
Valores

code
Resultado

success
Posibles errores

invalid_code

rewards-disable_promo_code

Deshabilita un código de promoción para que ya no se pueda usar (debe estar habilitado).

Comando rewards-disable_promo_code
Valores

code
Resultado

success
Posibles errores

invalid_code

rewards-list_promo_codes

Muestra todos los códigos de promoción que has creado.

Comando rewards-list_promo_codes
Valores

(ninguno)
Resultado

success
code description created status used
TESTCODE my promo code 2010-11-16 17:37:05 active 0
Posibles errores

(ninguno)

rewards-promo_details

Obtiene la información detallada de un código de promoción (debes haber creado el código).

Comando rewards-promo_details
Valores

code
Resultado

success
code description status discount_month discount_1year discount_2year bonus_domregs bonus_ips
TESTCODE my promo code active $0.00 $0.00 $0.00 2 1
Posibles errores

no_code
invalid_code

rewards-referral_summary

Da un resumen de todas las referencias para el período de tiempo dado.

Comando rewards-referral_summary
Valores

period : Un periodo como "1 day" o "3 months"
Resultado

success
code signups raw_clicks unique_clicks conversion
none 0 8 8 0.00%
TESTCODE 213 5515 5235 4.07%
Total 213 5523 5243 4.06%
Posibles errores

no_period
invalid_period

rewards-referral_log

Devuelve un registro sin procesar de referencias para el período de tiempo dado. Si la persona se registró, también muestra su webid y la hora de registro.

Comando rewards-referral_log
Valores

period : un período de tiempo como "1 día" o "3 meses"
Resultado

success
click_time ip code signup_time ref_url webid status

2010-11-16 17:32:27 9.10.11.12 TEST N/A https://www.example.com N/A no signup
Posibles errores

no_period
invalid_period
