Generalidades

Python es un lenguaje de programación como PHP y Ruby que te permite crear muchos tipos diferentes de aplicaciones. Se puede encontrar más información en el sitio web de Python:

  • https://www.python.org/about/

Visita los siguientes enlaces para ver las versiones actuales de Python.

  • Versiones de Ubuntu

También puedes instalar cualquier versión de Python 2 o Python 3 que desees.

Implementación

Tienes algunas opciones sobre cómo implementar tu aplicación Python en DreamHost:

  • Passenger – La forma preferida de implementar aplicaciones Python. Este es un módulo de Apache que se escribió originalmente para admitir la ejecución de aplicaciones Ruby on Rails, pero que también brinda soporte para aplicaciones compatibles con WSGI. Mira el artículo sobre Passenger para obtener detalles de configuración.
  • CGI – Lento y no recomendado.
  • FastCGI – Más rápido que CGI y utiliza mod_fcgi.

Marcos Web

Algunos marcos de sitios web de Python pueden ejecutarse en servidores DreamHost. Éstos incluyen:

  • Django
  • Pylons
  • Turbogears
  • Web.py
  • Flask
