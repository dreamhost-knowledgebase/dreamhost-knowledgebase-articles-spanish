Generalidades

Este artículo explica cómo usar el plugin WP Mail SMTP para enviar correos desde tu sitio de WordPress, usando tu dirección de correo de Google.

Si no deseas usar una dirección de Gmail para enviar desde tu sitio de WordPress, mira el siguiente artículo.

  • Configurar el plugin 'WP Mail SMTP'

Configurar el plugin WP Mail SMTP

  1. En el panel de WordPress, haz clic en 'Plugins > Añadir nuevo' en el panel izquierdo.
  2. Busca wp-mail-smtp. También puede encontrase aquí.
  3. Haz clic en el botón Instalar ahora, luego Activar.
  4. Después de que esté instalado y activado, navega a la página 'Plugins' en tu panel de WordPress.
  5. Haz clic en el enlace Ajustes debajo del plugin.
  6. Rellena los siguientes campos:
    • Correo electrónico remitente - Ingresa el correo DESDE el que deseas enviar.
    • Forzar remitente del mensaje - Marca esta casilla.
    • Nombre del remitente - Ingresa un nombre que te gustaría aparezca como REMITENTE.
    • Forzar el nombre del remitente - Marca esto si deseas forzar el nombre del REMITENTE.
    • Ruta de retorno - Marca esta casilla.
  7. Debajo de la sección Servicio de correo electrónico selecciona Otro SMTP.
  8. Desplázate hacia abajo a la sección Gmail.
    • ID cliente - Esta es una cadena de letras y números que crearás en tu cuenta de Google.
    • Clave secreta de cliente - Esta es una cadena de letras y números que crearás en tu cuenta de Google.
    • URI de redirección autorizada- Ingresarás esto en tu cuenta de Google.
  9. Haz clic en el botón Guardar ajustes.

Ahora debes configurar tus ajustes dentro de tu cuenta de Google.

Permitir que GMAIL envíe desde tu aplicación de correo

Ahora debes permitir que Google envíe correo desde tu sitio WordPress. Puedes hacer esto al habilitar la autenticación OAuth2 en tu cuenta de Google.

  • Habilitar el API OAuth 2 para usar con tu plugin Mail SMTP de WordPress

Agregar código para esconder tus credenciales

Las instrucciones anteriores agregan tus credenciales de Google al panel de WordPress. Para asegurar aún más tus credenciales, el plugin recomienda que agregues código a tu archivo wp-config.php en su lugar. Mira la siguiente página para más detalles.

  • Cómo proteger las configuraciones SMTP usando constantes

Agregar las constantes a tu archivo wp-config.php.

  1. Inicia sesión en tu servidor vía FTP o SSH.
  2. Usa uno de los siguientes artículos para editar tu archivo wp-config.php.
    • Crear y editar archivos vía FTP
    • Crear y editar un archivo vía SSH
  3. Baja en este archivo hasta la siguiente línea:

    /* That's all, stop editing! Happy blogging. */

  4. Agrega las siguientes líneas justo arriba de ésta línea. 
    define( 'WPMS_ON', true );
define( 'WPMS_GMAIL_CLIENT_ID', '********************.apps.googleusercontent.com');
define( 'WPMS_GMAIL_CLIENT_SECRET', '**********************');
define( 'WPMS_MAILER', 'gmail' );
  5. Las únicas partes que necesitas cambiar son WPMS_GMAIL_CLIENT_ID y WPMS_GMAIL_CLIENT_SECRET.
  6. Guarda el archivo.
  7. Si ves la página del plugin ahora, verás que no se puede acceder a tus credenciales.
