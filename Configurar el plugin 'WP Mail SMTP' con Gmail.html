Generalidades

Este artículo explica cómo usar el plugin WP Mail SMTP para enviar correos desde tu sitio de WordPress, usando tu dirección de correo de Google.

Si no deseas usar una dirección de Gmail para enviar desde tu sitio de WordPress, mira el siguiente artículo.

  • Configurar el plugin 'WP Mail SMTP'
 

Paso 1 — Permitir que GMAIL envíe desde tu aplicación de correo

Para usar una dirección de correo de Google, inicia sesión en tu cuenta de Google y habilitar la autenticación OAuth2. Esto te permite a Google enviar correos desde tu sitio WordPress. Revisa el siguiente artículo del sitio web del plugin WP Mail SMTP para ver las instrucciones completas:

  • Google Workspace / Gmail

Paso 2 — Configurar el plugin WP Mail SMTP

Revisa el artículo WP Mail SMTP para ver instrucciones sobre la instalación de este plugin. Una vez instalado, usa los siguientes pasos para configurarlo.

  1. Navega a la página WP MAIL SMTP > General.
  2. Rellena los siguientes campos:
    • Correo electrónico remitente - Ingresa el correo de Google DESDE el que deseas enviar.
    • Forzar remitente del mensaje - Marca esta casilla.
    • Nombre del remitente - Ingresa un nombre que te gustaría aparezca como REMITENTE.
    • Forzar el nombre del remitente - Marca esto si deseas forzar el nombre del REMITENTE.
    • Ruta de retorno - Marca esta casilla.
  3. Escoge Google / Gmail debajo de Mailer section.
  4. Llena los siguientes campos:

    Esas credenciales fueron creadas en el Paso 1 de arriba.

    • ID cliente - Esta es una cadena de letras y números que creaste en tu cuenta de Google.
    • Clave secreta de cliente - Esta es una cadena de letras y números que creaste en tu cuenta de Google.
    • URI de redirección autorizada- Ingresarás esto en tu cuenta de Google.
  5. Haz clic en Guardar ajustes.

Paso 3 — Agregar código para esconder tus credenciales

Las instrucciones anteriores agregan tus credenciales de Google al panel de WordPress. El plugin recomienda que agregues código a tu archivo wp-config.php para asegurar aún más tus credenciales,. Revisa la siguiente página para más detalles.

  • Cómo proteger las configuraciones SMTP usando constantes

Agregar las constantes a tu archivo wp-config.php.

  1. Inicia sesión en tu servidor vía FTP o SSH.
  2. Usa uno de los siguientes artículos para editar tu archivo wp-config.php.
    • Crear y editar archivos vía FTP
    • Crear y editar un archivo vía SSH
  3. Baja en este archivo hasta la siguiente línea:

    /* That's all, stop editing! Happy blogging. */

  4. Agrega las siguientes líneas justo arriba de ésta línea. 
    define( 'WPMS_ON', true );
define( 'WPMS_GMAIL_CLIENT_ID', '********************.apps.googleusercontent.com');
define( 'WPMS_GMAIL_CLIENT_SECRET', '**********************');
define( 'WPMS_MAILER', 'gmail' );
    Lo único que necesitas cambiar es WPMS_GMAIL_CLIENT_ID y WPMS_GMAIL_CLIENT_SECRET
  5. Guarda el archivo.

Si ahora ves la página del plugin, verás que tus credenciales no son accesibles.

Probar plugin

Ahora debes probar el plugin para confirmar que se esten enviando los correos correctamente desde tu dirección de Google. Revisa el siguiente artículo para obtener más instrucciones:

  • Probar el plugin WP Mail SMTP
