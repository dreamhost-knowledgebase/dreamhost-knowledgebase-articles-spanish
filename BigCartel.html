Características

BigCartel es un creador de sitios que se especializa en crear una tienda en línea para tu mercancía. Puedes usarlo para crear tu sitio web minorista para vender productos a tus clientes finales. Aprende más sobre BigCartel en el siguiente enlace:

  • https://www.bigcartel.com/

Apuntar tu sitio DreamHost a BigCartel

Si tu dominio está registrado en DreamHost, puedes apuntar tus DNS a BigCartel.

La siguiente captura de pantalla utiliza el dominio example.com. Tu dominio aparece en tu lugar cuando sigue estas instrucciones.

  1. Configura tu dominio en tu panel como Solo DNS.
  2. Revisa el siguiente enlace para obtener información sobre cómo crear un dominio personalizado en BigCartel. Este paso te proporciona tu ID de tienda:
    • Dominio personalizado
  3. En tu panel DreamHost, crea un registro CNAME.

    Host blog
    Apunta a el-nombre-de-tu-tienda.bigcartel.com
  4. Ingresa el Host y el Valor, y luego haz clic en el botón Agregar Registro.

Tu sitio ahora apunta a BigCartel. Sin embargo, solo se apunta la versión 'www' de tu sitio. La siguiente sección explica cómo crear una redirección a la versión 'www' de tu sitio para que siempre funcione.

Redireccionar tu URL de no-www a www

  1. Mira el artículo Redireccionar de dominio para obtener instrucciones detalladas sobre cómo crear una redirección.
  2. Ingresa los siguientes campos en tu panel:
    • Dominio a redireccionar: example.com
    • Redireccionar a URL: www.example.com
    Reemplaza example.com con tu URL
  3. Guarda tu redirección.

Tu sitio ahora siempre redirige a su versión 'www' que luego redirige a BigCartel.
