¿Qué es un VPS MySQL?

DreamHost MySQL VPS es el servicio premium de base de datos MySQL de DreamHost. Similar en concepto a VPS de DreamHost, DreamHost MySQL VPS usa Linux Containers (LXC) para aislar el servidor de la base de datos.

La diferencia entre servidores MySQL privados y compartidos

Todos los planes de alojamiento de DreamHost están configurados con un servidor MySQL compartido de forma predeterminada. Estos servidores de bases de datos compartidas 'comparten' los recursos del servidor con todos los usuarios y bases de datos.

VPS MySQL es diferente en que tus bases de datos y recursos están protegidos de otros usuarios.

La diferencia entre un VPS y un VPS MySQL

Lo siguiente describe la diferencia entre un plan de alojamiento de VPS y un servicio de VPS MySQL:

  • Un VPS es un plan de alojamiento que incluye almacenamiento gratuido para tus bases de datos en un servidor MySQL compartido.
  • Un VPS MySQL, por otro lado, es un servicio premium que solo almacena bases de datos MySQL y es recomendado si tienes bases de datos grandes que requieren más recurdos que un servidor MySQL compartido.

Un VPS MySQL puede ser agregado a cualquier cuenta de DreamHost con un plan de alojamiento Shared Unlimited, VPS, o Dedicado.

¿Está disponible el acceso a root?

El acceso a root y SSH no está disponible en un MySQL VPS. Si requieres de acceso Root/SSH, debes actualizar a un Servidor Dedicado donde las bases de datos de MySQL están ubicadas en la misma máquina, lo que permite el acceso a los archivos de configuración de MySQL.
