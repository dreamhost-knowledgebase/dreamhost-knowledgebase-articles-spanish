Generalidades

Agregar un certificado SSL a tu sitio de WordPress es un excelente primer paso para proteger tus datos. También tienes la opción de agregar Cloudflare a tu dominio además de tu certificado SSL. Esto te permite aprovechar los servicios de Cloudflare, mientras continúas asegurando tu sitio web. Las siguientes secciones explican cómo asegurarte de que Cloudflare esté configurado correctamente para tu sitio WordPress.

Configurar WordPress para usar un certificado SSL

Consulta el siguiente artículo para ver todos los pasos que debes seguir para ajustar tu sitio de WordPress para usar un certificado SSL.

  • ¿Cómo usar un certificado SSL con WordPress?

Cuando habilitas Cloudflare en el panel de DreamHost, tu sitio debe usar la versión 'www' de la URL. Por ejemplo:

  • www.example.com

Si tu sitio NO utiliza actualmente la versión 'www', asegúrate de que cuando actualices las URL en tu base de datos, también incluyas el www como parte de la URL actualizada.

Si no agregas www en este momento, tu sitio arrojará un mensaje de error de redireccionamiento después de agregar Cloudflare.

Ajustar la configuración SSL de Cloudflare

Estos pasos suponen que has instalado ya sea un certificado 'Let's Encrypt gratuito' o un certificado 'Sectigo pago'.

  1. Inicia sesión en tu cuenta de Cloudflare en https://dash.cloudflare.com/login.
  2. Verás tu sitio web en la lista. Haz clic en tu sitio web.

  3. En la fila superior, haz clic en el botón 'SSL/TLS'.
  4. Puedes ver tu opción de SSL en la lista. Con una cuenta gratuita de Cloudflare, el valor predeterminado es la opción 'Flexible'.
  5. Haz clic en el botón 'Full (strict)'.

Después de unas horas, el DNS se actualizará y todas las conexiones entre el usuario, Cloudflare y DreamHost serán seguras.
