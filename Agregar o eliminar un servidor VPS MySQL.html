Agregar un VPS MySQL

  1. Navega a la página VPS.
  2. Da clic en el botón Añadir un VPS MySQL ubicado en la parte inferior de la página.
  3. Escoge un plan, y luego de clic en el botón Obtener este VPS para comprarlo.

    Una vez agregado, puede tomar unas pocas horas para provisionar el nuevo servidor.

  • Cuando te registras para un VPS MySQL, todas tus bases de datos se trasladan a la medianoche, hora del Pacífico. (El panel lo muestra como una fusión).
  • El acceso SSH no está disponible.

Una vez aprovisionado, puedes administrarlo en la página del panel de VPS.

Eliminar un VPS MySQL

Si tienes un VPS MySQL con 3GB o más de datos y no tiense un Servidor Dedicado, no puede usar el botón Eliminar en el panel para eliminar tu MySQL VPS. Debes reducir el tamaño a menos de 3GB para cada base de datos MySQL VPS para poder eliminarla.

Si ya no necesitas tu servidor de base de datos privado, puedes eliminarlo en tu panel.

  1. Navega hasta la página VPS. Verás tu servidor listado como sqlXXXXX.
  2. Haz clic en el botón Eliminar a la derecha. Se te pide que confirmes que deses que los datos de tu base de datos se migren a un servidor MySQL compartido.
  3. Haz clic en el botón Mover todo en sqlXXXXX a shared hosting y luego destruir sqlXXXXX.

Tus datos se migrarán al servidor compartido dentro de 24 horas.
