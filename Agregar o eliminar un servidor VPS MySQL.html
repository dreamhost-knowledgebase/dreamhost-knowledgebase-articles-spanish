Generalidades

Este artículo explica el cómo agregar un VPS MySQL a tu cuenta o eliminar uno que haya sido agregado.

Agregar un VPS MySQL

El proceso de agregar un VPS MySQL es diferente dependiendo de si ya has agregado un VPS a tu cuenta.

Si has agregado un VPS

Las siguientes instrucciones aplican si ya has agregado un VPS a tu cuenta.

  1. Navega a la página VPS.
  2. Haz clic en el botón Agregar Servidor+ y selecciona Servidor Virtual Privado MySQL.
  3. Selecciona un plan y luego haz clic en el botón Obtener este VPS para comprarlo. Una vez agregado, puede llevar algunas horas aprovisionar el nuevo servidor.

    Cuando te registras en un VPS MySQL, todas tus bases de datos se trasladan a él a la medianoche, hora del Pacífico. (El panel lo muestra como una fusión).

  4. Cuando la página del Carrito se abra, revisa el resumen de la orden y da clic al botón Enviar Orden.

Una vez aprovisionado, puedes administrarlo en la página del panel de VPS.

Si NO has agregado un VPS

Las siguientes indicaciones aplican si NO has agregado un VPS a tu cuenta. Antes de continuar, asegúrate de haber agregado una nueva base de datos si no existe ninguna.

  1. Navega a la página Bases de datos MySQL.
  2. Debajo de la sección Bases de Datos MySQL Actuales, haz clic en el enlace ¡Encender un VPS MySQL!.
  3. En la siguiente página, selecciona un plan de VPS MySQL.
  4. Haz clic en el botón Pagar Ahora.
  5. Selecciona una de las siguientes opciones:
    • Mover todos los usuarios y dominios actuales a este nuevo VPS.
    • También me gustaría cancelar mi plan actual Shared Unlimited.
    • No mover nada a este nuevo VPS.
  6. Haz clic en Enviar Orden para agregar el servidor a tu cuenta.

Eliminar un VPS MySQL

Si tienes un VPS MySQL con 3GB o más de datos y no tienes un Servidor Dedicado, no puede usar el botón Eliminar en el panel para eliminar tu MySQL VPS. Debes reducir el tamaño a menos de 3GB para cada base de datos MySQL VPS para poder eliminarla.

Si ya no necesitas tu servidor de base de datos privado, puedes eliminarlo en tu panel.

  1. Navega a la página VPS. Verás a tu servidor como sqlXXXXX.
  2. A la derecha de tu VPS MySQL, da clic al botón de 'tres puntos verticales' y selecciona Eliminar.
  3. Selecciona la casita si quieres transferir toda la información a otro servidor.
  4. Haz clic en el botón Eliminar VPS cuando termines.

Tus datos se migrarán al servidor compartido dentro de 24 horas.
