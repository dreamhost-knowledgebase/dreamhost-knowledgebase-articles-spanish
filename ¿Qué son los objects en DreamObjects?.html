Generalidades

Los objects son archivos que se suben en DreamObjects.

Usar Object Viewer para ver objects en un bucket

Para usar el Object Viewer:

  1. Navega a la página DreamObjects en tu panel de DreamHost.

  2. Haz clic en el botón Ver Objects bajo tu nombre de usuario.

    Luego de hacer clic en el botón, el Visor de Objects carga una lista de tus buckets.

  3. Haz clic en el nombre del bucket en el panel izquierdo. Esto lista tus carpetas y object dentro de ese bucket, y también muestra cuatro botones de control en la parte inferior.

    El Object viewer en el panel solo puede mostrar 5000 object a la vez.

    Haz clic en el botón LOAD MORE OBJECTS para mostrar los siguientes 5000 objects.

Los botónes de control en la esquina inferior izquierda te permiten:

  • Switch Bucket — Ver los diferentes contenidos de un bucket.
  • Reload Bucket — Para ver los cambios (obejtos añadidos o removidos) del bucket.
  • Reload Folder — Recarga la carpeta actual que estás viendo.
  • Upload Objects — Subir un nuevo Object al bucket.

Otras formas de ver un Object

También puedes utilizar aplicaciones de terceros para conectarte a DreamObjects y ver o subir objects.

Crear un Object

Puedes crear Objects usando una aplicación de tranferencia de archivos compatible.

Propiedades de los Objects

Haz clic en cualquier object de tu lista para ver las propiedades del object, que incluyen:

  • Nombre del Object.
  • Permisos — (PERMISSIONS) (Public o Private determinado por el uso de las credenciales).
  • Tamaño (SIZE).
  • Última modificación (LAST MODIFIED).

Descargar un Object usando Object Viewer

  1. Haz clic en el Object que deseas descargar.

  2. Haz clic en el botón Download Object en la caja Object Properties.

    Aparece un cuadro emergente que le permite descargar el object.

    • También puedes enviarle a cualquier persona la URL del object para que puedan descargarlo ellos mismos.

Eliminar un Object

  1. Haz clic en el object que deseas eliminar.

  2. Haz clic en el enlace Remove Object en la caja Object Properties.

  3. Haz clic en el botón Remove en el cuadro emergente para eliminar el object.
