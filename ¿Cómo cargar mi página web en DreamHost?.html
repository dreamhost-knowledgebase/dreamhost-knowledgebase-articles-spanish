Clientes a usar para cargar tu página

Cuando tu página esté terminada y lista para publicarse, podrás cargar tus archivos en DreamHost para colocarlos en tu cuenta.

Esto se puede hacer usando un cliente FTP para cargar tus archivos:

  • Cliente de FTP — FileZilla, Cyberduck, CuteFTP, etc., dependiendo de tu sistema operativo. Usar un cliente FTP totalmente funcional, tal como los anteriores, es la forma recomendada de conectarte a tu servidor web.
  • Administrador de archivos — Visita files.dreamhost.com para conectarte a tu sitio usando el administrador de archivos del panel. Esta es la opción disponible de cliente FTP en tu panel de DreamHost.

Existen muchos otros paquetes de software para cliente FTP. Por favor, visita el artículo Lista de clientes FTP para más información.

Cliente FTP

DreamHost recomienda el uso de un cliente FTP para cargar tus archivos a DreamHost. Puedes descargar y usar cualquier cliente FTP que elijas (vea el artículo Lista de clientes FTP para más información).

Usa los siguientes valores de inicio de sesión de cuenta FTP cuando se le soliciten:

Nombre del Host/Servidor

Puedes usar cualquiera de los siguientes:

  • La URL de la página (sin el https://www): por ejemplo, si tu página es https://www.example.com, entonces tu solo usarías example.com para iniciar sesión en tu cliente FTP.
  • URL del Servidor: en tu panel, dirigete a Administrar Cuenta.

En la página Administrar Cuenta verás una sección llamada Dominios Alojados. En dicha sección, verás tus dominios y a la derecha de cada uno está el servidor web en el cual está configurado. Así que, si tu estás en el servidor "flower", tu dirección de servidor sería flower.dreamhost.com.

Nombre de usuario/Inicio de sesión

Este es el nombre de usuario para iniciar sesión solamente en tu cuenta FTP. Por favor, ten en cuenta que el WebID de su Panel NO ES el nombre de usuario FTP. Podrás encontrar esta información en el correo que DreamHost envía cuando tu te registraste o cuando creaste la cuenta de usuario FTP. También podrás ver todos tus usuarios en Archivos. Para ver qué usuario está asignado a cuál dominio, dirígete a Administrar Dominios. A la derecha de cada dominio (en la columna de Alojamiento Web), podrás encontrar al usuario asignado a ese dominio específicamente.

Contraseña

Esta es la contraseña exclusivamente para iniciar sesión en tu cuenta FTP. Por favor, ten en cuenta que esta NO ES la misma contraseña que usas para tu Panel. Puedes restablecerla en Archivos. Espera unos 15 minutos para que se actualice en el sistema antes de activarse.

Puerto

Consulta la página Archivos del panel para ver si tu usuario es un usuario de FTP/FTP/SHELL. Da clic en el botón Show Info al lado del usuario para ver el tipo. Los usuarios SFTP y SHELL (las opciones predeterminadas) usa el puerto 22. Si tienes un usuario FTP antiguo, usa en vez el puerto 21.

Para garantizarte una conexión segura con tu servidor, verifica que está usando el puerto 22. Esto también se conoce como "conectarse mediante FTP". Lee el artículo Seguridad en FTP para más detalles.

Solución de problemas

Si has cargado tu página, y todo parece indicar que se ha transferido con normalidad, pero solamente puedes ver un "directorio-padre", entonces lo más probable es que hayas cargado tus archivos en la carpeta equivocada, o en la cuenta de usuario FTP incorrecta. Por favor, verifícalo e inténtalo de nuevo.
