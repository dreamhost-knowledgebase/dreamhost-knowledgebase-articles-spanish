Generalidades

La barra de herramientas que aparece en la parte superior de cada página dentro de Remixer te permite acceder rápidamente a varias funciones de diseño y configuración.

Usar la barra de herramientas para seleccionar una función Remixer

Haz clic en los siguientes botones para abrir varias páginas relacionadas con las funciones de Remixer:

  1. El logo de DreamHost – abre la página principal 'Select Theme'.
  2. My Sites – Usa una herramienta interactiva para crear rápidamente una plantilla para tu sitio web.
  3. Create – Lista de temas que puedes usar para crear tu sitio web.
  4. Themes – Lista de temas que puedes utilizar para crear tu sitio web.
  5. Media – abre una página donde puede almacenar varios activos del sitio web, como imágenes, videos, audio, etc.
  6. Account – Abre un menú desplegable para contactar con el soporte, buscar artículos de la base de conocimientos o cerrar sesión en Remixer.
  7. Publish (o View)– Esto solo se muestra al editar el sitio. Si su sitio aún no se ha publicado, aparecerá el botón 'Publish'. Al hacer clic aquí, el sitio vive en el dominio que asignaste dentro de la página 'Configuración' (consulta la sección Settings a continuación). Si ya has publicado el sitio, el botón View te permite ver la página actual.

My Sites

El botón My Sites muestra todas tus páginas guardadas anteriormente.

Puedes hacer las siguientes acciones en esta página:

  • Edit — Edita una página guardada.
  • Settings — Ajusta la configuración de tu sitio, como SEO, Google Analytics y nombre de dominio.
  • Publish — Hace que la página esté activa en el dominio que le asignaste en 'Settings'.

Edit

Ve el artículo 'Editar una página en Remixer' para más información.

Settings 

Al hacer clic en el botón Settings se abre una página de diálogo.

Puedes realizar las siguientes acciones en esta página:

  • SEO – Agrega detalles de SEO y Google Analytics para tu nuevo sitio web.
  • DOMAINS – Selecciona un dominio para publicar tu sitio Remixer.
  • EMAIL – Accede al correo si está incluido en tu tipo de plan.
  • DELETE THIS PAGE – Elimina la página actual.

Asegúrate de hacer clic en el botón Save settings cuando termines.

Create

Al hacer clic en el botón 'Create' se abre una herramienta interactiva que te ayudará a crear tu sitio web. Primero eliges qué tipo de sitio te gustaría:

  • Business (Negocios).
  • Professional (Profesional).
  • Creative (Creativo).

Luego pasa a elegir una paleta de colores, imágenes y fuentes. Esta es una forma muy rápida de crear una plantilla para tu sitio web.

Themes

La página Themes muestra todas las plantillas de temas para que pueda basar tu sitio web. Si no hiciste clic en el botón 'Create' de arriba para crear un sitio web, puedes elegir manualmente una plantilla tú mismo. Una vez elegida, puedes editarla como desees.

Media

El botón Media abre la página Manage Content donde puedes cargar varios activos para tu sitio:

Imágenes

  • Sube archivos desde tu computador. Puedes cargar archivos en Remixer haciendo clic en el cuadro o arrastrando y soltando archivos en el cuadro.
  • Importa imágenes de varios sitios de redes sociales.
  • Agrega imágenes libres de derechos con pixabay.
  • Importa imágenes y texto de un sitio web ingresando una URL.

Vídeo

  • Agrega enlaces directos al contenido de video usando una URL.
  • Importa videos de varios sitios de redes sociales.
  • Busca en YouTube.
  • Busca en Vimeo.

Audio

  • Agrega enlaces directos al contenido de video usando una URL.
  • Importa audio de varios sitios de redes sociales.

Markdown

  • Crea un archivo Markdown para usarlo en tu sitio web. El botón abre un editor donde pueds editar, obtener una vista previa, guardar y eliminar un archivo Markdown.

Content Groups

  • Crea o ve tus etiquetas de imágenes, videos y grupos de audio guardadas previamente.

Publish/View

El botón 'Publish' solo se muestra al editar tu sitio. Haz clic en el botón Publish cuando termines de editar y estés listo para enviar el sitio Remixer a tu dominio en vivo que has asignado en la sección Settings.

Aparece un botón animado llamado Publish que te muestra el progreso de carga de tu sitio web:
Cuando tu sitio web se publica en tu dominio, aparece un icono actualizado en la esquina superior derecha.

Da clic en el botón View! para abrir el sitio web en vivo.
