Lo siguiente describe cómo hacer cambios usando la página antigua Dominios Alojados en el panel. DreamHost recomienda usar la página Administrar Sitios Web en vez para hacer ajustes a tus Configuraciones de DNS.

Revisa el siguiente artículo para más información:

  • Configurar tu dominio como 'Solo DNS'

Generalidades

Este artículo explica cómo eliminar los registros DNS predeterminados de alojamiento de DreamHost en tu dominio.

Cuando tu dominio está como 'Dominio Alojado', este tiene varias direcciones IP de DreamHost (registros A) asignadas que controlan dónde está alojado el sitio. Si los nameservers de tu sitio apuntan a DreamHost y tu sitio está como "Dominio Alojado", este está alojado en los servidores de DreamHost.

Si los nameservers de tu sitio apuntan a DreamHost pero NO deseas alojar el sitio en DreamHost, debes eliminar el alojamiento al cual elimina estas direcciones IP. Configurar el sitio como 'Solo DNS' elimina el alojamiento y los registros A de tu sitio. Luego de esto, podrás agregar registros DNS personalizados para apuntar tu sitio a otra empresa de alojamiento.

Esto solo funciona con el nombre de dominio principal como example.com. Esto NO funciona con subdominios como blog.example.com. Para los subdominios, debes agregar registros DNS personalizados al dominio principal como se explica a continuación.

Agregar un nuevo dominio como 'Solo DNS'

  1. Abre la página de Dominios Alojados.
  2. Haz clic en el enlace Agregar alojamiento a un dominio / subdominio.
  3. Cuando se abra la página siguiente, desplázate hacia abajo hasta 'Solo DNS'.
  4. Ingresa el dominio que deseas establecer como 'Solo DNS' y haz clic en el botón ¡Alojar solamente DNS!.

Cambiar un dominio alojado a 'Solo DNS'

  1. Navega a la página de Administrar Dominios.
  2. A la derecha del dominio debajo de la columna Alojamiento Web , haz clic en el botón Eliminar.

    Si el dominio está configurado como Mirror, verás un enlace Desactivar espejo en su lugar.

    Si el dominio está configurado como Redirect, verás un enlace Desactivar redirección en su lugar.

Esto configura el dominio como 'Solo DNS' (si no tienes un certificado SSL habilitado en el dominio).

Si tienes un certificado SSL habilitado en el dominio, la página Administrar Dominios solo muestra 'dns' a la derecha de tu dominio cuando debería decir 'Solo DNS'. Esto se debe al certificado SSL activo.

Debes eliminar el certificado SSL del dominio para poder eliminar todos los registros DNS y configurar el dominio como 'Solo DNS'. Visita la siguiente sección para obtener instrucciones.

Eliminar alojamiento seguro

Si has agregado un certificado SSL a tu dominio, debes eliminarlo para que se eliminen tus registros DNS.

  1. Navega a la página de Dominios Alojados.
  2. A la derecha del dominio puedes ver HTTPS Seguro. Haz clic en este enlace.
  3. Te llevará a la página de alojamiento seguro para este dominio. En esa página, haz clic en el botón Eliminar certificado.
  4. Navega de vuelta a la página de Dominios Alojados.
  5. A la derecha de tu dominio ahora verás 'Solo DNS'.

¿Qué registros DNS se eliminan?

Cuando configuras un dominio como 'Solo DNS', eliminas los siguientes registros DNS A:

  • Registro A no www
  • Registro A www
  • Registro A ssh
  • Registro A ftp

A continuación hay algunas cosas a tener en cuenta con respecto a estos cambios:

  • No presiones el botón Eliminar para el dominio en la página Dominios Alojados. Esto elimina el dominio de la cuenta junto con los contenidos que anteriormente estaban alojados en este.
  • Configurar el dominio en 'Solo DNS' no elimina los archivos alojados (excepto en los planes de DreamPress) ni elimina ningún registro DNS personalizado que hayas configurado previamente. Puedes hacer clic en el botón Editar para establecer el dominio como ‘Completamente alojado’. Esto crea de nuevo los registros DNS para tu dominio.
  • Ten en cuenta que esto solo afecta tu servicio de alojamiento web y deja intactos otros servicios como el correo (suponiendo que ya tienes un plan de alojamiento activo en tu cuenta). Si has eliminado tu plan de alojamiento pero deseas continuar alojando tu correo con DreamHost, puedes comprar un Plan de correo pagado.

Además, estos cambios solo son posibles para el dominio principal: los subdominios no se pueden establecer como Solo DNS.

Si este es tu único dominio en la cuenta y lo estás configurando para Solo DNS, NO necesitas un plan de alojamiento. Un plan de alojamiento solo es necesario si deseas alojar un dominio y su contenido en los servidores DreamHost.

¿Qué hacer después de eliminar los registros DNS?

Una vez que el dominio se establece en 'Solo DNS', puedes agregar registros DNS personalizados para apuntar tu sitio a donde quieras.
