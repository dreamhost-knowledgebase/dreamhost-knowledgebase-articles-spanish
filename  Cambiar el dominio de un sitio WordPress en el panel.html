Generalidades

Este artículo explica cómo cambiar el nombre de tu sitio WordPress a un nombre de dominio diferente. Estas instrucciones se pueden utilizar para un nombre de dominio normal como example.com o un dominio temporal como example.dreamhosters.com.

El contenido de tu sitio web no se modificará durante este proceso y el sitio continuará funcionando normalmente después de este cambio.

Puedes usar el cambiador de nombre de dominio en la página de Sitios Web para cambiar cualquier dominio temporal, no solo los subdominios de dreamhosters.com.

Prerrequisitos

Asegúrate de lo siguiente antes de continuar:

  • Asegúrate de que tienes una instalación de WordPress de DreamHost — La función de cambio de nombre de sitio web solo funciona con las instalaciones de WordPress de DreamHost. Confirma que el nombre del dominio desde el que estás cambiando sea una instalación de WordPress de DreamHost. Si no es así, puedes mover el sitio manualmente o comunícarte con el soporte para convertirlo.
  • Asegúrate de que tu instalación de WordPress de DreamHost no esté en un subdirectorio — Las instalaciones de WordPress de DreamHost en subdirectorios, como example.com/blog, no funcionará con el renombrador de sitios web. Aún puedes mover el sitio manualmente o contactar a soporte para mover la instalación al directorio root o raíz del sitio.
  • Eliminar los DNS personalizados — Asegúrate de que no hayan registros DNS personalizados A o CNAME en el dominio que estás cambiando a (si existe alguno que apunte a un servidor web diferente). Si los hay, elimina los registros personalizados. Otros registros DNS personalizados están bien para dejarlos en su lugar.
 

Paso 1 — Cambiar tu dominio

Para cambiar el dominio de WordPress:

  1. Navega a la página Administrar Sitios Web.
  2. Da clic en el botón Administrar para abrir la página Configuraciones de Dominios, la cual te permite ajustar múltiples configuraciones de tu sitio.
    • Si estás en la Vista de Malla, da clic en el botón Administrar en la parte inferior de la caja del sitio.
    • Si estás en la Vista de Lista, da clic en el botón Administrar a la derecha de tu dominio.
  3. Desplázate hacia abajo en la sección Nombre de dominio de la página.
  4. Haz clic en el enlace Cambiar.
  5. Selecciona un dominio. Tienes dos opciones para cambiar el nombre:

    Escoger un Dominio Existente

    Aparece una casilla de verificación que te solicita que confirmes que se eliminarán las copias de seguridad anteriores del dominio anterior.
    Esta opción te permite elegir entre un dominio que ya has agregado a tu panel.

    Ingresar un Dominio Nuevo 

    Puede aparecer una casilla de verificación pidiéndote que confirmes que el dominio no está registrado con DreamHost y que deberás actualizar el DNS.
    Esta opción te permite ingresar un nombre de dominio que ya posees o un dominio temporal. El panel luego lo agregará por tí.
  6. Haz clic en el botón Cambiar Dominio.
    Toma 4-6 para que el DNS se actualice para tu nuevo dominio en DreamHost.

Paso 2 — Esperar a que tu DNS DreamHost se actualice

Como se indicó anteriormente, el DNS puede tardar entre 4 y 6 horas en actualizarse en DreamHost. Si apuntas tu DNS a tu host actual a DreamHost antes de que el DNS se haya actualizado en los servidores de DreamHost, el sitio no se conectará.

Se recomienda que esperes de 4 a 6 horas antes de actualizar tu DNS en tu alojamiento actual.

Paso 3 — Actualizar tu DNS en tu hosting actual

Finalmente, inicia sesión en tu compañía de registro y apunta tu DNS a DreamHost. Luego de que el propague, tu sitio estará alojado con DreamHost.
