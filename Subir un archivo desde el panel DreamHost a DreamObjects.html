Generalidades

Existe una variedad de aplicaciones de terceros que te permiten subir y administrar tus datos almacenados en DreamObjects. El panel DreamHost también proporciona un 'DreamObjects Viewer' integrado que proporciona una forma de cargar datos en DreamObjects usando tu navegador.

Este artículo explica cómo subir datos en tu cuenta de DreamObjects. Si, en cambio, buscas instrucciones sobre cómo subir tu sitio web en tu servidor web DreamHost, visita la siguiente página:

  • ¿Cómo subir mi sitio a DreamHost?

Tamaño de subida máxima

Las subidas desde tu navegador tienen un tamaño limitado a 64MB. Para subir archivos más grandes, debes usar un cliente S3.

  • Clientes de DreamObjects

Subir una carpeta (directorio)

No es posible subir una carpeta (directorio) de tu computador a DreamObjects usando a herramienta de panel. Solo se pueden subir archivos con esta herramienta.

Si necesitas subir una carpeta (directorio) que incluye archivos, debes usar un cliente compatible con S3.

  • Clientes de DreamObjects

Subier un archivo

El siguiente ejemplo muestra la carga de un archivo .rtf, pero puedes subir cualquier tipo de archivo que desees.

  1. Navega a la página DreamObjects.

  2. Ubica el usuario que contiene el bucket con el que deseas trabajar, y luego haz clic en el botón Ver objetos.

    Los buckets disponibles aparecen en la siguiente ventana:

  3. Selecciona el bucket al que deseas subir objetos.

    Verás los objetos que se encuentran actualmente en el bucket en la siguiente ventana. Aparece un botón Subir objetos en la parte inferior del explorador de Objetos.

    • El navegador de objetos ya está listo para empezar a subir.

  4. Arrastra tus archivos al navegador de Objetos o haz clic en Selecciona archivos para seleccionar archivos usando el navegador de archivos.

  5. Cuando hayas terminado de subir, haz clic en el botón Hecho en la parte inferior derecha.

    El archivo ahora aparece en el Object viewer

  6. Haz clic en el botón Cambiar Bucket en la parte inferior para regresar a la página principal de navegador de Objetos.

  7. Para cerrar el Object viewer de DreamObjects cuando hayas finalizado, haz clic en la X gris en la esquina superior derecha.
