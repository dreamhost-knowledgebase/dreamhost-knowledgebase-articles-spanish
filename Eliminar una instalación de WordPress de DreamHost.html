Generalidades

Este artículo explica cómo eliminar WordPress si fue instalado usando el instalador de WordPress de DreamHost en el panel.

Estas instrucciones eliminarán el contenido de tu sitio web. Revisa el siguiente artículo en vez si solamente quieres desactivar las actualizaciones automáticas de WordPress:

  • Actualizaciones automáticas para las instalaciones de WordPress de DreamHost

Sitios web de DreamPress

Si tu sitio WordPress está configurado en un plan de alojamiento de DreamPress, los pasos para eliminarlo o cambiarlo difieren de los que están en este artículo. En vez, revisa los siguientes artículos para más información:

  • Eliminar DreamPress
  • Cambiar tu plan DreamPress a otro plan de alojamiento

¿Qué se elimina del servidor?

Se pueden eliminar varios archivos dependiendo de dónde esté instalado WordPress dentro de tu sitio web. Por ejemplo, se podría instalar en:

  • El directorio web Principal (example.com)
  • En un Subdirectorio (example.com/blog)

Asegúrate de leer las siguientes secciones antes de eliminar tu instalación de WordPress.

Confirma dónde está instalado WordPress

Para confirmar dónde está instalado WordPress en tu sitio web:

  1. Navega a la página Instalar WordPress.
  2. Da clic en Gestionar mis Aplicaciones.
  3. Da clic a tu sitio web.

El directorio principal

Si está ubicado en el directorio web principal, verás una barra inclinada junto al icono de WordPress.

El directorio principal del sitio web es el directorio desde del que tu sitio web carga archivos primero. En general, este directorio es el nombre de tu sitio. Por ejemplo, si tu sitio se llama example.com, el directorio web sería /home/username/example.com también.

Si aquí es donde instaló tu instalación de WordPress de DreamHost, ten mucho cuidado cuando la elimines en el panel. Si eliminas la instalación de WordPress de DreamHost almacenada en tu directorio web principal, TODOS LOS ARCHIVOS en el directorio web y todos los subdirectorios se eliminan. Esto quiere decir que si tienes subdirectorios como example.com/images o example.com/blog, también se eliminarán.

Asegúrate de usar esta opción solo si quieres que todo se elimine de tu sitio web incluyendo todos los demás archivos en este directorio web. Si necesitas alguno de esos archivos o subdirectorios, asegúrate de hacer una copia de seguridad antes de eliminar la instalación de WordPress de DreamHost.

  • Respaldar la información de un sitio web manualmente
  • Respaldar una cuenta en el panel

Un subdirectorio

Si WordPress está ubicado en un subdirectorio, ser mostrará una barra inclinada seguido del nombre del subdirectorio.

Eliminar una instalación en un subdirectorio solo eliminará los archivos dentro de ese subdirectorio específico. Por ejemplo, si tu sitio estaba en example.com/blog, solo los archivos dentro de ese subdirectorio /blog son eliminados, sin embargo, el directorio vacío /blog se mantendrá. No se eliminan archivos por encima de este directorio, por lo que cualquier cosa dentro del directorio /example.com no se tocará.

La diferencia entre Eliminar y Quitar

Quitar de la Lista

Dando clic al botón Quitar de la Lista desactiva una instalación de WordPress de DreamHost para que DreamHost ya no la administre. Si deseas continuar usando esta instalación de WordPress, serás responsable de mantenerla actualizada.

Esta opción lo quita de la lista, pero NO elimina ningún archivo.

Eliminar todos los archivos

Dando clic al botón Eliminar todos los Archivos elimina todo en el directorio de instalación de WordPress y lo elimina del panel. Solo usa esta opción si quieres que TODO el contenido dentro de tu instalación de WordPress se elimine como se menciona arriba.

Asegúrate de respaldar de los datos que no deseas que se eliminen antes de hacer clic en este botón.

  • Respaldar la información de un sitio web manualmente
  • Respaldar una cuenta en el panel

Eliminar una instalación de WordPress

  1. Navega a la página Instalar WordPress.
  2. Da clic a Gestionar mis Aplicaciones.
  3. Da clic a tu sitio web.
  4. Haz clic en el botón Eliminar todos los archivos.
    Aparece un mensaje de advertencia después de hacer clic en Eliminar todos los archivos:
  5. Da clic en Aceptar para confirmar.

Eliminar la base de datos

Cuando das clic en Eliminar todos los archivos, solo se eliminan los archivos del servidor web. Tu base de datos de WordPress permanece intacta en el panel. Para eliminar la base de datos:

  1. Navega a la página Bases de Datos MySQL.
  2. A la derecha de tu base de datos, da clic al botón Eliminar DB.
