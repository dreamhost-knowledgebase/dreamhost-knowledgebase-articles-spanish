Generalidades

A continuación se explica cómo encontrar las credenciales de inicio de sesión de tu base de datos, que te permiten iniciar sesión en ty base de datos en el panel usando phpMyAdmin o cualquier software que prefieras.

Por favor ten en cuenta que cualquier configuración incorrecta en tu base de datos podría provocar que tu sitio no funcione correctamente. Inicia sesión en tu base de datos solo si sabes exactamente qué necesitas ajustar.

Revisar las configuraciones de tu DNS

Antes de continuar, revisa este artículo para revisar a dónde apunta tu DNS, ya que esto afectará el poder iniciar sesión.

Si tu DNS está apuntando a DreamHost

Si tus nameservers están alojados en DreamHost, puedes continuar con los pasos de la siguiente sección.

Nuevas instalaciones de WordPress

Si acabas de instalar WordPress con el instalador de WordPress de DreamHost, deberás esperar unas horas para que se actualice el nuevo subdominio mysql. Una vez que el DNS se haya propagado en línea, tu sitio se conectará a tu base de datos.

Si tu DNS NO está apuntando a DreamHost

Si tus nameservers NO apuntan a DreamHost, debes crear un registro A (en tu compañía de alojamiento actual) para tu hostname mysql.example.com que apunte a tu registro A MySQL de DreamHost. Solo después de que este registro A se actualice en línea, funcionará el enlace phpMyAdmin en el panel.

Revisa este artículo para obtener instrucciones sobre cómo ubicar el registro A MySQL de DreamHost. Luego, puedes comunicarte con tu compañía de alojamiento actual para agregarlo a su sistema.

Solo necesitas crear el registro A en tu subdominio mysql, como mysql.example.com.

Como alternativa, puedes iniciar sesión utilizando los hostnames de los centros de datos de la Costa Este y Oeste en el Paso n° 4 a continuación.

Iniciar sesión en tu base de datos

Usa los siguientes pasos para encontrar las credenciales de tu base de datos para iniciar sesión con phpMyAdmin. También puedes utilizar estas credenciales para iniciar sesión a través de SSH u otro software.

 

Encontrar el nombre de tu base de datos

Visita la página Bases de Datos MySQL y desplázate hacia abajo hasta la sección titulada Bases de Datos en este servidor. A continuación verás una lista de tus bases de datos.

Si tienes varias bases de datos, deberás identificar la que corresponde a tu sitio web. Esto generalmente se puede encontrar en el archivo de configuración del sitio, pero cada aplicación lo coloca en una ubicación diferente. Revisa la base de conocimientos de tu aplicación para conocer las ubicaciones de los archivos de configuración y anota el nombre de la base de datos.

Encontrar tu usuario

El nombre de usuario específico vinculado a la base de datos aparece a la derecha de tus bases de datos.

Encontrar tu contraseña

Da clic en el nombre de usuario en la columna ACCESO DE USUARIOS. En la página siguiente, haz clic en el botón Mostrar para ver la contraseña del usuario.

Encontrar tu hostname

Navega de nuevo a la página Bases de Datos MySQL. Cerca de la parte superior, verás una sección llamada HOSTNAME. Busca el hostname que corresponde a tu sitio web. Lo más probable es que contenga el nombre del sitio web.

A la derecha del hostname hay un enlace titulado phpMyAdmin. Da clic en este enlace para abrir la página de inicio de sesión de phpMyAdmin.

También puedes utilizar el hostname oficial de phpMyAdmin de tu centro de datos. Visita la página Centro de Datos y luego desplázate hacia abajo para ubicar el centro de datos donde se encuentran tus servicios MySQL. Puede ser Costa Oeste o Costa Este. Luego puedes utilizar los siguientes hostnames para conectarse según la ubicación de tu servidor MySQL:

  • Costa Oeste:  https://west1-phpmyadmin.dreamhost.com
  • Costa Este:  https://east1-phpmyadmin.dreamhost.com

Iniciar sesión en phpMyAdmin

En el indicador de phpMyAdmin, ingresa tu hostname, nombre de usuario, y contraseña y luego da clic en Iniciar sesión.

phpMyAdmin requiere que el dominio de tu hostname de MySQL sea Completamente Alojado y use el DNS de DreamHost. Puede hacer esto de una de dos maneras:

  • Apunta tus nameservers a DreamHost, o
  • Apunta tu registro A a tu dirección IP de DreamHost. Para obtener detalles sobre cómo recuperar esa dirección IP, revisa el artículo sobre los registros DNS de DreamHost.

Si ninguno de tus hostnames funciona, revisa este artículo para obtener más información sobre cómo acceder a la base de datos antes de que se actualice su DNS.
