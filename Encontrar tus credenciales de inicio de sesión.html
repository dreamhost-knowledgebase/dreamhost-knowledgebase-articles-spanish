Generalidades

Las credenciales de tu base de datos te permiten iniciar sesión por medio del panel a tu base de datos usando. phpMyAdmin es una interfaz gráfica donde puedes navegar a través de tu base de datos y realizar cambios.

Estas mismas credenciales también se pueden usar para conectarse a la base de datos usando el software que desees.

Asegúrate de iniciar sesión en tu base de datos únicamente si sabes exactamente lo que necesitas ajustar. Cualquier configuración incorrecta en tu base de datos podría hacer que tu sitio no funcione correctamente.

Una observación sobre la configuración de DNS

Si tu DNS está alojado en DreamHost

Si tus nameservers están alojados en DreamHost, sigue los pasos a continuación.

Si acabas de instalar WordPress usando el instalador de WordPress de DreamHost

Si acabas de instalar WordPress usando el instalador de WordPress de DreamHost (y tus nameservers apuntan a DreamHost), tendrás que esperar algunas horas para que se actualice el nuevo subdominio mysql. Una vez que el DNS se haya propagado en línea, tu sitio se conectará a tu base de datos.

Si tu DNS NO está alojado en DreamHost

Si tus nameservers NO están alojados en DreamHost, debes crear un registro A (en tu proveedor de alojamiento actual) para que hostname mysql.example.com. Esto debe apuntar al registro mysql A de DreamHost. Solamente luego de que el registro A actualice en línea, el enlace phpMyAdmin funcionará en tu panel.

Revisa el siguiente artículo para ver instrucciones sobre cómo ubicar el registro mysql A en DreamHost. Luego puedes contactar a tu compañía de alojamiento actual para agregarlo en su sistema.

  • ¿A cuál registro A debo apuntar mi sitio?

Solo necesitas crear el registro 'A' para el subdominio mysql como en mysql.example.com.

Como alternativa, puedes ingresar usando los hostnames de los centros de datos de la costa Este y Oeste como se muestra en el Paso #4 debajo.

Paso 1 — Encontrar el nombre de tu base de datos

Visita la página Bases de datos MySQL y desliza hacia abajo hasta la sección titulada Bases de datos en este servidor. A continuación verás una lista de tus bases de datos.

Si tienes múltiples bases de datos, deberás identificar cuál de ellas corresponde a tu sitio. Usualmente esto lo encuentras en el archivo de configuración de tu sitio, pero cada aplicación lo puede poner en una ubicación diferente. Consulta el centro de ayuda de tu aplicación para conocer la ubicación de archivos de configuración. Anota el nombre de la base de datos.

Paso 2 — Encontrar tu usuario

El usuario específico ligado a la base de datos se encuentra a la derecha de cada una de las bases de datos bajo la columna 'Usuarios de acceso'. Anótalos.

Paso 3 — Encontrar tu contraseña

Da clic en el usuario debajo de la columna ACCESO DE USUARIOS. En la siguiente página, desliza hacia abajo y haz clic en el botón Mostrar para ver la contraseña de dicho usuario. Anótalo también.

Paso 4 — Encontrar el hostname

Navega de nuevo a la página Bases de Datos MySQL. Hacia la mitad de la página verás una sección titulada HOSTNAME. Busca el hostname que corresponde a tu sitio. Lo más probable es que el nombre de tu sitio esté incluido en dicho hostname.

A la derecha del hostname verás un enlace que dice phpMyAdmin. Haz clic en dicho enlace para acceder a la base de datos a través de phpMyAdmin.

También puedes usar el hostname oficial de phpMyAdmin de tu Centro de Datos. Primero, revisa la página Centro de Datos desde el panel. Desliza hacia abajo para ubicar el centro de datos donde tus servicios de MySQL están localizados. Puede ser costa Este o costa Oeste. Puedes entonces usar los hostnames que se muestran a continuación dependiendo de la ubicación de tu servidor MySQL.

  • Costa Oeste:  https://west1-phpmyadmin.dreamhost.com
  • Costa Este:  https://east1-phpmyadmin.dreamhost.com

Iniciar sesión en phpMyAdmin

En la página de inicio de phpMyAdmin, ingresa tu hostname, usuario, contraseña, y haz clic en el botón Iniciar sesión

phpMyAdmin requiere que el dominio usado para tu hostname de MYSQL esté Completamente Alojado y utilizando los DNS de DreamHost. Puedes hacer esto de dos maneras:

  • Apunta tus nameservers a DreamHost o
  • Apunta tu registro 'A' hacia la IP de DreamHost. Revisa el artículo registros DNS de DreamHost para obtener detalles de cómo encontrar tu registro IP de DreamHost.

Si ninguno de tus hostnames está funcionando, visita el siguiente artículo sobre cómo acceder a tu base de datos antes de actualizar los DNS:

  • Acceder a una base de datos antes de apuntar el DNS a DreamHost
