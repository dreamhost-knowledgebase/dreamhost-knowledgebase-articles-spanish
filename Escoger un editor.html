Generalidades

WordPress ofrece varios editores diferentes que puede elegir para editar su sitio web. Si estás usando BoldGrid para crear tu sitio, se recomienda el Post and Page Builder. Este artículo te brinda una descripción general rápida de las siguientes opciones.

  • Post and Page Builder
  • Wordpress Editor 
  • Classic editor

No debes usar varios creadores de sitios web, como BoldGrid y Elementor, en el mismo sitio. Si tu sitio fue construido con un creador de sitios web de WordPress y deseas cambiar a otro, primero debes usar un sitio Staging en lugar de instalar un nuevo constructor en el sitio en vivo.

Editar una página

  1. Inicia sesión en tu panel de WordPress.
  2. Haz clic en la pestaña Inspirations en el panel izquierdo.
  3. Bajo Site Content, haz clic en el ícono de lápiz a la derecha de la página que deseas editar.
  4. Haz clic en Okay, Let's Go! cuando se te presente la opción de configurar ajustes.
  5. Selecciona el editor que deseas usar. Tus opciones son:
    • Post and Page Builder — Este es el editor de BoldGrid (recomendado).
    • Wordpress Editor — Este es el editor de páginas estándar de WordPress que se encuentra en 'Pages > All Pages'.
    • Classic Editor — Similar al editor de WordPress, pero con más opciones mostradas.
    • Default — Por defecto usa Post and Page Builder.
  6. Haz clic en OKAY, GOT IT! para guardar tu configuración.

Elegir un editor

Post and Page Builder

Como se mencionó anteriormente, el Post and Page Builder es el editor predeterminado de BoldGrid y se recomienda debido a su simplicidad y fácil acceso a las herramientas. Puedes cambiar a un editor diferente haciendo clic en el menú desplegable titulado Post and Page Builder ubicado en la esquina superior derecha.

WordPress editor

La mayoría de las opciones están ocultas, pero se puedes acceder haciendo clic en los diferentes íconos. Se pueden agregar bloques en la parte superior izquierda, más opciones están a la derecha. Para cambiar a un editor diferente, haz clic en el icono de 3 puntos en la esquina superior derecha. En el menú, en Plugins, puedes cambiar a un editor diferente.

Classic editor

Similar al editor de WordPress, pero con más opciones mostradas por defecto. Haz clic en el menú desplegable Classic Editor en la parte superior para cambiar de editor.
