Generalidades

Comprar y registrar un dominio es un paso importante para configurar tu cuenta en DreamHost. Para que tu sitio web sea visible en Internet, debes configurar un plan de alojamiento para tu dominio.

Este artículo te guía a través de los sencillos pasos para agregar un plan compartido a tu dominio mediante el panel de DreamHost. También puedes agregar WordPress, correo y un certificado a tu dominio, que igualmente se describen a continuación.

Puedes agregar todos tus sitios web en un solo plan de Shared hosting.

Agregar un plan de Shared Hosting

  1. Navega a la página Administrar Cuenta.
  2. En la sección titulada Planes Activos, haz clic en el enlace Agregar un Plan de Alojamiento.
    Selecciona el plan Shared Starter o Shared Unlimited. Aparecen varias opciones de pago que te permiten elegir la frecuencia con la que se te facturará tu plan de alojamiento:
  3. Selecciona un plan de facturación y haz clic en el botón Agregar plan.
  4. Cuando se agrega tu plan de alojamiento, continúa agregando tu dominio al plan de alojamiento. Visita el siguiente artículo para obtener instrucciones sobre cómo agregar tu sitio al plan de alojamiento:
    • ¿Cómo agregar un dominio?

Agregar WordPress con el Instalador de Un-Clic

DreamHost ofrece una instalación con un clic de WordPress, que es una forma sencilla de instalar WordPress en tu sitio con unos pocos clics. Visita el siguiente artículo para obtener instrucciones:

  • Instalar WordPress usando el Instalador de Un Clic

Crear una dirección de correo

Después de agregar un plan de alojamiento, puedes agregar direcciones de correo a tu dominio. Visita el siguiente artículo para obtener instrucciones sobre cómo agregar un correo.

  • Guía de inicio rápido de correos electrónicos 

Instalar un certificado SSL con Let's Encrypt

Tener un sitio web seguro es fundamental hoy en día, y puedes lograrlo agregando un certificado SSL a tu dominio. Un certificado SSL ayuda a garantizar que los datos de tus visitantes permanezcan seguros e incluso ayuda potencialmente con en el posicionamiento en motores de búsqueda. DreamHost ofrece varios tipos de certificados SSL. Visita los siguientes artículos para obtener más detalles:
