Paso 1 — Configurar el alojamiento y un dominio

El primer paso es crear una cuenta DreamHost con un plan de alojamiento. También deberás ser propietario del registro de dominio (nombre de tu dominio).

1: Elige un plan de alojamiento y regístrate para obtener una cuenta

Antes de comprar una cuenta, primero debes elegir un plan de alojamiento. El plan de alojamiento contiene múltiples servicios, pero puede considerarse como el servidor al que se agregará tu sitio web. Agregar tu sitio al servidor le permite aparecer en Internet.

DreamHost ofrece varias opciones de planes de alojamiento diferentes según las necesidades de tu sitio web. El siguiente artículo enumera las diferentes opciones de planes disponibles en DreamHost.

  • ¿Qué tipos de planes de alojamiento están disponibles?

Para alojar un sitio web en línea, debes elegir uno de los siguientes planes de alojamiento:

  • Shared Starter o Shared Unlimited.
  • VPS (servidor privado).
  • Servidor Dedicado.
  • DreamPress (Alojamiento administrado de WordPress).

Una vez que hayas decidido el plan de alojamiento, puedes comprar el plan y abrir una cuenta de DreamHost:

  • Como inscribirte para una cuenta de DreamHost

2: Comprar un registro de dominio

Comprar un registro de dominio incluye poseer un nombre de dominio, como example.com. 

No puedes alojar (mostrar) tu dominio en línea hasta que seas propietario del registro.

Tu puedes comprar un nuevo registro de dominio o transferir a DreamHost un registro existente durante el proceso de inscripción. También puedes comprar un registro de dominio más adelante si lo deseas.

  • Cómo comprar un registro de dominio

Si ya has comprado el registro de dominio en una empresa diferente y no deseas transferirlo a DreamHost, puedes simplemente apuntar el DNS a DreamHost. Esta opción te permite mantener el registro en una compañía diferente mientras aloja el dominio en línea en los servidores de DreamHost.

3: Agrega el dominio a tu plan de alojamiento

Si compraste un registro de dominio, el nuevo dominio también debería haberse agregado al plan de alojamiento para que puedas verlo en línea. De lo contrario, siempre puedes agregar alojamiento al dominio usando el siguiente artículo:

  • ¿Cómo agregar un dominio?

Resumen

En este punto, has creado una cuenta DreamHost en la que puede iniciar sesión en panel.dreamhost.com. Dentro de esta cuenta, tienes un registro de dominio que se ha agregado a tu plan de alojamiento.

Paso 2 — Construir tu sitio web

Ahora que tu sitio está alojado en tu servidor DreamHost, puede comenzar a construirlo.

Hay muchas opciones diferentes disponibles para construir un sitio web. Algunas opciones (como un CMS) son más fáciles y te permiten crear un sitio sin necesitar de codificar nada. Otras opciones son más difíciles pero permiten una mayor personalización de tu sitio.

Opción 1 — Uso de un CMS (opción más fácil: no se requiere codificación)

La opción más fácil para construir un sitio es usar un CMS (Sistema de gestión de contenido). Este es un software preconstruido que instalas en tu sitio. Este software crea un sitio web totalmente funcional sin necesidad de codificación. Una vez instalado, puedes iniciar sesión en el sitio y comenzar a personalizarlo de la manera que desees.

Tu construyes y personalizas un sitio creado usando un CMS iniciando sesión en el backend del sitio web. Aquí es donde cargarás imágenes, configurarás la apariencia y agregarás texto.

WordPress

La opción de CMS más popular es WordPress. DreamHost ofrece esto como una Instalación de Un-Clic o como un plan de alojamiento actualizado especializado llamado DreamPress.

Instalación de Un-Click de WordPress

La instalación de Un-Clic instala inmediatamente WordPress en tu sitio:

  • Instalar WordPress usando el Instalador de Un-Clic

Se te envía un correo con instrucciones sobre cómo completar la instalación. Luego puedes iniciar sesión en el sitio en example.com/wp-login.php para comenzar a agregar tu contenido.

DreamPress

DreamPress es un plan de alojamiento específico que ofrece DreamHost. Es un paso adelante de alojar un sitio de WordPress en un plan de alojamiento compartido o VPS:

  • Generalidades DreamPress

Cuando compras un plan DreamPress, WordPress se instala automáticamente para ti. Este plan de DreamPres está específicamente optimizado para WordPress por simplicidad y rendimiento.

Opción 2 — Creación de un sitio web personalizado (Difícil, pero ofrece más personalización)

También puedes construir manualmente tu sitio web en la computador de tu hogar y subirlo a tu servidor DreamHost. Esto requeriría que al menos tengas un nivel básico de conocimiento de programación de sitios web. A continuación hay algunos enlaces para ayudarlo a construir un sitio web:

  • W3Schools — Tutoriales gratuitos sobre cómo construir un sitio web.
  • codingheroes.io — Muchos recursos gratuitos para ayudar a aprender a codificar un sitio web.
  • htmldog.com — Muchos recursos gratuitos para ayudar a aprender a codificar un sitio web.

Paso 4 — Optimizar tu sitio

Después de que tu sitio web esté configurado en el servidor y se muestre en línea, debes dedicar tiempo a optimizarlo. Optimizar un sitio significa hacer ajustes para que el sitio funcione más rápido y aparezca correctamente en diferentes dispositivos. También puedes incluir la realización de mejoras de diseño para mejorar la experiencia del usuario (UX).

Paso 4 — Atraer visitantes

Una vez que tu sitio esté en funcionamiento, debes publicitarlo para atraer visitantes. Permitir que tu sitio sea indexado por los motores de búsqueda ayudará a mostrar tu sitio en los resultados de búsqueda.

Enviar tu sitio a motores de búsqueda

Visita los principales motores de búsqueda y envía tu URL. La mayoría de estos sitios te permiten crear una cuenta donde puedes enviar tu sitio a su base de datos. Por ejemplo:

  • Google — Aparecer en Google
  • Bing — Herramientas Webmaster

Search Engine Optimization (SEO)

Hacer pequeños ajustes en tu sitio ayuda a los motores de búsqueda a rastrear y clasificar tu contenido. Esto a su vez ayuda a impulsar tu sitio hacia arriba en los resultados de búsqueda:
