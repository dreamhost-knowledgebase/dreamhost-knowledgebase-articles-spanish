Requisitos previos

Necesitarás tener el registro del dominio (la URL de tu sitio web) para que tu sitio web aparezca en línea.

  • Si ya lo compraste — Puedes transferirlo a Dreamhost al registrarte para obtener una cuenta. Alternativamente, también puedes transferirlo en cualquier momento en el futuro.
  • Si no lo has comprado aún — Puedes comprarlo durante el registro.

Cómo construir un sitio web

Los siguientes pasos te guiarán a través del proceso de creación de un sitio web.

 

Configurar el alojamiento y un dominio

El primer paso es crear una cuenta DE DreamHost con un plan de alojamiento.

1: Escoge un plan de alojamiento — Primero debes decidir qué tipo de plan de alojamiento te gustaría comprar. Este será el servidor que utilizará tu sitio web para aparecer en Internet. Algunos ejemplos de planes de alojamiento incluyen alojamiento Compartido, VPS, Dedicado y DreamPress.

2: Comprar la cuenta — Una vez que hayas decidido el plan de alojamiento, puedes registrarte para obtener una cuenta en DreamHost.

Comprar un registro de dominio

El registro de un dominio es la propiedad de un nombre de dominio, como example.com. No puedes alojar (mostrar) tu sitio web en línea hasta que seas el propietario del registro. Las siguientes opciones están disponibles:

  • Durante el proceso de registro, puedes comprar un nuevo registro de dominio o transferir un registro de dominio existente a DreamHost.
  • Si ya ha adquirido el registro de dominio en otra compañía y no desea transferirlo a DreamHost, simplemente puedes apuntar el DNS a DreamHost. Esta opción te permite mantener el registro en una compañía diferente mientras aloja el dominio en línea en los servidores de DreamHost.

3: Agrega el dominio a tu plan de alojamiento — Revisa el artículo Agregar un sitio web y alojamiento para obtener instrucciones sobre cómo añadir alojamiento a tu dominio para que aparezca en línea.

Construir tu sitio

Ahora que tu sitio está alojado en el servidor de DreamHost, puedes comenzar a crearlo.

Hay muchas opciones diferentes disponibles para crear un sitio web. Algunas opciones (como un CMS) son más sencillas y te permiten crear un sitio sin necesidad de codificar nada. Otras opciones son más difíciles pero permiten una mayor personalización del sitio.

Opción 1  — Usar un CMS (La opción más fácil: No se requiere código)

Un Sistema de Gstión de Contenido (CMS) es un software prediseñado que se instala en el sitio. Este software crea un sitio web completamente funcional sin necesidad de codificación.

WordPress — La opción de CMS más popular es WordPress. DreamHost lo ofrece como instalación de WordPress de DreamHost o como un plan de alojamiento especializado y mejorado llamado DreamPress (optimizado para la simplicidad y el rendimiento).

Opción 2  — Creación de un sitio web personalizado (difícil, pero ofrece más personalización)

También puedes crear manualmente tu sitio web en tu computador personal y subirlo a tu servidor de DreamHost. Esto requiere que tengas conocimientos básicos de programación de sitios web. A continuación, se incluyen algunos enlaces que te ayudarán a crear un sitio web:

  • W3Schools — Tutoriales gratuitos sobre cómo crear un sitio web.
  • codingheroes.io — Varios recursos gratuitos para ayudarte a aprender a codificar un sitio web.
  • htmldog.com — Varios recursos gratuitos para ayudarte a aprender a codificar un sitio web.

Crear una base de datos MySQL — Si tu sitio web personalizado requiere una base de datos, deberás crear una manualmente en el panel y cargar tu base de datos existente en ella.

Subir tu sitio al servidor — Una vez que hayas desarrollado tu sitio en st computador de hogar, carga el contenido a tu servidor de DreamHost utilizando un cliente FTP.

Atraer visitantes

Una vez que tu sitio esté en funcionamiento, debes publicitarlo para atraer visitantes.

Enviar tu sitio a los motores de búsqueda

Permitir que los motores de búsqueda indexen tu sitio ayuda a que el mismo aparezca en los resultados de búsqueda. Visita los principales motores de búsqueda y envíe tu URL. La mayoría de estos sitios te permiten crear una cuenta en la que puedes enviar tu sitio a su base de datos. Por ejemplo:

  • Google — Get on Google
  • Bing — Webmaster Tools

Search Engine Optimization (SEO)

Realizar pequeños ajustes en tu sitio ayuda a los motores de búsqueda a rastrear y clasificar tu contenido. Esto, a su vez, ayuda a que tu sitio suba en los resultados de búsqueda:

  • Yoast SEO plugin for WordPress
