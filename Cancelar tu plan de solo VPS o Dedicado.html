Generalidades

Este artículo explica cómo cerrar tu plan de alojamiento VPS o Dedicado en DreamHost.

Para mantener activo tu sitio web mientras migras a un nuevo plan de alojamiento, debes tener ya otro plan de alojamiento. Esto te permite mover tus datos antes de cerrar tu plan de alojamiento VPS o Dedicado. Sin otro servicio de alojamiento, tu sitio web quedará fuera de línea y tu cuenta solo estará disponible para registros de dominios.

Asegúrate de tener otro plan de alojamiento activo si deseas mantener tu sitio en funcionamiento.

Cerrar/cancelar tu plan de alojamiento VPS o Dedicado

Para eliminar el plan de alojamiento VPS o Dedicado, sigue estas instrucciones:

  • Haz clic aquí para obtener instrucciones completas sobre cómo eliminar el plan de VPS.
  • Da clic aquí para obtener instrucciones completas sobre cómo eliminar el plan de alojamiento Dedicado.

Para obtener más detalles sobre cómo cerrar tu cuenta, revisa este artículo.
