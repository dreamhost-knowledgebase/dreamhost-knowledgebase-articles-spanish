Descargar y descomprimir tus archivos de correo

Cuando tu usas la opción de Respaldo a Un Clic del panel, todos los datos se guardan como archivos .tar.gz en tu panel. Revisael siguiente artículo para obtener instrucciones sobre cómo descargar y descomprimir estos archivos:

  • Respaldar una cuenta en el panel

Agregar una extensión de archivo .eml a todos los archivos

Cuando descomprimes tu archivo de correo usando el artículo anterior, los archivos descomprimidos se colocan en una carpeta titulada algo así como x21563833.

Esta carpeta contiene una carpeta Maildir. Haz clic en esta carpeta.

Dentro de esa carpeta encontrarás tus carpetas y mensajes de correo. Para tus mensajes de la Bandeja de entrada, haz clic en las carpetas /cur o /new. Para todos los demás mensajes, haz clic en tu carpeta y ve tus directorios /cur o /new.

Los correos aparecerán en el siguiente formato:

1435986g36.V34I895305bS934136.homiemail-master

Si arrastras y sueltas este archivo en tu cliente de correo, no funcionará. Primero debes cambiar el nombre del archivo.

Agrega .eml al final de cualquier archivo de correo. Por ejemplo:

1435986g36.V34I895305bS934136.homiemail-master.eml

Solo cuando terminen en este formato tu cliente de correo reconocerá el tipo de archivo. Luego puedes arrastrar y soltar este archivo en tu cliente.

Cómo agregar la extensión .eml a varios archivos a la vez

Los nombres de las carpetas dentro del directorio /Maildir comúmnente se anteponen con un punto (.) y puede que estén escondidas, como .old-messages y .sent. Asegúrate de habilitar los archivos/carpetas escondidos para que se vean dentro de tu sistema operatido para poder verlos.

Mac o Linux

  1. Si estás en un computador Mac o Linux, abre una terminal y navega hasta el directorio donde se almacenan tus correos. Este debería ser el directorio /Maildir para una dirección de correo específica. 
  2. Corre el siguiente comando en el directorio de esta dirección: 
    [local ~]$ find . -type f -exec mv '{}' '{}'.eml \;
  3. Después de que todos los archivos en este directorio y todos sus subdirectorios tienen la extensión .EML agregada a ellos, arrástralos a tu cliente de correo para cargarlos a tu nueva dirección.

Windows

  1. Si estás en una computador con Windows, descarga este programa que te permitirá ejecutar comandos de Unix.
    • Git para Windows
  2. En Windows, navega hasta el directorio donde realizaste una copia de seguridad y descomprimiste tu archivo de correo. Como se muestra arriba, asegúrate de estar en el directorio /Maildir del correo.
  3. Haz clic derecho dentro de este directorio.
  4. En el menú que aparece, haz clic en Git Bash Here. Esto abre una terminal donde puedes ejecutar comandos de Unix.
  5. Corre el siguiente comando en el directorio de esta dirección: 
    [local ~]$ find . -type f -exec mv '{}' '{}'.eml \;
  6. Después de que todos los archivos en este directorio y todos sus subdirectorios tienen la extensión .EML agregada a ellos, arrástralos a tu cliente de correo para cargarlos a tu nueva dirección
