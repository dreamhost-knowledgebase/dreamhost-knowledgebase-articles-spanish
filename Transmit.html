Generalidades

Transmit es un cliente FTP para Mac OS X desarrollado por Panic que también incluye capacidades para montar conexiones SFTP, Amazon S3 y WebDAV. El cliente es un shareware, por lo que la evaluación es gratuita, pero una vez finalizado el período de prueba, está limitado a sesiones de siete minutos.

Esta guía describe cómo usar Transmit para montar tus buckets de DreamObjects. Para obtener más información, visita el sitio web de Transmit:

  • panic.com/transmit/

Usar Transmit con DreamObjects

Antes de comenzar, asegúrate de que ya tienes un bucket de DreamObjects creado. Luego descarga la última versión del sitio web de Panic.

  1. Lanza Transmit.
    Cuando lanzas Transmit la primera vez, debes seleccionar un tipo de protocolo:
  2. Del menú desplegable Protocol, selecciona 'Amazon S3'.
    • Address: objects-us-east-1.dream.io (no aparece si seleccionaste 'DreamObjects' como protocolo)
    • Access Key ID: Lo encuentras en tu panel de DreamHost Panel. Ve el artículo llaves de DreamObjects para más información.
    • Secret: Lo encuentras en tu panel de DreamHost. Ve el artículo llaves de DreamObjects para más información.
    • Remote Path: Puedes dejar la ruta remota en blanco o aceptar las configuraciones por defecto.
  3. Haz clic en el botón Connect cuando hayas finalizado para conectarte a DreamObjects.
  4. Para guardar el servidor como favorito, haz clic en el botón Add to Servers.
