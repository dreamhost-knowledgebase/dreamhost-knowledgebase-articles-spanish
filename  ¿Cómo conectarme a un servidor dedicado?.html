Conectarse a un Servidor Dedicado

Si el nombre de tu Servidor Dedicado es ds12345, puedes conectarte a él antes de agregar cualquier dominio usando el siguiente hostname:

ds12345.dreamservers.com

Una vez que se agrega tu dominio, puedes usarlo para el nombre de host. Pero puedes continuar usando el nombre del servidor si lo deseas. Mira los siguientes artículos para obtener información sobre cómo iniciar sesión en tu servidor:

  • Generalidades sobre FTP y credenciales
  • Generalidades sobre SSH
