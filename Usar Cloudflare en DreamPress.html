Generalidades

Este artículo cubre consideraciones específicas cuando se usa Cloudflare con DreamPress.

Configurar planes de socios de Cloudflare en DreamPress

Puedes registrarte a los planes de Cloudflare directamente en cloudflare.com. Por favor recuerda que si escoges esta opción, necesitarás modificar tus nameservers y administrar tus registros de DNS por ti mismo.

Configuración adicional

Se recomienda encarecidamente que instales el plugin Proxy Cache Purge en tu sitio DreamPress. Si bien Proxy Cache Purge se instala de manera predeterminada en todos los sitios nuevos de DreamPress, si migras un sitio de WordPress existente a DreamPress, los archivos del plugin podrían sobrescribirse. Puedes instalarlo tú mismo si tu sitio DreamPress no tiene el plugin:

  • Configurar el plugin Proxy Cache Purge en DreamPress

Si tienes el plugin instalado y activado, visita el menú "Proxy Cache" en tu WP-Admin para verificar que el campo "Set Custom IP" usa la IP del servidor (por defecto). Si no, ajusta el campo y da clic al botón Save IP.

Si ves algún problema extraño de almacenamiento en caché o rendimiento, el plugin Cloudflare, también disponible en el repositorio de WordPress.org, puede ayudarte a optimizar el rendimiento de tu sitio con Cloudflare. Sin embargo, este plugin no es obligatorio.

Pros y contras de usar Cloudflare en DreamPress

Debido a inconvenientes con la compatibilidad, DreamHost NO recomienda usar la Automatic Platform Optimizations (APO) de Cloudflare en tu sitio DreamPress.

DreamPress incluye almacenamiento en caché del servidor incorporado para que tus páginas se carguen muy rápidamente, lo que actúa como un "proxy inverso" de la misma manera que lo hace Cloudflare. Trabajar en tu sitio con Cloudflare activo en DreamPress a veces puede ser complicado ya que Cloudflare está almacenando en caché un sitio que ya está en caché utilizando el mismo método. Puedes reducir este efecto habilitando el modo de Desarrollo en Cloudflare y el plugin Proxy Cache Purge.

Si deseas una alternativa a Cloudflare para evitar posibles problemas de doble caché, la versión gratuita de Jetpack ofrece algunas características similares, como un CDN automático de archivos estáticos; y la suscripción a Jetpack Professional incluida con los planes DreamPress Plus y Pro contiene características de seguridad adicionales que pueden también ofrecer una alternativa a lo que proporciona Cloudflare.

Cloudflare ofrece características beneficiosas adicionales además del almacenamiento en caché de proxy inverso. Muchos clientes de DreamPress ejecutan con éxito tu sitio con Cloudflare activo. Estas características incluyen protección DDoS básica, desafío de captcha por IP o código de país, bloqueo de IP específicas dentro de la interfaz de Cloudflare, protección contra spam de comentarios, legrado de contenido, y más.
