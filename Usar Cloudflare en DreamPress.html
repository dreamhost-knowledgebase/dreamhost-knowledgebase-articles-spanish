Generalidades

Este artículo cubre consideraciones específicas cuando se usa Cloudflare con DreamPress.

Configurar planes de socios de Cloudflare en DreamPress

Puedes registrarte a los planes de Cloudflare directamente en cloudflare.com. Por favor recuerda que necesitarás modificar tus nameservers y administrar tus registros de DNS por ti mismo.

Configuración adicional

Mientras que el Plugin Nginx Helper se instala por defecto en todos los nuevos sitios DreamPress, los archivos del plugin puede que se sobrescriban si migras un si migras un sitio WordPress existente a DreamPress. Sin embargo, siempre puedes instalarlo por ti mismo si tu sitio DreamPress no tiene un plugin:

  • Administrar el caché de DreamPress con el plugin Nginx Helper

Plugin de Cloudflare

Si ves algún problema extraño de almacenamiento en caché o rendimiento, el plugin Cloudflare, también disponible en el repositorio de WordPress.org, puede ayudarte a optimizar el rendimiento de tu sitio con Cloudflare. Sin embargo, este plugin no es obligatorio.

Pros y contras de usar Cloudflare en DreamPress

Debido a inconvenientes con la compatibilidad, DreamHost NO recomienda usar la Automatic Platform Optimizations (APO) de Cloudflare en tu sitio DreamPress.

DreamPress incluye almacenamiento en caché del servidor incorporado para que tus páginas se carguen muy rápidamente, lo que actúa como un "proxy inverso" de la misma manera que lo hace Cloudflare. Trabajar en tu sitio con Cloudflare activo en DreamPress a veces puede ser complicado ya que Cloudflare está almacenando en caché un sitio que ya está en caché utilizando el mismo método. Puedes reducir este efecto habilitando el modo de Desarrollo en Cloudflare y el plugin Proxy Cache Purge.

Si deseas una alternativa a Cloudflare para evitar posibles problemas de doble caché, la versión gratuita de Jetpack ofrece algunas características similares, como un CDN automático de archivos estáticos.

Cloudflare ofrece características beneficiosas adicionales además del almacenamiento en caché de proxy inverso. Muchos clientes de DreamPress corren sus sitios exitosamente con Cloudflare activo. Estas características incluyen protección DDoS básica, desafío de captcha por IP o código de país, bloqueo de IP específicas dentro de la interfaz de Cloudflare, protección contra spam de comentarios, legrado de contenido, y más.
