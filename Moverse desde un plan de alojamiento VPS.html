Generalidades

A continuación se describen las diversas formas en las que puedes degradar o mejorar un plan de alojamiento VPS.

Cómo degradar un plan de alojamiento VPS

Las siguientes secciones explican cómo degradar un VPS a otro plan de alojamiento.

Degradar a Shared Starter

Si deseas degradar de un plan VPS a un plan de alojamiento Shared Starter, asegúrate de que tu sitio web se ajuste a las limitaciones del plan, y luego contacta a soporte para solicitarlo.

Degradar a Shared Unlimited

Para degradar a un plan de alojamiento Shared Unlimited, primero debes agregar el plan. Luego, puedes proceder a eliminar el VPS.

Al eliminar, aparecerá un mensaje informándote que todo en el VPS será movido al Servidor Shared

Si tienes tanto un Servidor Shared como un Servidor Dedicado en tu cuenta, solo el Servidor Dedicado aparecerá como una opción para mover al eliminar tu VPS.

Si prefieres moverte a un Servidor Shared, contacta a soporte. El soporte de DreamHost moverá los datos por ti.

Cómo mejorar un VPS a otro plan de alojamiento

Las siguientes secciones explican cómo actualizar un VPS a otro plan de alojamiento.

Mejorar a un Servidor Dedicado

Si deseas mover todo el contenido de tu VPS a un Servidor Dedicado, primero debes comprar el Servidor Dedicado.

Al agregar el Servidor Dedicado, selecciona tu VPS en el menú desplegable para mover todos los datos. Una vez que todo se haya movido, puedes eliminar tu VPS.

Mejorar a DreamPress

Revisa las instrucciones sobre cómo actualizar un sitio de WordPress existente a DreamPress.

  • Si solo tenías el sitio de WordPress, puedes entonces eliminar el VPS.
  • Si tienes otros sitios en el VPS, debes seguir pagando por ese plan ya que solo el sitio único se habrá movido a DreamPress.

Moverse a un plan de VPS diferente

Revisa el artículo cambiar tu plan de alojamiento VPS para obtener instrucciones sobre cómo mover tus sitios a un plan diferente.
