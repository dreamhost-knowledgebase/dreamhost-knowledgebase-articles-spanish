Generalidades

Este artículo proporciona información sobre las direcciones de correo Catch-All y opciones alternativas para administrar tu correo.

Direcciones de correo Catch-All en DreamHost

DreamHost no admite direcciones Catch-All. Esto se debe a que están diseñadas para aceptar todos los mensajes, lo que atrae un volumen abrumador de spam, degrada el rendimiento del sistema de correo para todos los clientes y retrasa la entrega de mensajes legítimos.

¿Qué alternativas tengo?

Las siguientes son algunas alternativas que puedes utilizar para administrar tu correo electrónico.

Google Workspace

DreamHost se ha asociado con Google para ofrecer Google Workspace. Cuando agrega correos en Google, puedes crear una dirección de Catch-All dentro de tu sistema.

Crea un alias en tu dirección de correo

DreamHost ofrece la posibilidad de crear un alias de un correo existente. Esta función está habilitada de manera predeterminada en todas las cuentas y no hay límite en la cantidad de alias que puedes crear.

Para crear un alias, agrega un nuevo correo en el panel. Al nombrarla, haz este ajuste después del nombre de usuario de correo, pero antes del símbolo @:

  • agrega el símbolo más (+) seguido de cualquier combinación de letras y dígitos.

For example:

Tipo Dirección
Correo principal nombre@example.com
Alias #1 nombre+shoppingsite@example.com
Alias #2 nombre+onlinebanking@example.com

Todas las direcciones alias se envían al correo en el que se basan, en este caso, nombre@example.com.

Este tipo de alias son útiles para rastrear si un sitio web en particular ha compartido o filtrado tu información de contacto o para categorizar de manera proactiva los registros de correo en varios sitios web.
