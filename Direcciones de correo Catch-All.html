Generalidades

DreamHost no admite direcciones de correo generales. Debido a que este tipo de direcciones no rechaza ningún mensaje, los dominios con un alcance general atraen un volumen abrumador de spam. Esto degrada el rendimiento del sistema de correo para todos los clientes y retrasa la entrega de mensajes legítimos.

A continuación, se describen algunas alternativas a las direcciones de correo catch-all que puedes usar en su lugar.

¿Cómo puedo crear un alias en mi correo?

Las cuentas de correo DreamHost permiten el uso de alias de direcciones de correo ‘desechables’ o ‘sobre la marcha’, donde puedes generar instantáneamente una dirección de correo única basada en un buzón existente. Para habilitar esta función, simplemente escribe un alias cada vez que se te pida tu correo de DreamHost.

Estos tipos de alias son útiles para rastrear si un sitio web en particular ha compartido o filtrado tu información de contacto, o para clasificar proactivamente las suscripciones de correo en varios sitios web.

Para crear un nuevo alias sobre la marcha, agrega un símbolo más (+) seguido de cualquier combinación de letras y dígitos después de tu nombre de usuario de correo, antes de la @. Por ejemplo:

Correo principal: jim@example.com
Nuevo alias: jim+somecoolshop22@example.com
Otro alias nuevo: jim+onlinebanking@example.com

El correo enviado a cualquiera de estas direcciones se enviará a jim@example.com.

No hay límite para la cantidad de alias que puedes crear de esta manera, y no requieren configuración previa. Esta función ya está habilitada en todas las cuentas de correo de DreamHost.
