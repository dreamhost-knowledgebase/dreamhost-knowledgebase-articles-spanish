Configuraciones de correo electrónico

Se recomienda enviar correo electrónico a través de SMTP. Cuando envías un correo electrónico directamente desde tu sitio web, el correo electrónico se envía DESDE el servidor web. Si la dirección FROM en Joomla no está configurada correctamente, es posible que el correo electrónico no se envíe.

Sin embargo, si has habilitado la autenticación SMTP en el panel de Joomla, el correo electrónico se envía desde el servidor de correo. Este es el método preferido para evitar problemas de envío.

Habilitar la autenticación SMTP

  1. En la esquina superior izquierda, haz clic en el elemento de menú ‘System’.
  2. En el menú desplegable, selecciona ‘Global Configuration’.
    La página Global Configuration se abre:
  3. Haz clic en la pestaña ‘Server’.
    En la parte inferior derecha de esta página está la sección ‘Mail Settings’.
  4. Haz las siguientes selecciones:
    • En el menú desplegable ‘Mailer’, selecciona ‘SMTP’.
    • Junto a ‘SMTP Authentication’, haz clic en el botón Yes.
    • Completa todos los demás campos, incluida la información de tu dirección de correo electrónico como se muestra en la captura de pantalla anterior.
  5. En la parte superior izquierda de la página, haz clic en el botón Save para guardar estos nuevos ajustes.

Los correos electrónicos ahora se envían a través de SMTP desde tu sitio web. Visite lo siguiente para más detalles:

  • Joomla! Documentation – Global configuration

Solución de problemas de correo electrónico

Si eliges no utilizar la autenticación SMTP y encuentras que no se envían correos electrónicos, marca la opción ‘Custom Reply’.

Ve a 'Components -> Contacts -> Options'. En la pestaña Form, asegúrate de que la "Custom Reply" está configurada en NO.

Si esto no se hace, Joomla puede que ni siquiera genere el correo electrónico y que el formulario no funcione.

Solución de problemas

Puedes ver el siguiente error al visitar tu sitio de Joomla.

Database connection error (1): The MySQL adapter 'mysql' is not available.

La función MySQL solo está disponible en versiones de PHP inferiores a 7.0. Este error indica que tu sitio está ejecutando una versión anterior de Joomla que no es compatible con las versiones más nuevas de PHP, como 7.2, 7.3 o 7.4. La solución es actualizar tu sitio a la versión #3 de Joomla. Puedes leer más sobre las versiones recomendadas de PHP para tu versión de Joomla aquí.
