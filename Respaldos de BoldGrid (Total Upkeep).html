Generalidades

Para usar el plugin Total Upkeep, primero debes configurar tu sitio web de WordPress con el Constructor Web de WP (BoldGrid). Luego si puedes habilitarlo usando las siguientes instrucciones.

Las instalaciones de BoldGrid viene con un plugin de respaldo llamado 'Total Upkeep'. Este plugin le permite configurar como, cuando y donde se almacenan tu sitio web de WordPress y los archivos de la base de datos.

El plugin Total Upkeep Premium también está preinstalado con tu instalación de BoldGrid, sin cargos extra. Debajo encontrarás algunas características incluidas con la versión premium:

  • Encriptación de base de datos
  • Actualizaciones automáticas oportunas
  • Historial de actualizaciones
  • Restauraciones de archivos individuales

Tienes las siguientes opciones para almacenar tus copias de seguridad:

  • Servidores web DreamHost 
  • Servidores Web externos
  • Cuenta Amazon S3 (Se requiere Total Upkeep Premium)
  • Cuenta Google Drive (Se requiere Total Upkeep Premium)
  • DreamObjects (Se requiere Total Upkeep Premium)

Este artículo te guía a través de la creación de una copia de seguridad y su visualización en tu servidor.

Crear una copia de seguridad

Seleccionar cuándo ejecutar la copia de seguridad

  1. Inicia sesión en tu panel de WordPress.
  2. En el menu Izquierdo, desplazate sobre Total Upkeep y selecciona Settings.
  3. En el menú de la izquierda, haz clic en Backup Schedule y selecciona qué días ejecutar la copia de seguridad.
  4. Haz clic en Save Changes.

Establecer la ubicación de la copia de seguridad

  1. Selecciona Backup Storage del menú Aquí puedes seleccionar entre las siguientes opciones donde se almacena la copia de seguridad.
    • Web Server — Tu servidor DreamHost.
    • FTP / SFTP — Cualquier servidor que te guste.
    • Amazon S3 — El servicio en la nube de Amazon.
    • Google Drive — Tu cuenta de Google.
  2. Marca la casilla junto a Web Server para almacenar tu copia de seguridad en tu servidor DreamHost.
  3. Haz clic en el botón Configure a la derecha si deseas hacer una copia de seguridad de cualquier otro servicio.

    Se recomienda dejar las opciones Full Backup seleccionadas para tus archivos y base de datos.

  4. Haz clic en Save Changes.

Configurar actualizaciones automáticas

Las seccion Auto Updates controla el manejo de las copias de seguridad justo antes de una actualización de WordPress, y que información se respalda.

Site Check

La sección Site Check habilita una herramienta para verificar si hay errores en tu sitio y registrarlos. Puedes encontrar más información sobre Site Check y los comandos utilizados aquí:

  • Comandos CLI

Notificaciones

La sección Notifications te permite establecer la configuración de notificación por correo para cuando se complete una copia de seguridad o cuando falle una verificación del sitio.

Preflight Check

Haz clic en la pestaña Preflight Check para asegurarte de que el entorno esté configurado correctamente. Deberías ver la palabra 'PASS' en la parte superior.

 Ver tus copias de seguridad en WordPress

La pestaña Backups muestra las copias de seguridad existentes que se han creado.

Haz clic en el botón View details para ver información más detallada de la copia de seguridad.

Ver tus copias de seguridad en el servidor web

Si eliges realizar una copia de seguridad en el servidor web, tus copias de seguridad se encuentran debajo de tu nombre de usuario en un directorio llamado /boldgrid_backup. Puedes iniciar sesión a través de FTP o SSH para ver las copias de seguridad.
