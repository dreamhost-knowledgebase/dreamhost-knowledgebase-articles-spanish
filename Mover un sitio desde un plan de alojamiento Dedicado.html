Generalidades

Este artículo describe las diversas formas en que puedes pasar de un plan de alojamiento de Servidor Dedicado a otro plan en DreamHost.

Cómo pasar a otro plan de alojamiento

Las siguientes secciones explican cómo pasar de un plan de alojamiento de Servidor Dedicado a otro plan.

Moverse a DreamPress

Revisa el artículo Mejorar un sitio de WordPress a DreamPress para aprender cómo mover un sitio de WordPress existente a DreamPress.

  • Si solo tenías un sitio WordPress en tu Servidor Dedicado, puedes cancelarlo.
  • Si tiene otros sitios en el Servidor Dedicado, debes continuar pagando el plan, ya que solo un sitio se habría movido a DreamPress.

Cambiar a un plan de Alojamiento Dedicado diferente

Contacta a soporte para obtener ayuda para mover información entre dos Servidores Dedicados.

Como degradar a un plan de alojamiento de Servidor Dedicado

Las siguientes secciones explican cómo degradar a otro plan.

Degradar a un Shared Starter

Si tienes un solo sitio web en tu plan de alojamiento de Servidor Dedicado y deseas degradar a un plan de alojamiento Shared Starter, asegúrate primero de entender las limitaciones descritas en el artículo Agregar un plan de alojamiento Shared Starter, luego contacta a soporte para solicitarlo.

Degradar a un plan Shared Unlimited o VPS

Asegúrate de haber agregado un plan de alojamiento Shared o un VPS. Luego, puedes eliminar el Servidor Dedicado.

Cuando lo elimines, aparece un mensaje informandote que todo en el Servidor Dedicado se moverá el servidor Shared o VPS.

  • Si tienes un servidor Shared y un VPS en tu cuenta, solo el servidor VPS se mostrará como una opción para moverse al eliminar tu Servidor Dedicado.
  • Si prefieres pasar a un Servidor Shared, comunícate con soporte . El soporte de DreamHost transferirá la información por ti.
