Generalidades

Este artículo te presenta los servicios más importantes de DreamHost necesarios para configurar tu sitio web. Si bien esto no cubre todo, sí cubre lo básico y es un buen lugar para comenzar después de comprar una cuenta.

Configuración Básica

 

Crear una cuenta en DreamHost

Si deseas utilizar algún servicio de DreamHost, debes crear una cuenta. Revisa el siguiente artículo para obtener instrucciones sobre cómo abrir una cuenta.

  • Cómo abrir una cuenta en DreamHost

Entrando al panel

Una vez terminado el proceso de registro, recibirás un mensaje para notificarte que tu cuenta está configurada. Desde allí, puedes iniciar sesión en el panel en:

  • https://panel.dreamhost.com/

Para iniciar sesión, usa las credenciales del correo electrónico que se le envió.

Comprar un dominio

Para cargar cualquier página web, el propietario debe primero ser propietario del registro del nombre de dominio. Si no posee el registro, no tiene derecho a utilizar el nombre de dominio.

Para comprar un dominio, ve a la página de Registrations.

En la página de Registrations, puedes registrar cualquier dominio que desees que aún esté disponible:

DreamHost permite el registro de muchos dominios de nivel superior diferentes (TLD). Puedes ver una lista completa de estos en la página de Registros haciendo click en el enlace 'Ver precio completo'.

Para más información, por favor revisa el artículo Domain Registration.

Agregar un plan de alojamiento

Después de comprar un dominio, debes agregar un plan de alojamiento para que el sitio resuelva en línea. Hay diferentes opciones de alojamiento que puedes usar para tu dominio.

  • Alojamiento completo (Fully hosted) — tu sitio se encuentra alojado en un servidor y es visible online
  • Remixer — tu sitio se encuentra alojado en el plan Remixer (nuestra herramienta para construir sitios web)
  • WordPress — tu sitio se encuentra alojado en un servidor (alojamiento completo) pero está ejecutando el programa WordPress
  • DreamPress — tu sitio tiene alojamiento completo ejecutando WordPress pero se encuentra en servidor de DreamPress

Cómo agregar un plan de alojamiento

Para agregar un plan de alojamiento, ve a la página Manage Account.

En la página Administrar cuenta, verás una sección titulada 'Planes de servicio':

Si ya has agregado un plan, lo verás aquí. En este ejemplo, se agregó un plan de alojamiento 'Compartido' (Shared) que se paga por adelantado un valor de $ 119.40 para todo el año.

Cómo alojar un dominio

Revisa el siguiente artículo para obtener instrucciones sobre cómo agregar alojamiento a tu dominio para que esté completamente alojado.

  • ¿Cómo agrego un dominio?

Incluso si configuras un sitio para que esté COMPLETAMENTE ALOJADO en tu panel, no quiere decir que el sitio esté alojado en DreamHost. Los registros DNS también deben estar apuntados a tu servidor de DreamHost. La compañía que aloja tus Nombres de servidor (nameservers), controla tus DNS. Puedes revisar adonde apuntas tus servidores de dominio al hacer una búsqueda pública de tu información WHOIS, aquí:

  • http://www.dreamhost.com/domains/

Si los nombres de servidor ya están apuntando a DreamHost, entonces to lo que necesitas es configurar el alojamiento. Revisa los siguientes artículos para más información:

  • Cómo Agregar Dominios y Subdominios
  • DNS
  • Nombres de Servidor

Alojamiento en Remixer

DreamHost también ofrece un creador de sitios web personalizado llamado 'Remixer' que no requiere experiencia en codificación. Remixer es un servicio de alojamiento que distinto a plan de Alojamiento Completo como el Compartido (Shared), VPS, Dedicado, o DreamPress. Con Remixer, puedes construir tu sitio web usando la interfaz en línea con sólo hacer click. Opciones de alojamiento en WordPress

 

Agregar WordPress

WordPress es el sistema de administración de contenido (CMS) más popular del mundo, y DreamHost es uno de los pocos proveedores recomendados por WordPress.org. Si eres nuevo en el alojamiento de un sitio web, querrás darle un vistazo a WordPress para ayudarte a construir tu sitio web. En cuestión de minutos, puedes crear un sitio que incorpore todas las características modernas que ahora incluyen los sitios web, con una capacidad casi ilimitada para agregar nuevas funciones a medida que las necesites.

Para ayudarte a empezar a construir tu sitio WordPress en DreamHost, consulta los siguientes artículos:

  • Cómo instalar WordPress usando el instalador de un click
  • Planes de Alojamiento de WordPress en DreamHost
  • Guía de DreamHost para WordPress

Alojamiento DreamPress

DreamPress es el Alojamiento de WordPress administrado de DreamHost, que proporciona funciones aún más potentes, como planes de memoria y almacenamiento más grandes, velocidades más rápidas y almacenamiento en caché mejorado.

Más información sobre DreamPress se puede encontrar aquí:

  • DreamPress

Opciones de servicios de migración de WordPress

DreamHost te puede ayudar a migrar tu sitio, ya sea a través del Servicio de Migración de WordPress pago o la herramienta de migración DreamPress gratuita que usa el panel, que se describen a continuación.

Opción paga para migrar tu sitio

Si tienes problemas para migrar tu sitio web de otro host, o simplemente no tienes tiempo, DreamHost actualmente ofrece un servicio que se encarga de eso por ti. Puedes encontrar más información en el artículo de DreamHost Servicio de migración de sitios web.

Opción de DreamPress para migrar tu sitio a un nuevo dominio

También puedes mejorar tu nuevo sitio y alojarlo en DreamPress, que ofrece una opción de importación.

Si tu instalador de DreamHost instaló y administró tu sitio de WordPress actual, puedes configurar DreamPress con el nuevo nombre de dominio e importar el contenido del sitio de WordPress actual al nuevo dominio de DreamPress. Puedes agregar DreamPress en un nuevo dominio en el panel de DreamHost en Managed WordPress. Por favor revisa la siguiente guía para importar contenido en DreamPress:

  • Importar una instalación de WordPress existente en DreamPress

Si tu actual sitio WordPress no está ejecutando el Instalador a un Click, puedes crear un caso para pedir que tu instalación la conviertan en el instalador a un click.

Agregar un certificado SSL

 

Agregar un certificado SSL a un dominio

Después de configurar el alojamiento para tu dominio, te recomendamos encarecidamente que también le agregues un certificado SSL al dominio (ya sea un certificado gratuito 'Let's Encrypt' o uno firmado profesionalmente). Agregar un certificado es un proceso simple que no solo asegura tu dominio para realizar transacciones monetarias y de eCommerce, sino que también logra que tu sitio sea calificado como seguro. De hecho, Google dará una calificación más favorable ya qe tener un sitio seguro es un factor importante para los motores de búsqueda.

Para ayudarte a iniciar el proceso de agregar un certificado, por favor dale un vistazo a los siguientes artículos:

  • Agregar un certificado Let's Encrypt gratis
  • ¿Cómo puedo comprar un certificado SSL firmado profesionalmente?
  • ¿Cómo puedo agregar un certificado SSL autofirmado?

Configuraciones adicionales

Los siguientes son servicios o productos adicionales que puedes agregar a su sitio, pero no son obligatorios para que tu sitio funcione.

Crear una dirección de correo electrónico

Una vez que agregues el alojamiento a un dominio, puedes crear una dirección de correo electrónico.

Para crear una dirección de correo electrónico:

  1. Ve a la página Manage Email.
  2. Haz click en el botón Crear nueva dirección de correo electrónico en la parte superior.
    Luego podrás ingresar un nuevo correo electrónico y una contraseña.

Una vez que guardes los cambios, los registros DNS necesarios para el correo se agregarán al DNS de tu sitio.

Puedes encontrar más información sobre la creación y configuración de cuentas de correo electrónico con DreamHost en la página Configuración de correo electrónico

También es posible alojar tu correo electrónico fuera de DreamHost. Para alojar en otro lugar, puede configurar registros MX personalizados para que apunten a tu servidor de correo externo. Visita el artículo Registros MX para más información.

Agregando estadísticas del sitio

Todas las cuentas de DreamHost tienen la capacidad de agregar estadísticas del sitio a cualquier sitio web totalmente alojado; las estadísticas las puedes habilitar en el panel. Allí, puede realizar un seguimiento de la información, como el número de solicitudes, el tráfico por dominio, el tráfico por archivo, las solicitudes por tipo de navegador / SO / URL, y más.

Para obtener más información sobre cómo agregar estadísticas del sitio, consulta los siguientes artículos:

  • Habilitar estadísticas
  • Hacer estadísticas accesibles con un archivo .htaccess
  • Generalidades sobre estadísticas análogas

Agregar el servicio de eliminación de malware DreamShield

DreamHost también ofrece un servicio de seguridad de pago en el panel. Cuando está habilitado, este servicio te permite escanear tu sitio web en busca de malware y te notifica si encuentra algún problema.

Para obtener más información sobre cómo habilitar el servicio de eliminación de malware, consulta el siguiente artículo:

  • ¿Cómo habilitar DreamShield?

Si hay algo en lo que el equipo de soporte te pueda ayudar con la configuración y carga de tu sitio web, no dudes en enviar un caso de soporte en la página Contact Support desde tu panel de DreamHost.
