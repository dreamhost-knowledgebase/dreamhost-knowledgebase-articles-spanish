Generalidades

Un archivo openrc es un pequeño script bash que establece variables de entorno para que las herramientas de línea de comandos de OpenStack sepan cómo comunicarse con DreamCompute. 

¿Cómo descargar mi archivo openrc?

Puedes encontrar tu archivo openrc en el panel de DreamCompute.

  1. Navega a la sección Compute > API Access.
  2. A la derecha, haz clic en el botón Download OpenStack RC File.

¿Cómo usar mi archivo openrc?

Las instrucciones de este artículo sobre cómo source (configurar) el archivo openrc.sh son solo para Mac OS X y Linux. Para obtener ayuda sobre el uso de tu archivo openrc.sh en un entorno Windows, consulta este artículo:

  • Cómo trabajo con DreamCompute en Windows

Si descargaste el archivo openrcfile.sh del Panel de control sin especificar una ubicación de destino, lo más probable es que se haya descargado en el directorio /Downloads. En un Mac sería:

/Users/username/Downloads

Este archivo que descargaste se llama algo así como dhc123456-openrc.sh. Puedes cambiar el nombre de esto a algo como openrc.sh si lo deseas.

[user@localhost]$ mv ~/Downloads/dhc-123456-openrc.sh ~/Downloads/openrc.sh

Usa el comando source para ejecutar este archivo. Este ejemplo muestra cómo ejecutarlo desde el directorio /Downloads de un Mac:

[user@localhost]$ source ~/Downloads/openrc.sh

Tu terminal te pedirá que ingreses una contraseña. Esta es tu contraseña del panel de DreamCompute, la misma que usas para iniciar sesión en el panel de DreamCompute en DreamCompute.

Después de ingresar la contraseña, se deben configurar todas las variables ambientales necesarias. Ahora puedes ejecutar un cliente de línea de comandos como el CLI de OpenStack.
