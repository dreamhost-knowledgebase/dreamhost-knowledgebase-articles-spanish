Generalidades

Lo siguiente es una visión general rápida de cómo conectarte a tu servidor web de DreamHost y subir los archivos de tu sitio web. Hay dos partes principales de tu sitio web que necesitarás subir:

  • Los archivos del sitio web
  • La base de datos del sitio web

Requisitos previos

Este artículo asume que ya has agregaro un plan de alojamiento a tu dominio en DreamHost.

Subir el contenido de tu sitio

Estos pasos permiten subir los archivos de tu sitio web, importar la base de datos, y apuntar el DNS.

 

Subir los archivos del sitio web

Descargar un cliente de FTP

Primero, necesitarás descargar un cliente de FTP en tu computador. Este software te permitirá subir los archivos de tu sitio web a tu servidor DreamHost.

Alternativamente, podrías usar el administrador de archivos de sitios web de DreamHost, que es un cliente FTP integrado al panel. Revisa el siguiente artículo para obtener más información:

  • Usar el administrador de archivos de sitios web del panel

Iniciar sesión en tu servidor

A continuación, iniciarás sesión en tu servidor usando tu cliente FTP. Revisa el siguiente artículo para obtener instrucciones:

  • Generalidades sobre FTP y sus credenciales

Subir los archivos del sitio web

Una vez que hayas iniciado sesión, verás varios archivos y carpetas. Necesitarás navegar al directorio web para cargar tus archivos. Revisa el siguiente artículo para obtener instrucciones sobre cómo localizar este directorio:

  • ¿Dónde debo subir mis archivos una vez conectado a FTP?

Importar la base de datos de tu sitio

Tu sitio web también puede usar una base de datos además de los archivos del sitio web. Si tienes una base de datos, revisa los siguientes artículos para obtener instrucciones sobre cómo cargarla en DreamHost:

  • ¿Cómo migrar mi base de datos a DreamHost?
  • phpMyAdmin — Cómo importar o restaurar una base de datos o una tabla

Apuntar el DNS

En este punto, tu sitio web está cargado en DreamHost. Sin embargo, para que se muestre en línea desde tu servidor, el DNS del dominio debe apuntar a DreamHost.

  • Si compraste el registro de tu dominio con nosotros, el DNS ya debería estar apuntando a DreamHost.
  • Si compraste el registro de tu dominio en otra compañía, necesitas apuntar el DNS a DreamHost.

Revisa este artículo para confirmar a dónde está apuntando actualmente tu DNS. Si es necesario, puedes seguir los pasos en el siguiente artículo para apuntar tu DNS a DreamHost desde tu registrador actual.

  • ¿A qué registros de DNS debo apuntar mi sitio?

Opciones de migración de WordPress

DreamHost ofrece varias opciones de migración para sitios web construidos con WordPress. Revisa el siguiente artículo para conocer las diferentes formas de migrar tu sitio de WordPress.

  • Generalidades de las migraciones de WordPress

Servicio de Migración de WordPress

Si estás teniendo problemas para migrar tu sitio web desde otro alojamiento o simplemente no tienes el tiempo, DreamHost ofrece un Servicio de Migración de WordPress que se encarga de esto por ti. Puedes encontrar más información en el artículo Servicios de migración de sitios web de DreamHost.

Si tu sitio WordPress ya está alojado en DreamHost y necesitas moverlo a otra cuenta de DreamHost, revisa el siguiente artículo en su lugar.

  •  Servicio de Content Move de WordPress

Solución de problemas

Es posible que después de subir tu sitio, solo veas un "Directorio principal" en la página. Esto suele ocurrir por las siguientes razones:

  • los archivos no se subieron al directorio de tu sitio web.
  • utilizaste las credenciales FTP de otro usuario al iniciar sesión en tu servidor.

Revisar los pasos anteriores para confirmar esta información debería solucionar el problema.
