¿Qué versiones de PHP soporta DreamHost?

Consulta el siguiente artículo para obtener una lista de las versiones actuales de PHP compatibles con DreamHost:

  • ¿Qué versiones de PHP están disponibles en DreamHost?

Ten en cuenta que las versiones de PHP se actualizan continuamente. Visita la siguiente página para ver una línea de tiempo de las versiones de PHP:

  • http://php.net/supported-versions.php

¿Puedo configurar un archivo personalizado php.ini?

Sí. Para obtener detalles sobre cómo configurar esto para su sitio, visita el siguiente artículo:

  • Generalidades de php.ini

¿Dónde puedo ver la configuración predeterminada de PHP de DreamHost?

Puedes crear un archivo llamado phpinfo.php y subirlo a tu sitio para ver tu configuración de PHP. Visita el siguiente artículo para más detalles:

  • Ver tu versión y configuración de PHP

¿Ofrecen el registro de errores de PHP?

Sí. Sin embargo, por defecto, los errores de PHP no se registran. Puedes habilitar y establecer la ruta al registro de PHP con el archivo phprc. Visita el siguiente artículo para más detalles:

  • Registro de errores PHP

  • ¿Qué versión de PHP está usando mi sitio?
