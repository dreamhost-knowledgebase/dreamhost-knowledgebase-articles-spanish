Generalidades

Google requiere la verificación de la propiedad del dominio para algunos de los servicios que ofrecen. Si has registrado tu dominio con DreamHost o administras DNS con DreamHost, hay varias opciones que Google proporciona para verificar tu nombre de dominio con ellos.

La verificación del sitio de Google es diferente de tu número de identificación de seguimiento de Google Analytics. Consulta el siguiente artículo para obtener información sobre cómo agregar tu código de seguimiento de Analytics a tu sitio:

  • Código de seguimiento de Google Analytics

¿Cómo puedo verificar mi dominio con Google?

Google ofrece cuatro opciones diferentes para verificar tu dominio con ellos. Los siguientes métodos se han tomado de la documentación de Google sobre verificación de dominio:

  1. Agrega una metaetiqueta a tu página de inicio (lo que demuestra que tienes acceso a los archivos de origen). Para utilizar este método, debes poder editar el código HTML de las páginas de tu sitio.
  2. Sube un archivo HTML de Google a tu servidor. Para utilizar este método, debes poder cargar nuevos archivos a tu servidor.
  3. Verifica a través de tu proveedor de nombre de dominio. Para utilizar este método, debes poder iniciar sesión en tu proveedor de nombres de dominio (por ejemplo: DreamHost) y agregar un nuevo registro DNS.
  4. Agrega el código de Google Analytics que usas para rastrear tu sitio. Para usar esta opción, debes ser un administrador en la cuenta de Google Analytics, y el código de seguimiento debe usar el nuevo fragmento asincrónico.

Métodos de verificación

Método 1: agregar una metaetiqueta a tu página de inicio

Este método requiere que tu:

  • Agregues una línea de código a los archivos de tu sitio, y es un método moderadamente avanzado para verificar tu dominio.
  • Obtengas un plan de alojamiento en tu cuenta DreamHost.

Una vez que localices esta línea de código del panel de Herramientas para webmasters de Google, edita manualmente el código de tu sitio web actual agregando la línea de código de Google para utilizar este método de verificación.

Para revisar más información, haz clic en el siguiente enlace de Google para obtener más instrucciones:

  • https://support.google.com/a/answer/186017

Método 2: subir un archivo HTML

Este método requiere que tu:

  • Agregues un plan de alojamiento en tu cuenta DreamHost.
  • Descargues un archivo HTML del panel de Herramientas para webmasters de Google y lo subas a través de FTP a tu servidor DreamHost.
  • Cargues manualmente este archivo.

Es la forma más rápida de verificar tu dominio con Google.

Para revisar más información, haz clic en el siguiente enlace de Google para obtener más instrucciones:

  • https://support.google.com/a/answer/63026

Método 3: Verificación a través de tu proveedor de nombre de dominio

Este método requiere que tu dominio esté usando los nameservers, de DreamHost, pero no necesitas tener un plan de alojamiento activo para verificar tu dominio usando este método.

Google te proporcionará la opción de utilizar un registro TXT o CNAME  para agregar a tu dominio en tu panel de DreamHost. Usar el registro TXT es la opción más simple en este caso.

El registro TXT es similar a "google-site-verification = 1234-abcd-5678-EFGH". Sigue estos pasos para agregar este registro a tu dominio en tu panel:

  1. Navega a la página Manage Domains.
  2. Encuentra tu nombre de dominio en esta página y luego haz clic en el enlace ‘DNS’ directamente debajo del nombre de dominio.
    Cuando la página 'Add a custom DNS record...' se abra, ingresa tu código de verificación TXT:
  3. Haz clic en el botón Add Record Now! Para guardar este registro TXT.

También puedes verificar utilizando un registro CNAME, sin embargo, es un poco más avanzado. Google te proporcionará dos elementos si eliges la opción de verificación CNAME:

  • CNAME Label / Host: 1234abcd.example.com (ejemplo)
  • CNAME Destination / Target: ab-cdefghijk.dv.googlehosted.com (ejemplo)

Una vez que agregues un registro TXT o CNAME, Google puede demorar hasta 72 horas en verificar el nuevo registro DNS en su sistema. Google también recomienda que dejes este registro en el dominio, incluso después de que se haya verificado.

Método 4: Agrega el código de Google Analytics que usas para rastrear tu sitio

Este método requiere que tengas una cuenta de Google Analytics. Para usar esta opción, debes ser administrador de la cuenta de Google Analytics y el código de seguimiento debe usar el nuevo fragmento asincrónico.

Para revisar más información sobre Google Analytics, haz clic en el siguiente enlace:

  • http://www.google.com/analytics/
