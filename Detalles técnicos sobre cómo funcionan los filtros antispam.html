Registros MX

El filtrado antispam se habilita cambiando los registros MX del dominio para enrutar los correos  entrantes a los servidores de filtrado primero antes de pasar el mensaje a las máquinas MX para su entrega. Los dominios que utilicen nameservers que no sean DreamHost deberás actualizar manualmente tus registros MX para reflejar el cambio.

Los registros de filtro son:

  • 0 mx1.mailchannels.net
  • 0 mx2.mailchannels.net

Consideraciones especiales para direcciones de reenvío

Hay ciertas cosas para recordar con el filtrado en relación con el reenvío de direcciones de correo. Los detalles de las cuatro combinaciones con esta configuración se describen a continuación:

  • Si se filtra el dominio y la dirección se reenvía a otra dirección de correo de DreamHost que también se filtra, el correo se clasifica según la configuración de correo del filtro de la dirección de reenvío original.
  • Si se está filtrando el dominio y la dirección se reenvía a otra dirección de correo de DreamHost que no está filtrada, se filtrará el correo.
  • Si se está filtrando el dominio y la dirección se reenvía a una dirección externa únicamente (como @gmail), se utilizará la configuración del filtro para la dirección externa.
  • Si el dominio no se filtra y se reenvía a una dirección de correo electrónico de DreamHost que se encuentra en un dominio filtrado, el correo no se filtrará.

El filtrado y lo qué hace

Actualmente, hay tres servicios configurados para filtrar el spam. A continuación se muestra la descripción de cada una:

Vade Secure (pronto será eliminado)
Este es el servicio de filtro que escanea los mensajes entrantes en busca de spam y otras categorías, como correos sociales o promocionales. Dependiendo de múltiples factores, el mensaje se etiquetará con un estado (SPAM, SN, etc.) para determinar si el mensaje debe enviarse a la bandeja de entrada del destinatario o filtrarse a la carpeta Spam. Sin embargo, se rechazarán los correos no deseados que superen un cierto umbral.
Procmail
Este es el agente de entrega del buzón que realmente coloca el correo en las carpetas Spam, Social o Promociones cuando detecta los encabezados de Vade Secure. Esto también es responsable de los filtros definidos por el usuario definidos en el panel.
Mailchannels (habilitado desde Sep 2020)
Mailchannels es un filtro de correo basado en la nube. Acepta y evalúa el correo, agrega un encabezado específico para los mensajes de spam y luego transmite los mensajes a los servidores MX.
