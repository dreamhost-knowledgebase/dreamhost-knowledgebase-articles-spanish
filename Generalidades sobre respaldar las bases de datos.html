Generalidades

Este artículo ilustra cómo hacer una copia de seguridad de tus bases de datos MySQL alojadas en DreamHost.

DreamHost proporciona un puñado de métodos de copias de seguridad para bases de datos MySQL, con acciones que van desde extremadamente simples hasta ligeramente técnicas. Las herramientas ofrecidas incluyen el panel DreamHost, la interfaz phpMyAdmin proporcionada, y el uso de la línea de comandos del servidor para generar una copia de seguridad disponible para descarga vía FTP.

Copia de seguridad con la herramienta desde el panel

La herramienta de copia de seguridad de la cuenta se encuentra en tu página Administrar Cuenta. Este descarga todos los datos de tu cuenta, incluidos todos los usuarios, cuentas de correo y bases de datos (también se capturan los procedimientos almacenados de MySQL), en una serie de archivos .zip.

phpMyAdmin es una herramienta de administración de bases de datos MySQL que proporciona una interfaz de usuario (relativamente) fácil de usar a través de un navegador web. Consulta el siguiente artículo para obtener instrucciones sobre cómo hacer una copia de seguridad de tu base de datos:
