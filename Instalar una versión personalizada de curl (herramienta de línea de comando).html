Generalidades

curl se instala en todos los servidores DreamHost de forma predeterminada; sin embargo, es posible que desees instalar una versión más nueva según tus necesidades. Este artículo te guía a través de la instalación de una versión personalizada de curl (herramienta de línea de comandos).

Estas instrucciones explican cómo instalar la herramienta de línea de comando curl que es diferente que cURL. Revisa el siguiente artículo para más información:

  • daniel.haxx.se/docs/curl-vs-libcurl.html

Instalar curl

  1. Inicia sesión en tu servidor por vía SSH.
  2. Asegúrate de estar en el directorio principal de tu usuario.
    [server]$ cd ~
  3. Busca aquí la versión que deseas instalar:
    • https://curl.se/download/

    Haz clic derecho en un enlace .zip y copie la URL. Luego puedes ejecutar el siguiente comando wget para descargar el archivo .zip.

    [server]$ wget https://curl.se/download/curl-8.3.0.zip
  4. Descomprime el archivo.
    [server]$ unzip curl-8.3.0.zip
  5. Navega al nuevo directorio.
    [server]$ cd curl-8.3.0
  6. Corre el siguiente comando para instalarlo:

    curl requiere que escojas cual librería de SSL prefieres usar cuando lo construyas. Estas instrucciones usan openssl. Revisa SELECT YOUR TLS para más detalles.

    [server]$ ./configure --prefix=$HOME/curl --with-openssl
[server]$ make
[server]$ make install
  7. Abre tu .bash_profile usando nano o vim. Por ejemplo: 
    nano ~/.bash_profile
  8. Agrega esta línea a tu .bash_profile: 
    export PATH=$HOME/curl/bin:$PATH
  9. Activar tu configuración nueva.
    [server]$ . ~/.bash_profile
  10. Verifica la ubicación y la versión de curl:
    [server]$ which curl
/home/username/curl/bin/curl
    
[server]$ curl --version
curl 8.3.0 (x86_64-pc-linux-gnu)
