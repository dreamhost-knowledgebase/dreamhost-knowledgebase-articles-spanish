Generalidades

Después de que hayas creado una Llave de Emparejamiento y lanzar una instancia, puedes saber ahora cómo conectar tu nueva instancia de DreamCompute usando esta Llave de Emparejamiento. Sin embargo, el usuario predeterminado no tiene establecida una contraseña. Solo usa esta Llave de Emparejamiento para iniciar sesión.

Una vez que hayas iniciado sesión, continúa con los siguientes pasos para crear una contraseña para el usuario predeterminado.

El siguiente ejemplo se ejecuta en una instancia de Ubuntu con el usuario predeterminado llamado 'ubuntu'.

Configurar la contraseña

  1. Cambiar al usuario root
    ubuntu@instance:~$ sudo su
root@instance:/home/ubuntu#
  2. Crea una nueva contraseña para el usuario predeterminado:
    root@instance:/home/ubuntu# passwd ubuntu
Enter new UNIX password:
Retype new UNIX password:
passwd: password updated successfully
  3. El usuario predeterminado ahora tiene una contraseña. Si alguna vez necesitas iniciar sesión en la instancia con esta contraseña, debes hacerlo desde el panel de DreamCompute.

Cómo iniciar sesión en tu instancia a través de la consola del panel de DreamCompute

  1. Inicia sesión en tu panel de DreamCompute.
  2. A la izquierda, haz clic en Instances y luego haz clic en el nombre de la instancia en la que deseas iniciar sesión.
  3. Haz clic en la pestaña Console.
  4. Ingresa el nombre de usuario predeterminado.
  5. Luego ingresa la contraseña que acabas de crear. Esto te registra en la instancia.
