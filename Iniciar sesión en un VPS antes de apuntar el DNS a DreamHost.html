Generalidades

Si deseas acceder al servidor para realizar configuraciones antes de mover tu dominio, puedes usar la siguiente forma de nombre de host en tu cliente FTP o SSH.

  • psXXXXX.dreamhostps.com

donde psXXXXX es el nombre de servidor de tu VPS.
