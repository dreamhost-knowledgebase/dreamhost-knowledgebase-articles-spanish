Generalidades

Este artículo explica qué es Wget y cómo utilizar esta herramienta en DreamHost.

Requisitos previos

Este artículo asume que has creado un usuario de Shell y ha iniciado sesión en tu servidor a través de SSH.

Antecedentes

Wget es un programa que recupera contenido de servidores web. Su nombre hace referencia a la World Wide Web y a Get.

Cuándo usar Wget

Wget se utiliza a menudo después de iniciar sesión en tu servidor a través de SSH para descargar archivos desde un servidor remoto a tu servidor de DreamHost. Esta es una opción mucho más rápida y más sencilla que descargar manualmente los archivos a través de FTP desde la fuente y luego subirlos a tu servidor.

rsync puede ser una opción mejor (más rápida y menos complicada) para los usuarios que mueven datos entre dos servidores (como mover archivos de un alojamiento Shared a un alojamiento VPS).

Cómo usar Wget

Puedes usar el comando Wget para ejecutar tareas simples y avanzadas en tu servidor.

Ver el manual

Para ver la página del manual de Wget, corre lo siguiente en tu terminal:

[server]$ man wget

Descargar el archivo

Para descargar un archivo en tu servidor:

Escribe wget seguido de la URL completa del archivo que desea descargar. Por ejemplo, corre el siguiente comando para descargar el archivo .tar.gz para la versión 3.8.1 de Python:

[server]$ wget https://www.python.org/ftp/python/3.8.1/Python-3.8.1.tgz
  • Esto descarga el archivo .tgz al directorio donde ejecutaste el comando.
  • Si el archivo que descargas está comprimido, descomprime el archivo usando gunzip, unzip, o tar para expandir y descomprimir la descarga.

Si necesitas pasar variables a un script, encierra la URL entre comillas simples, lo que evita que el carácter & se interprete como el comando de shell:

[server]$ wget 'https://www.example.com/myscript.php?var1=foo&var2=bar'
