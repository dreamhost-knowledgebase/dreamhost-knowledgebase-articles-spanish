Generalidades

La siguiente información describe cómo realizar una consulta WHOIS para un registro de dominio. Una consulta whois muestra información sobre un dominio, como dónde está registrado y a dónde apuntan tus nameservers.

Cómo hacer una consulta WHOIS

Hay dos opciones para realizar una consulta WHOIS.

Opción 1 — Usando una herramienta online

La forma más sencilla de buscar información sobre el registro de un dominio es usar una herramienta en línea de la siguiente lista:

  • lookup.icann.org
  • whois.com/whois
  • whois.domaintools.com
  • mxtoolbox.com/Whois

Opción 2 — Corriendo una consulta en la línea de comando

Para ejecutar una consulta whois desde la línea de comandos, debes iniciar sesión en un terminal a través de SSH. Esto puede ser un terminal en tu computador o un servidor de DreamHost.

Abre un terminal y corre el siguiente comando.

[server]$ whois example.com

Para algunas extensiones de dominio, es posible que necesites consultar el registro directamente en el servidor WHOIS específico de ese dominio. Sin embargo, no existe un estándar actual para determinar el servidor WHOIS responsable para una extensión de dominio. Algunas búsquedas con whois requieren trabajo adicional, como realizar la búsqueda a través del registrador específico del nombre de dominio para obtener detalles de propiedad. Puedes leer más sobre esto aquí.

En la mayoría de los casos, es más fácil usar un sitio de WHOIS de terceros para realizar tus consultas whois.

Entender una respuesta de consulta WHOIS

Hay dos partes en una consulta whois:

  • La información del registro central
  • La información del registrador

La información del registro central

Domain Name: DREAMHOST.COM
Registrar: DREAMHOST, LLC
Sponsoring Registrar IANA ID: 431
Whois Server: whois.dreamhost.com
Referral URL: http://www.dreamhost.com
Name Server: NS1.DREAMHOST.COM
Name Server: NS2.DREAMHOST.COM
Name Server: NS3.DREAMHOST.COM
Status: clientTransferProhibited
Updated Date: 27-aug-2014
Creation Date: 23-sep-1997
Expiration Date: 22-sep-2015

La información del registro central muestra el nombre del dominio en cuestión, la compañía que proporcionó los servicios de registro, un enlace a su servidor WHOIS, el estado del dominio, los nameservers y las fechas de creación/expiración.

La información del registrador

Después de que tu cliente recupere la información del registro central, se conectará al servidor del registrador para buscar la información de contacto.

La información en el registro central se actualiza aproximadamente cada 24 horas, por lo que si ves información en la sección WHOIS del registrador que es diferente a la del registro central, inténtalo de nuevo en aproximadamente 24 horas.
