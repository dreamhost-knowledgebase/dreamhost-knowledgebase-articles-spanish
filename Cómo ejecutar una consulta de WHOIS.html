Generalidades

Una consulta de whois muestra información sobre dónde está registrado tu sitio web. También muestra información en tus name servers que muestra dónde está alojado tu sitio.

Usando una herramienta en línea

Esta es la forma más sencilla de buscar el registro de dominio y la información de propiedad almacenada en la base de datos pública. Puedes usar sitios web como http://whois-search.com para verificar los registros de WHOIS actuales para dominios.

Ingresando una consulta WHOIS desde la línea de comando

También puedes realizar una consulta de whois desde tu computadora si tu sistema operativo lo admite. Abre una terminal en tu computadora y ejecuta una consulta de whois desde una línea de comando. Por ejemplo:

[server]$ whois example.com

Para algunas extensiones de dominio, es posible que deba verificar el registro directamente en el servidor de WHOIS de ese dominio específico. Sin embargo, no existe un estándar actual para determinar el servidor WHOIS responsable de una extensión de dominio. Algunas búsquedas de whois requieren un trabajo adicional, como realizar la búsqueda a través del registrador específico del nombre de dominio para recuperar los detalles de propiedad.

Puedes leer más sobre esto en:

  • Artículo de Wikipedia sobre Descubrir Servidores en WHOIS

En la mayoría de los casos, es más fácil usar un sitio de búsqueda de WHOIS de terceros para realizar sus consultas de búsqueda de whois.

Desglosar una consulta de WHOIS

Hay dos partes en una consulta de whois:

  • La información del registro central
  • La información del registrador

La información del registro central

Domain Name: DREAMHOST.COM
Registrar: DREAMHOST, LLC
Sponsoring Registrar IANA ID: 431
Whois Server: whois.dreamhost.com
Referral URL: http://www.dreamhost.com
Name Server: NS1.DREAMHOST.COM
Name Server: NS2.DREAMHOST.COM
Name Server: NS3.DREAMHOST.COM
Status: clientTransferProhibited
Updated Date: 27-aug-2014
Creation Date: 23-sep-1997
Expiration Date: 22-sep-2015

La información del registro central muestra el nombre de dominio en cuestión, qué compañía proporcionó servicios de registro, un enlace a su servidor WHOIS, el estado del dominio, los, nameservers, y las fechas de creación / caducidad. Una vez que tengas esta información, tu cliente de WHOIS generalmente se conectará al servidor del registrador para buscar la información de contacto. La información en el registro central se actualiza aproximadamente cada 24 horas, por lo que si ves información en la sección de registro de WHOIS que es diferente al registro central, intenta nuevamente en aproximadamente 24 horas.
