El siguiente artículo describe cómo instalar WordPress desde tu panel de DreamHost. También detalla las características adicionales que recibes si eliges la instalación gratuita 'Deluxe'.

Antes de instalar

Antes de instalar WordPress, verifica qué archivos ya están en el directorio de tu sitio web.

Si hay archivos de WordPress existentes

Si hay archivos de WordPress existentes, se les cambiará el nombre con una cadena de números al final. Por ejemplo:

  • index.php.1353108851

Si hay archivos que no son de WordPress en tu directorio

Si hay archivos o directorios que no sean de WordPress, la instalación no los tocará. Los nuevos archivos de WordPress se instalarán junto con tus archivos existentes.

Instalar WordPress

  1. Para instalar WordPress en DreamHost, inicia sesión en tu panel y ve a la página de One-Click Installs.
  2. De la lista de instalaciones con un clic, selecciona 'WordPress'.
  3. Introduce lo siguiente:
    • Install to: Selecciona el dominio en el que deseas instalarlo. El segundo cuadro de texto es si deseas instalarlo en un subdirectorio (si no deseas que esté almacenado en el directorio principal). Por ejemplo, si deseas que se instale en example.com/blog, primero seleccione example.com en el primer menú desplegable. En el segundo cuadro de texto escribe blog.
    • Select Database: Probablemente desees 'Crear automáticamente base de datos', ya que esto crea una nueva base de datos única para tu sitio. También puedes elegir una base de datos existente si lo deseas.
    • Deluxe Install: Se recomienda marcar esta casilla si deseas las funciones adicionales que se mencionan a continuación. Esta es una adición gratuita a tu instalación.
  4. Haz clic en el botón Install it for me now!. En los próximos 15 minutos, se te enviará un correo electrónico al correo electrónico principal que figura en tu cuenta de DreamHost. Este correo electrónico detalla los pasos a seguir para completar la instalación.

Cómo iniciar sesión en tu sitio de WordPress

Después de instalar WordPress, recibirás un correo a la dirección de correo electrónico principal en tu cuenta de DreamHost. Este correo electrónico enumera los pasos a seguir para completar la instalación.

Cuando WordPress finalmente está instalado en tu sitio, puedes iniciar sesión en él en:

  • example.com/wp-login.php

La instalación Deluxe

La instalación Deluxe gratuita incluye varias personalizaciones importantes para facilitar el uso de usuario principiantes. Incluye temas gratuitos y de código abierto de Automattic, una selección estándar de plugins y una mejora de la seguridad.

Temas

Las instalaciones Deluxe incluyen la siguiente selección de temas gratuitos y de código abierto de Automattic:

  • argent
  • button
  • dyad
  • illustratr
  • kelly
  • libre
  • orvis
  • resonar
  • scrawl
  • sketch
  • twentyfifteen (También presente en instalaciones normales)
  • twentyseventeen (También presente en instalaciones normales)
  • twentysixteen (También presente en instalaciones normales)

Plugins

La instalación de un solo clic de DreamHost también incluye los siguientes plugins populares:

  • Akismet (También presente en instalaciones normales)
  • Hello Dolly
  • Jetpack
  • WP Super Cache (habilitado automáticamente por defecto)
  • WPForms Lite (habilitado automáticamente por defecto)

Aunque estos plugins se instalan de manera predeterminada para las instalaciones Deluxe, es posible que necesites activarlos en tu panel de control de WordPress visitando 'Plugins '> 'Plugins instalados'.

Mejora de seguridad

Además, la instalación de WordPress agrega protección contra la ejecución de scripts dentro de tu directorio de cargas.

En términos más técnicos, esto significa que un archivo .htaccess con los siguientes contenidos se agrega automáticamente a tu directorio /wp-content/uploads/:

### Generated by DreamHost. DO NOT modify!!! ###
SetHandler no-handler
################################################

Si tu sitio se ve comprometido por la colocación de un script malicioso en tu directorio de subidas, el script se devuelve como una descarga cuando se accede. Esto evita que el script se ejecute y ayuda a evitar daños mayores.

DreamHost continuará agregando mejoras de seguridad adicionales a medida que estén disponibles.

Problemas de instalación

Si la instalación con un clic no se completa en 20 minutos o falla con un error, comunícate con Soprte.

Además, si la instalación se completa pero no recibes el correo electrónico con las instrucciones sobre cómo acceder a tu nueva instalación de WordPress, siempre puedes consultar tus Mensajes recientes ("Recent Messages") en la página Contact Support, y así obtener una copia archivada de ese correo electrónico.
