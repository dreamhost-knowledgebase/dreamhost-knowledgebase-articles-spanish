Generalidades

Con el CLI de OpenStack, puedes generar una nueva Llave Par o importar una llave pública existente. Este artículo asume que ya has instalado el CLI de OpenStack en tu computadora:

  • Iniciando con el CLI de OpenStack

Crear una Llave Par y añadirla a DreamCompute

Para generar una Llave Par, ejecuta el siguiente comando.

[user@localhost]$ openstack keypair create KEY_NAME > MY_KEY.pem

Este comando genera una Llave Par con los nombres que especificaste y los escribe en un archivo .pem. Luego, la Llave Par se registra en el panel de DreamCompute. Puedes ver la lista en la página Compute > Key Pairs.

También puedes ver si está corriendo ejecutando el siguiente comando:

[user@localhost]$ openstack keypair list
 

Importar una Llave Par existente

Si ya has generado una Llave Par en tu computador Linux o Mac, lo más probable es que la llave pública sea llamada id_rsa.pub y ubicada en el directorio: ~/.ssh/.
Ejecuta el siguiente comando para generar la nueva Llave Par en tu panel de DreamCompute.

El carácter de barra invertida \ se usa al ingresar un comando largo. Cuando ingresas \, el comando de terminal continúa a la siguiente línea hasta que termines de ingresarlo.

También puedes eliminar el carácter \ y poner cada parámetro en una sola línea.

[user@localhost]$ openstack keypair create --public-key \
~/.ssh/id_rsa.pub MY_KEY_NAME
+-------------+-------------------------------------------------+
| Field       | Value                                           |
+-------------+-------------------------------------------------+
| fingerprint | f2:fc:4f:ca:1c:b7:ba:43:1f:b8:e5:fc:63:84:79:13 |
| name        | My_KEY_NAME                                     |
| user_id     | 29815315c4ce4d48a27e767a63eb8320                |
+-------------+-------------------------------------------------+

Luego, la Llave Par se registra en el panel de DreamCompute. Puedes ver la lista en la página Compute > Key Pairs.

Nombra la nueva Llave Par con el nombre que especifiques para MY_KEY_NAME. Para asegurarte de que la Llave Par se haya importado correctamente, ejecuta el siguiente comando para enumerar las Llaves Par:

[user@localhost]$ openstack keypair list
