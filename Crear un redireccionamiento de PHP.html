Generalidades

Hay varias formas de redireccionar una URL. Esto es útil para las solicitudes de tu dominio principal a otro directorio bajo ese dominio. Como dirigir las solicitudes de example.com a example.com/blog. Esto se puede hacer usando un archivo .htaccess. Sin embargo, también es posible redireccionar usando PHP.

Este artículo da un ejemplo rápido de creación de una redirección usando PHP.

Crear una redirección simple en PHP

  1. Crea un archivo llamado index.php. Puedes hacer esto usando un cliente FTP como Filezilla o a través de SSH. Visita los siguientes artículos para obtener instrucciones sobre cómo crearlo (lo cual depende si estás utilizando un cliente FTP o SSH):
    • Crear y editar un archivo vía FTP
    • Crear y editar un archivo vía SSH
  2. Asegúrate de que el archivo está en el directorio del dominio. Este directorio es el que se especificas al agregar el dominio a tu panel. Revisa el artículo Cambiar el directorio web asignado a un dominio para ver instrucciones sobre cómo verlo.
  3. En este archivo index.php, agrega la siguiente línea de código:
    <?php header("Location: https://www.example.com/blog"); ?>

Recuerda cambiar la URL de muestra a la URL que quieres que el sitio redireccione.

Luego, tendrás una redirección simple y rápida que puedes usar para redirigir sin problemas una solicitud HTTP a cualquier URL.
