Generalidades

Java es un lenguaje de programación y un conjunto de bibliotecas desarrollado por Sun Microsystems. DreamHost tiene Java instalado de forma predeterminada en todos los servidores, sin embargo, algunas funciones no están disponibles a menos que las configures en un VPS (si no se requiere root/sudo) o un servidor Dedicado.

¿Qué funciones de Java admite DreamHost?

  • Java (en general)
  • Java Applets
  • Java2
  • Java SDK/JRE (Software Dev Kit/Java Runtime Environment)

Subprogramas de Java o applets

Sí. Java applets son aplicaciones que se ejecutan en el navegador del usuario, a diferencia de los programas CGI, que se ejecutan en el servidor.

JavaScript

Sí.

"Java" y "JavaScript" son dos cosas completamente diferentes.

Java es un software que se ejecuta en el servidor.

JavaScript, por otro lado, es un software que se almacena en el servidor, pero se ejecuta en su computadora en su navegador web. JavaScript ha crecido en los últimos años y ahora también se utiliza en el lado del servidor. Más notablemente con Node.js.

¿Qué funciones de Java NO admite DreamHost?

  • Java servlets
  • JSP
  • Tomcat/Jakarta (una maquina JSP)
  • EJB (Beans Java empresariales)
  • J2EE
  • Prácticamente cualquier otra cosa relacionada con Java.

Servlets de Java

Esto no es compatible con los planes Shared hosting. Sin embargo, puedes habilitar esto en un VPS (si no se requiere root/sudo) o un servidor Dedicado.

Si bien es sin duda una tecnología útil, a medida que pasa el tiempo será cada vez más difícil integrarla con otras tecnologías en las que opera DreamHost.

DreamHost ha evaluado de cerca la alternativa más obvia (Jakarta Tomcat), pero debido a una limitación inherente en la forma en que se diseñó Tomcat, no es una alternativa viable cuando se utiliza en un entorno de alojamiento compartido. También se han evaluado varias soluciones comerciales, pero no se ha encontrado que ninguna sea un reemplazo viable y rentable que funcione en todos los servidores de alojamiento compartido.

JSP?

Jakarta Server PagesJakarta Server Pages es un formato para usar Java para generar contenido web de forma dinámica.

DreamHost no ofrece JSP en planes Shared hosting. Sin embargo, puedes habilitar esto en un VPS(si no se requiere root/sudo) o en un servidor Dedicado.

Beans Java empresariales

Enterprise Java Beans, a arquitectura de componentes del lado del servidor para la plataforma J2EEpermite el desarrollo rápido y simplificado de aplicaciones Java distribuidas, transaccionales, seguras y portátiles.

DreamHost no es compatible con Enterprise Java Beans.

J2EE

J2EE (Java 2 Edición Empresarial) NO es compatible con DreamHost.

Motor JSP Tomcat/Jakarta

DreamHost no es compatible con Tomcat/Jakarta Motor JSP activado en planes Shared hosting. Sin embargo, puede habilitar esto en un VPS (si no se requiere root/sudo) o en un servidor Dedicado.
