Generalidades

Dentro de Remixer, puedes crear un hipervínculo que vincule tu texto a otra URL. Esto se puede hacer usando la barra de herramientas incorporada.

Crear el enlace

  1. Resalta el texto que deseas vincular. Aparece una barra de herramientas de edición.
  2. Da clic en el icono de hipervínculo. Se abre la barra de herramientas con cuatro pestañas:
    • URL — Ingresa una URL para vincular a otro sitio web.
    • Anchor — Ingresa una URL para vincular a otro sitio web.
    • Page — Selecciona del menú desplegable para vincular a otra de las páginas de Remixer de tu sitio. Esto supone que has creado un sitio Remixer de varias páginas.
    • Email — Ingresa un correo para vincularlo.
  3. Da clic en el botón Save.
