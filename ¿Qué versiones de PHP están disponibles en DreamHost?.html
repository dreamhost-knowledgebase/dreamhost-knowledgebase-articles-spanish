Versiones de PHP

DreamHost instala previamentelas siguientes versiones de PHP en sus servidores:

Versión de PHP Disponibilidad

PHP 8.3

 Disponible en el panel
PHP 8.2 Disponible en el panel
PHP 8.1 Disponible en el panel
PHP 8.0 No disponible en el panel. Oficialmente End-Of-Lifed por php.net.
PHP 7.4 No disponible en el panel. Oficialmente End-Of-Lifed por php.net.
PHP 7.3 No longer isponible en el panel. Oficialmente End-Of-Lifed by php.net.

PHP 7.1 No disponible en el panel. Oficialmente End-Of-Lifed por php.net.
PHP 7.0 No disponible en el panel. Oficialmente End-Of-Lifed por php.net.
PHP 5.6 No disponible en el panel. Oficialmente End-Of-Lifed por php.net.

Por favor ten en cuenta que las versiones de PHP se actualizan continuamente. Puedes revisar esta página para ver una cronología de las versiones de PHP.

Ver la versión predeterminada de PHP en un servidor web

Para ver la versión predeterminada en el servidor de DreamHost, inicia sesión en tu servidor a través de SSH y corre el comando php -v.

[server]$ php -v

Para utilizar una versión diferente al correr un comando de shell, debes especificar la ruta completa del archivo:

[server]$ /usr/local/php82/bin/php -v
