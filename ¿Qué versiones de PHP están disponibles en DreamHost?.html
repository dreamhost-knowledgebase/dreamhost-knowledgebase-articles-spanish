Generalidades

DreamHost tiene preinstaladas las siguientes versiones de PHP en sus servidores:

  • PHP 7.3.9 — Disponible en el panel.
  • PHP 7.2.22 — Disponible en el panel.
  • PHP 7.1.32 — Disponible en el panel.
  • PHP 7.0.33 — No disponible en el panel. Declarado Fin-de-vida-útil por php.net.
  • PHP 5.6.36 — No disponible en el panel. Declarado Fin-de-vida-útil por php.net.
  • PHP 5.5 — No compatible. Declarado End-Of-Lifed by php.net.
  • PHP 5.4 — No longer supported. Officially Fin-de-vida-útil por php.net.
  • PHP 5.3 — No longer supported. Officially Fin-de-vida-útil por php.net.
  • PHP 5.2 — No longer supported. Officially Fin-de-vida-útil por php.net.

Ten en cuenta que las versiones de PHP se actualizan continuamente. Consulta la siguiente página para ver una línea de tiempo de las versiones de PHP:

  • http://php.net/supported-versions.php

Version predeterminada de PHP en un servidor web compartido

Cuando ejecuta el comando php -v en un terminal shell al iniciar sesión en tu servidor, la salida muestra la versión predeterminada actual utilizada.

[server]$ php -v

Si quieres utilizar una versión de PHP diferente en el Shell, deberás especificar la ruta completa del archivo. Usa lo siguiente para PHP 7.1:

[server]$ /usr/local/php71/bin/php -v
PHP 7.1.19 (cli) (built: Jul 14 2018 02:46:25) ( NTS )
Copyright (c) 1997-2018 The PHP Group
Zend Engine v3.1.0, Copyright (c) 1998-2018 Zend Technologies
with Zend OPcache v7.1.19, Copyright (c) 1999-2018, by Zend Technologies

Estas son las versiones actuales que puede usar a través de SSH:

/usr/local/php56/bin/php
/usr/local/php70/bin/php
/usr/local/php71/bin/php
/usr/local/php72/bin/php
/usr/local/php73/bin/php
