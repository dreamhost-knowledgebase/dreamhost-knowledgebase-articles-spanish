Generalidades

DreamHost instala previamentelas siguientes versiones de PHP en sus servidores:

  • PHP 8.2 — Disponible en el panel
  • PHP8.1 — Disponible en el panel
  • PHP8.0 — Disponible en el panel
  • PHP 7.4 — No disponible en el panel. Oficialmente Fin-de-vida-útil por php.net.
  • PHP 7.3 — No disponible en el panel. Oficialmente Fin-de-vida-útil por php.net.
  • PHP 7.2 — No disponible en el panel. Declarado Fin-de-vida-útil por php.net.
  • PHP 7.1 — No disponible en el panel. Declarado Fin-de-vida-útil por php.net.
  • PHP 7.0 — No disponible en el panel. Declarado Fin-de-vida-útil por php.net.
  • PHP 5.6 — No disponible en el panel. Declarado Fin-de-vida-útil por php.net.

Ten en cuenta que las versiones de PHP se actualizan continuamente. Consulta la siguiente página para ver una línea de tiempo de las versiones de PHP:

  • php.net/supported-versions.php

Version predeterminada de PHP en un servidor web compartido

Cuando corres el comando php -v en un terminal shell al iniciar sesión en tu servidor, la salida muestra la versión predeterminada actual utilizada.

[server]$ php -v

Si deseas usar una versión de PHP diferente en el Shell, deberás especificar la ruta completa del archivo.

[server]$ /usr/local/php82/bin/php -v
PHP 8.2.0 (cli) (built: Dec 11 2022 07:26:39) (NTS)
Copyright (c) The PHP Group
Zend Engine v4.2.0, Copyright (c) Zend Technologies
with Zend OPcache v8.2.0, Copyright (c), by Zend Technologies

Estas son las versiones actuales que puede usar a través de SSH. Las versiones anteriores no están disponibles en algunos servidores.
