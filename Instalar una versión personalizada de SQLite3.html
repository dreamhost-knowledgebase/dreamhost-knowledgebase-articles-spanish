Instalar una versión personalizada de SQLite

SQLite ya está instalado en todos los servidores DreamHost, pero también puedes instalar tu propia versión localmente con tu usuario. Para instalar una versión personalizada:

  1. Inicia sesión en tu servidor a través de SSH.
  2. Asegúrate de estar en el directorio principal de tu usuario: 
    [server]$ cd ~
  3. Visita la página de descarga de SQLite para descargar la más reciente versión.
  4. Descárgalo al directorio de tu usuario usando el comando wget:q2 
    [server]$ wget https://sqlite.org/2020/sqlite-autoconf-3330000.tar.gz
  5. Descomprime el archivo: 
    [server]$ tar zxvf sqlite-autoconf-3330000.tar.gz
  6. Cambia al nuevo directorio. 
    [server]$ cd sqlite-autoconf-3330000
  7. Corre lo siguiente para configurarlo. 
    [server]$ ./configure --prefix=$HOME/opt/sqlite
  8. Ejecuta make y make install para completar la instalación: 
    [server]$ make
[server]$ make install
  9. Agrega las siguientes líneas al .bash_profile. Este un archivo ubicado en el directorio /home de tu usuario. 
    export PATH=$HOME/opt/sqlite/bin:$PATH
    export LD_LIBRARY_PATH=$HOME/opt/sqlite/lib
    export LD_RUN_PATH=$HOME/opt/sqlite/lib
  10. Corre el comando source en tu archivo .bash_profile: 
    [server]$ source .bash_profile
  11. Verifica la versión para confirmar que ahora se está usando: 
    [server]$ which sqlite3
/home/username/opt/sqlite/bin/sqlite3
[server]$ sqlite3
SQLite version 3.33.0 2024-04-02 13:23:32
  12. Sal del shell SQLite corriendo lo siguiente: 
    sqlite> .quit
