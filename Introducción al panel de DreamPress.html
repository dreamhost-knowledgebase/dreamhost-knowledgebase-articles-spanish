Generalidades

A continuación se describe cómo usar las diversas funciones en la página del panel de DreamPress.

¿Todavía no tienes DreamPress?

Puedes actualizar un sitio existente de WordPress a DreamPress utilizando el enlace Upgrade to DreamPress en la página Manage Domains o ir a la página Managed WordPress para comprarlo.

Acceso al panel de DreamPress

Para acceder al Panel de DreamPress, haz clic en el botón Managed WordPress en tu panel.

Cuando se abra la página Managed WordPress, haz clic en el botón Manage a la derecha de tu dominio DreamPress para abrir la página del panel de tu sitio web.

WP-Admin

El Panel de administración de WordPress (wp-admin) es donde puedes agregar y editar contenido y características en tu sitio web de WordPress:

Si el botón Manage WordPress no te conecta automáticamente a tu Panel de WordPress y en su lugar te lleva a una página de inicio de sesión, comunícate con Soporte para obtener ayuda.

Jetpack (Planes Plus y Pro)

Los clientes de DreamPress Plus y Pro tienen el beneficio adicional de acceder a una suscripción GRATUITA al plan de nivel profesional de Jetpack. Los usuarios de Jetpack Professional tienen acceso a cientos de temas de forma gratuita con soporte de temas, así como copias de seguridad externas en tiempo real gratuitas, alojamiento ilimitado de videos fuera del sitio, Elasticsearch y más.

Consulta el siguiente artículo para obtener más información sobre Jetpack:

  • Activación de Jetpack Professional para clientes de DreamPress Plus y DreamPress Pro

Características del panel de DreamPress

Las siguientes cuatro pestañas aparecen en el panel de control, estas te permiten realizar más ajustes en tu sitio DreamPress:

Cada una de las pestañas (Details, Domain, Backups, y Staging) se describe en las siguientes secciones.

Pestaña Details 

La pestaña Details te muestra información sobre tu plan actual de DreamPress:

En la pestaña Details, puedes ver lo siguiente:

  • El plan actual de DreamPress (así como un enlace que te permite editar o eliminar un plan)
  • La fecha de renovación
  • Cuánto almacenamiento ha utilizado tu sitio DreamPress (así como un enlace que te permite aumentar / disminuir el almacenamiento)

También puedes cargar archivos (usando el administrador de archivos) y hacer ajustes a la base de datos en la pestaña Details:

Subir archivos

Haz clic en el botón Manage Files para iniciar sesión automáticamente en el administrador de archivos para acceder a los archivos de tu sitio web. Cambie el selector Mostrar credenciales a 'Activado' para ver sus credenciales FTP.

Consulta el siguiente artículo para obtener más información sobre el administrador de archivos:

  • Usando el administrador de archivos del sitio web

Base de datos (MySQL)

Haz clic en el botón Manage Database para iniciar sesión automáticamente en tu cuenta phpMyAdmin para acceder a los archivos de tu base de datos. Cambia el selector Mostrar credenciales a 'Activado' para ver tus credenciales MySQL.

Consulta el siguiente artículo para obtener más información sobre cómo acceder a tu base de datos DreamPress:

  • Accede a tu base de datos DreamPress usando phpMyAdmin

Pestaña Domain

La pestaña Domain te muestra información sobre tu dominio DreamPress:

Puedes cambiar el nombre de dominio en tu sitio (seleccionando un dominio temporal o un dominio existente). También puedes registrar un nuevo dominio. Consulta el siguiente artículo para obtener más información sobre cómo cambiar un dominio DreamPress:

  • Cambiar el nombre de un dominio DreamPress

Pestaña Backups

La pestaña Backups te permite ver y restaurar fácilmente copias de seguridad anteriores, así como crear una nueva copia de seguridad:

Consulta el siguiente artículo para obtener más información sobre cómo crear y restaurar copias de seguridad:

  • Crear y restaurar copias de seguridad en DreamPress

Pestaña Staging

La pestaña Staging te permite configurar un sitio de prueba, para que puedas probar los cambios en tu sitio DreamPress antes de publicarlo:

La vista previa crea una copia de su sitio en un entorno "sandbox" donde puedes experimentar y obtener una vista preliminar de los cambios sin que afecte lo que ven tus visitantes. Cuando esté listo, envías los cambios a tu sitio en línea con un simple clic.

Consulta el siguiente artículo para obtener más información sobre la vista previa:

  • Cómo usar la Vista Previa en DreamPress
