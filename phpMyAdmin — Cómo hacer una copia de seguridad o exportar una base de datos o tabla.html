Crear una copia de seguridad de una base de datos

Antes de empezar

La siguiente sección está también disponible como video instructivo:

Estas instrucciones explican cómo hacer una copia de seguridad de toda tu base de datos.

  1. Inicia sesión de phpMyAdmin.
  2. Selecciona la base de datos en el panel izquierdo.
  3. Haz clic en la pestaña Export (Exportar) en el panel central superior.
  4. En la página siguiente, debes seleccionar un método de exportación: Quick (Rápido) o Custom (Personalizado).

    Elegir el método 'Rápido' te permite descargar el archivo .sql inmediatamente. El método 'Personalizado' te da más control sobre los datos. Utiliza el método 'Personalizado' solo si estás muy familiarizado con la administración de tablas en SQL.

  5. En el menú desplegable, elije el formato en el que deseas guardar el archivo. SQL es el formato más común.
  6. Haz clic en el botón Go (Ir) para continuar.
  7. Aparece una ventana emergente que te permite descargar el archivo. (Nota: Es posible que tu navegador lo descargue automáticamente.)

Exportar una sola tabla

Antes de empezar

La siguiente sección está también disponible como video instructivo:

Estas instrucciones explican cómo exportar una sola tabla dentro de tu base de datos.

  1. Inicia sesión en phpMyAdmin.
  2. Selecciona la base de datos en el panel izquierdo.
  3. Selecciona la tabla que deseas exportar en el panel izquierdo, debajo del nombre de la base de datos.
  4. Haz clic en la pestaña Export en el panel central superior.
  5. En la página siguiente, debes seleccionar un método de exportación: Quick o Custom.

    Elegir el método 'Rápido' te permite descargar el archivo .sql inmediatamente. El método 'Personalizado' te da más control sobre los datos. Utiliza el método 'Personalizado' solo si estás muy familiarizado con la administración de tablas en SQL.

  6. En el menú desplegable, elije el formato en el que deseas guardar el archivo. SQL es el formato más común.
  7. Haz clic en el botón Go para continuar.
  8. Aparece una ventana emergente que te permite descargar el archivo. (Nota: Es posible que tu navegador lo descargue automáticamente).
