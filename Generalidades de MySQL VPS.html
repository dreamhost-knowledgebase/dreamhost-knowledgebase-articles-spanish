¿Qué es MySQL VPS?

DreamHost MySQL VPS es el servicio premium de base de datos MySQL compartido de DreamHost. Similar en concepto a DreamHost VPS, DreamHost PS MySQL utiliza Linux-VServer para ofrecer un servidor de base de datos aislado.

Esto es diferente del servidor MySQL compartido estándar que una cuenta usa de manera predeterminada, ya que el servidor MySQL compartido comparte las bases de datos y recursos de otros usuarios. Con PS MySQL, sus bases de datos y recursos están protegidos de otros usuarios de la misma manera que un servidor privado web.

¿Está disponible el acceso de root?

Un root y acceso SSH en un MySQL VPS no está disponible. Debes actualizar a un servidor Dedicado donde la instancia de MySQL está en la misma máquina, lo que permite el acceso a los archivos de configuración de MySQL.
