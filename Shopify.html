Generalidades

Shopify es un creador de sitios que se especializa en soluciones de sitios web de comercio electrónico. Puedes usarlo para crear tu sitio web minorista para vender productos a sus clientes. Ver más sobre Shopify en el siguiente enlace:

  • http://www.shopify.com/about

Apuntando tu sitio DreamHost a Shopify

Si tu dominio está registrado en DreamHost, puedes apuntar tu DNS a Shopify.

En la siguiente captura de pantalla, se usa el dominio example.com. Tu dominio se mostrará en su lugar al configurar este registro.

  1. Configura su dominio en su panel para DNS Only.
  2. Ve el siguiente enlace para configurar su sitio en Shopify. Este paso proporciona tu ID de tienda Shopify:
    • Configurar tu dominio
  3. En su panel DreamHost, crea un 'registro A' para apuntar a Shopify.
    Name (dejar en blanco)
    Type A
    Value 23.227.38.32

  4. Crea un registro CNAME.
    Name www
    Type CNAME
    Value shops.myshopify.com

Tu sitio ahora apunta a Shopify.

Es posible que veas el siguiente error al agregar el CNAME:

You already have a record for this name. You can't have a CNAME and any other record on the same name.

Esto se muestra porque no configuraste su dominio en DNS Only primero. Ve el siguiente artículo para configurarlo como DNS Only. Entonces podrás agregar este registro.

  • Configurar tu dominio a DNS solamente

Apuntar un subdominio a Shopify

Si solo deseas que un subdominio como shop.example.com apunte a Shopify, solo necesitas crear un registro CNAME como se muestra arriba. El registro 'www' es técnicamente un subdominio, por lo que puede cambiar el 'www' a 'comprar' o cualquiera que sea el nombre de tu subdominio. Ve el siguiente artículo para más detalles:

  • Configurar tu subdominio en Shopify
