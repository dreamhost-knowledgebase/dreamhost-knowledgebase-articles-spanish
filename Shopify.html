Generalidades

Shopify es un creador de sitios que se especializa en soluciones de sitios web de comercio electrónico. Puedes usarlo para crear tu sitio web minorista para vender productos a sus clientes. Ver más sobre Shopify en el siguiente enlace:

  • www.shopify.com/about

Apuntar tu sitio DreamHost a Shopify

Si tu dominio está registrado en DreamHost, puedes apuntar tu DNS a Shopify.

En la siguiente captura de pantalla, se usa el dominio example.com. Tu dominio se mostrará en su lugar al configurar este registro.

  1. Configura tu dominio en tu panel como Solo DNS.
  2. Mira el siguiente enlace para configurar tu sitio en Shopify. Este paso proporciona tu ID de tienda Shopify:
    • Configurar tu dominio
  3. En tu panel DreamHost, crea un 'registro A' para apuntar a Shopify.
    Host (dejar en blanco)
    Apunta a 23.227.38.65

  4. Crea un registro CNAME.
    Host www
    Apunta a shops.myshopify.com

Tu sitio ahora apunta a Shopify.

Es posible que veas el siguiente error al agregar el CNAME:

El registro no puede coexistir con otro registro.

Esto se muestra porque no configuraste tu dominio en DNS Only primero. Revisa el siguiente artículo para configurarlo como Solo DNS. Entonces podrás agregar este registro.

  • Configurar tu dominio como Solo DNS

Apuntar un subdominio a Shopify

Si solo deseas que un subdominio como shop.example.com apunte a Shopify, solo necesitas crear un registro CNAME como se muestra arriba. El registro 'www' es técnicamente un subdominio, por lo que puede cambiar el 'www' a 'comprar' o cualquiera que sea el nombre de tu subdominio. Mira el siguiente artículo para más detalles:

  • Configurar tu subdominio en Shopify
