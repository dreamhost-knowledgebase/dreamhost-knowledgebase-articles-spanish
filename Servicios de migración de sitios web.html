Generalidades

DreamHost ofrece una variedad de servicios de migración de sitios web, como pasar de tu proveedor de alojamiento actual a DreamHost o mover un sitio dentro de DreamHost entre cuentas.

A continuación se describen las diversas opciones de migración a considerar para sitios WordPress y los que no lo son. También puedes contactar a soporte para analizar la opción que mejor se adapte a tí. Si actualmente no eres cliente y deseas migrar tu sitio de WordPress, puedes seleccionar esa opción durante el proceso de registro.

Servicios de migración de WordPress

Migrar WordPress desde tu proveedor de alojamiento actual

DreamHost puede mover tu sitio de WordPress a tu cuenta de DreamHost moviendo todos tus datos a un nuevo servidor, todo sin interrumpir a tus usuarios existentes. Visita la siguiente página para obtener más información sobre el servicio de transferencia de sitios web:

  • Servicio de Transferencia de Sitios Web

Precios

Si deseas migrar desde tu proveedor de alojamiento actual, otra cuenta DreamHost u otro TLD (por ejemplo, example.net a example.com en la misma cuenta):

  • $99 — Migración a un plan de alojamiento estándar, por sitio.
  • Gratis — Migrar a cualquier plan de DreamPress.

Limitaciones

  • El servicio de migración de WordPress no puede mover sitios de red de WordPress (multisitio) en este momento.
  • Si tu sitio está alojado en WordPress.com, solo está disponible una exportación de tus páginas, publicaciones e imágenes incrustadas. Deberás configurar tu propio tema, plugins y la mayoría de los elementos de diseño, como widgets y menús nuevamente.

Mover WordPress entre Cuentas DreamHost

El equipo de soporte técnico de DreamHost puede mover tu sitio de WordPress de una cuenta DreamHost existente a una cuenta DreamHost existente moviendo todos tus datos a un nuevo servidor, todo sin ninguna interrupción para tus usuarios existentes. Para obtener más información, revisa el siguiente artículo:

  • Servicio de Content Move de WordPress

Precios

Si deseas migrar desde otra Cuenta DreamHost:

  • $25 — Migración a un plan de alojamiento estándar, por sitio.
  • Gratuito — Migración a cualquier plan DreamPress.

Limitaciones

  • El servicio de Migración de WordPress no puede mover sitios de WordPress Network (multisitio) en este momento.
  • Como no es posible alojar el mismo dominio en dos cuentas simultáneamente, habrá un poco de tiempo de inactividad (generalmente menos de 5 minutos) mientras DreamHost hace el cambio. Si usas nameservers externos, deberás actualizar los registros A de tu dominio para que sean compatibles con la nueva IP.

Cambiar las URL del sitio WordPress

Si puedes mover tu sitio WordPress a un nuevo dominio en el panel y si este fue instalado usando el instalador de WordPress de DreamHost o DreamPress. Para obtener más información, revisa los siguientes artículos:

  • Cambiar el dominio de un sitio de WordPress en el panel
  • Cambiar el nombre de un dominio DreamPress

DreamPress es un plan de alojamiento administrado de WordPress que DreamHost ofrece como una actualización a un plan de alojamiento estándar.

Si has migrado recientemente tu sitio a un nuevo dominio, has cambiado las URL por tu cuenta o has agregado un certificado SSL a un dominio que anteriormente no usabas uno, deberás cambiar manualmente las URL de tu sitio de WordPress.

DreamHost puede cambiar las URL por ti por un cargo único de $19,99. Para enviar una solicitud de cambio de URL de WordPress, visita la página Contactar a Soporte en el panel e incluya el dominio/URL desde el que estás cambiando y el nuevo dominio/URL que te gustaría usar. El equipo de soporte de DreamHost responderá confirmando el cargo de $19,99 y el trabajo que se está realizando.

Este servicio es gratuito para los sitios de DreamPress.

Servicios de migración de sitios que no son WordPress

Migrar sitios que no son de WordPress desde otra cuenta de DreamHost

Si te encuentras en una situación que requiere cambios en tu sitio y tu desarrollador no está disponible para ayudarte, el equipo de soporte calificado de DreamHost puede ayudarte por una pequeña tarifa.

Puedes encontrar más información sobre los servicios soportados y costos asociados en el artículo Generalidades sobre Servicios Profesionales.

Si tu sitio no está construido en WordPress, DreamHost aún puede ayudarte a migrar tu sitio. El costo de este servicio es de $99 y se reembolsará si DreamHost no puede migrar tu contenido con éxito. Envía un ticket en la página de Contactar a Soporte en el panel e incluye cualquier información que tengas sobre tu sitio. Un representante de migración de DreamHost estará encantado de investigar esto por tí.

Limitaciones

  • Los servicios de alojamiento y constructores de sitios propietarios como Wix, Weebly o Squarespace no se pueden migrar a DreamHost.
  • Los sitios web de "Páginas de servidor activo" (ASP) basados en Windows Server no se pueden migrar a DreamHost.
