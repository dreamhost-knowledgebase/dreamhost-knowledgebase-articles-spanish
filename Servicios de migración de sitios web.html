Generalidades

DreamHost ofrece una variedad de servicios de migración de sitios web como por ejemplo, pasar de un host anterior a DreamHost o mover un sitio dentro de una cuenta DreamHost a otra.

A continuación se describen las diversas opciones de migración a tener en cuenta. También puedes ponerte en contacto con soporte para analizar la opción que mejor se adapte a tus necesidades. Si actualmente no eres cliente y deseas migrar tu sitio de WordPress, puedes seleccionar esa opción durante el proceso de registro.

WordPress

DreamHost puede mover tu sitio de WordPress a tu cuenta DreamHost. Consulta los Terms of Service (TOS) para obtener más detalles sobre el proceso. Si deseas que DreamHost siga adelante con la migración de tu sitio, envía un ticket con el nombre de dominio que deseas migrar y también incluye una autorización expresa con los WordPress migrations TOS y cualquier cargo, si aplica.

DreamPress es un plan de alojamiento administrado de WordPress que DreamHost ofrece como una actualización a un plan de alojamiento estándar.

Precios

Si deseas migrar desde otro host, otra cuenta DreamHost u otro TLD (por ejemplo, example.net a example.com en la misma cuenta):

  • $ 99 — Migración a un plan de alojamiento estándar, por sitio.
  • Gratis — Migrar a cualquier plan de DreamPress.

También es gratis si estás migrando desde una instalación con un clic (One click install) que no es de WordPress a una instalación con un clic (en la misma cuenta).

Limitaciones

  • El servicio de migración de WordPress multisite no está disponible en este momento.
  • Si tu sitio está alojado en WordPress.com, solo se puede hacer una exportación de las páginas, publicaciones e imágenes incrustadas. Deberás configurar nuevamente tu tema, complementos y la mayoría de los elementos de diseño, como widgets y menús.

Actualización de las URL de INICIO y SITIO de WordPress

WordPress usa dos URL para controlar en qué dominio debe ejecutarse el sitio. Son:

  • La URL de INICIO
  • La URL del SITIO

Estos son muy importantes ya que controlan dónde busca la base de datos dónde se encuentra WordPress. Si recientemente migraste tu sitio a un nuevo dominio, cambiaste las URL por tu cuenta o agregaste un certificado SSL a un dominio que anteriormente no usaba uno, necesitarás actualizar estas URL.

DreamHost puede cambiarlos por un cargo único de $19.99. Este servicio es gratuito para los sitios de DreamPress.

Otros tipos de sitios web

Si tu sitio no es WordPress, DreamHost tal vez podría ayudarte. El cargo es de $ 99 dólares, pero se reembolsará si DreamHost no puede migrar con éxito el contenido de tu sitio. Envía un ticket a soporte con cualquier información que tengas sobre el sitio, y un representante del Equipo de Migraciones de DreamHost estará feliz de analizar la información para dejarte saber cual es la mejor opción.

Limitaciones

  • Los servicios de alojamiento y sitios construidos con herramientas propias de otros proveedores como Wix, Weebly o Squarespace no se pueden mover a DreamHost.
  • Los sitios de Windows Server-based Active Server Pages (ASP) tampoco se pueden mover a DreamHost.
