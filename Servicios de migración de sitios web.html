Generalidades

DreamHost ofrece una variedad de servicios de migración de sitios web, como pasar de un host anterior a DreamHost o mover un sitio dentro de DreamHost entre cuentas.

A continuación se describen las diversas opciones de migración a considerar. También puedes contactar a soporte para analizar la opción que mejor se adapte a tí. Si actualmente no eres cliente y deseas migrar tu sitio de WordPress, puedes seleccionar esa opción durante el proceso de registro.

WordPress

Ahora puedes migrar tu propio sitio de WordPress a DreamHost de forma gratuita, utilizando el complemento de migración automatizada DreamHost. El complemento se puede usar en cualquier cuenta DreamHost con un plan de alojamiento activo. Vea el siguiente artículo para más detalles:

  • Uso del complemento de WordPress de migración automatizada DreamHost

DreamHost puede mover tu sitio de WordPress a su cuenta DreamHost. Consulta los Términos de servicio (TOS) para obtener más detalles sobre el proceso. Si deseas que DreamHost avance con la migración de tu sitio, envía un ticket con el nombre de dominio que deseas migrar y también incluye un acuerdo con los TOS de migraciones de WordPress y cualquier cargo, si corresponde.

DreamPress es un plan de alojamiento administrado de WordPress que DreamHost ofrece como una actualización a un plan de alojamiento estándar.

Precios

Si deseas migrar desde otro host, otra cuenta DreamHost u otro TLD (por ejemplo, example.net a example.com en la misma cuenta):

  • $99 — Migración a un plan de alojamiento estándar, por sitio.
  • Free — Migrar a cualquier plan de DreamPress.

Limitaciones

  • El servicio de migración de WordPress no puede mover sitios de red de WordPress (multisitio) en este momento.
  • Si tu sitio está alojado en WordPress.com, solo está disponible una exportación de sus páginas, publicaciones e imágenes incrustadas. Deberás configurar tu propio tema, complementos y la mayoría de los elementos de diseño, como widgets y menús nuevamente.

Otros sitios web

Si tu sitio no está construido en WordPress, DreamHost aún puede ayudarte a migrar tu sitio. El costo de este servicio es de $ 99 y se reembolsará si DreamHost no puede migrar tu contenido con éxito. Envía un ticket en la página de Contact Support en el panel e incluya cualquier información que tenga sobre tu sitio. Un representante de migración de DreamHost estará encantado de investigar esto por tí.

Limitaciones

  • Los servicios de alojamiento y construcción de sitios propietarios como Wix, Weebly o Squarespace no se pueden migrar a DreamHost.
  • Los sitios web de "Páginas de servidor activo" (ASP) basados en Windows Server no se pueden migrar a DreamHost.

Cambiar las URL del sitio de WordPress

Si recientemente migraste tu sitio a un nuevo dominio, cambió las URL por tu cuenta o agregaste un certificado SSL a un dominio que anteriormente no usabas, deberás cambiar las URL de su sitio de WordPress.

DreamHost puede cambiarlos por un cargo único de $ 19.99. Para enviar una solicitud de cambio de URL de WordPress, visita la página de Contact Support en el panel que incluya el dominio/URL desde el que está cambiando y el nuevo dominio/URL que desea usar. El equipo de soporte de DreamHost responderá confirmando el cargo de $ 19.99 y el trabajo que se está realizando.

Este servicio es gratuito para los sitios de DreamPress.
