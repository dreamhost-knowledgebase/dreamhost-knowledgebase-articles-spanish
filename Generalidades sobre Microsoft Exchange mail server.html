Generalidades

Microsoft Exchange server es un servidor de correo creado por Microsoft. Es posible alojar tu sitio web en DreamHost pero seguir utilizando Microsoft para el correo electrónico. Esto requiere que configures algunos registros DNS. Este artículo explica lo que debe ajustarse.

Encontrando tu configuración DNS

Lo siguiente asume que tus Nameservers son controlados en DreamHost. Si no lo están, debes hacer estos cambios en la compañía donde apuntan Nameservers.

Cuando creas tu cuenta de Exchange, se te proporcionan registros DNS específicos. Estos se pueden agregar en tu panel DreamHost. El siguiente enlace es una descripción general de los registros que Microsoft puede brindarle:

  • Registros DNS externos para Office 365

Una vez que tengas la lista de registros DNS de Microsoft, ingrésalos en tu panel DreamHost. El siguiente artículo te explica cómo crear diferentes tipos de registros DNS personalizados

  • DNS personalizado DNS

Consulta el siguiente artículo para obtener información sobre cómo cambiar tus registros MX:

  • Registros MX

Propagación DNS

Una vez que hayas agregado tus registros DNS personalizados, deben actualizarse en línea. Esto puede tardar varias horas. Ve el artículo propagación DNS para más detalles. También puedes revisar en línea en un sitio como este:

  • Revisa registros DNS al rededor del mundo
