Generalidades

Mantenerse al tanto de los complementos y las actualizaciones principales de WordPress es lo mejor que puedes hacer para mantener tu sitio actualizado y seguro: no solo mantendrá tu sitio menos propenso a intentos de hackeo, sino que casi todas las versiones nuevas del software tienen ajustes que ofrecer aumentos de velocidad.

Todas las instalaciones de WordPress de un clic en DreamHost tienen las actualizaciones automáticas habilitadas de forma predeterminada. Esta opción se puede desactivar después de instalar WordPress.

Funciones de actualización automática

Las actualizaciones automáticas de DreamHost incluyen las siguientes características:

  • DreamHost realiza una actualización de tu sitio de WordPress entre 1 y 3 semanas después del lanzamiento de una versión principal. Si deseas actualizar manualmente antes de esa fecha, puede hacerlo siguiendo estos pasos.
  • DreamHost también actualiza los complementos disponibles que permiten actualizaciones automáticas durante el proceso.
  • Si se encuentran errores durante el proceso de actualización, DreamHost revertirá la actualización y te notificará por correo. Las actualizaciones automáticas de DreamHost solo están disponibles para sitios con una instalación activa de WordPress de un clic. Si las actualizaciones automáticas están deshabilitadas, no se pueden volver a habilitar en el futuro.

Deshabilitar las actualizaciones automáticas

A partir de la versión 3.7 de WordPress, la aplicación de WordPress realiza automáticamente actualizaciones de revisiones menores, que son independientes de las actualizaciones automáticas de DreamHost. La desactivación de las actualizaciones automáticas de DreamHost no cambia este comportamiento. DreamHost recomienda que permitas que estas actualizaciones menores se produzcan automáticamente, ya que normalmente son correcciones de errores y/o actualizaciones de seguridad.

Si prefieres actualizar manualmente tu instalación de WordPress, puedes usar las siguientes instrucciones para optar por no recibir actualizaciones automáticas eliminando tu sitio de la lista Instalaciones de un clic:

  1. Navega a la página Instalaciones de Un-Clic.
    La página Instalaciones de Un Clic se abre.
  2. Haz clic en el botón Manage My Apps.
  3. Haz clic en el dominio que utiliza la instalación de un clic para expandirlo.
    Se muestran todas las instalaciones bajo el dominio.
  4. Haz clic en el botón Remove from List a la derecha.
    Un mensaje de advertencia aparece:
  5. Haz clic en Ok para confirmar.
    Un mensaje 'Success' aparece cuando el sitio se elimina de la lista Instalaciones de Un Clic:

Después de eliminar tu sitio, ya no se muestra en tu página Manage My Apps. DreamHost no puede actualizar automáticamente WordPress en este sitio en el futuro.
