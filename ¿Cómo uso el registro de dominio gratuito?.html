Generalidades

DreamHost ofrece un crédito de registro de dominio gratuito durante el primer año a los clientes Shared (Shared Starter y Shared Unlimited/Happy Hosting) y DreamPress en planes anuales, que puede utilizarse para un dominio .art, .blog, .club, .com, .design, .info, .me, .net, .online, .org, .shop, .site, .store, .tech, o .xyz.

Para calificar para un crédito de registro de dominio gratuito durante el primer año, te debes inscribir en un plan anual de 1 año o 3 años en un Shared Starter o Shared Unlimited/Happy Hostingo cualquier plan anual de DreamPress (los planes mensuales de Shared Hosting o DreamPress aplican). Este crédito gratuito NO está disponible en ningún otro plan de alojamiento.

El crédito se debe usar gratuito dentro del primer tres meses desde el momento que agregaste el plan anual Shared/DreamPress. Si no utilizas el registro gratuito durante el primer tres meses, ya no estará disponible para su uso.

¿Qué es el programa de registro de dominio gratuito?

Cuando te suscribes a un plan de alojamiento anual Shared Starter o Shared Unlimited/Happy Hosting o cualquier plan anual de DreamPress:

  • Recibes un (1) crédito de registro de forma gratuita.
  • Puedes utilizar este crédito de registro para registrar, transferir o renovar un nombre de dominio en DreamHost siempre y cuando usen las extensiones permitidas dentro de la promoción (.art, .blog, .club, .com, .design, .info, .me, .net, .online, .org, .shop, .site, .store, .tech, o .xyz).

Si usas (o transfieres) el registro de dominio gratuito y luego cierras tu hosting 'Shared Starter'/'Shared Unlimited' o el plan DreamPress antes de que se complete su período anual, tú eres responsable del costo del registro para el que se utilizó el crédito . Por lo general, esta cantidad se deduce de tu reembolso, a menos que haya un saldo adeudado en la cuenta antes de cancelar. También continúas siendo propietario del nombre de dominio hasta que caduque.

Preguntas Frecuentes

¿Cómo puedo verificar si tengo un crédito gratis en mi cuenta?

Si tienes un registro en tu cuenta

Puedes verificar esto en tu panel, en la página Registros.

Si tienes un crédito gratuito disponible, el precio de la renovación de los dominios seleccionados (mencionamos las extensiones anteriormente) se indica como "$0.00".

Si NO tienes un registro en tu cuenta

¿A quién pertenece el registro gratuito? ¿A DreamHost o a mi?

Tú eres el único propietario del registro de dominio para el que utiliza el crédito.

¿Puedo usar el crédito para otros servicios?

No. Este crédito no puede utilizarse para ningún otro servicio o artículo.

Si el registro se transfiere a otra compañía, ¿puedo renovarlo en esa otra compañía gratis?

No. El crédito de dominio gratuito solo se aplica a los dominios que se administran a través de tu panel de DreamHost.

¿En qué tipo de registros puedo usar el crédito de registro gratuito?

Si bien DreamHost ofrece muchas extensiones de dominio diferentes, el crédito gratuito solo está disponible para los siguientes dominios estándar:

  • .art
  • .blog
  • .club
  • .com
  • .design
  • .info
  • .me
  • .net
  • .online
  • .org
  • .shop
  • .site
  • .store
  • .tech
  • .xzy

El crédito no se puede utilizar para dominios especializados o dominios internacionales específicos de algún país.

Utilicé mi crédito gratis para un registro, pero tenía la intención de usarlo para otro registro. ¿Esto se puede cambiar?

Sí. El equipo de soporte de DreamHost puede ayudarte a aplicar el crédito gratuito a un dominio diferente, si así lo solicitas. Un ejemplo sería si renovaras dos dominios al mismo tiempo, pero deseas que el crédito se use en un dominio en lugar del otro. Ten en cuenta que el cambio de crédito gratuito no implicará un cambio en el monto de tu pago. Si el cambio de crédito gratuito da como resultado un saldo positivo en tu cuenta, el saldo se aplicará a los cargos futuros a medida que surjan.

DreamHost entiende que algunos clientes son revendedores y tienen una razón para usar el crédito para un dominio específico que figura en tus facturas. Si necesitas ayuda con esto, no dudes en ponerte en contacto con el soporte en la página de Contactar a Soporte.
