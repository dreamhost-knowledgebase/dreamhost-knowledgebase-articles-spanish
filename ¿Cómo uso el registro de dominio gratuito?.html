Generalidades

DreamHost ofrece un crédito de registro de dominio gratuito durante el primer año a los clientes Shared (Starter Shared y Unlimited Shared / Happy Hosting) y DreamPress en planes anuales, que puede utilizarse para un dominio .blog, club, .com, .info, .net , .online, .org, .shop, .tech, o .xzy.

Para calificar para un crédito de registro de dominio gratuito durante el primer año, te debes inscribir en un plan anual de 1 año o 3 años en un Shared Starter, Unlimited Shared / Happy Hosting o cualquier plan anual de DreamPress (los planes mensuales de Shared Hosting o DreamPress aplican). Este crédito gratuito NO está disponible en ningún otro plan de alojamiento.

El crédito se debe usar gratuito dentro del primer mes desde el momento que agregaste el plan anual Shared / DreamPress. Si no utilizas el registro gratuito durante el primer mes, ya no estará disponible para su uso.

  • Registrarse en Shared Hosting

El precio de una transferencia puede ser diferente al precio de una renovación. Visita las siguientes páginas para obtener información actualizada sobre precios:

¿Qué es el programa de registro de dominio gratuito?

Cuando te suscribes a un plan de alojamiento anual Starter Shared o Unlimited Shared / Happy Hosting o cualquier plan anual de DreamPress:

  • Recibes un (1) crédito de registro de forma gratuita.
  • Puede utilizar este crédito de registro para registrar, transferir o renovar un nombre de dominio en DreamHost siempre y cuando usen las extensiones permitidas dentro de la promoción (.blog, club, .com, .info, .net, .online, .org, .shop, .tech, o .xyz).

Si usas (o transfieres) el registro de dominio gratuito y luego cierras tu hosting 'Starter Shared' / 'Unlimited Shared' o el plan DreamPress antes de que se complete su período anual, tú eres responsable del costo del registro para el que se utilizó el crédito . Por lo general, esta cantidad se deduce de tu reembolso, a menos que haya un saldo adeudado en la cuenta antes de cancelar. También continúas siendo propietario del nombre de dominio hasta que caduque.

Preguntas Frecuentes

¿Cómo puedo verificar si tengo un crédito gratis en mi cuenta?

Puedes verificar esto en tu panel, en la página Registrations.

Si tienes un crédito gratuito disponible, el precio de la renovación de los dominios seleccionados (mencionamos las extensiones anteriormente) se indica como "$0.00".

Si no tienes ningún dominio registrado, verás un precio por el dominio cuando lo estés registrando. Este precio solo aparece en la primera página. Siempre que selecciones '1 año' como período de tiempo, verás que el costo es gratuito en la página final cuando envíes el pago.

¿A quién pertenece el registro gratuito? ¿A DreamHost o a mi?

Tú eres el único propietario del registro de dominio para el que utiliza el crédito.

¿Puedo usar el crédito para otros servicios?

No. Este crédito no puede utilizarse para ningún otro servicio o artículo.

Si el registro se transfiere a otra compañía, ¿puedo renovarlo en esa otra compañía gratis?

No. El crédito de dominio gratuito solo se aplica a los dominios que se administran a través de tu panel de DreamHost.

¿En qué tipo de registros puedo usar el crédito de registro gratuito?

Si bien DreamHost ofrece muchas extensiones de dominio diferentes, el crédito gratuito solo está disponible para los siguientes dominios estándar:

  • .blog
  • .club
  • .com
  • .info
  • .net
  • .online
  • .org
  • .shop
  • .tech
  • .xzy

El crédito no se puede utilizar para dominios especializados o dominios internacionales específicos de algún país.

Utilicé mi crédito gratis para un registro, pero tenía la intención de usarlo para otro registro. ¿Esto se puede cambiar?

Sí. El equipo de soporte de DreamHost puede ayudarte a aplicar el crédito gratuito a un dominio diferente, si así lo solicitas. Un ejemplo sería si renovaras dos dominios al mismo tiempo, pero deseas que el crédito se use en un dominio en lugar del otro. Sin embargo, esto realmente solo cambiaría qué dominio se muestra como renovado de forma gratuita en tu factura: no cambiará el monto de tu pago ni la fecha en que se utilizó el crédito.

DreamHost entiende que algunos clientes son revendedores y tienen una razón para usar el crédito para un dominio específico que figura en sus facturas. Si necesita ayuda con esto, no dude en ponerse en contacto con el soporte en la página de Contact Support.

¿Por qué debo pagar por el registro de mi dominio al momento de cancelar mi cuenta?

El registro solo es gratuito durante el primer año con un plan de alojamiento activo.

Si tu dominio califica para el crédito de registro gratuito, el registro de dominio se agrega a tu cuenta de forma gratuita durante el primer año. El tiempo mínimo por el que se puede registrar un dominio es de un año. Si cancelas tu plan de alojamiento (que viene con el registro gratuito), sigues siendo el propietario del registro durante el período restante.

Debido a que aún posees el registro (después de cancelar tu plan de alojamiento dentro del primer año), debes pagar el registro.
