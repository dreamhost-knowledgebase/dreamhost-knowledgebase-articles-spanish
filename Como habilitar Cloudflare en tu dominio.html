Generalidades

Este artículo explica cómo habilitar Cloudflare en tu sitio en el panel.

Paso adicional para sitios de WordPresss

Asegúrate de cambiar las URL en tu sitio de WordPress antes de activar Cloudflare en el panel. De lo contrario, puede provocar un bucle de redireccionamiento en tu sitio web.

Si tu sitio ejecuta WordPress, hay muchas URL codificadas en tu base de datos que pueden apuntar a la versión no www de tu dominio. Para usar Cloudflare, todas las URL en tu base de datos de WordPress deben apuntar a la versión www. Por ejemplo:

  • https://www.example.com

Si no actualizas las URL en tu base de datos, verás un error de bucle de redireccionamiento infinito.

  • Error de WordPress 'too many redirects'

Consulta el siguiente artículo para obtener información sobre cómo actualizar todas las URL en tu base de datos de WordPress.

  • ¿Cómo cambiar el SiteURL de WordPress?

Habilitar Cloudflare

  1. Navega a Administrar Dominios.
    • Debes agregar un plan de alojamiento a tu cuenta para poder utilizar cualquier Servicio de Cloudflare. Visita el artículo cómo agregar un dominio para obtener instrucciones.
    • Si tu dominio está en DreamPress, debe ponerte en contacto con soporte para habilitar un plan Cloudflare.
  2. Una vez que agregues un plan de alojamiento a tu dominio, haz clic en el botón Edit a la derecha del dominio en la columna Web Hosting.
  3. Si estás usando Cloudflare sin un subdominio, selecciona 'Add WWW' para el nombre de dominio cuando se te pregunte 'Do you want www in your URL?'.
  4. Desplázate hacia abajo a la sección Cloudflare Services.
  5. Marca la casilla de verificación 'Enable Cloudflare on this domain?'.
  6. Seleccione el tipo de cuenta Cloudflare:
    • Si no tienes una cuenta de Cloudflare, deberás seleccionar la opción New Account en el menú desplegable para la cuenta de Cloudflare. Luego deberás ingresar una dirección de correo válida que te gustaría usar para la cuenta y crear una contraseña para la nueva cuenta de Cloudflare. Utilizarás este nombre de usuario y contraseña para iniciar sesión y administrar tu cuenta en Cloudflare.com.
    • Si ya tienes una cuenta de Cloudflare, selecciona la opción Existing Account en el menú desplegable. Luego deberás ingresar la dirección de correo y la contraseña de tu cuenta de Cloudflare.
    • Si ya configuraste una cuenta de Cloudflare en tu panel para otro dominio o configuraste Cloudflare anteriormente, verás una opción para Existing Accounts en el menú desplegable. Si ves esta opción y deseas utilizar una cuenta existente, simplemente selecciona la dirección de correo que deseas usar.
  7. Una vez que hayas configurado tu cuenta de Cloudflare, selecciona el nivel de suscripción de Cloudflare.
    • Visita el artículo de Suscripciones de Cloudflare para obtener más detalles sobre estos planes.
  8. Desplázate hacia abajo y haz clic en el botón Change Settings para guardar tu nueva configuración.
    • Dentro de 5-10 minutos, el dominio se actualizará para usar Cloudflare en DreamHost.
    • Sin embargo, los cambios de DNS en el dominio tardarán más en propagarse completamente en línea para permitir que Cloudflare funcione.
    Si ves el siguiente error, debes asegurarte de que tu dominio esté configurado para usar WWW en el nombre de dominio:

Después de habilitar Cloudflare, también verás Cloudflare en la lista junto con la versión de PHP en la columna Web Hosting en la página Administrar Dominios. También verás una opción Clear Cache a la derecha del dominio en caso de que necesites purgar tu caché de Cloudflare.

Desactivar Cloudflare

Para deshabilitar Cloudflare, repite los pasos anteriores, pero asegúrate de desmarcar la casilla de verificación Enable Cloudflare on this domain? antes de guardar los cambios.
