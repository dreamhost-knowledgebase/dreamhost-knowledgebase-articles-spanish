Apuntar tu sitio DreamHost a CargoCollective

Si tu dominio está registrado en DreamHost, puedes apuntar tu DNS a CargoCollective. Mira el siguiente enlace para obtener instrucciones:

  • Apuntar a CargoCollective

Hay dos opciones: apuntar el dominio principal o apuntar un subdominio como blog.example.com.

Señalando un dominio primario

Las siguientes instrucciones le muestran cómo dirigir tu sitio example.com a CargoCollective.

Las siguientes capturas de pantalla utilizan el dominio example.com. Tu dominio aparecerá en su lugar cuando siga estas instrucciones.

  1. Configura tu dominio en tu panel como Solo DNS.
    Name (dejar en blanco)
    Type A
    Value 173.203.204.123
  2. En tu panel DreamHost, crea un registro A para tu dominio.
  3. Guarda este registro haciendo clic en el botón Add Record Now!.
    Name www
    Type A
    Value 173.203.204.123
  4. Repite este paso para el registro A 'www'.

Tu sitio ahora apunta a CargoCollective. Debido a la propagación de DNS, los registros pueden tardar varias horas en actualizarse en línea.

Apuntar un subdominio

Las siguientes instrucciones le muestran cómo dirigir tu sitio blog.example.com a CargoCollective utilizando un subdominio.

  1. Configura tu dominio en tu panel como Solo DNS.
    Name (dejar en blanco)
    Type A
    Value 173.203.204.123
  2. En tu panel DreamHost, crea un registro A para tu dominio.
  3. Guarda este registro haciendo clic en el botón Add Record Now!.
    Name blog
    Type CNAME
    Value cargocollective.com
  4. En tu panel DreamHost, crea un registro CNAME.

Tu subdominio ahora apunta a CargoCollective. Debido a la propagación de DNS, los registros pueden tardar varias horas en actualizarse en línea.
