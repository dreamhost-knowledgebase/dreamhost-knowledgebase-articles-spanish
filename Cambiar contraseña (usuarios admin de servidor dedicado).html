Cambiar tu contraseña

  1. Navega a la página de Admin User.
    • El Admin user's está en la página Admin Users.
  2. Haz clic en el botón Edit que está hacia el lado derecho del usuario.
  3. Tienes dos opciones para crear una contraseña nueva:
    • Ingresar la contraseña dos veces y guardar los cambios.
    • Hacer clic en la casilla 'Pick a password for me'. Luego el sistema la creará por ti.
