Generalidades

MIME es un acrónimo de Extensiones de correo de Internet multipropósito, aunque los tipos MIME modernos se aplican a otros servicios además del correo electrónico. Los tipos MIME son una forma de etiquetar tipos de datos para que los programas puedan procesarlos correctamente. Por ejemplo, los tipos MIME pueden ayudar a un navegador a determinar qué tipo de contenido está a punto de ser procesado para que el navegador pueda usar los protocolos, el reproductor multimedia, el códec o los complementos adecuados para mostrar los datos correctamente. Esto también permite que el navegador muestre contenido que no está en formato HTML.

Agregar tus propios tipos MIME

Puedes agregar tus propias extensiones a cualquier directorio usando un archivo .htaccess. Por ejemplo:

AddType application/andrew-inset ez
AddType video/x-flv flv
