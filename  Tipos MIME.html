Generalidades

Multipurpose Internet Mail Extensions o MIME (en español "extensiones multipropósito de correo de internet") se usaron originalmente para que los correos electrónicos pudieran incluir información que no fuera texto sin formato. Ahora se les conoce como Media Types o Tipos de Medios y son usados por los servidores para informar a los buscadores web del tipo de información que se les está enviando.

El navegador usa los Media Types (no la extensión del archivo) para determinar el tipo de contenido que se procesará a fin de seleccionar los protocolos adecuados para manejar los datos correctamente.

Media Types oficiales

La Autoridad de asignación de números de Internet (IANA) es la autoridad oficial responsable de los Media Types. Revisa la página de IANA para ver la lista actual:

  • iana.org/assignments/media-types/media-types.xhtml

Ver los Media Types en tu servidor

Para revisar cuales Media Types están ya instalados en tu servidor, inicia sesión en tu servidor a través de SSH. Una vez iniciado sesión corre el siguiente comando para ver el archivo mime.types.

[server]$ cat /etc/mime.types

También puedes usar grep para buscar un Media Type específico. Por ejemplo, si quieres ver mpeg, corre lo siguiente:

[server]$ cat /etc/mime.types | grep mpeg

Agregar tus propios Media Types

También puedes agregar tus propios Media Types a cualquier directorio usando un archivo .htaccess. Por ejemplo:

AddType video/x-flv flv

AddType es una directiva de Apache que especifica que estás agregando un Media Type. Luego se agrega el Media Type real. Revisa el siguiente enlace para obtener más información sobre la sintaxis de un Media Type:

  • w3docs.com/learn-html/mime-types

Ver los Media Types en un buscador

Abra el menú de Herramientas de Desarrollo en tu navegador para ver el Media Type de un archivo.

Chrome, Firefox, y Safari

  1. Abre el menú de Herramientas de Desarrollo en tu buscador.
    Windows: Mantén presionado CTRL + SHIFT y presiona la letra i. También podrías dar clic derecho a la página y seleccionar Inspeccionar del menú.
    Mac: Mantén presionado Opción () + Comando () y presiona la letra i. También puedes dar clic derecho (Ctrl ()+ clic) a la página y seleccionar Inspeccionar del menú.
  2. Da clic a la pestaña Red en el menú de Herramientas de Desarrollador.
  3. Vuelve a cargar la página en la que te encuentra actualmente haciendo clic en el botón Actualizar de tu navegador o presionando F5. Se muestra una lista de todos los archivos cargados en la página actual.
  4. Da clic en cualquier archivo debajo de la columna Nombre.
  5. Selecciona la pestaña Encabezados en el panel derecho.
    Los encabezados HTTP se muestran en el panel derecho. Por ejemplo, si haces clic en un archivo de imagen .png, el panel derecho muestra: 
    Content-Type: image/png
