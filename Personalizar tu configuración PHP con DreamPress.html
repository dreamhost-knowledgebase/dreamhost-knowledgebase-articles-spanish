Generalidades

Un .user.ini es un archivo de texto simple que le permite ajustar tu configuración de PHP. Por ejemplo, la versión de PHP en el servidor viene con algunas configuraciones predeterminadas, como un memory_limit de 256MB y puedes cambiar esto usando un archivo .user.ini.

¿Qué configuración de PHP puedo personalizar?

Mira la siguiente categoría para obtener una lista de artículos de personalizaciones PHP comunes:

Crear un archivo .user.ini

Para personalizar tu configuración de PHP, crea un archivo llamado .user.ini en el directorio de tu sitio web. Por ejemplo:

/home/username/example.com

Puedes crear un archivo .user.ini iniciando sesión en tu servidor a través de FTP o SSH. Las siguientes secciones describen cómo hacer esto para ambos métodos.

Crear un archivo .user.ini a través de FTP (recomendado para principiantes)

  1. Consulta el siguiente artículo para obtener instrucciones sobre cómo iniciar sesión en tu servidor web DreamPress. Puedes usar el administrador de archivos en el panel o un cliente de terceros para iniciar sesión.
    • Iniciar sesión en tu sitio DreamPress a través de FTP
  2. Una vez que hayas iniciado sesión, dirígete al directorio de tu sitio web donde se encuentran tus archivos.
  3. En este directorio, crea un nuevo archivo .user.ini.

    Asegúrate de que el nombre del archivo comience con un punto.

  4. Mira el artículo Creación y edición de un archivo a través de FTP para obtener instrucciones completas sobre cómo crear, editar y guardar este archivo con tu configuración de PHP.
  5. Una vez que lo guardes, tu configuración se actualizará automáticamente en aproximadamente 5 minutos.

Crear un archivo .user.ini a través de SSH (para usuarios avanzados)

  1. Consulta el siguiente artículo para obtener instrucciones sobre cómo iniciar sesión en tu servidor web DreamPress. Puedes usar el administrador de archivos en el panel o un cliente de terceros para iniciar sesión.
    • Iniciar sesión en tu sitio DreamPress a través de FTP
  2. La opción #2 explica cómo ubicar tus credenciales de inicio de sesión.
  3. Visita la Generalidades de SSH y los artículos de cliente de software de SSH para obtener instrucciones sobre cómo iniciar sesión en su servidor DreamPress. 
    [server]$ cd ~/example.com
  4. Navega al directorio de tu sitio web.
  5. En el directorio de tu sitio web, crea un nuevo archivo .user.ini.

    Asegúrate de que el nombre del archivo comience con un punto.

  6. Revisa el la artículo Crear y editar un archivo vía SSH para obtener instrucciones completas sobre cómo crear, editar y guardar este archivo con tu configuración de PHP.
  7. Una vez que lo guardes, tu configuración se actualizará automáticamente en aproximadamente 5 minutos.

Actualizar tus cambios

Después de crear tu archivo .user.ini, solo necesita esperar 5 minutos para que los cambios se actualicen. Este valor se establece en 300 segundos en su configuración de PHP, por lo que no necesitas reiniciar PHP manualmente para que los cambios surtan efecto.
