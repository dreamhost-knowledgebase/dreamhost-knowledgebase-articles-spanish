¿Qué es MySQL VPS?

DreamHost MySQL VPS es el servicio premium de base de datos MySQL de DreamHost. Similar en concepto a VPS de DreamHost, DreamHost MySQL VPS usa Linux Containers (LXC) para aislar el servidor de la base de datos.

Diferencia entre servidores MySQL privados y compartidos

Todos los planes de alojamiento de DreamHost están configurados con un servidor MySQL compartido de forma predeterminada. Estos servidores de bases de datos compartidas 'comparten' los recursos del servidor con todos los usuarios y bases de datos.

VPS MySQL es diferente en que tus bases de datos y recursos están protegidos de otros usuarios.

¿Está disponible el acceso de root?

Root y acceso SSH en un MySQL VPS no está disponible. Debes actualizar a un Servidor Dedicado donde las bases de datos de MySQL está en la misma máquina, lo que permite el acceso a los archivos de configuración de MySQL.
