Generalidades

PuTTY es un emulador de terminal gratuito que admite varios protocolos de red tal como SSH, Telnet, rlogin, y SCP. Originalmente estaba disponible sólo para Windows, pero ahora también está disponible en varias plataformas UNIX, con puertos de trabajo en desarrollo para Mac OS clásico y Mac OS X. En DreamHost, puedes usar PuTTY para iniciar sesión en tu servidor web a través de tu usuario Shell.

Descargar PuTTY

Sigue los siguientes pasos para descargar y configurar PuTTY en tu computador Windows.

  1. En tu computador Windows:
    • Navega a C:\
    • En ese directorio, crea una carpeta llamada \bin
    Esta es la ubicación en la que vas a descargar los archivos de PuTTY. No importa necesariamente dónde los descargues, pero agregar este directorio facilita el seguimiento de los archivos.
  2. Abre la página de descarga en tu navegador:
    • chiark.greenend.org.uk/~sgtatham/putty/download.html
  3. Descarga los siguientes archivos de ese sitio en tu carpeta C:\ bin:
    • putty.exe
    • puttygen.exe
    • pagent.exe
    • pscp.exe
    • psftp.exe
  4. Navega hasta el directorio C:\Users\bin\ y haz doble clic en el archivo putty.exe para abrirlo. O puedes crear un acceso directo en tu escritorio para facilitar el acceso:
    • Haz clic derecho en tu escritorio y elige 'New > Shortcut' (Nuevo > Acceso directo)
    • Busca la ubicación del archivo putty.exe (debe ser C:\Users\bin\putty.exe)
    • Guarda el acceso directo
    Cuando abres PuTTY, aparece el siguiente cuadro de configuración:
  5. Ingresa tus credenciales de DreamHost en el cuadro Configuración PuTTY (mira los detalles a continuación).
  6. Haz clic en el botón Save para guardar esta sesión.

Cuando abres una sesión en PuTTY, te solicita tu usuario Shell y contraseña.

Descripción de los campos en el cuadro de configuración PuTTY

Host Name or IP address (Nombre de Host o la dirección IP)
En este campo, ingresa tu nombre de servidor DreamHost. Por ejemplo, si tu servidor se llama flower, el nombre de host es flower.dreamhost.com
Port (Puerto)
22
Connection Type (Tipo de Conexión)
SSH
Saved Sessions (Sesiones Guardadas)
Ingresa un nombre para esta conexión. Por ejemplo, "Servidor DreamHost"
