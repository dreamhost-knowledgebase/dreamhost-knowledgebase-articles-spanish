Generalidades

Apache es un servidor web HTTP de código abierto para sistemas similares a UNIX (BSD, Linux y UNIX), Microsoft Windows y otras plataformas.

Versión Apache de DreamHost

DreamHost ejecuta las siguientes versiones:

  • 2.2.31 en servidores que ejecutan Ubuntu 14 (Trusty)
  • 2.4.29 en servidores que ejecutan Ubuntu 18 (Bionic)
  • 2.4.25 en servidores que ejecutan Debian (Stretch)

En el pasado, uno podía determinar información sobre el servidor web desde los encabezados de respuesta HTTP a través del encabezado "Server". Sin embargo, por razones de seguridad, se decidió establecer ServerTokens en "Prod" y, como resultado, el valor de este encabezado solo devuelve "Apache". No se puede modificar a través de archivos de configuración por directorio (.htaccess).

Por ejemplo, en el pasado, ejecutar curl -Is en tu sitio le daba la siguiente respuesta:

[server]$ curl -Is http://websitehelp.support | grep Server
Server: Apache/1.3.37 (Unix) mod_throttle/3.1.2 DAV/1.0.3 mod_fastcgi/2.4.2 mod_gzip/1.3.26.1a PHP/4.4.7  mod_ssl/2.8.22 OpenSSL/0.9.7e

Si ejecutas este mismo comando ahora, la única respuesta es 'Apache':

[server]$ curl -Is http://websitehelp.support |grep Server
Server: Apache

Para encontrar la versión de Apache que se ejecuta actualmente en una máquina Trusty, ejecuta lo siguiente:

[server]$ dpkg -l ndn-apache22

Para máquinas Bionic y Debian Stretch, ejecuta lo siguiente:

[server]$ /usr/sbin/apache2 -v

Módulos Apache para DreamHost

Ten en cuenta que algunos de los módulos solo funcionan si están habilitados a través de panel (por ejemplo, Cloudflare, Passenger, etc.) y algunos módulos dependen del tipo de servidor en el que te encuentres.

Si necesitas el módulo xsendfile habilitado, debes ponerte en contacto con el soporte en la página Contact Support. Asegúrate de mencionar en qué dominio deseas habilitarlo. Además, si necesitas el módulo proxy, eso solo puede habilitarse en un servidor VPS o Dedicado.

Para obtener más información sobre los módulos de Apache, visita la siguiente página:

  • http://httpd.apache.org/docs/2.2/mod/
