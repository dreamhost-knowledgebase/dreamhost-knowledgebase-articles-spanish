Generalidades

Apache es un servidor web HTTP de código abierto para sistemas similares a UNIX (BSD, Linux y UNIX), Microsoft Windows y otras plataformas.

Versión Apache de DreamHost

En el pasado, uno podía determinar información sobre el servidor web desde los encabezados de respuesta HTTP a través del encabezado Server. Sin embargo, por razones de seguridad, se decidió establecer ServerTokens en "Prod" y, como resultado, el valor de este encabezado solo devuelve Apache. No se puede modificar usando un archivo (.htaccess).

Por ejemplo, si ejecutas este mismo comando ahora, la única respuesta es Apache:

[server]$ curl -Is http://example.com | grep Server
Server: Apache

Para encontrar la versión de Apache que se ejecuta actualmente, corre lo siguiente:

[server]$ /usr/sbin/apache2 -v

Módulos Apache para DreamHost

Ten en cuenta que algunos de los módulos solo funcionan si están habilitados a través de panel (por ejemplo, Cloudflare, Passenger, etc.) y algunos módulos dependen del tipo de servidor en el que te encuentres.

Si necesitas el módulo xsendfile habilitado, debes ponerte en contacto con el soporte en la página Contactar a Soporte. Asegúrate de mencionar en qué dominio deseas habilitarlo. Además, si necesitas el módulo proxy, eso solo puede habilitarse en un Servidor VPS o Dedicado.

Para obtener más información sobre los módulos de Apache, visita la siguiente página:

  • http://httpd.apache.org/docs/2.2/mod/
