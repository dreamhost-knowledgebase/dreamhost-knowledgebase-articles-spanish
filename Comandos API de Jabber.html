Advertencia de fin de funcionamiento

Este comando de API será eliminado y no va a funcionar en Noviembre 2 de 2021.

El módulo API jabber te permite administrar lo usuarios de Jabber para tus dominios.

Comandos

jabber-list_users

Enumera todos los usuarios de Jabber en esta cuenta e incluye las contraseñas y el estado de los usuarios. El estado es uno de los siguientes: active, inactive, pending-active, pending-inactive, pending-delete

Comando jabber-list_users
Valores

(ninguno)
Resultado

success
username domain password status
testuser example.com abc123 active
testuser2 example.com 12345 active
Posibles errores

(ninguno)

jabber-list_users_no_pw

Igual que jabber-list_users, pero omite las contraseñas. Las contraseñas se muestran como ********.

Comando jabber-list_users_no_pw
Valores

(ninguno)
Resultado

(ninguno)

jabber-add_user

Crea un nuevo usuario de Jabber. Esto devuelve un token de servicio (consulta Comandos de control de servicio). Esto puede devolver el mismo token de cola de servicio que las llamadas anteriores a la API de Jabber. Si este es el caso, todos los cambios pendientes se aplican juntos.

Comando jabber-add_user
Valores

username : El usuario Jabber deseado.
domain : El dominio para el nuevo usuario.
password : La contraseña para el nuevo usuario.
Resultado

success
token
gcnaBcD12eFgH34iJkL
Posibles errores

no_username
no_domain
no_password
invalid_username
invalid_domain
invalid_password
user_exists
