Generalidades

Puedes buscar instancias usando una dirección IP con el cliente de línea de comandos OpenStack o usando el panel de DreamCompute.

Usar la CLI de OpenStack

Para correr los siguientes comandos, asegúrate de haber instalado el cliente de línea de comandos de OpenStack.

Puedes buscar una instancia desde la línea de comandos utilizando el parámetro de dirección IP. El siguiente ejemplo muestra los resultados de una búsqueda de la dirección IP 10.0.0.4.

[user@localhost]$ openstack server list --ip 10.0.0.4
+------------------+----------------------+--------+------------+-------------+------------------+
| ID               | Name                 | Status | Task State | Power State | Networks         |
+------------------+----------------------+--------+------------+-------------+------------------+
| 8a99547e-7385... | myInstanceFromVolume | ACTIVE | None       | Running     | private=10.0.0.4 |
+------------------+----------------------+--------+------------+-------------+------------------+

Uso del panel de DreamCompute

    1. Navega hasta la página DreamCompute.
    2. Haz clic en el botón Ver panel.
    3. A la izquierda, haz clic en 'Compute'.
    1. En el menú desplegable, haz clic en Instances.
  1. Selecciona el tipo de dirección IP que deseas buscar (IPv4 o IPv6), escribe la dirección en el cuadro de búsqueda y haz clic en el botón "Filter".

Se mostrarán todas las instancias con esa dirección IP.
