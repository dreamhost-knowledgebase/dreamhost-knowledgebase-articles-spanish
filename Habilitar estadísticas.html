Generalidades

Lo siguiente cubre el proceso de agregar usuarios de estadísticas, agregar y editar estadísticas y acceder a una URL de estadísticas de dominio.

¿Cómo obtengo estadísticas para mi sitio web?

Las estadísticas del sitio no están disponibles en los planes DreamPress (excluyendo los planes antiguos de DreamPress Legacy). Asegúrate de habilitar Jetpack si deseas capturar estadísticas del sitio en tu sitio DreamPress.

Los propietarios de sitios web a menudo desean saber cuánto tráfico recibe su sitio en un momento dado. DreamHost ofrece una manera de configurar las estadísticas del sitio para un dominio en la página Site Statistics. Desde la página de Site Statistics, puedes agregar un usuario de estadísticas, activar estadísticas para un dominio y configurar informes de estadísticas.

Agregar un usuario de estadísticas

Las estadísticas del sitio comienzan a registrarse solo después de crear un usuario. No puedes obtener ninguna estadística del sitio antes de eso.

Agregar un usuario de estadísticas también es útil para proteger las estadísticas del dominio y proporcionar acceso a otra persona.

Para agregar un usuario de estadísticas:

  1. Navega a la página Site Statistics.
  2. Haz clic en el enlace Add one? bajo la columna Stats users.
  3. Crea un nombre de usuario y contraseña. Este usuario solo se puede usar para acceder a tus estadísticas y no se puede usar para nada más en el panel.
  4. Haz clic en el botón Add User para guardar los cambios

Si deseas agregar un usuario de estadísticas adicional, puedes hacerlo haciendo clic en el botón Add Stats User a la derecha del dominio elegido.

Habilitar informes estadísticos específicos en un dominio

Para habilitar informes de estadísticas específicos:

  1. Navega a la página Site Statistics.
  2. Haz clic en el botón Configure a la derecha del dominio.
    Las casillas de verificación aparecen para los informes de Month, day, y Longterm en la siguiente página:
  3. Marca la casilla Active? para cualquiera que desees empezar a rastrear.
  4. Haz clic en el botón Change Web Report Settings Now para guardar tus cambios.

Personalizando tus informes de estadísticas

Para personalizar tus informes para cualquier dominio:

  1. Navega a la página Site Statistics.
  2. Haz clic en el botón Configure a la derecha del dominio.
  3. En la siguiente página, haz clic en el botón Edit a la derecha de cualquier reporte ‘Month’, ‘Day’, o ‘Longterm’.

  4. Una vez que hayas personalizado tu informe de estadísticas, haz clic en el botón Save Changes para guardar tu nueva configuración.

Accediendo a tu URL de estadísticas

Una vez que las estadísticas están activas para un dominio, puedes acceder a ellas cargando el subdirectorio /stats en tu navegador. Por ejemplo, si acabas de habilitar las estadísticas en el dominio example.com, entonces visitarías example.com/stats.

  • Aparece un mensaje de inicio de sesión después de ingresar a example.com/stats.
  • Si estás utilizando un CMS como WordPress, es posible que debas agregar un código adicional al archivo .htaccess de tu sitio para acceder a la página de inicio de sesión /stats.
  • Ve el artículo hacer las estadísticas accesibles con .htaccess para más instrucciones.
