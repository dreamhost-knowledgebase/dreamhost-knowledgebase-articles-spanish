Generalidades

Cuando agregas por primera vez un dominio a tu panel como Completamente Alojado, debes asignarle un directorio web. Esta será la carpeta en el servidor donde se encuentren los archivos del dominio.

En general, nunca deberías necesitar cambiar este directorio. Solo ciertas configuraciones personalizadas requerirán que lo hagas. Este artículo te muestra cómo cambiar el directorio web de tu dominio. 

Cambiar el directorio web de tu dominio

  1. Navega a la página Administrar Dominios.
  2. Haz clic en el enlace Edit a la derecha de su dominio, debajo de la columna Web Hosting.
  3. En la sección Users, Files, and Paths, busca el campo de texto del Web directory.
  4. En el campo de texto, ingresa el nuevo directorio en el servidor desde el que deseas cargar este dominio. Por ejemplo: example.com/blog.

    Ten en cuenta que este campo distingue entre mayúsculas y minúsculas, y la carpeta debe existir en el servidor bajo el usuario de tu dominio.

  5. Desplázate hacia abajo y haz clic en el botón Cambiar adjustes para guardar.
