Generalidades

Si no seleccionaste la opción para mover tus sitios y usuarios al VPS cuando realizaste el pedido, aún puedes moverlos más adelante.

Asegúrate de que tu nuevo VPS tenga suficiente espacio para migrar los contenidos de tu usuario. Cuando compras un nuevo VPS, puedes almacenar de 30GB a 240GB. Revisa la página Uso de Disco para confirmar que tus usuarios encajarán en el nuevo servidor.

Esta página puede ser accessada desde Administrar Cuenta.

Migrar tus sitios y usuarios

Cuando agregas un segundo VPS, no podrás mover automáticamente los datos de un VPS existente al nuevo VPS. Sin embargo, tu puede migrar tus usuarios y dominios después de aprovisionar el VPS usando estas instrucciones.

  1. Navega hasta la página VPS.
  2. Sobre el botón Agregar VPS, haz clic en el enlace titulado Haz clic aquí para mover todos los usuarios y dominios.
  3. Selecciona el VPS deseado del menú desplegable.
  4. Haz clic en el botón Mover todos los usuarios y dominios shared.

Tus usuarios y dominios se migran a tu VPS elegido.

Asegúrate de que el VPS tenga suficiente espacio para migrar todos tus usuarios y dominios. Si no hay suficiente espacio en el VPS, la migración falla y es posible que no pueda iniciar sesión en el servidor privado.

Puedes ver cuanto espacio necesitas revisando el valor de "Uso de Disco" en la página Usuarios & Archivos FTP. Da clic en el botón Mostrar Info al lado de usuario para mostrarla. El valor del Uso de Disco se muestra debajo del usuario.

Mover dominios específicos de Shared Hosting a un VPS

Solo puedes mover a todos los usuarios (y sus dominios) o ninguno. No puedes mover un solo usuario a la vez a un VPS.

Para mover un solo dominio a la vez desde un servidor compartido a un VPS, revisa el siguiente artículo:

  • ¿Cómo mover un solo dominio del Shared Hosting a un servidor privado?

¿Qué pasa con el correo electrónico?

Mover un usuario y dominio de un sitio web a un nuevo servidor no afecta el correo. Esto se debe a que el correo está en un servidor de correo completamente diferente. Tu correo seguirá funcionando normalmente.
