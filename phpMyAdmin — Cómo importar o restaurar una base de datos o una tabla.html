Generalidades

Este artículo explica cómo importar una base de datos o tabla usando phpMyAdmin.

Requisitos previos

  • Este artículo asume que ya has creado una base de datos en tu panel de DreamHost. No es posible crear una base de datos a través de SSH.
  • Para importar el archivo de respaldo, necesitarás las credenciales de tu base de datos. Revisa este artículo para obtener más información sobre cómo encontrar las credenciales de la misma.

Importar una base de datos o tabla

En los siguientes pasos, asegúrate de seleccionar tu archivo de base de datos o tabla .sql en el paso #4.

Por favor anota lo siguiente cuando importes:

  • Una sola tabla — Solo se puede importar una tabla si no existe en la base de datos actual. Si la tabla existe, se generará un error y la importación no se completará.
  • Una base de datos — Solo puede importar contenido de base de datos a una base de datos que ya se haya creado en el panel y que esté actualmente vacía.
  1. Inicia sesión en phpMyAdmin.
  2. Selecciona la base de datos de destino en el panel izquierdo.
  3. Haz clic en la pestaña Importar en el panel central superior.
  4. En la sección Archivo a importar, haz clic en Seleccionar archivo y localiza el archivo con la extensión .sql que deseas importar.
  5. Selecciona o deselecciona las casillas de Importación parcial y Otras opciones.
  6. En el menú desplegable Formato, selecciona SQL.
  7. Haz clic en el botón Continuar en la parte inferior para importar la base de datos.

Tu nueva base de datos o tabla apareceen el menú izquierdo de phpMyAdmin.

¿Qué pasa si el archivo es demasiado grande para importar a través de phpMyAdmin?

Si el archivo .sql que intentas importar es más grande de lo que permite phpMyAdmin, la única otra opción es importarlo mediante SSH. Revisa este artículo para obtener más información.
