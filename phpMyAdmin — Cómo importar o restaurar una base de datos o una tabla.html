Generalidades

Este artículo explica como importar lo siguiente:

  • Una sola tabla (que no existe) en una base de datos (que ya existe en el panel)
  • Todo el contenido de la base de datos en una base de datos vacía (que ya existe en el panel)

Primero debe screar la base de datos en tu panel DreamHost antes de importar contenido.

No es posible crear una base de datos a través de SSH.

  • Crear una base de datos MySQL  

Importar una tabla

Solo puedes importar una tabla si no existe en la base de datos actual.

Si la tabla existe, se produce un error y la importación no se completará.

Importar una base de datos

Solo puedes importar el contenido de la base de datos a una base de datos que ya se ha creado en el panel y que actualmente está vacía.

Ubicar las credenciales de tu base de datos

Visita el siguiente artículo para obtener detalles sobre cómo encontrar tus credenciales de inicio de sesión e inicia sesión en phpMyAdmin:

  • Encontrar tus credenciales de inicio de sesión de phpMyAdmin

Este artículo asume que la base de datos ya existe en tu panel de DreamHost.

Importar o restaurar una base de datos o tabla

  1. Inicia sesión en phpMyAdmin.
  2. Selecciona la base de datos de destino en el panel izquierdo.
  3. Haz clic en la pestaña Importar en el panel central superior.
  4. En la sección Archivo a importar, haz clic en Seleccionar archivo y localiza el archivo con la extensión .sql que deseas importar.
  5. Selecciona o deselecciona las casillas de 'Importación parcial' y 'Otras opciones'.
  6. En el menú desplegable Formato, selecciona 'SQL'.
  7. Haz clic en el botón Continuar en la parte inferior para importar la base de datos.

Si importaste una base de datos, todas las tablas se mostrarán al seleccionar la base de datos en el panel izquierdo de phpMyAdmin.

Si importaste una tabla, la tabla recién importada aparecerá en la lista de tablas.

¿Qué pasa si el archivo es demasiado grande para importar a través de phpMyAdmin?

Si el archivo .sql que intentas importar es más grande de lo que permite phpMyAdmin, la única otra opción es importar a través de SSH. Visita el siguiente artículo para obtener instrucciones de cómo realizar esto:

  • SSH — Importar una base de datos
