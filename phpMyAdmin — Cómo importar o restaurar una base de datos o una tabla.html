Este artículo asume que la base de datos ya existe en tu panel de DreamHost.

Importar o restaurar una base de datos o tabla

  1. Inicia sesión en phpMyAdmin.
  2. Selecciona la base de datos de destino en el panel izquierdo.
  3. Haz clic en la pestaña Import en el panel central superior.
  4. En la sección File to import, haz clic en Choose File y localiza el archivo con la extensión .sql que deseas importar.
  5. Selecciona o deselecciona las casillas de 'Importación parcial' y 'Otras opciones'.
  6. En el menú desplegable Format, selecciona 'SQL'.
  7. Haz clic en el botón Go en la parte inferior para importar la base de datos.

Si importaste una base de datos, todas las tablas se mostrarán al seleccionar la base de datos en el panel izquierdo de phpMyAdmin.

Si importaste una tabla, la tabla recién importada aparecerá en la lista de tablas.

¿Qué pasa si el archivo es demasiado grande para importar a través de phpMyAdmin?

Si el archivo .sql que intentas importar es más grande de lo que permite phpMyAdmin, la única otra opción es importar a través de SSH. Visita el siguiente artículo para obtener instrucciones de cómo realizar esto:
