Generalidades

Tener actualizados los archivos principales, temas, y plugins de WordPress es lo mejor que puedes hacer para mantener tu sitio web. Un software actualizado es menos propenso a los intentos de hackeo y, a menudo, ofrece optimización y aumentos de velocidad.

Actualizaciones de DreamHost

Todas las instalaciones de WordPress de DreamHost habilitan las actualizaciones automáticas por defecto, pero puedes deshabilitar este servicio si prefieres actualizar tu sitio WordPress tú mismo.

Funciones de actualización automática

Las actualizaciones automáticas de DreamHost solo están disponibles para sitios con un instalación de WordPress de DreamHost activa. Las actualizaciones automáticas incluyen las siguientes funciones:

  • DreamHost realiza una actualización de tu sitio de WordPress entre 1 y 3 semanas después del lanzamiento de una versión principal. Si deseas actualizar manualmente antes de esa fecha, puedes hacerlo actualizándolo manualmente en tu panel de WordPress.
  • DreamHost también actualiza los plugins y temas durante este proceso.
  • Si se encuentran errores durante el proceso de actualización, DreamHost revertirá la actualización y te notificará por correo.

Las actualizaciones automáticas de WordPress

Las actualizaciones automáticas de WordPress son independientes de las actualizaciones automáticas de DreamHost. Dado que suelen incluir correcciones de errores y/o actualizaciones de seguridad, DreamHost recomienda encarecidamente que permitas que estas actualizaciones se hagan automáticamente. Las siguientes versiones incluyen actualizaciones automáticas por parte de la aplicación WordPress.

  • Las versiones de WordPress 3.7 y siguientes — actualizaciones automaticas para todos los lanzamientos de revisión menores & mayores
  • Las versiones de WordPress 5.6 y siguientes — actualizaciones automaticas para todos los lanzamientos menores & mayores

Desactivar las actualizaciones automáticas de DreamHost no cambia este comportamiento

Deshabilitar las actualizaciones automáticas de DreamHost

Si prefieres actualizar manualmente tu instalación de WordPress, puedes usar las siguientes instrucciones para optar por no recibir actualizaciones automáticas eliminando tu sitio de la lista instalaciones de WordPress de DreamHost:

  1. Navega a la página Instalar WordPress.
    La página Instalar WordPress se abre.
  2. Haz clic en el botón Gestionar mis aplicaciones.
  3. Da clic en el dominio para expandir los ajustes de sus configuraciones.
  4. Haz clic en el botón Quitar de la Lista.
  5. Aparece un mensaje de advertencia que te notifica que esta instalación de WordPress ya no será administrada por DreamHost. Da clic en Ok para confirmar.

Actualizar en el futuro

Dando clic al botón Quitar de la Lista elimina la capacidad que tiene DreamHost para actualizar automáticamente la versión de WordPress.

En el futuro, debes revisar continuamente el Panel de Actualizaciones para ver las que están disponibles. Revisa el siguiente artículo para más instrucciones:

  • Actualizar WordPress Manualmente

¿Cómo puedo habilitar nuevamente las actualizaciones automáticas de DreamHost?

Contacta al soporte de DreamHost para pedir que el sitio que no está administrado por nosotros se revierta a una instalación de WordPress administrada.
