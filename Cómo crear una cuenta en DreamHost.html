Generalidades

Este artículo explica cómo crear una nueva cuenta en DreamHost. Este artículo está dirigido a clientes nuevos que deseen abrir una cuenta con sus propios datos personales, información de método de pago, y un nuevo plan de alojamiento web.

Este artículo muestra cómo abrir una cuenta con un plan de alojamiento. Esto quiere decir que pagarás por un servidor donde se alojará tu sitio. Si solamente quieres comprar el nombre del sitio (registro de nombre del dominio), y no estás listo para agregar alojamiento, puedes revisar la siguiente guía:

  • Cómo comprar el registro de un nombre de dominio

Correo Electrónico

Si deseas crear una cuenta solamente para tus correos electrónicos, y no quieres alojar tu sitio web en línea, puedes seguir las instrucciones en este enlace:

  • https://www.dreamhost.com/products/email/

Paso 1 — Escoge un plan de alojamiento

DreamHost ofrece varios planes de alojamiento para tu sitio web. Visita el siguiente enlace para ver la lista de planes disponibles:

  • Planes de Alojamiento

Si quieres alojar un sitio WordPress, revisa la página a continuación, ya que muestra una lista de las opciones de alojamiento para WordPress:

  • Alojamiento para WordPress

Cuando hayas decidido qué plan quieres, haz clic en el botón Sign up now debajo del plan que deseas para continuar con la creación de la cuenta.

Paso 2 — Crear una cuenta nueva

Después de haber seleccionado tu plan, puedes continuar creando la cuenta. Los pasos son generalmente los mismos para todos los planes de alojamiento, pero encontrarás un ejemplo del plan Starter Shared a continuación para mayor claridad.

Creando una Cuenta de alojamiento compartido

  1. Cuando decidas qué plan quieres comprar, haz clic en el botón debajo del plan para seguir. Al hacer clic en el botón, la página te llevará a panel.dreamhost.com/signup.
  2. Comprar el registro de un nombre de dominio, transferir un dominio existente, o escoger un dominio después.
  3. Hacia la derecha, selecciona el plan de pago y los servicios adicionales que desees.
  4. Selecciona si vas a realizar tu pago con tarjeta de crédito/débito o vía PayPal. Si escoges Paypal, serás redireccionado a la página de Paypal para completar la compra.
  5. Introduce una dirección de correo electrónico. Este será el correo electrónico principal que utilizarás para iniciar sesión en tu panel de DreamHost. También puedes registrarte utilizando el método de 'Inicio de Sesión Único' de Google. Esto te permite usar tu dirección de correo electrónico de Google para iniciar sesión sin utilizar una contraseña.
  6. Hacia la parte de abajo, hay algunas opciones que puedes habilitar:
    • Pre-Install WordPress: Selecciona esta opción para que WordPress sea instalado automáticamente en tu nuevo dominio.
    • DreamShield Protection: Este es un servicio adicional pago que escanea tu sitio semanalmente en busca de software malicioso.
    • DreamHost Email: Esta opción solo está disponible para el plan Starter Shared. Te permite agregar alojamiento de correo electrónico además del plan de alojamiento web. (NOTA: Todos los otros planes ofrecen alojamiento de correo electrónico incluido).
    • G Suite: esto agrega G Suite a tu dominio. Luego puedes usar el conjunto de herramientas de Google para administrar tu sitio web y correo electrónico.
  7. Cuando termines, haz clic en el botón Submit Order al final de la página.

Si tienes preguntas adicionales, por favor contacta al área de soporte de DreamHost.

  • Contactar Soporte
