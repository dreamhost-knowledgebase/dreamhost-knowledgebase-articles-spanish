Generalidades

Este artículo explica cómo crear una nueva cuenta en DreamHost. Este artículo está dirigido a clientes nuevos que deseen abrir una cuenta con sus propios datos personales, información de método de pago, y un nuevo plan de alojamiento web.

Este artículo muestra cómo abrir una cuenta con un plan de alojamiento. Esto quiere decir que pagarás por un servidor donde se alojará tu sitio. Si solamente quieres comprar el nombre del sitio (registro de nombre del dominio) y no estás listo para agregar alojamiento, puedes revisar la siguiente guía:

  • Cómo comprar el registro de un nombre de dominio

Correo Electrónico

Si deseas crear una cuenta solamente para tus correos, y no quieres alojar tu sitio web en línea, puedes seguir las instrucciones en este enlace:

  • dreamhost.com/es/productos/correo-electronico/
 

Paso 1 — Escoge un plan de alojamiento

DreamHost ofrece varios planes de alojamiento para tu sitio web. Visita el siguiente enlace para ver la lista de planes disponibles:

  • Planes de Alojamiento

Si quieres alojar un sitio WordPress, revisa la página a continuación, ya que muestra una lista de las opciones de alojamiento para WordPress:

  • Alojamiento para WordPress

Cuando hayas decidido qué plan quieres, haz clic en el botón Regístrate Ahora debajo del plan que deseas para continuar con la creación de la cuenta.

Paso 2 — Crear una cuenta nueva

Después de haber seleccionado tu plan, puedes continuar creando la cuenta. Los pasos son generalmente los mismos para todos los planes de alojamiento, sin embargo, encontrarás un ejemplo del plan Shared Starter a continuación para mayor claridad.

Agregar una Cuenta de Shared Hosting

  1. Cuando decidas qué plan quieres comprar, haz clic en el botón debajo del plan para seguir. Al hacer clic en el botón, la página te llevará a Escoge tu Plan de Alojamiento.
  2. Aquí puedes comprar el registro de un nombre de dominio, transferir un dominio existente, o escoger un dominio después.
  3. Hacia la derecha, selecciona el plan de pago y los servicios adicionales que desees.
  4. Selecciona si vas a realizar tu pago con tarjeta de crédito/débito o vía PayPal. Si escoges Paypal, serás redireccionado a la página de Paypal para completar la compra.
  5. Ingresa una dirección de correo. Esta será tu dirección principal para iniciar sesión en tu panel de DreamHost. También puedes usar el método 'inicio de sesión único de Google (SSO)'. Esto te permite usar tu dirección de correo de Google para iniciar sesión sin usar una contraseña.

    Algunos navegadores están habilitados con configuraciones de ventanas emergentes predeterminadas que deshabilitan la visualización de ventanas emergentes. Si es así, debes permitir que tu navegador muestre ventanas emergentes manualmente. Los siguientes enlaces muestran cómo hacer esto con navegadores populares.

    • Firefox
    • Safari
  6. Selecciona opciones adicionales que aparecen hacia la parte inferior:
    • Preinstalar WordPress: Selecciona esta opción para que WordPress sea instalado automáticamente en tu nuevo dominio.
    • Protección DreamShield: Este es un servicio adicional pago que escanea tu sitio semanalmente en busca de software malicioso.
    • El Kit de SEO de DreamHost: Este es un plugin de pago que te ayuda a optimizar tu sitio web para obtener una clasificación más alta en los motores de búsqueda. También continúa monitoreando tu sitio a lo largo del tiempo para garantizar que la puntuación de SEO de tu sitio permanezca alta.
    • Correo de DreamHost: Esta opción solo está disponible para el plan Shared Starter. Te permite agregar alojamiento de correo además del plan de alojamiento web. (NOTA: Todos los otros planes ofrecen alojamiento de correo incluido).
    • Google Workspace: Esto agrega Google Workspace a tu dominio. Luego puedes usar el conjunto de herramientas de Google para administrar tu sitio web y correo.
  7. Cuando termines, haz clic en el botón Enviar Orden al final de la página.
