General

¿Mi plan de alojamiento web incluye alojamiento de correo?

Todos los planes de alojamiento web DreamHost, excepto Remixer Standard y Shared Starter, incluyen alojamiento de correo. Si tienes Remixer Standard y deseas tener alojamiento de correo, puedes mejorar tu plan a Remixer All-Access. Si tienes Shared Starter y deseas tener alojamiento de correo , puedes mejorar tu plan a Shared Unlimited.

¿Cómo acceder a Webmail?

Por defecto, la URL para Webmail es https://webmail.dreamhost.com. Visita el siguiente artículo para más detalles:

¿El correo seguirá funcionando si mi registro de dominio ha caducado?

Lamentablemente, los correos no funcionarán. Si te ha enviado un mensaje y tu dominio expiró, el remitente recibirá una notificación de devolución de que el mensaje no se entregó. El registro del dominio debe estar activo y actualizados los records DNS usar el correo.

¿Necesito un plan de alojamiento activo para alojar el correo  en DreamHost?

No, DreamHost ofrece planes de alojamiento de solo pago por correo.

También puedes tener registrado tu dominio en DreamHost, luego dirigir su DNS a otra compañía de hosting para que use su servicio de correo. De esta manera, solo pagas por el registro del dominio y alojas su DNS de DreamHost. Consulta el siguiente artículo para obtener más información:

¿Dónde se almacena el correo?

El correo se almacena en un servidor de correo que está completamente separado de tu servidor web. No hay acceso diferente al servidor de correos que no sea usar un cliente de correo.

Recibo demasiado spam. ¿Puedo controlar esto en el panel?

DreamHost ofrece servicios Anti-spam services que puedes usar para filtrar spam. También puedes crear filtros de mensajes personalizados para bloquear mensajes específicos.

¿Puedo acceder al correo a través de una dirección IP?

No. Siempre debes usar una dirección URL. Los registros PTR no son compatibles.

Límites

¿Cuántas direcciones de correo puedo crear?

Si tienes un plan Shared Unlimited, puedes crear tantas direcciones de correo como desees. No hay un número establecido sobre cuántas direcciones puedes crear.

Si tienes un plan de solo pago por correo , puedes crear tantas direcciones de correo como hayas inscrito.

¿Existe un límite en la cantidad de espacio en disco que puede usar una dirección?

25GB es el límite actual. Sin embargo, debes mantenerlo por debajo de 1 GB para evitar problemas con la carga de tus carpetas. Cuando creas por primera vez una dirección de correo, el límite de disco se establece de manera predeterminada en 200M para el buzón. Puedes aumentar el tamaño del disco o eliminar el límite del disco. Puedes encontrar más información sobre el uso del disco en el siguiente artículo:

  • Advertencia de uso del disco de correo

¿Puedo usar mi dirección de correo para enviar mensajes masivos?

No. Visita el artículo de límites de cuota SMTP para ver las limitaciones. Si deseas enviar un mensaje masivo como un boletín informativo, puedes utilizar el servicio de Lista de Anuncios.

Listas

¿Qué es una Lista de Anuncios?

Esta es una Lista de suscripción que puedes crear y que te permite enviar un solo correo masivo a tus clientes. Visita el artículo de Listas de Anuncios para más información.

¿Qué es una Lista de Discusión?

Esto es similar a algo así como un mensaje de texto grupal. Es una forma de tener una conversación grupal por correo donde todos en la lista reciben respuestas de cualquiera que responda dentro del grupo. Visita el siguiente artículo para más detalles:

  • Listas de Discusión

¿Cuál es la diferencia entre una Lista de Anuncios y una Lista de Discusión?

Una Lista de Anuncios es un servicio que se utiliza cuando deseas enviar un solo correo a todos tus clientes. Los clientes no se comunican entre sí con una Lista de Anuncios. Una Lista de Discusión es cuando un solo usuario puede enviar un correo a una sola dirección que se entrega a todos los usuarios. De esta manera, los usuarios de la lista pueden comunicarse entre sí.
