Generalidades

DreamPress es el alojamiento administrado de WordPress en DreamHost. Es un servicio escalable que permite a los usuarios administrar sus sitios de WordPress con precios simples, beneficios adicionales de robustez del servidor, almacenamiento en caché y memoria.

Detrás de escena, DreamPress une diferentes piezas de la tecnología existente de DreamHost en una sola unidad cohesiva. A continuación se detallan todos los detalles de este nuevo servicio.

Detalles técnicos

Cada instancia de DreamPress se configura en el servicio de computación en la nube de DreamHost llamado DreamCompute. Nuevos servicios DreamPress:

  • se aprovisionan automáticamente
  • no permite acceso root, sudo y FTP (si puedes iniciar sesión a través de SFTP o SSH)
  • viene con un certificado seguro gratuito 'Let's Encrypt' instalado

Además, DreamHost tiene almacenamiento en caché incorporado, que:

  • se configura automáticamente para el cliente DreamPress
  • responde a las solicitudes web como proxy frente a la pila de WordPress de Apache y PHP
  • es extremadamente eficiente en el almacenamiento de contenido en caché
  • permite esta configuración para escalar a niveles muy altos de tráfico
  • viene con una IP estática

Restricciones del servicio

DreamPress es un servicio administrado de WordPress, lo que significa que DreamHost ha restringido ciertas acciones que los usuarios pueden tomar que podrían romper irreparablemente su sitio de WordPress.

Cada plan DreamPress está configurado para un dominio único junto con un sitio de vista previa. También puedes mezclarlo y combinarlo con alojamiento compartido (Shared), VPS y servidor Dedicado.

Restricciones actuales

  • No se puede configurar manualmente la memoria. La memoria está configurada y escala si es necesario.
  • No hay usuarios administradores ni claves SSH en root.
  • No se ofrece acceso FTP. Puedes tener un usuario de SFTP (SSH opcional) y el dominio que elijas se configurarán automáticamente.
  • No se pueden agregar o eliminar bases de datos o nombres de host. La base de datos y el nombre de host se crearán automáticamente para WordPress.
  • No se pueden agregar wildcard para DNS.
  • No se puede eliminar el certificado SSL instalado automáticamente.
  • No se puede ajustar la configuración HTTPS.
  • Las instalaciones de red (multisitio) no son compatibles.
  • DreamHost instala y actualiza automáticamente una pequeña cantidad de plugins necesarios.

Versión de WordPress

Dado que este es un servicio administrado, no es posible controlar la versión de WordPress. Cuando hay una nueva actualización disponible, el panel actualizará automáticamente tu sitio.

Soporte al usuario

DreamPress incluye el mejor soporte técnico de DreamHost, que también tienen experiencia en WordPress. Después de una capacitación y evaluación intensiva por parte de los colaboradores principales de WordPress, fueron seleccionados como los mejores en su clase para representar DreamHost y WordPress, y ayudarte con cualquier problema con tu sitio DreamPress. Si no saben la respuesta, saben cómo encontrar a alguien que sí lo sabe.

Dicho todo esto, ten en cuenta que el soporte de DreamHost no es un reemplazo para un desarrollador de sitio web real (si llegaras a necesitarlo); DreamHost puede ayudar a instalar / configurar plugins, pero no puede crear plugins o temas personalizados para ti. Además, si un plugin o tema está causando problemas, el único soporte que DreamHost puede proporcionar es deshabilitar el plugin/tema y sugerir alternativas.
