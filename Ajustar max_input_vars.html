Generalidades

La directiva max_input_vars controla cuántas variables de entrada pueden aceptarse en PHP.

  • php.net/manual/en/info.configuration.php#ini.max-input-vars

De forma predeterminada, se establece en 1000, pero tu sitio web puede generar un error que te informa que este valor es demasiado bajo. Puedes aumentar este valor en un archivo phprc utilizando los siguientes pasos.

Ajustando max_input_vars

  1. Edita tu archivo phprc usando FTP o SSH.
  2. En ese archivo, agrega la siguiente línea. Esto eleva el límite a 2000. 
    max_input_vars = 2000
  3. Elimina todos los procesos PHP para actualizar tu nueva configuración.
  4. Verifica tu nueva configuración utilizando un archivo phpinfo.php.
