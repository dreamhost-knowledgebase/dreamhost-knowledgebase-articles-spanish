En la página VPS, puedes agregar un nuevo VPS o administrar uno existente.

Si ya tienes un VPS, puedes hacer clic en el botón Configure a la derecha. Esto abre una nueva página donde puedes actualizar lo siguiente:

  • Cambiar el nombre del servidor
  • Establecer claves SSH de raíz (solo disponible en cuentas antiguas con un usuario administrador. Las cuentas nuevas no permiten usuarios administradores).
  • Cambiar el servidor web de Apache a Nginx
  • Desactivar FTP
  • Activar el servidor POP3/IMAP

Para más información sobre servidores privados, ve la siguiente categoría:
