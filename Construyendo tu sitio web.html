Generalidades

Hay algunos pasos importantes para construir un sitio web y mostrarlo en línea.

Este artículo cubre cómo configurar un sitio web y cómo atraer visitantes una vez que se sube en los servidores web de DreamHost.

Configurando tu sitio web

  1. Obten un registro de dominio.
    • Para comprar uno en el panel, ve a la página Registrations.
    • También puedes transferir un registro de dominio existente a DreamHost. Consulta el artículo Transfiere tu registo de dominio a DreamHost para más instrucciones.
    • Si tienes un dominio registrado a través de otro proveedor y no deseas transferir el registro, asegúrate de apuntar tu DNS a DreamHost.
    • Si aún no estás listo para apuntar tu DNS a DreamHost, hay otras maneras de ver tu sitio DreamHost antes de camibiar tu DNS.

    Debes configurar DNS en la empresa donde está registrado tu sitio. El soporte de DreamHost no puede ayudar con esto, ya que tu eres el propietario del dominio, no DreamHost.

  2. Añade un plan de alojamiento.
  3. Añade alojamiento a tu sitio.
  4. Crea tus páginas web en la computadora de tu hogar o usa una instalación de un clic para crear el sitio. A continuación hay algunos sitios para ayudarte.
    • W3Schools ofrece tutoriales gratuitos sobre cómo construir un sitio web.
    • También puedes usar un programa WYSIWYG como Dreamweaver.
    • HTML Goodies (un sitio de recursos) – http://www.htmlgoodies.com/
    • Tutoriales gratuitos para principiantes, intermedios y avanzados de XHTML y CSS – http://htmldog.com/

    Otra forma de crear un sitio web en DreamHost es usar una instalación de un clic. Vea los siguientes artículos para más detalles:

    • Cómo instalar una instalación de un Clic
    • Instalaciones de un Clic disponibles
  5. Si has desarrollado el sitio en la computadora de tu hogar, sube el contenido a tu servidor DreamHost usando un cliente FTP.
  6. Una vez que tu dominio esté registrado, alojado y hayas subido tu contenido al servidor web, tu sitio estará en vivo en Internet para que cualquiera lo visite.

Glosario de términos que se muestran en los pasos anteriores

Registro de Dominio
has pagado y estás registrado como propietario de un nombre de dominio específico, como example.com.
Alojamiento de Dominio
le has pagado a una empresa, como DreamHost, para almacenar los archivos de tu sitio web en su servidor.
Transfiriendo un dominio
estás cambiando la información de propiedad del registro de dominio para que tu sitio esté registrado en DreamHost.

¿Qué sucede si veo una lista de directorios vacía donde debería estar mi sitio web?

Cuando agregues el alojamiento por primera vez a tu sitio, verás una página ‘Coming Soon’ que DreamHost crea de manera predeterminada. Para subir un sitio que ya has creado, visita la página FTP para obtener instrucciones detalladas.

Si ya has subido los archivos de tu sitio, es posible que solo veas una lista de directorio de esos archivos y no el sitio web real. Si ese es el caso, debes crear una página de índice, como una página index.html o index.php. Esta página de índice se usa para cargar automáticamente tu sitio cuando se visita.

¿Cómo hago para que la gente visite mi sitio?

Una vez que tu sitio esté en funcionamiento, debes publicarlo. Nadie lo visitará si no puede encontrarlo a través de un motor de búsqueda o de otro sitio relacionado. A continuación hay algunos pasos para ayudar a que esto suceda:

  1. Visita cada uno de los principales motores de búsqueda y envía tu URL. La mayoría de estos sitios tienen un enlace que te permite enviar tu sitio a su base de datos. Algunos motores de búsqueda populares incluyen:
    • Google
    • Bing
    • Yahoo
    • Northern Lights
    • Dogpile

    También hay muchos otros servicios (gratuitos o de pago) que envían tu sitio a los motores de búsqueda por ti.

  2. Para facilitar que los motores de búsqueda rastreen y clasifiquen tu sitio, considera agregar algunas etiquetas META a tu página de índice.
