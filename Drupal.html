Generalidades

Drupal es un sistema de gestión de contenido (o CMS) que puede usar para construir tu sitio web. No requiere experiencia en codificación y ofrece varios modúlos y extensiones para expandir la funcionalidad de tu sitio.

Para utilizar Drupal, debes instalar manualmente este programa tu mismo siguiendo los pasos a continuación.

Servidores VPS o Dedicados son recomendados

Dado que Drupal usa muchos recursos, se recomienda que solo instales Drupal en un Servidor VPS o Dedicado. Si eliges ejecutar Drupal en un servidor compartido, puedes experimentar problemas de rendimiento.

Habilita OPcache antes de instalar Drupal

Cuando instalas Drupal, verifica ciertas especificaciones en tu servidor. Mostrará una advertencia si OPcache no está habilitado, por lo que se recomienda habilitarlo con fines de rendimiento. Consulta el siguiente artículo para obtener instrucciones sobre cómo habilitar OPcache:

  • Generalidades sobre OPcache

Instalar Drupal 8.8.1

Puedes descargar el archivo de instalación de Drupal usando un cliente FTP o usando SSH. Si estás usando un cliente FTP, visita la página de descarga de Drupal y descarga el archivo .zip en tu computador. Extráelo, y luego sube tu contenido a tu servidor web.

Los siguientes pasos muestran cómo descargar el archivo a través de SSH.

  1. Inicia sesión en tu servidor por medio de SSH.
  2. Corre el siguiente comando para descargar el archivo Drupal. 
    [server]$ wget https://www.drupal.org/download-latest/tar.gz
  3. Descomprime el archivo. 
    [server]$ tar -zvxf tar.gz
    Esto crea un nuevo directorio llamado drupal-8.8.1.
  4. Navega a este directorio, luego mueve todos los contenidos un nivel al directorio principal de tu sitio. 
    [server]$ cd drupal-8.8.1
[server]$ mv * .[^.]* ..
  5. Crea una base de datos MySQL en tu panel. Vista el artículo de MySQL para más instrucciones.
  6. Visita tu página donde cargaste los archivos de Drupal para inciar la instalación. 
  7. Durante el proceso de instalación, deberás conocer las credenciales de tu base de datos. También debes cambiar el campo 'Database Host' de localhost a tu nombre de host de base de datos DreamHost real. Mira el artículo phpMyAdmin para obtener información sobre cómo obtener las credenciales de tu base de datos.

No es necesario cambiar el puerto de la base de datos, pero el puerto de la base de datos DreamHost es 3306. Puedes escoger luego un a prefijo de tabla apropiado si te sientes inclinado a hacerlo.

Actualizar a un servidor privado MySQL

Drupal puede usar muchos recursos del servidor MySQL. Si notas lentitud, es posible que desees actualizar a un servidor privado MySQL.

Multisitio

Visita la guía de multi-sitios de Drupal para obtener instrucciones sobre esta configuración. Ten en cuenta que el soporte de DreamHost no puede ayudar con los errores que puedan ocurrir ya que se trata de una configuración personalizada.
