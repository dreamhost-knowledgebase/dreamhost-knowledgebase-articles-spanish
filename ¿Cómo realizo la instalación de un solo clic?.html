Generalidades

Este artículo cubre el proceso de instalación de una aplicación utilizando el instalador de un solo clic en el panel.

Para realizar la instalación de un solo clic, tu dominio primero debe estar 'completamente alojado'.

Esto significa que no se puede estar Redirected, Mirrored, Parked, o configurado como 'DNS Only'.

Instalar una aplicación

Si ya subiste archivos antes de usar el siguiente instalador de un clic, este cambiará el nombre de los que cause problemas y dejará todo lo demás intacto. También se te enviará un correo electrónico con la siguiente mención (los nombres de los archivos varían según el contenido de tu directorio):

Si ya tenías archivos que en su lugar que fueron sobrescritos por los archivos oficiales de WordPress, ¡no te preocupes! Se han guardado automáticamente con un sufijo xxxxxxxx.

Durante el proceso de instalación, cambiamos automáticamente el nombre de algunos archivos de índice antiguos que habíamos encontrado ('disabled_by_installer' se agregó al final de los nombres de archivo): index.htmll

Para instalar una aplicación desde la página de instalaciones de un clic:

  1. Ve a la página de One-Click Installs.
    Se abre la página Instalaciones de un solo clic que muestra un catálogo de aplicaciones disponibles:
  2. Haz clic en el icono de la aplicación sobre la que deseas obtener más información o instalar.
    Se abre la ventana de la aplicación:
    • La ventana de la aplicación muestra detalles de esta, como la versión actual y las calificaciones.
  3. Cuando estés listo para continuar con la instalación, selecciona el dominio que desees en el menú desplegable 'Instalar en'.
    También tienes la opción de instalar la aplicación en un subdirectorio o carpeta como se muestra a continuación:
    • Tienes la opción de usar una base de datos existente para esta aplicación o hacer que el panel cree una nueva.
  4. Haz clic en el botón Install it for me now! para proceder con la instalación.
    Una vez que hayas hecho clic en el botón Instalar, aparece un aviso con el mensaje de ¡éxito!:

Después de haber instalado el One-Click en el panel

Después de unos 10 minutos, debes recibir un correo electrónico con instrucciones para iniciar sesión. Sigue las instrucciones para finalizar la instalación de tu sitio.

Para WordPress, puedes iniciar sesión en example.com/wp-login.php.

Administrar tu instalación de un solo clic

Una vez que hayas instalado la aplicación del sitio, puedes verla en el panel en la página One-Click Installs.
