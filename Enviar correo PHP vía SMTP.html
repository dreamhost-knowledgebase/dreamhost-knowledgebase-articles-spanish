Generalidades

Cuando usas la funcionadolidad de PHP mail para enviar correo desde tu sitio web, estás enviando correo directamente desde tu servidor web. Este método no es recomemdado pues puede causar inconvenientes si el correo del REMITENTE no está configurado correctamente o si el mismo no está alojado en DreamHost.

Revisa el artículo Solución de problemas de PHP mail para más detalles.

SMTP mail

Enviar correo vía SMTP es recomendado pues este método envía correo desde el servidor de correo más que del servidor web. Esto ayuda significativamente a asegurar que tu correo se envíe y entregue correctamente.

PHPmailer

DreamHost recomienda usar PHPmailer para enviar correo vía SMTP.

  • PHPMailer está muy bien documentado con varios ejemplos en línea y páginas de ayuda.
  • PHPMailer se integra bien con los servidores de SMTP de Google SMTP.
