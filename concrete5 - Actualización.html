Generalidades

Este artículo explica cómo actualizar la versión de tu sitio de concrete5. Tienes dos opciones para actualizar.

  • Actualizar usando un cliente FTP (fácil)
  • Actualizar a través de SSH (avanzado)

Actualizar a través de FTP

  1. Mira el siguiente enlace para obtener instrucciones sobre cómo descargar concrete5 usando un cliente FTP.
    • FTP
  2. Sigue todos los pasos hasta extraer el contenido del archivo .zip.
  3. Una vez extraído, navega al directorio titulado concrete5.x.x.x. Verás una lista de archivos y directorios. Elimina todo excepto el directorio llamado concrete.
  4. En tu cliente FTP, navega hasta el directorio sobre el directorio concrete5.x.x.x. Esto significa que deberías ver el nombre del directorio en tu cliente FTP en el panel izquierdo. Por ejemplo:
  5. Carga este directorio en el directorio /update de tu sitio en el panel derecho.

    Esto llevará varios minutos, ya que hay muchos archivos para cargar.

  6. Inicia sesión en tu sitio en example.com/index.php/login.
  7. Haz clic en el icono de menú en la parte superior derecha.
  8. Selecciona System & Settings del menú emergente.
  9. Bajo Update concrete5 haz clic en Apply Update.
  10. Deberías ver la nueva versión disponible. Haz clic en Install Update.
  11. Aparece un mensaje de confirmación una vez que se completa la actualización.

Actualizar a través de SSH

  1. Ve el siguiente enlace para obtener instrucciones sobre cómo descargar concrete5 a través de SSH.
    • SSH
  2. Sigue todos los pasos hasta extraer el contenido del archivo .zip.
  3. Una vez extraído, navega al directorio titulado concrete5.x.x.x. Verás una lista de archivos y directorios. Usa el comando rm para eliminar todo excepto el directorio llamado concrete. Por ejemplo: 
    [server]$ rm FILENAME
[server]$ rm -rf DIRECTORY
  4. Navega un directorio hacia arriba. 
    [server]$ cd ..
  5. Copia el directorio concrete5.x.x.x en el directorio /updates de tu sitio. Cambia username a tu usuario Shell y example.com a tu sitio website. 
    [server]$ cp -r ./concrete5.x.x.x /home/username/example.com/updates
  6. Inicia sesión en tu sitio en example.com/index.php/login.
  7. Haz clic en el icono de menú en la parte superior derecha.
  8. Elige System & Settings del menu emergente.
  9. Bajo Update concrete5 haz clic en Apply Update.
  10. Deberías ver la nueva versión disponible. Haz clic Install Update.
  11. Aparece un mensaje de confirmación una vez que se completa la actualización.
