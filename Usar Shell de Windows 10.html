Generalidades

Windows 10 ofrece un cliente Shell nativo para que lo use. En el pasado era necesario usar un cliente de terceros como PuTTY. Ahora puedes usar el Shell integrado directamente en Windows 10 para acceder a tu servidor web DreamHost sin la necesidad de descargar ningún otro software.

Esto también está habilitado de forma predeterminada, por lo que no tienes que habilitarlo dentro de Windows.

Usar el cliente SSH dentro de la Línea de Comando

Para usar el cliente SSH, abre tu Símbolo del sistema. Desde allí puedes ejecutar comandos SSH.

  1. En la parte inferior izquierda, escribe 'cmd' en la barra de búsqueda.
  2. Haz clic en el icono Símbolo del sistema para abrirlo.
  3. Visita el artículo de Generalidades SSH para obtener instrucciones sobre cómo encontrar tus credenciales de inicio de sesión. Una vez encontrado, puedes ejecutar ssh user@server.dreamhost.com para iniciar sesión.

    La primera vez que inicies sesión, puedes ver una advertencia que dice 'The server's host key is unknown'. Esto es normal cuando inicias sesión por primera vez, sin embargo, puedes verificar la clave del servidor en tu panel si lo deseas. Ver el siguiente artículo para más detalles:

    • Alerta 'The server's host key is unknown' (La llave host del servidor es desconocida)

Usar el cliente SSH con PowerShell

Abre PowerShell. Desde allí puedes ejecutar comandos SSH.

  1. En la parte inferior izquierda, escribe 'powershell' en la barra de búsqueda.
  2. Haz clic en el icono PowerShell para abrirlo.
  3. Visita el artículo Generalidades sobre SSH para obtener instrucciones sobre cómo encontrar tus credenciales de inicio de sesión. Una vez encontrado, puedes ejecutar ssh user@server.dreamhost.com para iniciar sesión.

    La primera vez que inicies sesión, puedes ver una advertencia que dice 'The server's host key is unknown'. Esto es normal cuando inicias sesión por primera vez, sin embargo, puedes verificar la clave del servidor en tu panel si lo deseas. Ver el siguiente artículo para más detalles:

    • Alerta 'The server's host key is unknown' (La llave host del servidor es desconocida)
