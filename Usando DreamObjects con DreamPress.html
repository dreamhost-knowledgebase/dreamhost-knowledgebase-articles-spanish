Generalidades

El plugin 'DreamObjects Backups' viene preinstalado con tu instalación de WordPress en DreamPress. DreamObjects es el sistema de almacenamiento basado en la nube de DreamHost, y el plugin DreamObjects Backups en WordPress te permite almacenar los datos de tu sitio en DreamObjects.

Dado que siempre es una buena idea hacer una copia de seguridad de tus datos de WordPress de forma regular, puedes configurar DreamObjects Backups para guardar automáticamente tus datos. También recibirás una prueba gratuita de 30 días para usar DreamObjects Backups.

Primero debes crear un usuario y un bucket en DreamObjects para utilizar el plugin DreamObjects Backups. Si necesitas ayuda para crear un usuario y un bucket, consulta la sección ¿Qué necesito para usar DreamObjects? en el siguiente artículo:

  • ¿Qué es DreamObjects?

A continuación se describe en tres pasos cómo activar y usar el plugin DreamObjects Backups en WordPress.

Paso 1 — Activar el plugin DreamObjects Backups

Para activar el plugin:

  1. Abre la página del panel WP-Admin para tu sitio de WordPress y haz clic en 'Plugins > Installed Plugins' en el panel izquierdo.
    Se abre la página Plugins que muestra todos los plugins disponibles para tu sitio:
  2. Haz clic en el enlace Activate debajo de los plugins de DreamObjects Backups.
    Después de hacer clic en el enlace Activate, DreamObjects aparece en el panel izquierdo:
  3. Haz clic en el enlace DreamObjects en el panel izquierdo y continúa con el siguiente paso. Se abre la página DreamObjects Backups, que te solicita que ingreses los Access Keys para tu usuario de DreamObjects.

Paso 2 — encontrar los Access Keys de DreamObjects

Para utilizar el plugin DreamObjects Backups, primero debes crear un usuario y un bucket en el panel. Una vez que crees estos, puedes obtener tus keys de DreamObjects utilizando las siguientes instrucciones:

  1. Navega a DreamObjects en el panel de DreamHost.
    Se abre la página DreamObjects Manage Users:
  2. Haz clic en el nombre de usuario que deseas usar y luego copia AMBAS de las siguientes Keys:
    • Access Key (número alfanumérico que se muestra a la izquierda)
    • Secret Key (número alfanumérico que se muestra a la derecha—haz clic en 'Show Secret Key' para revelarlo)
  3. Regresa a la página principal de DreamObjects Backups en WP-Admin y continúa con el siguiente paso.

Paso 3 — Configurar DreamObjects Backups en WordPress

Después de obtener los keys de DreamObjects del panel DreamHost, usa las siguientes instrucciones para finalizar la configuración del plugin DreamObjects Backups:

  1. Abre la página del panel WP-Admin para tu sitio de WordPress y haz clic en 'DreamObjects' en el panel izquierdo.
    Se abre la página de configuración de DreamObjects Backups, que te solicita que ingreses tus Access Keys de DreamObjects:
  2. Bajo Access Keys, ingresa las siguientes keys que obtuviste de la sección anterior:
    • Key
    • Secret Key
  3. Cuando termines de ingresar tus access Keys, haz clic en el botón Save Changes.
    Aparece la siguiente página confirmando que el plugin se ha conectado correctamente a tu usuario de DreamObjects:
  4. Haz clic en 'DreamObjects > Backups' en el panel izquierdo.
    La página DreamObjects Settings se abre:
  5. Selecciona un bucket del menú desplegable 'Bucket Name' y luego haz clic en el botón Update Options.
    La página Settings confirma el bucket y te muestra configuraciones adicionales que puedes seleccionar:
  6. Selecciona las opciones que deseas habilitar para tus copias de seguridad:
    • What to Backup
    • Schedule
    • Backup Retention
    • Status Notification
  7. Haz clic en Update Options cuando termines.
    La parte inferior de la página DreamObjects Backup Settings te muestra el estado de las copias de seguridad recientes:
  8. Haz clic en el botón Backup ASAP si deseas ejecutar una copia de seguridad inmediata de tu sitio de WordPress.

El plugin DreamObjects Backups está configurado y listo para usar para hacer una copia de seguridad de tu sitio de WordPress.

¿Que sigue?

Cuando finalizas la prueba gratuita, DreamHost te cobra por el almacenamiento de DreamObjects. Actualmente, DreamObjects cuesta 2.5¢/GB de almacenamiento por mes. Si necesitas descargar y restaurar una copia de seguridad de DreamObjects, cuesta 5¢/GB por mes. Este es un almacenamiento muy económico para mantener tus datos de respaldo en un servidor diferente al de tu cuenta DreamPress.
