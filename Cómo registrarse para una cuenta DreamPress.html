Generalidades

Este artículo explica cómo inscribirse en un nuevo plan de hosting DreamPress. Este artículo es para nuevos clientes que desean abrir una cuenta con sus propios datos personales e información de pago.

Este artículo muestra cómo abrir una cuenta con un plan de alojamiento. Esto significa que pagarás por un servidor para alojar tu sitio web. Si solo deseas comprar el nombre del sitio web (nombre de dominio) y no estás listo para alojar el sitio en Internet, consulta el siguiente artículo:

  • ¿Cómo comprar un registro de dominio?

Paso 1 — Escoge el plan de alojamiento

DreamHost ofrece varios planes de alojamiento de DreamPress para que alojes tu sitio web. Visita el siguiente enlace para ver la lista de planes:

  • Alojamiento WordPress

Una vez que hayas decidido el plan, haz clic en el botón debajo de Get DreamPress para continuar con la creación de la cuenta.

Paso 2 — Crea una nueva cuenta

Después de haber seleccionado un plan, puedes continuar con la creación de tu cuenta.

Añadiendo una cuenta de DreamPress

  1. Cuando hayas decidido el plan que deseas comprar, haz clic en el botón debajo para continuar. Después de hacer clic en el botón, te lleva a  panel.dreamhost.com/signup.
  2. Aquí puedes ingresar un dominio para comprar, transferir un dominio existente o hacer clic en 'Elegiré un dominio más adelante' (I'll choose a domain later) para omitir la parte del dominio.
  3. Elije pagar con tarjeta de crédito o PayPal. A la derecha, selecciona el plan de pago y los complementos que desees.

    Si seleccionas PayPal, te llevarán a su sitio para completar la compra.

    Si seleccionas tarjeta de crédito, aparece el siguiente cuadro donde puedes ingresar tu información de facturación:
  4. Ingresa un número de tarjeta (incluye la fecha de vencimiento y el código de seguridad) y la dirección de facturación. Haz clic en el botón Add Credit Card.
  5. Ingrese la información de su dirección. 
  6. Ingresa un correo electrónico. Este será el correo primario que usarás para iniciar sesión dentro del panel de DreamHost. Puedes también registrarte usando el método 'Single Sign On' de Google. Esto te permitirá usar tu dirección de correo de Google para iniciar sesión sin usar una contraseña.
  7. Hacia el final hay varias opciones que puedes habilitar:
    • WP Website Builder — Usa una ayuda de creación de sitio y un editor de agarrar y soltar, puedes habilitar un bellísimo sitio WordPress en unos simples pasos.
    • DreamShield Protection — Provee seguridad extra y protección para tu sitio DreamPress.
    • WordPress Migration Service — Selecciona esta opción para hacer que DreamHost migre tu sitio de WordPress para tí.
    • Install WooCommerce — Esto agrega el pugin gratuito WooCommerce a tu sitio para que puedas hacer una tienda online.
  8. Cuando hayas terminado, haz clic en el botón Submit Order en la parte inferior de la página.
