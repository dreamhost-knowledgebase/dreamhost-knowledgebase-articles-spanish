Generalidades

Cualquier certificado de Comodo emitido después de enero de 2019 hace referencia a Sectigo como la autoridad de certificación, mientras que cualquier certificado emitido antes de esta fecha hace referencia a Comodo. Ambos nombres se refieren a la misma compañía.

El proceso de Validación de Control de Dominio (DCV) le permite a Sectigo verificar que tu tienes control sobre un dominio para el que solicitas un certificado. Esto evita que nadie más que tu obtenga acceso a la configuración de seguridad de tu dominio y tome el control de tu sitio.

¿Cómo valida DreamHost el Control de dominio para los certificados Sectigo?

DCV se puede verificar mediante uno de los siguientes tres métodos (DNS, HTTP/HTTPS o correo electrónico):

DNS

Colocando un registro en los nameservers: DreamHost ingresa un registro CNAME en tu zona de archivos con información que Sectigo puede encontrar, y luego valida que tu tienes control sobre el dominio. Una vez que se completa, se elimina el registro CNAME

HTTP/HTTPS

Ingresando un archivo de acceso público en tu dominio: DreamHost coloca un archivo con una versión hash de tu Solicitud de firma de certificado (CSR) en tu dominio donde Sectigo puede acceder a ella. Una vez que se verifica que el dominio está bajo tu control, se elimina el archivo.

Email

Enviando un correo electrónico: Sectigo envía un correo a una de una breve lista de direcciones administrativas comunes de correo asociadas con un dominio (admin @, webmaster @, etc.).

Asegúrate de que la dirección de correo que seleccionas cuando compres el certificado Sectigo exista y puedas recibir correos antes de seleccionarla. Si es necesario, puedes crear una a través del panel. Seleccionar una dirección de confirmación de la lista desplegable no significa que exista, ni DreamHost o Sectigo la crean por ti.

Esta es una última opción en el proceso DCV, por lo que es posible que nunca recibas este correo. Si recibes este correo de Sectigo, sigue los pasos para completar la verificación de tu dominio. 

¿Qué método se usa?

Cuando solicitas un certificado de Sectigo Domain Validated (DV), DreamHost intenta validar en el orden que se muestra arriba (DNS, HTTP/HTTPS y luego por correo). Los dos primeros métodos son preferibles (DNS y HTTP/HTTPS) ya que DreamHost puede automatizarlos en tu nombre con el menor inconveniente posible para ti.

Si los dos primeros métodos no se completan, DreamHost utilizará el método de correo si falla el primer intento de validación o si no alojas el dominio en DreamHost.

¿Cuánto tiempo lleva obtener mi certificado?

La validación automática a través del método DNS o HTTP/HTTPS generalmente se realiza dentro de las 24 horas. Si el certificado no se obtiene poco después de 24 horas completas, DreamHost utilizará el método de correo. DreamHost intentará validar enviándote un correo, usando la dirección que seleccionaste como la dirección de correo de DCV descrita anteriormente.
