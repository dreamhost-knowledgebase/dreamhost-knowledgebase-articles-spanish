Conectandote desde una cuenta shell

  1. Para iniciar sesión en tu base de datos MySQL, necesitarás las credenciales de tu base de datos. Consulta el artículo 'Encontrar las credenciales de tu base de datos' para obtener instrucciones. También puedes encontrarlas en el panel bajo MySQL Databases.
  2. A continuación, necesitarás una terminal para ejecutar el siguiente comando. Puedes hacer esto en una computadora Mac o Linux utilizando el terminal nativo, o simplemente podrías iniciar sesión en tu servidor web DreamHost a través de SSH. Si estás en Windows, usa PuTTY para conectarte a tu servidor DreamHost, luego ejecuta el siguiente comando.
  3. Asegúrete de usar tus credenciales correctos para las siguientes variables:
    • DHhostname - Tu URL de nombre de host mysql. Aparecerá como 'mysql.example.com'
    • DBuser - Tu nombre de usuario de la base de datos
    • DBpassword - La contraseña de tu nombre de usuario de la base de datos
    • DBname - El nombre de la base de datos
    [server]$ mysql -h <DBhostname> -u <DBuser> -p <DBname>
  4. A continuación, se te solicitará tu contraseña para iniciar sesión.
  5. Una vez que hayas iniciado sesión, tu símbolo del sistema cambiará al siguiente:
    mysql>

No puedes usar "localhost" para conectarte a tu base de datos, ya que los servidores MySQL de DreamHost están separados de los servidores de sitios web normales.
