Generalidades

Este artículo explica como ubicar y activar los DNS de correo de DreamHost.

Los dominios usan diferentes tipos de registros DNS. Este artículo es específicamente sobre registros DNS de correo.

¿Dónde son controlados los registros de correo?

Los registros DNS de correo se agregan automáticamente a tu cuenta de DreamHost cuando creas una dirección de correo en el panel. Sin embargo, esto no significa que se estén utilizando.

Los registros DNS (registros A o nameservers) de tu dominio controlan cuales registros DNS de correo están siendo usandos en cualquier momento.

Revisa el siguiente artículo para obtener una explicación completa de cómo se configura el DNS en un dominio.

  • Generalidades de los DNS

Configurar los DNS

Los nameservers de DreamHost

Si los nameservers de tu dominio están apuntando a DreamHost, el DNS de correo de DreamHost están siendo usados.

Nameservers que no son de DreamHost

Si los nameservers de tu dominio NO apuntan a DreamHost, tus registros DNS de correo de DreamHost apuntan a una empresa diferente. Para usar el correo de DreamHost, debes apuntar tu DNS a DreamHost usando una de las siguientes opciones:

  • Apuntas tus nameservers (recomendado)
  • Apuntar los Registros A individuales

La mejor opción es apuntar tus nameservers a DreamHost ya que esto apunta TODOS tus registros DNS a DreamHost, incluido el correo.

La única razón para no apuntar tus nameservers es si tienes servicios específicos que necesita continuar alojando en otra empresa. En ese caso, deberás configurar manualmente todos los registros A de DNS de correo individuales (que se muestran a continuación) en el host donde se encuentra el DNS de tu dominio está actualmente administrado.

Verificar donde están apuntados actualmente los registros DNS

Puedes verificar donde están apuntados tus nameservers en whoisweb.dreamhost.com.

  • Cuando un dominio tiene direcciones de correo configuradas en el panel de DreamHost, todos los registros DNS relacionados con el correo se crean automáticamente en la página Administrar Correo .
  • Puedes encontrar tus registros DNS de Correo usando la pestaña DNS en la página Administrar Sitios Web Mira el artículo Agregar DNS para más información.

Encontrar tus registros DNS de correo

  1. Da clic en la pestaña de DNS de Sitios Web en la pestaña Administrar Sitios Web para ver tus registros.

    Por favor recuerda que los registros de abajo son solo ejemplos. Tus registros DNS específicos serán diferentes. Puedes leer más sobre DNS en el siguiente artículo:

    • Generalidades de los DNS de DreamHost

    Hay dos tipos de registros DNS relacionados con el correo.

    Registros MX
    Le dice a los remitentes a dónde deben enviarse los correos.
    Si el filtro de spam está habilitado tus registros MX serán los siguientes:
    • mx1.mailchannels.net
    • mx2.mailchannels.net

    Registros A Los utiliza tu programa de correo cuando se conecta al servidor de correo para ver tu bandeja de entrada o enviar correos. Hay varios registros de correo A para tu dominio.

  2. Encuentra los siguientes registros:
    Registros DNS Ejemplo de los valores
    _domainkey Los Registros DKIM firman los correo para ayudar a la validación del remitente.
    _autodiscover._tcp (SRV) 5 0 443 autoconfig.dreamhost.com — Este es un registro SRV que ayuda a las aplicaciones de correo a aplicar la configuración correcta automáticamente.
    autoconfig (CNAME) autoconfig.dreamhost.com — Este es un registro CNAME que ayuda a las aplicaciones de correo a aplicar la configuración correcta automáticamente.
    *mail. (A) 64.90.62.162 — Este es un subdominio estándar que se puede utilizar para conectar tu cliente o programa al servidor de correo.
    *mail. (MX)

    Este dominio ha habilitado el filtro de spam, entonces los registros MX aparecen así:

    mx1.mailchannels.net

    mx2.mailchannels.net

    Si el filtro de spam no se ha habilitado, verás: mx1.dreamhost.com y mx2.dreamhost.com.
    *mailboxes. (A) 66.33.205.233 — Esto se usa para iniciar sesión en la Interfaz Mailboxes de DreamHost.
    *www.mailboxes. (A) 66.33.205.233 — Esto se usa para iniciar sesión en la Interfaz Mailboxes de DreamHost.
    *webmail. (A) 69.163.136.138 — Esto se usa para iniciar sesión en el panel de webmail.
    *www.webmail. (A) 69.163.136.138 — Esto se usa para iniciar sesión en el panel de webmail.
  3. Ingresa todos esos registros en el sistema donde se alojan y administran los DNS del dominio.

Después de realizar el ajuste de DNS, los cambios tardarán varias horas en actualizarse en línea debido a propagación de los DNS.
