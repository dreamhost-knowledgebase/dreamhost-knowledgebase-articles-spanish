Generalidades

Lo siguiente explica qué hacen las direcciones IP y se proporciona más información sobre cómo administrarlas en DreamHost.

Antecendentes

Una dirección de Protocolo de Internet (IP) es un número único asignado a cada dispositivo en Internet. Al igual que un número de teléfono o una dirección postal, una dirección IP identifica dispositivos conectados en red y define su ubicación en una red como Internet.

Actualmente existen dos versiones de direcciones IP:

  • IPv4 — El protocolo (históricamente) más utilizado. Este tipo aparece como 4 grupos de dígitos decimales (p. ej., 192.0.2.235). DreamHost asigna automáticamente una dirección IPv4 a todos los servidores, servicios, y dominios activos.
  • IPv6 — Un nuevo protocolo que aparece como 8 grupos de dígitos hexadecimales (p. ej., 2001:0db8:85a3:08d3:1319:8a2e:0370:7334). IPv6 se creó para gestionar el crecimiento de Internet y permite más direcciones que IPv4. DreamHost ofrece actualmente direcciones IPv6 únicas gratuitas.

Agregar una dirección IP única a un dominio totalmente alojado

Si es necesario, puedes agregar una única dirección IP única a cualquier dominio completamente alojado en el panel. Esto se aplica al registro A de la URL de tu sitio web.

Muchas otras direcciones IP se crean automáticamente cuando alojas un sitio web. Para actualizarlas, deberás crear registros DNS personalizados.
