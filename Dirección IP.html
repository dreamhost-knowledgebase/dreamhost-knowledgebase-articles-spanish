Generalidades

Una dirección de Protocolo de Internet (dirección IP) es un número único asignado a cada dispositivo en Internet. Esta guía cubre brevemente lo que hacen las direcciones IP y proporciona más información sobre la administración de direcciones IP en DreamHost.

¿Qué son las direcciones IP? ¿Qué hacen?

Al igual que un número de teléfono o una dirección, una dirección IP identifica los dispositivos en red y define su ubicación en una red como Internet. Actualmente hay dos versiones de dirección IP:

  • IPv4 – El protocolo anterior que se limita a la cantidad de dispositivos que puede admitir.
  • IPv6 – El reemplazo más nuevo que admite significativamente más dispositivos y se ha adoptado en los últimos 10 años.

DreamHost proporciona ambas versiones de dirección IP y asigna automáticamente una dirección IPv4 a todos los servidores, servicios y dominios activos.

Una dirección IP puede ser compartida por múltiples dominios sin problema. Si es necesario, se puede asignar una única dirección IP Única a cualquier dominio totalmente alojado utilizando el panel DreamHost. Un dominio también puede tener múltiples direcciones IP asignadas a sus registros del sistema de nombres de dominio (DNS).

De la misma manera que una dirección IP es similar a un número de teléfono o una dirección, DNS es como un mapa o una guía telefónica. Resuelve nombres de dominio fáciles de usar a la dirección IP a la que están asignados. Las direcciones IP asignadas a un dominio pueden ser cambiadas manualmente por usted o automáticamente por DreamHost como resultado de un cambio en el servidor.

Versiones de protocolo de internet

Hay dos protocolos de capa entre redes en uso en Internet y compatibles con DreamHost:

IPv4
El protocolo más ampliamente implementado, limitado a direcciones de 32 bits (4 bytes) normalmente expresadas como 4 grupos de dígitos decimales (por ejemplo, 192.0.2.235). IPv4 permite aproximadamente 4,3 mil millones de direcciones y, a medida que aumenta la cantidad de dispositivos conectados a Internet, se alcanza este límite. La asignación de una dirección IPv4 única en DreamHost conlleva un costo mensual adicional debido a la disponibilidad limitada de direcciones IPv4.
IPv6
Un nuevo protocolo con un esquema de direccionamiento de 128 bits normalmente expresado como 8 grupos de dígitos hexadecimales (por ejemplo, 2001:0db8:85a3:08d3:1319:8a2e:0370:7334). IPv6 se creó para gestionar el crecimiento de Internet y permite aproximadamente 3,4 × 1038 direcciones. DreamHost actualmente ofrece direcciones IPv6 únicas gratuitas.
