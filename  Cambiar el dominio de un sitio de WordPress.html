Generalidades

Este artículo explica cómo cambiar el nombre de tu sitio de WordPress a un nombre de dominio diferente. Estas instrucciones se pueden utilizar para un nombre de dominio normal como example.com o un dominio temporal como example.dreamhosters.com.

Puedes usar el cambiador de nombre de dominio en la página de Sitios Web para cambiar cualquier dominio temporal, no solo los subdominios de dreamhosters.com.

Prerrequisitos

Asegúrate de lo siguiente antes de continuar:

  • Asegúrate de que tienes una Instalación de Un-Clic — La función de cambio de nombre de sitio web solo funciona con las Instalaciones de Un-Clic de WordPress. Confirma que el nombre del dominio desde el que estás cambiando sea una Instalación de Un-Clic. Si no es así, puedes mover el sitio manualmente o comunícarte con el soporte para convertirlo.
  • Elimina DNS personalizado — Asegúrate de que no hayan entradas de DNS personalizado A o CNAME en el dominio al que estás cambiando (si existe alguno que apunte a un servidor web diferente). Si los hay, elimina los registros personalizados. Otros registros DNS personalizados están bien de dejarlos en su lugar.

Paso 1 — Cambiar tu dominio

Para cambiar el dominio de WordPress:

  1. Navega a la página Adminstrar Sitios Web.
  2. Haz clic en el botón Manage de tu dominio.
  3. Desplázate hacia abajo en la sección Domain de la página.
  4. Haz clic en el enlace Change bajo Domain Name.
  5. Selecciona un dominio. Tienes dos opciones para cambiar el nombre:

    Escoger un dominio Existente

    Esta opción te permite elegir entre un dominio que ya has agregado a tu panel.

    Ingresa un nuevo dominio

    Esta opción te permite ingresar un nombre de dominio que ya posees o un dominio temporal. El panel luego lo agregará por tí.
  6. Haz clic en el botón Change Domain.
    Si seleccionaste un dominio existente, aparece la siguiente casilla de verificación pidiéndote que confirmes que se eliminarán las copias de seguridad anteriores del nombre de dominio anterior.
    Si ingresaste un nuevo dominio, puede aparecer la siguiente casilla de verificación, pidiéndote que confirmes que el dominio no está registrado con DreamHost y deberás actualizar el DNS.
    Toma 4-6 para que el DNS se actualice para tu nuevo dominio en DreamHost.

Paso 2 — Esperar a que tu DNS DreamHost se actualice

Como se indicó anteriormente, el DNS puede tardar entre 4 y 6 horas en actualizarse en DreamHost. Si apuntas tu DNS a tu host actual a DreamHost antes de que el DNS se haya actualizado en los servidores de DreamHost, el sitio no se conectará.

Se recomienda que esperes de 4 a 6 horas antes de actualizar tu DNS en tu alojamiento actual.

Paso 3 — Actualizar tu DNS en tu hosting actual

Finalmente, inicia sesión en tu compañía de registro y apunta tu DNS a DreamHost. Luego de que el propague, tu sitio estará alojado con DreamHost.
