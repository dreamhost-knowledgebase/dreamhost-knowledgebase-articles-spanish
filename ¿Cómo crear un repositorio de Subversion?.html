Subversion se ha eliminado del panel DreamHost. Si previamente habías configurado Subversion en tu dominio, seguirá funcionando normalmente.

Crear el repositorio

  1. Agrega el sitio web a tu panel como totalmente alojado. Ve el artículo Agregar un dominio para conocer los pasos detallados.
  2. Espera hasta que este dominio se resuelva en línea, lo que puede demorar algunas horas. Consulta el artículo sobre propagación de DNS para más detalles.
    Una vez que el dominio se resuelve en línea, puedes continuar con la creación de tu repositorio.
  3. Navega hasta la página Subversion.
  4. Actualiza los siguientes campos:
    • Project Name: Nombre de ejemplo 'exampleProjectName'.
    • Project ID: ID de ejemplo 'exampleProjectID'.
    • Install to: Elige tu dominio, luego haz clic en el campo de texto en blanco a la derecha que completa automáticamente tu ID de proyecto.
    • Users: Usar el usuario de ejemplo 'mysvnuser' y 'password12345'.
    • Visibility: Elige si deseas que este repositorio sea público o privado.
    • HTTP/HTTPS: Si deseas usar HTTPS, debes agregar un certificado SSL.
  5. Da clic en el botón Create my new project repository now! para crear un repositorio.
