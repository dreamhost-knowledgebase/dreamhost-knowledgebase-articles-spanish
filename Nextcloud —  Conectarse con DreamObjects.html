Generalidades

La aplicación External Storage Support te permite utilizar un servicio de almacenamiento externo (como DreamObjects) para almacenar los archivos de tu instalación de Nextcloud en lugar de almacenar los archivos en tu servidor web. Esto también se recomienda para mantenerte de acuerdo con las políticas de almacenamiento de DreamHost.

Esta guía describe como conectar Nextcloud con el servicio de almacenamiento en la nube DreamObjects.

Paso 1 — Instalar Nextcloud

Antes de que puedas comenzar a usar Nextcloud con DreamObjects, debes instalar Nextcloud en tu servidor DreamHost siguiendo esta guía paso a paso.

  • Nextcloud — Instalación

Paso 2 — Crear un bucket y usuario de DreamObjects

También debes habilitar DreamObjects en tu cuenta DreamHost. Ve los siguientes artículos para configurar DreamObjects y crear un bucket y una Llave Par.

  • Crear un usuario de DreamObjects
  • Crear llaves de DreamObjects

Paso 3 — Conectarse con DreamObjects

  1. Inicia sesión en tu sitio Nextcloud. Si seguiste la guía de instalación anterior, esta se ubicará en example.com/nextcloud.
  2. Haz clic en tu icono de usuario en la parte superior derecha. Selecciona + Apps en el menú desplegable.
  3. Desplázate hacia abajo en la lista para localizar External storage support.
  4. Haz clic en el botón Enable a la derecha.
  5. Haz clic en tu icono de usuario en la parte superior derecha. Selecciona Settings del menú desplegable.
  6. Selecciona External storages en el menú a la izquierda.
  7. Selecciona Amazon S3 del menú desplegable External storage.

    Ignora la advertencia "smbclient". Se utiliza para conectarse a Windows y no es necesario para esta configuración.

  8. Folder name: Ingresa el nombre de tu carpeta Nextcloud. Este será el nombre de la carpeta en la que almacenas tus datos y puede tener el nombre que desees.
  9. External storage: Configurado a Amazon S3.
  10. Authentication: Configurado para Access key.
  11. Configuration:
    • Bucket name — El nombre de tu bucket de DreamObjects.

      El nombre de tu bucket no puede contener un guion bajo, ya que Nextcloud no se conectará.

    • Hostname — objects-us-east-1.dream.io
    • Port — Déjalo en blanco.
    • Region — us-east-1
    • Enable SSL — Selecciona esta casilla.
    • Enable Path Style — Selecciona esta casilla.
    • Access key — Tu Llave de acceso de DreamObjects.
    • Secret key — Tu Llave Secreta DreamObjects.
  12. Available for: Si tienes varios usuarios de tu instalación de Nextcloud, puedes limitar quién tiene acceso a la carpeta escribiendo users o groups en la casilla Available for a la derecha de la información de configuración.
  13. Haz clic en la marca de verificación a la derecha. Si ingresaste todo correctamente, aparecerá un círculo verde a la izquierda del nombre de tu carpeta.

Ver tus archivos de almacenamiento externo

Ahora que has vinculado Nextcloud a DreamObjects, puedes ver los archivos que has subido a tu bucket de DreamObjects.

  1. Inicia sesión en tu sitio de Nextcloud.
  2. Haz clic en Files en el menú superior.
    Se muestra una lista de carpetas y archivos. En esta lista, verás el nombre de la carpeta que creaste anteriormente para almacenar el almacenamiento externo. En este ejemplo se llama MyNextcloudFolder.
  3. Haz clic en MyNextcloudFolder. En esta carpeta, verás todo lo que has subido a tu bucket de DreamObjects.

Cualquier objeto que subas a tu bucket de DreamObjects automáticamente cargará en tu carpeta Nextcloud.
