Generalidades

Si tienes problemas para migrar tu sitio web desde otro host, o simplemente no tienes tiempo, DreamHost actualmente ofrece un servicio que se encarga de esto por ti. Puedes encontrar más información en el artículo de DreamHost Servicios de migración para sitios web.

La migración de un sitio de WordPress es diferente de la migración de un sitio web HTML sencillo ya que los sitios de WordPress dependen tanto de los archivos web como de una base de datos MySQL para ejecutarse. Esta descripción general explica tus opciones según el tipo de migración que necesitas.

Migración de un sitio de WordPress desde un host diferente

Si estás migrando un sitio de WordPress desde una empresa de alojamiento diferente (host), hay muchas opciones disponibles para ti. Estos son algunos de los métodos más populares entre los que puedes elegir:

  • Migrar tu sitio manualmente usando FTP y phpMyAdmin
  • Migrar tu sitio usando el DreamHost Automated Migration WordPress plugin
  • Pedir el Servicio Profesional de DreamHost para migrar tu sitio por una tarifa (como se explica en el cuadro de notas en la parte superior de esta página)

Los pasos de cada método pueden cambiar dependiendo de si estás manteniendo el mismo nombre de dominio (también conocido como su URL) o si estás cambiando a uno nuevo.

Mantener el mismo nombre de dominio

Si mantienes el mismo nombre de dominio y deseas minimizar el tiempo de inactividad en el sitio actual mientras estás migrando, puedes beneficiarte de configurar un mirror de tu sitio DreamHost utilizando un subdominio temporal .dreamhosters.com:

  • Vista previa de tu sitio usando un subdominio gratuito dreamhosters.com

Cambiar el nombre de dominio

Si estás cambiando tu sitio de WordPress a un nuevo nombre de dominio cuando se mueva a DreamHost, podrías usar un plugin. Muchos plugins te ayudan a realizar este cambio automáticamente, aunque cada plugin lo maneja a su manera.

Si eliges no usar un complemento y migrar manualmente usando FTP y phpMyAdmin, debes actualizar manualmente tu sitio migrado al nuevo nombre de dominio. Después de completar los cinco pasos en las instrucciones de Migración Manual, los pasos 5 a 7 de la siguiente guía te ayudarán a cambiar el nombre de tu sitio de WordPress manualmente:

  • ¿Cómo mover WordPress a un dominio diferente en DreamHost?

Migrar un sitio de WordPress entre dominios en tu cuenta DreamHost

Si deseas mantener el contenido de un sitio de WordPress, pero quieres cambiar su nombre de dominio a uno nuevo, no solo necesitarás hacer cambios en tu cuenta DreamHost, sino que también debes hacer cambios en tu Panel de administración de WP. La siguiente guía explica un método para asegurarse de que tu contenido cambie completamente al nuevo nombre de dominio:

  • ¿Cómo mover WordPress a un dominio diferente en DreamHost?

Opción de DreamPress para migrar tu sitio a un nuevo dominio

Si tienes problemas para migrar un sitio de WordPress usando las instrucciones anteriores o si deseas una forma más rápida, puedes actualizar tu nuevo dominio a DreamPress, que ofrece una opción de importación. DreamPress es el alojamiento de WordPress administrado de DreamHost. Es un servicio escalable que permite a los usuarios administrar sus sitios de WordPress con precios simples y beneficios adicionales de robustez en el servidor, almacenamiento en caché y memoria.

Si tu sitio actual de WordPress fue instalado y administrado por el instalador de un clic de DreamHost, puedes configurar DreamPress en el nuevo nombre de dominio e importar el contenido del sitio actual de WordPress al nuevo dominio de DreamPress. Puedes agregar DreamPress en un nuevo dominio en tu panel DreamHost en Managed WordPress. Por favor, visita la siguiente guía sobre cómo importar contenido en DreamPress:

  • Importar una instalación de WordPress existente a DreamPress

Si tu sitio actual de WordPress no es administrado por el Instalador de un clic, puedes enviar un ticket para solicitar que tu sitio se convierta en una Instalación de un clic.
