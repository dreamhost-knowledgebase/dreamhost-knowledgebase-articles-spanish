Generalidades

A continuación se describe cómo configurar la versión de prueba gratuita de WebYog de SQLyog para conectarte a tu base de datos DreamHost MySQL.

Conectarte a tu base de datos usando SQLyog

  1. Descarga e instala SQLyog.
    La caja de dialogo 'Connect to MySQL Host' aparece cuando abres SQLyog:
  2. Haz clic en el botón New. 
    La caja de dialogo 'New Connection' se abre:
  3. Ingresa un nombre para la nueva conexión y haz clic en el botón OK. Puedes darle el nombre que desees.
  4. Ingresa la siguiente información en la pestaña MySQL:
    • MySQL Host Address — La dirección de host MySQL se puede encontrar en la página de Bases de Datos MySQLen tu panel. Para obtener más información, consulta el artículo Encontrar las credenciales de inicio de sesión de tu base de datos.
    • Username — El nombre de usuario de la base de datos.
    • Password — La contraseña de la base de datos.
  5. Ingresa la siguiente información en la pestaña SSH:
    • SSH Host Address — La dirección Host SSH puede ser encontrada en la página Administrar Cuenta en tu panel, en la sección Dominios Alojados (bajo la columna Tipo de Alojamiento). Para más información, visita el artículo Generalidades sobre SSH.
    • Username — El nombre de usuario puede ser encontrado en la página Administrar Dominios de tu panel, a la derecha de tu dominio bajo la columna Web Hosting.
    • SSH Port — Ingresa '22' como el valor de puerto SSH.
    • Password — La contraseña del usuario. Puedes visitar la página Archivos en tu panel para cambiar tu contraseña si es necesario.
  6. Cuando hayas finalizado, haz clic en el botón Test Connection.

Una vez que puedas conectarte, haz clic en el botón Connect para conectarte a tu base de datos DreamHost MySQL.
