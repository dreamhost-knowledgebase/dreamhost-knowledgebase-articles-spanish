Si está teniendo problemas para migrar un sitio de WordPress de otro servidor o simplemente no tiene tiempo, DreamHost actualmente ofrece un servicio que se encarga de esto por usted: por solo $99 y en menos de 48 horas, DreamHost moverá su sitio de WordPress sin ningún tiempo de inactividad. Por favor, consulte los Términos de servicio (TOS) para obtener más información sobre el proceso. Clientes a usar para cargar su página

Cuando su página esté terminada y lista para publicarse, podrá cargar sus archivos en DreamHost para colocarlos en su cuenta.

Esto se puede hacer usando un cliente FTP para cargar sus archivos:

  • Cliente FTP — FileZilla, Cyberduck, CuteFTP, FireFTP, etc., dependiendo de su sistema operativo. Usar un cliente FTP totalmente funcional, tal como los anteriores, es la forma recomendada de conectarse a su servidor web.
  • Web FTP — Visite http://webftp.dreamhost.com para conectarse a su página mediante WebFTP. Esta es la opción disponible de cliente FTP en su panel de DreamHost.

Existen muchos otros paquetes de software para cliente FTP. Por favor, visite el artículo Lista de clientes FTP para más información.

Use los siguientes valores de inicio de sesión de cuenta FTP cuando se le soliciten:

Nombre del Host/Servidor

Puede utilizar cualquiera de los siguientes:

  • URL de la Página (sin el http://www): por ejemplo, si su página es http://www.example.com, usted usaría solamente example.com para iniciar sesión en su cliente FTP.
  • URL del Servidor: en su panel, vaya a Manage Account.

En esta página usted verá una sección llamada "Dominios alojados". En dicha sección, usted verá sus dominios. A la derecha de cada una está el servidor web en el cual está configurado. Así que, si usted está en el servidor "flower", su dirección de servidor sería "flower.dreamhost.com".

Nombre de usuario / Inicio de sesión

Este es el nombre de usuario para iniciar sesión solamente en su cuenta FTP. Por favor, tenga en cuenta que el WebID de su Panel NO ES el nombre de usuario FTP. Usted podrá encontrar esta información en el correo electrónico que DreamHost le envió cuando usted se registró o cuando creó la cuenta de usuario FTP. Usted también podrá ver todos sus usuarios en Manage Users. Para ver qué usuario está asignado a cuál dominio, vaya a Manage Domains. A la derecha de cada dominio (en la columna de alojamiento web), podrá encontrar al usuario asignado a ese dominio específicamente.

Contraseña

Esta es la contraseña exclusivamente para iniciar sesión en su cuenta FTP. Por favor, tenga en cuenta que esta NO ES la misma contraseña que usa para su Panel. Puede restablecerla en Manage Users. Harán falta unos 15 minutos para que se actualice en el sistema antes de activarse.

Puerto

Consulte esta página para ver si su usuario es un usuario de FTP/SFTP/SHELL Manage Users. Si su usuario es un usuario de FTP, debe usar el puerto 21. Si es un usuario de SFTP o SHELL, debe usar el puerto 22. NOTA: si usted usa WebFTP en el panel, este solamente acepta el puerto 21 por el momento.

Para garantizarle una conexión segura con su servidor, verifique que está usando el puerto 22. Esto también se conoce como "conectarse mediante FTP". Lea Seguridad en FTP para más detalles.

Resolución de problemas

Si usted ha cargado su página, y todo parece indicar que se ha transferido con normalidad, pero solamente puede ver un "directorio-padre", entonces lo más probable es que usted haya cargado sus archivos en la carpeta equivocada, o en la cuenta de usuario FTP incorrecta. Por favor, verifíquelo e inténtelo de nuevo.
