Otras opciones de migración

Servicio de migración de WordPress

Si tienes problemas migrando tu sitio web desde otra compañía de alojamiento, o simplemente no tienes el tiempo de hacerlo, DreamHost ofrece actualmente un servicio que se encarga de ello por ti. Puedes encontrar más información en el artículo Servicios de Migración de DreamHost.

Remixer

DreamHost también ofrece un editor de página web llamado Remixer. Este servicio le permite crear una página web sin necesitar ninguna clase de experiencia con programación, y sin necesidad de usar un cliente FTP para cargar los datos de tu página. Clientes a usar para cargar tu página

Cuando tu página esté terminada y lista para publicarse, podrás cargar tus archivos en DreamHost para colocarlos en tu cuenta.

Esto se puede hacer usando un cliente FTP para cargar tus archivos:

  • Cliente FTP — FileZilla, Cyberduck, CuteFTP, FireFTP, etc., dependiendo de tu sistema operativo. Usar un cliente FTP totalmente funcional, tal como los anteriores, es la forma recomendada de conectarte a tu servidor web.
  • WebFTP — Visita http://webftp.dreamhost.com para conectarte a tu página mediante WebFTP. Esta es la opción disponible de cliente FTP en tu panel de DreamHost.

Existen muchos otros paquetes de software para cliente FTP. Por favor, visita el artículo Lista de clientes FTP para más información.

Use los siguientes valores de inicio de sesión de cuenta FTP cuando se le soliciten:

Nombre del Host/Servidor

Puedes utilizar cualquiera de los siguientes:

  • URL de la Página (sin el http://www): por ejemplo, si tu página es http://www.example.com, tu usarías solamente example.com para iniciar sesión en tu cliente FTP.
  • URL del Servidor: en tu panel, dirigete a Manage Account.

En esta página verás una sección llamada "Dominios alojados". En dicha sección, verás tus dominios. A la derecha de cada uno está el servidor web en el cual está configurado. Así que, si tu  estás en el servidor "flower", tu dirección de servidor sería "flower.dreamhost.com".

Nombre de usuario / Inicio de sesión

Este es el nombre de usuario para iniciar sesión solamente en tu cuenta FTP. Por favor, ten en cuenta que el WebID de su Panel NO ES el nombre de usuario FTP. Podrás encontrar esta información en el correo electrónico que DreamHost te envió cuando tu ye registraste o cuando creaste la cuenta de usuario FTP. Tu también podrás ver todos tus usuarios en Manage Users. Para ver qué usuario está asignado a cuál dominio, ve a Manage Domains. A la derecha de cada dominio (en la columna de alojamiento web), podrás encontrar al usuario asignado a ese dominio específicamente.

Contraseña

Esta es la contraseña exclusivamente para iniciar sesión en tu cuenta FTP. Por favor, ten en cuenta que esta NO ES la misma contraseña que usas para tu Panel. Puedes restablecerla en Manage Users. Espera unos 15 minutos para que se actualice en el sistema antes de activarse.

Puerto

Consulta esta página para ver si tu usuario es un usuario de FTP/SFTP/SHELL Manage Users. Si tu usuario es un usuario de FTP, debes usar el puerto 21. Si es un usuario de SFTP o SHELL, debes usar el puerto 22. NOTA: si tu usas WebFTP en el panel, este solamente acepta el puerto 21 por el momento.

Para garantizarte una conexión segura con tu servidor, verifica que está usando el puerto 22. Esto también se conoce como "conectarse mediante FTP". Lea Seguridad en FTP para más detalles.

Resolución de problemas

Si tu has cargado tu página, y todo parece indicar que se ha transferido con normalidad, pero solamente puedes ver un "directorio-padre", entonces lo más probable es que hayas cargado tus archivos en la carpeta equivocada, o en la cuenta de usuario FTP incorrecta. Por favor, verifícalo e inténtalo de nuevo.
