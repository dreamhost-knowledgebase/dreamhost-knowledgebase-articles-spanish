Generalidades

Cuando el tema actual se rompe o tiene un problema, puedes hacer que todo el sitio sea inaccesible con errores. Esto hace que sea difícil cambiar a un tema de trabajo porque incluso el panel de administración no carga.

Hay dos formas de cambiar el tema cuando no tienes la capacidad de iniciar sesión en tu sitio:

  • Usando la línea de comando
  • Cambiando el tema en la base de datos (usando phpMyAdmin)

Usando la línea de comandos

  1. Conéctate a tu servidor a través de SSH y usa la herramienta de línea de comando de WordPress.
  2. Escribe lo siguiente para ver una lista de todos los temas disponibles: 
    [server]$ wp theme status
  3. Escribe lo siguiente para cambiar tu tema: 
    [server]$ wp theme activate twentytwenty
  4. Si no tienes twentytwenty, instálalo y actívalo en un solo comando: 
    [server]$ wp theme install twentytwenty --activate

Cambiando el tema en la base de datos

Para cambiar a un tema diferente usando la base de datos:

  1. Inicia sesión en phpMyAdmin.
  2. Haz clic en el nombre de la base de datos a la izquierda.
  3. Encuentra la tabla _options en la columna izquierda (el nombre de la tabla tiene un prefijo pero termina en "_options")
  4. Ve a la segunda página y busca las opciones con los nombres template y stylesheet.
  5. Haz clic en el icono de lápiz a la izquierda de cada una.
    En el campo de texto grande, puedes ver el nombre del tema actual:
  6. Cambia esto a cualquier tema que desees que use tu sitio.
  7. Haz clic en el botón Go cuando hayas terminado para guardar tus cambios.

Solución de problemas

Si ves un error al ejecutar cualquiera de los comandos anteriores, por favor contacta el equipo de soporte para más asistencia.
