Generalidades

Este artículo describe las diferentes formas en que puedes mejorar tu plan de alojamiento Shared Starter.

Mejorar a un Alojamiento Shared

Revisa el siguiente artículo para obtener instrucciones sobre cómo actualizar a un Alojamiento Shared.

  • Actualizar un plan Shared Starter a Shared Unlimited

Mejorar el plan a un VPS

Revisa el siguiente artículo para obtener instrucciones sobre cómo agregar un plan de alojamiento VPS.

  • Agregar un VPS

Al momento de la compra del VPS, tendrás la oportunidad de migrar TODOS tus usuarios y dominios a este nuevo servidor. También puedes optar por no migrar ningún dato.

Si solo deseas pagar por un VPS y eliminar el Alojamiento Shared, marca el botón de opción para migrar todos los usuarios y dominios durante este proceso.

Mejorar a un Servidor Dedicado

Revisa el siguiente artículo para obtener instrucciones sobre cómo agregar un nuevo Servidor Dedicado.

  • Agregar un servidor Dedicado

Mientras creas el Servidor Dedicado, puedes elegir migrar todos tus usuarios compartidos a la nueva máquina.

Mejorar a DreamPress

Si ya tienes un sitio de WordPress en Shared hosting, puedes usar el siguiente artículo para actualizarlo a DreamPress:

  • Actualizar un sitio de WordPress a DreamPress

También puedes agregar un nuevo dominio a DreamPress usando el siguiente artículo:

  • Agregar DreamPress a un dominio nuevo
