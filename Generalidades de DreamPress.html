Generalidades

DreamPress es el alojamiento administrado de WordPress en DreamHost. Es un servicio escalable que permite a los usuarios administrar sus sitios de WordPress con precios simple y los beneficios adicionales de robustez del servidor, almacenamiento en caché y memoria.

Detrás de escena, DreamPress combina de la tecnología existente de DreamHost en una sola unidad cohesiva. A continuación se detallan todos los detalles de este nuevo servicio.

Comenzar

Una vez que hayas añadido un plan DreamPress a tu cuenta, puedes actualizar tu sitio de WordPress o agregar un nuevo sitio web a DreamPress.

Detalles técnicos

Dado que DreamPress funciona con Nginx, no es posible realizar cambios de configuración. Por favor contacta a soporte si necesitas personalizar el archivo nginx.conf o hacer campos que normalmente irían al archivo .htaccess.

Cada instancia de DreamPress se configura en el servicio de computación en la nube de DreamHost llamado OpenStack. Un nuevos servicio de DreamPress:

  • Se aprovisiona automáticamente.
  • No permite acceso root, sudo y FTP (si puedes iniciar sesión a través de SFTP o SSH).
  • Viene con un certificado seguro gratuito 'Let's Encrypt' instalado.
  • Puede ajustarse su memoria y almacenamiento disponible basado en el plan seleccionado.
  • Incluye las versiones de WordPress Core más recientes (así como los dos anteriores) (mira numeralización de versiones para más información).
  • Incluye cualquier versión de PHP que WordPress recomiende para sus tres últimas versiones principales.

Además, DreamHost tiene almacenamiento en caché incorporado, que:

  • Se configura automáticamente para el cliente DreamPress.
  • Responde a las solicitudes web como proxy frente a la pila de WordPress de Nginx y PHP.
  • Es extremadamente eficiente en el almacenamiento de contenido en caché.
  • Permite esta configuración para escalar a niveles muy altos de tráfico.
  • Viene con una IP estática.

Mejoras de seguridad adicionales

Como se mencionó anteriormente, todos los planes de DreamPress incluyen un certificado Let's Encrypt SSL gratuito. Si es necesario, puedes agregar un certificado SSL firmado profesionalmente (como Sectigo) por una tarifa adicional.

También puedes agregar el servicio de Seguridad Web de DreamShield para cualquier plan de DreamPress o DreamPress Plus (todos los planes de DreamPress Pro incluyen un plan de DreamShield sin costo adicional.

Restricciones del servicio

DreamPress es un servicio administrado de WordPress, lo que significa que DreamHost ha restringido ciertas acciones que los usuarios pueden tomar que podrían romper irreparablemente Tu sitio de WordPress.

Cada plan DreamPress está configurado para un dominio y un sitio de vista previa. Puedes mezclarlo y combinarlo con Alojamiento compartido, VPS y Servidor Dedicado.

Restricciones actuales

  • No hay usuarios administradores ni claves SSH en root.
  • No se ofrece acceso FTP. Puedes tener un usuario de SFTP (SSH opcional) y el dominio que elijas se configurarán automáticamente.
  • No se pueden agregar o eliminar bases de datos o nombres de host. La base de datos y el hostname se crearán automáticamente para WordPress.
  • No se pueden agregar wildcard para DNS.
  • No se puede eliminar el certificado SSL instalado automáticamente.
  • No se puede ajustar la configuración HTTPS.
  • No es posible añadir directamente reglas de caché personalizadas o .htaccess. Puedes contactar a soporte para obtener ayuda si tu sitio requiere alguna de estas opciones.
  • Las instalaciones de red (multisitio) no son compatibles.
  • DreamHost instala y actualiza automáticamente una pequeña cantidad de plugins necesarios.
  • No pueden agregar direcciones IP únicas. 

Soporte al usuario

DreamPress incluye el mejor soporte técnico de DreamHost, que también tienen experiencia en WordPress. Después de una capacitación y evaluación intensiva por parte de los colaboradores principales de WordPress, fueron seleccionados como los mejores en su clase para representar DreamHost y WordPress, y ayudarte con cualquier problema con tu sitio DreamPress. Si no saben la respuesta, saben cómo encontrar a alguien que sí lo sabe.

Dicho esto, por favor ten en cuenta que el soporte de DreamHost no es un reemplazo para un desarrollador de sitio web real (si llegaras a necesitarlo); DreamHost puede ayudar a instalar/configurar plugins, pero no puede crear plugins o temas personalizados. Además, si un plugin o tema está causando problemas, el único soporte que DreamHost puede proporcionar es deshabilitar el plugin/tema y sugerir alternativas.
