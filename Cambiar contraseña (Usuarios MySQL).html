Actualizar el archivo de configuración de tu sitio web

Cuando cambies la contraseña del usuario de la base de datos también debes actualizar el archivo de configuración de la aplicación de tu sitio web. De no hacerlo, tu sitio no va a cargar. El archivo de configuración es distinto para cada sitio. A continuación encontrarás un listado con los ejemplos mas comunes:

Software Nombre del archivo de configuración Ubicación del archivo de configuración
Concrete CMS database.php application/config/
Joomla configuration.php En el directorio principal del sitio
MediaWiki Localsettings.php En el directorio principal del sitio
phpBB config.php En el directorio principal del sitio
WordPress wp-config.php En el directorio principal del sitio
Zen Cart configure.php includes/

Puedes actualizar el archivo config de tu sitio, al ingresar al servidor por FTP o SSH.

Busca documentación online sobre cómo ajustar el respectivo archivo de configuración (config), específicamente para la aplicación que estés usando.

Cambiar la contraseña de tu base de datos

  1. Navega a la página Bases de Datos MySQL.
  2. Haz clic en el usuario que se encuentra al lado derecho del nombre de tu base de datos.
  3. Al final de la página de edición del usuario puedes ingresar la contraseña nueva y guardar los cambios.
