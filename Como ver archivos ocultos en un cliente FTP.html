Generalidades

La configuración de tu cliente FTP para mostrar archivos ocultos es un proceso que difiere significativamente entre cada cliente FTP. Las instrucciones para hacerlo con muchos clientes FTP populares se encuentran a continuación:

Deberías usar SFTP para conectarte (en lugar de FTP) ya que SFTP garantiza una conexión segura a tu servidor.

  • Seguridad SFTP

Si ya se estás conectando usando SFTP, es posible que ya puedas ver archivos ocultos. Esto depende del cliente que estés usando.

Usando FileZilla

  1. Haz clic en la pestaña Server en la parte superior.
  2. Selecciona Force showing hidden files.

Usando CuteFTP 9.0

  1. Haz clic derecho dentro de la ventana del lado del servidor.
  2. Selecciona la opción Filter ...
  3. Bajo Server side filtering selecciona la casilla de verificación Enable server side filtering (server applies filters).
  4. Escribe -al en la caja Remote filter.
  5. Haz clic para Apply.

Usando Fetch

  1. Al abrir Fetch, se te presentará la siguiente pantalla de configuración de conexión. Aquí hay un ejemplo de conexión con un usuario llamado 'user' en el servidor charles-pickney:
  2. Después de iniciar sesión, verás el contenido de tu servidor web:
  3. Habilita archivos ocultos navegando a 'Fetch > 'Preferences':
  4. Selecciona la pestaña Miscellaneous.
    1. En esta página, asegúrate de que 'Omitir elementos ocultos cuyos nombres comiencen con un punto' no esté marcado.
    2. Luego marca la casilla 'Use 'LIST -al' command to reveal hidden items'.
  5. Después de realizar este cambio, asegúrate de actualizar tu vista yendo a 'View > Refresh':

Usando Transmit

  1. Cuando abra Transmitir, a la derecha verás la configuración de tu conexión. Esta es una captura de pantalla que usa un usuario FTP llamado 'user' y un servidor llamado 'charles-pickney':
  2. Una vez conectado, puedes habilitar archivos ocultos yendo al menú en la parte superior y seleccionando 'View > Show Invisible Files':

Usando FTP Voyager

  1. Selecciona la opción Administrador de perfiles de sitio en la parte superior izquierda.
  2. Selecciona Advanced Options en la ventana que aparece.
  3. Selecciona la pestaña de Connection en las opciones avanzadas de Site Profile en la ventana que se abre.
  4. Debajo de la columna General, escriba lo siguiente junto al campo Parámetro LIST:
    [server]$ -la
  5. Haz clic en OK.
