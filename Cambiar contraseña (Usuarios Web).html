Generalidades

Puedes crear dos tipos de usuarios web diferentes.

  • SFTP
  • Shell

Los tipos de usuarios mencionados arriba pueden iniciar sesión en un servidor para acceder a los archivos web, que es diferente al usuario que usa para iniciar sesión en un sitio CMS que ejecuta software como WordPress. Si necesitas restablecer la contraseña de ese tipo de usuario, revisa el siguiente artículo en vez.

  • Cambiar contraseña (Aplicaciones de terceros)

Lo siguiente describe cómo cambiar la contraseña para el archivo de acceso SFTP/Shell usando la página Archivos y Usuarios SFTP

Cambiar la contraseña de usuario de tu sitio web

Para restablecer la contraseña de tu usuario en el panel:

  1. Navega a la página Archivos y Usuarios SFTP.
  2. Da clic en el botón Datos de Ingreso a la derecha para abrir la caja de configuraciones del usuario.
  3. Haz clic en el botón Cambiar Contraseña.
    La caja de Restablecer Contraseña aparece, pidiéndote que ingreses una contraseña.
  4. Haz clic o ingresa uno de los siguientes:
    • Generar contraseña segura — Da clic en este enlace para crear una contraseña generada por el sistema, o
    • Caja de campo de contraseña — ingresa manualmente una nueva contraseña.
  5. Cuando termines de crear la nueva contraseña, da clic en el botón Restablecer contraseña para cambiarla.
    Cuando la contraseña se actualice correctamente, una pequeña marca de verificación aparece al lado de tu usuario.

Por favor permítele hasta 5 minutos antes de iniciar sesión.
