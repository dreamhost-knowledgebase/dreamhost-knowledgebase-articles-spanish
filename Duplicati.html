Generalidades

Duplicati es un cliente de respaldo gratuito que almacena de forma segura respaldos encriptados, incrementales y comprimidos en servicios de almacenamiento en la nube y servidores de archivos remotos. Funciona con Linux, Mac y Windows.

Duplicati tiene encriptación AES-256 incorporada, y las copias de seguridad se pueden firmar usando GNU Privacy Guard. Un programador integrado garantiza que las copias de seguridad estén siempre actualizadas. El proyecto Duplicati se inspiró en la duplicity, aunque similares, no son compatibles.

Este artículo te explica cómo configurar Duplicati con tu cuenta de DreamObjects.

Configurar Duplicati

  1. Descarga Duplicati y abre el programa.
  2. Haz clic en Add backup en la parte derecha.
  3. Selecciona la opción para Configure a new backup.
  4. Dale un nombre y selecciona AES-256 encryption, built-in.
  5. Ingresa un passphrase fuerte, que puedas recordar. 
  6. Ingresa lo siguiente en los campos en la siguiente página:
    • Storage Type: S3 Compatible.
    • Use SSL: Selecciona esta casilla.
    • Server: Selecciona Custom server url e ingresa 'objects-us-east-1.dream.io'.
    • Bucket name: El nombre de tu bucket.
    • Bucket create region: Elige US East (Northern Virginia) (us-east-1).
    • Storage class: Dejar como predeterminado.
    • Folder path: Ingresa una ruta de carpeta específica si es necesario.
    • AWS Access ID: La llave de acceso de tu usuario.
    • AWS Access Key: La clave secreta de tu usuario.
  7. Haz clic en el botón Test connection para confirmar que estás conectándote con DreamObjects.
    Recibirás un mensaje sobre el nombre de usuario siendo incorrecto. Haz clic en No. Luego verás una ventana emergente con el mensaje 'Connection worked!'.
  8. Haz clic en el botón Next.
  9. Elige qué carpetas deseas respaldar en la siguiente página.
  10. Establece un horario para tus copias de seguridad.
  11. Sube el tamaño del volumen predeterminado a 50MB.
  12. Selecciona 'AES-256 encryption, built-in' debajo de Advanced options.
  13. Haz clic en el botón Save.
  14. Haz clic en Run now para generar la copia de seguridad de tus archivos.
