Generalidades

A partir de diciembre de 2019, las versiones de PHP 5.6, 7.0 y 7.1 se han eliminado del panel DreamHost. Todos los sitios en los servidores compartidos que ejecutan PHP 5.6 o 7.0 están en proceso de actualización a PHP 7.2. Los sitios web en VPS y servidores dedicados continuarán ejecutando versiones anteriores de PHP por el momento.

Si deseas cambiar manualmente la versión PHP de tu sitio, las opciones actuales son 7.2, 7.3 o 7.4. Para obtener más información, consulta el siguiente artículo:

  • ¿Cómo cambio la versión PHP de mi sitio?

What if my site was previously using PHP 5.6, 7.0, or 7.1?

¿Qué sucede si mi sitio utilizaba PHP 5.6, 7.0 o 7.1 anteriormente?
Todos los sitios en un servidor compartido que anteriormente ejecutaban PHP 5.6, 7.0 o 7.1 están en proceso de actualizarse automáticamente a PHP 7.2.

Se recomienda encarecidamente que actualices tus sitios a PHP 7.2, 7.3 o 7.4 ya que las versiones anteriores pronto se eliminarán por completo de todos los servidores DreamHost.

Para obtener más información sobre cómo actualizar la versión de PHP de tu sitio, consulta el siguiente artículo:

  • Generalidades sobre la actualización de PHP
