Generalidades

Este artículo explica cómo inscribirse en un nuevo plan de hosting DreamPress. Este artículo es para nuevos clientes que desean abrir una cuenta con sus propios datos personales e información de pago.

Este artículo muestra cómo abrir una cuenta con un plan de alojamiento. Esto significa que pagarás por un servidor para alojar tu sitio web. Si solo deseas comprar el nombre del sitio web (nombre de dominio) y no estás listo para alojar el sitio en Internet, consulta el siguiente artículo:

  • ¿Cómo comprar un registro de dominio?
 

Paso 1 — Escoger el plan de alojamiento

DreamHost ofrece varios planes de alojamiento de DreamPress para que alojes tu sitio web. Visita el siguiente enlace para ver la lista de planes:

  • Alojamiento WordPress

Haz clic en el botón debajo de tu plan elegido para crear una cuenta DreamPress.

Paso 2 — Crear una nueva cuenta

Después de haber seleccionado un plan, puedes continuar con la creación de tu cuenta.

Agregando una cuenta de DreamPress

  1. Cuando hayas decidido el plan que deseas comprar, haz clic en el botón debajo para continuar. Después de hacer clic en el botón, te lleva a registrarte en DreamPress.
  2. Aquí puedes ingresar un dominio para comprar, transferir un dominio existente o hacer clic en 'Escoge un dominio después' para omitir la parte del dominio.
  3. Elige pagar con tarjeta de crédito/débito o PayPal. A la derecha, selecciona 'Anual' o 'Mensual' y los complementos que desees.

    Si eliges PayPal, te llevarán a su sitio para completar la compra.

    Si seleccionas tarjeta de crédito, aparece un nuevo campo donde puedes ingresar tu información de facturación:
  4. Ingresa un número de tarjeta (incluye la fecha de vencimiento y el código de seguridad) y la dirección de facturación. Haz clic en el botón Añadir Tarjeta de Crédito.
  5. Ingresa la información de tu dirección. 
  6. Ingresa un correo. Este será el correo primario que usarás para iniciar sesión dentro del panel de DreamHost. Puedes también registrarte usando el método 'inicio de sesión único de Google (SSO)' de Google. Esto te permitirá usar tu dirección de correo de Google para iniciar sesión sin usar una contraseña.
  7. Hacia el final, selecciona lo siguiente (opcional):
    • Creador de sitios web para WP — Usa una ayuda de creación de sitio y un editor de agarrar y soltar, puedes habilitar un bellísimo sitio WordPress en unos simples pasos.
    • Protección DreamShield — Provee seguridad extra y protección para tu sitio DreamPress.
    • Instalar WooCommerce — Esto agrega el plugin gratuito WooCommerce a tu sitio para que puedas hacer una tienda online.
  8. Cuando hayas terminado, haz clic en el botón Enviar Orden en la parte inferior de la página.
