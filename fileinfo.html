Generalidades

Las funciones en el módulo fileinfo tratan de adivinar el tipo de contenido y la codificación de un archivo buscando ciertas secuencias de bytes mágicos en posiciones específicas dentro del archivo. Si bien este no es un enfoque a prueba de balas, las heurísticas utilizadas hacen un muy buen trabajo.

Puedes encontrar más información sobre fileinfo en php.net – File Information.

Habilitar fileinfo

Este módulo está habilitado de forma predeterminada en todas las cuentas de DreamHost. Puedes confirmar agregando un archivo phpinfo.php a tu sitio web. Consulta el siguiente artículo para obtener instrucciones sobre cómo agregar este archivo:

  • phpinfo.php

Una vez que ha sido creado, visita la siguiente URL en tu buscador:

  • example.com/phpinfo.php
