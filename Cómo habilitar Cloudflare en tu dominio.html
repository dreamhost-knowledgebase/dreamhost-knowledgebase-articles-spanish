Generalidades

Para habilitar Cloudflare para tu sitio:

  1. Navega a Manage Domains.
    • Debes agregar un plan de alojamiento a tu cuenta para poder utilizar cualquier Servicio de Cloudflare. Visita el artículo cómo agregar un dominio para obtener instrucciones.
    • Si tu dominio está en DreamPress, debe ponerte en contacto con soporte para habilitar un plan Cloudflare.
  2. Una vez que agregues un plan de alojamiento a tu dominio, haz clic en el botón Edit a la derecha del dominio en la columna 'Web Hosting'.
  3. Si estás utilizando Cloudflare sin un subdominio, selecciona 'Add WWW' para el nombre de dominio cuando se te pregunte 'Do you want www in your URL?'.
  4. Desplázate hacia abajo a la sección 'Cloudflare Services'.
  5. Marca la casilla de verificación 'Enable Cloudflare on this domain?'.
  6. Seleccione el tipo de cuenta Cloudflare:
    • Si no tienes una cuenta de Cloudflare, deberás seleccionar la opción 'New Account' en el menú desplegable para la cuenta de Cloudflare. Luego deberás ingresar una dirección de correo electrónico válida que te gustaría usar para la cuenta y crear una contraseña para la nueva cuenta de Cloudflare. Utilizarás este nombre de usuario y contraseña para iniciar sesión y administrar tu cuenta en Cloudflare.com.
    • Si ya tienes una cuenta de Cloudflare, selecciona la opción 'Existing Account' en el menú desplegable. Luego deberás ingresar la dirección de correo electrónico y la contraseña de tu cuenta de Cloudflare.
    • Si ya configuraste una cuenta de Cloudflare en tu panel para otro dominio o configuraste Cloudflare anteriormente, verás una opción para 'Existing Accounts' en el menú desplegable. Si ves esta opción y deseas utilizar una cuenta existente, simplemente selecciona la dirección de correo electrónico que desea utilizar.
  7. Una vez que hayas configurado tu cuenta de Cloudflare, selecciona el nivel de suscripción de Cloudflare.
    • Visite el artículo de Suscripciones de Cloudflare para obtener más detalles sobre estos planes.
  8. Desplázate hacia abajo y haz clic en el botón Change Settings para guardar tu nueva configuración.
    • Dentro de 5-10 minutos, el dominio se actualizará para usar Cloudflare en DreamHost.
    • Sin embargo, los cambios de DNS en el dominio tardarán más en propagarse completamente en línea para permitir que Cloudflare funcione.

Después de habilitar Cloudflare, también verás Cloudflare en la lista junto con la versión de PHP en la columna ‘Web Hosting’ en la página ‘Manage Domains’. También verás una opción Clear Cache a la derecha del dominio en caso de que necesites purgar tu caché de Cloudflare.

Desactivar Cloudflare

Para deshabilitar Cloudflare, repite los pasos anteriores, pero asegúrate de desmarcar la casilla de verificación Enable Cloudflare on this domain? antes de guardar los cambios.
