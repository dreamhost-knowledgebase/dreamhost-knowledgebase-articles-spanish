Lo siguiente se describe cómo especificar una salida personalizada y una ubicación de registro de errores al iniciar tu aplicación Node.js.

Especificar la ubicación del registro de errores y de salida en Node.js

Corre el siguiente comando para especificar la ubicación del registro de errores y de salida.

Asegúrate de cambiar example.js al nombre de tu aplicación antes de correr el comando:

[server]$ node example.js > node_log.log 2> node_error.log
