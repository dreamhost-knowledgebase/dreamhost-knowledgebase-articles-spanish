Generalidades

Node.js es un nuevo entorno de tiempo de ejecución que se ha vuelto muy popular entre los desarrolladores web. Utiliza JavaScript, tradicionalmente un lenguaje de programación front-end en el lado del servidor.

Prerrequisitos

Un servidor VPS o Dedicated.  Node.js no se puede instalar en un servidor compartido.

¿Se puede usar Node.js en servidores compartidos?

No. DreamHost no admite Node.js en servidores web compartidos, ya que la Configuración de Seguridad

en los servidores compartidos DreamHost es incompatible con la compilación o ejecución de Node.js.

Si intentas compilar Node.js en uno de los servidores web compartidos, tu usuario será bloqueado automáticamente a través de grsec (eliminando todos los sitios web PHP que se ejecutan bajo ese usuario) y el servidor tendrá que reiniciarse antes de que tu usuario pueda ser excluido. Si lo haces una vez más, serás obligado a pasar a un VPS.
