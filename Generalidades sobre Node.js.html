Generalidades

Node.js es un nuevo entorno de tiempo de ejecución que se ha vuelto muy popular entre los desarrolladores web. Usa JavaScript, tradicionalmente un lenguaje de programación front-end en el lado del servidor.

¿En que servidores puedo instalar Node.js?

Node.js puede ser instalado en Servidores compartidos, VPS, y Dedicados.

Instalar una versión personalizada de NVM y Node.js

Node.js no está instalado de forma predeterminada en los servidores DreamHost más nuevos. Debes instalar manualmente una versión personalizada con el nombre de usuario de tu servidor. Consulta el siguiente artículo para obtener instrucciones sobre cómo instalar una versión personalizada de Node.js.

  • Instalar una versión personalizada de NVM y Node.js

Implementar Node.js con Passenger usando el panel DreamHost

Si bien no es necesario usar Passenger con Node.js, podrías ayudar a facilitar la implementación de tu aplicación. También te permite cargar tu sitio en el puerto 80 o 443, suponiendo que el archivo para cargar tu aplicación se llama app.js.

¿Es posible ejecutar Node.js en los puertos 80 o 443?

Sí, pero solo si has habilitado Passenger en el panel y el archivo para cargar tu aplicación se llama app.js.

La versión de Passenger que se corre en los servidores DreamHost no funciona actualmente con las versiones 14+ de Node.js.
