Generalidades

Node.js es un nuevo entorno de tiempo de ejecución que se ha vuelto muy popular entre los desarrolladores web. Utiliza JavaScript, tradicionalmente un lenguaje de programación front-end en el lado del servidor.

¿En qué servidores se puede instalar Node.js?

Node.js se puede instalar en servidores compartidos Shared, VPS y Dedicados.

Consulta el siguiente artículo para obtener instrucciones sobre cómo instalar una versión personalizada de Node.js.

  • Instalar una versión personalizada de NVM y Node.js

Para garantizar la compatibilidad con los paquetes Node.js, siempre se recomienda instalar una versión personalizada de Node.js con tu nombre de usuario, en lugar de utilizar la versión predeterminada del servidor.
