Generalidades

Los metacomandos de API te brindan información sobre la API, como, qué comandos y claves de API están disponibles en tu cuenta. Hay dos metacomandos de API disponibles para correr.

  • Enumera todos los comandos disponibles (api-list_accessible_cmds)
  • Enumera todas las llaves de API (api-list_keys)

Revisa el siguiente artículo para obtener instrucciones sobre cómo conectarte a la API de DreamHost. Usarás las instrucciones de este artículo para crear tu clave de API.

  • Conectarse con una API de DreamHost

api-list_accessible_cmds

El comando api-list_accessible_cmds enumera todos los comandos a los que tiene acceso la Llave API usada en el comando.

Por ejemplo, el siguiente comando solo generaría comandos que la Llave 1A2B3C4D5E6F7G8H se le han otorgado permisos de acceso.

Estos permisos fueron configurados cuando la llave fue creada originalmente.

https://api.dreamhost.com/?key=1A2B3C4D5E6F7G8H&cmd=api-list_accessible_cmds

api-list_keys

El comando api-list_keys enumera todas las llaves de API que la cuenta ha creado.

Debes tener una llave API con acceso a todos los comandos ("*") para usar este comando.

https://api.dreamhost.com/?key=1A2B3C4D5E6F7G8H&cmd=api-list_keys
