Generalidades

Zen Cart es un software gratuito que se utiliza para ejecutar un sitio web de comercio electrónico. Esto te permite vender tus productos directamente desde tu sitio web como una tienda en línea.

Puedes instalar Zen Cart manualmente siguiendo las instrucciones de este artículo.

 

Paso 1 — Instalar un certificado SSL

Dado que Zen Cart se utiliza para vender productos en tu sitio web, primero debes instalar un certificado SSL en tu sitio. Esto asegura que todas las transacciones hacia y desde tu sitio sean seguras. Hay dos opciones de certificado SSL en el panel DreamHost.

  • Let's Encrypt (Gratuito)
  • Sectigo (De pago)

Ambos certificados se pueden usar para ejecutar un sitio de comercio electrónico. Sin embargo, se recomienda un certificado Sectigo pagado, ya que parecerá más confiable para los clientes potenciales. Revisa el siguiente artículo para obtener más detalles.

  • ¿Qué tipo de certificado debería usar para mi sitio de comercio electrónico?

Revisa el siguiente artículo para obtener instrucciones sobre cómo instalar el certificado elegido.

  • Generalidades sobre como añadir un certificado SSL

Paso 2 — Crear una cuenta con un procesador de pagos

El certificado SSL solo protege los datos que se transmiten en tu sitio web. También debes crear una cuenta con un procesador de pagos. Este procesador de pagos es la empresa que procesará los datos de la tarjeta de crédito.

DreamHost no recomienda ningún procesador de pagos en particular, pero puedes encontrar muchas opciones en línea para elegir.

Paso 3 — Crear una base de datos

Zen Cart requiere una base de datos para almacenar y procesar los datos de tu carrito de compras. Ve el siguiente artículo para crear una nueva base de datos en tu panel DreamHost.

  • Crear una base de datos MySQL

Paso 4 — Descargar el software de Zen Cart

Hay dos opciones para agregar el software Zen Cart a tu sitio.

  • FTP (fácil) — Puedes descargar el software a tu computador y luego cargarlo en tu sitio usando un cliente FTP.
  • SSH (avanzado) — También puede descargar el software directamente a su sitio a través de SSH.

Las siguientes instrucciones descargan e instalan la versión 1.5.7.

Descargar a través de FTP

  1. Visita la siguiente página para descargar el software Zen Cart.
    • Descargar Zen Cart
  2. Haz clic en el primer enlace titulado CURRENT.
  3. En la página siguiente, verás varios archivos que puedes descargar. Haz clic en el archivo .zip. Esto descarga el archivo a tu computador.
  4. Busca el archivo que acabas de descargar. Se titulará algo así como zen-cart-v1.5.7-06232020.zip. La mayoría de las veces puede encontrarlo en tu directorio /descargas.
  5. En Windows, haz clic con el botón derecho en el archivo y elige Extraer todo... en el menú emergente
  6. Se abre una nueva ventana. En la nueva ventana, haz clic en el botón Extraer. Todos los archivos se extraen a una nueva carpeta con el mismo título que el archivo .zip.
  7. Inicia sesión en tu servidor usando un cliente FTP.
  8. En el panel izquierdo, navega hasta el directorio de tu computador donde extrajo el archivo .zip.
  9. Haz clic en este directorio. Deberías ver una lista de archivos.
  10. En el panel derecho, asegúrate de estar en el directorio de tu sitio web. No debería haber ningún archivo aquí que no sea un posible archivo favicon.ico.
  11. Resalta todos los archivos en el panel izquierdo y arrástralos al panel derecho para subirlos a tu sitio.

    Esto puede tardar varios minutos, ya que hay muchos archivos para cargar.

Descargar a través de SSH

  1. Inicia sesión en tu servidor a través de SSH.
  2. Navega en el directorio home de tu usuario. 
    [server]$ cd ~
  3. Visita la siguiente página para descargar el software Zen Cart.
    • Descargar Zen Cart
  4. Haz clic en el primer enlace titulado CURRENT.
  5. En la página siguiente, verás varios archivos que puedes descargar. Haz clic derecho en el archivo .zip. En el menú emergente, elige Copiar dirección de enlace.
  6. En el directorio de tu sitio web, corre el siguiente comando para descargar Zen Cart. 

    Cambia la URL por la que copiaste anteriormente.

    [server]$ wget https://sourceforge.net/projects/zencart/files/CURRENT%20-%20Zen%20Cart%201.5.x%20Series/zen-cart-v1.5.7-06232020.zip/download
  7. Esto descarga el software en un nuevo archivo titulado download. Corre el siguiente comando para descomprimir el archivo. 
    [server]$ unzip download
  8. Los archivos se extraen en un nuevo título directamente zen-cart-v1.5.7-06232020.
  9. Navega a este directorio. 
    [server]$ cd zen-cart-v1.5.7-06232020
  10. Copia todos el contenido en el directorio de tu sitio web. Cambia username a tu usuario Shell y example.com a tu sitio website. 
    [server]$ cp -r ./* /home/username/example.com
  11. Verifica el directorio de tu sitio web para confirmar que los archivos se hayan migrado. 
    [server]$ ls -la /home/username/example.com

Paso 5 — Instalar el software

Ahora que los archivos están en el directorio de tu sitio web, puedes completar la instalación.

  1. Visita tu sitio web en un navegador. Verás la página de inicio de Zen Cart.
  2. Haz clic en el enlace para proceder CLICK HERE.
  3. La primera página te informa que hay un problema con la base de datos MySQL. Ignora esto y haz clic en el botón Continue.
  4. En la página siguiente, ingresa la información de tu tienda y haz clic en Continue.
  5. Ingresa tus credenciales de base de datos en la siguiente página. 

    Deja la opción SQL Cache Method como en No SQL Caching. Siempre puede ajustar esto más tarde si es necesario.

  6. Haz clic en el botón Continue. Esto instala el software en tu base de datos.
  7. Ingresa tus credenciales de dominio, y luego haz clic en Continue.

    Toma nota de la 'Contraseña de administrador' en este momento.

  8. Aparece un mensaje de confirmación informándote que se ha instalado Zen Cart. Ahora debes eliminar el directorio zc_install como se muestra en el siguiente paso.

Paso 6 — Eliminar el directorio zc_install

Antes de poder utilizar Zen Cart, primero debe eliminar el directorio zc_install en el servidor. Una vez más, puedes optar por utilizar un cliente FTP o SSH.

Eliminar el directorio zc_install a través de FTP

  1. En tu cliente FTP, navega hasta el directorio de tu sitio web. Si previamente cargaste los archivos de Zen Cart, ya está en este directorio.
  2. Haz clic derecho en el directorio titulado zc_install y selecciona Eliminar.
  3. Elimina el directorio.

Visita el enlace de la página de administración desde la página de confirmación. Aquí puedes iniciar sesión en el backend de tu sitio web y comenzar a cargar tu contenido.

Eliminar el directorio zc_install a través de SSH

  1. Inicia sesión en tu servidor a través de SSH.
  2. Navega al directorio de tu sitio web. 
    [server]$ cd ~/example.com
  3. Elimina el directorio zc_install. 
    [server]$ rm -rf zc_install

Visita el enlace de la página de administración desde la página de confirmación. Aquí puedes iniciar sesión en el backend de tu sitio web y comenzar a cargar tu contenido.

 

Actualizar Zen Cart

Actualmente, Zen Cart no ofrece una forma sencilla de actualizar a la versión más reciente de su software. La actualización es un proceso manual complejo que requiere la capacidad de leer, comparar y editar código dentro de los archivos de tu sitio web. El proceso implica los siguientes pasos básicos:

  1. Descargar tu sitio actual a tu computador.
  2. Descarga del nuevo software a tu computador.
  3. Usar un programa tercero para comparar tus archivos actuales con los archivos de la nueva versión.
  4. Ajustar manualmente tus archivos para que funcionen con la versión más reciente.
  5. Crear una tienda temporal para probar tus nuevos archivos.
  6. Cambiar tu tienda de prueba a tu tienda en vivo.

Si no te sientes cómodo con este proceso, te recomendamos que busque un desarrollador que pueda actualizar tu sitio por ti. A continuación se muestran las instrucciones oficiales de Zen Cart sobre cómo realizar este proceso.
