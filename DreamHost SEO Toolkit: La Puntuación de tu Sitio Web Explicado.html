Generalidades

Tu puntaje se actualiza automáticamente todos los días y también cada vez que haces cambios en tu sitio web y usas el complemento. Esto activa el escaneo de todo tu sitio web para que siempre puedas saber cómo los cambios que has hecho para cada tarea afectan tu progreso de SEO.

¿Cómo debo leer mi puntaje?

Tu puntaje de SEO puede serte útil de varias maneras y debe usarse como un medio para:

  • Medir éxito — Analiza la evolución de tu puntuación como indicador de rendimiento a medio y largo plazo.
  • Identificar inconvenientes — Una disminución importante en la puntuación debería advertirte sobre los problemas del sitio web que debes abordar.
  • Comparar sitios — Compara tu puntaje de SEO con otros sitios para determinar cómo se está desempeñando tu sitio web frente a tus competidores.

Completar tareas para optimizar tus páginas puede ayudarte a aumentar la puntuación de SEO de tu sitio, pero es posible que este trabajo por sí solo no te brinde el impulso que te gustaría ver — también deberás monitorear y mantener tu sitio con regularidad para ver a largo plazo mejoras en tu puntaje de SEO.

Para resumir

  • Debes analizar el progreso de tu puntuación como indicador de rendimiento de tu sitio web a corto y largo plazo para SEO.
  • Los cambios importantes en la puntuación deberían advertirte sobre problemas en tu sitio.
  • Cada tarea que completes dentro de la herramienta te indicará si está progresando o no.
