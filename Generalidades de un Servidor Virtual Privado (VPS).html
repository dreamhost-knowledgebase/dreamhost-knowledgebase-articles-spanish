Generalidades

DreamHost Shared, VPS, Dedicated, y DreamPress son cuatro de nuestros planes de alojamiento disponibles para tu sitio web.

El servidor privado virtual DreamHost (VPS) es una solución de alojamiento premium basada en discos de estado sólido (SSD). Está disponible como complemento de un plan existente en DreamHost o un servicio independiente. Los VPS de DreamHost usan Linux-VServer para crear una "máquina virtual" que protege los recursos de CPU y RAM para que sean solo utilizados por un usuario, y no sean utilizados por otros usuarios en la misma máquina física.

Por ejemplo, en un servidor compartido los recursos son compartidos por otros usuarios. En un VPS, estos recursos están más aislados, lo que mejora el rendimiento y ayuda a mantener la estabilidad.

¿Qué características incluye un VPS?

  • CPU y RAM protegida.
  • Se permiten procesos persistentes (y nunca se eliminan).
  • Capacidad para alterar la memoria sobre la marcha, sin necesidad de reiniciar.
  • Posibilidad de reiniciar el servidor desde el panel.
  • La memoria histórica y los gráficos de carga se pueden ver en el panel.
  • Unidades basadas en SSD que ofrecen velocidades 20 veces más rápidas que las unidades de disco SATA tradicionales.

Direcciones IP

Cada VPS viene con su propia dirección IP incluida. Luego puedes agregar una IP única a cada dominio (IPv4 o IPv6) por una tarifa adicional. Puede leer más sobre IP únicas aquí:

  • Generalidades de IP únicas

Por favor ten en cuenta que DreamHost no puede ofrecer direcciones IP estáticas. Esto significa que la IP de tu sitio (incluidas las IP únicas) está sujeta a cambios. Si bien no sucede tan a menudo, DreamHost no garantiza que una dirección IP nunca cambie. Sin embargo, cuando se produce un cambio de IP, se notificará por correo de manera oportuna.

Procesos persistentes

A diferencia de una configuración de Shared Hostingregular, los VPS permiten la ejecución de procesos persistentes. Independientemente de cuánto tiempo lleve ejecutar estos procesos (invocado en la web, invocado por la consola, cron o de otro modo), el servidor nunca elimina los procesos.

Ajuste dinámico de la configuración del servidor

Muchas de las opciones de configuración clave para tu servidor VPS están configuradas sobre la marcha por el sistema de automatización de DreamHost para aprovechar al máximo los recursos disponibles de tu servidor. Por ejemplo, si tienes un servidor VPS de 300 MB, Apache está configurado para usar una cantidad de memoria menor que si tuviera un servidor de 1000 MB. La configuración de Apache es proporcional a la asignación de memoria y se ajusta cuando se aumenta la asignación para el servidor.

¿Qué es un VPS Legacy?

El Legacy VPS se refiere a los antiguos planes de VPS que DreamHost ofrecía anteriormente en el pasado. Si bien estas configuraciones están desactualizadas y ya no están disponibles, aún pueden existir instancias en algunas cuentas. Por favor pónte en contacto con el soporte técnico si tienes un VPS Legacy y deseas actualizar a una de las ofertas más nuevas de DreamHost.

Cómo verificar la versión del sistema operativo

Para ver el sistema operativo, inicia sesión en tu servidor a través de SSH. Una vez que hayas iniciado sesión, corre el siguiente comando:

[server]$ cat /etc/issue

¿Las bases de datos MySQL están en el VPS?

No. Un VPS sólo ejecuta archivos de sitios web. Todas las bases de datos que agregues en el panel se alojarán en un servidor MySQL compartido.

También puedes comprar un servidor MySQL privado, pero este sería un segundo servidor, adicional a tu VPS. Mira el siguiente artículo para más detalles:
