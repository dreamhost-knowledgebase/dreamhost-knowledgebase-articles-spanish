Generalidades

El Servidor Virtual Privado de DreamHost (VPS) es un plan de alojamiento premium disponible como adición a un plan de DreamHost existente o un servicio independiente.

Un VPS usa Contenedores de Linux (LXC) para crear un servidor "virtual" que protege los recursos RAM de otros usuarios en la misma máquina física. Por ejemplo, en un servidor compartido, los recursos son compartidos entre otros usuarios. En un VPS, esos recursos están más asolados, lo que mejora el rendimiento y ayuda a mantener la estabilidad.

Este artículo te da una visión general de los planes de alojamiento VPS de DreamHost, también como los pasos siguientes para comenzar.

Características de los VPS

La siguientes es una lista de características a las que tendrás acceso luego de agregar un VPS.

  • RAM protegida.
  • Se permiten procesos persistentes (y nunca se eliminan).
  • Habilidad de mejorar la RAM sin reiniciar (degradar reiniciará el servidor)
  • Posibilidad de reiniciar el servidor desde el panel.
  • La memoria histórica y los gráficos de carga se pueden ver en el panel.

Direcciones IP

Cada VPS viene con su propia dirección IP de servidor incluida. También puedes agregar una IP única a cada dominio (IPv4 o IPv6) por una tarifa adicional. Revisa el siguiente artículo para más detalles:

  • Generalidades de las IP Únicas

Por favor ten en cuenta que DreamHost no puede ofrecer direcciones IP estáticas. Esto significa que la IP de tu sitio (incluidas las IP únicas) está sujeta a cambios.

Si bien no sucede tan a menudo, DreamHost no garantiza que una dirección IP nunca cambie. Sin embargo, cuando se produce un cambio de IP, se te notificará por correo de manera oportuna.

Procesos persistentes

A diferencia de un plan de Shared Hosting regular, los VPS permiten la ejecución de procesos persistentes. Independientemente de cuánto tiempo lleve ejecutar estos procesos (invocado en la web, invocado por la consola, cron o de otro modo), el servidor nunca elimina los procesos.

DNS de Wildcard

Es posible agregar un DNS Wildcard a tu VPS, pero esto debe ser habilitado por soporte técnico. Por favor visita el artículo DNS Wildcard para más información.

Versión del sistema operativo

Los planes de VPS ejecutan el sistema operativo de Ubuntu.

¿Las bases de datos MySQL están en el VPS?

No. Un VPS sólo ejecuta archivos de sitios web. Todas las bases de datos que agregues en el panel se alojarán en un servidor MySQL compartido.

También puedes comprar un MySQL VPS, pero este sería un segundo servidor, adicional a tu VPS. Revisa el siguiente artículo para más detalles:

  • Generalidades de MySQL VPS
