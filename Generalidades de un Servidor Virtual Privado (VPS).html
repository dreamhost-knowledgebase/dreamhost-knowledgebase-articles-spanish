Generalidades

El Servidor Virtual Privado de DreamHost (VPS) es un plan de alojamiento premium disponible como adición a un plan de DreamHost existente o un servicio independiente.

Los nuevos VPS agregados en el panel ejecutan Ubuntu Focal y usan Linux-Containers (LXC) para crear un servidor "virtual" que protege los recursos de RAM de un usuario de otros usuarios en la misma máquina física. Por ejemplo, en un servidor compartido, los recursos son compartidos por otros usuarios. En un VPS, estos recursos están más aislados, lo que mejora el rendimiento y ayuda a mantener la estabilidad.

¿Qué características incluye un VPS?

  • RAM protegida.
  • Se permiten procesos persistentes (y nunca se eliminan).
  • Habilidad de mejorar la RAM sin reiniciar (degradar reiniciará el servidor)
  • Posibilidad de reiniciar el servidor desde el panel.
  • La memoria histórica y los gráficos de carga se pueden ver en el panel.

Direcciones IP

Cada VPS viene con su propia dirección IP incluida. Luego puedes agregar una IP única a cada dominio (IPv4 o IPv6) por una tarifa adicional. Revisa el siguiente artículo para más detalles:

  • Generalidades de IP únicas

Por favor ten en cuenta que DreamHost no puede ofrecer direcciones IP estáticas. Esto significa que la IP de tu sitio (incluidas las IP únicas) está sujeta a cambios.

Si bien no sucede tan a menudo, DreamHost no garantiza que una dirección IP nunca cambie. Sin embargo, cuando se produce un cambio de IP, se te notificará por correo de manera oportuna.

Procesos persistentes

A diferencia de una configuración de los Shared Hosting regular, los VPS permiten la ejecución de procesos persistentes. Independientemente de cuánto tiempo lleve ejecutar estos procesos (invocado en la web, invocado por la consola, cron o de otro modo), el servidor nunca elimina los procesos.

¿Qué es un VPS Legacy?

El Legacy VPS se refiere a los antiguos planes de VPS que DreamHost ofrecía anteriormente en el pasado. Si bien estas configuraciones están desactualizadas y ya no están disponibles, aún pueden existir instancias en algunas cuentas. Por favor ponte en contacto con el soporte técnico si tienes un VPS Legacy y deseas actualizar a una de las ofertas más nuevas de DreamHost.

Cómo verificar la versión del sistema operativo

Para ver el sistema operativo, inicia sesión en tu servidor a través de SSH. Una vez que hayas iniciado sesión, corre el siguiente comando:

[server]$ cat /etc/issue

¿Las bases de datos MySQL están en el VPS?

No. Un VPS sólo ejecuta archivos de sitios web. Todas las bases de datos que agregues en el panel se alojarán en un servidor MySQL compartido.

También puedes comprar un servidor MySQL privado, pero este sería un segundo servidor, adicional a tu VPS. Mira el siguiente artículo para más detalles:

  • Generalidades de MySQL VPS
