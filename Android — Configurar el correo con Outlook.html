Generalidades

A continuación se describe cómo agregar una cuenta de correo en un dispositivo Android usando la aplicación Outlook.

Revisa los siguientes artículos para obtener una lista de configuraciones de servidor aceptables:

  • Configuración de Clientes de Correo
  • Protocolos y números de puertos de clientes de correo

Para obtener una lista de aplicaciones de correo alternativas, dirígete a la sección Ver también en la parte inferior de esta página. Sin embargo, la configuración es generalmente la misma para cualquier aplicación que elijas.

Configurar tu correo

  1. Asegúrate de haber instalado la aplicación Outlook de Google Play Store. Una vez instalada, abre la aplicación de correo.
  2. Presiona el botón Agregar una cuenta.
  3. Ingresa la dirección de correo electrónico a la que deseas conectarse. En este ejemplo, el correo es ejemplo@example.com.
  4. Presiona el botón Continuar.
  5. Elije qué tipo de cuenta quisieras usar. Se recomienda IMAP.
  6. Da clic en Configuraciones Avanzadas en la parte inferior de la página.
  7. Ingresa lo siguiente en la configuración Servidor de correo entrante IMAP / POP3:
    • Nombre de Host IMAP / POP3 – Revisa el artículo Nombre de servidor de correo para asegurarte de que estás usando el nombre del servidor de correo correcto. Esto debe ser imap.dreamhost.com o pop.dreamhost.com.
    • Puerto – Esto debe ser 993 cuando se usa IMAP o 995 cuando sea POP3.
    • Tipo de seguridad – Selecciona SSL/TLS del menú desplegable.
    • Nombre de usuario de IMAP / POP3 – Tu dirección de correo completa.
    • Contraseña de IMAP / POP3 – La contraseña de tu dirección de correo.
  8. Ingresa la siguiente configuración Servidor de correo saliente SMTP:
    • Nombre de Host SMTP – Esto siempre debe ser smtp.dreamhost.com.
    • Puerto – En la mayoría de los casos, se debe usar 465.
    • Tipo de seguridad – Selecciona SSL/TLS del menú desplegable.
    • Nombre de usuario SMTP – Tu dirección de correo completa.
    • Contraseña de SMTP – Tu contraseña.
  9. Haz clic en el botón de marca de verificación para completar la configuración.

Editar la configuración de tu correo

Una vez que se haya agregado tu correo electrónico, no podrás editar los ajustes de configuración. Para editar la configuración del correo, debes eliminarlo Outlook y agregarlo nuevamente. Las siguientes instrucciones describen cómo eliminar tu correo de Outlook.

  1. Abre la aplicación de Outlook
  2. Da clic en el icono de 'sobre'.
  3. Da clic en el icono de 'engranaje'.
  4. Selecciona la cuenta de tu correo debajo de Cuentas de correo.
  5. Selecciona la opción ELIMINAR CUENTA.
  6. Toca el botón ELIMINAR para eliminar el correo.
  7. Agrega el correo de nuevo a la aplicación usando los nuevos detalles de configuración descritos arriba.
