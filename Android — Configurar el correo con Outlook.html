Generalidades

A continuación se describe cómo agregar una cuenta de correo en un dispositivo Android usando la aplicación Outlook.

Ve los siguientes artículos para obtener una lista de configuraciones de servidor aceptables:

  • Configuración de Clientes de Correo
  • Protocolos y números de puertos de clientes de correo

Para obtener una lista de aplicaciones de correo alternativas, consulta la sección Ver también en la parte inferior de esta página. Sin embargo, la configuración es generalmente la misma para cualquier aplicación que elijas.

Configurar tu correo

  1. Asegúrate de haber instalado la aplicación Outlook de Google Play Store. Una vez instalada, abre la aplicación de correo.
  2. Presiona el botón Add Account.
  3. Ingresa la dirección de correo electrónico a la que deseas conectarse. En este ejemplo, el correo es example@dreamhostexample.com.
  4. Presiona el botón Continue.
  5. Presiona el botón del signo de interrogación.
  6. Selecciona la opción Change account provider en el menú que aparece.
    La pantalla Choose account type se abre:
  7. Elige qué tipo de cuenta te gustaría usar en la sección Advanced. Se recomienda IMAP.
  8. Ingresa lo siguiente en la configuración IMAP / POP3 Incoming Mail Server:
    • IMAP / POP3 Host Name – Ve  el artículo nombre de servidor de correo para asegurarte de que estás usando el nombre del servidor de correo correcto. Esto debe ser imap.dreamhost.com o pop.dreamhost.com.
    • IMAP / POP3 Username – Tu dirección de correo completa.
    • IMAP / POP3 Password – La contraseña de tu dirección de correo.
  9. Ingresa la siguiente configuración SMTP Outgoing Mail Server:
    • SMTP Host Name – Esto siempre debe ser smtp.dreamhost.com.
    • SMTP Username – Tu dirección de correo completa.
    • SMTP Password – Tu contraseña.
  10. Haz clic en el botón de marca de verificación para completar la configuración.

Editar la configuración de tu correo

Una vez que se haya agregado tu correo electrónico, no podrás editar los ajustes de configuración. Para editar la configuración del correo, debes eliminarlo Outlook y agregarlo nuevamente. Las siguientes instrucciones describen cómo eliminar tu correo de Outlook.

  1. Abre la aplicación de Outlook
  2. Da clic en el icono de 'sobre'.
  3. Da clic en el icono de 'engranaje'.
  4. Selecciona la cuenta de tu correo debajo de Mail accounts.
  5. Selecciona la opción DELETE ACCOUNT.
  6. Toca el botón DELETE para eliminar el correo.
  7. Agrega el correo de nuevo a la aplicación usando los nuevos detalles de configuración descritos arriba.
