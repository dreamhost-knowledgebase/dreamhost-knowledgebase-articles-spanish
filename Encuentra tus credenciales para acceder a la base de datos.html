Generalidades

Las credenciales de tu base de datos te permiten iniciar sesión por medio del panel a tu base de datos usando. phpMyAdmin es una interfaz gráfica donde puedes navegar a través de tu base de datos y realizar cambios.

Estas mismas credenciales también se pueden usar para conectarse a la base de datos utilizando el software que desees.

Asegúrate de iniciar sesión en tu base de datos únicamente si sabes exactamente lo que necesitas ajustar. Cualquier configuración incorrecta en tu base de datos podría hacer que tu sitio no funcione correctamente.

Una observación sobre la configuración de DNS

Si tu DNS está alojado en DreamHost

Si tus nombres de servidores (nameservers) están alojados en DreamHost, sigue los pasos a continuación.

Si acabas de instalar una aplicación de un solo clic (One-click install)

Si acabas de instalar una aplicación de un solo clic (y tus nombres de servidores o “nameservers” apuntan a DreamHost), tendrás que esperar algunas horas para que se actualice el nuevo subdominio mysql. Una vez que el DNS se haya propagado en línea, tu sitio se conectará a tu base de datos.

Si tu DNS NO está alojado en DreamHost

Si tus nombres de servidores o 'nameservers' NO están alojados en DreamHost, debes crear un registro A (en tu proveedor de alojamiento actual) para que tu nombre de host 'mysql.ejemplo.com' apunte a DreamHost. Solo después de que el registro A de mysql se actualice, funcionará el enlace phpMyAdmin en tu panel. Revisa el siguiente artículo para obtener instrucciones sobre cómo ubicar tu registro A de mysql de DreamHost. Luego puedes contactar a tu proveedor de alojamiento actual para agregarlo a su sistema.

  • A cuál registro A debo apuntar mi sitio?

Solo necesitas crear el registro 'A' para el subdominio de mysql como en 'mysql.example.com'.

Como alternativa, puedes ingresar usando los nombres de host de los centros de datos de la costa Este y Oeste como se muestra en el Paso #4 debajo.

Paso 1 — Encuentra el nombre de tu base de datos

Visita la página MySQL Databases y desliza hacia abajo hasta la sección titulada 'Bases de datos en este servidor'. A continuación verás una lista de tus bases de datos.

Si instalaste un One-Click Install desde el panel, verás el nombre de la aplicación bajo la columna 'Descripción'. Si tienes múltiples bases de datos, deberás identificar cuál de ellas corresponde a tu sitio. Usualmente esto lo encuentras en el archivo de configuración de tu sitio, pero cada aplicación lo puede poner en una ubicación diferente. Consulta el centro de ayuda de tu aplicación para conocer la ubicación de archivos de configuración.

Anota el nombre de la base de datos.

Paso 2 — Encuentra tu usuario

El usuario específico ligado a la base de datos se encuentra a la derecha de cada una de las bases de datos bajo la columna 'Usuarios de acceso'. Haz una nota de ellos.

Paso 3 — Encuentra tu contraseña

Haz click en el usuario de la base de datos.

En la siguiente página, desliza hacia abajo y haz click en el botón Show para ver la contraseña de dicho usuario. Anótalo también.

Paso 4 — Encuentra tu nombre de Host

Navega de nuevo a la página MySQL Databases.

Hacia la mitad de la página verás una sección titulada 'Nombres de Host para este Servidor de MySQL' (Hostnames for this MySQL server). Busca el nombre de host que corresponde a tu sitio. Lo más probable es que el nombre de tu sitio esté incluído en dicho nombre de Host.

A la derecha del nombre de host verás un link que dice phpMyAdmin. Haz click en dicho link para acceder a la base de datos a través de phpMyAdmin.

También puedes usar el nombre de Host oficial de phpMyAdmin de tu Data Center. Primero revisa la página Centro de Datos desde el panel. Desliza hacia abajo para ubicar el centro de datos donde tus servicios de MySQL están localizados. Puede ser costa Este o costa Oeste. Puedes entonces usar los nombres de host que se muestran a continuación dependiendo de la ubicación de tu servidor MySQL.


West Coast: https://west1-phpmyadmin.dreamhost.com
East Coast: https://east1-phpmyadmin.dreamhost.com

Log into phpMyAdmin

En la página de inicio de phpMyAdmin, ingresa tu nombre de host, usuario, contraseña, y haz click en el botón Ir. (Go)

phpMyAdmin requiere que el dominio usado para tu nombre de host de MYSQL esté Completamente Alojado y utilizando los DNS de DreamHost. Puedes hacer esto de dos maneras:

  • Apunta tus Nombres de Servidor (nameservers) to DreamHost, a DreamHost, o
  • Apunta tu registro 'A' hacia la IP de DreamHost. Revisa el Artículo de DNS para obtener detalles de cómo encontrar tu registro IP de DreamHost.

Si ninguno de tus nombres de host está funcionando, visita el siguiente artículo sobre cómo acceder a tu base de datos antes de actualizar los DNS:
