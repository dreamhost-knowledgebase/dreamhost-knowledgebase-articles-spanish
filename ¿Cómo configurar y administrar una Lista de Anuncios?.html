Configurar la lista

  1. Navega hasta la página Listas de Anuncios.
  2. Da clic en el botón Agregar nueva Lista de Anuncios. La página 'Crear nueva Lista de Anuncios' se abrirá. Las siguientes capturas de pantalla se han dividido para mayor claridad:
  3. Ingresa lo siguiente:
    • De nombre: El nombre de tu lista o boletín aquí.
    • Dirección de correo de la lista: El correo que deseas crear como el correo de la lista. Además, asegúrate de que el menú desplegable a la derecha esté lleno con el dominio en el que deseas que se cree la lista.
    • Asunto predeterminado de anuncio: Este es un campo opcional donde puedes ingresar el asunto de tu mensaje.
    • Confirmación de Asunto de Correo: Cuando un usuario se suscribe a tu lista, se envía un correo de confirmación al usuario. Ingresa en este campo cualquier línea de asunto que te gustaría usar para este mensaje.
    • Confirmación de Mensaje de Correo: Cuando un usuario se suscribe a tu lista, recibe un correo de confirmación con los detalles enumerados en el campo a la derecha. El script está configurado para completar automáticamente los detalles entre los guiones dobles (--) para esta lista en particular. Puedes editar el aviso como quieras. Ten en cuenta que el código HTML ingresado aquí no se representa correctamente.
    • Instrucciones para Dar de Baja: Este aviso aparece en la parte inferior de cada anuncio que envía en caso de que el usuario ya no desee recibir este correo. El código HTML ingresado aquí se procesa correctamente, sin embargo, debes asegurarte de marcar la casilla de verificación Is this HTML? en la página final al enviar.
    Los campos restantes son URL opcionales que puede crear para personalizar la experiencia del usuario. Ajústalos según sea necesario:
  4. Cuando termines, haz clic en el botón ¡Crea esta lista! al final de la página.

Explicación de las variables en el cuadro 'Confirmation Email Message'

A continuación se enumeran las variables y de dónde se extraen:

Variable De donde es sacado
--their_email-- Este es el correo del cliente/usuario que ingresaste al editar la Lista de Anuncios. Estos correos se agregaron en la sección Emails to Add.
--listname-- El nombre de tu Lista de Anuncios. Cuando editas tu lista, este es el primer campo titulado 'List Name'.
--ip-- Esta es la dirección IP de la computadora que estás usando para enviar la Lista de Anuncios.
--ref_url-- ¿Éste predeterminado es https://panel.dreamhost.com/index.cgi?
--url-- Esta es una URL específica en la que el cliente/usuario hace clic para confirmar que desea unirse a la lista. Se parece a https://scripts.dreamhost.com/add_list.cgi?g=b8ccge71a6ae4767e70b2f8br77e02c2

Editar la lista

  1. Navega hasta la página Listas de Anuncios.
  2. A la derecha de la lista, haz clic en el botón Editar.
    Las páginas de edición se abren permitiéndote hacer tus ajustes.
    Solo hay unos pocos campos que se pueden cambiar después de hacer clic en el botón 'Edit' para la lista en cuestión:
    • De Nombre
    • Asunto predeterminado de anuncio
    • Confirmación de Asunto de Correo
    • Confirmación de Mensaje de Correo
    • Instrucciones para Dar de Baja
    • Eliminar direcciones incorrectas
    • Todos los campos de URL.

No puedes cambiar el nombre de la lista después de que se haya creado. Debes eliminarlo y crear una nueva lista con el nuevo nombre.

"Eliminar direcciones incorrectas"

Después de crear la lista y hacer clic en el botón Editar, verás una nueva opción titulada 'Eliminar direcciones incorrectas' donde puedes elegir eliminar las direcciones de correo de tu lista después de un número específico de rebotes.

También puedes establecerlo en 'Nunca' para que nunca se elimine ninguna dirección.
