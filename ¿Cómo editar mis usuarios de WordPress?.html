Generalidades

A continuación se describe cómo editar tu usuario de WordPress en su panel de administración.

Editar tu usuario en el panel WordPress

Puedes cambiar tu nombre a mostrar, dirección de correo electrónico o preferencias personales dentro del panel de WordPress.

  1. Inicia sesión en tu cuenta administrativa de WordPress.
  2. En el panel izquierdo, haz clic Usuarios > Todos los usuarios.
  3. De la lista de todos los nombres de usuario actuales que aparecen, haz clic en el usuario que deseas editar.
  4. Realiza cambios en tu usuario de WordPress. Consulta las Secciones de Perfil a continuación para obtener una lista de opciones de usuario.
  5. Haz clic en Actualizar perfil para guardar tus cambios.

Roles de usuario

Existen distintos tipos de usuarios que puedes asignar en WordPress:

Usuario Descripción
Administrador tiene acceso a todas las funciones de administración en un solo sitio
Editor puede publicar y administrar publicaciones, incluidas las publicaciones de otros usuarios
Autor puede publicar y administrar sus propias publicaciones
Colaborador puede escribir y administrar sus propias publicaciones, pero no puede publicarlas
Subscriptor solo puede administrar su perfil

Secciones de Perfil

Las opciones de usuario dentro de la página de perfil se dividen en varias secciones, las cuales se describen a continuación.

Opciones personales

  • Editor visual: Deshabilita el editor visual.
  • Resaltado de sintaxis: Deshabilita el resaltado de sintaxis.
  • Esquema de color de administración: Cambia el esquema de color de la interfaz de usuario del panel de administración.
  • Atajos de teclado: Habilita las abreviaciones de teclado para moderación de comentarios.
  • Barra de herramientas: Habilita la barra de herramientas de administración cuando inicies sesión y veas el sitio.

Nombre

  • Nombre de usuario: Esto enumera el nombre de usuario interno de WordPress del usuario, que no se puede cambiar una vez creado el usuario.
  • Nombre y Apellidos: Estos campos son opcionales, pero se pueden completar para asociar al usuario con una persona específica.
  • Alias (obligatorio): Edita este campo para especificar un nombre personalizado para el usuario, el cual predetermina el nombre de usuario del usuario.
  • Mostrar este nombre públicamente: Haz clic en el menú desplegable para elegir qué opción de nombre desead que WordPress muestre a los visitantes para el usuario.

Información de contacto

  • Correo electrónico (obligatorio): Edita este campo para cambiar la dirección de correo asociada con el usuario. Si se cambia la dirección de correo del usuario, a la nueva dirección se te enviará un correo de confirmación antes de que pueda activarse.
  • Web: Edita este campo para agregar un sitio web personal al perfil del usuario.

Acerca de tí

  • Información biográfica: Este campo es opcional, pero se puede completar para agregar información biográfica al perfil del usuario.
  • Imagen de perfil: El avatar del usuario no se puede personalizar directamente en esta página, pero se puede editar a través del servicio Gravatar.

Administración de la Cuenta

  • Nueva contraseña: Haz clic en el botón Establecer una nueva contraseña para generar una nueva contraseña para el usuario. Revisa este artículo para más información.
  • Sesiones: Haz clic en el botón Desconectar del resto de sitios para finalizar todas las sesiones del usuario actual, excluyendo la actual.
