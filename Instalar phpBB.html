Generalidades

phpBB es un programa de foro que te permite crear un espacio en tu sitio web donde los miembros pueden hablar sobre diferentes temas e ideas sobre cualquier tema que elijan.

Para obtener más información, visite el sitio oficial de phpBB:

  • phpbb.com

DreamHost no ofrece phpBB como instalación con un solo clic; sin embargo, puedes instalarlo manualmente siguiendo las instrucciones de este artículo.

  • Paso 1 — Crear una base de datos
  • Paso 2 — Descargar el software phpBB (FTP o SSH)
  • Paso 3 — Instalar el software
  • Paso 4 — Quitar el directorio de instalación
 

Paso 1 — Crear una base de datos

phpBB requiere una base de datos para almacenar y procesar los datos de tu foro. Ve el siguiente artículo para crear una nueva base de datos en tu panel DreamHost.

  • Crear una base de datos MySQL

Paso 2 — Descargar el software phpBB

Hay dos opciones para agregar el software phpBB a tu sitio.

  • FTP (fácil) — Puedes descargar el software a tu computador y luego cargarlo en tu sitio usando un cliente FTP.
  • SSH (avanzado) — También puedes descargar el software directamente a tu sitio a través de SSH.

Las siguientes instrucciones descargan e instalan la versión 3.3.1.

Descarga vía FTP

  1. Visita la siguiente página para descargar el software phpBB.
    • Descargas phpBB
  2. En la página de Descargas, verás varias opciones. Haz clic en el botón titulado Download Latest. Esto descarga el archivo a tu computador.
  3. Busca el archivo que acaba de descargar. Se titulará algo así como phpBB-3.3.1.zip.La mayoría de las veces puedes encontrarlo en tu directorio /downloads.
  4. En Windows, haz clic con el botón derecho en el archivo y elige Extract All... en el menú emergente.
  5. Se abre una nueva ventana. En la nueva ventana, haz clic en el botón Extract Todos los archivos se extraen a una nueva carpeta con el mismo título que el archivo .zip.
  6. Inicia sesión en tu servidor usando un cliente FTP.
  7. En el panel izquierdo, navega hasta el directorio de tu computador donde extrajiste el archivo .zip.
  8. Haz clic en este directorio. Deberías ver una lista de archivos.
  9. En el panel derecho, asegúrate de estar en el directorio de tu sitio web. No deberías haber ningún archivo aquí que no sea un posible archivo favicon.ico.
  10. Resalta todos los archivos en el panel izquierdo y arrastralos al panel derecho para cargarlos en tu sitio.

    Esto puede tardar varios minutos ya que hay muchos archivos para cargar.

Descarga a través de SSH

  1. Inicia sesión en tu servidor vía SSH.
  2. Navega al directorio home de tu usuario. 
    [server]$ cd ~
  3. Visita la siguiente página para descargar el software phpBB.
    • Descargas de phpBB
  4. En la página de Descargas, verás varias opciones. Haz clic con el botón derecho en el botón titulado Download Latest. En el menú emergente, elige Copy link address.
  5. En el directorio de tu sitio web, corre el siguiente comando para descargar phpBB.

    Cambia la URL por la que copió anteriormente.

    [server]$ wget https://download.phpbb.com/pub/release/3.3/3.3.1/phpBB-3.3.1.zip
  6. Esto descarga el software en tu servidor. Corre el siguiente comando para descomprimir el archivo.

    Cambia el nombre del archivo zip por el que descargó anteriormente.

    [server]$ unzip phpBB-3.3.1.zip
  7. Los archivos se extraen en un nuevo directorio titulado phpBB3.
  8. Navega a este directorio. 
    [server]$ cd phpBB3
  9. Copia todo el contenido en el directorio de tu sitio web. Cambia username a tu usuario Shell y example.com a tu sitio web. 
    [server]$ cp -r ./* /home/username/example.com
  10. Verifica el directorio de tu sitio web para confirmar que los archivos se hayan migrado. 
    [server]$ ls -la /home/username/example.com

Paso 3 — Instalar el software

Ahora que los archivos están en el directorio de tu sitio web, puedes completar la instalación.

  1. Visita tu sitio web en un navegador. Verás la página de inicio de phpBB.
  2. Haz clic en la pestaña Install, seguido del botón Install para continuar.
  3. La primera página le pide que crea un inicio de sesión de administrador. Completa los campos y haz clic en el botón Submit.
  4. Ingresa las credenciales de tu base de datos en la página siguiente y haz clic en el botón Submit.
  5. Edita la ruta de tu sitio y la configuración de URL si deseas realizar algún cambio y luego haz clic en el botón Submit.
  6. Si deseas utilizar la autenticación SMTP para los correos de tu sitio, ingresa tus credenciales de correo aquí. Cuando hayas terminado, haz clic en Submit.

    DreamHost recomienda usar la autenticación SMTP, que te permite usar servicios de correo de terceros y ayuda a evitar que tus mensajes sean marcados incorrectamente por filtros de spam.

  7. En la siguiente página, completa el nombre de tu foro y una breve descripción, y luego haz clic en Submit.
    Esto inicia el instalador de phpBB.
  8. Aparece un mensaje de confirmación informandote que phpBB se ha instalado. Da click al enlace ACP para proceder.
  9. phpBB te solicita que habilites o deshabilites las estadísticas y la monetización antes de continuar. Haz tus elecciones y haz clic en Submit.

Ahora debes eliminar el directorio install como se muestra en el siguiente paso

Paso 4 — Quitar el directorio de instalación

Antes de que puedas utilizar phpBB, primero debes eliminar el directorio de install en el servidor. Una vez más, puedes optar por utilizar un cliente FTP o SSH.

Eliminar el directorio install vía FTP

  1. En tu cliente FTP, navega hasta el directorio de tu sitio web. Si previamente cargaste los archivos phpBB, ya está en este directorio.
  2. Haz clic con el botón derecho en el directorio titulado install y selecciona Delete.
  3. Elimina el directorio.

Eliminar el directorio install vía SSH.

  1. Inicia sesión en tu servidor vía SSH.
  2. Navega al directorio de tu sitio web. 
    [server]$ cd ~/example.com
  3. Elimina el directorio install. 
    [server]$ rm -rf install
