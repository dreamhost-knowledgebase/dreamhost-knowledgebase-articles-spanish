Generalidades

El rango de bytes es un proceso que permite que el entorno del servidor sirva partes específicas de una solicitud sin la necesidad de ejecutar la solicitud completa. El cliente puede solicitar partes de un archivo grande en lugar de la necesidad de todo el archivo. Este tipo de funcionalidad puede ser útil cuando se solicitan partes de un archivo multimedia grande (PDF, podcast, video, audio, etc.) mientras se carga el resto del contenido en segundo plano. El rango de bytes también se usa al continuar la descarga de un archivo que se ha descargado o interrumpido parcialmente.

¿Cómo habilitar el rango de bytes?

El rango de bytes ya está integrado en Apache, por lo que no es necesario "habilitarlo".

  • Si estás usando iTunes o tienes problemas de rango de bytes con archivos de audio/video/podcast, el problema es, en cambio, cómo se maneja el archivo dentro de ese software de terceros.
  • Si el archivo al que estás accediendo se encuentra en un sitio web de WordPress, el propietario o desarrollador del sitio web debe hacer ajustes al código ya que no hay nada que ajustar en el servidor.
