Generalidades

Este artículo explica cómo eliminar un dominio de la página Administrar Sitios Web.

Respaldar la información del dominio

Antes de eliminar tu dominio, asegúrate de hacer una copia de seguridad de todos los datos que desees conservar. Revosa los siguientes artículos para obtener instrucciones sobre los diferentes métodos para respaldar tu información:

  • Copias de Seguridad
  • Respaldar la información de un sitio web manualmente
  • Respaldar correo
  • Generalidades sobre el respaldo de las bases de datos

Subdominios

Eliminar el dominio principal

Si tu dominio principal (example.com) tiene un subdominio como por ejemplo blog.example.com, ambos dominios se eliminarán.

No elimines el dominio principal si aún deseas alojar el subdominio. En vez de eso, cámbialo a Solo DNS.

Eliminar un subdominio

El proceso para eliminar un subdominio (blog.example.com) es el mismo que para los dominios de primer nivel (example.com).

Eliminar un subdominio no elimina el dominio principal. Solo se eliminará el subdominio del panel.

¿Que información se elimina?

La siguiente información se remueve cuando se elimina un dominio.

  • El sitio web en el panel
  • Todas las direcciones de correo y su contenido
  • El hostname de la base de datos

Qué no se elimina

La siguiente información no se remueve. Deberás eliminarla por separado en tu panel.

  • Las bases de datos y los archivos del sitio
  • El usuario del sitio web en el servidor
  • Tu plan de alojamiento
  • El registro de dominio

Eliminar un dominio

Eliminar un dominio no se puede deshacer. Asegúrate de estar seguro de eliminar el dominio antes de continuar.

  1. Navega a la página Administrar Sitios Web.
  2. Da clic en el botón Administrar para abrir la página Configuraciones de Dominios, la cual te permite ajustar múltiples configuraciones de tu sitio.
    • Si estás en la Vista de Malla, da clic en el botón Administrar en la parte inferior de la caja del sitio.
    • Si estás en la Vista de Lista, da clic en el botón Administrar a la derecha de tu dominio.
  3. Cuando la página Administrar se abra, selecciona la pestaña Alojamiento.
  4. Desplazate hacia abajo y da clic en la sección Eliminar Permantemente el Dominio.
  5. Haz clic en el botón Continuar con la Eliminación para finalizar el proceso.
