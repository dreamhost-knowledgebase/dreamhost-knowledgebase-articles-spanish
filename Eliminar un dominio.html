Generalidades

Asegúrate de que deseas eliminar un dominio. Eliminar un dominio no se puede deshacer.

Lo siguiente describe cómo eliminar un dominio usando el panel Sitios Web.

Eliminar un dominio

  1. Navega a la página Administrar Sitios Web.
  2. Haz clic en el botón Administrar para el dominio que quieres eliminar.
  3. Cuando la página Admnisrar se abra, selecciona la pestaña Hosting.
  4. Desplazate hacia abajo y selecciona la sección Permanently Delete Domain.
  5. Haz clic en el botón Proceed with Deletion para finalizar el proceso.
