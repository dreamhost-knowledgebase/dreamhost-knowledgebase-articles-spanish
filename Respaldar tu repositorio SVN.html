Generalidades

Este artículo explica cómo crear una copia de seguridad de tu repositorio SVN. Se asume lo siguiente para todos los ejemplos de comandos:

  • Tu repositorio se llama SVN_Test
  • El repositorio se encuentra en tu directorio /home/username/svn/

Crear un archivo volcado de tu repositorio

  1. Inicia sesión en tu servidor vía SSH.
  2. Navega hasta el directorio en el que deseas crear la copia de seguridad. Este artículo la crea en tu directorio home de tu usuario. 
    [server]$ cd ~
  3. Corre lo siguiente para crear un archivo volcado de tu repositorio SVN. Este comando también lo comprime y te permite darle un nuevo nombre al archivo de respaldo. 
    [server]$ svnadmin dump ~/svn/SVN_Test | gzip -9 > SVN_Test_backup.dump.gz
  4. Corre lo siguiente para ver la copia de seguridad en el archivo .gz en el directorio home de tu usuario. 
    [server]$ ls -la ~ | grep .gz
SVN_Test_backup.dump.gz

Ahora puedes cargar el repositorio copiado en el nuevo servidor.
