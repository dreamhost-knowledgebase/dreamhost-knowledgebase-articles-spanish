Generalidades

Este artículo es una descripción general de las opciones disponibles para configurar tu dominio en el panel. Para obtener una descripción general de las opciones de alojamiento de dominios, consulta el artículo del Menú Domains.

Ver las opciones de configuración del dominio

Para ver las opciones disponibles para configurar tu dominio, navega hasta la página Manage Domains.

Hacia el lado derecho de tu dominio, haz clic en el botón Edit. Esto abre la página de configuración de tu dominio. A continuación se enumeran todas las secciones y subsecciones junto con una explicación de cada una.

Fully Hosted

Esta sección contiene opciones para sitios que están totalmente alojados en un servidor DreamHost.

Domain name

Esta opción te permite controlar cómo aparece la URL de tu sitio cuando lo visitas. Para obtener más información, revisa el artículo Agregar o quitar el "www" de la URL de tu dominio.

Users, Files, and Paths

  • En esta sección es donde asignas un usuario FTP/SFTP/Shell para acceder al directorio web del dominio.

Run this domain under the user (manejar este dominio bajo el usuario)

El menú desplegable te permite seleccionar un usuario existente o crear un nuevo usuario para asignar al dominio. Todos los usuarios se encuentran en la página Manage Users.

Web directory

Este campo muestra el directorio en el servidor donde se almacenan tus archivos. De forma predeterminada, está configurado en una carpeta con el mismo nombre que el nombre de tu dominio. En el ejemplo anterior, el nombre del sitio web es example.com.

Logs directory

Esto muestra la ruta en el servidor web a los registros de tu sitio. Puedes acceder a ellos si tu usuario es un usuario de SFTP o shell.

Web Options

PHP Mode

Este menú desplegable te permite elegir qué versión de PHP ejecutar en tu sitio.

Automatically upgrade PHP

Si esta opción está marcada, estás permitiendo que DreamHost actualice la versión de PHP en tu dominio a una versión recomendada. Si estás ejecutando una versión desactualizada, DreamHost la actualiza automáticamente después de notificártelo.

Extra Web Security

Al seleccionar Extra Web Security (Seguridad web adicional), habilitas el uso de un módulo de seguridad especial para tu sitio web. Se recomienda que mantengas esta opción marcada.

Para obtener más información sobre esta opción, visita mod_security.

PHP OPcache support

Esta opción solo está disponible para sitios que se alojan en un VPS. Esto habilita OPcache lo que mejora las aplicaciones PHP al almacenar en caché el código PHP compilado en la memoria.

Passenger (Ruby/NodeJS/Python apps only)

La habilitación de esta función está destinada principalmente a ejecutar programas basados en Ruby on Rails o Ruby/Python. Si no necesitas esta función, déjala deshabilitada. Para obtener más información, consulta el artículo sobre Passenger.

Cloudflare Services

Cloudflare proporciona varias herramientas que ayudan a acelerar y optimizar tu sitio web. Si habilitas esta función, debes crear una cuenta o ingresar tus credenciales de cuenta de Cloudflare existentes para configurarla en tu dominio. Para obtener más información, consulta el artículo de Cloudflare.

Redirect

Esta sección te permite redirigir tu dominio actual a cualquier otro dominio que desees. Para obtener más información, consulta el artículo Redirigir un dominio.

Mirrored

Un espejo te permite cargar tu contenido a tu servidor para un sitio (sitio A), pero usa la URL de otro sitio (sitio B) en la misma cuenta para ver el sitio original (sitio A). Para obtener más información, revisa el artículo de Reflejar un dominio.

Parked

La sección Parked no tiene configuraciones. Simplemente haces clic en el botón Park this domain para habilitarlo, lo que establece la opción de alojamiento web en “estacionado”. Una vez que un dominio está configurado como Estacionado, DreamHost muestra un sitio web de marcador de posición con un mensaje genérico de DreamHost 'próximamente'. Consulta el siguiente artículo para más detalles:

  • Dominio estacionado

DNS Only

La opción DNS Only (sólo DNS) proporciona el DNS básico para el dominio sin configurar el sitio en el servidor web que lo alojará. Si deseas modificar su configuración, por favor revisa los artículos de DNS y DNS personalizados.
