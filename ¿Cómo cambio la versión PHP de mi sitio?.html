Generalidades

DreamHost ofrece varias versiones de PHP. Cuando agrega un sitio, hay una opción para elegir qué versión le gustaría usar, pero también puede cambiar esta versión en cualquier momento en el panel.

No es posible cambiar la versión PHP de los sitios de DreamPress en el panel, ya que es un servicio administrado por DreamHost. Si necesitas cambiar la versión PHP de un sitio DreamPress, comunícate con el soporte para obtener ayuda.

Cambiando la versión de PHP

Para cambiar la versión de PHP en cualquiera de sus dominios / subdominios dentro del panel:

  1. Navegar a la página Manage Domains.
  2. A la derecha de su dominio, haga clic en el botón Edit en la columna 'Web Hosting'.
    Aparece la página 'Manage Domains':
  3. En la sección 'Opciones web'(Web Options), seleccione qué versión de PHP del menú desplegable desea que se ejecute su sitio.
  4. Puede marcar o desmarcar Actualizar automáticamente PHP (PHP Priority Upgrade). Esta opción solo actualiza PHP si la versión que está ejecutando está desactualizada.
  5. Desplácese hacia abajo y haga clic en el botón Change settings para guardar los cambios.

Cambiar la versión de PHP en su sitio usando un archivo .htaccess

Además de cambiar la configuración en el panel de DreamHost, también puede cambiar la configuración de PHP usando un archivo .htaccess.

View the following articles for instructions on how to create or edit your .htaccess file:

  • Creando y editando un archivo vía FTP
  • Creación y edición de un archivo a través de SSH

Simplemente agregue las siguientes dos líneas en su archivo .htaccess que obliga a PHP a usar PHP 7.2:

AddHandler fcgid-script .php 
FCGIWrapper "/dh/cgi-system/php72.cgi" .php

Si esto causa algún problema, elimine las líneas para volver a la configuración anterior.
