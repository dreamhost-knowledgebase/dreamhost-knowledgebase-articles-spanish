Generalidades

Este artículo describe las diversas formas en que puedes cambiar tu plan desde un plan de Alojamiento Dedicado.

Cambiar el plan a Shared Starter

Si deseas cambiar el plan de un Servidor Dedicado a un plan de alojamiento Shared Starter, asegúrate de cumplir con las limitaciones descritas en el siguiente artículo:

  • Agregar un plan de alojamiento Shared Starter

Si quieres proceder, Contacta a Soporte para solicitar esto.

Cambiar el plan a compartido o VPS

Primero, asegúrate de haber agregado un plan de alojamiento compartido o VPS.

  • Agregar un plan de Shared Hosting
  • Agregar un servidor privado (VPS)

Luego puedes eliminar el Servidor Dedicado:

  • Eliminar un Servidor Dedicado

Al eliminar, verás un mensaje que te informa que todo en el Servidor Dedicado se migrará al servidor compartido o VPS.

Si tienes un servidor compartido y un VPS en tu cuenta, solo el servidor VPS se mostrará como una opción para migrar al eliminar tu Servidor Dedicado.

Si prefieres migrar a un servidor compartido, contacta a soporte. El soporte de DreamHost migrará los datos por tí.

Migrar a DreamPress

Consulta el siguiente artículo para obtener instrucciones sobre cómo migrar un sitio de WordPress existente a DreamPress:

  • ¿Cómo actualizar una página WordPress a DreamPress?

Si solo tenías el único sitio de WordPress en tu Servidor Dedicado, puedes cancelarlo.

  • Eliminar un Servidor Dedicado

Si tienes otros sitios en el Servidor Dedicado, debes continuar pagando ese plan, ya que solo el sitio único se habría migrado a DreamPress.

Migrar a un plan de Alojamiento Dedicado diferente

Contacta a soporte si deseas migrar datos de un Servidor Dedicado a un Servidor Dedicado diferente. El soporte luego migrará los datos por tí.
