Generalidades

Este artículo describe las diversas formas en que puedes cambiar o degradar tu plan desde un plan de alojamiento dedicado.

Cambiar el plan a Starter Shared

Si deseas cambiar el plan de un servidor dedicado a un plan de alojamiento Starter Shared, asegúrate de cumplir con las limitaciones descritas en el siguiente artículo:

  • Añadir un plan de alojamiento Starter Shared

Si quieres proceder, Contact Support para solicitar esto.

Cambiar el plan a Shared o VPS

Primero, asegúrate de haber agregado un plan de alojamiento Shared o VPS.

  • Agregar un plan de alojamiento Shared
  • Agregar un servidor privado (VPS)

Luego puedes eliminar el servidor dedicado:

  • Eliminar un servidor Dedicado

Al eliminar, verás un mensaje que te informa que todo en el servidor dedicado se migrará al servidor Shared o VPS.

Si tienes ambos un servidor Shared y un VPS en tu cuenta, solo el servidor VPS se mostrará como una opción para migrar al eliminar tu servidor dedicado.

Si prefieres migrar a un servidor compartido, contacta soporte. El soporte de DreamHost migrará los datos por tí.

Migrando a DreamPress

Consulta el siguiente artículo para obtener instrucciones sobre cómo migrar un sitio de WordPress existente a DreamPress:

  • ¿Cómo mejorar una página WordPress a DreamPress?

Si solo tenías el único sitio de WordPress en tu servidor dedicado, puedes cancelarlo.

  • Eliminar un servidor Dedicado

Si tienes otros sitios en el servidor dedicado, debes continuar pagando ese plan, ya que solo el sitio único se habría migrado a DreamPress.

Migrar a un plan dedicado diferente

Contacta soporte si deseas migrar datos de un servidor dedicado a un servidor dedicado diferente. El soporte luego migrará los datos por tí.
