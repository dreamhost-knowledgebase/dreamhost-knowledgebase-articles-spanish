Generalidades

La página DNS proporciona una interfaz gráfica que te permite realizar cambios importantes en la configuración de tu dominio. Puedes usar la página DNS para verificar tus registros actuales, ver el estado de propagación, cambiar sus servidores de nombres y más.

Asegúrate de entender cómo funciona el DNS antes de intentar hacer cambios en tu configuración de DNS. Consulta el siguiente artículo para obtener más información sobre cómo ver y realizar cambios en tu DNS en DreamHost:

  • Descripción general del DNS de DreamHost

Registros

Las siguientes opciones aparecen en la sección Registros, que se describe a continuación:

Estado de propagación del dominio

El indicador te muestra si el dominio se propaga. Aparecen las siguientes opciones:

  • Propagado (verde) — Los registros DNS de tu sitio se han actualizado en todo el mundo.
  • Verificando (naranja) — DreamHost está comprobando si los registros DNS de tu dominio se han propagado por todo el mundo.

Vé el siguiente artículo para más información:

  • Descripción general de la propagación de DNS

DNS Checker

El Comprobador de propagación de DNS te permite seleccionar varios registros y luego verificar instantáneamente la dirección IP actual de un sitio web y la información del registro de DNS con múltiples servidores de nombres ubicados en diferentes partes del mundo. Aparece un mapa interactivo y una lista de citas, que te muestran el estado actual de la búsqueda de tu dominio:

Refresh DNS

Haz clic en el botón Refresh DNS para actualizar tu configuración de DNS. Aparece un indicador que indica el progreso.

Add Record

Haz clic en el botón Add Record para realizar cambios en tus registros DNS para un dominio. Las siguientes opciones están disponibles:

  • A
  • AAAA
  • CNAME
  • MX
  • TXT
  • SRV (avanzado)
  • NS (avanzado)
