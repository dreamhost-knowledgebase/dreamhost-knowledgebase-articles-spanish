Generalidades

Si deseas acceder al servidor para realizar configuraciones antes de mover tu dominio, puedes usar la siguiente forma de nombre de host en tu cliente FTP o SSH.

  • psXXXXX.dreamhostps.com

donde psXXXXX es el nombre de host de tu VPS.

Usando un navegador web

Si ingresas ftp://psXXXXX.dreamhostps.com en un navegador web, funcionaría; pero http://psXXXXX.dreamhostps.com no funciona. Esto se debe a que las solicitudes http y FTP se manejan de manera diferente: para las solicitudes http, el servidor comparte direcciones IP entre dominios, mientras que para las solicitudes FTP, el servidor solo necesita el nombre de usuario/contraseña FTP para saber qué contenido está dentro del alcance de su respuesta.
