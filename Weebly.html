Generalidades

Weebly es un servicio de alojamiento web con un creador de sitios web de arrastrar y soltar.

Puedes leer más sobre esto en Weebly.com.

Apuntar tu dominio a Weebly

Si tu dominio está registrado en DreamHost, puedes apuntarlo a Weebly usando registros 'A'. Antes de empezar, por favor revisa como mover el alojamiento a otra compañía (mira la Opción #2 en el siguiente artículo):

  • Transferir el hosting web a otra compañía

Usa los siguientes pasos para apuntar tu dominio a Weebly.

  1. Asegúrate de que tu dominio esté configurado solo para DNS. Visita los siguientes artículos para más detalles:
    • Configurar tu dominio como 'Solo DNS'
  2. Una vez que esté configurado solo para DNS, revisa el siguiente artículo para aprender cómo agregar registros 'A' personalizados usando los registros DNS de Weebly:
    • Agregar un registro 'A' personalizado
  3. Crea DOS registros 'A' (usando ambos hostnames en blanco y uno 'www'). Usa el botón Añadir Más + para agregar más registros:
    • Registro #1: Un registro 'A' con un hostname en blanco.
    • Registro #2: Un 'A' registro con el hostname 'www'.
    • Registro #1 y #2: Contacta a Weebly para obtener la dirección IP correcta y luego introdúcela en el cuadro de campo 'Apunta a'.

Una vez que se guardan esos registros, la actualización en línea demora 6 horas o más. Puedes verificar el estado de los cambios de DNS usando cualquiera de los siguientes enlaces:

  • whatsmydns.net — Un sitio popular que verifica los registros DNS en todo el mundo.
  • Ver tus registros DNS de DreamHost — Usa el verificador de propagación de DNS de DreamHost en el panel.

Cuando que los DNS se hayan actualizado, tu sitio estará listo para ser alojado en Weebly.
