Generalidades

Weebly es un servicio de alojamiento web con un creador de sitios web de arrastrar y soltar.

Puedes leer más sobre esto en Weebly.com.

Apuntar tu dominio a Weebly

Si tu dominio está registrado en DreamHost, puedes apuntarlo a Weebly utilizando registros A:

  1. Asegúrate de que tu dominio esté configurado solo para DNS. Visita los siguientes artículos para más detalles:
    • DNS Personalizados
  2. Una vez que esté configurado solo para DNS, agrega los registros DNS de Weebly. Deberás agregar dos registros A. El siguiente artículo explica cómo agregar estos registros:
    • Transferir el alojamiento del sitio web a otra compañía
  3. Agrega el registro A pero deja el nombre de host en blanco.
  4. Agrega el mismo registro A con el nombre de host 'www'.

Una vez que se guardan esos registros, la actualización en línea demora 6 horas o más. Puedes verificar el estado de los cambios de DNS en el siguiente sitio:

  • https://www.whatsmydns.net/

Una vez que los DNS se hayan actualizado, tu sitio estará listo para ser alojado en Weebly.
