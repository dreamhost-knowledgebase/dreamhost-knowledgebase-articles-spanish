Weebly is a web-hosting service featuring a drag-and-drop website builder.

You can read more about it at Weebly.com.

Pointing your domain to Weebly

If your domain is registered at DreamHost, you can point it to Weebly using A records:

  1. Make sure your domain is set to DNS only. Visit the following articles for further details:
    • Custom DNS
  2. Once it's set to DNS only, add Weebly's DNS records. You'll need to add two A records. The following article explains how to add these records:
    • Transferring website hosting to another company
  3. Add the A record with a blank hostname.
  4. Add the same A record with the 'www' hostname.

Once those records are saved, it make take 6 hrs or more to update online. You can check the status of your DNS changes on the following site:

  • https://www.whatsmydns.net/

Once the DNS has updated, your site is ready to be hosted at Weebly.
