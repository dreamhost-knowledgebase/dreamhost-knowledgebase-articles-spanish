Generalidades

DreamHost no admite direcciones de correo electrónico generales. Debido a que este tipo de direcciones no rechaza ningún mensaje, los dominios con un alcance general atraen un volumen abrumador de spam. Esto degrada el rendimiento del sistema de correo para todos los clientes y retrasa la entrega de mensajes legítimos.

¿Cómo puedo crear un alias en mi dirección de correo electrónico?

Las cuentas de correo electrónico DreamHost permiten el uso de alias de direcciones de correo electrónico ‘desechables’ o ‘sobre la marcha’, donde puedes generar instantáneamente una dirección de correo electrónico única basada en un buzón existente.

Este tipo de alias son útiles para rastrear si un sitio web en particular ha compartido o filtrado tu información de contacto, o para clasificar proactivamente las suscripciones de correo electrónico en varios sitios web.

Para crear un nuevo alias sobre la marcha, agrega un símbolo más (+) seguido de cualquier combinación de letras y dígitos después de tu nombre de usuario de correo electrónico, antes de la @. Por ejemplo:

Correo electrónico principal: jim@example.com
Nuevo alias: jim+somecoolshop22@example.com
Otro alias nuevo: jim+onlinebanking@example.com

El correo enviado a cualquiera de estas direcciones se enviará a jim@example.com.

No hay límite para la cantidad de alias que puedes crear de esta manera, y no requieren configuración previa. Esta función ya está habilitada en todas las cuentas de correo electrónico de DreamHost.
