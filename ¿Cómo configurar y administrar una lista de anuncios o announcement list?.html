Configurando la lista

  1. Naviega hasta la página Announcement Lists.
  2. Da click en el botón Add New Announcement List. La página 'Create a New Announcement List' se abrirá. Las siguientes capturas de pantalla se han dividido para mayor claridad:
  3. Ingresa lo siguiente:
    • From name: ingresa el nombre de tu lista o boletín aquí.
    • Email address on the list: ingress el correo electrónico que deseas crear como el correo electrónico de la lista. Además, asegúrate de que el menú desplegable a la derecha esté lleno con el dominio en el que deseas que se cree la lista.
    • Default Announcement Subject: este es un campo opcional donde puedes ingresar el asunto de su mensaje.
    • Confirmation Email Subject: cuando un usuario se suscribe a tu lista, se envía un correo electrónico de confirmación al usuario. Ingresa en este campo cualquier línea de asunto que te gustaría usar para este mensaje.
    • Confirmation Email Message: cuando un usuario se suscribe a tu lista, recibe un correo electrónico de confirmación con los detalles enumerados en el campo a la derecha. El script está configurado para completar automáticamente los detalles entre los guiones dobles (--) para esta lista en particular. Puedes editar el aviso como quieras. Tenga en cuenta que el código HTML ingresado aquí no se representa correctamente.
    • Unsubscribe Directions: este aviso aparece en la parte inferior de cada anuncio que envía en caso de que el usuario ya no desee recibir este correo electrónico. El código HTML ingresado aquí se procesa correctamente, sin embargo, debes asegurarse de marcar "¿Es esto HTML?" casilla de verificación en la página final al enviar.
    Los campos restantes son URL opcionales que puede crear para personalizar la experiencia del usuario. Ajústalos según sea necesario:
  4. Cuando termines, haz clic en el botón Create This List al final de la página.

Explicación de las variables en el cuadro 'Confirmation Email Message'

A continuación se enumeran las variables y de dónde se extraen:

Variable De donde es sacado
--their_email-- Este es el correo electrónico del cliente/usuario que ingresaste al editar la Lista de anuncios. Estos correos electrónicos se agregaron en la sección 'Correo electrónico para agregar'.
--listname-- El nombre de tu lista de anuncios. Cuando editas tu lista, este es el primer campo titulado 'List Name'.
--ip-- Esta es la dirección IP de la computadora que estás utilizando para enviar la Lista de anuncios.
--ref_url-- ¿Este predeterminado es https://panel.dreamhost.com/index.cgi?
--url-- Esta es una URL específica en la que el cliente/usuario hace clic para confirmar que desea unirse a la lista. Se parece a https://scripts.dreamhost.com/add_list.cgi?g=b8ccge71a6ae4767e70b2f8br77e02c2

Editar la lista

  1. Naviega hasta la página Announcement Lists.
  2. A la derecha de la lista, haz clic en el botón Edit.
    Las páginas de edición se abren permitiéndole hacer sus ajustes.
    Solo hay unos pocos campos que se pueden cambiar después de hacer clic en el botón 'Edit' para la lista en cuestión:
    • From Name
    • Default Announcement Subject
    • Confirmation Email Subject
    • Confirmation Email Message
    • Unsubscribe Directions
    • Remove Bad Addresses
    • Todos los campos de URL.

No puedes cambiar el nombre de la lista después de que se haya creado. Debes eliminarlo y crear una nueva lista con el nuevo nombre.

"Eliminar direcciones incorrectas"

Después de crear la lista y hacer clic en el botón Edit, verás una nueva opción titulada 'Eliminar direcciones incorrectas' donde puedes elegir eliminar las direcciones de correo electrónico de tu lista después de un número específico de rebotes.

También puedes establecerlo en 'Never' para que nunca se elimine ninguna dirección.
