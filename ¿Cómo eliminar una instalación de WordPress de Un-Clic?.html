Este artículo explica cómo eliminar WordPress instalado a través del instalador de Un-Clic en el panel.

¿Qué se elimina en el servidor?

Cuando eliminas una instalación de Un-Clic, se pueden eliminar varios archivos dependiendo de dónde se encuentre la instalación. Asegúrate de leer los ejemplos a continuación antes de eliminar la instalación de un clic.

Si tu instalación está en el directorio web principal

El directorio web principal de tu sitio es el directorio que el servidor busca para servir archivos. Mira el artículo ¿Cómo agregar un dominio?. El paso 4 muestra dónde se encuentra el directorio web. También puedes visitar la página Instalaciones de Un-Clic

Haz clic en el botón Administrar mis aplicaciones.

Luego haz clic en el nombre de tu sitio web. Mostrará una barra diagonal al lado del ícono Un-Clic.

En general, este directorio es el nombre de tu sitio web. Por ejemplo, si tu sitio se llama example.com, tu directorio web también sería /home/username/example.com. Esto significa que cuando visitas tu sitio en example.com, estás viendo la Instalación de Un-Clic.

Si aquí es donde instalaste tu aplicación de Un-Clic, ten mucho cuidado al eliminarlo en el panel. Si eliminas la aplicación de Un-Clic almacenada en tu directorio web principal, TODOS LOS ARCHIVOS en el directorio web y todos los subdirectorios se eliminan. Esto significa que si tienes subdirectorios como example.com/images o example.com/blog, también se eliminan.

Asegúrate de usar esta opción solo si deseas que todo se elimine de tu sitio web y servidor, incluidos todos los demás archivos de este directorio web. Si necesitas alguno de esos archivos o subdirectorios, asegúrate de hacer una copia de seguridad antes de eliminar la aplicación de Un-Clic.

  • Hacer una copia de seguridad manual de tu información
  • Hacer una copia de seguridad completa de tu cuenta

Si tu instalación está en un subdirectorio

Si tu instalación está en un subdirectorio como example.com/blog, solo se eliminan los archivos dentro de ese subdirectorio.

Por ejemplo, si tu sitio era example.com/blog, solo se eliminarán los archivos dentro de ese subdirectorio /blog. Sin embargo, el directorio vacío /blog permanecerá. No se tocan archivos encima de este directorio, por lo que queda todo dentro del directorio /example.com.

Eliminar una aplicación de instalación de Un-Clic

  1. Navega a la página Instalaciones de Un-Clic.
    Se abre la página Instalaciones de un clic:
  2. Haz clic en el botón Administrar mis aplicaciones.
  3. Haz clic en el dominio que está usando la aplicación para expandirlo.
    Se muestran bajo el dominio todas las aplicaciones instaladas de un clic.
  4. Haz clic en el botón Borrar todos los archivos que se encuentra a la derecha.
    Aparece un mensaje de advertencia después de hacer clic en Borrar todos los archivos:
  5. Haz clic en Ok para confirmar y espera que la desinstalación se complete. Un mensaje que dice 'Éxito' se mostrará cuando la aplicación sea eliminada:
    • Después de desinstalar la aplicación, ya no se muestra en la página Administrar Aplicaciones.
    • Si aún aparece en la lista, haz clic en el botón Quitar de la lista.

¿Cómo eliminar la base de datos?

Cuando haces clic en el botón Borrar todos los archivos, solo los archivos en el servidor son eliminados. Tu base de datos de WordPress sigue en tu panel. Para eliminar tu base de datos, navega a la página Bases de Datos MySQL. Hacia la derecha de la respectiva base de datos, haz clic en el botón Eliminar BD.

¿Qué sucede si no se muestra el botón Delete?

Si encuentras que el botón Eliminar no se muestra en tu panel junto a la Instalación de Un-Clic, contacta a soporte para obtener más ayuda.
