Generalidades

BoldGrid es un creador de sitios de WordPress fácil de usar. Utiliza un editor de arrastrar y soltar, que te permite crear y personalizar rápidamente un sitio web profesional sin necesidad de codificación.

BoldGrid (también llamado 'WP Website Builder') viene preconfigurado con una lista de temas optimizados para dispositivos móviles para que puedas elegir según las necesidades de tus sitios. También hay disponibles herramientas adicionales como copias de seguridad para garantizar que tus datos estén seguros.

Habilitar BoldGrid en tu sitio web

Existen tres opciones para habilitar BoldGrid en un sitio WordPress.

Opción 1 — Regístrate en BoldGrid cuando abras una cuenta de DreamHost

Mira el siguiente artículo para abrir una nueva cuenta DreamHost.

  • Cómo registrarse para obtener una cuenta DreamHost 

Asegúrate de marcar la casilla titulada WP Website Builder. Esto instala BoldGrid dentro de tu instalación de WordPress.

Opción 2 — Habilitar BoldGrid en una instalación de WordPress existente

Si no instalaste Bolgrid originalmente cuando te registraste en el plan de WordPress de DreamHost, aún puedes habilitarlo en el panel.

Deshabilitar otros editores

BoldGrid no debe usarse si otro creador de sitios web, como Elementor, ya está en uso. Si tu sitio fue construido con un creador de sitios web de WordPress y deseas cambiar a otro, primero debes usar un sitio Staging en lugar de instalar ambos constructores en el sitio en vivo.

  1. Navega a la página Instalar WordPress.
  2. Da clic en el botón Gestionar mis Aplicaciones.
  3. Da clic en tu dominio para expandir sus configuraciones.
  4. Debajo de la columna Acciones, da clic en el botón Instalar Constructor de Sitios WP.
    Se abre un cuadro emergente de confirmación que solicita confirmar la instalación.
  5. Haz clic en el botón Instalar para confirmar. WP Website Builder estará disponible en tu instalación de WordPress en unos minutos.

Opción 3 — Nuevas instalaciones de WordPress

Esta opción instala BoldGrid cuando instalas WordPress en el panel. Revisa el artículo instalador de WordPress de DreamHost para ver las instrucciones completas.

Luego de dar clic en Instalar WordPress, la siguiente página te permite seleccionar un propósito de sitio para guiar la instalación de WordPress.

Para instalar BoldGrid:

  1. Escoge la opción Otro.
  2. A la derecha, da clic en el botón Siguiente.
  3. Cuando la página de los plugins aparezca, asegúrate de que WP Website Builder esté seleccionado.
  4. Da clic a Instalar para proceder con la instalación de un nuevo sitio WordPress con BoldGrid.

Administrar tu sitio BoldGrid

Llave Premium

Luego de habilitar BoldGrid usando las opciones número 2 o 3 de arriba, una BoldGrid Premium Connect Key se instala.

Para confirmar que una Premium Connect Key fue instalada, inicia sesión en tu sitio WordPress y navega a la pestaña Escritorio en el panel izquierdo. En la parte derecha, debajo de BoldGrid Notifications, busca la BoldGrid Connect Key. Si no hay una llave instalada, contacta al soporte de DreamHost.

Restaurar la contraseña de WordPress de tu sitio web

Después de instalar WordPress en tu sitio web, se te envía por correo un enlace para restablecer tu contraseña. Haz clic en este enlace para aceptar la contraseña actual (y anótala) o ingresa una nueva contraseña para tu inicio de sesión de WordPress. Revisa el siguiente artículo para obtener más información:
