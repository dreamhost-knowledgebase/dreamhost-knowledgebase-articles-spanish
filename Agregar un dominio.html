Generalidades

Para que un sitio web sea visible en internet, necesitas dos cosas:

  • Comprar el registro de un dominio.
  • Agregar alojamiento a un dominio.

Este artículo asume que ya has comprado el registro del nombre de dominio que deseas usar.

¿Qué es un dominio Completamente Alojado?

Cuando un dominio está completamente alojado quiere decir que está configurado en un servidor y tiene los Registros DNS asignados. Una vez los registros DNS han sido asignados, puedes entonces visitar y ver el sitio en internet. Las instrucciones para agregar un dominio a tu cuenta como 'completamente alojado' se muestran debajo. Esto requiere algunos pasos:

  1. Comprar el registro del dominio.
  2. Comprar el plan de alojamiento.
  3. Seguir las instrucciones en este artículo para agregar tu dominio al plan de alojamiento.

Apuntar los DNS a DreamHost

Tu sitio puede ya encontrarse alojado en los servidores de DreamHost, pero si los DNS de tu sitio no apuntan aún a DreamHost, el sitio no resolverá en línea. Revisa las siguientes secciones dependiendo de dónde compraste el registro del dominio.

Si compraste el registro con DreamHost

Si compraste tu registro del dominio en DreamHost, los DNS ya se encuentran configurados correctamente. (A menos de que tu los personalices).

Si compraste el registro con otra compañía

Necesitas apuntar los DNS a DreamHost. Revisa el siguiente artículo para obtener los detalles:

Mensajes de error al agregar un dominio

En algunos casos, aparece un mensaje de error cuando intentas agregar alojamiento para un dominio en tu cuenta. Los mensajes de error más comunes se describen a continuación.

Debes agregar el dominio principal

Si ves:

You can't add that domain: you must add the parent domain example.com to your account before adding a subdomain to it.

Esto significa que el dominio de segundo nivel aún no está en el sistema. Un subdominio solo se puede crear si Tu dominio de segundo nivel se crea primero (example.com)— incluso si no planeas alojar el dominio de segundo nivel aquí en DreamHost.

No tienes permiso para agregar ese dominio

Si ves:

You can't add that domain: you don't have permission to add.

Significa que el dominio que estás intentando agregar a su cuenta ya está alojado en otra cuenta DreamHost. Si estás moviendo el dominio de una cuenta existente a la que tiene acceso, recuerda eliminar primero el dominio de la cuenta anterior antes de agregarlo a una cuenta nueva. Si no sabes en qué cuenta está alojado actualmente el dominio, pónte en contacto con el equipo de soporte.

El nombre de dominio ya existe en el sistema.

El error "domain name already exists in the system" puede ocurrir en situaciones similares a "you don't have permission to add". Si ves este mensaje de error, confirma que el dominio no está alojado en otra cuenta DreamHost y luego pónte en contacto con el equipo de soporte.
