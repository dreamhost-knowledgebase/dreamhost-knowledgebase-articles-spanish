El módulo de API de DNS te permite administrar los registros del Servicio de nombres de dominio.

Comandos

dns-list_records

Vuelca una lista de todos los registros DNS para todos los dominios (sin incluir los registros) en todas las cuentas a las que tiene acceso. Ten en cuenta que esto omite las zonas dreamhosters.com, dreamhost.com, dreamhostps.com y newdream.net.

Comando dns-list_records
Valores (ninguno)
Resultado

dns-add_record

Agrega un nuevo registro DNS a un dominio que ya ha alojado con DreamHost. Sin embargo, no puedes agregar registros de dreamhosters.com. Ten en cuenta que los cambios de DNS pueden tardar un tiempo en realizarse.

Comando dns-add_record
Valores record : El nombre completo del registro que deseas agregar, ejemplo: testing.groo.com

type : A,CNAME,NS,PTR,NAPTR,SRV,TXT, oo AAAA value : El valor del registro DNS comment : Comentario opcional para este registro.
Resultado

success
record_added

dns-remove_record

Elimina un registro DNS editable existente que tiene con DreamHost. Sin embargo, no puedes eliminar los registros de dreamhosters.com. Ten en cuenta que los cambios de DNS pueden tardar un poco en propagarse.

Comando dns-remove_record
Valores

record : El nombre completo del registro que deseas eliminar, ejemplo: testing.groo.com
type : El tipo (mira dns-add_record) del registro que deseas eliminar.
value : El valor (mira dns-add_record) del registro que deseas eliminar.
Resultado

success
record_removed
Posibles errores

no_record
no_type
no_value
no_such_record
no_such_type
no_such_value
not_editable
internal_error_could_not_destroy_record
internal_error_could_not_update_zone
