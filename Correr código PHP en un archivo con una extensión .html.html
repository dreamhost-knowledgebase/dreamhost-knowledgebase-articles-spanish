Generalidades

Este artículo explica cómo utilizar un archivo .htaccess para ejecutar código PHP dentro de un archivo HTML.

Antecedentes

Normalmente, cuando se combinan PHP y HTML en el mismo archivo, el archivo utiliza una extensión .php para mostrar el contenido en el navegador. Por ejemplo, el siguiente archivo se llama hello.php:

<html
<body>

<?php
  echo "hello";
?>

</body>
</html>

Sin embargo, si necesitas colocar código PHP en un archivo con una extensión .html, primero debes agregar un controlador a tu archivo .htaccess para que el código PHP pueda funcionar.

Usar un carchivo .htaccess

Debes usar un archivo .htaccess para ejecutar el código PHP. Si este archivo no existe en el directorio principal de tu sitio web, revisa los siguientes artículos para obtener instrucciones sobre cómo crear el archivo mediante FTP o SSH:

  • Crear y editar archivos a través de FTP
  • Crear y editar archivos a través de SSH

Agrega estas líneas al archivo .htaccess.

Este código también cambia la versión de PHP que has configurado en el panel a la versión de la segunda línea.

AddHandler fcgid-script .html
FcgidWrapper "/dh/cgi-system/php82.cgi" .html

Cuando cargas el archivo .html en un navegador, ahora se carga como un archivo PHP normal.
