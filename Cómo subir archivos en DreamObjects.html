Generalidades

Hay una amplia variedad de formas de almacenar archivos en DreamObjects, principalmente según tus preferencias y habilidades.

Subiendo en el panel de DreamHost

Hay una interfaz gráfica en tu panel DreamHost que puedes usar para cargar directamente desde tu computadora. Consulta el siguiente artículo para obtener más detalles:

  • Subiendo un archivo desde el panel de DreamHost a DreamObjects

Usando un cliente compatible de S3 (Recommended)

Usar un cliente S3 es la mejor opción, ya que un cliente especializado te ofrecerá más funciones para ayudarte a administrar tus datos. Hay varias aplicaciones, desde utilidades de escritorio y de línea de comandos hasta complementos de WordPress y aplicaciones móviles, que te permiten almacenar archivos en DreamObjects. Ve la siguiente categoría para obtener una lista de clientes disponibles:

  • Clientes de DreamObjects

Usando la API REST

Los usuarios avanzados pueden utilizar la API REST y cualquiera de las bibliotecas que admiten
API S3.

  • API REST

Tamaños de Objetos

Los objetos están limitados a 10 TB, pero deben cargarse en fragmentos de 5 GB. Ten en cuenta que no todos los clientes pueden realizar cargas de partes múltiples. Para cualquier valor superior a 5 GB, debes utilizar un cliente que admita cargas de partes múltiples.

DreamObjects está diseñado para una durabilidad del 99,99999% durante un año determinado y tus datos se replican tres veces, con consistencia inmediata, en todo el centro de datos de DreamObjects, lo que garantiza una mayor seguridad de respaldo.

Tiempo de actividad de DreamObjects

DreamObjects es capaz de almacenar casi cualquier tipo de archivo, siempre que cumpla las condiciones de la Política de uso aceptable.
