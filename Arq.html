Generalidades

Arq Backup es una aplicación para Mac y Windows que te permite realizar copias de seguridad en línea compatibles con DreamObjects. Las copias de seguridad de Arq se cifran con una contraseña antes de que se carguen para que tus datos estén protegidos.

La instalación y configuración de Arq varía según tu sistema operativo. Las instrucciones a continuación muestran cómo configurarlo dentro de Windows.

Conectarte con Arq en Windows

  1. Lanza Arq.
    Al iniciar Arq por primera vez, se te solicitará que elijas el tipo de destino de la copia de seguridad:
  2. Haz clic en la opción 'Other S3-Compatible Service' y luego haz clic en el botón Continue.
  3. La siguiente información puede ser encontrada en la página DreamObjects en tu panel.
    • S3-Compatible Server URL: Ingresa el nombre de host https://objects-us-east-1.dream.io. Asegúrate de añadir https:// al frente de la URL.
    • Access Key ID: Encontrado en tu panel de DreamHost. Visita el artículo Llaves para más información.
    • Secret Access Key: Encontrado en tu panel de DreamHost. Visita el artículo de Llaves para más información.
    • Request Signature Version: DreamObjects soporta ambos versión de firma 2 y 4 (no hay versión 3).
  4. Cuando finalices, haz clic en el botón Add Destination para continuar.
    Arq te solicita que ingreses un nuevo nombre de bucket o selecciones un bucket:
  5. Crea un bucket (e ingresa un nombre) o selecciona un bucket existente, luego haz clic en el botón Add para finalizar.
    Una vez que tu bucket está configurado, Arq te solicita que configures una nueva programación de copias de seguridad o que restaures archivos de este bucket (lo que requiere copias de seguridad existentes en tu bucket):
  6. Haz clic en el botón Set Up Backups.
  7. Ingresa una contraseña encriptada que proporciona seguridad adicional para la copia de seguridad que crea Arq.
  8. Después de ingresar una contraseña encriptada, asegúrate de escribir la contraseña que creaste y guardarla en un lugar seguro.
  9. Se te informa que la copia de seguridad se realizará cada hora y puedes seleccionar la ubicación de la copia de seguridad.
  10. Elige las carpetas para respaldar.
    • En los sistemas Windows, Arq realiza de forma predeterminada una copia de seguridad de toda la unidad C: excluyendo solo la Papelera de reciclaje y el archivo pagefile.sys.
  11. Para añadir o remover archivos o carpetas, haz clic en el botón Edit backup suggestions, el cual te permite personalizar la selección.

Arq ahora está configurado para respaldar tus datos.

¡Comienza a almacenar tus datos hoy!
