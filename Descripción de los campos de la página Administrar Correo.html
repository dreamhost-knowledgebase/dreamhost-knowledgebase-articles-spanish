Generalidades

Visita la página Adminsitrar Correo en tu panel:

Crear Nueva Dirección de Correo: Se debe hacer clic en este botón para crear una nueva dirección de correo como dice.

Editar Direcciones de Solo-Reenvío Masivamente: Este botón es para la creación de una dirección de correo de "solo reenvío". Esto te permite agregar múltiples reenvíos muy rápidamente.

Mostrar estos dominios: Show these domains: Esta sección enumera todos tus dominios alojados con DreamHost. Puedes ver los usuarios de correo en ese dominio resaltando el nombre del dominio y luego haciendo clic en el botón Mostrar los Dominios Seleccionados. (También puedes seleccionar varios dominios para ver manteniendo presionado el botón "Control" y haciendo clic en los dominios que deseas ver).

Las siguientes son las tres columnas que enumeran información para cada dirección de correo:

Buzón: Los detalles enumerados en esta columna incluyen lo siguiente:

  • El nombre del buzón.
  • Enlace al interfaz de Adminstrador de Buzones para ese usuario.
  • Detalles del tamaño del buzón para esa dirección junto con un porcentaje de cuán lleno está actualmente el buzón.

El panel se actualiza cada 24 horas. Si recientemente eliminaste una gran cantidad de correo, notarás que el tamaño no ha cambiado; El nuevo tamaño del buzón puede tardar hasta 24 horas en mostrarse correctamente.

También reenvía a: Este campo muestra los alias de reenvío que configuraste para la dirección de correo que estás viendo.

En el pasado, era posible crear un correo totalmente alojado que también se reenviaba a otra bandeja de entrada. Esta característica ya no es compatible, y las nuevas direcciones de correo creadas no tendrán esta opción. Las direcciones de correo existentes que usan esta función no se verán afectadas hasta que se editen, momento en el que deberás seleccionar entre reenviar solo o totalmente alojado.

Si no hay alias de reenvío configurados para la dirección de correo, mostrará este mensaje:

“There are no additional forwards; messages are only delivered to this mailbox.”

Acciones: Esta columna consta de tres botones para cada dirección de correo : Editar, Dehabiliatar, y Eliminar.

  • Editar: Este botón Editar te lleva a otra página donde puede realizar cambios en tu dirección de correo, como aumentar el tamaño del buzón.
  • Deshabilitar: Este botón deshabilita la dirección de correo dentro de una hora. Al hacer clic en este botón, cambia lo que ves para ese usuario en la página Adminstrar Correo marcando una línea con el nombre y el tamaño del buzón del usuario, y configurando un nuevo mensaje en la columna También Reenvía a. El nuevo mensaje leerá:  
    Deleted with bounce.

    (Lo que esto significa es que cualquier correo enviado a este usuario deshabilitado será eliminado y rechazado de nuevo al remitente con un error de devolución).

    Una vez configurado como deshabilitado, una dirección de correo no puede iniciar sesión ni utilizar para enviar/recibir correo.
  • Eliminar: Al hacer clic en este botón, se elimina el usuario, pero aparece una ventana emergente que te solicita que verifique esta solicitud primero con este mensaje: 
    REALLY DELETE user@example.com AND ALL EMAIL IN ITS MAILBOX PERMANENTLY?!!

    Al hacer clic en "OK" se elimina el usuario y el buzón de ese usuario. Esto requiere unos minutos, pero una vez hecho esto, ya no verás a ese usuario en la lista de la página Administrar Correo para ese dominio. Esto es diferente a cuando deshabilita al usuario, que aún mantendrá el nombre de usuario en la página Administrar Correo en caso de que desees configurarlo como una dirección de correo alojada nuevamente en el futuro.

Para encontrar rápidamente tu configuración de correo en el panel, abre la página Administrar Correo y busca el enlace en la esquina superior derecha. Haz clic en el texto para abrir las instrucciones de acceso rápido para tu dirección de correo:

Entrante
  • imap.dreamhost.com
  • pop.dreamhost.com
Saliente
  • smtp.dreamhost.com
