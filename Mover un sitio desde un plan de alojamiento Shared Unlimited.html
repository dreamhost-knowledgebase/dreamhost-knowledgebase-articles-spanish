Generalidades

Este artículo describe las diferentes formas en que puedes mejorar o cambiar desde un plan de Alojamiento Shared Unlimited.

Cambiar tu plan a Shared Starter

Si deseas cambiar de un plan de Alojamiento Shared Unlimited a un plan de Alojamiento Shared Starter, asegúrate de cumplir con las limitaciones mencionadas en el siguiente artículo:

  • Agregar un plan de Alojamiento Shared Starter

Si quieres continuar, por favor Contacta a Soporte para solicitarlo.

Mejorar a un alojamiento VPS

Revisa el siguiente artículo para obtener instrucciones sobre cómo agregar un nuevo plan de alojamiento VPS:

  • Agregar un VPS

Al comprar, tienes la oportunidad de migrar TODOS tus usuarios y dominios al nuevo VPS. También puedes optar por no migrar ningún dato.

Si solo deseas pagar por un VPS y eliminar el Alojamiento Shared, marca el botón de opción para migrar todos los usuarios y dominios durante este proceso.

Después de que todos tus sitios se hayan migrado a tu nuevo VPS y se resuelvan correctamente, debes cancelar manualmente tu plan de Alojamiento Shared.

  • Cancelar el Alojamiento Shared si también tienes otro tipo de plan de alojamiento

Mejorar a un Servidor Dedicado

Revisar el siguiente artículo para obtener instrucciones sobre cómo agregar un nuevo Servidor Dedicado:

  • Agregar un Servidor Dedicado

Mientras creas el Servidor Dedicado, puedes elegir migrar todos tus usuarios compartidos a la nueva máquina.

Mejorar a un DreamPress

Si ya tienes un sitio de WordPress en Alojamiento Shared, puedes usar el siguiente artículo para actualizarlo a DreamPress:

  • Actualizar un sitio de WordPress a DreamPress

También puedes agregar un nuevo dominio a DreamPress usando el siguiente artículo:

  • Agregar DreamPress a un dominio nuevo
