SWIG (el Simplified Wrapper and Interface Generator) es usado por algunos paquetes para generar fácilmente enlaces a lenguajes de programación (por ejemplo, Python).

Estas instrucciones son para instalar SWIG versión 3.0.5 desde la fuente.

Instalación

[server]$ mkdir ${HOME}/soft
[server]$ cd ${HOME}/soft

Descarga y extrae el paquete

[server]$ wget http://prdownloads.sourceforge.net/swig/swig-3.0.5.tar.gz
[server]$ tar xvzf swig-3.0.5.tar.gz
[server]$ cd swig-3.0.5

Suponiendo que has seguido las instrucciones de virtualenv, configura usando:

[server]$ ./configure --prefix=$HOME --with-perl5=/usr/local/bin/perl --with-python=$HOME/opt/python-2.7.7/bin/python  --with-ruby=/usr/bin/ruby

Compila y verifica

[server]$ make
[server]$ make check

Es posible que obtengas algunos errores sobre ocaml. Estos no importan (a menos que tengas la intención de utilizar esta versión de SWIG para generar enlaces ocaml también.

[server]$ make install
