Generalidades

setuptools es una colección de mejoras a las distutils de Python que te permiten construir y distribuir más fácilmente paquetes de Python, especialmente aquellos que tienen dependencias en otros paquetes.

Para ejecutar los comandos de este artículo, debe siniciar sesión en tu servidor a través de SSH con tu usuario de Shell. Ve los siguientes artículos para obtener más información:

  • Generalidades SSH
  • Crear un usuario con acceso Shell (SSH)

Puedes instalar fácilmente setuptools para que funcione con tu versión de Python. (Ten en cuenta que virtualenv configurará setuptools automaticamente.)

[server]$ mkdir ${HOME}/soft
[server]$ cd ${HOME}/soft
[server]$ wget http://peak.telecommunity.com/dist/ez_setup.py
[server]$ python ez_setup.py

Ahora tienes el script easy_install instalado en tu directorio ${RUN}/bin.
