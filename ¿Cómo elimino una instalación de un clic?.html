Generalidades

Este artículo proporciona detalles sobre cómo eliminar de forma rápida y segura una aplicación instalada a través del instalador One-Click en el panel.

¿Qué se elimina en el servidor?

Cuando eliminas una instalación con un clic, se pueden eliminar varios archivos dependiendo de dónde se encuentre la instalación. Asegúrate de leer los ejemplos a continuación antes de eliminar la instalación con un clic.

Si tu instalación está en el directorio web principal

El directorio web principal de tu sitio es el directorio que el servidor busca para servir archivos. Mira el artículo '¿Cómo agregar un dominio?'. El paso 4 muestra dónde se encuentra el directorio web. También puedes visitar la página One-Click Installs.

Haz clic en el botón Manage Installed Applications.

Luego haz clic en el nombre de tu sitio web. Mostrará una barra diagonal al lado del ícono One-Click.

En general, este directorio es el nombre de tu sitio web. Por ejemplo, si tu sitio se llama 'example.com', tu directorio web también sería '/home/username/example.com'. Esto significa que cuando visitas tu sitio en 'example.com', estás viendo la Instalación de un clic.

Si aquí es donde instalaste tu aplicación de un clic, ten mucho cuidado al eliminarlo en el panel. Si eliminas la aplicación de un clic almacenada en tu directorio web principal, TODOS LOS ARCHIVOS en el directorio web y todos los subdirectorios se eliminan. Esto significa que si tienes subdirectorios como 'example.com/images' o 'example.com/blog', también se eliminan.

Asegúrate de usar esta opción solo si deseas que todo se elimine de tu sitio web y servidor, incluidos todos los demás archivos de este directorio web. Si necesitas alguno de esos archivos o subdirectorios, asegúrate de hacer una copia de seguridad antes de eliminar la aplicación de un clic.

  • Haz una copia de seguridad manual de tu información
  • Haz una copia de seguridad completa de tu cuenta

Si tu instalación está en un subdirectorio

Si tu instalación está en un subdirectorio como 'example.com/blog, solo se eliminan los archivos dentro de ese subdirectorio.

Por ejemplo, si tu sitio era 'example.com/blog', solo se eliminarán los archivos dentro de ese subdirectorio /blog. Sin embargo, el directorio vacío /blog permanecerá. No se tocan archivos encima de este directorio, por lo que queda todo dentro del directorio /example.com.

Eliminar una aplicación de instalación con un clic

  1. Navega a la página One-Click Installs.
    Se abre la página One-Click Installs:
  2. Haz clic en el botón Manage Installed Applications.
  3. Haz clic en el dominio que está usando la aplicación para expandirlo.
    Se muestran bajo el dominio todas las aplicaciones instaladas de 1 clic.
  4. Haz clic en el botón Delete all Files que se encuentra a la derecha.
    Aparece un mensaje de advertencia después de hacer clic en Delete all Files:
  5. Haz clic en Ok para confirmar y espera que la desinstalación se complete. Un mensaje que dice 'Success' se mostrará cuando la aplicación sea eliminada:
    • Después de desinstalar la aplicación, ya no se muestra en la página Manage Application.
    • Si aún aparece en la lista, haz clic en el botón Remove from List.

¿Cómo eliminar la base de datos?

Cuando haces clic en el botón Delete all Files, solo los archivos en el servidor son eliminados. Tu base de datos de WordPress sigue en tu panel. Para eliminar tu base de datos, navega a la página MySQL Databases. Hacia la derecha de la respectiva base de datos, haz clic en el botón Delete DB.

¿Qué sucede si no se muestra el botón Delete?

Si encuentras que el botón Delete no se muestra en tu panel junto a la Instalación con un clic, pónte en contacto con el soporte para obtener más ayuda.
