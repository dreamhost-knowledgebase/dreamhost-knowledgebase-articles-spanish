Generalidades

Este artículo describe como crear una página estática de Próximamente que se establecerá como la página de inicio de tu sitio web WordPress. Este tipo de página es útil cuando necesitas mostrar una página general a los visitantes antes de que tu sitio web completo se publique.

Crear una página estática

  1. Inicia sesión en tu panel de WordPress.
  2. En el panel izquierdo, navega a Páginas > Añadir nueva.
  3. Edita la página Próximanente como quisieras que apareciera.
  4. En la parte superior derecha, haz clic en el botón Publicar.
    Aparece un mensaje de confirmación en la parte superior derecha.
  5. En la parte superior izquierda, da clic en el icono de WordPress para volver al panel.
  6. En el panel izquierdo, navega a Ajustes > Lectura.
  7. En la página Ajustes de Lectura, da clic en el menú desplegable Portada y escoge la página Coming Soon que acabaste de crear.
  8. Da clic en el botón Guardar cambios para guardar esta nueva página de inicio.
  9. Visita la página de inicio de tu sitio para ver la nueva página que creaste.

Habilitar manualmente la página de mantenimiento programado de WordPress

Una alternativa de de crear una página de Próximanente es habilitar la página de mantenimiento programado de WordPress.

  1. Inicia sesión en tu servidor y crea un archivo llamado .maintenance.
  2. Agrega las siguientes lineas al archivo: 
    <?php
      $upgrading = time();
    ?>
  3. Visita tu sitio y verás un mensaje de mantenimiento predeterminado.

Para eliminar la página de mantenimiento programado, elimina o renombra el archivo .maintenance.

 

Cómo administrar el modo de mantenimiento

Cuando actualizas tus plugins, temas o archivos principales de WordPress, tu sitio de WordPress muestra el siguiente mensaje:

Briefly unavailable for scheduled maintenance. Check back in a minute.

WordPress elimina automáticamente ese mensaje una vez que se completan las actualizaciones. Sin embargo, en algunos casos, los conflictos o errores dejan el mensaje activo en el sitio, incluso después de que se completen las actualizaciones.

Eliminar el mensaje de "mantenimiento programado" de WordPress

Si el mensaje de 'mantenimiento programado' permanece activo en tu sitio, usa las siguientes instrucciones para eliminarlo:

  1. Consulta el siguiente artículo para obtener instrucciones sobre cómo iniciar sesión en tu servidor web. Puede usar el administrador de archivos en el panel o un cliente de terceros para iniciar sesión:
    • Generalidades sobre FTP y credenciales
  2. Una vez que hayas iniciado sesión, navega hasta el directorio de tu sitio web donde se encuentran tus archivos.
  3. En este directorio, busca un archivo llamado .maintenance y elimínalo.
  4. Una vez que elimines el archivo, asegúrate de borrar cualquier caché en tu sitio y luego actualiza tu navegador. El mensaje ya no debería aparecer en tu sitio.
