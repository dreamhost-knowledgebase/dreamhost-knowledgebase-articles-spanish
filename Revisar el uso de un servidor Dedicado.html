Generalidades

Es importante realizar un seguimiento del número de recursos que estás usando en tu Servidor Dedicado para evitar sobrepasar los límites. Hay una página en el panel que te permite ver estadísticas de usuarios:

Uso de Disco

La página Uso de Disco desglosa el ancho de banda por 'Máquina', 'Dominio' y 'Usuario'.

Esta página puede ser accesada desde la página Administrar Cuenta.

Baja hasta la otra sección Servicios Activos y da clic en el enlace Ver Uso de Ancho de Banda.

Revisar el uso de RAM en tiempo real

El panel no muestra el uso de RAM para tu Servidor Dedicado. Para ver esta información, debes iniciar sesión en tu servidor a través de SSH y correr comandos para verificar esta información. Mira el siguiente enlace para ver ejemplos. Ten en cuenta que aunque el artículo hace referencia a una máquina
