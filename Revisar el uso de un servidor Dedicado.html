Generalidades

Es importante realizar un seguimiento de la cantidad de recursos que estás utilizando en tu Servidor Dedicado para evitar sobrepasar los límites. Hay dos páginas en el panel que te permiten ver estadísticas de usuarios y ancho de banda:

Uso de Disco

La página Uso de Disco muestra las estadísticas de uso de 'Usarios', 'Dominios', 'Correo', y 'Bases de Datos'. La parte superior de la página muestra un resumen general.

Uso de ancho de banda

La página Uso de Ancho de Banda desglosa el ancho de banda por 'Maquina', 'Dominio', y 'Usuario.

Revisando el uso de RAM en tiempo real

El panel no muestra el uso de RAM para tu Servidor Dedicado. Para ver esta información, debes iniciar sesión en tu servidor a través de SSH y correr comandos para verificar esta información.
