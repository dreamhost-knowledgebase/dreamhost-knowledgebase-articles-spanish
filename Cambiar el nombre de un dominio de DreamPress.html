Generalidades

Este artículo explica cómo cambiar el nombre de tu sitio DreamPress a un nombre de dominio diferente. Estas instrucciones se pueden usar para un nombre de dominio regular como example.com o un dominio temporal de DreamPress como example.dream.press.

Asegúrate de que el dominio al que está cambiando el nombre ya esté configurado como Solo DNS. Luego puedes usar el cambiador de nombres de dominio en el panel.

Puedes usar el cambiador de nombres de dominio en el panel de DreamPress para cambiar cualquier dominio temporal, no solo los subdominios dream.press.

Preparar tu dominio para la migración

Confirma lo siguiente para el dominio al que estás moviendo. Si alguno de los siguientes elementos no está configurado, tu sitio no se conectará después de moverlo.

Eliminar alojamiento

El dominio al que mueves no se puede alojar. Si es así, debes eliminar el alojamiento del dominio antes de continuar. Puedes hacer esto configurando el dominio como Solo DNS. Esto es para que el dominio se pueda mover de forma segura sin romper un sitio en vivo.

Eliminar DNS personalizados

Asegúrate de que no hayan Registros de DNS personalizados como registros A o CNAME en el dominio al que estás moviendo (si existe alguno que apunte a un host web diferente). Si los hay, elimina los registros personalizados. Otros registros DNS personalizados están bien para dejarlos en su lugar.

 

Paso 1 — Cambiar tu dominio

Para cambiar el dominio de DreamPress:

  1. Navega hasta la página WordPress Administrado.
  2. Da clic en el botón Administrar al lado de tu dominio.
  3. Da clic en la pestaña Dominio en la parte superior.
  4. Tienes dos opciones para cambiar el nombre:

    Usarun dominio de tu propiedad

    Esta opción te permite ingresar un nombre de dominio que ya posees. El panel lo agregará por ti.

    Elige dominios existentes

    Esta opción te permite elegir un dominio que ya has agregado a tu panel. Aparece la siguiente advertencia que te notifica que se eliminarán las copias de seguridad anteriores del antiguo dominio:
    'Usar esto eliminará tu sitio de ensayo y todas las copias de seguridad anteriores y creará una nueva con el nombre example.com.'
  5. Haz clic en el botón Cambiar Domino.
    El DNS tarda entre 4 y 6 horas en actualizarse para tu nuevo dominio en DreamHost.

Paso 2 — Esperar a que se actualice tu DNS DreamHost

Como se indicó anteriormente, el DNS puede tardar entre 4 y 6 horas en actualizarse en DreamHost. Si apuntas tu DNS en tu host actual a DreamHost antes de que el DNS se haya actualizado en los servidores de DreamHost, el sitio no se conectará.

Se recomienda que esperes de 4 a 6 horas antes de actualizar tu DNS en tu host actual.

Paso 3 — Actualización de tu DNS en tu alojamiento actual

Finalmente, inicia sesión en tu registrador actual y apunta tu DNS a DreamHost. Después de que el DNS propague, tu sitio estará alojado con DreamPress.
