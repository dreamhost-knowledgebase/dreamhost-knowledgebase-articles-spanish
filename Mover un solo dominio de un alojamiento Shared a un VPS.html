Generalidades

Puedes mover un solo dominio a otro servidor haciendo lo siguiente:

  • Creando un nuevo usuario en el nuevo servidor
  • luego cambiando tu sitio para usar este nuevo usuario.

Cuando guardas los cambios, todos los archivos se migrarán al nuevo servidor. Revisa el siguiente artículo para más detalles:

  • Migrar un solo sitio web a un servidor diferente
