Generalidades

Un archivo php.ini es un archivo de texto simple que te permite ajustar la configuración de PHP. Por ejemplo, el servidor viene con algunas configuraciones predeterminadas, como un límite de memoria PHP de 256 MB. Otros límites que a menudo se cambian son para el tiempo de ejecución de PHP y el tamaño de carga. Este artículo te guía a través de cómo configurar un archivo php.ini (phprc) y agregar tus ajustes personalizados.

El término estándar para este archivo es php.ini, sin embargo, DreamHost usa un archivo phprc en su lugar. Esto actúa de la misma manera: solo el nombre es diferente. En este artículo, solo se usa el término phprc.

Determina qué versión de PHP usa tu sitio web

Antes de crear un archivo phprc, necesitarás saber qué versión de PHP está ejecutando tu sitio. Los siguientes artículos explican cómo ver la versión PHP de tu sitio y cambiarlo:

  • ¿Qué versión de PHP está usando mi sitio?
  • ¿Cómo cambiar la versión PHP de mi sitio?

Cuando crees tu archivo phprc en el servidor, usarás esta versión de PHP para especificar la ubicación correcta del archivo.

Cómo crear un archivo phprc

Puedes crear un archivo phprc iniciando sesión en tu servidor a través de FTP o SSH. Los siguientes enlaces te guían a través de cada opción:

  • Crear un archivo phprc vía FTP (más fácil para principiantes)
  • Crear un archivo phprc vía SSH (para usuarios avanzados)

Actualizar tus cambios

Después de crear tu archivo phprc, debes actualizar tu configuración de PHP para asegurarse de que la nueva configuración surta efecto. Visita el siguiente artículo para más detalles:
