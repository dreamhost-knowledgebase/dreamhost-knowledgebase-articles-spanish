Generalidades

Este artículo te guía a través de la configuración de un archivo php.ini (phprc) para ajustar la configuración de PHP. Editar este archivo de texto te permite modificar la configuración predeterminada de PHP, como memory_limit.

Sitios web de DreamPress

Revisa este artículo si necesitas ajustar la configuración de PHP dentro de DreamPress.

Crear un archivo php.ini

Los siguientes pasos explican cómo crear un archivo php.ini para administrar la configuración de PHP.

El término estándar para este archivo es php.ini; sin embargo, DreamHost utiliza el término phprc en su lugar. Este funciona de la misma manera, solo el nombre es diferente. En este artículo, solo se utiliza el término phprc.

 

Determinar cuál versión de PHP está usando tu sitios

Antes de crear un archivo phprc, necesitarás saber qué versión de PHP está ejecutando tu sitio. Los siguientes artículos explican cómo ver la versión PHP de tu sitio y cambiarlo:

  • ¿Qué versión de PHP está usando mi sitio?
  • Cambiar la versión PHP de un sitio

Cuando crees tu archivo phprc en el servidor, usarás esta versión de PHP para especificar la ubicación correcta del archivo.

Crear un archivo phprc

Puedes crear un archivo phprc iniciando sesión en tu servidor a través de FTP o SSH. Los siguientes enlaces te guían a través de cada opción:

  • Crear un archivo phprc vía FTP (más fácil para principiantes)
  • Crear un archivo phprc vía SSH (para usuarios avanzados)

Actualizar tus cambios

Después de crear el archivo phprc, debes actualizar la configuración de PHP para garantizar que la nueva configuración surta efecto. Revisa este artículo para obtener más información.
