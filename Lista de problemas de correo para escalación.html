Generalidades

Si crees que tu correo no funciona como debería, el soporte de DreamHost necesita información específica para ayudarte. Este artículo te proporciona una lista de verificación de los elementos que debes tener el soporte para solucionar tu problema. Los siguientes tipos de inconvenientes de correo están cubiertos en este artículo:

  • Inconvenientes con Enviar/Recibir/Conectividad
  • Inconvenientes con demoras
  • Si no se recibió el correo

Una vez recopilada la información que se describe a continuación, dirígete a la página Contactar a Soporte para enviar tu requerimiento de soporte.

Problemas de envío/recepción/conectividad

  • Error message – Describe el error exacto que ves en el cliente, incluidos los códigos de error. Pega el error que ves en tu solicitud de soporte.
  • Email address – La dirección que estás utilizando cuando se produce el problema.
  • Source IP (Es decir, la IP del cliente) – Puedes obtener esto en whatismyipaddress.com.
  • Provider – El ISP que está utilizando, como Verizon o Comcast.
  • Email client (nombre y número de versión) – Los ejemplos incluyen Webmail, Outlook, Thunderbird, y Mac Mail.
  • OS y número de versión – ¿Qué sistema operativo (y número de versión) o dispositivo estás usando?
  • Port # – ¿Cuáles son los puertos que estás usando? Visita el artículo de Configuración de cliente de correo para más detalles.
  • SSL – ¿Estás usando puertos seguros para conexiones entrantes y salientes?
  • Protocol – ¿Estás usando IMAP o POP3 para correo entrante? Visita el artículo POP3 vs IMAP para más detalles.
  • Authentication – ¿Estás usando la autenticación de contraseña para el correo saliente? Si
  • Time – Hora del error incluyendo tu zona horaria (u, hora actual).
  • Traceroute – Realiza un Traceroute a tu servidor de correo. Puede identificar en qué servidor de correo te encuentras visitando la página Data Center.
  • Server name – Consulta el siguiente artículo para obtener instrucciones sobre cómo localizar tu nombre de servidor de correo.
    • Cómo encontrar tu nombre de servidor de correo

Inconvenientes con demoras

El soporte de DreamHost necesita lo siguiente:

  • Full email headers – Revisa el artículo Ver encabezados de correo electrónico para obtener instrucciones sobre cómo obtenerlos.
  • Time – La hora en que se envió y recibió el correo, incluido zona horaria (u, hora actual).
  • Addresses – Dirección de envío y dirección de recepción
  • Where was the email sent from? – ¿Se envió el correo desde el servidor web usando phpmail/sendmail y así sucesivamente o mediante un cliente de correo?
  • Mail server – ¿En qué servidor de correo estás? Puedes identificar en qué servidor de correo te encuentras visitando la página Centro de Datos.

Si no se recibió el correo

  • Addresses – La dirección de correo que envió el correo y la dirección a la que se envió el correo.
  • Time – La fecha y hora en que se envió la dirección de correo, incluidos zona horaria (u, hora actual).

Los registros de correo de DreamHost solo retroceden unos días, por lo que debe proporcionar ejemplos recientes.
