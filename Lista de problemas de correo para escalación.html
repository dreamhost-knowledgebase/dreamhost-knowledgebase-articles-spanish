Generalidades

A continuación se proporciona una lista de verificación de elementos que el soporte de DreamHost necesita para solucionar cualquier problema de correo electrónico que estés experimentando.

Una vez que hayas reunido la siguiente información, procede a la página Contactar a soporte técnico para enviar tu solicitud de ayuda.

Lista de elementos a reunir

Por favor intenta hacer todo lo posible para recopilar la siguiente información para que el soporte pueda ayudarte mejor.

Problemas de envío/recepción/conectividad

  • Mensaje de error — Describe el error exacto que ves en el cliente, incluidos los códigos de error. Pega el error que ves en tu solicitud de soporte.
  • Direcciones de correo — La dirección que estás utilizando cuando se produce el problema.
  • Origen de la IP (Es decir, tu IP) — Puedes obtener esto en whatismyipaddress.com.
  • Proveedor – El ISP que está utilizando, como Verizon o Comcast.
  • Cliente de correo (nombre y número de versión) – Los ejemplos incluyen Webmail, Outlook, Thunderbird, y Mac Mail.
  • Sistema Operativo y número de versión – ¿Qué sistema operativo (y número de versión) o dispositivo estás usando?
  • Número de puerto – ¿Cuáles puertos estás usando? Visita el artículo de Configuración de cliente de correo para más detalles.
  • SSL – ¿Estás usando puertos seguros para conexiones entrantes y salientes?
  • Protocolo — ¿Estás usando POP o IMAP para el correo entrante? Revisa el artículo POP vs IMAP para más detalles.
  • Autenticación – ¿Estás usando la autenticación de contraseña para el correo saliente? Si
  • Hora – Hora del error, incluyendo tu zona horaria (u, hora actual).
  • Traceroute – Realiza un Traceroute a tu servidor de correo. Puede identificar en qué servidor de correo te encuentras visitando la página Data Center.
  • Nombre del servidor – Revisa este artículo para aprender más sobre como ubicar el nombre del servidor.

Inconvenientes con demoras

  • Encabezados de mensajes completos — Revisa el artículo Ver encabezados de correo electrónico para obtener instrucciones sobre cómo obtenerlos.
  • Hora — La hora en que se envió y recibió el correo, incluido zona horaria (u, hora actual).
  • Direcciones — Dirección de envío y dirección de recepción
  • ¿De dónde se envió el correo? — ¿Se envió el correo desde el servidor web usando PHP mail/sendmail o mediante un cliente de correo?
  • Servidor de correo — ¿Qué servidor de correo estás usando? Puedes identificar en qué servidor de correo te encuentras visitando la página Centro de Datos.

Si no se recibió el correo

Los registros de correo de DreamHost solo se remontan a unos pocos días, por lo que debes proporcionar ejemplos recientes.

  • Direcciones — Direcciones de envío y recepción
  • Hora — La fecha y hora en que se envió la dirección de correo, incluidos zona horaria (u, hora actual).
