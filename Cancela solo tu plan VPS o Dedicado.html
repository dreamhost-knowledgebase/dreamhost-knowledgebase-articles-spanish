¿Cierre por COVID-19?

Pónte en contacto con el soporte de DreamHost si estás considerando cancelar tu cuenta debido a COVID-19. DreamHost está aquí para ayudarte y puede ofrecerte aplazamiento de pago y otras opciones durante este tiempo.

Ver tu tipo de plan de alojamiento

Puedes ver todos los servicios facturados a tu cuenta en la página Administrar Cuenta. Allí verás una lista de todos los servicios que utilizas actualmente.

Cerrar/cancelar tu plan de alojamiento VPS

Si deseas mantener tu sitio web activo y migrarlo a un plan de alojamiento diferente, ya debes tener un plan de alojamiento compartido o dedicado. Solo entonces tendrás la oportunidad de migrar los datos a este nuevo servidor cuando canceles el VPS.

Si no hay otro servicio de alojamiento, el sitio web se cerrará y tu cuenta solo estará disponible para registros de dominio.

Asegúrate de tener otro plan de alojamiento activo si deseas mantener tu sitio en funcionamiento.

Revisa el siguiente artículo para obtener instrucciones completas:

  • Eliminar un servidor privado

Cerrar/cancelar tu plan de alojamiento Dedicado

Si deseas mantener tu sitio web activo y migrarlo a un plan de alojamiento diferente, ya debes tener un plan de alojamiento compartido o VPS activo. Solo entonces tendrás la oportunidad de migrar los datos a este nuevo servidor cuando canceles el plan Dedicado.

Si no hay otro servicio de alojamiento, el sitio web se cerrará y tu cuenta solo estará disponible para registros de dominio.

Asegúrate de tener otro plan de alojamiento activo si deseas mantener tu sitio en funcionamiento.

Revisa el siguiente artículo para obtener instrucciones completas:

  • Eliminar un servidor Dedicado

¿Qué sucede con mis registros de dominio cuando cierro mi cuenta?

DreamHost no eliminará los registros de tus dominios cuando cierres tu cuenta. Todos los registros de dominio adquiridos previamente continúan activos durante el resto de sus períodos de registro.

Puedes continuar iniciando sesión en tu panel y administrar tus registros de dominio como lo hiciste antes.
