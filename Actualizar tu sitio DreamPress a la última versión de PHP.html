Generalidades

PHP es el programa de lenguaje que potencia tu sitio WordPress. DreamHost recomienda usar la última versión de PHP en tu sitio WordPress para asegurar un rendimiento y seguridad óptimos.

Por favor asegúrate de revisar el siguiente artículo para ver las generalidades de todo lo que necesitas considerar antes de actualizar la versión de PHP de tu sitio WordPress:

  • Actualizar tu sitio WordPress a la última versión de PHP

Este artículo explica cómo actualizar el PHP de un sitio DreamPress. Si instalaste WordPress usando el instalador del panel o instalaste WordPress manualmente, revisa el siguiente artículo en su lugar para ver instrucciones sobre la actualización de la versión de PHP:

  • Cambiar la versión de PHP de un sitio web

Preparar tu sitio DreamPress para una actualización de PHP

Si ya está en uso un sitio provisional de DreamPress, asegúrate de publicar cualquier cambio que desees guardar antes de trabajar en el sitio staging.

Por favor ten en cuenta que si bien seguir esta lista puede ayudar a minimizar los problemas, es importante saber que debido a la gran cantidad de temas, plugins, y personalizaciones disponibles en WordPress, siempre existe una pequeña posibilidad de encontrar errores o problemas durante el proceso de actualización de PHP. En tales casos, puedes intentar cambiar a la versión anterior de PHP.

Estos pasos no son obligatorios, pero son muy recomendables para mantener la seguridad y el buen funcionamiento de tu sitio:

  1. Verifica si hay actualizaciones pendientes en tu panel de administración de DreamPress. Ten en cuenta que las instalaciones de DreamPress están configuradas para actualizarse automáticamente.
  2. Identifica y revisa temas o plugins más antiguos que el desarrollador no haya actualizado en más de un año. Considera encontrar alternativas adecuadas o ponerte en contacto con el mismo para obtener actualizaciones.
  3. Revisa si tienes temas o plugins pagos con licencia vencida para actualizaciones. Renueva las licencias o explora opciones alternativas.
  4. Si tienes personalizaciones escritas en PHP, se recomienda que un desarrollador las revise para verificar la compatibilidad con la nueva versión de PHP.

Probar la actualización usando un sitio provisional o staging

Una forma segura de probar la nueva versión de PHP es crear un sitio web provisional o staging. Este sitio provisional duplicará tu sitio web activo existente, lo que te permitirá hacer pruebas para ver cómo reacciona a la nueva versión de PHP.

Si no sucede nada inusual, puedes actualizar tu sitio web en vivo. Revisa el siguiente artículo para obtener información sobre cómo crear un sitio web staging para DreamPress:

  • Cómo hacer un sitio provisional (o staging) en DreamPress

Ver la versión de PHP de un sitio DreamPress

Puedes ver la versión PHP de tu sitio DreamPress en el panel.

  1. Navega a la página Administrar Sitios Web.
  2. Da clic en el botón Administrar para abrir la página Configuraciones de Dominios, la cual te permite ajustar múltiples configuraciones de tu sitio.
    • Si estás en la Vista de Malla, da clic en el botón Administrar en la parte inferior de la caja del sitio.
    • Si estás en la Vista de Lista, da clic en el botón Administrar a la derecha de tu dominio.

La versión de PHP de tu sitio aparece debajo de la sección PHP en la pestaña Sitio Web, y la versión de WordPress está debajo de la sección WordPress.

Actualizar la versión de PHP de un sitio DreamPress

En el momento, la única opción disponible para actualizar la versión de PHP de un sitio DreamPress a través de un sitio web provisional. Revisa el siguiente artículo para ver las instrucciones completas:

  • Probar las nuevas versiones de PHP en Staging

Los sitios de prueba recién creados en DreamPress utilizarán por defecto la versión de PHP que está usando tu sitio en vivo. Si necesitas ayuda para cambiar la versión de PHP, por favor contacta a soporte.
