Generalidades

Es posible que encuentres un error cuando agregas un sitio web y alojamiento en DreamHost. Este artículo muestra algunos errores comunes y cómo resolverlos.

El mensaje 'example.com is almost here!' aparece en mi sitio web

Mensajes de error agregando un dominio

Debes agregar el dominio principal

No puedes agregar ese dominio: 
debes agregar el dominio principal example.com a tu cuenta antes de agregarle el subdominio.

Este error quiere decir que el dominio principal (example.com) aún no está agregado a tu cuenta. Un subdominio (blog.example.com) solo puede ser agregado si su dominio principal es creado primero. El dominio principal debe ser agregado primero incluso si no planeas alojar el dominio de segundo nivel aquí en DreamHost.

No tienes permiso para agregar este dominio

No puedes agregar ese dominio: no tienes permiso para agregarlo.

Este error significa que el dominio que intentas agregar a tu cuenta ya está alojado en otra cuenta en DreamHost. Si estás moviendo el dominio de una cuenta existente a la que ya tienes acceso, primero debes eliminar el dominio de la cuenta anterior antes de agregarlo a una cuenta nueva. Si no sabes en qué cuenta está alojado el dominio en el momento, por favor contacta a soporte.

El dominio ya existe en el sistema

El nombre de dominio ya existe en el sistema.

Si ves este error, por favor confirma que el dominio no está alojado en otra cuenta de DreamHost a la que tengas acceso. Si no sabes en qué cuenta está alojado el dominio actualmente, por favor contacta a soporte.
