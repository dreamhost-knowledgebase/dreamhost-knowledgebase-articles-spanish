Generalidades

Primero, inicia sesión en tu panel DreamHost. Cuando hayas ingresado, navega a la página Bases de Datos MySQL. En ésta página, verás una lista completa de tus nombres de host y bases de datos MySQL.

Acceder a la información de usuario de tu base de datos

En la página Bases de Datos MySQL, haz clic en el nombre de usuario a la derecha de la base de datos para ver su configuración.

Eliminar

Al hacer clic en el botón Eliminar BD se elimina este acceso de los usuarios de MySQL a la base de datos que figura en la parte superior de la página. Cuando un usuario de MySQL se elimina de todas las bases de datos, el usuario de MySQL deja de existir y se considera completamente eliminado.

Administrar usuarios de MySQL, hosts permitidos y privilegios

La página Bases de Datos MySQL aparece.

¿Qué puede hacer <USUARIO> a las tablas en estas bases de datos?

Esta sección permite agregar o eliminar privilegios específicos del usuario MySQL, lo que afecta su acceso para leer, escribir o cambiar datos, así como administrar el contenido de la base de datos.

  • Seleccionar — Seleccionar: otorga la capacidad de leer filas de tablas.
  • Insertar — Insertar: otorga la capacidad de agregar nuevas filas a las tablas.
  • Actualizar — Actualizar: otorga la capacidad de modificar las filas existentes en las tablas.
  • Eliminar — Eliminar: otorga la capacidad de eliminar filas en las tablas.
  • Crear — Crear: otorga la capacidad de crear nuevas tablas. Esto no permite la creación de nuevas bases de datos en la línea de comando.
  • Soltar — Soltar: otorga la capacidad de eliminar tablas y vistas.
  • Índice — Índice: otorga la capacidad de agregar o eliminar índices de las tablas existentes.
  • Host permitidos — Alterar: otorga la capacidad de cambiar la estructura de las tablas.

Hosts permitidos

Una lista de hosts que pueden iniciar sesión en este usuario de MySQL. Por razones de seguridad, el valor predeterminado es %.dreamhost.com, que restringe las conexiones desde fuera de la red DreamHost, incluso si se proporciona un nombre de usuario y contraseña correctos. Se pueden agregar direcciones IP o comodines adicionales en las nuevas líneas para permitir el acceso a una red externa. Modificar o eliminar %.dreamhost.com puede resultar en la inaccesibilidad de MySQL para los sitios web alojados en DreamHost.

Contraseña actual

Haz clic en el botón Mostrar para mostrar la contraseña de este usuario de MySQL.

Nueva contraseña

Ingresa una nueva contraseña aquí para cambiar la contraseña de los usuarios. Ten en cuenta que todos los sitios que usan este usuario de MySQL dejan de funcionar en cuestión de minutos, hasta que todos los archivos de configuración se actualicen a la nueva contraseña.

Repetir nueva contraseña

Confirma la nueva contraseña ingresándola nuevamente aquí.

Cuando hayas realizado todos los cambios deseados, haz clic en ¡Modificar usuario ahora! para confirmar.

Los cambios pueden tardar hasta 5 minutos en surtir efecto.
