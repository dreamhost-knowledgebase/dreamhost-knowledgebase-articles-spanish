Generalidades

Este artículo describe las diversas formas en que puedes mejorar o cambiar tu plan desde un plan de alojamiento VPS.

Cambiar tu plan a Shared Starter

Si deseas cambiar de un plan de servidor privado (VPS) a un plan de alojamiento Shared Starter, asegúrate de cumplir con las limitaciones descritas en el siguiente artículo:

  • Agregar un plan de alojamiento Shared Starter

Si deseas continuar, Contacta a Soporte para solicitar

Cambiar tu plan a Shared

Primero, asegúrate de haber agregado un plan de Shared Hosting.

  • Agregar un plan de Shared Hosting

Luego puedes eliminar el servidor privado.

  • Eliminar un VPS

Al eliminar, verás un mensaje que te informa que todo en el VPS se migrará al servidor compartido.

Si tienes un servidor compartido y un Servidor Dedicado en tu cuenta, solo el Servidor Dedicado se mostrará como una opción para migrar al eliminar tu VPS. 

Si preferirías migrar a un servidor compartido, contacta a soporte. El soporte de DreamHost migrará los datos por tí.

Mejorar tu plan a Servidor Dedicado

Si deseas migrar todo el contenido de tu VPS a un Servidor Dedicado, primero debes comprar el Servidor Dedicado como se muestra en el siguiente artículo:

  • Agregar un servidor Dedicado

Durante ese proceso, selecciona tu VPS en el menú desplegable para migrar todos los datos. Una vez que todo ha sido migrado, puedes eliminar tu VPS:

  • Eliminar un VPS

Mejorar tu plan a DreamPress

Consulta el siguiente artículo para obtener instrucciones sobre cómo actualizar un sitio de WordPress existente a DreamPress:

  • ¿Cómo mejorar una página WordPress a DreamPress?

Si solo tenías el único sitio de WordPress en tu VPS, puedes cancelarlo.

  • Eliminar un VPS

Si tienes otros sitios en el VPS, debes continuar pagando ese plan, ya que solo el sitio único se habría migrado a DreamPress.
