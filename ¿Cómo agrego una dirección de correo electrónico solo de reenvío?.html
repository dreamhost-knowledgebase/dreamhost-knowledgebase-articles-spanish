Generalidades

Una dirección de correo electrónico de solo reenvío es una dirección de correo que no tiene un buzón activo. Esto se debe a que la función principal de este usuario es reenviar todo el correo electrónico enviado a otro usuario de correo. El mensaje solo se almacenará en el usuario configurado para recibir el correo electrónico reenviado.

NO establezcas una dirección de correo electrónico activa en una dirección de solo reenvío. Esto eliminará todo el correo electrónico guardado anteriormente con esta dirección. Si deseas reenviar una dirección de correo electrónico existente, simplemente agrega los correos electrónicos adicionales para reenviarlos editando la dirección en la página Manage Email.

Creando una redirección de correo

  1. Visita la página Manage Email en tu panel.
  2. Haz clic en el botón Create New Email Address.
  3. Desplázate hacia abajo a la sección Forward Only.
    Email Address
    A la izquierda del símbolo "@", ingresa tu alias deseado. A la derecha del símbolo "@", selecciona el dominio o subdominio en el que se creará el usuario.
    List all email addresses to forward to, one per line
    Ingresa la dirección o direcciones de correo electrónico a las que deseas reenviar a este usuario. Asegúrate de tener cada dirección de correo electrónico para reenviar y configurar en tu propia línea dentro del campo vacío.

  4. Haz clic en el botón Forward to These Addresses! para crear el reenvío.
    Las columnas en la captura de pantalla anterior son para direcciones de correo electrónico de solo reenvío, que no tienen un buzón activo. Debido a esto, estos usuarios se enumeran por separado de los usuarios que tienen buzones activos.

    Las direcciones de correo electrónico de solo reenvío son direcciones de correo electrónico que no tienen un buzón activo porque estos usuarios están destinados a reenviar todos los correos electrónicos que se le envían a otra dirección de correo electrónico.

Descripción de los campos en la página de correo electrónico de solo reenvío

Forwarding-Only Addresses: En esta columna, solo verás la dirección de correo electrónico del usuario reenviador porque no hay un buzón activo para mostrar detalles para.

Forwards to: Esta columna mostrará la dirección de correo electrónico del usuario que recibirá los mensajes enviados a este usuario reenviador. Puedes agregar más de uno si lo deseas.

Actions: Esta columna tiene dos opciones para cada usuario enumerado: Edit y Delete.

Edit: El botón editar lo lleva a otra página donde puedes realizar cambios en tu dirección de reenvío.
Delete: Al hacer clic en este botón se elimina al usuario, pero aparece una ventana emergente que te solicita que verifiques esta solicitud primero con el mensaje:
Really delete forwarduser@example.com? (There is no email stored with it.)
Al hacer clic en “OK” se eliminará el usuario y el buzón de ese usuario. Esto requerirá unos minutos, pero una vez hecho esto, ya no verás a ese usuario en la lista en la página Manage Email para ese dominio.

Remover reenvío de correo

Hay tres opciones para eliminar el reenvío de correo electrónico:

  • Cambiar de 'forwarding' a una dirección completamente alojada o correo basura.
  • Deshabilita 'forwarding'.
  • Elimina la dirección de reenvío.
  1. Visita la página Manage Email en tu panel.
  2. A la derecha de la dirección de reenvío hay tres botones: 'Edit', 'Disable', 'Delete'.
  3. Haz clic en cualquiera de esos botones según lo que te gustaría hacer con la dirección.
