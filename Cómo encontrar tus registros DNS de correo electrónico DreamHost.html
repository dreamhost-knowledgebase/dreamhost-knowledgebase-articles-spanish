Generalidades

Si tu dominio no utiliza los nameservers de DreamHost, los registros DNS relacionados con el correo no estarán automáticamente activos para el dominio. Deberás configurar manualmente esos registros en el host donde se administra los DNS tu dominio. Puedes verificar hacia dónde apuntan tus nameservers en whoisweb.dreamhost.com.

  • Cuando un dominio tiene direcciones de correo electrónico configuradas en el panel de DreamHost, todos los registros DNS relacionados con el correo se crean automáticamente en la página Administrar Dominios.
  • Puedes encontrar tus registros DNS de correo en la página Administrar Dominios.

Si buscas información sobre cómo configurar tu correo electrónico en un cliente de correo, mira los siguientes artículos:

  • Generalidades de las configuraciones del cliente de correo
  • Protocolos y números de puerto para clientes de correo

Encontrar tus registros DNS de correo electrónico

  1. Haz clic en el enlace DNS debajo del dominio para ver tus registros.

    Los registros a continuación son solo ejemplos. Tus registros DNS específicos serán diferentes.

    Los siguientes ejemplos SÓLO muestran registros DNS de correo electrónico. Tu sitio web tendrá registros adicionales, sobre los cuales puedes leer más en el siguiente artículo:

    • Generalidades de los DNS de DreamHost
    Hay dos tipos de registros DNS relacionados con el correo.
    Registros MX
    le dice a los remitentes a dónde deben enviarse los correos.
    Por ejemplo, si el filtro de spam NO se habilitó, los registros se ven así:
    • mx1.dreamhost.com
    • mx2.dreamhost.com
    Si el filtro de spam está habilitado tus registros MX serán los siguientes:
    • vade-in1.mail.dreamhost.com
    • vade-in2.mail.dreamhost.com
    Registros A
    Los utiliza tu programa de correo cuando se conecta al servidor de correo para ver tu bandeja de entrada o enviar correos electrónicos. Hay varios registros de correo A para tu dominio.
  2. Encuentra los siguientes registros:
    Registros DNS Ejemplo de los valores
    _domainkey Los Registros DKIM firman los correo electrónicos para ayudar a la validación del remitente.
    _autodiscover._tcp (SRV) 5 0 443 autoconfig.dreamhost.com — Este es un registro SRV que ayuda a las aplicaciones de correo electrónico a aplicar la configuración correcta automáticamente.
    autoconfig (CNAME) autoconfig.dreamhost.com — Este es un registro CNAME que ayuda a las aplicaciones de correo electrónico a aplicar la configuración correcta automáticamente.
    *mail. (A) 64.90.62.162 — Este es un subdominio estándar que se puede utilizar para conectar su cliente o programa al servidor de correo.
    *mail. (MX)

    Este dominio ha habilitado el filtro de spam, entonces los registros MX aparecen así:
    vade-in1.mail.dreamhost.com
    vade-in2.mail.dreamhost.com.

    o

    mx1.mailchannels.net
    mx2.mailchannels.net

    Si el filtro de spam no se ha habilitado, verás: mx1.dreamhost.com y mx2.dreamhost.com.
    *mailboxes. (A) 66.33.205.233 — Esto se usa para iniciar sesión en la Interfaz Mailboxes de DreamHost.
    *www.mailboxes. (A) 66.33.205.233 — Esto se usa para iniciar sesión en la Interfaz Mailboxes de DreamHost.
    *webmail. (A) 69.163.136.138 — Esto se usa para iniciar sesión en el panel de webmail.
    *www.webmail. (A) 69.163.136.138 — Esto se usa para iniciar sesión en el panel de webmail.
  3. Ingresa todos esos registros en el sistema donde se alojan y administran los DNS del dominio.

Después de realizar el ajuste de DNS, los cambios tardarán varias horas en actualizarse en línea debido a propagación de los DNS.
