Generalidades

Cuando agregas un dominio en tu panel, puedes configurarlo de varias maneras:

  • Fully Hosted — Esto quiere decir que aparecerá en Internet
  • Redirect — Esto redirecciona tu dominio a un sitio web completamente alojado
  • Mirrored — Esto refleja un sitio web completamente alojado
  • DNS Only — Esto elimina el alojamiento y te permite apuntar tu DNS fuera de DreamHost

Hay una opción final llamada 'Parked'. Un dominio parqueado no hace nada ni tiene configuraciones. Todas las opciones de arriba muestran un sitio o apuntan a un sitio para mostrar. Un dominio parqueado es solo un estado inerte hasta que decidas hacer algo con el sitio web.

Cómo parquear un dominio

Cuándo compras un registro de dominio, el dominio es agregado automáticamente a la página Manage Domains. También puedes agregar un dominio para parquearlo, o cambiar un dominio existente a Parqueado.

Agregar un dominio como Parqueado

  1. Navega hasta la página Manage Domains.
  2. Da Click en Add Hosting to a Domain / Sub-Domain.
  3. Baja hasta la sección Parked.
  4. Ingresa tu nuevo dominio que deseas parquear, luego da click en el botón Park this domain para habilitarlo.
  5. De nuevo en la página Manage Domains, verás que ahora está parqueado.

Configurar un dominio existente como Parqueado

  1. Navega hasta la página Manage Domains.
  2. A la derecha de tu dominio da click en el enlace Edit.
  3. Baja hasta la sección Parked y da click en Park this domain.

Una vez que tu dominio está parqueado, DreamHost muestra un mensaje genérico por defecto que dice 'coming soon'.

Los dominios parqueados no están alojados, entonces no hay servidor al que iniciar sesión para subir información.

Cómo desparquear un dominio

Para desparquear un dominio, necesitas asignar un tipo de alojamiento diferente. Lo siguiente son opciones que puedes cambiar a:

  • Fully Hosted — Esto quiere decir que aparecerá en Internet
  • Redirect — Esto redirecciona tu dominio a un sitio web completamente alojado
  • Mirrored — Esto refleja un sitio web completamente alojado
  • DNS Only — Esto elimina el alojamiento y te permite apuntar tu DNS fuera de DreamHost

Después de cambiar el tipo de alojamiento, los nuevos valores de DNS necesitan propagar en línea lo cual puede tomar hasta 6 horas. Vé el siguiente artículo para ver más detalles.
