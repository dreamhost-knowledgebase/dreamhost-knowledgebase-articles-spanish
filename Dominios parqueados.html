Generalidades

Cuando agregas un dominio en tu panel, puedes configurarlo de varias maneras:

  • Completamente Alojado — Esto quiere decir que aparecerá en Internet
  • Redirección — Esto redirecciona tu dominio a un sitio web completamente alojado
  • Espejo — Esto refleja un sitio web completamente alojado
  • Solo DNS— Esto elimina el alojamiento y te permite apuntar tu DNS fuera de DreamHost

Hay una opción final llamada 'Parqueo'. Un dominio parqueado no hace nada ni tiene configuraciones. Todas las opciones de arriba muestran un sitio o apuntan a un sitio para mostrar. Un dominio parqueado es solo un estado inerte hasta que decidas hacer algo con el sitio web.

Cómo parquear un dominio

Cuando compras un registro de dominio, el dominio es agregado automáticamente a la página Administrar Sitios Web. También puedes agregar un dominio para parquearlo, o cambiar un dominio existente a Parqueado.

Agregar un dominio como Parqueado

  1. Navega hasta la página Administrar Sitios Web.
  2. Da clic en Añadir un Sitio Web.
  3. Tes opciones aparece.
    • Ingresa un dominio o subdominio que ya posees: Selecciona esta opción si ya has registrado tu dominio.
    • Crea un subdominio: Selecciona esta opción para crear un subdominio en un dominio que ya hayas registrado.
    • Regista un dominio nuevo: Selecciona esta opción para registrar primero tu nombre de dominio.
  4. Ingresa el nuevo dominio que deseas parquear, luego haz clic en el botón Siguiente: Alojamiento para proceder.
  5. Selecciona la opción Parquear dominio (sin alojamiendo) en la página siguiente y luego haz clic en el botón Parquear dominio.
  6. De nuevo en la página Administrar Sitios Web, verás que ahora está parqueado.

Configurar un dominio existente como Parqueado

  1. Navega a la página Administrar Sitios Web.
  2. Da clic en el botón Administrar para abrir la página Administrar Sitios Web, la cual te permite ajustar múltiples configuraciones de tu sitio.
    • Si estás en la Vista de Malla, da clic en el botón Administrar en la parte inferior de la caja del sitio.
    • Si estás en la Vista de Lista, da clic en el botón Administrar a la derecha de tu dominio.
  3. Da clic a la pestaña Alojamiento.
  4. Baja hasta la sección Opciones de no alojamiento y selecciona la opción Parquear Dominio.
  5. Da clic al botón Parquear este Dominio.
  6. Da clic al botón Si, Parquear este Dominio que aparece.

Una vez que tu dominio está parqueado, DreamHost muestra un mensaje genérico por defecto que dice 'PARKED DOMAIN'.

Los dominios parqueados no están alojados, entonces no hay servidor al que iniciar sesión para subir información.

Como desparquear un dominio

Para desparquear un dominio, necesitas asignar un tipo de alojamiento diferente. Lo siguiente son opciones que puedes cambiar a:

  • Completamente Alojado — Esto quiere decir que aparecerá en Internet
  • Redirección — Esto redirecciona tu dominio a un sitio web completamente alojado
  • Espejo — Esto refleja un sitio web completamente alojado
  • Solo DNS— Esto elimina el alojamiento y te permite apuntar tu DNS fuera de DreamHost

Para Alojar Completamente de nuevo un dominio que tenías Parqueado, debes completar los siguientes pasos:

  • Asignar el dominio a el mismo usuario con el que fue configurado anteriormente.
  • Asegurarse de que el 'directorio Web' (donde los archivos del sitio web están localizados) es el mismo que usaste cuando lo Parqueaste.

Después de cambiar el tipo de alojamiento, los nuevos valores de DNS necesitan propagar en línea lo cual puede tomar hasta 6 horas. Mira el siguiente artículo para ver más detalles.
