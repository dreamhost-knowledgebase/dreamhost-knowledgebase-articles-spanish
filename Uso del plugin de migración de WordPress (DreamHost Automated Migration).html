Generalidades

DreamHost se ha asociado con los creadores de BlogVault para proporcionar DreamHost Automated Migration, un complemento gratuito de WordPress que te permite migrar rápida y fácilmente tu sitio de WordPress a DreamHost.

El complemento de migración automatizada de DreamHost funciona con la mayoría de los sitios de WordPress. Si crees que puedes tener una configuración de sitio u host web inusual, consulta la sección de Preguntas Frecuentes al final del artículo para obtener posibles soluciones.

Antes de comenzar una migración

Asegúrate de hacer lo siguiente antes de instalar el complemento de migración automatizada de DreamHost:

  • Prueba e instala todas las actualizaciones disponibles para WordPress, incluidos todos los temas y complementos/plugins instalados. Asegúrate de que tu sitio no tenga errores o código pirateado.
  • Para evitar cualquier tiempo de inactividad en tu sitio, mantén los DNS del dominio apuntando a tu host anterior hasta que se complete la migración y hayas tenido la oportunidad de mirar el sitio migrado en DreamHost.

Cómo instalar el complemento de migración automatizada de DreamHost

Paso uno: Encuentra la información de tu cuenta DreamHost

A continuación se explica qué información necesitas y dónde encontrarla en el panel web de DreamHost.

  1. Si aún no lo has hecho, Agrega Hosting a tu dominio en tu panel de DreamHost. Puedes agregar WordPress como una Instalación de 1 Clic.
    • Está bien hacer esto mientras tu dominio está alojado en tu host anterior. Agregar alojamiento en DreamHost no afecta tu alojamiento actual. Tu sitio existente permanecerá en vivo por ahora.
    • Si te preocupa afectar tu sitio existente, puedes agregar un subdominio temporal que termine con .dream.press (para los sitios de DreamPress) o .dreamhosters.com para todos los demás planes de alojamiento. Ten en cuenta que crear un sitio de prueba temporal creará trabajo adicional al final del proceso, ya que deberás actualizar tu base de datos a la URL correcta.
  2. Busca la siguiente información sobre tu cuenta de DreamHost:
    • Usuario FTP/SFTP: Navega a la página Manage Domains. Toma nota de tu usuario a la derecha de tu dominio (se muestra como 'Usuario: nombre').
    • Dirección de Host/Servidor: Haz clic en el usuario para abrir la página 'Editing User'. El nombre de tu servidor se muestra entre paréntesis junto a tu nombre de usuario en la parte superior (por ejemplo, 'en el nombre del servidor'). Usa el siguiente formato de dirección del servidor según el tipo de plan de alojamiento que tengas:
      • Shared Hosting — servername.dreamhost.com
      • DreamPress o VPS — servername.dreamhostps.com
      • Dedicated Server — servername.dreamservers.com
    • FTP o SFTP: Asegúrate de que tu tipo de usuario sea SFTP o SSH. Si es FTP, cambia el tipo de usuario a SFTP o SSH (que también da acceso SFTP).
    • Contraseña FTP/SFTP: Configura una contraseña para este usuario y haz clic en Save Changes. Mantén esta información lista para ingresar en el siguiente paso.

Paso dos: Instalar y activar el plugin

Para instalar y activar el plugin DreamHost Automated Migration en tu sitio WordPress desde tu host anterior:

  1. Abre el Panel de administración de WordPress para tu sitio.
  2. Selecciona 'Plugins > Add New' en la barra de navegación lateral izquierda.
  3. Escribe 'DreamHost Automated Migration' en el campo de búsqueda en la parte superior derecha.
  4. Haz clic en el botón Install Now en la ventana cuando aparezca.
  5. Haz clic en el botón Activate.
    La página de configuración 'Migrate Your Sites' se abre:
  6. Ingresa lo siguiente:
    • Email — Una dirección de correo electrónico donde te gustaría recibir notificaciones sobre el estado de la migración y la finalización.
    • Destination Site URL — La dirección web (URL del sitio de destino) de tu sitio. Por lo general, esta es la misma dirección de tu sitio en tu host actual. Si estás utilizando un subdominio temporal de DreamHost (dream.press or dreamhosters.com) o si vas a cambiar el dominio para tu sitio, usa esa dirección en su lugar. Asegúrate de ingresar 'https://' al comienzo de la dirección web si planeas usar un certificado SSL.
    • Type — Seleciona sFTP. Es lo más recomendable.
    • Host/Server Address — La dirección del servidor. Ver arriba para más detalles.
    • FTP/SFTP User — El nombre de usuario. Ver arriba para más información.
    • FTP/SFTP Password — La contraseña. Ver arriba para más detalles.
    • Revisa y acepta (haciendo clic en la casilla de verificación) los Términos del servicio y la Política de privacidad si estás de acuerdo y deseas continuar.
  7. Haz clic en el botón Migrate cuando termines.
  8. Marca la URL como favorita en tu navegador para verificar el progreso de tu migración y cierra la pestaña o ventana del navegador mientras se está ejecutando.

La mayoría de las migraciones tardan menos de una hora en completarse, pero si tu sitio es más grande que un par de GB de datos, podría llevar más de una hora. Una vez que se complete, recibirás un correo electrónico de notificación a la dirección que ingresaste durante el paso anterior.

Verifica tu sitio migrado y publícalo en DreamHost

Si tu dominio es el mismo en DreamHost y el host anterior

Agrega '.dream.website' al final de tu dominio para ver el sitio migrado en DreamHost. Por ejemplo, si tu dominio es 'example.com', revisa tu sitio migrado en 'example.com.dream.website'.

Una vez que estés satisfecho con el sitio migrado, cambia los DNS de tu dominio para que apunten a DreamHost.

Si eliges un nombre de dominio diferente en DreamHost del host anterior

Puedes usar un nombre de dominio diferente para ver tu sitio en DreamHost mientras el sitio actual permanece activo. Asegúrate de que cualquier nuevo nombre de dominio que elijas ya esté utilizando los registros DNS de DreamHost.

Si estás utilizando un subdominio temporal que termina en .dreamhosters.com o .dream.press

Puedes usar el nombre de subdominio temporal para ver de inmediato el sitio migrado.

  • Dominios de DreamPress: cuando estés listo para publicar tu sitio en DreamHost usando tu dominio real, sigue estas instrucciones para cambiar el nombre de tu dominio.
  • Todos los demás planes de alojamiento: cuando estés listo para publicar tu sitio en DreamHost usando tu nombre de dominio real, sigue estas instrucciones para cambiar el nombre de tu dominio.

Preguntas Frecuentes

¿Puedo usar este complemento para migrar un sitio múltiple de WordPress?

Sí, el complemento Migrate to DreamHost permite la migración de una red de WordPress o una instalación multisitio. Después de instalar el complemento, asegúrate de 'Activar la red' ('Network Activate') para asegurarte de que se pueda ejecutar desde el panel de administración de la red. Sin embargo, ten en cuenta las siguientes consideraciones:

  • La migración debe mover toda la red a DreamHost. No permite la migración de subsitios individuales o la importación de un subsitio a una red diferente.
  • Los planes de alojamiento recomendados para WordPress Multisite son DreamHost VPS, Servidor Dedicado o DreamCompute. Las instalaciones multisitio en planes Shared y DreamPress no se recomiendan ni son compatibles.
  • El área de soporte de DreamHost no ofrece asistencia técnica con instalaciones de WordPress multisite.

¿Hay restricciones o limitaciones?

El complemento funcionará con la gran mayoría de los sitios, temas, complementos y configuraciones de WordPress.

Sin embargo, ten en cuenta que los sitios de WordPress configurados en WordPress.com no se pueden migrar utilizando este o cualquier otro complemento como una copia exacta del sitio original a cualquier otro host. Sin embargo, puedes importar el contenido de tus páginas, publicaciones e imágenes incrustadas utilizando las instrucciones que se encuentran en esta página.

Tu sitio ya debe ser un sitio de WordPress para migrar una copia exacta de tu sitio usando este o cualquier otro complemento. Esto significa que el complemento Migrate to DreamHost no se puede usar para migrar otros tipos de sitios, como los creados con Wix, Weebly, SquareSpace, Joomla u otros creadores de sitios.

En este momento, puedes migrar un sitio de WordPress de cualquier tamaño usando este complemento. Ten en cuenta que el tiempo para completar la migración aumenta para sitios más grandes.
