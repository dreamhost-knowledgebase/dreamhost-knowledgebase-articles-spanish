Generalidades

Si tu cuenta de DreamHost solo tiene un registro de dominio y no tiene alojamientos web, aún puedes crear un sitio web para tu nuevo registro de dominio usando Remixer.

Crear un sitio Remixer para tu registro de dominio.

  1. Navega a la página Remixer en tu panel web.
  2. Cuando el la página del panel de Remixer se abra, da clic al botón Start Using Remixer casi al final.

Esto creaun nuevo plan de Remixer para alojat tu sitio web. Puedes ahora usar las guías de soporte en la Base de Conocimientos para crear tu sitio web:

  • Guías de soporte de Remixer

Periodo de prueba gratuito

Tienes derecho a una prueba gratuita de 14 días de Remixer sin costo alguno. Durante tu prueba de Remixer, solo podrás publicar tus proyectos de Remixer en una URL gratuita de DreamHost Cloud generada automáticamente. Aparecerá algo como Este:

  • http://911dc58e-7f21-4728-8cf9-193cfef80.remixer.website

Puedes usar esta URL Cloud y previsualiza tu sitio web Remixer durante el periodo de prueba.

Actualizar

Si deseasque t sitio web Remixer aparezca en línea con el nombre de t registro de dominio, puedesvolver a visitar la URL anterior y actualizar a un plan Remixer pago. Para obtener más detalles sobre el plan y los precios de Remixer, por favor visita siguiente artículo:

  • Planes y Precios de Remixer
