Generalidades

Este artículo enumera algunos inconvenientes comúnes que puedes encontrar cuando actualizas WordPress, y algunas soluciones para resolver estos problemas.

¡Respalda tu Sitio!

Antes de continuar, asegúrate de hacer una copia de seguridad tu sitio web. Esto siempre debe hacerse antes de hacer cualquier cambio. Esto te permitirá restaurar rápidamente tu sitio si algo no funciona correctamente después de la actualización. Revisa el siguiente artículo para obtener instrucciones completas:

  • Preparar tu sitio WordPress para una actualización

El siguiente artículo de WordPress cubre inconvenientes comunes a los que puedes referirte:

  • Common WordPress Errors

Lo siguiente son algunos problemas potenciales que ocasionalmente surgen luego de una actualización:

Inconvenientes comúnes de actualización

Plugin/temas conflictivos que causan errores en la página principal

Algunos problemas que puedes ver en la página principal son:

  • white screen of death/la página blanca de la muerte
  • 'site experiencing difficulties/el sitio está experimentando dificultades'

Primero, determina qué plugin o tema está causando el problema. Puedes hacer esto de la siguiente manera:

  • Revisa el registro de errores de tu sitio: El registro de errores puede apuntar a un tema o plugin específico que causa el problema.
  • Cambia tu sitio a un tema predeterminado: Esta es una forma rápida de confirmar si el tema en sí está causando el error.
  • Apaga todos los plugins: Deshabilitar todos los pluginses una forma rápida de confirmar si un plugin está causando los errores. Luego puedes deshabilitar los plugins uno a la vez hasta que encuentres el que causa el error. Si el plugin no es una parte vital del sitio web (como un plugin sin usar), puedes dejarlo desactivado o desinstalarlo de manera segura.

Plugin WP Rollback

Si la fuente del error es una parte vital del sitio web (como el tema o un plugin importante), puedes intentar usar el siguiente plugin. Esta es una manera fácil de revertir tu sitio a la versión anterior del tema o plugin:

  • Usar el plugin WP Rollback

El panel administrativo está Inaccesible

Si no puedes iniciar sesión en tu sitio WordPress, debes revisar los registros de errores de tu sitio.

  • Ver tus registros de acceso y de errores vía SFTP
  • Ver y examinar un registro de errores vía SSH

Si los registros especifican un plugin, dirígete al directorio de plugins y desactívalo. Revisa el siguiente artículo para obtener instrucciones sobre cómo deshabilitar un plugin:

  • Deshabilitar plugins

Deshabilitar un plugin a menudo es suficiente para que el panel de administración de WordPress vuelva a funcionar para que puedas iniciar sesión.
