Generalidades

Hay varias formas de verificar si Cloudflare funciona en un sitio web.

Ver encabezados

Dado que Cloudflare inserta la información del encabezado en las solicitudes, necesitarás una forma de verificar estos encabezados para ver si Cloudflare está habilitado y funciona en st sitio web.

Al mirar estos encabezados, está buscando el siguiente encabezado específico de Cloudflare:

CF-Cache-Status: HIT
