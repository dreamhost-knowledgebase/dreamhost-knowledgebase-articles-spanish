Usando Windows

Un programa común para ssh en windows es PuTTY o putty-nd. Si seguiste las instrucciones para crear una llave de emparejamiento en el panel de DreamCompute, ya has descargado un archivo .pem. PuTTY requiere que uses este archivo .pem para convertir tu clave privada en un archivo .ppk antes de que pueda usarse para conectarse a través de ssh.

  1. Abre el programa de puttygen. Si no lo tienes, descárgalo aquí.
  2. Haz clic en el menú Conversions y selecciona Import Key.
  3. Busca la carpeta que contiene el archivo .pem que descargaste del panel de DreamCompute o la llave privada que creaste fuera del panel.
  4. Puedes asignar un nombre a la clave en el campo Key comment.
  5. Cuando termines, haz clic en el botón Save private key para guardarla en un archivo .ppk.
  6. Abre PuTTY y navega a Connection > SSH > Auth.
  7. Haz clic en el botón Browse cerca del campo Private key file for authentication.
  8. Selecciona el archivo .ppk que generaste y luego haz clic en Open.
  9. Navega a la categoría Session e ingresa la dirección IP de tu instancia en el campo Host Name. En el campo Saved Sessions, dale a esta conexión un nombre. Por ejemplo, dh-compute.
  10. Haz clic en el botón Save a la derecha para guardar esta sesión.
  11. Haz clic en el nombre de la sesión que acabas de guardar y haz clic en el botón Open.
  12. PuTTY se conecta a tu instancia y pregunta por tu nombre de usuario. 
    login as:
  13. Ingresa tu usuario y presiona Enter. Ahora has iniciado sesión en la instancia.
