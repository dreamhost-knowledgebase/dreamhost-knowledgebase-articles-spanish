Eliminar una base de datos

Puedes eliminar una base de datos creada en DreamHost a través de la página Bases de Datos MySQL.

A la derecha de la base de datos, haz clic en el botón Delete DB.

La papelera de reciclaje

Las bases de datos eliminadas se mueven a la papelera de reciclaje. Puedes acceder a esto haciendo clic en el botón MySQL Recycle Bin en la parte superior izquierda de la página. Esto permite una fácil restauración de las bases de datos que se eliminan por error.

Las bases de datos se almacenan aquí durante 30 días. Después de 30 días, se eliminan permanentemente.

Si deseas eliminar permanentemente una base de datos, puedes iniciar sesión en MySQL usando el comando y emitir una declaración de "caída de la base de datos" para acelerar el proceso.

¿Qué le sucede al usuario de la base de datos?

Si el usuario de la base de datos que eliminó NO está asignado a ninguna otra base de datos, también se eliminará junto con la base de datos.
