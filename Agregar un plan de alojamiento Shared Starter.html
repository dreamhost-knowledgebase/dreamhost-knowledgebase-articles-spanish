Generalidades

Este artículo describe qué es un plan de alojamiento Shared Starter y cómo crear una cuenta con este tipo de plan.

Preguntas Frecuentes

¿Qué es un plan de alojamiento Shared Starter?

Shared Starter es el plan de alojamiento más básico y de nivel de entrada de DreamHost. El plan Shared Starter es una excelente opción para poner en marcha un solo sitio web rápidamente a un costo muy bajo.

¿Hay alguna limitación?

Dado que el plan Shared Starter es un plan introductorio, hay algunas limitaciones que debes tener en cuenta:

  • Solo se puede alojar un sitio web (excluyendo dominios Solo DNS, espejo, estacionados o redireccionados)
  • Correo electrónico no incluido (pero puedes agregar un plan de correo pago)
  • 6 bases de datos MySQL
  • 5 subdominios
  • 50GB ode espacio en disco
  • 6 usuarios de servidor web

¿Puedo agregar un plan de alojamiento Shared Starter si solo tengo una cuenta de registro?

Si tu cuenta se abrió previamente como una cuenta solo de registro de dominio, no hay planes de alojamiento activos. Debes contactar a soporte si deseas agregar un plan de alojamiento Shared Starter.

Cómo agregar un plan de alojamiento Shared Starter

Las siguientes secciones explican cómo configurar un plan de alojamiento Shared Starter.

Crear un plan de alojamiento Shared Starter

Visita la página de alojamiento Shared Starter para crear un nuevo plan.

En Shared Starter, haz clic en el botón Regístrate Ahora y luego sigue las indicaciones para registrarte en una nueva cuenta de DreamHost. Una vez completado, tendrás acceso a tu panel de DreamHost.

Agregar tu sitio al plan

Después de agregar el plan, usa los siguientes pasos para iniciar sesión en tu panel y agregar tu sitio web al nuevo plan:

  1. Navega a la página de Usuarios & Archivos SFTP y agrega un nuevo usuario a un servidor.
  2. Una vez agregado, navega a la página de Administrar Sitios Web y agrega un sitio web y alojamiento.
  3. Después de agregar el dominio, espera unas horas para que se propague en línea. Después de eso, podrás visitarlo.

Cambiar qué dominio está alojado en un plan Shared Starter

Puedes usar los siguientes pasos para cambiar el dominio 'Completamente Alojado' de tu plan Shared Starter a otro que poseas.

Solo se puede alojar completamente un dominio a la vez con un plan Shared Starter. Sin embargo, no necesitas contactar al soporte para cambiar el dominio activo. Para cambiar el sitio web 'Completamente Alojado' en el plan:

  1. Navega a la página de Administrar Sitios Web y elimina el alojamiento del dominio actualmente activo.
  2. Después de que el dominio antiguo se configure como 'Solo DNS', agrega tu nuevo dominio como 'Completamente Alojado' en tu servidor.
  3. Abre un navegador y visita el nuevo dominio para asegurarte de que aparezca.
