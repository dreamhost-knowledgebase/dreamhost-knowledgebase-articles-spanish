Generalidades

En este artículo se explica cómo limpiar el caché en distintos navegadores. Esto resulta útil si has realizado cambios en tu sitio web, pero estos no aparecen al cargar la página.

 

Forzar una página a cargar

Antes de limpiar tu caché, primero intenta correr los siguientes comandos para forzar a la página a recargar

Windows
Ctrl + F5
Mac
CMD + R
(Command key) – En algunos teclados de Apple, esta tecla también tiene un logo () de Apple (Fuente: Funciones rápidas de teclado en Mac).
Linux
F5

Evitar tu caché

A veces, forzar una recarga no es suficiente para mostrar el nuevo contenido de tu sitio. En ese caso, intenta evitar el caché del navegador. Cuando evitas la caché, obligas a tu navegador a volver a descargar el contenido completo y actualizado de una página web. Esto debería ser suficiente para que veas los nuevos cambios. Revisa la siguiente página de Wikipedia para obtener instrucciones sobre cómo hacer esto en diferentes sistemas operativos y navegadores:

  • Evitar tu caché

Vaciar fuertemente caché para cargar (Solo para Chrome)

Chrome también permite hacer una carga fuerte. Esto puede ser usado si ves que limpiar tu caché no funciona como debe.

  1. Abre las herramientas de desarrollador.
    • Windows: Ctrl + Shift + i
    • Mac: Option + CMD + i
  2. Da clic derecho en el icono refrescar.
  3. En el menú desplegable, da clic en Vaciar caché y volver a cargar de manera forzada.
