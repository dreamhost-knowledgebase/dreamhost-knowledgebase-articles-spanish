Generalidades

Si has hecho cambios en tu sitio web pero no aparecen cuando cargas la página, puede que necesites o limpiar el caché del buscador.

 

Forzar una página a cargar

Antes de limpiar tu caché, primero intenta correr los siguientes comandos para forzar a la página a recargar

Windows
Ctrl + F5
Mac
CMD + R
(Command key) – En algunos teclados de Apple, esta tecla también tiene un logo () de Apple (Fuente: Funciones rápidas de teclado en Mac).
Linux
F5

Saltar tu caché

Algunas veces, solamente refrescando tu página no muestra el nuevo contenido del sitio. En ese caso, intenta saltar el caché del buscador. Cuando saltas tu caché, fuerzas a tu buscador a re-descargar la página web completa, el contenido actualizado. Esto debe ser suficiente para ver los nuevos cambios.

Revisa el siguiente artículo de Wikipedia para ver instrucciones de cómo hacerlo en diferentes sistemas operativos y buscadores:

  • Saltar tu caché

Limpiar el caché del buscador

Algunas veces forzar la carga no es suficiente, entonces puede que necesites limpiar el almacenamiento de tu buscador de archivos guardados. Selecciona tu buscador que usas y sigue las siguientes instrucciones para limpiar el caché.

Chrome

Revisa el siguiente enlace para ver instrucciones sobre cómo limpiar tu caché en Chrome:

  • Cómo borrar datos de navegación

Vaciar fuertemente caché para cargar (Solo para Chrome)

Chrome también permite hacer una carga fuerte. Esto puede ser usado si ves que limpiar tu caché no funciona como debe.

  1. Abre las herramientas de desarrollador.
    • Windows: Ctrl + Shift + i
    • Mac: Option + CMD + i
  2. Da clic derecho en el icono refrescar.
  3. En el menú desplegable, da clic en Vaciar caché y volver a cargar de manera forzada.
