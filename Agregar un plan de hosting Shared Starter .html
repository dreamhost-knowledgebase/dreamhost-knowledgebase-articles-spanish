Este artículo describe que es un plan de alojamiento Shared Starter y cómo crear una cuenta con este tipo de plan.

¿Qué es un plan de hosting Shared Starter?

El plan Shared Starter, es el plan más básico de alojamiento ofrecido por DreamHost.

Este plan, es una excelente opción para poner en funcionamiento un solo sitio web a un costo muy bajo. Ya que es un plan de inicio hay algunas limitaciones a tener en cuenta que se mencionan a continuación.

¿DreamHost ofrece otros planes de hosting?

Además del plan Shared Starter, también están disponibles los siguientes planes de alojamiento:

  • Shared — Tu sitio web es alojado en un servidor compartido.
  • VPS — Tu sitio web es alojado en un servidor privado.
  • Dedicados — Tu sitio web es alojado en un servidor privado.
  • DreamPress — Tu sitio web es alojado en un servidor privado corriendo una versión de WordPress administrada por DreamHost.

¿Hay alguna limitación?

Ya que el Shared Starter es un plan introductorio, hay ciertas limitaciones que debes tener en cuenta:

  • Solo se puede alojar un sitio web (excluyendo dominios de Solo DNS, Espejo, Parqueo, o Redirección).
  • No incluye correo electrónico.
  • 6 Bases de datos MySQL.
  • 5 subdominios.
  • 50GB de espacio de almacenamiento.
  • 6 usuarios web server.

¿Cómo agregar un plan Shared Starter?

Para crear un plan Shared Starter, visita la siguiente página:

  • https://www.dreamhost.com/es/hosting/shared/

Debajo de 'Shared Starter' haz clic en el botón Regístrate Ahora, y sigue las indicaciones para inscribirte a una nueva cuenta DreamHost. Una vez completado, tendrás acceso a tu panel desde:

  • panel.dreamhost.com

Agregar tu sitio web al plan

Después de que el plan ha sido agregado, inicia sesión en tu panel. Puedes ver más información sobre cómo iniciar sesión en tu panel de control de DreamHost, visitando el siguiente artículo:

  • Generalidades del panel de control de DreamHost

Para agregar tu página al nuevo plan:

  1. Navega a la página Usuarios & Archivos FTP. Usa el siguiente artículo para agregar un nuevo usuario a tu servidor.
    • Agregar un usuario a un VPS
  2. Una vez agregado, navega a la página Administrar Dominios. Usa el siguiente artículo para agregar un nuevo dominio a tu servidor.
    • ¿Cómo agregar un dominio?
  3. Después de agregar el dominio, espera unas horas mientras es propagado en línea. Una vez sea completado, podrás visitar tu sitio web.

Cambiar cual dominio está alojado con un plan de Hosting Shared Starter

Puedes usar los siguientes pasos si quieres que el plan de tu dominio en Shared Starter como 'Fully Hosted' a otro que tengas.

Solo un dominio puede ser alojado al tiempo con un plan Shared Starter. Sin embargo, no necesitas contactar a Soporte para cambiar el dominio activo. Para cambiar el sitio web a 'Fully Hosted' en el plan:

  1. Navega a la página Administrar Dominios. Ve el siguiente artículo para eliminar el alojamiento para el dominio activo actual:
    • Configura tu dominio como 'Solo DNS'
  2. Después que el dominio esté como 'Solo DNS', ve el siguiente artículo para agregar tu nuevo dominio como 'Completamente Alojado' en tu servidor.
    • ¿Cómo agregar un dominio?

Agregar un plan Shared Starter si ya tienes una cuenta de solo registros de dominio

Si tu cuenta fue previamente creada como solo registros de dominio, no tienes ningún plan de alojamiento activo. Contacta el equipo de soporte si quieres agregar un plan de alojamiento
Shared Starter:

  • Contactar a Soporte

O puedes agregar cualquier otro plan de alojamiento que te guste, incluyendo:

  • Shared — Haz clic en el link Upgrade to Shared Unlimited debajo de tu plan actual.
  • VPS — Visita el artículo Agregar un servidor privado.
  • Dedicado — Visita el artículo Agregar un Servidor Dedicado.
  • DreamPress — Visita los artículos Instalar DreamPress.
