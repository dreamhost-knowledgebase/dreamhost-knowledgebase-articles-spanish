El módulo cuenta de API contiene comandos para verificar el estado o el uso de tu cuenta.

Comandos

account-domain_usage

Enumera el uso de ancho de banda para todos los servicios de dominio visibles. El uso del ancho de banda está en bytes desde el comienzo del ciclo de facturación actual (que se puede determinar usando el estado de la cuenta).

Comando account-domain_usage
Valores

(ninguno)
Resultado

success
domain type bw
example.com http 1048576
Posibles errores

(ninguno)

account-list_keys

Muestra todas las claves públicas SSH de esta cuenta. Las claves SSH se pueden utilizar para proporcionar acceso de root a cualquiera de tus VPS o Servidores Dedicados.

(ninguno)

account-user_usage

Enumera el uso del ancho de banda y del disco para todos los usuarios visibles. El uso del disco está en kilobytes; el uso del ancho de banda está en bytes desde el inicio del ciclo de facturación (consulta el estado de la cuenta para determinar cuándo es). En la mayoría de los casos, el uso de ancho de banda aparece como cero aquí. Consulta account-domain_usage para conocer el ancho de banda del dominio.

Comando account-user_usage
Valores

(ninguno)
Resultado

success
user disk disk_as_of bw
username 4700.56 2009-12-22 9.412
Posibles errores

(ninguno)
