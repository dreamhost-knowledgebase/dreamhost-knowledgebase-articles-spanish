Generalidades

Este artículo proporciona información sobre cómo transferir un registro de dominio entre cuentas de DreamHost.

Antecedentes

El proceso de transferencia entre cuentas de DreamHost es muy similar a la transferencia de un dominio desde un proveedor externo. Revisa este artículo para obtener más información sobre este proceso.

Tarifas

Al transferir entre cuentas DreamHost, la cuenta receptora paga la tarifa de transferencia, que extiende la fecha de vencimiento del dominio por un año. Revisa esta página para obtener una lista de las tarifas de transferencia.

Dominios internacionales

Algunos dominios internacionales no se pueden transferir mediante este proceso. La cuenta receptora debe contactar al soporte de DreamHost para obtener más ayuda.
