Generalidades

Lo siguiente describe cómo ver el uso de tu Servidor Dedicado en el panel. Esto es importante para ayudarte a llevar un seguimiento de la cantidad de recursos que estás utilizando y evitar alcanzar cualquier límite. 

Ver el uso del disco

Para acceder a la página Uso de Disco:

  1. Navega a la página Administrar Cuenta.
  2. Baja a la sección Otros Servicios Activos.
  3. Haz clic en el enlace Ver Uso de Disco para abrir la página.

En la parte superior, aparece un resumen general, seguido por las estadísticas de us de los Usuarios, Dominios, Correo, y Bases de datos.

Revisar el uso de RAM en tiempo real

El panel no muestra el uso de RAM en tiempo real para tu Servidor Dedicado. Para ver esta información, necesitas iniciar sesión en tu servidor a través de SSH y correr los comandos que se encuentran en el siguiente artículo.

  • Usando el comando 'ps' para solucionar problemas de tu sitio web

Por favor, ten en cuenta que, aunque el artículo hace referencia a una máquina VPS, aún puedes usar esos comandos en tu Servidor Dedicado.
