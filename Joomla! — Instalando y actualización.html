Generalidades

Joomla! es un software gratuito de Sistema de administración de contenido (CMS) que puedes utilizar para crear un sitio web. Una vez instalado, podrás iniciar sesión en el backend de tu sitio para comenzar a personalizarlo como desees. Joomla! también proporciona un Directorio de Extensiones donde puedes descargar nuevos temas y plugins para personalizarlo aún más.

  • Directorio de muestras Joomla!

Paso 1 — Instala un certificado SSL

Todos los sitios web modernos deben estar protegidos con un certificado SSL, ya que garantiza que todo el tráfico hacia y desde tu sitio sea seguro. Hay dos opciones de certificado SSL en el panel DreamHost:

  • Let's Encrypt (gratuito)
  • Sectigo (pago)

Ambos certificados se pueden utilizar para proteger tu sitio. Sin embargo, si vas a vender productos en tu sitio, se recomienda un certificado Sectigo pagado, ya que parecerá más confiable para los clientes potenciales. Consulta el siguiente artículo para obtener más detalles:

  • ¿Qué tipo de certificado debería usar para mi sitio de comercio electrónico?

Ve el siguiente artículo para instrucciones como instalar el certificado que elegiste:

  • Generalidades sobre añadir un certificado SSL

Paso 3 — Descarga Joomla!

Hay dos opciones para descargar el software de Joomla! a tu sitio.

  • FTP (fácil) — Descarga el software en tu computadora, luego cárgalo en tu sitio usando un cliente FTP.
  • SSH (avanzado) — Descarga el software directamente en tu sitio a través de SSH.

Descargando a través de FTP

  1. Visita la siguiente página para descargar el software de Joomla!:
    • Descargar Joomla! 3.9.20
  2. Haz clic en el enlace para descargar el archivo .zip a tu computador.
  3. Busca el archivo que acabas de descargar. La mayoría de las veces puedes encontrarlo en tu directorio /downloads y se titula algo como Joomla_3.9.20-Stable-Full_Package.zip.
  4. En Windows, haz clic derecho en el archivo y elige Extract All... en el menú emergente.
  5. En la nueva ventana que se abre, haz clic en el botón Browse para seleccionar dónde deseas guardar la carpeta de instalación de Joomla!.
  6. Cuando estés listo, haz clic en el botón Extract. Todo el contenido se extrae a una nueva carpeta titulada igual que el archivo .zip.
  7. Inicia sesión en tu servidor usando un cliente FTP.
  8. En el panel izquierdo, navega hasta el directorio de tu computadora en el que extrajiste el archivo .zip. Deberías ver una lista de archivos.
  9. En el panel derecho, asegúrate de estar en el directorio principal de tu sitio web. No debería haber ningún archivo aquí que no sea un posible archivo favicon.ico.
  10. Resalta todos los archivos en el panel izquierdo y arrástralos al panel derecho para subirlos a tu sitio.

    Esto podría llevar varios minutos (o más, dependiendo de la velocidad de tu conexión a Internet), ya que hay muchos archivos para cargar.

Descargando a través de SSH

  1. Inicia sesión en tu servidor a través de SSH.
  2. Navega al directorio de tu sitio. Cambia username a tu usuario Shell y example.com a tu sitio website. 
    [server]$ cd /home/username/example.com
  3. Visita la siguiente página para descargar el software de Joomla!:
    • Descargar Joomla! 3.9.20
  4. Haz clic derecho en el botón de descarga. En el menú emergente, selecciona Copy link address.
  5. Ejecuta el siguiente comando para descargar Joomla, que descarga el archivo zip de instalación de Joomla!:

    Asegúrate de cambiar la URL por la que copiaste arriba.

    [server]$ wget https://downloads.joomla.org/us/cms/joomla3/3-9-20/Joomla_3-9-20-Stable-Full_Package.zip?format=zip
  6. Ejecuta el siguiente comando para descomprimir el archivo: 
    [server]$ unzip 'Joomla_3-9-20-Stable-Full_Package.zip?format=zip'

    Todos los archivos de instalación de Joomla! se extraen y guardan en el directorio principal de tu sitio web.

  7. Verifica el directorio principal de tu sitio web para confirmar que los archivos se hayan migrado. Cambia username a tu usuario Shell y example.com a tu sitio website. 
    [server]$ ls -la /home/username/example.com

Paso 4 — Instala Joomla!

Joomla! recomienda usar PHP versión 7.3+. Estas instrucciones se probaron en todos los diferentes tipos de servidores en DreamHost, utilizando PHP versiones 7.2, 7.3 y 7.4 (CGI y FastCGI).

Consulta el siguiente artículo para obtener información más detallada sobre lo que Joomla! soporta:

  • Requerimientos técnicos de Joomla!

Ahora que los archivos están en el directorio de tu sitio web, puedes completar la instalación. Las siguientes instrucciones describen cómo hacerlo.

  1. Visita tu sitio web en un navegador.
    La página de instalación de Joomla! aparece:
  2. Ingresa lo siguiente en la pestaña Configuration:
    • Select Language — El idioma principal de tu sitio Joomla!. El idioma predeterminado aparece en la lista desplegable, que normalmente se lee en tu navegador.
    • Site Name — Requerido. Un nombre descriptivo para tu sitio.
    • Description — Opcional.
    • Email — Requerido. Una dirección de correo electrónico, que se utiliza para comunicar cualquier mensaje que se genere desde tu sitio Joomla!.
    • Username — Requerido. El nombre de usuario de tu sitio web, como administrador, webmaster o incluso un nombre.
    • Password — Requerido. Se recomienda una contraseña segura.
    • Confirm Password — Requerido. Ingresa tu contraseña de nuevo.
    • Site Offline — La configuración predeterminada es No, lo que significa que tu sitio estará en línea cuando finalice la instalación. En su lugar, selecciona Yes si deseas poner el sitio fuera de línea cuando haya terminado. Puedes ajustar esto más tarde a través del menú Global Configuration en Joomla.
  3. Cuando finalices, haz clic en el botón Next.
  4. Ingresa lo siguiente en la pestaña Database:

    Asegúrate de configurar una base de datos MySQL antes de continuar, como se describe en el paso 2 (arriba). Puedes encontrar más detalles en la página MySQL Databases en tu panel.

    • Database Type — Requerido. Puedes usar la configuración predeterminada, MySQLi.
    • Host Name — Requerido. El hostname para el servidor MySQL, típicamente en el formato mysql.example.com. NO uses el predeterminado localhost.
    • Username — Requerido. El nombre de usuario de la base de datos.
    • Password — Requerido. La contraseña del usuario de la base de datos.
    • Database Name — Requerido. El nombre de la base de datos en el servidor.
    • Table Prefix — Requerido. Puedes usar el generado aleatoriamente o ingresar un nuevo prefijo de tabla.
    • Old Database Process — Requerido. Joomla! te permite hacer una copia de seguridad o eliminar instalaciones de tablas Joomla!
  5. Cuando finalices haz clic en el botón Next.
    El siguiente mensaje aparece después de hacer clic en el botón Next, que indica que la base de datos MySQL no se encuentra en tu servidor local (esto es lo esperado):
  6. Para continuar, simplemente elimina el archivo .txt mencionado en el mensaje de error. En este ejemplo, el archivo se llama _JoomlaNGIqTFnEEWsp3aeADgdHE.txt y está ubicado en la carpeta de installation. Joomla! utiliza este método para garantizar la seguridad al verificar la propiedad adecuada de la base de datos.
  7. Después de eliminar el archivo .txt, vuelve a ingresar la contraseña de usuario de la base de datos y luego haz clic en el botón Next para continuar.
    Después de que Joomla! verifica que tienes las credenciales adecuadas para acceder a la base de datos, aparece la pestaña Overview:
  8. Revisa la información en la pestaña Overview, especialmente:
    • Install Sample Data — Selecciona una plantilla. El Default English (GB) Sample Data es una opción popular.
    • Email Configuration — Al seleccionar Yes, Joomla! envía una copia por correo electrónico de todas las configuraciones de tu sitio, incluida la contraseña.
    • Other Configurations — Asegúrate que la configuración bajo Main Configuration, Database Configuration, Pre-Installation Check, y Recommended settings son satisfactorias. Si es necesario, puedes realizar cambios en tu plan de alojamiento o en la instalación de Joomla! en sí, al hacer clic en el botón Previous.
  9. Cuando hayas finalizado, haz clic en el botón Install.
    Después de Joomla! se instala correctamente, aparece el siguiente mensaje:
  10. Haz clic en el botón Remove "installation" folder. Debes eliminar esta carpeta antes de continuar, ya que Joomla! requiere esta acción para mejorar la seguridad en tu sitio.
    Después de hacer clic en el botón, aparece el siguiente mensaje que indica que la carpeta se eliminó:
  11. Haz clic en el botón Site para ver tu sitio en vivo, o haz clic en el botón Administrator para iniciar sesión en el panel de administración de tu instalación Joomla!.

¡Felicidades! ¡Has instalado correctamente el software de Joomla!.

Actualización de Joomla!

Actualizaciones de tu instalación Joomla! se pueden realizar fácilmente desde el panel de control del sitio.

  1. Inicia sesión en tu panel de control Joomla!.
    El panel muestra un mensaje cuando hay una actualización disponible para tu instalación:
  2. Haz clic en el botón Update Now.
    Aparece una página con los detalles de la actualización:
  3. En la página siguiente, haz clic en el botón Install the Update.
    Aparece un mensaje de confirmación cuando se completa la actualización:
  4. Espera mientras se ejecuta la actualización. Cuando el mensaje "Your site has been updated" (Tu sitio ha sido actualizado) aparece el mensaje, la actualización está completa.
