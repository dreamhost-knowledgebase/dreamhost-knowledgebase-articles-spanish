Generalidades

Para mostrar un sitio web en Internet, necesitas hacer dos cosas:

  • Comprar el registro de dominio
  • Agregar alojamiento al dominio

Este artículo lo guía a través de las diferentes opciones en el panel para agregar un dominio. Los pasos son diferentes dependiendo de si ya has registrado el dominio o si estás agregando un subdominio. En este artículo se tratan los siguientes temas.

  • Opción 1 — Ingresa un dominio o subdominio que ya posees
  • Opción 2 — Crear un subdominio
  • Opción 3 — Registrar un nuevo dominio
  • Opción 4 — Crear un Subdominio DreamHost gratuito (subdominio dreamhosters.com)

Errores de solución de problemas

Es posible que puedas encontrar un error cuando agregas un dominio. Si es así, por favor revisa el siguiente artículo para ver más soluciones comunes:

  • Solucionar problemas al agregar un sitio web
 

Agregar un dominio

Los siguientes pasos describen como agregar un dominio a un plan de alojamiento o registrar un dominio nuevo.

  1. Navega a la página Administrar Sitios Web.
  2. En la parte superior derecha, da clic en el botón Añadir un Sitio Web.
  3. Haz una selección de las siguientes cuatro opciones:
    • Ingresa un dominio o subdominio que ya posees: Selecciona esta opción si ya registraste tu dominio.
    • Crea un subdominio: Selecciona esta opción para crear un subdominio de un dominio que ya has registrado.
    • Registra un nuevo dominio: Selecciona esta opción para registrar tu nombre de dominio primero.
    • Crear un Subdominio DreamHost gratuito: Selecciona esta opción para crear un dominio dreamhosters.com gratuito.
 

Opción 1 — Ingresa un dominio o subdominio que ya posees

  1. Ingresa un nombre de dominio que ya has registrado. Si está registrado en otra compañía, selecciona la caja para confirmar que eres el dueño de este nombre de dominio.
  2. En la parte inferior derecha, da clic en el botón Siguiente: Alojamiento.
    Aparece una lista de los planes de alojamiento.
  3. A la derecha del plan al que deseas agregar este dominio, da clic en el botón Seleccionar.
  4. En la parte inferior derecha, da clic en el botón Configurar Sitio Web para agregar el alojamiento al dominio.
    Se muestra un mensaje exitoso confirmando que el sitio ha sido añadido al plan de alojamiento.

Apuntar el DNS a DreamHost

El sitio web ahora esta alojado en los servidores de DreamHost, pero a menos de que el DNS esté apuntado a DreamHost, este no resolverá en línea. Revisa las siguientes secciones dependiendo de donde hayas comprado tu registro de dominio.

Si has comprado un registro con DreamHost

Si has comprado el registro en DreamHost, el DNS se configura correctamente (a menos de que lo hayas personalizado).

Si compraste el registro en otra compañía

Necesitarás apuntar el DNS a DreamHost. Revisa el siguiente artículo para más detalles:

  • ¿A qué registros DNS debo apuntar mi sitio?
 

Opción 2 — Crear un subdominio

  1. Ingresa tu subdominio en el primer campo de texto. Por ejemplo, blog o foro.
  2. Del menú desplegable Dominio principal, selecciona un dominio existente en tu cuenta.
  3. En la parte inferior derecha, da clic en el botón Siguiente: Alojamiento.
    Aparece una lista de los planes de alojamiento.
  4. A la derecha del plan al que deseas agregar este subdominio, da clic en el botón Seleccionar.
  5. En la parte inferior derecha, da clic en el botón Configurar Sitio Web.
    Se muestra un mensaje un mensaje exitoso confirmando que el sitio ha sido añadido al plan de alojamiento.
 

Opción 3 — Registra un nuevo dominio

La opción Registra un nuevo te lleva a la página Administrar Registros de Dominio donde puedes comprar el registro para el dominio. Revisa el siguiente artículo para más detalles.

  • Cómo comprar un registro de dominio

Una vez comprado, puedes entonces añadir alojamiento al dominio.

 

Opción 4 — Crear un Subdominio DreamHost gratuito

Selecciona la opción Crear un Subdominio DreamHost gratuito para crear un Usar un subdominio .dreamhosters.com.

  1. Ingresar tu subdominio en el primer campo de texto. Por ejemplo, blog o foro.
  2. En la parte inferior derecha, da clic en el botón Siguiente: Alojamiento.
    Aparece una lista de los planes de alojamiento.
  3. A la derecha del plan al que deseas agregar el dominio, da clic en el botón Seleccionar.
  4. En la parte inferior derecha da clic en el botón Siguiente: Opciones.
  5. Esta página te permite elegir varias opciones para agregar a tu sitio web (Ejemplo: instalar WordPress). Haz clic en el botón Seleccionar solo si deseas agregar WordPress.
  6. En la parte inferior derecha, da clic en el botón Configurar Sitio Web.
    Aparece un mensaje exitoso que confirma que el sitio se ha agregado al plan de alojamiento.

Después de agregar un dominio o subdominio, el DNS debe propagarse, lo que puede demorar de 4 a 6 horas. Para obtener más información sobre la propagación de DNS, revisa el siguiente artículo:

  • Generalidades de la propagación de DNS

Siguientes Pasos

En este momento, tu sitio web está alojado y se resuelve en línea, pero si lo visitas en un navegador, verá una página 'Coming Soon' generada por DreamHost. Ahora debes subir el contenido de tu sitio web en el servidor de DreamHost.

Si ya tienes un sitio web creado

Si ya tienes un sitio web en otra compañía de alojamiento o uno que creaste en el computador de tu hogar, simplemente puedes subir el contenido a tu servidor a través de FTP. Revisa el siguiente artículo para obtener instrucciones sobre cómo subir tanto los archivos de tu sitio web como la base de datos.

  • ¿Cómo subir mi página web a DreamHost?

Si no tienes un sitio web

Si necesitas crear un sitio web nuevo, puedes instalar WordPress en el panel DreamHost. Esto te permite crear un sitio web sin necesidad de escribir código. Revisa los siguientes artículos para obtener más información:

  • Generalidades de WordPress
  • Instalar WordPress usando el instalador de WordPress de DreamHost
