Generalidades

Este artículo lo guía para agregar un dominio. Los pasos son diferentes dependiendo de si ya ha registrado el dominio o si está agregando un subdominio. En este artículo se tratan los siguientes temas.

Añadir un dominio

  • Opción 1 — Agregar alojamiento para un dominio de tu propiedad
  • Opción 2 — Agregar un subdominio de tu propiedad
  • Opción 3 — Registrar un nuevo dominio

Añadir un dominio

Los siguientes pasos añaden un dominio a un plan de alojamiento.

  1. Navega a la página Websites.
  2. Haz clic en el botón Add Website en la izquierda superior.
  3. Aparecen tres opciones.
    • Ingresa un dominio o subdominio de tu propiedad: Selecciona esta opción si ya registraste tu dominio.
    • Crea un subdominio: Selecciona esta opción para crear un subdominio de un dominio que ya has registrado.
    • Registra un nuevo dominio: Selecciona esta opción para registrar tu nombre de dominio primero.

Opción 1 — Ingresa un dominio o subdominio de tu propiedad

  1. Ingresa un nombre de dominio que ya has registrado. Si está registrado en otra compañía, selecciona la caja para confirmar que eres el dueño de este nombre de dominio.
  2. Haz clic en el botón Next: Hosting en la parte inferior derecha.
    Una lista de planes de alojamiento aparece.
  3. Haz clic en el botón Select a la derecha de un plan al que deseas añadir un dominio.
  4. Haz clic en el botón Next: Options en la derecha inferior.
  5. La siguiente página te permite escoger algunas opciones para añadir a tu sitio web. Por ejemplo, instalar WordPress. Haz clic en el botón Select solo si deseas añadir WordPress.
  6. Haz clic en el botón Set Up Website en la parte inferior derecha para añadir alojamiento al dominio.
    Un mensaje de éxito se muestra confirmando que el sitio ha sido añadido al plan de alojamiento.

Opción 2 — Crea un subdominio

  1. Ingresa tu subdominio en el primer campo de texto. Por ejemplo, blog o foro.
  2. Selecciona un dominio existente en tu cuenta del menú desplegable Parent domain.
  3. Haz clic en el botón Next: Options en la parte inferior derecha.
  4. La siguiente página te permite escoger algunas opciones para añadir a tu sitio web. Por ejemplo, instalar WordPress. Haz clic en el botón Select solo si deseas añadir WordPress.
  5. Haz clic en el botón Set Up Website en la parte superior derecha.
    Un mensaje de éxito se muestra confirmando que el sitio ha sido añadido al plan de alojamiento.

Opción 3 — Registra un nuevo dominio

Esta opción te lleva a la página Registrations donde puedes comprar el registro de dominio. Ve el siguiente artículo para más detalles.

  • Cómo comprar un registro de dominio

Una vez comprado, puedes entonces añadir alojamiento al dominio.
