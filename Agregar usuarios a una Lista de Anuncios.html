Agregar usuarios

  1. Navega hasta la página Lista de anuncios.
  2. Da clic en el botón Editar al extremo derecho de tu lista.
  3. Desplázate hacia abajo a la sección Editar lista de Suscriptores.
  4. Edita la lista. Los campos disponibles para editar son:
    • Lista actual de suscriptores (): Muestra las direcciones de correo de aquellos que ya están suscritos a la lista. El paréntesis vacío se completa con el número de usuarios en esta lista. Este ejemplo no muestra usuarios agregados a la lista.
    • Correos para agregar: Para agregar una dirección de correo, ingresa la dirección de correo completa en el campo provisto. Asegúrate de agregar sólo una dirección de correo por línea.
    • Correos para quitar: Ingresa la dirección de correo que deseas eliminar permanentemente de tu lista.
  5. Da clic en el botón Modificar suscriptores... para guardar.
  6. Da clic ¡Enviar Confirmaciones Ahora! para enviar la solicitud.

    Correos de confirmación de suscripción:

    • Se debe responder para verificar una suscripción.
    • Se envían por defecto desde donotreply@dreamhost.com, que no se puede modificar. Solo los anuncios enviados a los suscriptores actuales se enviarán desde la dirección de la lista, no desde el correo de confirmación inicial.

El usuario luego recibe un correo con un enlace para unirse a la lista.

Si a un usuario ya se le ha enviado un mensaje de confirmación para la lista en las últimas 48 horas, no se puede volver a agregar a la lista hasta que haya expirado ese período de 48 horas. Esto es para evitar que el sistema envíe spam a los usuarios con avisos de suscripción potencialmente no deseados.

Ver/editar detalles del usuario

Una vez que un usuario acepta ser agregado a la lista, los verá enumerados en la sección Editar lista de Suscriptores al lado de las palabras 'Current Subscriber List':

Aparecen tres opciones a la izquierda:

  • Mostrar fechas.
  • Mostrar nombres.
  • Mostrar rebotes.

En la captura de pantalla anterior, los tres ya están seleccionados, lo que muestra la información del usuario a la derecha de tu dirección de correo.

Como puedes ver en la captura de pantalla anterior, el nombre no aparece. Desafortunadamente, no hay forma de editar un usuario existente para cambiar el nombre. Si necesitas un nombre diferente, debes eliminar el usuario y luego agregarlo nuevamente como se muestra en la siguiente sección.

¿Cómo puedo asignar un nombre a un usuario?

Cuando agregas el usuario al campo 'Emails to Add' Por primera vez, puedes especificar su nombre en este formato:

email@example.com John Doe

Solo puedes cambiar el nombre cuando se agregó originalmente. Si deseas cambiarlo más tarde, elimina la dirección y luego agregarla nuevamente con un nuevo nombre.

¿Cuántos correos electrónicos se pueden agregar a la lista?

No hay límite para la cantidad de personas que puedes agregar a una lista. Esa es la razón por la cual se debe utilizar el servicio de lista en lugar de enviar un correo de alta velocidad a múltiples contactos a través de tu dirección de correo estándar que tiene límites de cuota. Las limitaciones de SMTP evitan esto con tu dirección de correo habitual y otros proveedores de correo también pueden tener restricciones de correo saliente. Por lo tanto, el servicio de listas es ideal para enviar un solo correo a una gran cantidad de suscriptores.

Sin embargo, ten en cuenta que puede llevar un tiempo que todos los mensajes salgan si tiene una lista muy grande.

Si un mensaje enviado para la lista no ha sido entregado y ha pasado más de unas horas, no reenvíes el mensaje Contacta a soporte para que se puedan verificar los registros de correo.
