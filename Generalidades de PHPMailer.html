Generalidades

Este artículo describe información general sobre PHPMailer

Antecedentes

PHPMailer es una potente biblioteca para crear y enviar correos a través de SMTP en PHP. Admite una variedad de funciones como archivos adjuntos, formato HTML, y validación del lado del servidor. Para obtener más detalles, visita la página de GitHub de PHPMailer.

Cómo instalarlo

Puedes instalar PHPMailer en un servidor compartido o VPS/Dedicado.

Los correos masivos de cualquier tipo enviados desde una cuenta DreamHost están sujetos a la política de spam de DreamHost, que forma parte de los términos de servicio. Por favor lee esas páginas junto con la cuota de SMTP para asegurarte de no infringir ninguna de estas políticas.

Permitir a GMAIL enviar correo desde tu aplicación

Si usas tu dirección de GMAIL para enviar mensajes a través de SMTP, primero debes permitir que tu aplicación acceda a la dirección de GMAIL. Si no lo haces, tu correo no se autenticará ni se enviará. Revisa este artículo para obtener más detalles.
