Generalidades

PHPMailer es una clase de creación y transporte de correo electrónico con todas las funciones para PHP que te permite enviar correos electrónicos a través de SMTP. Visita la página de PHPMailer’s GitHub para más detalles sobre lo que tiene para ofrecer.

Nota importante sobre el correo masivo

El correo electrónico masivo de cualquier tipo enviado desde una cuenta de DreamHost está sujeto a la la política de spam de DreamHost, que forma parte de los términos de servicio. Lee éstas páginas junto con la SMTP quota para asegurarte de no infringir ninguna de estas políticas.

Solución de problemas

Errores de certificado usando PHP 5.6

PHP 5.6 maneja los certificados de manera diferente a las versiones anteriores como se explica en la siguiente página:

  • http://php.net/manual/en/migration56.openssl.php

Si usas un nombre de host de correo incorrecto, puedes ver el siguiente error al intentar enviar un correo electrónico:

Warning: stream_socket_enable_crypto(): Peer certificate CN=`*.mail.dreamhost.com' did not match expected CN=`mail.example.com'

Esto se debe al nombre de host que se está utilizando. Ve el siguiente artículo para obtener una solución detallada:

  • Errores de certificado usando PHP 5.6

Permitir que GMAIL envíe desde tu aplicación de correo

Si estás utilizando su dirección de GMAIL para enviar a través de SMTP, primero debes permitir que tu aplicación acceda a tu dirección de GMAIL. Si no haces esto, su correo electrónico no se autenticará y no se enviará. Ve el siguiente artículo para más detalles:

  • Solución de problemas de errores de autenticación SMTP de GMAIL
