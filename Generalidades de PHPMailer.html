Generalidades

PHPMailer es una clase de creación y transporte de correo con todas las funciones para PHP que te permite enviar correos a través de SMTP. Visita la página de PHPMailer’s GitHub para más detalles sobre lo que tiene para ofrecer.

Nota importante sobre el correo masivo

El correo masivo de cualquier tipo enviado desde una cuenta de DreamHost está sujeto a la la política de spam de DreamHost, que forma parte de los términos de servicio. Lee éstas páginas junto con la límites de cuota SMTP para asegurarte de no infringir ninguna de estas políticas.

Permitir que GMAIL envíe desde tu aplicación de correo

Si estás usando tu dirección de GMAIL para enviar a través de SMTP, primero debes permitir que tu aplicación acceda a tu dirección de GMAIL. Si no haces esto, tu correo no se autenticará y no se enviará. Mira el siguiente artículo para más detalles:

  • Solución de problemas de errores de autenticación SMTP de GMAIL
