Generalidades

git está instalado en todos los servidores DreamHost de forma predeterminada, pero si deseas ejecutar la versión más reciente, puedes instalarla localmente con tu nombre de usuario. Este artículo te guía a través de cómo instalar una versión personalizada de git que tu Shell usará cuando inicies sesión vía SSH.

Instalar git

  1. Inicia sesión en tu servidor DreamHost vía SSH.
  2. Visita la página de descarga de Linux.
    • Descarga Git
  3. Hay un enlace en el primer párrafo a la versión más reciente. Haz clic derecho aquí y elige Copiar Dirección de Enlace.
  4. Asegúrate de estar en el directorio home del usuario. 
    [server]$ cd ~
  5. Corre el siguiente comando para descargar git.

    Cambia la versión al enlace que copiaste.

    [server]$ wget  https://www.kernel.org/pub/software/scm/git/git-2.34.1.tar.gz
  6. Decomprime el archivo. 
    [server]$ tar -zxf git-2.34.1.tar.gz

    Esto crea un nuevo directorio con el nombre del número de versión.

  7. Cambia a este nuevo directorio. 
    [server]$ cd git-2.34.1
  8. Corre los siguientes comandos uno a la vez para instalar. Asegúrate de cambiar el nombre de usuario a tu usuario SSH. 
    [server]$ make prefix=/home/username/local all
[server]$ make prefix=/home/username install
[server]$ exec bash
  9. Corre el siguiente comando para confirmar que se estás usando la nueva versión de git. 
    [server]$ git --version

Actualiza tu .bash_profile

Ahora que git está instalado, debes actualizar tu .bash_profile para cargarlo automáticamente cuando inicies sesión en tu servidor a través de SSH.

  1. Inicia sesión en tu servidor a través de SSH.
  2. Edita el archivo .bash_profile para agregar la siguiente línea. 
    [server]$ export PATH=$HOME/local/bin:$PATH

Cuando inicies sesión en tu servidor en el futuro, la nueva versión de git se cargará automáticamente.
