Manage Domains

La página Manage Domains te permite administrar las siguientes funciones para tus dominios.

La página 'Manage Domains' te permite administrar el alojamiento de tu dominio. Aquí puedes agregar alojamiento y ajustar la configuración del dominio, y puedes restaurar los archivos web respaldados.

Ver el siguiente artículo para más detalles:

  • Generalidades del Panel de Dominios

Websites

La página de Websites proporciona una presentación gráfica mejorada de todos tus dominios, ofreciéndote una vista simplificada donde puedes realizar fácilmente ajustes de funciones y ajustes a cualquiera de tus dominios en DreamHost.

Ve el siguiente artículos para más detalles:

  • Uso de la página del panel 'Websites'
  • Uso de la nueva página Administrar sitios web

Registrations

La página Registrations es donde puedes registrar un nuevo dominio y administrar los registros de dominio actuales que tienes con DreamHost. Las modificaciones de dominio WHOIS (incluyendo nameservers) se pueden hacer desde aquí, así como la configuración de renovación de registro y el bloqueo de dominio.

Ver el siguiente artículo para más detalles:

  • Registro de Dominio

Reg. Transfer

La página Reg. Transfer es donde puedes transferir un registro de dominio hacia o desde DreamHost, así como obtener el código de autorización del dominio.

Vea los siguientes artículos para más detalles:

  • Transferir tu registro de dominio  fuera de DreamHost
  • Transferir tu registro de dominio a DreamHost

SSL/TLS Certificates

En la página SSL/TLS Certificates, puedes agregar un certificado SSL a tu dominio, lo que permite acceder al dominio utilizando el protocolo HTTPS.

Para más información ver el siguiente artículo:

  • Generalidades de certificados SSL
