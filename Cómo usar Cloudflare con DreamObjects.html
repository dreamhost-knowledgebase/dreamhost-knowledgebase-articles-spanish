Generalidades

Este artículo explica cómo configurar un bucket de DreamObjects para que tu contenido se pueda acelerar y proteger con Cloudflare.

Requisitos previos

Este artículo asume lo siguiente sobre la configuración de tu sitio web:

  • Tu domino está registrado con DreamHost.
  • Tienes una cuenta de DreamObjects configurada.
 

Paso 1 — Agrega tu dominio a tu cuenta Cloudflare

Revisa el siguiente artículo para ver instrucciones de como crear una cuenta gratuita con Cloudflare:

  • Crear una cuenta de Cloudflare gratuita

Después de que hayas agregado tu dominio, Cloudflare asigna tus nameservers específicos. Ellos aparecerán así:

Continua siguiendo las instrucciones en el artículo de arriba para actualizar los nameservers de tu dominio en DreamHost para usar estos nuevos nameservers de Cloudflare.

Paso 2 — Crear un bucket en tu panel de DreamObjects

Revisa el siguiente artículo para ver instrucciones de como crear un nuevo bucket en tu panel de DreamObjects:

  • Buckets de DreamObjects

El nuevo bucket deberá ser llamado algo como micdn.example.com. Esto es como se vería luego de agregarlo a tu panel de DreamObjects:

Puedes nombrar el bucket como desees, pero el nombre debe ser el mismo que le das a tu subdominio en el panel de Cloudflare. Este nombre también debe ser un nombre de dominio completo (FQDN).

Por ejemplo, si tu nuevo bucket se llama minuevobucket.example.com, la entrada de DNS que crees en el panel de Cloudflare debería ser minuevobucket.example.com.

Este ejemplo usa micdn.example.com.

Migrar un bucket existente

Si tienes un bucket existente que te gustaría usar con esta solución, deberás "cambiar el nombre" del bucket migrando su contenido. Mira el siguiente artículo para obtener instrucciones sobre cómo hacerlo:

  • Renombrar un bucket usando la herramienta de migración del panel

Paso 3 — Crea un registro CNAME en tu cuenta de Cloudflare

Revisa el siguiente artículo para ver instrucciones de cómo agregar agregar un registro DNS en tu panel DNS Cloudflare:

  • Actualizar los registros de DNS en Cloudflare

Crea un registro CNAME de DNS con las siguientes configuraciones. Asegúrate de cambiar micdn al nombre de tu bucket y example.com al nombre de tu dominio:

  • Tipo: En el menú desplegable, selecciona CNAME
  • Nombre: micdn
  • Destino: micdn.example.com.objects-us-east-1.dream.io

Da clic en el botón Guardar para guardar este registro de DNS nuevo.

Para obtener más información sobre cómo agregar registros CNAME en Cloudflare, consulta su documentación aquí:

  • Configuración de un CNAME

¿Cuánto debo esperar hasta que funcione?

Tan pronto como se propague la información de DNS, se podrá acceder al contenido de tu bucket mediante el registro de subdominio que creaste.

Por ejemplo, si has cargado una imagen a tu bucket llamado imagen.png, puedes verla ahora en:

  • micdn.example.com/imagen.png
