Generalidades

La extensión intl se habilita por defecto en DreamHost. Puedes ver para confirmar si la extensión está habilitada creando un archivophpinfo.php. También deberías ver lo siguiente cuando visites esta página en tu sitio web:

Error que puedes ver si esto no está instalado

Si falta esta extensión, tu sitio puede arrojar diferentes errores. Aquí hay un error que puede aparecer en WordPress.

Fatal error: Uncaught Error: Class 'NumberFormatter' not found

Si notas cualquier error en tu sitio relacionado con esta extensión, por favor Contacta a Soporte para más asistencia.
