To edit the httpd.conf file, your user must have admin access to the server. However, admin access is not available on VPS machines. If you need to edit this file, the only option is to upgrade to a Dedicated server and add an admin user.

Location of the httpd.conf file

On your Dedicated server, you can find this file in the following location:

/dh/apache2/httpd-INSTANCE/etc/httpd.conf
