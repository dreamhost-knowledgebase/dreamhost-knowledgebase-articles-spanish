Generalidades

A continuación se describe cómo realizar cambios rápidamente en tu base de datos MySQL en DreamPress.

Opción 1 — Iniciar sesión automáticamente en phpMyAdmin

  1. Navega a la página Administrar Sitios Web.
  2. Da clic en el botón Administrar para abrir la página de Configuraciones de Dominios, la cual te permite ajustar múltiples configuraciones de tu sitio.
    • Si estás en la Vista de Malla, da clic en el botón Administrar en la parte inferior de la caja del sitio.
    • Si estás en la Vista de Lista, da clic en el botón Administrar a la derecha de tu dominio.
  3. Selecciona la pestaña Sitio Web.
  4. Baja hasta la sección Administra Tu Sitio.
  5. En la parte derecha de Administrar Base de Datos, haz clic en el botón Administrar para abrir phpMyAdmin.
    Se abre la página de phpMyAdmin.

Puedes realizar cambios en tu base de datos MySQL de DreamPress directamente en phpMyAdmin, independientemente de si el DNS del dominio apunta a DreamHost. phpMyAdmin es particularmente útil para realizar copias de seguridad, importar, exportar y restaurar una base de datos.

Para obtener más información sobre phpMyAdmin, revisa los enlaces al final de este artículo.

Opción 2 — Ubicar tus credenciales

Revisa el siguiente artículo para obtener más información sobre cómo conectarte a una base de datos mediante un programa de terceros:

  • Conectase a una base de datos usando MySQL Workbench

A continuación, puedes usar las siguientes instrucciones para ubicar las credenciales de inicio de sesión de tu base de datos para acceder a tu base de datos de DreamPress usando FTP:

  1. Navega a la página Administrar Sitios Web.
  2. Da clic en el botón Administrar para abrir la página de Configuraciones de Dominios, la cual te permite ajustar múltiples configuraciones de tu sitio.
    • Si estás en la Vista de Malla, da clic en el botón Administrar en la parte inferior de la caja del sitio.
    • Si estás en la Vista de Lista, da clic en el botón Administrar a la derecha de tu dominio.
  3. Selecciona la pestaña Sitio Web.
  4. Baja hasta la sección Administrar Tu Sitio.
  5. En la parte derecha de Administrar Base de Datos, haz clic en el botón Datos de Ingreso para ver tus credenciales.
