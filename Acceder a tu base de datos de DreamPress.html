Generalidades

A continuación se describe cómo realizar cambios rápidamente en tu base de datos MySQL en DreamPress.

Opción 1 — Iniciar sesión automáticamente en phpMyAdmin

  1. Navega a la página WordPress Administrado.
  2. Haz clic en el botón Administrar.
    La página 'Wordpress Administrado' se abre:
  3. En la parte inferior derecha debajo 'BASE DE DATOS (MYSQL)', haz clic en el botón Administrar Base de Datos.
    La siguiente página se abre.

Puedes realizar cambios en tu base de datos MySQL de DreamPress directamente en phpMyAdmin, independientemente de si el DNS del dominio apunta a DreamHost. phpMyAdmin es particularmente útil para realizar copias de seguridad, importar, exportar y restaurar una base de datos.

Para obtener más información sobre phpMyAdmin, consulta los enlaces en la siguiente sección.

Opción 2 — Ubicar tus credenciales

Revisa el siguiente artículo para obtener más información sobre cómo conectarte a una base de datos mediante un programa de terceros:

  • Conectarse a una base de datos mediante un programa de terceros

A continuación, puedes usar las siguientes instrucciones para ubicar las credenciales de inicio de sesión de tu base de datos para acceder a tu base de datos de DreamPress mediante FTP:

  1. Navega a la página WordPress Administrado.
  2. A la derecha de tu dominio, haz clic en el botón Administrar.
  3. En la parte inferior derecha, verás una sección BASE DE DATOS (MYSQL). Enciende Mostrar Credenciales para ver las credenciales de tu base de datos.
  4. Haz clic en el icono de copia a la derecha de cada elemento y pega las credenciales en tu cliente MySQL elegido.

Consulta el siguiente artículo para obtener instrucciones sobre cómo cambiar la contraseña de tu usuario de la base de datos.
