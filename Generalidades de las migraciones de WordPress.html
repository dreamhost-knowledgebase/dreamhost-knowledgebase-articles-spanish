Generalidades

Si tienes problemas para migrar tu sitio web desde otro alojamiento, o simplemente no tienes tiempo, DreamHost actualmente ofrece un servicio que se encarga de esto por ti. Puedes encontrar más información en el artículo de DreamHost Servicios de migración para sitios web.

La migración de un sitio de WordPress es diferente de la migración de un sitio web HTML sencillo ya que los sitios de WordPress dependen tanto de los archivos web como de una base de datos MySQL para ejecutarse. Esta descripción general explica tus opciones según el tipo de migración que necesitas.

Cambiar tu nombre de dominio de WordPress

Es posible que debas cambiar el dominio de tu sitio de WordPress o transferir un sitio a un dominio diferente que ya está alojando con DreamHost. Esto sucede si está usando un nombre de dominio diferente o si está moviendo un sitio de un sitio de prueba o provisional a un sitio de producción en vivo. DreamHost ofrece dos formas sencillas de cambiar un dominio WordPress:

Opción 1 — Usar la herramienta para renombrar en tu panel de DreamHost

DreamHost provee una herramienta en la página del panel Sitios Web que te permite cambiar el nombre del dominio de cualquier sitio WordPress. Mira el siguiente artículo par ampas información sobre como usar la herramienta:

  • Cambiar el nombre del dominio a un sitio WordPress en el panel

Opción 2 — Deja de DreamHost renombre el sitio por ti

DreamHost ofrece un Microservicio Profesional para cambiar la URL del sitio por tí. Mira el siguiente artículo para más información sobre este servicio:

  • Servicios de migración del sitio web

Mover un sitio WordPress a otra cuenta de DreamHost

En algunas circunstancias, es posible que desees mover un sitio a una cuenta DreamHost diferente, como cuando transfieres la propiedad de un sitio a un nuevo desarrollador o individuo. DreamHost proporciona dos métodos para hacerlo:

Opción 1 — Deja que DreamHost mueva tu sitio por tí

El Microservicio Profesional de DreamHost puede mover tu sitio entre cuentas por tí. Mira el siguiente artóculo para más información sobre este servicio:

  • Servicios de migración de sitios web

Opción 2 — Mover manualmente tu sitio

Puede mover manualmente los datos de tu sitio entre cuentas usando las herramientas proporcionadas en el panel DreamHost. Lee el siguiente artículo para más instrucciones sobre este proceso:

  • Como mover el alojamiento de un dominio entre cuentas

Opción 1 (Fácil) — Usa el plugin de Migración Automatizada de WordPress

El plugin de Migración Automatizada de DreamHost te permite migrar tu sitio fácilmente a tu servidor en DreamHost. Para más información sobre el plugin, mira el siguiente artículo:

Solución de problemas al descargar archivos PHP después de una migración

En algunas circunstancias, un sitio migrado a DreamHost desde otro proveedor de alojamiento no carga correctamente. Cuando accedas al sitio, el buscador descargará un archivo PHP en vez de mostrar el sitio web. Esto es causado típicamente por código que sobre del anterior alojador, muy comunmente por la prescencia de una entrada AddHandler en el archivo .htaccess del sitio como el ejemplo de abajo:

AddHandler application/x-httpd-php70 .php

Este código no es necesario en los servidores de DreamHost y debería ser eliminado del archivo .htaccess de WordPress luego de que el sitio se haya migrado.
