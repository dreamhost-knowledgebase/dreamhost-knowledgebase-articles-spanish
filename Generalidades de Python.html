Python en DreamHost

Python se instala en todos los servidores de DreamHost predeterminadamente. Sin embargo, es probable que quieras instalar una versión personalizada.

  • Instalar una versión personalizada de Python 3

Si vas a instalar paquetes, primero debes crear un entorno virtual para almacenarlos localmente bajo tu nombre de usuario.

  • Instalar y usar virtualenv con Python 3

Luego puedes usar pip para instalarlo.
