Generalidades

Python es un lenguaje de programación como PHP y Ruby que te permite crear muchos tipos diferentes de aplicaciones. Se puede encontrar más información en el sitio web de Python:

  • https://www.python.org/about/

Desarrollo

DreamHost recomienda usar FastCGI para desarrollar tu aplicación en Python. Fast CGI usa mod_fcgi.

Marcos Web

Algunos marcos de sitios web de Python pueden ejecutarse en servidores DreamHost. Éstos incluyen:

  • Django
  • Pylons
  • Turbogears
  • Web.py
  • Flask
