  • https://www.python.org/about/

También puedes instalar cualquier versión de Python 2 o Python 3 que desees.

Implementación

Tienes algunas opciones sobre cómo implementar tu aplicación Python en DreamHost:

  • Passenger – La forma preferida de implementar aplicaciones Python. Este es un módulo de Apache que se escribió originalmente para admitir la ejecución de aplicaciones Ruby on Rails, pero que también brinda soporte para aplicaciones compatibles con WSGI. Revisa el artículo Passenger para obtener detalles de configuración.
  • CGI – Lento y no recomendado.
  • FastCGI – Más rápido que CGI y utiliza mod_fcgi.

Marcos Web

Algunos marcos de sitios web de Python pueden ejecutarse en servidores DreamHost. Éstos incluyen:

  • Django
  • Pylons
  • Turbogears
  • Web.py
  • Flask
