Generalidades

Es posible configurar tu sitio de WordPress en DreamHost antes de apuntar tus DNS a DreamHost. Esto requiere que crees un subdominio dreamhosters.com.

  • Usar un subdominio dreamhosters.com

Por ejemplo, si tu sitio se llama example.com, puede agregar un nuevo dominio en tu panel llamado example.dreamhosters.com. Este nuevo sitio se utiliza para acceder a tu sitio oficial antes de que apuntes tus DNS.

Sube tu sitio de WordPress a tu sitio DreamHost

Este paso hace una copia de tu sitio actual de WordPress (en otra compañía) en DreamHost.

  1. Abre la página Administrar Sitios Web.
  2. Asegúrate de que tu dominio está Completamente Alojado. De ser así, el plan de alojamiento específico se muestra debajo de la columna Plan de Alojamiento.
  3. Carga todo el contenido del sitio web de tu antigua empresa de alojamiento en tu sitio en DreamHost (example.com). Revisa los siguientes artículos que explican cómo cargar los archivos, la base de datos y el correo electrónico de tu sitio web.
    • Generalidades sobre FTP y credenciales
    • ¿Cómo migrar mi base de datos a DreamHost?
    • Importar el correo otro alojamiento a DreamHost

Agregar un subdominio dreamhosters.com

Para más información sobre añadir un subdominio dreamhosters.com, visita el siguiente artículo:

    • Generalidades del reflejo de un dominio

Crear un hostname de base de datos

Este paso permite que tu subdominio dreamhosters.com vea la información de la base de datos de tu sitio en DreamHost. De lo contrario tu sitio apuntaría a la empresa de alojamiento actual.

  1. Crea un hostname mysql en el panel para este nuevo subdominio dreamhosters.com en la página Bases de Datos MySQL.
  2. Haz clic en el botón Agregar nuevo hostname para crear el hostname. Debe ser nombrado algo como mysql.example.dreamhosters.com.
  3. Haz clic en el botón ¡Crear este hostname MySQL ahora! para guardar.

Editar tus archivos de WordPress

Este paso asegura que tu subdominio dreamhosters.com apunte a tu base de datos DreamHost.

  1. Usando FTP, sube el contenido de tu sitio web principal a tu servidor DreamHost. Visita el artículo FTP para obtener más detalles sobre cómo cargar:
    • FTP
  2. Una vez subido, edita tu archivo wp-config.php. Verás la siguiente línea dentro del archivo:
    define('DB_HOST', 'mysql.example.com');
    • Esto hará que tu sitio no cargue, ya que intentará conectarse a una base de datos externa.
  3. Cambia el hostname a la URL de tu subdominio dreamhosters.com:
    define('DB_HOST', 'mysql.example.dreamhosters.com');

Sube y edita tu base de datos

Este paso actualiza los enlaces de tu base de datos para que puedas ver el sitio en tu subdominio dreamhosters.com.

  1. Carga tu base de datos a tu cuenta DreamHost en tu panel en Bases de Datos MySQL. Visita el siguiente artículo para obtener más información sobre cómo importar tu base de datos:
    • Migrar MySQL
  2. Una vez cargado, actualizar las URL de HOME y SITE URL en tu base de datos. Accede a la interfaz de phpMyAdmin en la misma página de MySQL Database.
    • Debes usar el hostname mysql.example.dreamhosters.com que acabas de crear para iniciar sesión en tu base de datos.
  3. Cuando accedas, haz clic en el nombre de tu base de datos en la izquierda.
    Cuando haces clic en la base de datos, se abre una lista de las tablas de tu base de datos.
  4. Haz clic en la tabla llamada wp_options. Puedes notar que tienes un prefijo de tabla diferente. En este ejemplo, el prefijo de la tabla es wp_bnkih8_.
  5. Haz clic en la pestaña Browse en la parte superior izquierda.
    Cuando haces clic en dicha pestaña, los campos 'siteurl' y 'home' aparecen.
  6. Edita estos campos y cámbialos a tu nueva URL de dreamhosters.com.

Ahora puedes visitar el sitio duplicado example.dreamhosters.com que creaste para ver el contenido del sitio principal que has cargado en el servidor DreamHost.
