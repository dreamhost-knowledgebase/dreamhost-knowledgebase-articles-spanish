Crear una lista de discusión

  1. Navega a la página Discussion Lists.
  2. Haz clic en el botón Create New Discussion List en la parte superior.
    La página 'Creating Discussion List' se abre:
  3. Ingresa la siguiente información sobre tu lista de discusión.
    • List Name: Ingresa el nombre de tu lista de discusión. Por ejemplo, si tu lista es para dueños de perros, puedes llamarla "doglovers" o algo similar. Esta es también la primera parte de la dirección de correo electrónico de la lista, por ejemplo, doglovers@lists.example.com.
    • List Owner Email Addresses, one per line: Ingresa las direcciones de correo electrónico de los propietarios/administradores de la lista, quienes luego tendrán acceso completo para editarla dentro del panel. Puedes establecer múltiples propietarios de la lista si es necesario.
    • List Password: Ingresa tu contraseña elegida para la lista. Al igual que cualquier contraseña, deseas que esto sea seguro. Visita el artículo Generalidades sobre contraseñas para obtener detalles sobre cómo crear una contraseña segura. Además, ten en cuenta que esta contraseña solo se aplica a esta lista, y nada más dentro de tu cuenta.
    • Use an existing domain or Create a new sub-domain: Tienes la opción de incluir un subdominio para tu discusión. Si eliges la primera opción, el subdominio se convierte automáticamente en ‘lists’. Si eliges la segunda opción, puedes ingresar cualquier cosa que desees como subdominio.
    El segundo campo es un menú desplegable para que selecciones en qué dominio deseas crear la lista.
    • Create Convenience Addresses: Si marcas esta casilla, se crean una serie de direcciones de correo electrónico adicionales específicamente para tu lista. Por ejemplo:
    admin@lists.example.com
    subscribe@lists.example.com
    owner@lists.example.com
    • Esta opción está marcada de forma predeterminada y, por lo general, es una buena idea crear estas direcciones.
    • Estas direcciones de conveniencia solo funcionan si alojas un correo electrónico con DreamHost. Si estás utilizando un servidor de correo electrónico de terceros como Google, las direcciones de conveniencia NO funcionarán.
  4. Haz clic en el botón Create Discussion List Now! para crear la lista.

    Dado que el subdominio es un nuevo registro DNS, tarda un tiempo para que el DNS se actualice en línea, y puede tardar hasta 6 horas. La lista en sí solo debería tomar unos 30 minutos para crearse en tu panel.
