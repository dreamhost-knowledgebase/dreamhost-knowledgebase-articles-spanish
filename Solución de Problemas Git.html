gnutls_handshake() failed: Handshake failed

Es posible que veas este error cuando te conectes a ciertos repositorios desde un VPS DreamHost antiguo o un Servidor Dedicado que ejecute Ubuntu 14 (Trusty). El error está relacionado con un error de Ubuntu 14. A continuación se explica en qué servidores puede ocurrir esto.

Shared

DreamHost ya ha actualizado los servidores web compartidos a Ubuntu Bionic 18, donde ya no se encontrará con este error.

VPS

Los servidores privados comprados antes de marzo de 2020 ejecutan Ubuntu 14.

Los servidores privados comprados después del 9 de marzo de 2020 ejecutan el sistema operativo Debian Stretch donde ya no se encontrará con este error. DreamHost pronto actualizará todos los servidores privados a Debian Stretch.

Dedicados

Cualquier Servidor Dedicado recién comprado ejecuta Bionic. DreamHost está actualizando Servidores Dedicados antiguos a Bionic donde ya no se encontrará con este error.
