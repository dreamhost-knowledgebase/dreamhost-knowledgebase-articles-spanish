Generalidades

Cuando creas la primera dirección de correo electrónico para un dominio (que está configurado para ser alojado con DreamHost), el sistema también configurará los nombres de host para el servicio de correo web.

Después de agregar la primera dirección de correo electrónico, los DNS pueden tardar entre 4 y 6 horas en actualizarse en línea. Solo después de las actualizaciones de DNS podrás acceder al correo vía la interfaz web. Cualquier dirección adicional que agregues estará disponible de inmediato.

Webmail se encuentra en:

https://webmail.dreamhost.com

Advertencia de error de coincidencia del certificado SSL al conectarte a tu URL de correo web personalizado

En el pasado, era posible iniciar sesión en el correo web en https://webmail.example.com (donde example.com es su sitio web). Debido a las mejoras en la infraestructura de correo electrónico de DreamHost, ya no debes usar esta URL para conectarte. Si utilizas la URL personalizada de tu dominio, verás el siguiente error de SSL en tu navegador:

Advertencia en la barra de direcciones del navegador

Mensaje avanzado en Chrome

Es bastante recomendado que solo utilices https://webmail.dreamhost.com para conectarte a tu cuenta de correo web para así evitar estos problemas.
