Generalidades

Cuando creas la primera dirección de correo electrónico para un dominio (que está configurado para ser alojado con DreamHost), el sistema también configurará los nombres de host para el servicio de correo web.

Después de agregar la primera dirección de correo electrónico, los DNS pueden tardar entre 4 y 6 horas en actualizarse en línea. Solo después de las actualizaciones de DNS podrás acceder al correo vía la interfaz web. Cualquier dirección adicional que agregues estará disponible de inmediato.

Webmail se encuentra en:

https://webmail.dreamhost.com

Advertencia de error de coincidencia del certificado SSL al conectarse a correo web

Si ves una advertencia de SSL en tu navegador cuando visitas la página de inicio de sesión de webmail, es porque estás utilizando una URL que no es webmail.dreamhost.com. Esta es la única dirección que debes usar para iniciar sesión en tu panel de correo.
