Generalidades

Puedes localizar las direcciones IP de tu instancia con el cliente de línea de comandos de OpenStack o usando el panel de DreamCompute.

Usar el OpenStack CLI

Para correr los siguientes comandos, asegúrate de haber instalado el Cliente de línea de comandos OpenStack.

Puedes ver instancias en tu proyecto usando el comando openstack server list. Todas las direcciones IP asignadas a la instancia se mostrarán en la columna Networks.

[user@localhost]$ openstack server list
+------------------+--------------+--------+--------------------------------------------------------------+-------+---------------+
| ID               | Name         | Status | Networks                                                     | Image | Flavor        |
+------------------+--------------+--------+--------------------------------------------------------------+-------+---------------+
| 8a99547e-7385... | dhc-instance | ACTIVE | public=208.113.132.158, 2607:f298:5:101d:f816:3eff:fe66:28e9 |       | gp1.semisonic |
+------------------+--------------+--------+--------------------------------------------------------------+-------+---------------+

Usando el panel de DreamCompute

  1. Navega la página DreamCompute.
  2. Haz clic en el botón "Ver panel".
  3. A la izquierda, haz clic en "Compute".
  4. De las opciones múltiples, selecciona Instances.
  5. La dirección IPv4 de tu instancia se muestra en la columna IP Address.
