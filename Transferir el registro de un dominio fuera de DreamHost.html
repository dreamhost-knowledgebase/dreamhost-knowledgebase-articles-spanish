Generalidades

A continuación se describe cómo transferir el registro de tu dominio a otra compañía de registro.

Los detalles enumerados aquí son solo para transferir el registro de tu dominio a otro registrador. La transferencia del alojamiento de tu sitio es un proceso completamente diferente. Puedes leer más sobre ese proceso en el siguiente artículo:

  • Transferir el alojamiento web a otra compañía

Preparar tu registro para la transferencia

Debido a la política de transferencia de ICANN, cualquier cambio realizado en el nombre, organización o dirección de correo del registrante pondrá el dominio en un bloqueo de transferencia de 60 días; también activará correos de notificación que se envían al antiguo y nuevo registrante notificándoles el cambio. Antes de iniciar una transferencia, asegúrate de desactivar el bloqueo de transferencia de 60 días usando el panel.

Para obtener más información sobre la nueva política y cómo optar por el bloqueo de transferencia de 60 días, visita el siguiente artículo para obtener más detalles:

  • Política de transferencia de ICANN

Por favor ten en cuenta que incluso si cambias tu configuración de privacidad, se activa el bloqueo de 60 días. Así que asegúrate de que la casilla de verificación esté seleccionada para optar por no participar.

Antes de enviar la solicitud de transferencia a tu nuevo registrador, debes completar algunos pasos. Esta sección ofrece una visión general de estos pasos, y los pasos completos aparecen a continuación.

Se deben hacer los siguientes cambios en tu nombre de dominio:

  1. Desbloquea el registro de dominio para transferir en el panel en Administrar Registros de Dominio.
  2. Obtén el código de autorización, también conocido como el código EPP, desde el panel en la página de Transferencias de Registros. Necesitas este código para completar la solicitud de transferencia en tu nuevo registrador.

¿Cuándo deberías comenzar la transferencia?

Debido a las restricciones de ICANN, no puedes transferir un registro de dominio dentro de los 60 días de la compra original o si ha sido transferido a DreamHost en los últimos 60 días.

La transferencia puede tardar varios días en completarse, por lo que es mejor iniciar el proceso lo antes posible antes de que expire el registro. DreamHost recomienda iniciar la transferencia al menos una semana completa antes de que expire el dominio.

 

Paso 1 — Desbloquear tu dominio

El estado de bloqueo se muestra en la página Administrar Registros de su panel y en una consulta WHOIS.

  • La página Administrar Registros de Dominio muestra la columna Bloqueo de transferencia.
  • El estado indica 'Bloqueado' o 'Desbloqueado'.
  • Al hacer clic en el botón de alternar cambia el estado.

Tu dominio puede estar bloqueado para evitar la transferencia del mismo. Un dominio bloqueado aparece de la siguiente manera:

Si está bloqueado, debes desbloquearlo antes de continuar con la solicitud de transferencia. Haz clic en el botón de deslizar para desbloquearlo:

Si realizas una búsqueda de WHOIS para el registro del dominio, también puedes ver el estado de bloqueo allí. Consulta la sección Estado de Dominio de la búsqueda de WHOIS para el dominio. Si está bloqueado, aparece el siguiente estado:

Domain Status: clientTransferProhibited https://www.icann.org/epp#clientTransferProhibited

Si el dominio está desbloqueado, aparece el siguiente estado:

Domain Status: ok

Paso 2 — Adquirir el código de autorización o EPP (auth) de tu dominio

El segundo paso para preparar el registro de tu dominio para la transferencia es la recuperación de su código de autorización (o código EPP). Este código es lo que te permite realizar la solicitud de transferencia con la compañía de registro receptora.

Para ver/enviar el código de autorización (código EPP):

  1. Navega a la página Transferir Registros de Dominio.
  2. Haz clic en el enlace titulado O transferir fuera de DreamHost.
    Se abre una página que muestra tus dominios.
  3. Da clic en el botón Revelar Código de Autorización.

Según tu Dominio de nivel superior (TLD) puedes revelar tu código de autenticación (código EPP) en el panel o enviarlo a la dirección de contacto del Registrante (según lo especificado por tu WHOIS).

Al enviar el código EPP/Auth a través de la página Transferir Registros de Dominio, por favor revisa la carpeta de correo basura/spam para la dirección de correo electrónico de contacto del Administrador que configuraste para el registro.

 

Paso 3 — Solicitar la transferencia

Las transferencias de registro de dominio deben iniciarse a través de la interfaz de la nueva compañía de registros. Esto se realiza después de completar la preparación para la transferencia.

Cada compañía de registros tiene un método diferente para iniciar transferencias de registro de dominio. Si no estás seguro de qué hacer, comunícate con tu compañía de registros y pregúntales cómo puedes iniciar una transferencia de dominio.

¿Cómo transferir un dominio *.uk fuera de DreamHost?

  • Los dominios .uk no tienen un código EPP. Para transferir un dominio *.uk que posees en DreamHost a otra compañía de registro, solicita a tu nueva compañía de registro su etiqueta IPS. Las etiquetas IPS distinguen entre mayúsculas y minúsculas, y una vez que se cambia una etiqueta IPS, no se puede revertir.

  • Cuando envíes tu solicitud, aclara que la etiqueta IPS en tu dominio debe cambiarse para tí.
  • Pónte en contacto con tu nueva compañía de registro después de que se haya confirmado el cambio de etiqueta IPS, ya que su sistema puede requerir pasos adicionales para completar el proceso.
