Generalidades

ShareX es una aplicación de Windows gratuita y de código abierto que puedes usar para tomar capturas de pantalla y luego compartirlas con otros. A partir de la versión 9.0.0, puedes cargar fácilmente capturas de pantalla en DreamObjects. ShareX aprovecha la compatibilidad de la API S3 de DreamObjects para permitir que se configure como destino.

Este artículo asume que ya tienes una cuenta de DreamObjects configurada y has creado un Usuario, Bucket, y Llaves de acceso.

Configurar ShareX con DreamObjects

  1. Lanza ShareX.
  2. A la izquierda, haz clic en el menú ‘Destinations’ y luego selecciona ‘Destination settings...’
  3. Desde la opción Destination Settings, desplázate hacia abajo a la sección File Uploaders
  4. Selecciona ‘Amazon S3’.
  5. Selecciona ‘DreamObjects’ del menú desplegable Endpoint en la ventana principal.
  6. Ingresa la Llave de acceso y llave secreta en los campos seleccionados.
  7. Para enrutar el destino de las capturas de pantalla y otra información a DreamObjects, vuelve a la opción de carpeta ‘Destinations’ dentro de ShareX.
  8. Selecciona la opción ‘Image uploader’ > ‘File uploader’ > ‘Amazon S3’ como se muestra en la captura de pantalla anterior.
  9. Repite el paso anterior para habilitar cargas de texto y archivos.
