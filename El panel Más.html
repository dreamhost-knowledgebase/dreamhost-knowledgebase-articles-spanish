Generalidades

El menú Avanzado contiene enlaces para otras partes del panel DreamHost que ofrecen adiciones a tu sitio web. Estos se pueden usar para optimizar tu experiencia general.

Llaves de SSH

En la página Llaves de SSH, puedes ver las Huellas Digitales de SSH asignadas a tu servidor. Luego, puedes verificar que se estás conectando al servidor correcto a través de SSH cuando se te solicite que guardes la Huella Digital y la clave SSH.

Cron Jobs

La página Cron Jobs es donde puedes agregar/administrar un cron job. Los cron jobs son una forma de realizar tareas programadas en sistemas UNIX. Cada entrada en el archivo "crontab" de un usuario se denomina "cronjob".

Revisa el siguiente artículo para más detalles:

  • Generalidades de Crontab

Htaccess/WebDAV

La página Htaccess/WebDAV es donde puedes habilitar la protección con contraseña o WebDAV.

Puedes usar esta interfaz web para crear nombres de usuario y contraseñas para que tus amigos y asociados accedan a secciones protegidas por contraseña de tus sitios web. Incluso puedes usar un archivo .htaccess para ayudar a evitar el enlace directo de los archivos de tu sitio web por parte de terceros o prohibir extensiones de archivo específicas.

La opción de panel sólo puede agregar las características mencionadas anteriormente. Si necesitas agregar cualquier otro código a un archivo .htaccess, debes crearlo manualmente. Revisa el siguiente artículo para más detalles:

  • Generalidades del archivo .htaccess

Y mira lo siguiente para obtener una descripción general de WebDAV:

  • Generalidades de WebDAV

Bloquear Spiders

En la página Bloquear Spiders, puedes crear un archivo .htaccess para bloquear motores de búsqueda específicos.

Mira el siguiente artículo para más detalles:

  • Controlar los bots, spiders, y rastreadores

Proxy

En un Servidor Virtual Privado de DreamHost (VPS) o Servidor Dedicado, puedes configurar un Servidor Prixy. Puedes configurarlo en una página Proxy en el panel de DreamHost.

Mira el siguiente artículo para más detalles:

  • Servidor Proxy

Estadísticas del Sitio Web

Puede habilitar las estadísticas del sitio en la página Estadísticas del Sitio Web. La aplicación de estadísticas del sitio se ejecuta a través de un programa llamado Analog y proporciona varios tipos de informes, incluidos informes estadísticos para su dominio.

Revisa el siguiente artículo para más detalles:
