DreamHost ahora ofrece Google Workspace, el cual es una colección de aplicaciones populares que te ayudan a organizar y simplificar tus correos/archivos personales o el flujo de trabajo de tu organización. Los planes de Google Workspace  incluyen correo, almacenamiento, edición/uso compartido de documentos, informes y más. Una de las mejores cosas de tener tu propio dominio es que puedes tener cualquier dirección personal que te guste para ese dominio, que se ve más profesional que una dirección de correo genérica, como @gmail.com.

Agregar una dirección de correo solo requiere unos pocos pasos simples que esta guía explica. Sin embargo, si necesitas más detalles sobre otros aspectos de una dirección de correo, revisa el artículo Configuración de cliente de correo.

Prerrequisitos

Debes tener un plan de alojamiento activo, o un plan pago adicional de correo para usar correo.

Todos los planes de alojamiento, excepto Shared Starter, ofrecen correo de forma gratuita.

Los planes de alojamiento shared starter y solo registro de dominio no vienen originalmente con correo , sin embargo, puedes agregar un plan de correo pagado a tu cuenta para hasta 5 direcciones de correo. Consulta el siguiente artículo para obtener más información.

  • Plan pago adicional de correo

Crear una dirección de correo

Para crear una dirección de correo:

  1. Abre la página Administrar Correo.
  2. Haz clic en el botón Crear Nueva Dirección de Correo.
  3. Ingresa lo siguiente:
    • La dirección de correo deseada en el primer campo.
    • Elige el dominio al que deseas agregarlo de la lista desplegable.
  4. Ingresa un nombre para el buzón (este puede ser tu nombre o algo para describir el propósito del buzón).
  5. Selecciona la casilla para generar una contraseña (o ingresa una de tu elección si lo prefieres).
  6. Hay otros campos como 'Límite uso total del disco', pero no necesitas ajustarlos para crear la dirección. Si deseas leer más sobre esas configuraciones, visita el siguiente artículo:
    • Generalidades sobre advertencia de uso de disco de correo
  7. Haz clic en el botón Crear Dirección. Aparece un mensaje de éxito que confirma que se ha agregado la dirección.

Ahora estás listo para configurar tu Cliente de correo y comenzar a enviar y recibir correos.

Para encontrar rápidamente tu configuración de correo en el panel, abre la página Administrar Correo y busca el enlace Información de configuración de correo en la esquina superior derecha. Haz clic en el texto para abrir las instrucciones de acceso rápido para tu dirección de correo:

Entrante
imap.dreamhost.com
pop.dreamhost.com
Saliente
smtp.dreamhost.com

Posibles errores

Los pasos anteriores suponen que tu DNS está alojado con DreamHost. Si tienes Registros MX personalizados para un dominio, no podrás crear nuevas direcciones. Esto se debe a que si estás usando Gmail (u otro proveedor de correo alternativo), debes crear nuevas direcciones de correo en ese proveedor.

Otro problema a tener en cuenta es que, a veces, cuando ya tienes una gran cantidad de direcciones creadas, recibes un mensaje de error que dice:

“Your account cannot create additional e-mail addresses at this time. 

This is usually a temporary situation and should be resolved within 24 hours.

Please contact Support if you have any questions.”

En caso de que reciba este mensaje, por favor contacta a soporte
