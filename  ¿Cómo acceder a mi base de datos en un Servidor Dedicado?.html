Generalidades

Con un plan de alojamiento compartido, VPS o DreamPress, todas las bases de datos MySQL están en una máquina MySQL separada. En un Servidor Dedicado, todas las bases de datos MySQL se almacenan localmente en la misma máquina. Esto te brinda acceso completo a la raíz para modificar tus bases de datos MySQL según lo necesites.

Administrar tu base de datos

Puedes administrar tus bases de datos en el panel de DreamHost como cualquier otro plan de alojamiento. Si eres un usuario avanzado con conocimientos de UNIX Shell y te gustaría realizar personalizaciones más allá de lo que puede ofrecer el panel, debes utilizar un usuario de Shell para iniciar sesión en el servidor. Mira la siguiente categoría para ver artículos sobre cómo crear un usuario de Shell e iniciar sesión en tu servidor.

  • Temas SSH

Ten en cuenta que el soporte de DreamHost no puede ayudar con ningún tipo de personalización o modificación al servidor MySQL. 

Actualización a un Servidor Dedicado

Cuando estás en un plan Shared VPS, tus bases de datos MySQL están en un servidor MySQL compartido.

Cuando actualizas, DreamHost mueve todos tus servicios MySQL a tu nuevo Servidor Dedicado.

El servicio MySQL se mueve incluso si se selecciona la opción 'don't move data'.

Cambiar de hosting Dedicado a compartido

Si cancelas el Servidor Dedicado, tus servicios MySQL volverán a migrar a un servidor MySQL compartido.
