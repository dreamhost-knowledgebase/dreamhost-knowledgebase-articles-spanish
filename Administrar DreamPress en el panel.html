Generalidades

Este artículo explica cómo administrar las configuraciones de tu sitio web DreamPress en el panel de DreamHost. Aquí es donde puedes administrar las configuraciones generales del sitio web, archivos, bases de datos, plan, sitio web provisional, o cambiar el nombre de dominio.

Acceder a la página de configuraciones de DreamPress

  1. Navega a la página Administrar Sitios Web.
  2. Da clic en el botón Administrar para abrir la página Configuraciones de Dominios, la cual te permite ajustar múltiples configuraciones de tu sitio.
    • Si estás en la Vista de Malla, da clic en el botón Administrar en la parte inferior de la caja del sitio.
    • Si estás en la Vista de Lista, da clic en el botón Administrar a la derecha de tu dominio.
  3. Da clic en las pestañas Sitio Web, Alojamiento, o Sitio Provisional en la parte superior de la página Sitios Web Administrados para administrar tu sitio DreamPress.

Administrar tus configuraciones de DreamPress

Las siguientes tres pestañas ubicadas en la parte superior de la página Sitios Web Administrados te permiten hacer ajustes al plan y al sitio web de DreamPress site.

Sitio Web

La pestaña Sitio Web muestra una vista general de las configuraciones de tu sitio. Aquí puedes:

  • Administrar las bases de datos y los archivos del sitio
  • Administrar las configuraciones de WordPress
  • Ver/Cambiar la versión de PHP
  • Cambiar el nombre del dominio
  • Ver herramientas de seguridad
  • Correo Electrónico
  • SEO

Alojamiento

La pestaña Alojamiento muestra tu plan actual de DreamPress.

Haz clic en el botón Cambiar Plan para ver otros planes de DreamPress y sus características disponibles para tu sitio.

Sitio Web Provisional

La pestaña Sitio Web Provisional te permite crear un sitio web provisional. Esta es una forma de probar de forma segura los cambios en tu sitio DreamPress antes de publicarlo en vivo. Revisa el siguiente artículo para obtener más información sobre los sitios web provisionales:

  • Sitio provisional en DreamPress

DNS

La pestaña DNS te permite ver y personalizar los registros DNS de tu sitio web. Revisa el siguiente artículo para aprender más sobre el ajuste de los registros DNS:

  • Agregar DNS
