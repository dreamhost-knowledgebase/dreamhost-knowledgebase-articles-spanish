Generalidades

Este artículo explica cómo crear una copia de seguridad o exportar una base de datos o tabla en phpMyAdmin.

Respaldar una base de datos

Los siguientes pasos están también disponibles como video instructivo:

Para exportar (respaldar) toda tu base de datos:

  1. Inicia sesión de phpMyAdmin.
  2. Selecciona la base de datos en el panel izquierdo.
  3. Haz clic en la pestaña Exportar en el panel central superior.
  4. Selecciona un método de exportación Rápido ó Personalizado.

    El método Rápido te permite descargar el archivo .sql inmediatamente.

    El método Personalizado te brinda más control sobre los datos. Usa el método Personalizado solo si estás muy familiarizado con la administración de tablas SQL.

  5. Escoge el formato en el que deseas guardar el archivo del menú desplegable. SQL es el formato más común.
  6. Haz clic en el botón Continuar para continuar.
Aparece una ventana emergente que te permite descargar el archivo. (Nota: Es posible que tu navegador lo descargue automáticamente.)

Exportar una sola tabla

Los siguientes pasos también están disponibles como vídeo instructivo:

Para exportar (o respaldar) de una sola tabla dentro de tu base de datos:

  1. Inicia sesión en phpMyAdmin.
  2. Selecciona la base de datos en el panel izquierdo.
  3. Selecciona la tabla que deseas exportar en el panel izquierdo debajo del nombre de la base de datos.
  4. Haz clic en la pestaña Exportar en el panel central superior.
  5. Selecciona un método de exportación Rápido ó Personalizado.

    El método Rápido te permite descargar el archivo .sql inmediatamente.

    El método Personalizado te brinda más control sobre los datos. Usa el método Personalizado únicamente si estás muy familiarizado con la administración de tablas SQL.

  6. Escoge el formato en el que deseas guardar el archivo del menú desplegable. SQL es el formato más común.
  7. Haz clic en el botón Continuar para continuar.
Aparece una ventana emergente que te permite descargar el archivo. (Nota: Es posible que tu navegador lo descargue automáticamente).
