Generalidades

En el panel, la página Uso de Disco detalla varios detalles de uso enumerados a continuación:

  • servidor (server)
  • usuario (username)
  • dominio (domain)
  • dirección de correo electrónico (email address)
  • base de datos (database)
  • Listas de discusión  (discussion lists)

Disk Usage (Uso del disco)

La sección Disk Usage enumera lo siguiente:

  • Día de tu ciclo de facturación
  • Fechas de inicio y finalización de tu ciclo de facturación
  • Uso estimado del disco del servidor

Disk Usage by Server (Uso de disco o servidor)

La sección Disk Usage by Server muestra el uso del disco de cualquier VPS, MySQL privado o servidores dedicados agregados a tu cuenta.

Disk Usage by User (Uso de disco o usuario)

La sección Disk Usage by User enumera los usuario que hayas agregado a tu servidor junto con la cantidad de espacio en el disco que ocupa cada usuario. Si haces clic en el botón View report (Ver informe), el uso del disco del usuario se muestra en función de cada día de uso.

Disk Usage by Fully Hosted Domain (Uso de disco por dominio alojado)

La sección Disk Usage by Fully Hosted Domain enumera el uso en función de cualquier dominio al que hayas alojado en el servidor. Por defecto, no se muestran valores. Haz clic en el botón Go! para actualizar esta página. Aparece un mensaje de confirmación que notifica que puede tardar algunas horas actualizar esta información y que te enviaremos un correo electrónico una vez se haya actualizado.

Disk Usage by Address (Uso del disco por cuenta de correo)

La sección Disk Usage by Address enumera el uso del disco de correo electrónico. Haz clic en el botón View Report a la derecha de cada dirección para ver el uso diario.

Disk Usage by Database (Uso de disco por base de datos)

El Disk Usage by Database enumera el uso del disco de cada base de datos.

Disk Usage by Discussion List (Uso del disco por lista de discusión)

La sección Disk Usage by Discussion List) enumera todas las Listas de discusión que haya agregado a tu cuenta y cuánto espacio en disco están usando.
