Generalidades

Zen Cart es un carrito de compras completo con una comunidad de soporte activa. Se basa en el antiguo motor de osCommerce que puedes usar para vender artículos a través de Internet. Zen Cart también se puede personalizar con plantillas para modificar el aspecto del carrito de compras y se pueden instalar varios módulos para personalizar la funcionalidad del carrito de compras. Zen Cart es un sistema de carrito de compras de código abierto cada vez más popular escrito en el lenguaje de programación PHP y está disponible para los usuarios de DreamHost como instalación de un clic.

Ten en cuenta que aún tendrás que hacer arreglos para recibir y procesar pagos de manera segura. Consulta el artículo de Generalidades de certificados SSL para obtener más información.
