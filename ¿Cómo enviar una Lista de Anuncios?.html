Generalidades

Este artículo describe cómo enviar una Lista de Anuncios dentro de tu panel DreamHost.

Enviar una lista

  1. Navega hasta la página Lista de Anuncios.
  2. A la derecha de tu lista, haz clic en el botón Mail.
  3. Ingresa lo siguiente:
    • Remitente: La dirección de correo que envías el mensaje no se puede cambiar, pero puede cambiar el nombre From si es necesario.
    • Asunto: Ingresa la línea de asunto de la publicación que estás enviando. Este campo solo acepta texto sin formato.
    • Tu mensaje: Ingresa el cuerpo del mensaje aquí. Este campo aceptará otros caracteres no estándar, a diferencia del campo "Asunto" anterior.
    • ¿Es esto HTML? Si ingresa HTML en el campo "Tu mensaje", debes marcar la casilla de verificación junto a Preview as HTML in a new window. Para obtener una vista previa del mensaje antes de enviarlo, haz clic en el enlace titulado Preview as HTML en una nueva ventana.
    • Pie de página para cancelar la suscripción: Este es el mensaje incluido automáticamente en la parte inferior de todas tus publicaciones para permitir a los usuarios darse de baja de la lista si así lo desean. El mensaje a la derecha de este campo es el mensaje de cancelación de suscripción que has configurado actualmente para las publicaciones de tu lista. Si deseas editar ese mensaje, haz clic en el pequeño botón ‘edit debajo de ‘Unsubscribe Directions Footer:’.
    • Enviar anuncio a la hora: Ingresa la fecha y la hora en que deseas enviar la publicación a los usuarios suscritos. El campo utiliza un reloj de 24 horas en lugar del reloj de 12 horas y está configurado en Hora del Pacífico (EE. UU.).
  4. Una vez que hayas configurado tu mensaje, haz clic en el botón Submit Announcement Now para enviar tu Lista de Anuncios.

¿Puedo enviar mensajes codificados en HTML a mi lista?

Si. Marca la casilla titulada Es este HTML que permite que las etiquetas HTML en tu mensaje se interpreten como HTML para las personas cuyos clientes de correo tienen capacidades HTML. Debes pegar tu código HTML directamente en el área de texto que dice "Your message’.

Además, incluso si algunas personas en tu lista no pueden leer correos con formato HTML, tu mensaje se envía como un mensaje de varias partes: aquellos usuarios con clientes de correo electrónico que no pueden leer HTML recibirán una versión de texto sin formato.

Las etiquetas <html> no deben incluirse en el campo ‘Your message’. Si se agregan, no se representa ningún código HTML en el pie de página.

¿Cómo publican los miembros en la lista?

Ellos no pueden. Esta es una Lista de Anuncios, no una Lista de Discusión. Si necesitas crear una Lista de Discusión, visita el siguiente artículo.
