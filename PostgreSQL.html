Generalidades

Este artículo contiene información/ejemplos que usan usuarios root/sudo/admin. Si necesitas acceso a sudo/admin, debes actualizar a un Servidor Dedicado.

PostgreSQL es un producto de base de datos SQL comparable a MySQL. PostgreSQL es más comúnmente la plataforma para PostGIS, un conjunto de extensiones GIS para manejar datos geoespaciales. DreamHost no es compatible con PostgreSQL. Siempre puedes agregar una sugerencia en el foro de usuarios para agregar soporte.

Instalación de PostgreSQL en un Servidor Dedicado

El soporte de DreamHost no puede proporcionar asistencia para estas instrucciones. Si algo no funciona según lo previsto, debes consultar el foro de DreamHost para obtener ayuda del usuario del foro. El soporte no puede ayudar de ninguna manera con esta configuración.

  1. Usa SSH para iniciar sesión en tu Servidor Dedicado usando tu Usuario Administrador.
  2. Corre los siguientes comandos:
    [server]$ sudo apt-get update
[server]$ sudo apt-get install postgresql postgresql-contrib
  3. PostgreSQL ahora debería estar instalado. Corre el siguiente comando para ver si funciona:
    [server]$ psql --version
psql (PostgreSQL) 9.1.14
contains support for command-line editing
