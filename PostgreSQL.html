Generalidades

Este artículo contiene información/ejemplos que usan usuarios root/sudo/admin. Si necesitas acceso a sudo/admin, debes actualizar a un Servidor Dedicado.

PostgreSQL es un producto de base de datos SQL comparable a MySQL. PostgreSQL es más comúnmente la plataforma para PostGIS, un conjunto de extensiones GIS para manejar datos geoespaciales. DreamHost no es compatible con PostgreSQL.

Instalar PostgreSQL en un Servidor Dedicado

El soporte de DreamHost no puede proporcionar asistencia con esta configuración.

  1. Usa SSH para iniciar sesión en tu Servidor Dedicado usando tu Usuario Administrador.
  2. Corre los siguientes comandos:
    [server]$ sudo apt-get update
[server]$ sudo apt-get install postgresql postgresql-contrib
  3. Corre el siguiente comando para ver si funciona. PostgreSQL ahora debería estar instalado:
    [server]$ psql --version
psql (PostgreSQL) 9.1.14
contains support for command-line editing
