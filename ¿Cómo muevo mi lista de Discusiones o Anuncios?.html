Generalidades

Este artículo describe cómo migrar una lista de Discusiones o Anuncios existente de una cuenta a otra.

No elimines el dominio de tu cuenta hasta que el soporte de DreamHost haga una copia de seguridad de tus listas. Si eliminas el dominio antes de que se haga una copia de seguridad de las listas, es posible que los datos se pierdan de forma permanente.

Haciendo una copia de seguridad de tu sitio

Para mover una lista a una nueva cuenta, eventualmente debes eliminar tu sitio de la cuenta actual. Antes de hacer esto, asegúrate de hacer una copia de seguridad de todos tus datos. Visita la siguiente página para obtener instrucciones paso a paso:

  • ¿Cómo puedo hacer una copia de seguridad manual de mis datos?

Migrando una lista de Discusiones

Sigue cada paso para evitar cualquier pérdida de datos al migrar una lista de Discusión:

  1. Asegúrate de que la nueva cuenta no tenga problemas para agregar el dominio a su panel. Esto significa que la nueva cuenta no puede usar una promoción activa ni tener un referido en la lista. Si no estás seguro de esto, contacta el equipo de soporte. Puedes hacer esto en la página Contact Support.

    Asegúrate de que la solicitud para exportar el contenido de la lista se envíe desde la dirección de correo electrónico principal de la cuenta antigua. Esto es obligatorio antes de que soporte pueda migrar la lista.

  2. Hazle saber al equipo de soporte que deseas migrar tu lista a otra cuenta. Especifica el nombre de lista exacto y el número de cuenta de la nueva cuenta a la que migrar.
  3. Solicítale al propietario de la "Cuenta receptora" que se comunique con el servicio de asistencia desde la dirección de correo electrónico desde la que te registraste. El propietario de esta cuenta debe declarar que aprueba la transferencia del dominio y la o las listas de discusión a tu cuenta.
    • El soporte de DreamHost puede hacer una copia de seguridad de tu lista.
    • El soporte te notificará que puedes eliminar la lista y el dominio de tu panel.

Recreando las listas

  1. Una vez que haya eliminado el dominio de la cuenta anterior, agréguelo a la cuenta nueva.
  2. Recrea la lista en la página Discussion Lists. El siguiente artículo explica como:
    • Creando una lista de Discusión
  3. Después de crear la nueva lista, haz que el nuevo propietario de la cuenta se comunique con el soporte de DreamHost desde la dirección de correo electrónico principal de la nueva cuenta y solicita que la lista de respaldo se migre a la nueva lista.
  4. El soporte terminará de importar la lista por ti y te notificará cuando haya terminado.

Migrando una lista de Anuncios

Primero asegúrate de hacer una copia de seguridad de los datos de tu sitio como se mencionó anteriormente. Una vez completado, procede con los siguientes pasos.

  1. Asegúrate de que la nueva cuenta no tenga problemas para agregar el dominio a su panel. Esto significa que la nueva cuenta no puede usar una promoción activa ni tener un referido en la lista. Si no estás seguro de esto, comunícate con el equipo de soporte.
  2. Solicita al propietario de la "Cuenta receptora" que se comunique con el equipo de soporte desde la dirección de correo electrónico desde la que se registró. El propietario de esta cuenta debe declarar que aprueba la transferencia del dominio y la o las listas de anuncios a su cuenta.
  3. El propietario de la "Cuenta antigua" debe ponerse en contacto con el servicio de asistencia para confirmar que desea que la Lista de anuncios se migre a la nueva cuenta.
  4. Espera a que el soporte responda. NO ELIMINES TU DOMINIO.
  5. Solo después de que el soporte se haya puesto en contacto contigo para notificarte que la lista está respaldada y que está bien eliminar el dominio, puedes proceder.

El soporte luego completará el proceso.
