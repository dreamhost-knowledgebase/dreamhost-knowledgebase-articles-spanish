El módulo announcement_list de API te permite administrar Listas de Anuncios y suscriptores.

Comandos

announcement_list-list_lists

Descarga una lista de todas las listas de anuncios que tienes en tu(s) cuenta(s).

Comando announcement_list-list_lists
Valotrd (ninguno)
Resultado

success
account_id listname domain name start_date max_bounces num_subscribers
1 fun example.com "DH Fun" <fun@dreamhost.com> 2007-12-13 5 102
1 announcements dreamhost.com "DH Announcements" <announcements@dreamhost.com> 2001-05-09 6 302420
Posibles erorres (ninguno)

announcement_list-list_subscribers

Volcar una lista de todos los suscriptores a una lista de anuncios.

Comando announcement_list-list_subscribers
Valores

listname : El nombre de la Lista de Anuncios, como anuncios.
domain : EL dominio de la Lista de Anuncios, como example.com
Resultado

success
email confirmed subscribe_date name num_bounces
josh@gmail.com 1 2007-12-13 16:55:15 Josh 0
joe@hotmail.com 1 2007-12-13 16:55:08 Joe 0
john@yahoo.com 1 2007-12-19 09:16:32 John 0
Posibles errores

no_listname
no_domain
no_such_listname
no_such_domain

announcement_list-add_subscriber

Envía un correo de suscripción voluntaria a una dirección de correo para pedirles que se suscriban a una Lista de anuncios.

Comando announcement_list-add_subscriber
Valores

listname : El nombre de la Lista de Anuncios, como anuncios.
domain : El dominio de la Lista de Anuncios, como example.com
email : El correo para suscribirse, como test@test.com
name : El nombre de la persona, como Josh Jones (opcional).
Resultado

success
sent_opt_in_email
Posibles errores

no_listname
no_domain
no_such_listname
no_such_domain
no_email
invalid_email (puede tener detalles después de una pestaña)
already_subscribed
email_may_not_be_added_to_any_dreamhost_list_ever
email_requested_to_be_added_in_last_two_days_already (may have specifics after a tab)
internal_error_opting_in

announcement_list-remove_subscriber

Elimina inmediatamente una dirección de correo de una lista de anuncios.

Comando announcement_list-remove_subscriber
Valores

listname : El nombre de la Lista de Anuncios, como anuncios.
domain : El dominio de la Lista de Anuncios, como example.com
email : El correo para desuscribirse, como test@test.com
Resultado

success
unsubscribed
Posibles errores

no_listname
no_domain
no_such_listname
no_such_domain
no_email
not_subscribed
internal_error_unsubscribing

announcement_list-post_announcement

Publica un anuncio en una lista de anuncios.

Comando announcement_list-post_announcement
Valores

listname : El nombre de la Lista de Anuncios, como anuncios.
domain : EL dominio de la Lista de Anuncios, como example.com
subject : El asunto del mensaje (opcional).
message : El texto del mensaje a enviar.
name : El nombre para la lista a usar, como: "nombre de lista" <lista@example.com>
stamp : El tiempo a enviar el mensaje, como 2009-05-28 19:40:00 (opcional).
charset : El caracter configurado en el cual el mensaje es codificado (opcional).
type : El formato del mensaje, tanto texto o HTML (opcional).
duplicate_ok: Ya sea para permitir que se envíen mensajes duplicados, como 1 o 0 (opcional).
Resultado

success
posted
Posibles errores

no_listname
no_domain
no_such_listname
no_such_domain
message_required
invalid_stamp
invalid_type
invalid_name
duplicate_message (may have specifics after a tab)
internal_error_updating_list
internal_error_sending_mail
