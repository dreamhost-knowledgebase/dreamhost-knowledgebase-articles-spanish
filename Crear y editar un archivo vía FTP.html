Generalidades

DreamHost ofrece un cliente FTP incorporado en el panel. Visita el siguiente artículo para obtener más detalles:

  • Usar el administrador de archivos del sitio web

Puedes usar un cliente FTP para crear y editar archivos para tu sitio web. Esta es una forma más fácil de crear y editar archivos en comparación con el uso de la línea de comandos. Puedes usar cualquier cliente FTP que desees, pero este artículo usa el cliente FTP Filezilla. Visita la siguiente página para descargar este programa gratuito:

  • Descargar Filezilla

Si deseas obtener instrucciones sobre cómo editar un archivo a través de SSH, consulta el siguiente artículo:

  • Crear y editar un archivo via SSH

¿Qué explica este artículo?

Este artículo solo te brinda un ejemplo de cómo crear y editar un archivo usando un cliente FTP.

Este archivo creado es llamado .htaccess, pero puedes crear cualquier archivo que desees.

Este artículo NO te dice qué poner en el archivo.

Al seguir estas instrucciones, asegúrate de elegir tu archivo que deseas modificar y editalo con los contenidos que necesites.

Crear un archivo

Sigue todos los pasos a continuación para iniciar sesión en tu servidor y crear un archivo.

  1. Visita el artículo Generalidades sobre FTP y credenciales para iniciar sesión en tu servidor usando un cliente FTP.
  2. Asegúrate de que tu cliente esté configurado para mostrar archivos ocultos.
  3. Una vez que hayas iniciado sesión, confirma estar en el directorio correcto para crear el archivo. Puedes ver el directorio en el panel derecho. Es un campo titulado "Sitio remoto". En este ejemplo, el directorio es /home/example_username/example.com.
  4. Haz clic derecho y elige Create new file.
  5. Aparece un mensaje para que ingreses el nombre del archivo. En este ejemplo, se está creando un archivo llamado .htaccess. Haz clic en el botón OK para guardar.
  6. Ahora puedes ver que el archivo vacío existe en el servidor.

Editar un archivo

Sigue todos los pasos a continuación para editar un archivo existente.

  1. Visita el artículo Generalidades sobre FTP y credenciales para iniciar sesión en tu servidor utilizando un cliente FTP.
  2. Una vez que hayas iniciado sesión, confirma estar en el directorio correcto para crear el archivo. Puedes ver el directorio en el panel derecho. Es un campo titulado "Sitio remoto". En este ejemplo, el directorio es /home/example_username/example.com.
  3. Haz clic derecho en el archivo y elige View/Edit
  4. Aparece un mensaje que te pide que elijas un editor de texto. Puedes elegir un editor específico que ya hayas instalado, pero la opción más sencilla es usar el editor predeterminado.

    Para Windows, este programa es el Bloc de notas.

    Para Mac, este programa es el Editor de texto o TextEdit.

  5. Agrega el texto que necesitas al archivo.
  6. Cuando hayas terminado de editarlo, haz clic en File > Save.
  7. Aparece un aviso. Marca la casilla y haz clic en Yes.

El archivo ahora está actualizado en tu servidor web.

Eliminar un archivo

Para eliminar un archivo existente:

  1. Revisa el artículo Generalidades de FTP para iniciar sesion usando un cliente FTP.
  2. Una vez estes logueado, asegúrate que estás en el mismo directorio que el archivo.
    Puedes ver el directorio en el panel derecho. Es un campo llamado 'Sitio Remoto':
  3. Da clic derecho en el archivo y selecciona Delete
  4. Si el cliente FTP te pregunta confirmar la acción, como muestra arriba, da clic a Si para eliminar el archivo.
