Generalidades

Hay varias formas de descargar archivos de DreamObjects, principalmente según tus preferencias, habilidades y permisos establecidos en los objects.

DreamObjects admite permisos básicos en el panel y Listas de Control de Acceso (ACL) compatibles con Amazon S3 en buckets y objects. Los permisos se pueden establecer como públicos o privados.

  • Cualquiera puede acceder a buckets y objects públicos.
  • Solo se puede acceder a buckets y objects privados a través de una URL firmada o la API con las credenciales adecuadas.

Descargar archivos en el panel DreamHost

Navega a la página de DreamObjects. Desde ahí, haz clic en tu nombre de usuario DreamObjects para ver tus Buckets. Luego haz clic en el botón Ver objetos para abrir el 'Visor de Objetos'. Desde ahí, navega al archivo que deseas descargar. Haz clic en el object, luego haz clic en el enlace de descarga a la derecha para guardar el archivo.

Descargar un archivo con una URL firmada

El object puede tener permisos PUBLICO o PRIVADO establecidos en él. Si se configuran los permisos PRIVATE, solo puedes descargar el archivo usando una URL firmada. Esto permite acceder al objects independientemente de sus permisos.

Navega a tu objects en el 'Visor de Objetos'. Haz clic en él para ver la dirección 'COMPARTIR URL'. Haz clic derecho en el enlace de descarga para copiar una URL. Por defecto, este enlace es válido por una hora. 

Descargar un archivo con permisos PÚBLICOS

Puedes hacer clic en el botón 'toggle' para cambiar los permisos a PÚBLICOS o usar un cliente como Cyberduck para ello.

Un object con permisos de lectura PUBLIC se puede descargar a través de cualquier navegador web simplemente navegando hasta su URL. Por ejemplo, un object llamado 'MY_OBJECT' en el bucket llamado 'MY_BUCKET' se puede descargar navegando a esta URL:

https://objects-us-east-1.dream.io/MY_BUCKET/MY_OBJECT

Descargar usando un cliente

La mejor opción para descargar objects es mediante un cliente. Esto te permite descargar depósitos completos y todos los objetos dentro de ellos. La siguiente sección provee una lista de clientes.

  • Clientes de DreamObjects

Limites de carga

Los objects están limitados a 10 TB, pero deben cargarse en fragmentos de 5 GB. Ten en cuenta que no todos los clientes pueden realizar cargas de varias partes (multi-part uploads). Para cualquier valor superior a 5 GB, debes utilizar un cliente que admita cargas de varias partes (multi-part uploads).
