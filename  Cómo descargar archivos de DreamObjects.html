Generalidades

Hay una amplia variedad de formas de descargar archivos de DreamObjects, principalmente según tus preferencias, habilidades y permisos establecidos en los objects.

DreamObjects admite permisos básicos en el panel, así como Listas de Control de Acceso (ACL) compatibles con Amazon S3 en buckets y objects. En el panel, los permisos se pueden establecer como públicos o privados.

  • Cualquiera puede acceder a buckets y objects públicos.
  • Solo se puede acceder a buckets y objects privados a través de una URL firmada o mediante la API con las credenciales adecuadas.

Descargar archivos en el panel DreamHost

Navega a la página de DreamObjects. Desde ahí, haz clic en tu nombre de usuario DreamObjects para ver tus Buckets. Luego haz clic en el botón 'View Objects' para abrir el 'Object Viewer'. Desde ahí, navega al archivo que deseas descargar. Haz clic en el object, luego haz clic en el enlace de descarga a la derecha para guardar el archivo.

Descargar un archivo con una URL firmada

El object puede tener permisos PUBLIC o PRIVATE establecidos en él. Si se configuran los permisos PRIVATE, solo puedes descargar el archivo usando una URL firmada. Esto permite acceder al objects independientemente de sus permisos.

Navega a tu objects en el 'Object Viewer'. Haz clic en él para ver la dirección 'SHARE URL'. Haz clic derecho en el enlace de descarga para copiar una URL. Por defecto, este enlace es válido por una hora. 

Descargar un archivo con permisos PUBLIC

Puedes hacer clic en el botón 'toggle' para cambiar los permisos a PUBLIC. También puedes cambiar los permisos utilizando un cliente como Cyberduck.

Un object con permisos de lectura PUBLIC se puede descargar a través de cualquier navegador web simplemente navegando hasta su URL. Por ejemplo, un object llamado 'MY_OBJECT' en el bucket llamado 'MY_BUCKET' se puede descargar navegando a esta URL:

https://objects-us-east-1.dream.io/MY_BUCKET/MY_OBJECT

Descargar usando un cliente

La mejor opción para descargar objects es mediante un cliente. Esto te permite descargar depósitos completos y todos los objetos dentro de ellos. Mira la siguiente sección para obtener una lista de clientes.

  • Clientes de DreamObjects

Limites de carga

Los objects están limitados a 10 TB, pero deben cargarse en fragmentos de 5 GB. Ten en cuenta que no todos los clientes pueden realizar cargas de varias partes (multi-part uploads). Para cualquier valor superior a 5 GB, debes utilizar un cliente que admita cargas de varias partes (multi-part uploads).
