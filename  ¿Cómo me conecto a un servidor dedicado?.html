Conectarse a un servidor dedicado

Si el nombre de tu servidor dedicado es ds12345, puedes conectarte a él antes de agregar cualquier dominio usando el siguiente nombre de host:

ds12345.dreamservers.com

Una vez que se agrega tu dominio, puedes usarlo para el nombre de host. Pero puedes continuar usando el nombre del servidor si lo deseas. Ve los siguientes artículos para obtener información sobre cómo iniciar sesión en tu servidor:

  • Generalidades sobre SSH
