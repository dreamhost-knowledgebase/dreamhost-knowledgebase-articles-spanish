Generalidades

Las instancias de DreamCompute creadas con las imágenes proporcionadas por DreamHost no tendrán una contraseña establecida para sus usuarios predeterminados. Esto significa que no puedes iniciar sesión en estas instancias a través de la consola sin establecer una contraseña para el usuario predeterminado. Revisa el siguiente artículo para obtener instrucciones sobre cómo hacerlo.

  • Establecer una contraseña para el usuario predeterminado

VNC o SPICE se usa para ver la salida de la consola de una instancia, independientemente de si el registro de la consola tiene salida o no. Esto permite transmitir la actividad del teclado y el mouse hacia y desde una instancia.

DreamCompute admite noVNC para el acceso remoto de una instancia a través de una consola. noVNC es un cliente VNC en el navegador implementado usando HTML5 Canvas y WebSockets.

Puedes acceder a una instancia a través de una consola remota generando una URL única con el cliente de línea de comandos de OpenStack o accediendo a las propiedades de la instancia en el panel de DreamCompute.

El soporte de DreamHost no tiene acceso a tu instancia de DreamCompute. Se recomienda que sigas los pasos del artículo Cómo evitar quedarse fuera de las instancias de DreamCompute para evitar quedar bloqueado en tu instancia. Si no estableces una contraseña de usuario y tus claves ya no funcionan, no habrá forma de acceder a la instancia.

Si necesitas el soporte de DreamHost para solucionar problemas de tu instancia, debes proporcionar las credenciales de inicio de sesión para la instancia. Revisa el siguiente artículo para obtener más detalles:

  • Compartir contraseñas con el soporte de DreamHost

Usar el CLI de OpenStack

Para acceder a una instancia a través de una consola remota, corre el siguiente comando:

[user@localhost]$ openstack console url show INSTANCE_NAME
+-------+--------------------------------------------------------------------------------+
| Field | Value                                                                          |
+-------+--------------------------------------------------------------------------------+
| type  | novnc                                                                          |
| url   | https://iad2.dreamcompute.com:6080/vnc_auto.html?token=c83ae3a3-15c4-4890-8d45 |
+-------+--------------------------------------------------------------------------------+

El comando devuelve una URL desde la que puedes acceder a tu instancia.

Usar del panel de DreamCompute

  1. Navega a la página DreamCompute.
  2. Haz clic en el botón View Dashboard.
  3. A la izquierda, haz clic en Compute.
  4. Desde el menú desplegable, haz clic en Instances.
  5. Haz clic en la instancia de la consola a la que deseas acceder.
  6. Haz clic en la pestaña Console.

    Si tu consola no usa una GUI, deberías poder interactuar con la instancia desde aquí con tu ratón y teclado.

    Si la consola no parece responder o está usando una GUI, deberás hacer clic en el enlace Click here to show only console.
