Generalidades

Una IP única es una dirección IP que es específica de tu dominio. Cuando agregas hosting a un dominio/subdominio, se le asigna una IP en el servidor Apache que se comparte con otros sitios. Esta dirección IP compartida suele ser la única IP que necesitará. Sin embargo, ciertas aplicaciones pueden requerir que agregue una IP única a su sitio. Puedes agregar una IP única a cualquier dominio/subdominio totalmente alojado alojado en DreamHost en la página Administrar Dominios.

DreamHost no puede ofrecer direcciones IP estáticas. Esto significa que la IP de tu sitio (incluidas las IP únicas) está sujeta a cambios si es necesario. DreamHost no garantiza que una dirección IP nunca cambie. Sin embargo, cuando se produce un cambio de IP, se te notificará por correo electrónico de manera oportuna.

¿Necesito una IP única?

En la mayoría de los casos, no. Las IP únicas se usan con mayor frecuencia con dominios que tienen un certificado SSL habilitado. Si bien es posible utilizar un alojamiento seguro sin una IP única, algunos navegadores más antiguos que no admiten la Indicación de Server Name muestran una advertencia de certificado al ver su sitio. Si bien no es una lista completa, esto incluye:

  • Versiones de Windows XP de Internet Explorer.
  • El navegador predeterminado en Android 2.4 "Gingerbread" y versiones anteriores. (4.0 "Ice Cream Sandwich" y posteriores funcionará correctamente).
  • Varios navegadores móviles más antiguos, incluidos BlackBerry, Symbian OS y versiones anteriores de Opera Mobile.

Si bien es muy poco probable, si esperas que un número significativo de visitantes a tu sitio seguro use estos navegadores, necesitarás una IP única. Ve el siguiente enlace para los navegadores más utilizados por año:

  • Los buscadores más populares

Correo electrónico

Las IP únicas son solo para sitios web. Agregar una IP única no tendrá ningún efecto en tu servicio de correo ya que tu correo está en una máquina completamente diferente. Actualmente no hay forma de ajustar la dirección IP de un servidor de correo.

Otros casos de uso

Se necesita una IP única cuando:

  • Si estás interesado en acceder al sitio directamente con una dirección IP (esto es útil cuando el dominio en cuestión aún no utiliza los nameservers/registros DNS de DreamHost). Si intentas acceder a tu sitio a través de la IP del servidor compartido, se abre la página DreamHost "Sitio no encontrado", sobre la que puede leer en el siguiente artículo:
    • Página web no encontrada
  • Si otro dominio alojado en la misma IP compartida que la tuya está bloqueado por ciertos firewalls restrictivos (por ejemplo, GFW), agregar una dirección IP única al sitio puede ayudarlo a omitir el filtrado (suponiendo que el filtro se base en la IP compartida). Este enfoque también puede ser de interés si deseas asegurarte de que tu sitio no esté asociado con otros sitios, independientemente de la presencia de filtros. Un ejemplo de esto se puede encontrar en el artículo "China Firewall":
    • Generalidades del Firewall de China

¿Una IP única ayuda a mi ranking de página?

No, no lo hace. Con la escasez actual de direcciones IP a medida que continúa la adopción de IPv6 (y el hecho de que hay un costo adicional asociado con agregarlas a tus dominios), es importante saber que agregar una IP única a tu dominiono afectará la clasificación de tu motor de búsqueda

Si esta es la única motivación para agregar una IP a su dominio, entonces no hay necesidad de agregar una.
