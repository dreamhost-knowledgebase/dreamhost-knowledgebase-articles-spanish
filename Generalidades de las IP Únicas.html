Generalidades

A continuación se explica qué es una IP única y cuándo utilizarla.

Antecedentes

Una IP única es una dirección IP específica de tu dominio. Cuando agregas un alojamiento a un dominio o subdominio, se le asigna una IP en el servidor, que se comparte con otros sitios. Esta dirección IP compartida suele ser la única IP que necesitará. Sin embargo, es posible que ciertas aplicaciones requieran agregar una IP única a tu sitio.

Una IP por dominio/subdominio

Cuando compras una IP Única, se aplica a un sólo dominio o subdominio. Si quieres una IP Única para el dominio (example.com) y el subdominio (blog.example.com), deberás comprar una IP Única para cada uno separadamente.

Direcciones IP Estáticas

DreamHost no puede ofrecer direcciones IP estáticas. Esto significa que la IP de tu sitio (incluidas las IP únicas) está sujeta a cambios si es necesario. DreamHost no garantiza que una dirección IP nunca cambie. Sin embargo, cuando ocurra un cambio de IP, se te notificará por correo a tiempo.

¿Necesito una IP única?

En la mayoría de los casos, no es necesario tener una dirección IP única; sin embargo, algunas situaciones pueden requerirla. Si es necesario, puedes agregar una IP única a cualquier dominio o subdominio completamente alojado en la página Administrar Sitios Web.

No se puede agregar una dirección IP única a los subdominios dreamhosters.com ni a los sitios de DreamPress.

Permitir a los buscadores antiguos visitar tu sitio

El alojamiento Seguro se puede usar sin una IP Única, pero algunos navegadores más antiguos que no admiten el Server Name Indication mostrarán una advertencia de certificado al ver tu sitio.

Si bien es muy poco probable, si espera una cantidad significativa de visitantes a tu sitio seguro que utilizan estos navegadores más antiguos, necesitarás una IP Única

Aceder a un sitio directamente

Si intentas acceder a tu sitio a través de la IP del servidor compartido, aparecerá el mensaje Site not found de DreamHost.

Es necesaria una IP Única si estás interesado en acceder al sitio directamente con una dirección IP. Esto también es útil cuando el dominio aún no utiliza los nameservers o registros DNS de DreamHost.

Saltar los firewalls restrictivos

Una IP Única es necesaria si otro dominio alojado en la misma dirección IP compartida que la tuya está bloqueado por ciertos firewalls restrictivos (por ejemplo, Great Firewall). Agregar una IP Única al sitio puede ayudarte a eludir el filtrado (suponiendo que el filtro se basa en la dirección IP compartida). Este enfoque también puede ser de interés si deseas asegurarte de que tu sitio no esté asociado con otros sitios, independientemente de la presencia de filtros. Puedes encontrar un ejemplo de esto en el artículo Firewall de China.

Agregar una IP Única al servidore de correo

Las IP Única son solo para sitios web. Agregar una dirección IP única no afectará tu servicio de correo, ya que el mismo está en un servidor completamente diferente. Actualmente no hay forma de ajustar la IP de un servidor de correo.

Mejora el ranking de mi página

Agregar una IP única no mejorará la clasificación de tu sitio en los motores de búsqueda.
