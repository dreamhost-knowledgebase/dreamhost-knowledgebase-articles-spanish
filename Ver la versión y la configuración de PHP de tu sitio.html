Generalidades

A continuación se explica cómo utilizar un archivo phpinfo.php para ver la versión de PHP que ejecuta tu sitio.

Antecedentes

El lenguaje PHP tiene una función especial incorporada llamada phpinfo() que recupera información sobre el entorno PHP que se ejecuta en un sitio web. Puedes ver esta información creando un archivo phpinfo.php. Cuando se ve en un navegador, aparecen detalles como los siguientes:

  • Versión de PHP
  • Extensiones de PHP
  • Rutas
  • Valores locales de las opciones de configuración

Cómo crear un archivo phpinfo.php

Esta sección explica cómo crear este archivo en el servidor web usando FTP o SSH.

Usando FTP

  1. Accede a tu servidor usando una herramienta de FTP como Filezilla.
  2. Crea un archivo en blanco titulado phpinfo.php. Asegúrate de que tiene la extensión .php y no la extensión .txt.
  3. Edita el archivo phpinfo.php y agrega las siguientes líneas: 
    <?php 
     phpinfo(); 
    ?>
  4. Visita el archivo en un navegador. Por ejemplo: example.com/phpinfo.php

La página muestra una lista de toda la configuración de PHP que tiene tu sitio en el momento.

Usando SSH

  1. Inicia sesión en tu servidor a través de SSH.
  2. Cambia al directorio de tu sitio web.
  3. Crea un archivo llamado phpinfo.php.
  4. Agrega las siguientes líneas a este archivo: 
    <?php 
     phpinfo(); 
    ?>
  5. Guarda el archivo y regresa a tu shell.
  6. Visita el archivo en un buscador. Por ejemplo, example.com/phpinfo.php.

La página muestra una lista de toda la configuración de PHP que tiene tu sitio en el momento.
