Generalidades

Hay varios plugins de almacenamiento en caché que puede utilizar para sus sitios de WordPress, lo que puede ayudar a mejorar el tiempo de carga de su sitio web y el rendimiento general. También pueden ayudar a su sitio a manejar grandes picos de tráfico como el efecto Slashdot. Lo siguiente es una descripción general de los plugins recomendados que se puede usar en los servidores de DreamHost:

WP Super Cache

WP Super Cache crea una caché de archivos estática de su sitio y carga eso en lugar de usar PHP para generar la página. Recogiendo donde WP-Cache dejó, este plugin incluye características avanzadas, así como un sistema de plugins y ganchos. WP Super cache ya está incluido y activado con todas las nuevas instalaciones Deluxe de One Click WordPress desde el panel Web de DreamHost. Si estás en hosting compartido, esta es la mejor opción.

Consulte el siguiente artículo para obtener más información sobre la configuración de WP Super cache:

  • Instalación de WP Super Cache

Hyper Cache

Hyper Cache es un plugin de caché efectivo, fácil de usar y es una alternativa sobresaliente para la mayoría de los usuarios de WordPress.

Sólo DreamPress

DreamPress utiliza una caché de servidor incorporada y la caché de objetos, junto con plugins preinstalados que están específicamente configurados para ayudar a WordPress a aprovechar al máximo estas características. No es necesario instalar ningún plugin addicional de caché al usar DreamPress.

Para obtener más información sobre las opciones de almacenamiento en caché de DreamPress, consulte el siguiente artículo:

  • Gestionando el cache de DreamPress
