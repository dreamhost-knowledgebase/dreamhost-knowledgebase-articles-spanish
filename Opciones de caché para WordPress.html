Generalidades

Hay varios plugins de almacenamiento en caché que puedes usar para tus sitios de WordPress, lo que puede ayudar a mejorar el tiempo de carga de tu sitio web y el rendimiento general. También pueden ayudar a tu sitio a manejar grandes picos de tráfico como el efecto Slashdot. Lo siguiente es una descripción general de los plugins recomendados que se puede usar en los servidores de DreamHost:

Habilitar el Almacenamiento de Caché de Página en el panel

Esta opción no está disponible (ni es necesaria) para instalaciones de DreamPress, ya que DreamPress usa su propio un servidor de caché incorporado.

Los sitios WordPress pueden activar el Almacenamiento de Caché de Páginas en el panel, lo que activa el plugin de Simple Caché

  1. Navega a la página Administrar Sitios Web.
  2. Da clic en el botón Administrar para abrir la página de Configuraciones de Dominios, la cual te permite ajustar múltiples configuraciones de tu sitio.
    • Si estás en la Vista de Malla, da clic en el botón Administrar en la parte inferior de la caja del sitio.
    • Si estás en la Vista de Lista, da clic en el botón Administrar a la derecha de tu dominio.
  3. Haz clic en la pestaña Sitio Web.
  4. Baja hasta la sección WordPress.
  5. A la derecha de Almacenamiento de Caché de Página, habilita esta opción.
    Aparece una marca de verificación cuando está activo.

WP Super Cache

WP Super Cache crea una caché de archivos estática de su sitio y carga eso en lugar de usar PHP para generar la página. Recogiendo donde WP-Cache dejó, este plugin incluye características avanzadas, así como un sistema de plugins y ganchos. WP Super cache ya está incluido y activado con todas las nuevas instalaciones Deluxe de WordPress desde el panel Web de DreamHost. Si estás en Shared Hosting, esta es la mejor opción.

Revisa los siguientes artículos para obtener más información sobre la configuración de WP Super cache:

El plugin Docket Cache

El plugin Docket Cache — Object Cache Accelerator es un caché de objetos a nivel de sistema de archivos que se usa como alternativa a servicios como Redis o Memcache. Almacena datos al convertir objetos PHP en código PHP simple, lo que ayuda a mejorar el rendimiento. 

Revisa el siguiente artículo para más información sobre el plugin Docket Cache:

  • Usar el plugin Docket Cache

Solo DreamPress

DreamPress usa una caché de servidor incorporado junto con plugins preinstalados que están específicamente configurados para ayudar a WordPress a aprovechar al máximo estas características. No es necesario instalar ningún plugin addicional de caché al usar DreamPress.

Para obtener más información sobre las opciones de almacenamiento en caché de DreamPress, consulte el siguiente artículo:
