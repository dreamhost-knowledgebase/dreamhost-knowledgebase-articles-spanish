Administrar Sitios Web

La página Administrar Sitios Web proporciona una presentación gráfica y mejorada de todos tus dominios, ofreciéndote una vista optimizada donde puedes hacer fácilmente ajustes de funciones y configuraciones a cualquiera de tus dominios en DreamHost.

Revisa los siguientes artículos para más detalles:

  • Usar la página 'Sitios Web' del panel
  • La página Administrar Sitios Web

Archivos y Usuarios SFTP

La página Archivos y Usuarios SFTP es donde puedes acceder a los archivos de tu dominio y agregar nuevos usuarios de FTP/SFTP/Shell a tu servidor.

Revisa el siguiente artículo para más detalles:

  • Diferentes tipos de usuarios de DreamHost

Bases de Datos MySQL

La página Bases de Datos MySQL es donde puedes agregar/administrar una base de datos que se puede usar con tu sitio web.

Visita el siguiente artículo para más detalles:

  • Generalidades de MySQL

Certificados Seguros

En la página Certificados Seguros, puedes agregar un certificado SSL a tu dominio, lo que permite acceder al dominio mediante el protocolo HTTPS.

Para más información dirígete al siguiente artículo:

  • Generalidades sobre los certificados SSL
