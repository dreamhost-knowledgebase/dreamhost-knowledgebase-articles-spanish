Generalidades

A continuación se describe cómo cambiar el plan de facturación para varios servicios de DreamHost que puedas estar utilizando, como:

  • Shared Starter y Shared Unlimited
  • VPS
  • DreamPress
  • Servidor Dedicado
  • Correo Electrónico Pago
  • Remixer

Cada uno de los servicios se describe a continuación, ya que los pasos para realizar cambios difieren entre los tipos de planes.

Reembolsos al cambiar de un plan anual a un plan mensual

Ten en cuenta que si cambias tu plan de facturación de Anual a Mensual, no tienes derecho a un reembolso. Sin embargo, se proporcionará un crédito a tu cuenta de DreamHost por el saldo restante. Ve el siguiente artículo para ver las políticas de reembolso de DreamHost.

  • Reembolsos

Shared Starter y Shared Unlimited

Para hacer cambios a tu plan Shared Starter o Shared Unlimited en la página Administrar Cuenta selecciona una opción bajo la columna Service Plans. Revisa el siguiente artículo para obtener más información:

  • Cambiar tu plan de facuración de Shared Starter o Shared Unlimited

Los reembolsos solo se permiten durante los primeros 97 días, o durante el período de gracia de 30 días después de la fecha de renovación anual.

VPS

Para modificar tu plan VPS, navega a la página VPS. Haz clic en el botón Cambiar Plan a la derecha de tu VPS actual para ver las opciones de tu plan. Consulta el siguiente artículo para obtener más información:

  • Cambiar tu plan de VPS o tu plan de facturación

DreamPress

Para hacer cambios a tu plan DreamPress, abre la página WordPress Administrado. Haz clic en el botón Administrar del plan que quieres editar, y luego haz clic en el botón Editar/Eliminar Plan. Revisa el siguiente artículo para más información:

  • Cambiar un plan DreamPress

Servidor Dedicado

Pónte en contacto con el área de soporte si deseas migrar a un plan de facturación/configuración de servidor dedicado diferente. El área de soporte migrará los datos por ti. Consulta el siguiente artículo para obtener más información:

  • ¿Cómo agregar un Servidor Dedicado?

Correo Electrónico Pago

Navega a la página Administrar Correo para modificar tu plan de correo pago. Haz clic en el botón Administrar Suscripción de Correo para ver las opciones de facturación disponibles. Mira el artículo a continuación para más información:

  • Plan de correo pago

Remixer

Si quieres hacer cambios a tu plan Remixer, envía un ticket por correo.

  • Contactar a DreamHost vía correo

Mira el siguiente artículo para más información.

  • Cancelar un plan Remixer
