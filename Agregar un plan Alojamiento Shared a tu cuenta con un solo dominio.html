Generalidades

Este artículo te guía a través del proceso de agregar un plan de alojamiento Shared a un registro de dominio que hayas añadido previamente a tu cuenta.

Un plan de alojamiento permite que tu sitio web sea visible en Internet. Una vez añadido, puedes agregar servicios adicionales como WordPress, correo electrónico y un certificado SSL.

Requisitos previos

Este artículo asume que ya has comprado o transferido un registro de dominio a tu cuenta de DreamHost.

Agregar un plan de Alojamiento Shared

Para agregar un plan:

  1. Navega a la página Administrar Cuenta.
  2. En la sección titulada Planes Activos, haz clic en el enlace Agregar un Plan de Alojamiento.
  3. Elige el plan Shared Starter o Shared Unlimited.
  4. Selecciona el término de facturación.
  5. Haz clic en el botón Agregar Plan.

Cómo configurar tu dominio

Después de agregar tu plan de alojamiento, puedes proceder con la configuración de tu sitio web y sus servicios.

Agregar tu dominio

Revisa el artículo agregar un sitio web para obtener instrucciones sobre cómo agregar tu registro de dominio existente a tu nuevo plan de alojamiento. Una vez añadido, puedes agregar subdominios adicionales.

Agregar WordPress con el instalador de WordPress de DreamHost

Si deseas ejecutar WordPress en tu sitio web, el instalador de WordPress de DreamHost es una forma sencilla de instalarlo con solo unos pocos clics en el panel.

Crear una dirección de correo

Revisa el artículo Agregar un correo electrónico para obtener instrucciones sobre cómo agregar correos a tu dominio.

Instalar un certificado SSL con Let's Encrypt

Un certificado SSL ayuda a garantizar que los datos de tus visitantes permanezcan seguros e incluso puede ayudar potencialmente con las clasificaciones en los motores de búsqueda. DreamHost ofrece varios tipos de certificados SSL sobre los que puedes leer en los siguientes artículos:
