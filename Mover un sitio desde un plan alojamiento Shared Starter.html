Generalidades

Este artículo describe las diversas formas en que puedes pasar de un plan de alojamiento Shared Starter:

Mejorar a un Shared Hosting

Revisa el artículo Mejorar un plan Shared Starter a Unlimited Hosting para obtener instrucciones sobre cómo actualizar a Shared Hosting.

Mejorar a un alojamiento VPS

Revisa el artículo Agregar un VPS para obtener instrucciones sobre cómo mover tus sitios web a un nuevo plan de alojamiento VPS.

  • Al realizar la compra, puedes optar por mover TODOS tus usuarios y dominios al nuevo VPS.

Mejorar a un Servidor Dedicated

Revisa el artículo Agregar un servidor Dedicado para obtener instrucciones sobre cómo mover tus sitios web a un nuevo plan de Alojamiento Dedicado.

  • Al realizar la compra, puedes optar por mover TODOS tus usuarios y dominios al nuevo servidor Dedicado.

Mejorar a DreamPress

Si ya tienes un sitio de WordPress en Alojamiento Shared, puedes utilizar el artículo Mejorar un sitio de WordPress a DreamPress para actualizarlo a DreamPress. También puedes agregar un nuevo dominio a DreamPress utilizando el artículo Agregar DreamPress a un nuevo dominio.
