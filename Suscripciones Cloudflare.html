Generalidades

Cuando habilitas Cloudflare en su panel DreamHost, puede elegir entre dos tipos de suscripciones:

Cloudflare Basico (gratuito)
Todos los dominios habilitados para Cloudflare tienen acceso a la suscripción gratuita básica de Cloudflare sin costo.
Cloudflare Plus
La suscripción a Cloudflare Plus proporciona características adicionales como la compresión de imágenes.

Las suscripciones de Cloudflare se compran y administran desde tu panel DreamHost, y el que elijas controla qué funciones están disponibles para un dominio habilitado para Cloudflare. Algunas funciones requieren el uso del panel de control de Cloudflare.

Para obtener más información sobre Cloudflare y cómo habilitarlo, por favor visita el artículo generalidades sobre Cloudflare.

Cómo funcionan las suscripciones de Cloudflare

Cualquier suscripción que habilites para un dominio, junto con todas las características de Cloudflare para ese plan, están disponibles para TODOS sus subdominios sin costo adicional. Sin embargo, ten en cuenta que debes habilitar específicamente Cloudflare en cada subdominio.

Para explicarlo, una sola suscripción cubre example.com y cualquier subdominio, como blog.example.com. Sin embargo, un dominio diferente como dreamhostexample.com requiere una suscripción separada a Cloudflare. Puedes habilitar Cloudflare para un número ilimitado de subdominios.

No necesitas agregar 'www' a su subdominio. Por ejemplo, si tu subdominio es blog.example.com, puedes activar Cloudflare para blog.example.com.

Habilitar suscripciones de Cloudflare

Visita el artículo de Cloudflare para obtener instrucciones sobre cómo habilitar este servicio en tu dominio. Cuando agregas un servicio, verás las dos opciones de suscripción que se describen a continuación.

Cloudflare Basico (GRATUITO)

Cloudflare Basico incluye:

  • CDN global con múltiples puntos de presencia en todo el mundo
  • Optimización de contenido web front-end con RockLoader y AutoMinify
  • Amplia seguridad
  • Soporte SSL
  • Compatibilidad IPv6
  • Análisis del sitio (retraso de 24 horas)

Si actualmente tienes la opción Plus para el soporte SSL y ya no la quieres, puedes cambiar a Basic cancelando la opción Plus y luego cambiando a la opción gratuita. Para facilitar el cambio a la opción Básica, debes habilitar la opción 'Agregar WWW' en el panel DreamHost para completar el proceso. Para obtener más información sobre cómo agregar 'www' a su dominio, visita el siguiente artículo:

  • ¿Cómo agregar o remover www de mi dominio?

Cloudflare Plus

Además de todos los beneficios del servicio gratuito, el plan Cloudflare Plus incluye:

  • Rendimiento más rápido del sitio web y optimización avanzada de imágenes (mejor rendimiento móvil) a través del acceso a las funciones de Cloudflare Polish, Mirage y Lazy Loading
  • 10 reglas URL
  • Análisis del sitio (retraso de 1 hora)

Comparación de planes

Suscripciones DreamHost vs. suscripciones Cloudflare

Como socios de hosting con Cloudflare, las suscripciones a Cloudflare de DreamHost te brindan muchos de los mismos beneficios de usar los servicios de Cloudflare directamente dentro de tu panel DreamHost. No necesitas cambiar los nameservers de tu dominio a Cloudflare para obtener estos beneficios.

El plan “gratuito“ de Cloudflare frente al plan “básico (¡gratis!)“ De DreamHost

No hay diferencia entre los dos.

El plan “Pro“ de Cloudflare vs. el plan “Plus“ de DreamHost

Cloudflare también ofrece un plan “Pro” con algunas características adicionales como:

  • Cortafuegos de aplicaciones web. El plan DreamHost's Plus no tiene el Cortafuegos de Aplicaciones Web (WAF). Sin embargo, todas las cuentas Shared, VPS y DreamPress incluyen el WAF instalado en el servidor web Linux.
  • El plan Pro de Cloudflare tiene 20 vs. 10 (plan DreamHost Plus) PageRules.

Visita la página del ‘Plan pro’ de Cloudflare para más detalles.
