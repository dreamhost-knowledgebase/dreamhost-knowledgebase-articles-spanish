Generalidades

Este artículo muestra diferentes aplicaciones de Windows, Mac, y líneas de comando que puedes usar para conectarte a tu bucket de DreamObjects.

AWS CLI

AWS CLI es:

  • Una herramienta de línea de comandos para cargar, recuperar y administrar datos en Amazon S3 y otros proveedores de servicios de almacenamiento en la nube que usan el protocolo S3 como DreamHost DreamObjects.
  • Más adecuado para usuarios que no temen a la línea de comando.
  • Ideal para scripts, copias de seguridad automatizadas activadas desde cron, etc.

Echa un vistazo al tutorial de AWS CLI de DreamObjects para más información.

Cyberduck

Cyberduck es un cliente FTP, SFTP y WebDav, así como un navegador de almacenamiento en la nube para Mac y Windows. Puedes usar Cyberduck para conectarte a tu cuenta de DreamObjects.

Echa un vistazo al tutorial de Cyberduck  de DreamObjects para más información.

S3 Browser

S3 Browser es un cliente gratuito de Windows S3 que proporciona una interfaz web simple que puedes usar para almacenar y recuperar datos.

Mira el sitio S3 Browser para más información.

S3cmd

S3cmd es:

  • Una herramienta de línea de comandos para cargar, recuperar y administrar datos en Amazon S3 y otros proveedores de servicios de almacenamiento en la nube que usan el protocolo S3 como DreamObjects.
  • Más adecuado para usuarios que no temen a la línea de comando.
  • Ideal para scripts, copias de seguridad automatizadas activadas desde cron, etc.

Echa un vistazo al tutorial de S3cmd de DreamObjects para más información.

UpdraftPlus

UpdraftPlus trae copias de seguridad confiables y fáciles de usar, restauraciones y copias del sitio (clones/migraciones) a tu sitio de WordPress.

Echa un vistazo al tutorial de UpdraftPlus de DreamObjects para más información.
