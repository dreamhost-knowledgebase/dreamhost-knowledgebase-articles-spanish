Arq

Arq es una aplicación para Mac y Windows que te permite hacer copias de seguridad en línea. Las copias de seguridad de Arq se cifran con una contraseña antes de que se carguen para que tus datos estén protegidos.

Echa un vistazo al tutorial Arq de DreamObjects para más información.

AWS CLI

AWS CLI es:

  • una herramienta de línea de comandos para cargar, recuperar y administrar datos en Amazon S3 y otros proveedores de servicios de almacenamiento en la nube que usan el protocolo S3 como DreamHost DreamObjects.
  • más adecuado para usuarios que no temen a la línea de comandos.
  • ideal para scripts, copias de seguridad automatizadas activadas desde cron, etc.

Echa un vistazo al tutorial de AWS CLI de DreamObjects para más información.

CloudBerry

CloudBerry Backup es un programa de Windows que automatiza las copias de seguridad de datos encriptados y comprimidos.

Echa un vistazo al tutorial de CloudBerry de DreamObjects para más información.

Cyberduck

Cyberduck es un cliente FTP, SFTP y WebDav, así como un navegador de almacenamiento en la nube para Mac y Windows. Puedes usar Cyberduck para conectarte a tu cuenta de DreamObjects.

Echa un vistazo al tutorial de Cyberduck  de DreamObjects para más información.

Dropshare

Dropshare es un llamado menulet para OS X. Te permite arrastrar y soltar archivos, carpetas y cualquier otra cosa en tu propio servidor, DreamObjects u otros servicios en la nube. Una vez cargada, la aplicación copia el enlace al archivo en el portapapeles de tu Mac. Luego puedes compartir con quien quieras.

Echa un vistazo al tutorial de Dropshare de DreamObjects para más información.

ExpanDrive

ExpanDrive es una aplicación que te permite conectarse a DreamObjects al igual que una unidad USB. Está disponible para Mac y Windows y te permite acceder a DreamObjects desde cualquier aplicación en tu computadora.

Echa un vistazo al tutorial de ExpanDrive de DreamObjects para más información.

GoodSync

GoodSync es un sincronizador de archivos fácil, seguro y confiable. La versión gratuita está limitada a 3 trabajos o menos y 100 o menos archivos y carpetas en cada trabajo.

Muchos clientes han informado problemas con GoodSync fallando con archivos de más de 100MB. Hasta que se encuentre una solución, se recomienda que uses otro cliente.

Echa un vistazo al tutorial de GoodSync de DreamObjects para más información.

S3 Browser

S3 Browser es un cliente gratuito de Windows S3 que proporciona una interfaz web simple que puedes usar para almacenar y recuperar datos.

Ve el sitio S3 Browser para más información.

S3cmd

S3cmd es:

  • Una herramienta de línea de comandos para cargar, recuperar y administrar datos en Amazon S3 y otros proveedores de servicios de almacenamiento en la nube que usan el protocolo S3 como DreamObjects.
  • más adecuado para usuarios que no temen a la línea de comandos.
  • ideal para scripts, copias de seguridad automatizadas activadas desde cron, etc.

Echa un vistazo al tutorial de S3cmd de DreamObjects para más información.

Transmit

Transmit es un cliente FTP para Mac OS X desarrollado por Panic que también incluye capacidades para montar conexiones SFTP, Amazon S3 y WebDAV. El cliente es un shareware, por lo que es gratuito de evaluar, pero una vez finalizado el período de prueba, se limita a sesiones de siete minutos.

Echa un vistazo al tutorial de Transmit de DreamObjects para más información.

UpdraftPlus

UpdraftPlus trae copias de seguridad confiables y fáciles de usar, restauraciones y copias del sitio (clones/migraciones) a tu sitio de WordPress.

Echa un vistazo al tutorial de UpdraftPlus de DreamObjects para más información.
