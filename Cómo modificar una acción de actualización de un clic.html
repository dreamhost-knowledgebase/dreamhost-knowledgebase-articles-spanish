Generalidades

De forma predeterminada, las instalaciones con un clic están configuradas para actualizar WordPress automáticamente con cada actualización importante. Recibirás una notificación por correo electrónico una vez que se haya completado una actualización.

DreamHost recomienda que siempre mantengas habilitada la opción de actualización automática, ya que esto asegurará que tu instalación de WordPress se mantenga actualizada con las últimas configuraciones de seguridad.

Para cambiar la acción de actualización predeterminada para tu dominio, sigue estas instrucciones.

Modificar la acción de actualización de un clic

  1. Ve a One-Click Installs en el panel.
    La página One-Click Installs se abre
  2. Haz clic en el botón Manage Installed Applications para ver tus aplicaciones de un clic.
  3. Haz clic en tu nombre de dominio. 
    Aparece una página que enumera tus dominios con aplicaciones instaladas:
  4. Bajo la columna Upgrade Action, haz clic en el enlace edit.
    Verás las siguientes opciones:
  5. Haz una selección de las siguientes opciones:
    • No action: DreamHost no realizará actualizaciones de un clic para este dominio.
    • Notify me: Solo te avisa por correo electrónico una vez que hay una actualización de revisión importante disponible. No se realiza ninguna actualización.
    • Upgrade automatically: Actualiza el sitio y te envía correos electrónicos automáticamente.
    • Upgrade without notification: Actualiza el sitio pero no te envía un correo electrónico.
