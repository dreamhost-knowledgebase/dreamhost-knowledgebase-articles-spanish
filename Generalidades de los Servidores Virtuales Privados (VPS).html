Generalidades

Este artículo te brinda una visión general del plan de alojamiento VPS de DreamHost, así como los siguientes pasos para comenzar. Este tipo de plan de alojamiento está disponible como un complemento a un plan de DreamHost existente o como un plan de alojamiento independiente para tus sitios web.

¿Qué es un plan de VPS?

El Servidor Privado Virtual (VPS) de DreamHost es un plan de alojamiento premium en el cual los recursos de RAM de un usuario están protegidos de otros usuarios en la misma máquina física. Esta aislamiento de recursos ayuda a mejorar el rendimiento y mantener la estabilidad.

Esto es diferente de un servidor compartido, donde los recursos del servidor son compartidos por otros usuarios en la misma máquina.

Características

La siguiente es una lista de características a las que tendrás acceso después de agregar un VPS.

  • RAM Protegida
  • Capacidad de actualizar la RAM sin reiniciar (la degradación reiniciará el servidor)
  • Capacidad de reiniciar el servidor desde el panel
  • Se pueden ver gráficos históricos de memoria y carga en el panel.
  • Cada VPS viene con su propia dirección IP de servidor incluida. También puedes agregar una IP única a cada dominio (IPv4 o IPv6) por una tarifa adicional.
  • Se permite ejecutar procesos persistentes (a diferencia de un plan de alojamiento Shared). Independientemente de cuánto tiempo tarden en ejecutarse estos procesos, nunca son terminados por el servidor.

Preguntas Frecuentes

¿Cuánto cuesta un VPS?

Revisa la página de Alojamiento VPS para obtener información actual sobre las características y los planes de precios de VPS.

¿Se me reembolsará si cancelo un plan de alojamiento VPS?

Los reembolsos solo se aplican a los planes de alojamiento Shared.

¿Qué sistema operativo se ejecuta?

Los planes VPS corren el sistema operativo Ubuntu.

¿Las bases de datos MySQL están en el VPS?

No. Un VPS solo ejecuta archivos de sitios web. Cualquier base de datos que agregues en tu panel se configurará por defecto en un servidor MySQL compartido.

También puedes comprar un VPS MySQL, pero este sería un segundo servidor además de tu VPS.

¿Se admite el DNS Wildcard DNS?

Es posible agregar DNS comodín a tu VPS, pero esto debe ser habilitado por el soporte técnico. Por favor, visita el artículo de Wildcard DNS para más información.
