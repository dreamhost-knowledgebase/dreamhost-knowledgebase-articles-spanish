Generalidades

WordPress ofrece varios editores diferentes que puede elegir para editar su sitio web. Si estás utilizando BoldGrid para crear tu sitio, se recomienda el 'Post and Page Builder'. Este artículo te brinda una descripción general rápida de las siguientes opciones.

  • Post and Page Builder
  • Wordpress Editor 
  • Classic editor

Editando una página

  1. Inicia sesión en tu panel de WordPress.
  2. Haz clic en la pestaña 'Inspirations' en el panel izquierdo.
  3. Bajo 'Site Content', haz clic en el ícono de lápiz a la derecha de la página que deseas editar.
  4. Al principio, se te presenta la opción de configurar algunos ajustes. Haz clic en
    Okay, Let's Go!.
  5. Selecciona el editor que deseas usar. Tus opciones son:
    • Post and Page Builder — Este es el editor de BoldGrid (recomendado)
    • Wordpress Editor — Este es el editor de páginas estándar de WordPress que se encuentra en 'Pages > All Pages'.
    • Classic Editor — Similar al editor de WordPress, pero con más opciones mostradas.
    • Default — Por defecto usa Post and Page Builder.
  6.  En el paso final, haz clic en OKAY, GOT IT! para guardar tu configuración.

Elegiendo un editor

Post and Page Builder

Como se mencionó anteriormente, el 'Post and Page Builder' es el editor predeterminado de BoldGrid y se recomienda debido a su simplicidad y fácil acceso a las herramientas.

Puedes cambiar a un editor diferente haciendo clic en el menú desplegable titulado 'Post and Page Builder'.

WordPress editor

La mayoría de las opciones están ocultas, pero se puedeS acceder haciendo clic en los diferentes íconos.

Se pueden agregar bloques en la parte superior izquierda, más opciones están a la derecha.

Para cambiar a un editor diferente, haz clic en el icono de 3 puntos en la esquina superior derecha. En el menú, en 'Plugins', puedes cambiar a un editor diferente.

Classic editor

Similar al editor de WordPress, pero con más opciones mostradas por defecto.

Haz clic en el menú desplegable 'Classic Editor' en la parte superior para cambiar de editor.
