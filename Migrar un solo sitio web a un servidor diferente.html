Generalidades

Este artículo explica cómo mover un dominio entre cualquier servidor web en DreamHost. El dominio puede ejecutar una instalación con un clic, como WordPress, ya que solo se migran los archivos del servidor web. La base de datos permanece en el mismo servidor MySQL.

Mover un solo dominio a un servidor diferente

Puedes mover un solo dominio a un servidor diferente simplemente cambiando el usuario con el que se ejecuta el dominio. Puedes hacer esto editando el dominio en tu panel:

  1. Navega a la página Manage Domains.
  2. Haz clic en el botón Edit a la derecha del dominio debajo de la columna ‘Web Hosting’.
  3. Desplázate hacia abajo hasta la sección 'Users, Files, and Paths'.

    Las capturas de pantalla a continuación muestran cómo se vería al pasar a un servidor compartido o privado.


    Migrar a un servidor Compartido
    Migrar a un VPS
  4. A la derecha de ‘Run this domain under the user:',
    ingresa lo siguiente:
    • Haz clic en el menú desplegable y selecciona 'Create a New User' (recomendado).
    • Ingresa el nuevo nombre de usuario.
    • Asegúrate que el servidor correcto está seleccionado.
    • Deja la casilla 'Move files to the new user?' seleccionada.
  5. Baja y da click al botón Change Settings para guardar los cambios.
    Se abrrirá una página de confirmación que indica que los cambios se guardaron correctamente.
    • En 15 minutos, el nuevo usuario está listo para usar.
    • Según la cantidad de datos que se muevan, puede llevar varias horas completar la migración al nuevo usuario.
    • Para verificar el estado de la migración de un solo dominio, consulta la página Manage Domains. A la derecha de tu dominio, puedes ver un icono de reloj y un texto que indica que los datos se están moviendo al nuevo usuario. Una vez completado, el icono y el texto ya no aparecerán.

Propagación de DNS después de migrar

No hay tiempo de inactividad durante el movimiento ya que los datos se copian mientras el servidor anterior continúa sirviendo al sitio web. Sin embargo, no debes trabajar en tu sitio durante este período de 4 a 6 horas. Esto se debe a que es posible que sus cambios se realicen en el sitio anterior, no en el nuevo. Debes esperar hasta que el DNS se haya actualizado completamente para continuar trabajando en tu sitio web.

Cuando migras tu dominio a un servidor diferente, la IP de su sitio cambia. Si tu dominio usa los servidores de nombres de DreamHost, la IP del dominio se actualiza automáticamente.

Si tu dominio no está utilizando los nameservers de DreamHost, deberá actualizar sus registros DNS manualmente en la compañía donde están alojados sus servidores de nombres.

Puedes ver todos los registros DNS configurados para cualquier sitio que aloje en DreamHost en su panel. Ten en cuenta que después de cualquier cambio de DNS, los registros deben propagarse en línea.
