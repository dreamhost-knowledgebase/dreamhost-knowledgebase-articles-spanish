Generalidades

Este artículo utiliza el protocolo SFTP para conectarse. Asegúrate de que tu usuario sea un usuario de SFTP antes de continuar. Visita el siguiente artículo para obtener más detalles:

  • SFTP

Después de que se inicia WinSCP, se te presenta un cuadro de diálogo para ingresar los detalles de tu conexión:

  1. Visita el artículo credenciales SFTP para recuperar tus credenciales SFTP para conectarte.
  2. Ingresa las credenciales en los campos de texto correspondientes en el cuadro de diálogo WinSCP:
    • File protocol: SFTP
    • Hostname:
    • Port number: 22
    • User name:
    • Password:
  3. Haz clic en el botón Advanced bajo el campo password.
    Una nueva caja de dialogo aparece:
  4. A la izquierda bajo la categoría Environment, selecciona la opción SFTP.
  5. Asegúrate de que la casilla Allow SCP fallback no esté marcada.
  6. Haz clic en el botón OK para regresar a la ventana de WinSCP.
  7. Haz clic en el botón Login para conectarte a través de SFTP.
    Aparece un mensaje de advertencia que indica que has recibido una nueva clave de host del servidor al que acabas de conectarte. Esto siempre sucederá la primera vez que establezcas una conexión:
  8. Deberías visitar el artículo Alerta 'The server's host key is unknown' (La llave host del servidor es desconocida) para instrucciones sobre como verificar la huella. Esto asegura que te estás conectando a tu servidor DreamHost.
  9. Una vez verificado, haz clic en el botón Yes para aceptar la clave de host y guardarla en tu caché para referencia futura.
    • Solo tendrás que hacer esto una vez (a menos que la clave de host del servidor cambie).
    • Puedes comprobar tus Llaves SSH en la página Llaves de SSH de tu panel para compararlas con esta advertencia.
  10. Cuando inicias sesión, estás en el directorio de tu usuario. Aquí verás varias carpetas. Busca la carpeta de tu sitio web y haz clic en ella. En esa carpeta, verás los archivos de tu sitio que ahora puedes editar.
