Si tienes problemas para mover tu sitio web entre cuentas, o simplemente no tienes tiempo, DreamHost actualmente ofrece un Servicio de Content Move que se encarga de eso por ti. Puedes encontrar más información en el artículo de DreamHost Servicios de Migración de Sitios Web.

Primer paso para propietarios de cuentas NUEVAS

Si el dominio que deseas mover a tu cuenta se encuentra alojado actualmente en otra cuenta de DreamHost, debes crear una cuenta nueva y usar un subdominio dreamhosters.com. Por ejemplo, si el sitio (en la cuenta existente de DreamHost) que deseas mover a tu cuenta, se llama example.com, vas a agregar el dominio example.dreamhosters.com al momento de crear la cuenta nueva.

Una vez creada la cuenta con este subdominio dreamshosters.com, puedes seguir con los siguientes pasos para mover el sitio a tu cuenta.

Restricciones

Debes tener en cuenta algunas restricciones que se aplican a las migraciones de dominios internos:

  • La cuenta a la que vas a agregar el dominio no puede tener una promoción activa, lo que significa que si la cuenta aún se encuentra dentro del primer año de alojamiento y se registró con un descuento promocional, no es posible agregar el dominio.
  • La cuenta a la que vas a agregar el dominio no puede estar activa en el programa Referiere & Gana (esto también se aplica si la cuenta está dentro del año inicial de servicio).

Ponte en contacto con soporte si tienes alguna pregunta sobre las restricciones que mencionamos anteriormente.

Paso 1 — Hacer copia de seguridad de la información de tu dominio

Si deseas mover el alojamiento de un dominio a una nueva cuenta, primero debes hacer una copia de seguridad de todos los archivos necesarios. Esto incluye archivos del servidor web, bases de datos y correo.

NO elimines el dominio de tu panel hasta que hayas hecho una copia de seguridad de todos los datos. Esto puede eliminar permanentemente tus datos.

Los siguientes artículos proporcionan información sobre cómo hacer una copia de seguridad de cada uno:

  • FTP: Este artículo explica cómo conectarte a tu servidor DreamHost usando un cliente FTP. Una vez conectado, puedes descargar todos los archivos de tu sitio web a tu computador local. Estos archivos finalmente se cargarán en la cuenta nueva.
  • Copia de seguridad de MySQL: Este artículo explica cómo hacer una copia de seguridad de tu base de datos (suponiendo que tengas una). Algunos sitios web simples no usan una base de datos. Si ese es el caso con tu sitio web, puedes omitir este paso.
  • Transferir correos: Si tienes direcciones de correo alojadas con DreamHost para el dominio que estás moviendo, debes hacer una copia de seguridad antes de eliminar el alojamiento. Este artículo explica cómo usar un cliente de correo para conectarte a tu dirección utilizando el protocolo POP3. Esto te permite descargar tus archivos de correo a tu computador.
  • Copia de seguridad: Este es un artículo que detalla cómo hacer una copia de seguridad de todos los datos de tu cuenta.

Paso 2 — Mover Listas de Anuncios/Discusión

Debido a que las Listas de Discusión y Anuncios requieren atención especial, comunícate con el equipo de soporte en la página Contactar a Soporte antes de comenzar el proceso. De esta forma, el soporte de DreamHost puede hacer arreglos para ayudarte a moverlos. Visita el siguiente artículo para obtener más información:

Paso 3 — Mover el hosting del dominio

Después de hacer una copia de seguridad de todos los datos de tu dominio, puedes continuar con la migración a la cuenta nueva.

Habrá un tiempo de inactividad inevitable a medida que migres el dominio entre cuentas.

  1. Dirígete a la página Administrar Dominios.
  2. Haz clic en el botón Delete al extremo del lado derecho del dominio del que realizaste una copia de seguridad.
  3. Después de hacer clic en el botón Delete se presentará el mensaje de advertencia anterior. Esto es para confirmar que ya has realizado una copia de seguridad de tus datos y que estás seguro de que deseas continuar con la eliminación de esta cuenta.
  4. Haz clic en el botón Remove para eliminar este sitio web. Luego verás un mensaje de "Success!" Que confirma que el dominio ha sido eliminado.
  5. Inicia sesión en la segunda cuenta de DreamHost en panel.dreamhost.com.
  6. Dirígete a la página Administrar Dominios.
  7. Mira el artículo agregar un dominio para ver los detalles sobre cómo agregar este dominio para ser alojado en la cuenta nueva.
  8. Una vez que el dominio esté completamente alojado, puedes proceder a cargar todo el contenido del que hiciste una copia de seguridad en la sección anterior.

Una vez que los datos se han cargado en la cuenta nueva y los DNS hayan propagado, el sitio va a resolver desde la cuenta nueva en DreamHost.

Paso 7 — Mover el registro de tu dominio

Algunos dominios internacionales no pueden ser transferidos de cuenta a cuenta por el cliente. En estos casos, el panel del cliente que recibe el dominio debe solicitar que se comuniquen con el soporte de DreamHost para obtener más ayuda.

Mover un registro de dominio entre cuentas es lo mismo que si lo moviera a otro registrador. Para obtener instrucciones sobre cómo mover solo el registro, visita los siguientes artículos:

  • Transferir el registro de tu dominio a DreamHost
  • Transferir el registro de tu dominio fuera de DreamHost

No es necesario mover el registro de dominio a la cuenta nueva para habilitar el alojamiento del sitio web. Sin embargo, tener tanto el alojamiento como el registro en la misma cuenta facilita el seguimiento de ambos.
