Generalidades

Los objetos son archivos que se suben en DreamObjects.

Usando el Object Viewer para ver objetos en un bucket

Para usar el Object Viewer:

  1. Navega a la página DreamObjects en tu panel de DreamHost.

  2. Haz clic en el botón View Objects bajo tu nombre de usuario.

    Luego de hacer clic en el botón, el Object Viewer carga una lista de tus buckets.

  3. Haz clic en el nombre del bucket en el panel izquierdo. Esto lista tus carpetas y objetos dentro de ese bucket, y también muestra cuatro botones de control en la parte inferior.

    El Object viewer en el panel solo puede mostrar 5000 objetos a la vez.

    Haz clic en el botón LOAD MORE OBJECTS para mostrar los siguientes 5000 objetos.

    Los botónes de control en la esquina inferior izquierda te permiten:

    • Switch Bucket — Ver los diferentes contenidos de un bucket
    • Reload Bucket — Para ver los cambios (obejtos añadidos o removidos) del bucket
    • Reload Folder — Recarga la carpeta actual que estás viendo
    • Upload Objects — Subir un nuevo Objeto al bucket

Otras formas de ver Objetos

También puedes utilizar aplicaciones de terceros para conectarte a DreamObjects y ver o subir objetos.

Creando un Object

Puedes crear Objetos usando una aplicación de tranferencia de archivos compatible.

Propiedades de los Objetos

Haz clic en cualquier objeto de tu lista para ver las propiedades del objeto, que incluyen:

  • Nombre del Objeto
  • Permisos — (PERMISSIONS) (Public o Private determinado por el uso de las credenciales)
  • Tamaño (SIZE)
  • Última modificación (LAST MODIFIED)

Descargando un Objeto usando Object Viewer

  1. Haz clic en el Objeto que deseas descargar.

  2. Haz clic en el botón Download Object en la caja Object Properties.

    Aparece un cuadro emergente que le permite descargar el objeto.

    • También puedes enviarle a cualquier persona la URL del objeto para que puedan descargarlo ellos mismos.

Eliminando un Objeto

  1. Haz clic en el objeto que deseas eliminar.

  2. Haz clic en el enlace Remove Object en la caja Object Properties.

  3. Haz clic en el botón Remove en el cuadro emergente para eliminar el objeto.
