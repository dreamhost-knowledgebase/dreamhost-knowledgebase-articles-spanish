Generalidades

Puedes ajustar el color de las áreas de fondo de texto y sin imagen dentro de una 'Web Section'. Puedes personalizar los colores usando la barra deslizante de color, mediante la entrada de código hexadecimal o haciendo clic en un color anterior sugerido, en las opciones del selector de color.

Ajustar color de texto

Haz clic en el texto para el que deseas ajustar el color. Lo verás resaltado y la barra de herramientas de edición de texto aparecerá debajo.

En la esquina superior derecha, haz clic en el icono de gota de color. Esto abre el panel de personalización de color.

Ficha de color personalizado

Cuadro superior derecho — Este es un cuadro de entrada manual para códigos de color hexadecimales. Mira un sitio como color-hex.com para obtener un código hexadecimal.

A continuación, verás 3 barras deslizantes horizontales, cada una con su propio propósito.

Control deslizante superior — Selecciona la gama de colores general.

Control deslizante medio — Elige el color específico dentro de esa gama de colores general. Esto aplicará automáticamente el cambio.

Último deslizante — Ajusta la opacidad de su texto (ejemplo: efecto de desvanecimiento).

Caja cuadrada — En la parte inferior derecha hay una vista previa del color personalizado.

Ficha Color anterior

También puedes hacer clic en la pestaña Previous Color para ver los colores sugeridos o un historial de cambios de color.

Ajustar el color de fondo de una sección web

Para ajustar el color de fondo de una sección web, haz clic en el icono color palette en la esquina superior derecha.

Verás pestañas separadas para Paleta, Historial y Personalizado.

Palette — Opciones de color sugeridas.

History — Muestra los colores anteriores utilizados en la página.

Custom — Esta pestaña tiene las mismas opciones de personalización que el editor de color de texto mencionado anteriormente.

Los colores no se pueden cambiar para los colores de fondo en los botones de texto.
