Ver tus registros DNS en el panel

Puedes revisar todos los registros DNS configurados para cualquier sitio que alojes en DreamHost en la página Administrar Dominios.

Debajo del nombre de dominio, haz clic en el enlace DNS:

Los detalles de cada columna consisten en el registro DNS configurado para tu dominio:

Registro Tipo Valor
  • Muestra el servicio para este registro.
  • Por ejemplo, si tienes un registro para el servicio ‘www’, el nombre de ‘www’ aparecerá en esta columna.
  • Muestra el tipo de registro.
  • Para el servicio de alojamiento ‘www’, verás ‘A’ aquí, ya que se configurará como un registro ‘A’ para ese servicio.
  • Muestra el valor para ese registro. Verás un valor diferente aquí dependiendo del tipo de registro que estés configurado.
  • Para el registro 'www', esta columna mostrará una dirección IP para señalar el registro 'www' para que esté alojado con DreamHost.

Esta información es muy útil si tu dominio apunta a nameservers que no son DreamHost y necesitas configurar alguno o todos estos registros manualmente para que apunten a los servicios de DreamHost.

Si estás usando los servicios de alojamiento de DreamHost (incluido el estacionamiento o una redirección del sitio) y tu dominio está registrado en DreamHost, debes usar estos nameservers:

  • ns1.dreamhost.com
  • ns2.dreamhost.com
  • ns3.dreamhost.com

Mientras los nameservers apunten a DreamHost, los registros en la página DNS de tu dominio se aplicarán a tu dominio.
