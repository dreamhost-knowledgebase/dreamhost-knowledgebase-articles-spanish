General

¿Qué es Remixer?

¡Remixer es la nueva herramienta de creación de sitios de clic para editar de DreamHost!

Puedes crear fácilmente un sitio web listo para publicar siguiendo estos simples pasos:

  1. Selecciona un tema
  2. Edita un tema
  3. Vista previa de tu página
  4. Asigna la página a tu dominio

Los artículos de configuración de Remixer describen todo el proceso, desde elegir un tema hasta publicar tus activos en un dominio DreamHost.

¿Cuanto cuesta?

Visita el artículo Remixer planes y precios para más detalles.

¿Remixer es compatible con dispositivos móviles?

Sí. Remixer está optimizado para uso móvil.

¿Cómo empiezo a usar Remixer?

  1. Abre https://remixer.panel.dreamhost.com/.

    El navegador Google Chrome es preferido..

  2. Ingresa las mismas credenciales de inicio de sesión del propietario de la cuenta que usas para el panel DreamHost.
  3. Cuando aparezca el mensaje 'Autorizar Remixer ...', haz clic en el botón Conceder acceso para abrir Remixer.

¿Necesito algo más?

No. ¡Todo lo que necesitas es una computadora/navegador y la capacidad de apuntar, hacer clic, arrastrar y soltar!

DreamHost recomienda usar una computadora de escritorio con el navegador Google Chrome para Remixer.

¿Cómo obtengo soporte para Remixer?

Puedes enviar un ticket de soporte regular Contactar Suporte o enviar un correo electrónico Contactar Soporte de Remixer a remixer@dreamhost.com durante las horas regulares de personal, de 10 a.m. a 6 p.m., hora del Pacífico, de lunes a viernes.

Soporte para Remixer no está disponible actualmente por medio de chat en vivo.

Una vez que empiezo a construir, ¿puedo guardarlo y volver más tarde para publicar?

Sí. Remixer guarda automáticamente tu trabajo. Puedes encontrar sus sitios guardados en la parte superior derecha haciendo clic en 'Sitios web'.

¿Puedo configurar una tienda de comercio electrónico con un certificado SSL?

En este momento, no hay un servicio integrado para comercio electrónico o certificados SSL en Remixer.

Sin embargo, si tienes un plan de alojamiento Remixer All-Access o Shared, puedes considerar usar la función "Exportar a Wordpress" de DreamHost Remixer, que luego incluye un certificado Let's Encrypt SSL y está listo para WordPress-ecommerce. Preguntas de codificación

¿Necesito saber HTML, JavaScript o CSS?

No. Remixer está completamente basado en plantillas, y no requiere codificación manual alguna. Puedes crear un sitio web totalmente funcional en cuestión de minutos utilizando un creador de sitios fácil de usar y de hacer clic para editar.

¿Puedo agregar JavaScript a una página?

Actualmente no es posible agregar JavaScript personalizado a una página Remixer.

¿Puedo acceder a los archivos HTML directamente? ¿Remixer tiene un editor HTML para modificar manualmente el tema o 'Sección web'?

Consulta el siguiente artículo para obtener una descripción general de 'Secciones web'.

  • 'Secciones Web' de Remixer

En este momento, Remixer no tiene un editor HTML incorporado ni ningún método para acceder a los archivos HTML reales.

Los temas de Remixer y las 'Secciones web' se han diseñado específicamente para verse y ajustarse de cierta manera, y DreamHost recomienda que uses la opción Remix en cada 'Sección web' si deseas probar un estilo de sección diferente.

¿Se puede exportar una página Remixer como HTML?

No. Las páginas de Remixer están completamente construidas y operadas dentro del servicio Remixer.

Acabo de publicar, pero mi dominio aún no está cargando el sitio

Vuelve a verificar para asegurarte de haber asignado el dominio o subdominio a tu página en el menú Páginas. Cuando editas una página Remixer, esta sección se muestra en el panel izquierdo.

Se requieren cambios de DNS si está publicando en su dominio o subdominio por primera vez.

  • Si los servidores de nombres de tu dominio apuntan a DreamHost, se actualiza automáticamente en 24 a 48 horas.
  • Si los servidores de nombres de tu dominio no apuntan a DreamHost, debes actualizar tus registros DNS en la empresa que aloja tus servidores de nombres.

Remixer requiere que tu dominio DreamHost esté configurado en “Sólo DNS” antes de publicar. Para configurar solo DNS, visita el siguiente artículo:

  • Configurar tu dominio como Sólo DNS

Su sitio debe estar visible después de unas horas de Propagación de DNS.
