Plugins

Los plugins de WordPress amplían la funcionalidad de tu sitio. Ofrecen nuevas adiciones que mejoran las características actuales y agregan nuevas características. Mira el siguiente artículo de WordPress para obtener más información sobre la instalación y administración de plugins:

  • Administrar Plugins

Verificar las versiones de tus plugins

Asegúrate de que son compatibles con la versión de WordPress

Es muy importante asegurarte de que los plugins son compatibles con la versión de WordPress que estás usando. Esto es algo que puedes verificar aquí en WordPress' Plugin Directory.

Localizar tu versión de WordPress

Puedes encontrar la versión de WordPress que estás usando, en tu panel de WordPress. Haz clic en 'Dashboard' en la parte superior izquierda.

Instalar el plugin

  1. Inicia sesión en tu sitio WordPress.
  2. En el menú de la izquierda, haz clic en 'Plugins > Add New'.
  3. Busca el nombre del plugin. En este ejemplo, instalamos 'UpdraftPlus'. Cuando lo encuentres, haz clic en el botón Install Now.
  4. Al volver a la página 'Plugins', asegurate de hacer clic en 'Activate' que está debajo del plugin.

Actualizar plugins

También debes confirmar que estés usando la versión más reciente de cada plugin. Actualizar tus plugins te puede ayudar con problemas de seguridad y también a mejorar el rendimiento de tu sitio web. En el panel de WordPress puedes ver si algún plugin tiene disponible alguna actualización:

Haz clic en 'Plugins' en el lado izquierdo de la ventana. Así puedes ver todos los plugins que tienes y cuales de ellos tiene actualizaciones disponibles.

Como mencionamos anteriormente, asegúrate de que la nueva versión del plugin es compatible con la versión actual de WordPress que estés usando. Esta información está disponible en la página web del plugin.

Confirmar si un plugin es compatible con tu versión de WordPress

El siguiente ejemplo muestra que hay una actualización disponible para 'Akismet'. Lo que debes hacer a continuación es revisar la página del plugin:

  • https://wordpress.org/plugins/akismet/

Al lado derecho puedes ver la información sobre la versión.

Actualización masiva

También puedes hacer clic en el menú desplegable en la parte superior de la list. En el menú 'Bulk Actions' selecciona 'Update', y luego haz clic en 'Apply' para actualizar todos los plugins de esa lista. Sin embargo, lo más recomendable es confirmar que las versiones más recientes sean compatibles con tu versión de WordPress antes de actualizarlos.

Deshabilitar plugins

Si estás experimentando lentitud en el sitio y deseas ver si está relacionado con el uso de recursos (y no algo relacionado con DreamHost), simplemente desactive todos los plugins y cambia el tema por uno predeterminado.

Si tu instalación de WordPress de repente se está ejecutando más rápido, te recomendamos revisar tus plugins y habilitarlos uno por uno, para ver si alguno de ellos está utilizando más recursos que los demás.

Cómo deshabilitar los plugins

La manera más rápida para deshabilitar los plugins es renombrar el directorio de plugins en el servidor.

  1. Inicia sesión en tu servidor ya sea por FTP o SSH. (En este ejemplo usamos Filezilla).
  2. Asegúrate de estar en el directorio de tu sitio WordPress.
  3. Navega al directorio '/wp-content'. Aquí puedes ver el directorio /plugins.
  4. Renombra el directorio /plugins a /plugins_OFF. Esto deshabilita inmediatamente todos los plugins.
  5. Prueba tu sitio nuevamente para ver si esto soluciona el problema que estuvieras experimentado.

Si te das cuenta de que tu sitio mejora repentinamente, sabrás que uno de los plugins estaba causando el problema.

  1. Renombra el directorio /plugins_OFF de vuelta a su nombre original /plugins.
  2. Haz clic en el directorio /plugins para ver todos los plugins.
  3. Ahora puede cambiarles el nombre de uno en uno para deshabilitarlos. A medida que los renombras uno a la vez, carga tu sitio para ver si ese plugin fue el que causó el problema.
  4. Una vez que descubras cual es el plugin que estuvo causando el inconveniente, comunícate directamente con el desarrollador del plugin. Todos los plugins y temas alojados en WordPress.org tienen un foro de soporte.

Modo de mantenimiento

Cuando actualizas tus complementos, temas o archivos principales de WordPress, tu sitio de WordPress muestra el siguiente mensaje:

No disponible por ahora debido a mantenimiento programado. Vuelve en un minuto.

WordPress elimina automáticamente ese mensaje una vez que se completan las actualizaciones. Sin embargo, en algunos casos, los conflictos o errores dejan el mensaje activo en el sitio, incluso después de que se completen las actualizaciones.

Eliminar el mensaje de "mantenimiento programado" de WordPress

Si el mensaje de 'mantenimiento programado' permanece activo en tu sitio, usa las siguientes instrucciones para eliminarlo:

  1. Consulta el siguiente artículo para obtener instrucciones sobre cómo iniciar sesión en tu servidor web. Puedes usar el portal WebFTP en el panel o una aplicación de terceros para iniciar sesión:
    • Generalidades sobre FTP y credenciales
  2. Una vez que hayas iniciado sesión, navega hasta el directorio de tu sitio web donde se encuentran tus archivos.
  3. En este directorio, busca un archivo llamado .maintenance y elimínalo.
  4. Una vez que elimines el archivo, asegúrate de borrar cualquier caché en tu sitio y luego actualiza tu navegador. El mensaje ya no debería aparecer en tu sitio.

El plugin Jetpack

El plugin Jetpack es algo especial. Puedes intentar deshabilitar módulos individuales en el plugin para mejorar el rendimiento.

Estadísticas de Plugins

Si bien tener un plugin que maneje tus estadísticas (y las muestre en tu interfaz de administrador) es bueno, en realidad pueden inflar la base de datos casi tan rápido como el spam no verificado. Esto significa que debes tener cuidado de usarlos y prestar especial atención a la base de datos si esto llega a ocurrir. Lo óptimo sería, usar algo que no dependa del entorno de tu servidor, algo como Google Analytics.

Fallo en la actualización de un plugin

Si notas que la actualización del plugin falla, ponte en, contacto con soporte para que tengas ayuda adicional.
