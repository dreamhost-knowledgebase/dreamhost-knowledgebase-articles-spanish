Generalidades

Este artículo explica el programa de registro de dominio gratuito en DreamHost y cómo utilizar el crédito para un dominio elegible.

El programa de registro gratuito

DreamHost ofrece un crédito de registro gratuito bajo las siguientes condiciones:

  • un (1) crédito de registro de dominio gratuito
  • por cuenta
  • por el primer año
  • para cuentas nuevas que se registren con cualquiera de los siguientes planes:
    • Shared Starter o Shared Unlimited (planes anuales de 1-3 años)
    • VPS (planes anuales de 1-3 años)
    • DreamPress (cualquier plan anual)

Este crédito gratuito NO está disponible para ningún otro plan de alojamiento.

Además, no se aplican los planes Shared mensuales ni los planes mensuales de DreamPress.

¿Qué dominios son elegibles para este crédito?

Si bien DreamHost ofrece muchas extensiones de dominio diferentes, el crédito gratuito solo está disponible para su uso en los siguientes dominios:

  • .art
  • .bar
  • .blog
  • .click
  • .cloud
  • .club
  • .co
  • .com
  • .design
  • .forum
  • .food
  • .help
  • .info
  • .io
  • .love
  • .me
  • .net
  • .online
  • .org
  • .property
  • .shop
  • .site
  • .store
  • .tech
  • .us
  • .xyz

Ningún otro dominio es elegible para este crédito.

Cómo verificar si hay un crédito gratuito disponible

Si ya has agregado un registro

Visita la página Administrar Registros de Dominio. Si tienes un crédito gratuito disponible, el precio de la renovación (de cualquier dominio elegible mencionado anteriormente) aparecerá como $0.00.

Si NO has agregado un registro

Visita la página Administrar Registros de Dominio y usa la barra Buscar Nombres de Dominio que se encuentra en la parte superior de la página (para cualquier dominio elegible mencionado anteriormente). A la derecha, verás un mensaje que confirma que hay un crédito gratuito disponible.

Términos y exclusiones

Periodo

Debes utilizar el crédito de registro de dominio gratuito dentro de los primeros tres meses de registrar tu nueva cuenta. Si no lo utilizas dentro de este período, caducará y ya no estará disponible para utilizarlo en el futuro.

Cuentas cerradas

Tú es responsable del costo de inscripción si:

  • Usas el crédito de registro gratuito para comprar o transferir un dominio
  • y luego cierras el plan de alojamiento antes de que se complete el período anual.

Por lo general, este monto se deduce de tu reembolso a menos que haya un saldo pendiente en la cuenta antes de cancelar. Revisa el artículo Cerrar una cuenta con un registro de dominio gratuito para obtener más detalles.

Preguntas frecuentes

¿Puedo utilizar el crédito para otros servicios?

No. Este crédito no se puede utilizar para ningún otro servicio o producto.

¿Soy elegible para obtener más créditos gratuitos si tengo dos planes de alojamiento calificados?

No. Agregar planes o servicios de alojamiento adicionales no proporcionará un crédito adicional de registro de dominio gratuito.

¿Quién es el propietario del registro gratuito? ¿DreamHost o yo?

Tú es el propietario total del registro de dominio en el que utilizas el crédito.

Si transferí el registro a otra compañía, ¿puedo renovarlo allí gratis?

No. El crédito de registro gratuito solo se aplica a los dominios administrados a través del panel de DreamHost.

¿Puedo cambiar el dominio al que se aplicó el crédito gratuito?

Sí. El soporte de DreamHost puede ayudarte a aplicar el crédito gratuito a un dominio diferente si se requiere. Un ejemplo sería si renovaste dos dominios simultáneamente pero deseas que el crédito se use en un dominio en lugar del otro.

Por favor tena en cuenta que el cambio de crédito gratuito no implicaría cambiar el monto de tu pago. Si el cambio de crédito gratuito genera un saldo positivo en tu cuenta, el saldo se aplicará a los cargos futuros a medida que surjan.

DreamHost entiende que algunos clientes son revendedores y tienen un motivo para usar el crédito para un dominio específico que figura en sus facturas. Por favor, Contacta a Soporte si necesitas más ayuda.
