Generalidades

Este artículo enumera algunos ejemplos simples de Node.js que puedes usar para probar tu instalación.

Ejemplo de línea de comando

Crea y ejecuta este script dentro de tu terminal:

  1. Crea un archivo llamado HolaMundo.js y luego agrega lo siguiente en el archivo: 
    console.log('Hola Mundo!');
  2. Ejecuta lo siguiente en tu terminal para ejecutar el script: 
    [server]$ node HolaMundo.js
Hola Mundo!

    Si está instalado correctamente, verás que la consola responde con Hello World!

Ejemplo de Sitio Web

También puedes crear un servidor HTTP básico para cargar un sitio.

  1. Crea un archivo en tu directorio web llamado app.js con el siguiente código: 
    var http = require("http");

http.createServer(function(request, response) {
  response.writeHead(200, {"Content-Type": "text/plain"});
  response.write("Hola Mundo. Esta página está ejecutando Node.js versión: ");
  response.write(process.version);
  response.end();
}).listen(8888);
    • Este servidor HTTP escuchará en el puerto 8888.
  2. Inicia el servidor ejecutando lo siguiente: 
    [server]$ node app.js
  3. Visita la página en un navegador con el puerto 8888 agregado al final. Por ejemplo: 
    http://example.com:8888
    Si se ejecuta correctamente, tu sitio mostrará lo siguiente: 
    Hola Mundo. Esta página está ejecutando Node.js versión: v12.16.3

Cargando tu aplicación en los puertos 80 y 443
