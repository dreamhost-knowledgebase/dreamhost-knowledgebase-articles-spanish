Generalidades

SQLite ya está instalado en todos los servidores DreamHost, sin embargo, la versión es diferente según la versión de Ubuntu que esté ejecutando tu servidor (Trusty o Bionic). Puedes ver esos detalles aquí:

  • SQLite

También puedes instalar tu propia versión localmente con tu nombre de usuario. Este artículo te guía a través de este proceso.

Instalación

  1. Inicia sesión en tu servidor a través de SSH.
  2. Asegúrate de estar en el directorio de inicio de tu usuario. 
    [server]$ cd ~
  3. Visita la  página de descarga de SQLite para descargar la más reciente versión.
  4. Ejecuta lo siguiente para descargarlo al directorio de tu usuario. 
    [server]$ wget https://sqlite.org/2020/sqlite-autoconf-3330000.tar.gz
  5. Descomprime el archivo. 
    [server]$ tar zxvf sqlite-autoconf-3330000.tar.gz
  6. Cambia al nuevo directorio. 
    [server]$ cd sqlite-autoconf-3330000
  7. Ejecuta lo siguiente para configurarlo. 
    [server]$ ./configure --prefix=$HOME/opt/sqlite
  8. Ejecuta make y make install para completar la instalación. 
    [server]$ make
[server]$ make install
  9. Añade lo siguiente a tu .bash_profile. 
    export PATH=$HOME/opt/sqlite/bin:$PATH
    export LD_LIBRARY_PATH=$HOME/opt/sqlite/lib
    export LD_RUN_PATH=$HOME/opt/sqlite/lib
  10. source a tu .bash_profile. 
    [server]$ source .bash_profile
  11. Finalmente, verifiva la versión para confirmar que ahora se está usando. 
    [server]$ which sqlite3
/home/username/opt/sqlite/bin/sqlite3
[server]$ sqlite3
SQLite version 3.33.0 2020-08-14 13:23:32
  12. Luego puedes salir del shell sqlite ejecutando lo siguiente. 
    sqlite> .quit
