Generalidades

Este artículo es una descripción general rápida de cómo se configuran las bases de datos MySQL en DreamHost.

Preguntas frecuentes

¿Qué es MySQL?

MySQL es uno de los sistemas de bases de datos de código abierto más utilizados, compatible con una multitud de aplicaciones de sitios web.

¿Qué versión corre DreamHost ?

Los servidores MySQL compartidos, VPS MySQL, DreamPress, y los servidores Dedicados ejecutan la versión 8 de MySQL.

Los clientes que estén familiarizados con la ejecución de su propio servidor MySQL y deseen utilizar una versión más nueva pueden actualizar a un plan Dedicado o DreamCompute, en el que pueden instalar cualquier software de MySQL que deseen.

¿Hay limitaciones?

Cuando creas una base de datos en el panel, esta se agrega a un servidor MySQL compartido de manera predeterminada. Debido a la naturaleza del alojamiento compartido, algunos privilegios no están disponibles para administrar tu base de datos. Revisa este artículo para obtener una lista de los privilegios disponibles.

¿Cuántas bases de datos puede contener mi cuenta?

No hay límite para la cantidad de bases de datos, pero en general tendrás una para cada sitio que requiera una base de datos.

¿La encripción de la base de datos es soportada?

DreamHost no proporciona cifrado de bases de datos. Por favor ten en cuenta que si planeas crear una solución personalizada para encriptar tu base de datos y necesitas acceso root/sudo al servidor de base de datos, deberá comprar un Servidor Dedicado o DreamCompute. Solo en esos planes tendrá acceso directo a la base de datos.

¿Hay dispinible un servidor MySQL privado?

Sí. Revisa el artículo Generalidades de un VPS MySQL para obtener más detalles.

Comparación de los productos de MySQL

La siguiente tabla muestra las principales diferencias entre las opciones de MySQL compartidas, privadas, y dedicadas en DreamHost.

Producto Descripción Local/En red Apropiado para las Necesidades de Alojamiento
MySQL compartido Un servidor MySQL compartido que está incluido en los planes de alojamiento Shared y VPS. En red Apropiado para la mayoría de necesidades de los sitios web.
VPS MySQL Un servidor privado que solo ejecuta software MySQL. En red Este es un servidor MySQL de alto rendimiento para clientes que han superado la capacidad de MySQL compartido.
MySQL Dedicado Un servidor web Dedicado ejecuta un servidor MySQL localmente junto con los demás servicios web. Local Un Servidor Dedicado es ideal para el software que envía grandes cantidades de datos entre aplicaciones web y MySQL, ya que no hay una red en el medio que limite el rendimiento.
