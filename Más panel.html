Generalidades

El menú Avanzado contiene enlaces para otras partes del panel DreamHost que ofrecen adiciones a tu sitio web. Estos se pueden utilizar para optimizar su experiencia general.

Bases de datos MySQL

La página MySQL Databases es donde puede agregar/administrar una base de datos que puede usarse con su sitio web.

Ve el siguiente artículo para más detalles:

  • Generalidades de MySQL

Cron Jobs

La página Cron Jobs Cron Jobs es donde puede agregar / administrar un trabajo cron. Cron es una forma de realizar tareas programadas en sistemas UNIX. Cada entrada en el archivo "crontab" de un usuario se denomina "cronjob".

Ve el siguiente artículo para más detalles:

  • Generalidades de Crontab

Htaccess/WebDAV

La página Htaccess/WebDAV es donde puedes habilitar la protección con contraseña o WebDAV.

Puedes usar esta interfaz web para crear nombres de usuario y contraseñas para que tus amigos y asociados accedan a secciones protegidas por contraseña de tus sitios web. Incluso puedes usar un archivo .htaccess para ayudar a evitar el enlace directo de los archivos de tu sitio web por parte de terceros o prohibir extensiones de archivo específicas.

La opción de panel sólo puede agregar las características mencionadas anteriormente. Si necesitas agregar cualquier otro código a un archivo .htaccess, debes crearlo manualmente. Ve el siguiente artículo para más detalles:

  • Generalidades del archivo .htaccess

Y ve lo siguiente para obtener una descripción general de WebDAV:

  • Generalidades de WebDAV

Bloquear Spiders

En la página Block Spiders, puedes crear un archivo .htaccess para bloquear motores de búsqueda específicos.

Ve el siguiente artículo para más detalles:

  • Bots spiders y crawlers

Estadísticas de un sitio

Puede habilitar las estadísticas del sitio en la página Site Statistics. La aplicación de estadísticas del sitio se ejecuta a través de un programa llamado Analog y proporciona varios tipos de informes, incluidos informes estadísticos para su dominio.

Ve el siguiente artículo para más detalles:

  • Generalidades de las Estadísticas
