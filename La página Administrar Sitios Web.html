Generalidades

Articulo relacionado: Página del panel Sitios Web

El botón Administrar aparece en la página Administrar Sitios Web:

  • Si estás en la Vista de Malla, pasa sobre tu dominio y da clic al botón Administrar.
  • Si estás en la Vista de Lista, da clic en el botón Administrar a la derecha de tu dominio.

Haz clic en el botón azul Administrar para abrir la página Administrar Sitios Web (excepto para planes DreamPress y Remixer, que en su lugar enlazan a su propia página de administración).

Los siguientes temas se describen en este artículo:

  • Pantalla superior
  • Pestaña Sitio Web
  • Pestaña Alojamiento
  • Pestaña DNS

Pantalla superior

La pantalla superior proporciona información de primera línea sobre tu sitio web, herramientas de navegación y dos pestañas (Sitios Web y Alojamiento) que proporcionan información más detallada.

Regresar a la página Sitios web (migas de pan)

Haz clic en el enlace Sitios Web en la parte superior para volver a la página de Sitios Web.

Información del plan de alojamiento

La URL de tu sitio web, así como el tipo de plan de alojamiento, se encuentra en la parte superior.

Desplázate sobre la URL de tu sitio web (un iconoaparece al lado) y haz clic para abrir el sitio web seleccionado. La URL del sitio web se vuelve azul al desplazarte sobre el.

Pestañas Sitios Web, Alojamiento y DNS

La página Administrar Sitios Web tambinén incluye las pestañas Sitios Web, Alojamiento y DNS donde puedes hacer más ajustes a tu sitio web.

Haz clic en una pestaña para ver configuraciones y características específicas, que se describen en las siguientes secciones.

Pestaña Sitios Web

Dependiendo del tipo de plan de alojamiento que tengas y las características que están habilitadas (o deshabilitadas) en él, algunas de las siguientes características pueden no aparecer en tu panel.

Los siguientes elementos aparecen en la pestaña Sitios Web de tu sitio web.

Agregar Alojamiento

Si tu sitio web no tiene un plan de alojamiento, aparece un mensaje informativo en la parte superior.

Haz clic en el botón Agregar alojamiento, el cual abre la página del Plan de Alojamiento de tu sitio.

Administrar Tu Sitio

El cuadro Administrar Tu Sitio incluye información sobre cómo acceder a los archivos de tu servidor usando tu propio cliente FTP de terceros o el administrador de archivos de DreamHost.

  • Haz clic en el enlace Mostrar info de inicio de sesión para ver la configuración de tu servidor, incluido el hostname, el nombre de usuario y la contraseña, así como un enlace a la página Usuarios & Archivos SFTP.
  • Haz clic en el botón Administrar para abrir el administrador de archivos.

WordPress

Si WordPress está instalado en tu sitio web, aparece una sección para este.

  • Haz clic en el botón Administrar para abrir la página wp-admin de tu sitio WordPress.
  • Da clic en enlace Desinstalar WordPress para desinstalar WordPress.
  • Haz clic en el botón Obtener la Llave para obtener una llave de migración que te permite usar el plugin de migración automatica de DreamHost

Otra sección aparece si WordPress aún no está instalado en tu sitio web, lo que te permite instalar WordPress rápidamente haciendo clic en el botón Instalar.

PHP

El cuadro PHP muestra la versión PHP actual en tu sitio web.

Haz clic en la flecha Modificar, que expande el cuadro donde puedes hacer ajustes a la versión de PHP para tu sitio web.

Dominio

Muestra el estado de tu registro de dominio (incluida la fecha de renovación) y los registros de DNS.

Seguridad

Muestra si un certificado SSL está activo en tu dominio y si DreamShield está habilitado.

Correo

Muestra el tipo de plan de correo y la cantidad de direcciones de correo configuradas en tu dominio.

Pestaña Alojamiento

Dependiendo del tipo de plan de alojamiento que tengas y las características que están habilitadas (o deshabilitadas) en él, algunas de las siguientes descripciones pueden no aparecer en tu panel.

Los siguientes elementos aparecen en la pestaña Sitio Web para tu sitio web.

Plan de Alojamiento

Muestra el plan actual, la fecha de facturación, el nombre del servidor, así como un enlace a la página Administrar Cuenta donde puedes cambiar tu plan.

Estado de alojamiento

Si tu sitio web está configurado para redirigir, reflejar, u estacionar, aparece el cuadro Estado de alojamiento y muestra el estado actual.

IP Única/Restauración de archivos

Puedes agregar una IP única o restaurar tus archivos.

Opciones de No Alojamiento (Redirección, Parqueo, Solo DNS, etc.)

También puedes configurar el dominio solo DNS, redirección, parqueo, y más.

Eliminar Dominio

Puedes eliminar un dominio. Visita el artículo eliminar un dominio para más información.
