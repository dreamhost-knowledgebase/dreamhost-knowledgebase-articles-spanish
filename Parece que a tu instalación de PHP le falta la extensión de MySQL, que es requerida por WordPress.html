Generalidades

Este artículo proporciona información sobre un mensaje de error común de MySQL en WordPress y varias formas de solucionarlo.

Antecedentes

El siguiente error puede aparecer en tu sitio WordPress cuando el código PHP de tu sitio no es compatible con la versión de PHP que tu sitio está usando actualmente. Más específicamente, el problema es con la extensión obsoleta de MySQL, que se eliminó a partir de PHP 7.0::

Parece que a tu instalación de PHP le falta la extensión MySQL que requiere WordPress.

Si estás usando una versión de PHP superior a la 7.0 y ves este error, se debe a que el código es antiguo y debe actualizarse. Debes actualizar los archivos principales de WordPress a la versión actual, así como también cualquier plugin que esté usando tu sitio.

Cómo corregir el error

Para solucionar este error, es necesario verificar la versión de PHP de tu sitio y asegurarte de que tu código de WordPress esté actualizado.

Revisar la versión de PHP en tu sitio

Revisa qué versión de PHP usa tu sitio con un archivo phpinfo.php. Asegúrate de que coincida con la versión de PHP que aparece en el panel de tu dominio.

Actualizar el código

Si tu sitio web se ha actualizado a una versión de PHP superior a 7.0 y ves este error, significa que debes actualizar la versión principal de WordPress y tus plugins para usar las extensiones MySQLi o PDO_MySQL más nuevas.

Actualizar WordPress

Las instalaciones de WordPress de DreamHost realizadas a través del panel siempre están configuradas para actualizarse automáticamente. También deberías revisa tus plugins y actualizar WordPress manualmente si es necesario:

  • Actualizar los plugins de WordPress
  • Actualizar WordPress manualmente
