Generalidades

Al visitar tu sitio de WordPress, es posible que veas este error:

Your PHP installation appears to be missing the MySQL extension which is required by WordPress.

Esto ocurre cuando el código PHP en tu sitio no es compatible con la versión de PHP que tu sitio está usando actualmente. Más específicamente, el problema es con la extensión obsoleta de MySQL que se eliminó a partir de PHP 7.0.

  • https://www.php.net/manual/en/intro.mysql.php

Si estás utilizando cualquier versión de PHP anterior a 7.0 y estás viendo este error, se debe a un código antiguo que debe actualizarse. Deberás actualizar los archivos principales de WordPress a la versión actual, así como cualquier complemento que esté utilizando tu sitio.

Comprueba qué versión de PHP está usando tu sitio

Primero, verifica qué versión de PHP está usando tu sitio con un archivo phpinfo.php.

  • ¿Cómo veo la configuración de mi entorno PHP?

Asegúrate de que coincida con la versión de PHP que muestra tu panel para tu dominio.

  • ¿Qué versión de PHP está usando mi sitio?

Si hay una discrepancia, hay una personalización potencial en algún lugar que hace que las versiones sean diferentes.

Actualizando tu código

Si tu sitio web se ha actualizado a una versión de PHP superior a 7.0 y estás viendo este error, significa que debes actualizar la versión principal de WordPress y sus complementos para usar las extensiones MySQLi o PDO_MySQL más nuevas.

  • https://www.php.net/manual/en/mysql.requirements.php

Actualización de WordPress

Si has instalado WordPress como Instalación de un clic, puedes actualizarlo manualmente en tu panel, o configurarlo para que se actualice automáticamente.

  • Cómo actualizar una instalación de un clic
  • Cómo configurar tu instalación de WordPress de un clic para actualizar automáticamente

Y puedes actualizar tus complementos en el tablero.

  • Actualización de complementos de WordPress

Si has instalado WordPress manualmente, puedes actualizarlo dentro del panel de WordPress.

  • Actualización manual de WordPress

Para servidores Dedicated

Si estás en un servidor Dedicated, tu versión de MySQL puede ser 5.6. Si es así, y estás viendo este error, agrega la siguiente línea al archivo phprc de su sitio.

extension = mysql.so
