Generalidades

Los servidores web Apache de DreamHost están configurados en modo basado en procesos. Esto significa que cada proceso sirve una conexión simultánea y utiliza una cantidad significativa de memoria para eso, lo que limita el recuento de usuarios simultáneos. DreamHost ofrece la opción de cambiar tu servidor web a Nginx dentro del panel para VPS y Servidores Dedicados.

Cambiar tu servidor

Cambiar a Nginx no afectará los archivos de tu sitio web. Permanecerán donde están y no serán tocados por este cambio.

  1. Navega hasta la página VPS o Paner de Dedicados.
  2. Da click en el botón Configurar a la derecha de tu servidor.
  3. Selecciona 'nginx' del menú desplegable HTTP Server:
    Debajo de la sección Web Server Configuration es un menú desplegable HTTP Server:
    • Apache
    • Nginx
    • No web server
  4. Da click en el botón Guardar Cambios en la parte inferior para guardar sus cambios.
  • Los servidores Nginx usan async E/S y pueden manejar una gran cantidad de conexiones simultáneas sin consumir mucha RAM, especialmente cuando sirven archivos estáticos.
  • La mayoría de las veces, Apache es más que suficiente, pero si tu sitio recibe mucho tráfico, es posible que desees buscar en Nginx.
