Administrar

¿Cómo puedo crear una base de datos?

Las bases de datos se pueden crear únicamente en el panel de DreamHost. En el momento, no es posible crear bases de datos a través de SSH o phpMyAdmin. Revisa el siguiente artículo para obtener más detalles:

  • Cómo administro mi base de datos?

¿Hay algún límite en la cantidad de bases de datos que puedo crear?

No hay ningún límite y puedes crear todas las bases de datos que desees en prácticamente todos los planes de alojamiento. (Excepto por el plan Starter Shared donde el límite son 5 bases de datos).

¿Dónde puedo ver los registros de MySQL?

Infortunadamente, los registros no están disponibles en los servidores compartidos de MySQL. Sin embargo, si has adquirido un MYSQL VPS, el área de soporte puede localizar los registros por ti. Sólo necesitas enviar un ticket a través del panel solicitando dicha información.

  • Generalidades sobre contactar a soporte

¿Es compatible con el cifrado de bases de datos?

El cifrado de bases de datos no es ofrecido o proporcionado por DreamHost. Ten en cuenta que si deseas crear una solución personalizada para cifrar tu base de datos y esto requiere acceso a la raíz (root/sudo) del servidor de tus bases de datos, necesitarás obtener un plan Dedicado o DreamCompute. Esos son los únicos planes en los que tendrías acceso directo al servidor de la base de datos.

Copia de Seguridad

¿Ofrecen copias de seguridad a largo plazo para mis bases de datos?

Las copias de seguridad a largo plazo no están disponibles para las bases de datos. DreamHost ofrece solamente copias de seguridad de 5 días y honestamente no son garantizadas. Es altamente recomendable que hagas un respaldo local de tus bases de datos frecuentemente.

¿Cómo puedo hacer una copia de seguridad de mis bases de datos?

A través de phpmyadmin o conectándote vía SSH puedes realizar el respaldo de tus bases de datos. Las instrucciones las puedes encontrar en el siguiente artículo:

  • ¿Cómo hago una copia de seguridad de mis bases de datos MySQL?

Mi dominio no apunta a los DNS de DreamHost o ha expirado. ¿Cómo puedo hacer una copia de seguridad de mi base de datos?

Si tienes un usuario Shell, puedes acceder a tu servidor web via SSH y ejecutar mysqldump. Una vez completes el respaldo, puedes acceder via FTP para descargar el archivo de la copia de seguridad.

¿Puedo acceder a mi base de datos con mi usuario de FTP?

No. Los usuarios de los dominios como Shell, SFTP, y FTP están aislados de los usuarios de las bases de datos, son completamente diferentes ya que tienen propósitos diferentes y están en servidores diferentes. Además, sólo puedes acceder al servidor de la base de datos si tienes un servidor Dedicado.

Permisos y Especificaciones

¿Tengo acceso a la raíz/sudo en un servidor privado de bases de datos (MySQL VPS)?

No. El servicio MySQL VPS es administrado por DreamHost por lo que el acceso al servidor no está disponible. Solo en un servidor Dedicado tendrás acceso ya que las bases de datos se alojan localmente en el mismo servidor dedicado que los dominios. 

¿Los usuarios de la base de datos tienen todos los privilegios en dicha base de datos?

No todos los privilegios están habilitados. Revisa el siguiente artículo para ver la lista actual:

  • MySQL: limitaciones en el servidor compartido

¿Puedo darle acceso limitado a otros usuarios para mi base de datos?

Sí. Puedes crear múltiples usuarios para acceder a la base de datos y configurar sus privilegios. Revisa el siguiente artículo para obtener más detalles:

  • Administrar usuarios MySQL, hosts permitidos, y privilegios

Acceder

Mi dominio apunta a los nombres de servidor (nameservers) de otro proveedor, pero apunta al registro A de DreamHost. ¿Por qué no puedo acceder a la base de datos vía phpMyAdmin?

Hay un registro A de los DNS asociado específicamente al nombre del host además del registro A del sitio. Así como con el dominio, debes agregar el registro A del nombre del host para que funcione correctamente. Revisa el artículo de DNS para más detalles.

¿Puedo acceder a mi base datos con una IP?

No. No puedes usar la IP como una alternativa para acceder directamente a la base de datos.

General

¿Puedo actualizar la versión de MySQL?

No.

¿Cómo puedo importar mi copia de seguridad de la base de datos?

Puedes hacer esto con un usuario SSH o phpMyAdmin. Para mayor información, visita el enlace a continuación:

  • ¿Cómo migrar una base de datos a DreamHost?

¿Cuáles son los servicios disponibles para bases de datos?

Todos los planes de alojamiento vienen con un servidor compartido para las bases de datos MySQL. También tienes la opción de adquirir por medio del panel un servidor privado para tus bases de datos. En el enlace debajo verás todos los detalles:

  • Generalidades de MySQL VPS

¿Por qué no funciona el nombre del host de mi base de datos?

Varios factores pueden causar esto:

  • El dominio no apunta a los DNS de DreamHost
  • El dominio no está completamente alojado en el panel
  • El registro del dominio ha expirado
  • Has habilitado Passenger, o estás usando Nginx en lugar de Apache.
