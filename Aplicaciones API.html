A continuación, se muestran algunas aplicaciones que los usuarios han escrito que utilizan la API de DreamHost.

Desktop Apps

  • https://sourceforge.net/projects/dreamhostadmin/ — Aplicación de escritorio de Windows para la gestión remota de cuentas

Línea de comando de Linux

  • https://deekoo.net/technocracy/lsdh/ — Interfaz de línea de comandos interactiva
  • https://github.com/helderco/Dreamhost-Full-Auto-Backup/ — Script que realiza una copia de seguridad de todas tus bases de datos MySQL y usuarios de Shell en tu cuenta de copia de seguridad

Administrador de Listas de Anuncios

  • https://www.domador.net/dalmatian/ — Aplicación de Windows para administrar listas de anuncios
  • https://terrywysocki.com/ckeditor_announcementlist_api.php — Editor WYSIWYG basado en web a través de API para publicaciones de listas de anuncios

Programación de librerias

  • WWW::DreamHost::API — Interfaz de Perl para la API
  • Another PHP Class — Otra clase de PHP
  • Fog — Admite la API de DNS de DreamHost. Mira los ejemplos.

DNS

  • https://github.com/gsiametis/dreampy_dns/ — Otro script de DNS dinámico de Python con soporte para IPv6
  • https://github.com/clempaul/dreamhost-dynamic-dns — Script de bash de DNS dinámico
  • https://github.com/Erutan409/dreamhost-api-powershell — Funcionalidad de cmdlet de PowerShell
