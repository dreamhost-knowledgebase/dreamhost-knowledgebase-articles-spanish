Generalidades

DreamHost ya no ofrece acceso de administrador (sudo) en instancias de VPS DreamHost.

Este artículo detalla por qué esto se hizo junto con ejemplos de cómo administrar tu VPS sin acceso de administrador (sudo).

¿Por qué se eliminaron los usuarios administradores?

DreamHost actualmente administra todos los servidores privados para proporcionar actualizaciones rápidas y confiables necesarias para la seguridad y estabilidad del servidor. Sin embargo, cuando los usuarios toman la administración del servidor en sus propias manos con el comando "sudo", limita directamente la capacidad de DreamHost de proporcionar un entorno de alojamiento administrado seguro y confiable.

¿Cómo puedo instalar software y administrar mi VPS sin un usuario administrador?

La mayoría del software no necesita compilarse en el sistema operativo directamente como root. En general, puedes instalarlo en tu directorio de inicio y ejecutarlo como tu usuario no root. Esto también puede ser una forma más segura de ejecutar estos programas.

A continuación se presentan algunos ejemplos con enlaces que muestran cómo administrar su VPS sin acceso de administrador (sudo).

Cambiar la configuración de PHP

Los cambios de PHP como los siguientes se pueden hacer usando un archivo PHP.ini:

  • Incrementar el limite PHP de subida
  • Incrementar el el limite de memoria PHP

Instalar módulos PHP/herramientas/frameworks

Los módulos PHP también se pueden instalar localmente. Ve lo siguiente para ejemplos detallados:

  • IonCube Loader
  • PEAR
  • Composer
  • Zend framework

Ejemplos de adición

  • Ruby Version Manager to install a custom version of Ruby and Rails
  • Python modules and custom version using virtualenv
  • Django

¿Qué otras opciones tengo si necesito un usuario administrador?

Servidor Dedicado

Puedes comprar un servidor Dedicado que permite el acceso de root. En este servidor, puedes configurar un usuario root e instalar software utilizando sudo.

DreamCompute

Otra opción es que el servicio de computación en la nube de DreamHost, DreamCompute, está completamente sin administrar.

Puedes instalar el software que desees en él. ¿No necesitas un servidor web? ¡No hay problema! DreamHost te proporciona una instalación básica del sistema operativo (o incluso puedes cargar la tuya propia). En esta máquina, puedes usar sudo o iniciar sesión como root si lo deseas.

DreamCompute proporciona el control de servidor remoto completo y completo que tantos usuarios de DreamHost VPS han estado ansiando durante años. Visita el siguiente artículo para más detalles:
