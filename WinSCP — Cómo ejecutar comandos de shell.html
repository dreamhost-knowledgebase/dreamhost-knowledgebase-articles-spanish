Generalidades

WinSCP ofrece una función para correr comandos de shell/SSH.

Para utilizar esta función, tu usuario debe ser un usuario de SHELL. En este artículo, el usuario se ha configurado como usuario de SFTP. Visita el siguiente artículo que explica cómo ajustar tu usuario para convertirse en un usuario de SHELL:

  • Habilitar acceso shell

WinSCP no admite comandos que requieran emulación de terminal o entrada del usuario. Si necesitas un terminal con todas las funciones, puedes encontrar que PuTTY sea más útil.

Correr un comando Shell

  1. Selecciona la pestaña Commands del menú superior.
  2. Del menú desplegable Commands, elige Open Terminal.
    La siguiente ventana se muestra después de seleccionar este:
  3. Haz clic en el botón OK para proceder.
    Entonces aparece la siguiente ventana
    • Ahora podrás ingresar comandos de shell/SSH para ejecutar en tu entorno de servidor remoto.
    • Ingresa comandos de shell/SSH en el campo de texto 'Enter command'
    • Haz clic en el botón Execute para ejecutar el comando
