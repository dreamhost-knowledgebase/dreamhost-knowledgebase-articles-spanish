DreamHost admite una amplia variedad de tecnologías para ayudar a respaldar y personalizar tu entorno de alojamiento. Si no encuentras tu tecnología en las tablas a continuación, comunícate con Soporte de DreamHost para confirmar si es soportada.

Lo siguiente se aplica a las plataformas de alojamiento administradas. por DreamHost. Si tienes un Servidor Dedicado con un usuario admin o estás usando DreamCompute, estas limitaciones pueden no aplicarse.

Temas y Plugins de WordPress

WordPress tiene una gran variedad de temas y plugins que puedes usar para personalizar tu sitio web:

  • Administrar Temas y Plugins
  • wordpress.org/themes
  • wordpress.org/plugins

Puedes usar cualquier tema o plugins que desees con tu sitio de WordPress, sin embargo, algunos temas o plugins pueden usar mucha memoria. Asegúrate de probar cualquier plugins nuevo para asegurarte de que no afecte la capacidad de respuesta de tu sitio.

Aplicaciones de construcción de sitios

Las aplicaciones de creación de sitios como WordPress, MediaWiki y Joomla se pueden usar para construir rápidamente un sitio web sin conocimiento de HTML.

Servicios de construcción de sitios

Los servicios de creación de sitios como Squarespace o Wix no son compatibles. Por lo general, estos servicios no te permiten ver el código del sitio o exportarlo a otro proveedor de alojamiento fuera de su plataforma.

Aplicaciones de creación de sitios no soportadas

Estas aplicaciones no son compatibles, sin embargo, aún puedes instalarlas manualmente si lo deseas.

Si se requiere acceso root/sudo para alguna instalación, la única opción es adquirir un Servidor Dedicado o DreamCompute ya que estos son los únicos planes de alojamiento que permiten el acceso sudo.
