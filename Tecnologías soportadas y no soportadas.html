DreamHost admite una amplia variedad de tecnologías para ayudar a respaldar y personalizar tu entorno de alojamiento. Si no encuentras tu tecnología en las tablas a continuación, comunícate con Soporte de DreamHost para confirmar si es soportada.

Lo siguiente se aplica a las plataformas de alojamiento administradas. por DreamHost. Si tienes un servidor Dedicadocon un usuario admin o estás usando DreamCompute, estas limitaciones pueden no aplicarse.

WordPress Temas y Plugins

WordPress tiene una gran variedad de temas y complementos que puedes usar para personalizar tu sitio web:

  • Administrando Temas y Plugins
  • wordpress.org/themes
  • wordpress.org/plugins

Puedes usar cualquier tema o complemento que desees con tu sitio de WordPress, sin embargo, algunos temas o complementos pueden usar mucha memoria. Asegúrate de probar cualquier complemento nuevo para asegurarte de que no afecte la capacidad de respuesta de tu sitio.

Aplicaciones de construcción de sitios

Las aplicaciones de creación de sitios como WordPress, MediaWiki y Joomla se pueden usar para construir rápidamente un sitio web sin conocimiento de HTML.

Aplicaciones de creación de sitios no soportadas

Estas aplicaciones no son compatibles, sin embargo, aún puedes instalarlas manualmente si lo deseas.

Si se requiere acceso root/sudo para alguna instalación, la única opción es adquirir un servidor Dedicado o DreamCompute ya que estos son los únicos planes de alojamiento que permiten el acceso sudo.
