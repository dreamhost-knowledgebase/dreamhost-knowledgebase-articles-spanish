Generalidades

Lo siguiente presenta una variedad de tecnologías que DreamHost admite para ayudarte a gestionar y personalizar tu entorno de alojamiento. Si tu tecnología no está listada en las tablas a continuación, por favor, contacta a soporte de DreamHost para confirmar si es compatible.

Lo siguiente se aplica a las plataformas de alojamiento administradas de DreamHost. Si tienes un Servidor Dedicado con un usuario administrador o estás utilizando DreamCompute, estas limitaciones pueden no aplicarse.

Temas y Plugins de WordPress

WordPress ofrece una gran variedad de temas y plugins que puedes usar para personalizar tu sitio web:

  • Administrar Temas y Plugins
  • es.wordpress.org/themes
  • es.wordpress.org/plugins

Puedes usar cualquier tema o plugin que desees en tu sitio de WordPress; sin embargo, algunos temas o plugins pueden usar mucha memoria. Asegúrate de probar cualquier nuevo plugin para garantizar que no afecte la capacidad de respuesta de tu sitio.

WordPress Multisitio

El soporte no puede asistir con instalaciones personalizadas de WordPress Multisitio. Antes de continuar, asegúrate de tener conocimientos básicos de PHP, MySQL, edición de archivos y cómo revisar los registros de errores. Además, hay dos formas en las que puedes configurar tu multisite:

  • Subdominios
  • Subdirectorios

La opción de subdominio solo está disponible en un VPS o en un Servidor Dedicado, ya que se requiere DNS de Wildcard para esta configuración. Si deseas configurar Multisitio en un servidor compartido, debes usar la opción de subdirectorio.

NO utilices el panel para ingresar un asterisco * como un registro CNAME. Esto no funciona como una regla comodín y podría causar problemas adicionales.

Aplicaciones de constructores de sitios web

Las aplicaciones de creación de sitios web, como WordPress, MediaWiki y Joomla, se pueden utilizar para construir rápidamente un sitio web sin conocimientos de HTML.

Aplicaciones de creación de sitios web no compatibles

Estas aplicaciones no están soportadas. Sin embargo, puedes instalarlas manualmente si lo deseas.

Si se requiere acceso root/sudo para cualquier instalación, la única opción es actualizar a un Servidor Dedicado o DreamCompute, ya que esos son los únicos planes de alojamiento que permiten el acceso sudo.
