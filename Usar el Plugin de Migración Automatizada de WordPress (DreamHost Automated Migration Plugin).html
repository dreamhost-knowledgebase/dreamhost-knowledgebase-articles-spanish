Generalidades

DreamHost se ha aliado con los creadores de BlogVault para proveer el Plugin de Migración Automatizada de DreamHost (DreamHost Automated Migration plugin), un plugin gratuito de WordPress que te permito migrar fácil y rápidamente tu sitio WordPress a DreamHost.

Este artículo da una descripción general de los pasos de configuración iniciales que debes seguir antes de usar el plugin. Cuando estés listo, puede usar cualquiera de las siguientes opciones para migrar tu sitio WordPress:

  • Migrar un sitio WordPress usando una llave de migración
  • Migrar WordPress usando las credenciales de FTP

Preguntas Frecuentes

¿Puedo usar este plugin para migrar un Multisitio de WordPress?

Sí. Después de instalar el plugin, asegúrate de 'Activarlo en Red' para confirmar de que pueda ejecutarse desde el panel de administración de red. Sin embargo, por favor ten en cuenta las siguientes consideraciones:

  • La migración debe mover toda la red a DreamHost. Este no permite la migración de subsitios individuales ni la importación de un subsitio a una red diferente.
  • Los planes de alojamiento recomendados para los Multisitios de WordPress son VPS, servidores Dedicados, o DreamCompute. No se recomiendan ni admiten instalaciones de Multisitios en planes Shared y DreamPress.
  • El Soporte de DreamHost no ofrece asistencia técnica con instalaciones de Multisitios.

¿Existen restricciones o limitaciones?

El plugin funcionará con la gran mayoría de sitios, temas, plugins, y configuraciones de WordPress. Sin embargo, por favor ten en cuenta que no puedes migrar los siguientes tipos de sitios usando este plugin:

  • Sitios de WordPress configurados en WordPress.com. Sin embargo, puedes importar el contenido de tus páginas, publicaciones e imágenes incrustadas.
  • Sitios creados con Wix, Weebly, SquareSpace, Joomla, u otros creadores de sitios
  • Sitios de WordPress desarrollados en entornos locales (por ejemplo, ServerPress)
  • Tu sitio ya debe ser un sitio en WordPress para poder migrar una copia exacta del mismo usando este o cualquier otro plugin.

¿Existen límites de tamaño en la migración?

En este momento, puedes migrar un sitio de WordPress de cualquier tamaño usando este plugin. Por favor ten en cuenta que el tiempo para completar la migración aumenta para sitios más grandes.
