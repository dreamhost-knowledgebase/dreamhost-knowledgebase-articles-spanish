Generalidades

Debes tener una cuenta activa y un plan de alojamiento en DreamHost para poder usar el Plugin de Migración Automatizada de WordPress. Si aún no eres cliente de DreamHost, consulta el siguiente artículo para obtener más información sobre cómo puedes registrarte para obtener una cuenta de DreamHost:

  • Cómo crear una cuenta en DreamHost

DreamHost se ha asociado con los creadores de BlogVault para proporcionar la Migración Automatizada de DreamHost, un plugin gratuito de WordPress que te permite migrar rápida y fácilmente tu sitio de WordPress a DreamHost.

El Plugin de Migración Automatizada de DreamHost funciona con la mayoría de los sitios de los sitios WordPress. Si crees que puedes tener una configuración de sitio o alojamiento web inusual, consulta la sección de Preguntas Frecuentes al final del artículo para obtener posibles soluciones.

Este plugin no debe usarse para cambiar el nombre de tu sitio de WordPress. En su lugar, consulta el siguiente artículo si necesitas cambiar todas las URL de tu sitio para usar un nombre de dominio diferente.

  • Cambiar el dominio de un sitio de WordPress

A continuación se describe en detalle cómo usar el Plugin de Migración Automatizada de DreamHost.

Para Empezar

Asegúrate de hacer lo siguiente antes de instalar el Plugin de Migración Automatizada de DreamHost:

  • Prueba e instala todas las actualizaciones disponibles para WordPress, incluidos todos los temas y plugins instalados. Asegúrate de que tu sitio no tenga errores o código pirateado.
  • Para evitar cualquier tiempo de inactividad en tu sitio, mantén los DNS del dominio apuntando a tu alojamiento anterior hasta que se complete la migración y hayas tenido la oportunidad de mirar el sitio migrado en DreamHost.

Cómo instalar el Plugin de Migración Automatizada de DreamHost

Paso 1 — Encontrar la información de tu cuenta de DreamHost

A continuación se explica qué información necesitas y dónde encontrarla en el panel web de DreamHost.

  1. Si aún no lo has hecho, Agrega un alojamiento a tu dominio en tu panel de DreamHost.
  2. Puedes agregar WordPress como una Instalación de Un-Clic.
    • Está bien hacer esto mientras tu dominio está alojado en tu host anterior. Agregar alojamiento en DreamHost no afecta tu alojamiento actual. Tu sitio existente permanecerá en vivo por ahora.
    • Si te preocupa afectar tu sitio existente, puedes agregar un subdominio temporal que termine con .dream.press (para los sitios de DreamPress), o .dreamhosters.com para todos los demás planes de alojamiento. Ten en cuenta que crear un sitio de prueba temporal creará trabajo adicional al final del proceso, ya que deberás actualizar tu base de datos a la URL correcta.
  3. Asegúrate de que el usuario asignado a tu dominio es un usuario Shell o SFTP. Un usuario de FTP hará que la migración falle.
  4. Ubica tu clave de migración en el panel DreamHost:
    • Planes DreamPress: Navega a la página Wordpress Administrado. Haz clic en el botón Manage a la derecha de tu dominio, haz clic en la pestaña Migration en la siguiente página y luego haz clic en el botón Generate Migration Key.
    • Todos los otros planes: Navega a la página Administrar Sitios Web y sigue estos pasos:
      1. Haz clic en una de las siguientes:
        • Desde la vista de Cuadrícula, pon tu cursor sobre la imagen de vista previa de tu dominio y haz clic en el botón Manage.
        • Desde la vista de Lista, haz clic en el botón Manage a la derecha de tu dominio.
      2. Cuando la página Manage Websites se abre, haz clic en la pestaña Website.
      3. Encuentra la sección Migrating this site to DreamHost? (en la caja de WordPress).
      4. Haz clic en el botón Get Key.

        Si la clave de migración de tu sitio no aparece en tu panel, Contacta a Soporte.

Paso 2 — Instalar y activar el plugin

Para instalar y activar el plugin de Migración Automatizada de DreamHost en tu sitio WordPress desde tu alojamiento anterior:

  1. Abre el Panel de administración de WordPress de tu sitio.
  2. Selecciona 'Plugins > Add New' en la barra de navegación lateral izquierda.
  3. Escribe 'DreamHost Automated Migration' en el campo de búsqueda en la parte superior derecha.
  4. Haz clic en el botón Install Now en la ventana cuando aparezca.
  5. Haz clic en el botón Activate.
    La página Migrate Your Sites to DreamHost se abre:
  6. Ingresa lo siguiente:
    • Email — Una dirección de correo donde te gustaría recibir notificaciones sobre el estado y finalización de la migración.
    • Migration Token — La Migration Key obtenida de tu panel DreamHost.
    • Revisa y acepta (haciendo clic en la casilla de verificación) los Términos de servicio y la Política de privacidad si estás de acuerdo y deseas continuar.
  7. Haz clic en el botón Migrate cuando hayas finalizado.
  8. Migration Key: Ingresa tu token de migración.
  9. Las cuatro opciones son opcionales y solo deben completarse si es necesario.
    • Source Website Password Protected: ¿Tu sitio web de origen está protegido con contraseña actualmente para simplemente ver la página de inicio? - Generalmente, esto no está protegido por contraseña.
    • Destination Website Password Protected: ¿Tu sitio web de origen está protegido con contraseña actualmente para simplemente ver la página de inicio? - Generalmente, esto no está protegido por contraseña.
    • Advanced Options: Por lo general, no es necesario completarlo.
    • Add folders: Esta opción te permite seleccionar archivos no específicos del core de WordPress, en caso de que tengas una configuración personalizada.
  10. Marca la URL para verificar el progreso de tu migración y cierra la pestaña o ventana del navegador mientras se está ejecutando.

La mayoría de las migraciones tardan menos de una hora en completarse, pero si tu sitio tiene más de un par de GB de datos, podría tardar más de una hora. Una vez que se complete, recibirás un correo de notificación a la dirección que ingresaste durante el paso anterior.

Verificar tu sitio migrado y publícalo en DreamHost

Si tu dominio es el mismo en DreamHost y el host anterior

Agrega '.dream.website' al final de tu dominio para ver el sitio migrado en DreamHost. Por ejemplo, si tu dominio es example.com, revisa tu sitio migrado en example.com.dream.website.

Una vez que estés satisfecho con el sitio migrado, cambia los DNS de tu dominio para que apunten a DreamHost.

Si estás usando un subdominio temporal que termina en .dreamhosters.com o .dream.press

Puedes usar el nombre de subdominio temporal para ver de inmediato el sitio migrado.

  • Dominios de DreamPress: Cuando estés listo para publicar tu sitio en DreamHost usando tu dominio real, sigue estas instrucciones para cambiar el nombre de tu dominio.
  • Todos los demás planes de alojamiento: Cuando estés listo para publicar tu sitio en DreamHost usando tu nombre de dominio real, sigue estas instrucciones para cambiar el nombre de tu dominio.

Migrar con credenciales FTP, SFTP, o SSH

Los siguientes pasos describen cómo migrar tu sitio usando credenciales FTP, SFTP o SSH.

Paso 1 — Ubicar tus credenciales de DreamHost

Busca la siguiente información sobre tu cuenta DreamHost:

  • FTP/SFTP/SSH User: Navega a la página Administrar Dominios. Anota tu usuario a la derecha de tu dominio (se muestra como 'User: name').
  • Host/Server Address: Haz clic en el nombre de usuario para abrir la página 'Editing User'. El nombre de tu servidor se muestra entre paréntesis junto a tu nombre de usuario en la parte superior (por ejemplo 'on servername'). Utiliza el siguiente formato de dirección de servidor según el tipo de plan de alojamiento que tengas:
    • Plan de Servidor Compartido — servername.dreamhost.com
    • Plan DreamPress o VPS — servername.dreamhostps.com
    • Plan de Servidor Dedicado — servername.dreamservers.com
  • FTP, SFTP, o SSH: Anota el tipo de usuario asignado a tu dominio.
  • FTP/SFTP/SSH Password: Establece una contraseña para este usuario y haz clic en Save Changes. Manten esta información lista para ingresar en el siguiente paso.

Paso 2 — Ingresar tus credenciales de DreamHost en tu compañía de alojamiento antigua

  1. Visita la página de DreamHost Automated Migration plugin en tu sitio WordPress en la compañía de alojamiento antigua.
    La página Migrate Your Sites to DreamHost se abre:
  2. Ingresa una dirección de correo donde te gustaría recibir notificaciones sobre el estado y finalización de la migración.
  3. Marca la casilla para aceptar los términos del servicio.
  4. Haz clic en el botón Migrate cuando hayas finalizado.
  5. En la derecha arriba haz clic en el enlace llamado Manually Input Host Details.
    La siguiente página aparece
  6. Introduzce lo siguiente:
    • Destination Site URL — La dirección web de tu sitio. Por lo general, esta es la misma dirección que tu sitio en tu compañía de alojamiento actual. Si estás utilizando un subdominio temporal de DreamHost (dream.press o dreamhosters.com) o si vas a cambiar el dominio de tu sitio, usa esa dirección en su lugar. Asegúrate de ingresar https:// al comienzo de la dirección web si planeas usar un certificado SSL.
    • Destination Server FTP IP — La dirección del servidor. Ve arriba para más detalles.
    • FTP Type — Selecciona FTP para usuarios de FTP. Selecciona SFTP para usuarios de SFTP/SSH.
    • FTP Username — El nombre de usuario. Ve arriba para más detalles.
    • FTP Password — La contraseña. Ve arriba para más detalles.
  7. Las cuatro opciones son opcionales y solo deben completarse si es necesario.
    • Source Website Password Protected: ¿Tu sitio web de origen está protegido con contraseña actualmente para simplemente ver la página de inicio? - Generalmente, esto no está protegido por contraseña
    • Destination Website Password Protected: ¿Tu sitio web de destino está protegido con contraseña actualmente para ver simplemente la página de inicio? - Generalmente, esto no está protegido por contraseña
    • Advanced Options: Por lo general, no es necesario completarlo.
    • Add folders:  Esta opción te permite seleccionar archivos no específicos de core de WordPress, en caso de que tengas una configuración personalizada.
  8. Marca la URL para verificar el progreso de tu migración y cierra la pestaña o ventana del navegador mientras se está ejecutando.

Preguntas Frecuentes

¿Puedo usar este plugin para migrar un sitio múltiple de WordPress?

Sí, el plugin Migrate to DreamHost permite la migración de una red de WordPress o una instalación multisitio. Después de instalar el plugin, asegúrate de 'Activar la red' ('Network Activate') para asegurarte de que se pueda ejecutar desde el panel de administración de la red. Sin embargo, ten en cuenta las siguientes consideraciones:

  • La migración debe mover toda la red a DreamHost. No permite la migración de subsitios individuales o la importación de un subsitio a una red diferente.
  • Los planes de alojamiento recomendados para WordPress Multisite son DreamHost VPS, Servidor Dedicado o DreamCompute. Las instalaciones multisitio en planes Shared y DreamPress no se recomiendan ni son compatibles.
  • El área de soporte de DreamHost no ofrece asistencia técnica con instalaciones de WordPress multisite.

¿Hay restricciones o limitaciones?

El plugin funcionará con la gran mayoría de los sitios, temas, plugins y configuraciones de WordPress. Sin embargo, ten en cuenta que no puedes migrar los siguientes tipos de sitios con este plugin:

  • Sitios de WordPress configurados en WordPress.com.
  • Sitios creados con Wix, Weebly, SquareSpace, Joomla u otros creadores de sitios.
  • Sitios de WordPress desarrollados en entornos locales (por ejemplo, ServerPress).

Los sitios de WordPress configurados en WordPress.com no se pueden migrar usando este o cualquier otro plugin como una copia exacta del sitio original a cualquier otro host. Sin embargo, puedes importar el contenido de tus páginas, publicaciones e imágenes incrustadas usando las instrucciones que se encuentran en esta página.

Tu sitio ya debe ser un sitio de WordPress para migrar una copia exacta de tu sitio usando este o cualquier otro plugin. Esto significa que el plugin Migrate to DreamHost no se puede usar para migrar otros tipos de sitios, como los creados con Wix, Weebly, SquareSpace, Joomla u otros creadores de sitios.

En este momento, puedes migrar un sitio de WordPress de cualquier tamaño usando este plugin. Ten en cuenta que el tiempo para completar la migración aumenta para sitios más grandes.
