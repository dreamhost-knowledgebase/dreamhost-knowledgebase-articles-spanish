Generalidades

El usuario asignado a tu sitio web es el nombre de usuario que utiliza para iniciar sesión en tu servidor utilizando un programa FTP o una aplicación de terminal. Este usuario se puede crear como usuario SFTP o SSH. La diferencia es que solo un usuario de Shell puede ejecutar comandos Unix en el servidor desde una terminal.

DreamHost provee acceso shell a todos sus clientes, pero debe ser habilitado específicamente para cada usuario agregado a tu cuenta. En este artículo, encontrarás información sobre cómo crear un nuevo usuario con acceso FTP/SFTP/Shell y cómo modificar un usuario existente para conceder o remover accesso FTP/SFTP/Shell.

Creando un nuevo usuario con acceso shell

  1. Visita la página Manage Users de tu panel de control.
  2. Haz clic en el botón de Add A New User en la parte superior de la página.
    La página Agregar un nuevo usuario abre:
    La descripción del los campos que verás en esta página es la siguiente:
    Type of User Account
    Tendrás tres opciones para tu 'tipo de cuenta de usuario'. Haz clic en el botón de radio de  'usuario Shell - permite iniciar sesión vía SSH'.
    Shell Type
    Tienes algunas opciones en este menu desplagable. Está bien dejarlo como /bin/bash.
    Server
    Selecciona el servidor donde quieres crear este usuario.
    Username
    Ingresa el nombre de este nuevo usuario SHELL.
    Enhanced security?
    Deja esto marcado.
    Full Name
    Puedes simplemente ingresar el nombre de usuario de tu usuario SHELL.
    Pick a password for me
    Marca esto si quiere que el sistema cree una contraseña para el usuario.
    Password/Password Again
    Ingresa tu contraseña personalizada aquí si 'Seleccionar una contraseña por mí' no está marcado.
  3. Cuando termines de ingresar tus configuraciones, haz clic en el botón azul Add User en la parte inferior.

    Ten en cuenta que tarda 20 minutos para que este cambio sea efectuado.

Editar un usuario existente para que sea un usuario SHELL

Los siguientes pasos explican cómo cambiar un usuario existente SFTP o FTP a usuario SHELL.

  1. Visita la página de Manage Users en tu panel de control.
  2. Haz clic en el botón de Edit a la derecha de el usuario correspondiente.
  3. Para el Tipo de Usuario selecciona 'Usuario Shell - permite iniciar sesión vía SSH'.
    • El acceso Shell también provee acceso SFTP para este usuario.
    • A menos que la opción 'Rechazar FTP' esté marcada, también el usuario tendrá acceso FTP.
  4. Haz clic en el botón azul Save Changes  en la parte inferior para guardar los cambios.

Crear un usuario Shell en el servidor a través de SSH

Todos los usuarios deben crearse en el panel DreamHost. No es posible agregar un usuario Shell en el servidor después de iniciar sesión a través de SSH.
