Generalidades

Este artículo explica cómo crear un filtro simple para mensajes de correo electrónico que mueva un correo electrónico entrante a un directorio específico. El siguiente ejemplo muestra cómo filtrar los correos electrónicos enviados desde una aplicación de Apple Mail.

Este ejemplo solo mueve el correo electrónico. NO hace que los filtros aprendan a identificar qué es y qué no es spam. Para enseñarle al filtro a aprender qué es el correo no deseado, debes iniciar sesión en Webmail y etiquetar manualmente el mensaje.

Ejemplos de encabezados

Este ejemplo muestra cómo filtrar el correo enviado por la aplicación Apple Mail. Cuando esta aplicación envía un correo electrónico, los encabezados mostrarán la siguiente línea:

X-Mailer: Apple Mail

Las instrucciones a continuación capturan cualquier mensaje con la misma línea de encabezado y lo mueven a una carpeta específica.

Crear el filtro personalizado

  1. Navega a la página Message Filters.
  2. Al lado derecho de una dirección de correo electrónico, haz clic en el botón Edit para abrir la página de los filtros.
  3. Si aún no has agregado filtros, haz clic en el botón Add New Filter.
    Se abre la página Add New Filter, donde puedes configurar un filtro:
  4. Dado que este ejemplo busca algo en el encabezado, elige Headers y contains de los primeros dos menús desplegables.
  5. El tercer campo es el contenido exacto que estás buscando. En este ejemplo es X-Mailer: Apple Mail.

    Para asegurarse de captar el texto, agregue un punto y un asterisco antes del código de esta manera:
    .*textoaqui

  6. Elige el botón de radio Move it to folder:.
  7. Justo debajo del botón de radio, escriba el nombre de la carpeta a la que desea que se envíe este correo electrónico.
  8. Cuando termines de crear tu regla, haz clic en el botón Add Filter en la parte inferior.

Qué esperar

Si se te envía un correo electrónico desde una aplicación Apple Mail, ahora se moverá a tu carpeta personalizada Apple Messages una vez recibido.

Puedes crear filtros de encabezado como este para que coincidan con otros correos electrónicos, según los detalles de sus encabezados. Al revisar los encabezados de los mensajes de un correo electrónico que te gustaría filtrar, puedes crear tus propios filtros personalizados basados en encabezados.
