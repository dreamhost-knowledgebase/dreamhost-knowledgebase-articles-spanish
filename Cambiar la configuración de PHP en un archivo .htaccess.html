En algunas configuraciones de servidor, es posible ajustar la configuración de PHP dentro de tu archivo .htaccess. Por ejemplo:

php_value  upload_max_filesize  600M

DreamHost no permite este tipo de configuración. Si necesitas ajustar tu configuración de PHP, debes hacerlo dentro de un archivo phprc. Mira los siguientes artículos para obtener información sobre cómo crear y configurar este archivo.

  • Crear un archivo phprc vía FTP
  • Crear un archivo phprc vía SSH
  • Incrementar el límite de carga de PHP
  • Incrementar el límite de memoria de PHP
