Generalidades

El menú 'Dedicated Servers' te permite agregar/administrar un Servidor Dedicado a tu cuenta. También ofrece la posibilidad de agregar un usuario administrador al servidor que te permite correr comandos como sudo/root.

Dashboard

La página Servidor Dedicado muestra una lista de Servidores Dedicados de esta cuenta.

Admin Users

Un usuario administrador tiene acceso completo a la raíz del Servidor Dedicado a través del comando sudo. Estos son usuarios especiales con acceso completo al servidor.

Ten cuidado cuando uses un usuario administrador, ya que tendrás la capacidad de alterar la configuración del sistema que podría deshabilitar tus sitios o servidor. Asegúrate de saber lo que haces cuando usas un usuario administrador.
