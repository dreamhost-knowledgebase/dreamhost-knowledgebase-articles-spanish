Generalidades

Lo siguiente describe cómo usar el plugin WP Rollback para revertir un plugin/tema a su versión anterior cuando se necesite.

Antes de usar este plugin para revertir a una versión de un tema o plugin, se recomienda encarecidamente que hagas una copia de seguridad de la base de datos de tu sitio. Revisa el siguiente artículo para obtener más información:

  • Generalidades sobre respaldar las bases de datos

El plugin WP Rollback plugin es útil para revertir la versión de tus plugins/temas según sea necesario. Un ejemplo puede ser si actualizaste un plugin y la nueva actualización causó problemas en tu sitio o entró en conflicto con otro aspecto del mismo, como otro plugin o la versión de WordPress. Luego puedes usar el plugin de reversión para volver a la versión anterior para solucionar problemas.

Luego de instalar y activar el plugin, puedes usarlo con unos simples pasos que están descritos a continuación.

Revertir un plugin

  1. Dirígete a la sección Plugin de tu panel administrativo de WordPress.
    El siguiente ejemplo muestra el plugin WPForms Lite:
  2. Busca el plugin que deseas revertir en la lista de plugins y da clic en el enlace Rollback.
  3. Selecciona la versión a la que deseas retroceder. Por lo general, es la versión anterior a la última versión del plugin o, en algunos casos, la versión anterior que usaste antes de la actualización si el plugin no se ha actualizado en un largo tiempo.
  4. Confirma tu decisión dando clic en el botón Rollback.
    Una página de estado aparece mostrándote el progreso:
  5. Da clic a uno de los siguientes enlaces en la parte inferior de la página de estado del plugin:
    • Activar plugin — para encender el plugin de reversión.
    • Ir a la página de plugins — para devolverte a la página del plugin.

Si continúas tenniendo problemas con el plugin que has revertido, puedes intentar también revertirlo a una versión anterior.

Revertir un tema

  1. Dirígete a la sección Apariencia > Temas de tu panel administrativo de WordPress.
  2. Pasa el cursor sobre el tema que deseas revertir y da clic en el botón Detailles del tema que aparece.
  3. Da clic al botón Rollback en la parte inferior derecha.
  4. Selecciona la versión a la que deseas retroceder. Por lo general, es la versión anterior a la última versión del tema o, en algunos casos, la versión anterior que usaste antes de la actualización si el tema no se ha actualizado en un largo tiempo.

Después de que el plugin revierta el tema, puedes obtener una vista previa de tu sitio y asegurarte de que la apariencia funcione según lo previsto.
