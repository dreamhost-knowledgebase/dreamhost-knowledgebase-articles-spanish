Generalidades

En este artículo se describe cómo migrar una Lista existente de Anuncios o Discusiones de una cuenta a otra.

Respaldar la información de tu sitio web

Asegúrate primero de hacer una copia de seguridad de tu sitio y de todos sus datos.

Cuando mueves una lista a una nueva cuenta, eventualmente debes eliminar tu sitio de la cuenta actual. Tener un respaldo garantiza que tus datos no se pierdan durante este proceso.

Confirmar que la nueva cuenta pueda agregar una lista

Para mover una Lista de Anuncios o Discusión, primero deberás confirmar que la nueva cuenta no esté usando una promoción activa o tenga un referido en la lista. Si es así, no podrá agregar su dominio a ella.

Si no estás seguro de esto, contacta a soporte.

Cómo mover una Lista de Discusión

Las siguientes secciones explican cómo mover y recrear una Lista de Discusión.

Paso 1 — Contactar a soporte

Para mover una Lista de Discusión:

  1. Contacta a soporte para solicitar la migración de tu lista a otra cuenta.
    • Especifica el nombre exacto de la lista y el número de la cuenta nueva a la que deseas migrarla.
    • Asegúrate de que la solicitud para exportar el contenido de la lista se envíe desde el correo principal de la cuenta anterior. Esto es obligatorio antes de que el servicio de asistencia pueda migrar la lista.
  2. El propietario de la "Cuenta Receptora" debe contactar a soporte técnica utilizando el correo desde la que se registró. El propietario de esta cuenta debe declarar que aprueba la transferencia del dominio y las listas de discusión a su cuenta.
  3. Una vez confirmada esta información, el soporte de DreamHost hará un respaldo de tu lista.

    No elimines el dominio de tu cuenta hasta que el soporte de DreamHost haga una copia de seguridad de tus listas. Si eliminas el dominio antes de que se haga el respaldo de las listas, los datos pueden perderse de forma permanente.

  4. Luego, soporte te notificará que puedes eliminar la lista y el dominio de tu panel.

Paso 2 — Recrear la Lista de Discusión

Para recrear la Lista de Discusión:

  1. Agrega tu dominio a la nueva cuenta.
  2. Crea la lista en la página Listas de Discusión.
  3. Una vez creada la nueva lista, pide al nuevo propietario de la cuenta que se comunique con el soporte de DreamHost desde el correo principal para solicitar que la lista respaldada se migre a la nueva lista.

El soporte de DreamHost completará la importación por tí y te notificará cuando haya finalizado.

Mover una Lista de Anuncios

Para mover una Lista de Anuncios:

  1. El propietario de la "Cuenta que Cede" debe comunicarse con el servicio de soporte para confirmar que desea que la lista de anuncios se migre a la nueva cuenta.
  2. El propietario de la "Cuenta Receptora" debe comunicarse con el servicio de asistencia técnica utilizando el correo desde el que se registró para declarar que aprueba la transferencia del dominio y las listas de anuncios a su cuenta.
  3. Espera a que el soporte responda.

    No elimines el dominio de tu cuenta hasta que el soporte de DreamHost responda. Si eliminas el dominio, los datos pueden perderse de forma permanente.

  4. El soporte de DreamHost se comunicará contigo y te informará cuando se haya movido la lista.
