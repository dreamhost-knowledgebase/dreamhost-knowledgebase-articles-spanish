Generalidades

Los pasos para crear un alias de DNS en este artículo no te permite conectar tu bucket de DreamObjects con una conexión (HTTPS) segura. Si requieres de una conexión segura a tus buckets de DreamObjects, por favor mira este artículo en vez.

  • Cómo usar Cloudflare con DreamObjects

DreamObjects te permite crear alias que apunten a tus buckets usando registros CNAME. El uso de un registro CNAME de esta manera hace que tus enlaces se vean más profesionales, ya que no muestran la parte completa 'objects-us-east-1.dream.io' de la URL.

Por ejemplo, puedes usar un registro CNAME para hacer un para a la URL del bucket de DreamObjects a un dominio que controles, como media.example.com. De esta manera, una URL de tu objeto sería media.example.com/object en vez de bucket.objects-us-east-1.dream.io/object.

Ejemplos de registros CNAME y buckets

Registro CNAME Bucket de DreamHost al cual apunta
bucket.example.com bucket.objects-us-east-1.dream.io
bucket.example.com/object bucket.objects-us-east-1.dream.io/object

La opción de configuración del panel no es necesaria para que un alias funcione correctamente. Si no estás usando los nameservers de DreamHost, todo lo que necesitas hacer es sustituir el nombre de tu bucket en el ejemplo anterior y crear manualmente un registro DNS personalizado de tipo CNAME en tus nameservers de terceros.

Agregar un registro CNAME de DreamObjects

Puedes crear, eliminar y modificar tus registros CNAME de DreamObjects dentro del panel usando la interfaz creada. La interfaz incluye:

  • Una sección para permisos
  • Un enlace para el bucket
  • Una sección para configurar aliases como subdominios o dominios en la cuenta.

  1. Navega a la página DreamObjects.

  2. A la derecha de tu bucket, da clic al enlace Change Settings.

  3. Prende la opción de los permisos a ‘PRIVATE’ o ‘PUBLIC’.
  4. Ingresa tu alias y escoge el dominio al que lo deseas asignar.
  5. Da clic al botón Save Settings para guardar.

Si estás usando un DNS tercero

Una vez el registro es creado, puedes verlo en la página Administrar Dominios. Da clic al enlace DNS debajo de tu dominio para ver el registro:

Si estás usando un DNS tercero, debes copiar el registro CNAME del panel del alojamiento actual de tus registros DNS.

Enlazar un archivo usando el registro CNAME

El siguiente ejemplo usa el CNAME para enlazar a un archivo de imagen:

https://blog.example.com/image.png

Si tu imagen está en un subdirectorio (en DreamObjects), debes especificar el camino completo. Este ejemplo muestra que la el archivo image.png está en la carpeta /savedimages/.

https://blog.example.com/savedimages/image.png
