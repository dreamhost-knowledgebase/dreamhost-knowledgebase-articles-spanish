Artículo relacionado: La página Administrar Sitios Web.

La página del panel Administrar Sitios Web proporciona una presentación gráfica y mejorada de todos tus dominios, ofreciéndote una vista optimizada en la que puedes realizar fácilmente ajustes de funciones y ajustes a cualquiera de tus dominios en DreamHost.

Los siguientes temas se describen en este artículo:

  • Abrir la página Sitios Web
  • La barra de navegación superior
  • Acciones que puedes hacer en vista de Lista o Cuadricula

Acciones que puedes hace en la Vista de Malla o Lista

Dependiendo de la configuración de tu sitio web, puedes realizar varios ajustes en la Vista de Malla o lista. A continuación se describen todas las opciones disponibles en cada vista.

Malla

Descripción de los elementos de la Malla

  • Enlace a tu sitio web — Coloca el cursor sobre la URL de tu sitio web que se muestra como exampledreamhost.com y example.com en los ejemplos anteriores. Unicono aparece junto a él. Haz clic en él para abrir el sitio web seleccionado. La URL del sitio web también cambia a azul cuando pasas el mouse sobre ella.
  • o (Certificado SSL) — Muestra un () candado bloqueado o un () candado desbloqueado. A candado cerrado (verde) indica que el sitio web tiene un certificado SSL instalado y es seguro, y un candado desbloqueado (rojo) indica que no hay ningún certificado instalado y que no es seguro. Al hacer clic en el candado desbloqueado, se abre un cuadro de diálogo con un enlace a la página Certificados Seguros.
  • Plan de Alojamiento web (mostrado como DreamPress Advanced y Shared Unlimited en los ejemplos anteriores) — el tipo de plan de alojamiento aparece debajo de la URL del sitio web. Por ejemplo: Shared Unlimited, VPS, DreamPress o Remixer (por nombrar algunos). También puede mostrar la configuración del dominio para el sitio web, como Espejo, Solo DNS o Redirección.
  • (WordPress) — Si es un sitio WordPress, el logo aparece en la esquina superior derecha de la caja de Malla. Haz clic en el botón WordPress para abrir la página wp-admin para tu sitio WordPress.
  • (Email) — El icono muestra si correo está configurado en ese sitio web o no, y si lo está, cuantas direcciones de correo han sido agregadas.
  • (DreamShield) — El logo aparece si el Eliminador de Malware DreamShield está instalado.
  • Botón Administrar — Abre la página Administra Sitios Web de tu sitio web o la página WordPress Administrado (si es un sitio DreamPress).
  • (Archivos) — Abre el administrador de archivos para el sitio web.
  • Editar Sitio Web (no se muestra en el ejemplo anterior) — Si es un sitio de Remixer, aparece el botón Editar Sitio Web que abre el panel Remixer.
  • (no se muestra en el ejemplo anterior) — Si no hay un plan de alojamiento en tu sitio, aparece el botón Agregar Alojamiento, el cual abre la página Hosting, donde puedes agregar alojamiento a un dominio. Revisa el artículo Agregar un sitio y alojamiento para más información.
  • (Más información) — Haz clci en el icono de tres puntos () para abrir un menú que provee enlaces a varias áreas del panel web de DreamHost, incluyendo registro de dominio, configuración de DNS, plan de alojamiento, administración de correo y más.

Lista

Descripción de los elementos de la columna de lista

  • Nombre de dominio — La URL de tu sitio web. Pasa el cursor sobre la URL de tu sitio web (unicono aparece junto a el) y haz clic en él para abrir el sitio web seleccionado. La URL del sitio web también se vuelve azul al pasar el cursor.
  • Registro — La fecha de vencimiento del dominio de tu sitio web.
  • Plan de Alojamiento — El tipo de plan de alojamiento de tu sitio web. Por ejemplo: Shared Unlimited, VPS, DreamPress o Remixer (por nombrar algunos). También puede mostrar la configuración del dominio para el sitio web, como Mirror, Solo DNS o Redirección.
  • PHP — La versión de PHP configurada actualmente en su sitio.
  • SSL — Muestra un candado () bloqueado o un candado () desbloqueado. Bloqueado (verde) indica que el sitio web tiene un certificado SSL instalado y es seguro, y desbloqueado (rojo) indica que no hay ningún certificado instalado y no es seguro. Al hacer clic en el candado desbloqueado, se abre un cuadro de diálogo con un enlace a la página Certificados Seguros.
  • DreamShield — Indica si el Eliminador de Malware DreamShield está instalado () o no instalado (). Si DreamShield no está disponible en un dominio (p. Ej., Es solo DNS, redireccionamiento o no tiene alojamiento en él), aparece una marca '' en esta columna.
  • Correo electrónico — Indica la cantidad de direcciones de correo configuradas en el dominio. Un '0' indica que aún no se han configurado direcciones de correo.
  • Enlaces — Hay varios tipos de iconos que aparecen en la columna Enlaces:
    • — Abre la página Dominios Alojados, donde puedes agregar un plan de alojamiento.
    • — Abre la página DNS, donde puedes hacer ajustes a los registros DNS de tu dominio.
    • — Abre el administrador de archivos de tu sitio web, donde puedes conectarte a tu servidor para ver, subir, descargar archivos y más.
    • — Abre Remixer.
    • — Abre la página wp-admin de tu sitio WordPress.
  • Columnas — Puedes ajustar qué elementos de la columna mostrar haciendo clic en el icono en el extremo derecho.
  • Botón Administrar — Abre la página Administra Sitios Web de tu sitio web o la página WordPress Administrado (si es un sitio DreamPress).
  • More information () — Haz clic en el icono de tres puntos () en el extremo derecho, que abre un menú que proporciona más enlaces a varias áreas del panel web de DreamHost, incluido el registro de dominio, la configuración de DNS, el plan de alojamiento, la administración de correo y más (las opciones disponibles están basadas en el plan de alojamiento actual).
