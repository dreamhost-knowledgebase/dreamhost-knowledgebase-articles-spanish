Conectarte desde una cuenta shell

  1. Inicia sesión en tu base de datos MySQL usando las credenciales de tu base de datos. Revisa el artículo Encontrar las credenciales de tu base de datos para obtener instrucciones. También puedes encontrarlas en el panel bajo Bases de Datos MySQL.
  2. Abre la aplicación de terminal. Puedes hacer esto en un computador Mac o Linux utilizando el terminal nativo, o simplemente podrías iniciar sesión en tu servidor web DreamHost a través de SSH. Si estás en Windows, usa PuTTY para conectarte a tu servidor DreamHost, luego ejecuta el siguiente comando.
  3. Corre el siguiente comando. Asegúrate de usar tus credenciales correctos para las siguientes variables:
    • hostname - Tu URL de hostname mysql. Aparecerá como mysql.example.com
    • user - Tu nombre de usuario de la base de datos.
    • password - La contraseña de tu nombre de usuario de la base de datos.
    • database - El nombre de la base de datos.
    [server]$ mysql -h hostname -u user -p database

A continuación, se te solicitará tu contraseña para iniciar sesión. Una vez que hayas iniciado sesión, tu símbolo del sistema cambiará al siguiente:

mysql>

No puedes usar "localhost" para conectarte a tu base de datos, ya que los servidores MySQL de DreamHost están separados de los servidores de sitios web normales.
