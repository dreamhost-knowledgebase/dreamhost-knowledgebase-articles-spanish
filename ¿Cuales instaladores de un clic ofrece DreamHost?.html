Generalidades

Construir un sitio web desde cero puede ser una tarea muy lenta y difícil. Sin embargo, hay varias aplicaciones pre-ensambladas que puedes instalar que crean inmediatamente un sitio completamente funcional. En DreamHost, se denominan instalaciones con un clic y se puede encontrar una lista completa de aplicaciones en la página de One-Click Install.

Una vez que hayas elegido e instalado tu aplicación, el instalador de un clic de DreamHost hace todo el trabajo por ti. Una vez que el software se instala automáticamente, recibirás un correo electrónico con las instrucciones adicionales que debes seguir para completar la instalación de la aplicación elegida.

Contactar a Soporte

Si no recibes el correo electrónico de instalación que proporciona las instrucciones sobre cómo finalizar la instalación, siempre puedes consultar tu "Historial de soporte" en el Panel de control en la página Support History para una copia archivada de ese correo electrónico.

Instaladores de un clic disponibles

Haz clic en el logotipo en una sección a continuación para visitar el sitio web oficial de cada aplicación para obtener más información y ejemplos.

Application Description
Concrete5 es un sistema de gestión de contenido de código abierto (CMS) que permite a los usuarios con habilidades técnicas básicas crear un sitio web.
  • http://www.concrete5.org/
Joomla! es otro sistema de gestión de contenidos (CMS) que ofrece DreamHost. Es similar a un sitio web preconstruido en el que puedes ingresar tu propia información.
  • http://www.joomla.org/
MediaWiki es un programa que te permite a ti y a los visitantes ingresar información sobre un tema. De hecho, DreamHost actualmente usa MediaWiki. Es similar a una enciclopedia en línea y contribuida por el usuario. Puedes visitar el sitio web de MediaWiki para obtener más detalles y ver ejemplos de cómo puedes usarlo.
  • http://www.mediawiki.org/wiki/MediaWiki
phpBB es un programa de foro que te permite crear un espacio en tu sitio web donde los miembros pueden hablar sobre diferentes temas e ideas sobre cualquier tema que elijas. 
  • http://www.phpbb.com/
WordPress es una herramienta de blogs del Sistema de gestión de contenido (CMS), similar a un diario. Puedes usarlo para publicar historias, imágenes y enlaces a cualquier cosa que desees. Puedes visitar el sitio web de WordPress para obtener más detalles y ver ejemplos de cómo puedes usarlo.
  • http://wordpress.org/
Zen Cart es un carrito de compras basado en el antiguo motor de comercio electrónico, pero ofrece más funciones y tiene una comunidad de soporte activa. Puedes usarlo para vender artículos a través de Internet. Puedes visitar el sitio web de Zen Cart para obtener más detalles y ver ejemplos de cómo puedes usarlo.
  • http://www.zen-cart.com/

Nota: Debes establecer arreglos para recibir y procesar pagos de manera segura.

Solución de problemas

Errores de MySQL

Si recibes un error de MySQL inmediatamente después de instalar una aplicación de un clic, a menudo es porque la base de datos aún no se ha configurado por completo. Solo espera unos minutos para que todo se configure a través de DreamHost y para que los DNS se actualicen en línea. Cuando regrese a su sitio después de una breve espera, la instalación debería estar lista para usar.

Si el error de MySQL persiste y no puedes conectarte a la base de datos, asegúrate de usar el nombre de host de MySQL, el nombre de la base de datos, el nombre de usuario de MySQL y la contraseña de MySQL adecuados cuando completes la instalación.
