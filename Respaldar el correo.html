Generalidades

Este artículo explica varias opciones para hacer una copia de seguridad de tu correo en tu computador.

La interfaz de Webmail es una interfaz de solo lectura para el correo. Esto significa que no puedes descargar el correo de Webmail. Necesitas un cliente como Microsoft Outlook, Apple Mail o Thunderbird.

Generar una copia de seguridad de tu correo utilizando una conexión POP

Si configuras tu dirección de correo dentro de tu cliente como una cuenta POP, tu correo se descargará automáticamente a tu computador. Así es como debe funcionar una configuración POP.

Mira los siguientes artículos para obtener información sobre cómo configurar tu dirección como conexión POP.

  • Generalidades sobre configuración de cliente de correo
  • Protocolos y numeros de puertos de cliente de correo

Este método requiere el uso de un programa de correo de terceros configurado como una conexión POP3 para descargar los correos de los servidores de correo DreamHost para almacenarlos localmente en el computador de tu hogar.

Ya no está en el servidor de correo

Dado que estos extraen los correos del servidor, ya no verá estos mensajes a través de una conexión IMAP o por correo web. Una vez que hayas descargado tu correo de un servidor DreamHost, las copias de seguridad de esos mensajes ya no estarán disponibles, así que asegúrate de mantener las copias de seguridad almacenadas en tu computador al menos hasta que las cargues en un nuevo servidor de correo.

Carpeta de Bandeja de Entrada

La configuración POP solo descargará el correo de la carpeta Bandeja de entrada del usuario. Todas las demás carpetas y contenidos dentro de ellas deben moverse a la bandeja de entrada cuando estés listo para descargarlas. Esto incluirá carpetas como ‘mensajes antiguos’, ‘Enviados’, ‘Borradores’, y cualquier otra carpeta fuera de la carpeta Bandeja de entrada.

Flujo de Trabajo

Aquí hay un flujo de trabajo general que puedes seguir:

  1. Configura tu correo para usar una conexión POP.
  2. El correo del que solamente la Bandeja de entrada se descarga.
  3. En tu computador, ubica el directorio donde el cliente de correo descargó los correos.
  4. Crea una carpeta en tu computador llamada Bandeja de entrada y arrastra esos correos a la carpeta.
  5. Para las subcarpetas, deberás mover tu contenido a la Bandeja de entrada. Por ejemplo, si tienes una carpeta llamada 'Enviado', mueve el correo de esta carpeta a la Bandeja de entrada para que se descarguen en tu computador. Luego puedes crear otra carpeta en tu computador llamada 'Enviados' y arrastrarlos a ella.
  6. Haz cada subcarpeta una a la vez para evitar confusiones.

El punto a recordar es que la configuración POP solo descarga el correo del directorio de la Bandeja de entrada, no ningún subdirectorio.

Generar una copia de seguridad de tus correos en otro alojador

El proceso de copia de seguridad del correo con otro host puede variar de un host a otro. El servidor de correo que tienes ahora puede tener otra solución para hacer una copia de seguridad de los correos de tus servidores. Deberás ponerte en contacto con tu servidor de correo actual para obtener detalles sobre cómo hacer una copia de seguridad de los correos de tus servidores para poder importarlos a DreamHost.
