Generalidades

Este artículo ofrece varias opciones a considerar al respaldar correos electrónicos en tu computador.

Requisitos previos

Necesitarás configurar un cliente de correo de tu elección para descargar los correos a tu computador.

La interfaz de Webmail es una interfaz de solo lectura para el correo, por lo que no es posible descargar utilizando este servicio.

Respaldar el correo

Las siguientes secciones describen tres opciones para hacer un respaldo de tu correo.

Opción uno — Respaldar de tu correo utilizando una conexión POP

Si configuras tu dirección de correo dentro de tu cliente como una cuenta POP, tu correo se descargará automáticamente a tu computador. Así es como debe funcionar una configuración POP. Revisa este artículo para obtener información sobre cómo configurar tu dirección como conexión POP.

Correos eliminados del servidor

Dado que la conexión POP descarga los correos del servidor, ya no verás los mensajes en webmail (ya que utiliza IMAP).

Una vez que hayas descargado tu correo, asegúrate de conservar los respaldos hasta que las subas a un nuevo servidor de correo.

Carpeta de Bandeja de Entrada

La configuración POP solo descargará el correo de la carpeta Bandeja de entrada del usuario. Todas las demás carpetas y contenidos dentro de ellas deben moverse a la Bandeja de entrada cuando estés listo para descargarlas. Esto incluirá carpetas como old-messages, Enviados, Borradores, y cualquier otra carpeta fuera de la carpeta Bandeja de entrada.

Ejemplo de respaldo de correo

Aquí hay un flujo de trabajo general que puedes seguir:

  1. Configura tu correo para usar una conexión POP.
  2. El correo del que solamente la Bandeja de entrada se descarga.
  3. En tu computador, ubica el directorio donde el cliente de correo descargó los correos.
  4. Crea una carpeta en tu computador llamada Bandeja de entrada y arrastra esos correos a la carpeta.
  5. Para las subcarpetas, deberás mover tu contenido a la Bandeja de entrada. Por ejemplo, si tienes una carpeta llamada Enviados, mueve el correo de esta carpeta a la Bandeja de entrada para que se descarguen en tu computador. Luego puedes crear otra carpeta en tu computador llamada Enviados y arrastrarlos a ella.
  6. Crea cada subcarpeta una a la vez para evitar confusiones.

El punto a recordar es que la configuración POP solo descarga el correo del directorio de la Bandeja de entrada, no ningún subdirectorio.

Opción dos — Exportar el correo dentro de Mac Mail

Mac Mail tiene una opción para descargar rápidamente todo el correo en una sola dirección a tu computador. Revisa este artículo para obtener instrucciones.

Opción tres — Respaldar tus correos en otro alojador

El proceso de copia de seguridad del correo con otro host puede variar de un host a otro. El servidor de correo que tienes ahora puede tener otra solución para hacer una copia de seguridad de los correos de tus servidores. Deberás ponerte en contacto con tu servidor de correo actual para obtener detalles sobre cómo hacer una copia de seguridad de los correos de tus servidores para poder importarlos a DreamHost.
