Generalidades

A continuación se explica cómo determinar dónde está alojado tu sitio web.

Preguntas frecuentes

¿Dónde está alojado mi sitio web?

Tu sitio web está alojado en la compañía donde el DNS de tu dominio está apuntando. Esto es una combinación de dos cosas.

  • El Registrador: La compañía donde tu dominio está registrado
  • La Compañía de Alojamiento: La empresa que muestra tu sitio web en Internet

Revisa este artículo para saber más sobre estos términos.

¿Cómo puedo ver dónde está registrado mi dominio?

Puedes confirmar la compañía donde está registrado tu dominio ejecutando una consulta WHOIS. Esto se puede hacer usando sitios en línea o a través de SSH.

¿Cómo puedo ver dónde está alojado el DNS de mi dominio?

Revisa este artículo para ver más sobre cómo ver dónde están alojados tus DNS actualmente.

Varios métodos para apuntar tu DNS

Hay dos opciones de DNS que controlan dónde está alojado tu sitio (nameservers o registros A). Las siguientes secciones describen diferentes formas en que puede apuntar estas opciones según las necesidades de tu sitio web.

Cambiar los nameservers

Todos los registros DNS de tu sitio están controlados en la compañía donde apuntan los nameservers. Esto significa que cuando cambias tus nameservers, TODOS los registros DNS de tu sitio (sitio web, correo, MySQL, etc.) apuntan a una nueva empresa.

Para cambiar los nameservers, debes iniciar sesión en la compañía donde compraste el registro de tu dominio (el Registrador). La siguiente tabla describe dónde se aloja un sitio según varios ajustes de nameservers:

Cambiar los registros A

Los registros A son las direcciones IP para varios servicios del sitio web. Dos de estos registros A corresponden a las URLs no www y www de tu sitio.

  • example.com
  • www.example.com

Si solo apuntas los registros A de tu sitio web a una compañía diferente, solo tu sitio web se alojará en la nueva compañía. Todos tus otros registros DNS (correo, mysql, ftp, ssh) continuarán alojados donde apuntan tus nameservers.
