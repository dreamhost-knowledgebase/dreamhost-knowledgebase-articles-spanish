Generalidades

La página Administrar Cuenta enumera todos los servicios principales para tu cuenta DreamHost:

La sección Generalidades muestra lo siguiente:

  • La fecha en que se abrió tu cuenta
  • El número de ID de tu cuenta
  • El nombre del titular de la cuenta
  • Las fechas de su ciclo de facturación
  • Saldos pendientes y Autopago

También puedes cerrar tu cuenta desde esta página.

Planes de Servicio

La sección Planes de Servicio enumera los diferentes servicios que ha agregado a tu cuenta DreamHost. Estos podrían incluir lo siguiente:

  • Cualquier plan de alojamiento agregado.
  • DreamCompute.
  • Almacenamiento de DreamObjects.
  • DreamShield.
  • DreamPress.
  • Remixer.
  • Direcciones IP únicas.

Puedes editar el servicio haciendo clic en el enlace Editar a la derecha del mismo, debajo de la columna Inicio.

Servidores

Esta sección Servidores enumera todos los Servidores VPS, MySQL privados o Dedicados que hayas agregado a tu cuenta.

Usuarios

Esta sección Usuarios enumera todos los usuarios FTP, SFTP o Shell que hayas agregado a tu cuenta. Estos son nombres de usuario que puede usar para iniciar sesión en tu servidor utilizando un cliente de FTP o a o una terminal de Shell.

Buzones

Esta sección de Buzones enumera las direcciones de correo que tiene agregado a tu sitio web.

Dominios Alojados

Esta sección Dominios Alojados enumera todos dominios agregados a tu cuenta junto con el tipo de alojamiento que se le asigna.

Dominios Registrados

Esta sección Dominios Registrados enumera cualquier registros de dominio que ha comprado.

Bases de Datos

Esta sección Bases de Datos enumera todas las bases de datos has agregado a tu cuenta.

Direcciones IP Únicas

Esta sección Direcciones IP Únicas enumera cualquier Dirección IP únicas has agregado a tus sitios web.
