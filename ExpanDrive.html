Generalidades

ExpanDrive Es una aplicación que te permite conectarte a DreamObjects tal como una USB. Está disponible para ambos Mac y Windows y te permite acceder a DreamObjects desde cualquier aplicación en tu computador. Antes de comenzar, necesitarás crear un bucket en la página DreamObjects.

Versión Mac

  1. Descarga la última versión para Mac.
  2. Lanza ExpanDrive y selecciona DreamObjects como tipo de disco.
  3. Ingresa lo siguiente:
    • Server: objects-us-east-1.dream.io
    • Access Key ID: Tu llave de acceso de DreamObjects
    • Secret Access Key: Tu clave secreta de DreamObjects
    • Visita el artículo Llaves de DreamObjects para detalles sobre como encontrar tus llaves.
  4. Haz clic en el botón Connect.
  5. Es posible que veas una ventana la primera vez que te conectes pidiéndote que ingreses la contraseña de usuario de Mac. Ingresa y haz clic OK. Esto solo debe hacerse la primera vez que te conectas.
  6. La ventana del Finder en tu computador se abre con tu bucket de DreamObjects. DreamObjects ahora se mostrará como una unidad montada. Puedes verlo visible en la barra de menú superior. También lo verás en los cuadros de diálogo para abrir y guardar en tus aplicaciones, lo que te permitirá acceder fácilmente a los datos de DreamObjects.

Versión Windows

  1. Descarga la última versión para Windows.
  2. Lanza ExpanDrive y selecciona DreamObjects como tipo de disco.
  3. Ingresa lo siguiente:
    • Server: objects-us-east-1.dream.io
    • Access Key ID: Tu llave de acceso DreamObjects
    • Secret Access Key: Tu llave secreta DreamObjects
    • Visita el artículo Llaves de DreamObjects para detalles sobre como encontrar tus llaves.
  4. Haz clic en el botón Connect.
  5. ExpanDrive intentará conectarse a nuestro bucket de ‘DreamObjects’.
  6. Se abre una ventana del explorador en tu computador con tus buckets de DreamObjects. Tus datos de DreamObjects ahora son accesibles como una unidad montada que te permite acceder fácilmente a tus datos de DreamObjects.
