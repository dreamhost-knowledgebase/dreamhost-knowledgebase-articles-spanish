Alojamiento

¿Cuántos dominios puedo alojar?

Los planes Shared, VPS y Dedicated no tienen un límite en el número de dominios que puedes hospedar. Los únicos servicios que están limitados a hospedar un solo dominio son DreamPress y Starter Shared. Esto es debido a que DreamPress es un plan dedicado y especializado solo para un sitio WordPress, y Starter Shared es un plan básico para comenzar.

¿Cuánto cuesta el alojamiento?

Visita la siguiente página para ver los precios actualizados:

  • dreamhost.com/hosting/

¿Puedo alojar solo un subdominio?

Sí. Solo debes agregar el dominio principal en el panel, luego podrás agregar el subdominio. Después podrás ver la IP para dicho subdominio. Revisa el artículo Aregar un dominio para más detalles.

¿Puedo alojar dominios de primer nivel OpenNIC (ejemplo: Namecoin, Emercoin, New Nations, etc...)?

El sistema de alojamiento administrado de DreamHost no está configurado aún para soportar dominios OpenNIC. Pero puedes configurar si deseas una instancia DreamCompute para dicho propósito. Infortunadamente, debido a que esto requiere un alto nivel de personalización, no es actualmente ofrecido. Puedes ver el siguiente enlace para más información:

  • Sitio Oficial OpenNIC

¿Puedo restaurar mi sitio si elimino accidentalmente el alojamiento del dominio?

Es posible. Si tu sitio estaba alojado en un servidor Shared, VPS o Dedicated, los archivos del sitio no se eliminan automáticamente cuando se elimina el dominio. Si configuras el dominio como Fully Hosted nuevamente, puedes seleccionar el usuario y directorio FTP que se utilizó anteriormente para restaurar el sitio.

Los servidores DreamPress se eliminan automáticamente cuando el dominio se elimina de DreamPress, y tus datos no se pueden restaurar una vez que se elimina el servidor.

Acceder

¿Cómo puedo subir y acceder a los archivos de mi dominio?

Usando una herramienta como FileZilla, CyberDuck, etc… puedes acceder al servidor vía FTP o SFTP. También puedes usar una terminal para acceder via SSH. Puedes encontrar más información en el artículo de Usuarios.

¿Cómo veo y accedo a mi dominio si los DNS no apuntan a DreamHost aún?

Tienes algunas opciones. Por favor revisa la guía sobre previsualizar tu sitio antes de cambiar los DNS usando un subdominio dreamhosters.com.

¿Puedo acceder al dominio usando una IP?

Esto lo puedes lograr solo si compras una IP Única. Así tu sitio tendrá una IP específica asignada. De otra manera, esto no es posible.

General

He alojado completamente mi dominio. ¿Dónde está el archivo public_html?

Cuando agregas alojamiento para alojar completamente tu dominio, debes ingresar un directorio web. Dicho directorio es dónde pondrás el contenido de tu sitio en el servidor. Puedes poner tu archivo de índice en ese directorio web para que se muestre cuando visites la página. Este directorio web es el mismo archivo que conoces como el folder "public_html". Revisa el artículo Agregar Dominios y Subdominios para más detalles.

Agregué un dominio en el panel completamente alojado y los nombres de servidores ya están apuntando a DreamHost. ¿En cuánto tiempo puedo acceder a mi sitio?

Si recientemente agregaste alojamiento al dominio, puede tomar alrededor de 4 a 8 horas para resolver en línea. Puedes usar herramientas como la página whatsmydns.net para hacer un seguimiento de la estado de la propagación.

¿Cumple DreamHost con HIPAA?

DreamHost no cumple con HIPAA.

La Ley de Responsabilidad y Portabilidad del Seguro de Salud (HIPAA) fue creada para establecer estándares nacionales sobre cómo se protegen los registros médicos. Más específicamente, la 'Regla de seguridad' especifica estándares para proteger la información de salud transferida electrónicamente. La Regla de Seguridad requiere salvaguardas administrativas, físicas y técnicas apropiadas para garantizar la confidencialidad, integridad y seguridad de la información médica protegida electrónica. Puedes encontrar más información en el sitio web del Departamento de Salud y Servicios Humanos de EE.UU:

  • hhs.gov/hipaa/for-professionals/security
