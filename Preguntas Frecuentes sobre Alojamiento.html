Alojamiento

¿Cuántos dominios puedo alojar?

Los planes de Alojamiento Shared, VPS y Dedicados no tienen un límite en el número de dominios que puedes alojar. Los únicos servicios que están limitados a hospedar un solo dominio son DreamPress y Shared Starter. Esto es debido a que DreamPress es un plan dedicado y especializado solo para un sitio WordPress, y Shared Starter es un plan básico para comenzar.

¿Cuánto cuesta el alojamiento?

Visita la siguiente página para ver los precios actualizados:

  • dreamhost.com/es/hosting/

¿Puedo alojar solo un subdominio?

Sí. Solo debes agregar el dominio principal en el panel, luego podrás agregar el subdominio. Después podrás ver la IP para dicho subdominio. Revisa el artículo Agregar un dominio para más detalles.

¿Puedo alojar dominios de primer nivel OpenNIC (ejemplo: Namecoin, Emercoin, New Nations, etc...)?

El sistema de alojamiento administrado de DreamHost no está configurado aún para soportar dominios OpenNIC. Pero puedes configurar si deseas una instancia DreamCompute para dicho propósito. Infortunadamente, debido a que esto requiere un alto nivel de personalización, no es actualmente ofrecido. Puedes ver el siguiente enlace para más información:

  • Sitio Oficial OpenNIC

¿Puedo restaurar mi sitio si elimino accidentalmente el alojamiento del dominio?

Tal vez. Si tu sitio estaba alojado en un servidor compartido, VPS o Dedicado, los archivos del sitio no se eliminan automáticamente cuando se elimina el dominio. Si configuras el dominio como completamente alojado nuevamente, puedes seleccionar el usuario y directorio FTP que se utilizó anteriormente para restaurar el sitio.

Los servidores DreamPress se eliminan automáticamente cuando el dominio se elimina de DreamPress, y tus datos no se pueden restaurar una vez que se elimina el servidor.

Acceder

¿Cómo puedo subir y acceder a los archivos de mi dominio?

Usando una herramienta como FileZilla, CyberDuck, etc… puedes acceder al servidor vía FTP o SFTP. También puedes usar una terminal para acceder vía SSH. Puedes encontrar más información en el artículo Usuarios.

¿Cómo ver y acceder a mi dominio si los DNS no apuntan a DreamHost aún?

Tienes algunas opciones. Por favor revisa la guía sobre previsualizar tu sitio antes de cambiar los DNS usando un subdominio dreamhosters.com.

¿Puedo acceder al dominio usando una IP?

Esto lo puedes lograr solo si compras una IP Única. Así tu sitio tendrá una IP específica asignada. De otra manera, esto no es posible.

General

He alojado completamente mi dominio. ¿Dónde está el archivo public_html?

Cuando agregas alojamiento a tu dominio, se te solicita que ingreses a un directorio web. Este directorio es donde coloca los archivos de tu sitio web en tu servidor web. Carga tu archivo de index en el directorio web que lleva el nombre de tu sitio (/example.com)por lo que se muestra cuando visitas tu página.

Otos alojamientos referencian esto como el directorio public_html. Este funciona de la misma manera que DreamHost. Visita el artículo Agregar un Sitio Web y Hosting para más detalles.

Agrega un dominio en el panel completamente alojado y los nameservers ya están apuntando a DreamHost. ¿En cuánto tiempo puedo acceder a mi sitio?

Si recientemente agregaste alojamiento al dominio, puede tomar alrededor de 4 a 8 horas para resolver en línea. Puedes usar cualquiera de los siguientes enlaces para verificar el estado de propagación de DNS:

  • whatsmydns.net — Un sitio popular que verifica los registros DNS en todo el mundo.
  • Ver tus registros DNS de DreamHost — Usa el verificador de propagación de DNS de DreamHost en el panel.

¿Cumple DreamHost con HIPAA?

DreamHost no cumple con HIPAA y, por lo tanto, no puede emitir un BAA (Business Associate Agreement).

La Ley de Responsabilidad y Portabilidad del Seguro de Salud (HIPAA) fue creada para establecer estándares nacionales sobre cómo se protegen los registros médicos. Más específicamente, la 'Regla de seguridad' especifica estándares para proteger la información de salud transferida electrónicamente. La Regla de Seguridad requiere salvaguardas administrativas, físicas y técnicas apropiadas para garantizar la confidencialidad, integridad y seguridad de la información médica protegida electrónica. Puedes encontrar más información en el sitio web del Departamento de Salud y Servicios Humanos de EE.UU:

  • hhs.gov/hipaa/for-professionals/security
