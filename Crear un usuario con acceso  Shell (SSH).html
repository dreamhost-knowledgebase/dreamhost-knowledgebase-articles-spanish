Generalidades

El usuario asignado a tu sitio web es el usuario que usas para iniciar sesión en tu servidor mediante un programa FTP o una aplicación de terminal. Este usuario se puede crear como usuario SFTP o SSH. La diferencia es que solo un usuario de Shell puede ejecutar comandos Unix en el servidor desde una terminal. DreamHost proporciona acceso de shell a todos sus clientes, pero debe habilitarse específicamente para cada usuario agregado a tu cuenta.

Este artículo explica cómo crear un nuevo usuario con acceso SFTP o Shell y cómo modificar un usuario existente para poder otorgar o eliminar el acceso SFTP o Shell.

Crear un nuevo usuario con acceso shell

Para agregar un nuevo usuario de sitio web con acceso shell:

  1. Navega a la página Archivos y Usuarios SFTP desde el panel.
  2. En la parte superior derecha, da clic en el botón Crear Usuario.
    Aparece una ventana emergente:
  3. Ingresa lo siguiente:
    • Nombre de Usuario — Un nombre de usuario para el nuevo usuario.
    • Servidor — Da clic en la flecha hacia abajo y selecciona el servidor al cual el usuario tendrá acceso.
    • Contraseña (Generar contraseña segura) — Ingresa una contraseña o da clic en el botón para que el panel cree una contraseña fuerte por tí.
    • Acceso Seguro Shell (SSH) — Usa el botón de activación para habilitar el Shell/SSH Seguro (actívalo hacia la derecha). Cuando se activa, el botón se torna azul.
    • Tipo de Shell — Cuando el acceso de shell Seguro se habilita, el menú desplegable Tipo de Shell aparece. Puedes seleccionar bash, tcsh, ksh, o zsh. Revisa los usuarios Shell para más información sobre los varios tipos de Shell que puedes usar.
  4. Cuando termines, haz clic en el botón Crear Usuario.

El panel luego muestra un estado de actualización a medida que el usuario se va creando.

 

Editar un usuario existente para convertirlo en un usuario SHELL

Si ya has creado varios usuarios para el sitio web, puedes usar las siguientes instrucciones de este artículo para filtrarlos por su tipo (SFTP o Shell). Esto te ayuda a identificar rápidamente al usuario de SFTP que deseas convertir en un usuario de Shell:

  • Filtrar usuarios de los sitios web

Para editar un usuario existente:

  1. Navega a la página Archivos y Usuarios SFTP desde el panel.
  2. Selecciona un usuario y haz clic en el botón Datos de Ingreso.
    Aparecerá lo siguiente si tu usuario es SFTP:
  3. A la derecha de Acceso Seguro Shell (SSH), habilita el botón de activación.
    El panel comienza a actualizar los permisos para este usuario. Cuando termine, una marca de verificación verde aparece.
  4. Da clic en el botón Datos de Ingreso nuevamente a la derecha del usuario.
    Los siguientes cambios se muestran:
    • Acceso a Archivos — Cambia a SFTP/SSH.
    • Acceso Seguro Shell (SSH) — El botón de activación se torna azul cuando es habilitado.
    • Tipo de Shell— Selecciona un tipo de Shell de la lista del menú desplegable que deseas usar.

¿Puedo crear un usuario Shell en mi servidor a través de SSH?

No. Todos los usuarios deben crearse en el panel de DreamHost. No es posible agregar un usuario Shell en el servidor después de iniciar sesión a través de SSH.
