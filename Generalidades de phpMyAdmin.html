Generalidades

phpMyAdmin es un paquete de software de sitio web gratuito y de código abierto escrito en PHP destinado a manejar la administración de una base de datos en un navegador web.

Las operaciones de uso frecuente (gestión de bases de datos, tablas, columnas, relaciones, índices, usuarios, permisos, etc.) se pueden realizar a través de la interfaz de usuario además de tener la capacidad de ejecutar directamente cualquier instrucción SQL.

DreamHost proporciona phpMyAdmin para clientes en opciones de alojamiento de Servidor compartido, VPS, DreamPress y Dedicado. Este artículo te ayuda a acceder a tus bases de datos usando el software phpMyAdmin administrado por DreamHost.

phpmyadmin.dreamhost.com ya no funciona para abrir phpMyAdmin. De ahora en adelante, debes usar el nombre de host correcto para tu base de datos que se muestra en el panel, o usar el nombre de host phpMyAdmin oficial de tu centro de datos. Primero revisa la página Centro de Datos. Desplázate hacia abajo para ubicar el centro de datos en el que se encuentran tus servicios MySQL. Las opciones serán Costa Oeste o Costa Este. A continuación, puedes usar los siguientes nombres de host para conectarte según la ubicación de tu servidor MySQL.


Costa Oeste:  https://west1-phpmyadmin.dreamhost.com
Costa Este:  https://east1-phpmyadmin.dreamhost.com

Requisitos previos

Antes de usar phpMyAdmin en DreamHost, debes completar los siguientes pasos:

  1. Configura un dominio en alojamiento completo o reflejado en la página Administrar Sitios Web. Para más información sobre esto, por favor visita el artículo Generalidades del reflejo de un dominio.
  2. No habilites Passenger n un dominio alojado, al que vas a querer acceder por phpMyAdmin, ya que intercepta las solicitudes HTTP y mostrará un error.
  3. Agrega un nombre de host MySQL en la página Bases de Datos MySQL.
  4. Dale suficiente tiempo para que los registros DNS progaguen después de agregar el nombre de host y el activar el alojamiento completo.

Una vez que completes estos pasos, phpMyAdmin estará disponible en tu dominio.

Si deseas borrar/eliminar una base de datos

No puedes borrar una base de datos en DreamHost usando phpMyAdmin; solo puedes eliminar una base de datos en la página Bases de Datos MySQL.

Diferencias en el tamaño de las bases de datos reportada

Con el tiempo, el tamaño de la base de datos reportado por phpMyAdmin comenzará a diferir del tamaño real de la base de datos en el disco de tu servidor MySQL. Esto ocurre porque phpMyAdmin muestra el tamaño de la base de datos usando los índices de la base de datos, que pueden ser más pequeños que el uso real del disco después de muchos ciclos de lectura y escritura.

Una forma posible de resolver este problema es optimizar tu base de datos, aunque esto no está garantizado. Hay dos métodos relativamente simples que puedes usar para optimizarla:

  • Usando phpMyAdmin
  • Usando wp-cli de Wordpress
