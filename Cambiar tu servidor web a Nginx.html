Generalidades

DreamHost ofrece la opción de cambiar tu servidor web de Apache a Nginx dentro del panel de Servidores VPS y Servidores Dedicados.

Antes de cambiar a Nginx, revisa el siguiente artículo para ver una comparación detallada entre Apache y Nginx y asegurarse de que Nginx es la mejor opción para tu sitio web.

  • Comparación de rendimiento del servidor web

Archivos .htaccess

Nginx no soporta archivos de configuración a nivel de directorio, como el archivo .htaccess de Apache. Esto significa que si has creado previamente un sitio web que utiliza un archivo .htaccess, todo el código dentro de él dejará de funcionar. Esto podría ser particularmente problemático para ciertas configuraciones de WordPress. Revisa el siguiente artículo para obtener más información:

  • WordPress and Nginx

Asegúrate de que tu sitio web no dependa del archivo .htaccess antes de continuar.

Si tu sitio web depende de ciertas reglas de .htaccess para funcionar, revisa la documentación de tu software para opciones de configuración de Nginx.

Cambiar tu servidor

Cambiar a Nginx no afectará los archivos de tu sitio web. Permanecerán donde están y no serán tocados por este cambio.

  1. Navega a la página Servidores o al Panel de Dedicados.
  2. Haz clic en el botón Administrar a la derecha de tu servidor.
  3. Desplázate hasta la sección Configuración de Servidor.
  4. Al lado de Servidor HTTP, haz clic en el menú desplegable y selecciona la opción nginx con PHP #.#.
  5. Da clic en el botón Confirmar en la parte inferior para guardar tus cambios.
