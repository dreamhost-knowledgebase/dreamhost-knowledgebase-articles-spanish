Generalidades

Los nuevos usuarios de web se crean como usuarios tanto de SFTP como Shell. Este artículo explica la diferencia entre estos tipos de usuarios.

Usuarios de SFTP

Se requiere un usuario de SFTP para cargar un sitio web que hayas creado en los servidores de DreamHost o para hacer cambios en un sitio web existente.

El protocolo SFTP usa una conexión cifrada para acceder a un servidor. El puerto 22 se usa en tu cliente para conectarse al servidor a través de SFTP.

La página Archivos y Usuarios SFTP muestra una lista con el nombre de usuario y el tipo de con el que está configurado.

Haz clic en el botón Datos de Acceso para mostrar los detalles de este usuario. En la siguiente página puedes ver si el usuario está configurado como usuario SFTP.

  • Acceso a Archivos solo muestra SFTP
  • El interruptor Acceso Seguro Shell está apagado

Usuarios Shell

Los usuarios de Shell son usuarios que pueden conectarse a través de SSH y SFTP. El "Shell" es la línea de comandos de Linux en tu servidor de alojamiento que puedes usar para correr comandos.

Cambiar a un usuario Shell Para cambiar un usuario SFTP a uno Shell, solo da clic en el botón de interruptor para que se vuelva azul. Revisa el siguiente artículo para más instrucciones.
  • Editar un usuario existente para convertirlo en un usuario SHELL

Aquí podrás ver si el usuario configurado como un usuario SSH.

  • Acceso a Archivos muestra SFTP/SSH
  • El interruptor Acceso Seguro Shell está encendido
