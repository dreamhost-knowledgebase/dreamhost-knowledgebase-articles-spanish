Generalidades

DreamHost ofrece un servicio de seguridad pago que puedes habilitar a través del panel, que te permite escanear tu sitio web en busca de malware. Aquí están los detalles:

  • DreamShield (anteriormente llamado servicio de eliminación de malware) es un complemento que puedes usar en cualquier dominio (o subdominio) que desees para asegurarte de que esté libre de código malicioso que podría comprometer la información confidencial.
  • Cuando agregas DreamShield, este inmediatamente ejecuta un escaneo en tu sitio, y automáticamente lo escaneará semanalmente después de eso.
  • Se te notificará por correo electrónico si DreamShield encuentra algún problema. Tu panel también muestra esta información.
  • Si el sitio está limpio, tu panel muestra esta información. También se envía un correo electrónico que te informa que tu sitio está limpio.

A continuación se describe cómo agregar (o eliminar) DreamShield a un dominio usando tu panel DreamHost.

Por favor, tómate un momento para revisar los términos y el servicio de DreamShield.

Agregar DreamShield a tu dominio (o subdominio)

Solo puedes instalar DreamShield en un solo dominio o subdominio único. Si deseas instalarlo en example.com y blog.example.com, debes agregar un servicio DreamShield separado a cada dominio.

  1. Abre la página Administrar Dominios.
    Aparece la página de Manage Domains.
  2. Desplázate hacia abajo y luego haz clic en el enlace Add Malware Remover debajo de tu dominio.
    Se abre la página del removedor de malware DreamShield:
  3. Selecciona un plan y luego da click en el botón Start scanning... para confirmar y empezar el escaneo en tu dominio.
    Se abre la siguiente página que le muestra el progreso de su escaneo:
    DreamShield muestra lo siguiente cuando finaliza. Si DreamShield encuentra algún problema, también aparece aquí:
  4. Regresa a la página Administrar Dominios. 
  5. Haz clic en el enlace Site Clean (o Issues found) para ver resultados recientes o ajustar la configuración de DreamShield.

Agregar DreamShield al registrarte para obtener una cuenta DreamHost

Remover DreamShield

  1. Navega a la página Administrar Dominios. 
  2. Haz clic en el enlace Site Clean (o Issues found) debajo de tu dominio.

    Es posible que veas Outdated Software en lugar del enlace Site Clean.

    Si es así, haz clic para revelar qué software está desactualizado. Es muy recomendable ejecutar siempre el software más actualizado para garantizar que tu sitio esté seguro. Una vez que actualices el software de tu sitio, el enlace Site Clean aparecerá nuevamente.

     
  • Haz clic en el enlace Edit DreamShield settings.
  • Haz clic en el botón rojo Cancel DreamShield para que su dominio desactive el servicio.

¿Qué hacer si DreamShield encuentra algún problema con mi sitio web?

Cuando DreamShield completa un análisis de tu sitio web, se te envía un correo con los resultados e incluye cualquier problema que se encuentre. El correo también proporciona los siguientes pasos a seguir (incluidos los enlaces) para corregir cualquier intrusión u otras amenazas.

Aquí hay algunos enlaces a artículos populares para ayudarte a abordar cualquier problema en el caso de que DreamShield detecte cualquier problema en tu sitio web:

  • Cómo actualizar aplicaciones de tu sitio

Probar tu sitio después de haberlo limpiado

Después de haber limpiado tu sitio de cualquier código comprometido, es posible que desees confirmar que ahora es seguro. DreamShield se ejecuta una vez por semana, por lo que te informará automáticamente de cualquier problema la próxima vez que se ejecute.

Sin embargo, si deseas un escaneo inmediato, por favor genera un ticket con el equipo de soporte con el tema establecido en 'Abuso/Seguridad', que luego lo escaneará por ti.
