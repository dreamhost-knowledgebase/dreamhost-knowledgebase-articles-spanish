Descarga y descomprime tus archivos de correo

Cuando tu usas la opción de copia de seguridad de un clic del panel, todos los datos se guardan como archivos .tar.gz en tu panel. Consulta el siguiente artículo para obtener instrucciones sobre cómo descargar y descomprimir estos archivos:

  • Cómo generar una copia de seguridad de mi cuenta completa

Agregar una extensión de archivo .eml a todos los archivos

Cuando descomprimes tu archivo de correo usando el artículo anterior, los archivos descomprimidos se colocan en una carpeta titulada algo así como x15861576.

Esta carpeta contiene una carpeta Maildir. Haz clic en esta carpeta.

Dentro de esa carpeta encontrarás tus carpetas y mensajes de correo. Para tus mensajes de la Bandeja de entrada, haz clic en las carpetas /cur o /new. Para todos los demás mensajes, haz clic en tu carpeta y ve tus directorios /cur o /new.

Los correos aparecerán en el siguiente formato:

1435986g36.V34I895305bS934136.homiemail-master

Si arrastras y sueltas este archivo en tu cliente de correo, no funcionará. Primero debes cambiar el nombre del archivo.

Agrega .eml al final de cualquier archivo de correo. Por ejemplo:

1435986g36.V34I895305bS934136.homiemail-master.eml

Solo cuando terminen en este formato tu cliente de correo reconocerá el tipo de archivo. Luego puedes arrastrar y soltar este archivo en tu cliente.

Cómo agregar la extensión .eml a varios archivos a la vez

Mac o Linux

  1. Si estás en un computador Mac o Linux, abre una terminal y navega hasta el directorio donde se almacenan tus correos. Este debería ser el directorio /Maildir para una dirección de correo específica. 
  2. Corre el siguiente comando en el directorio de esta dirección: 
    [local ~]$ find . -type f -exec mv '{}' '{}'.eml \;
  3. Esto agrega la extensión .eml a todos los archivos en este directorio y todos tus subdirectorios. Ahora puedes arrastrarlos a tu cliente de correo para cargarlos en tu nueva dirección.

Windows

  1. Si estás en una computadora con Windows, descarga este programa que te permitirá ejecutar comandos de Unix.
    • Git para Windows
  2. En Windows, navega hasta el directorio donde realizaste una copia de seguridad y descomprimiste tu archivo de correo. Como se muestra arriba, asegúrae de estar en tu directorio /Maildir para una dirección de correo específica.
  3. Haz clic derecho dentro de este directorio.
  4. En el menú que aparece, haz clic en 'Git Bash Here'. Esto abre una terminal donde puedes ejecutar comandos de Unix.
  5. Corre el siguiente comando en el directorio de esta dirección: 
    [local ~]$ find . -type f -exec mv '{}' '{}'.eml \;
  6. Esto agrega la extensión .eml a todos los archivos en este directorio y todos tus subdirectorios. Ahora puedes arrastrarlos a su cliente de correo para cargarlos en tu nueva dirección.
