Generalidades

Este artículo detalla cómo transferir tu sitio web y alojamiento de correo de GoDaddy a DreamHost. El proceso traerá todos tus datos a su nuevo servidor DreamHost y luego actualizará el DNS una vez que todo esté listo para funcionar. La transferencia del registro de dominio es opcional, pero también se trata en esta guía.

Los siguientes pasos se describen en detalle:

  • Paso 1 — Crear una cuenta de DreamHost
  • Paso 2 — Respaldar tu información
  • Paso 3 — Subir tu información a DreamHost
  • Paso 4 — Ver tu sitio en DreamHost antes de apuntar los DNS
  • Paso 5 — Apuntar tu DNS a DreamHost

Servicio de migración

Si tienes problemas para migrar tu sitio web desde otro alojamiento o simplemente no tienes tiempo, DreamHost actualmente ofrece un servicio que se encarga de eso por ti. Puedes encontrar más información en el artículo de Servicios de Migración de Sitios Web de DreamHost.

Paso 1 — Crear una cuenta de DreamHost

  1. Regístrate en cualquier plan de DreamHost. Revisa el siguiente artículo para  más detalles:
    • Cómo inscribirte para una cuenta de DreamHost
  2. Durante el proceso de inscripción, selecciona la opción de I already have a domain para añadir tu dominio actual a tu nueva cuenta. Esto NO afecta tu sitio web actual.
  3. Completa el proceso de inscripción.
  4. Una vez que tu cuenta es creada, asegúrate de que tu dominio está configurado como Completamente Alojado. Es posible que también necesites crear una base de datos MySQL para tu sitio si depende de una base de datos.

DreamHost te envía un correo de 'solicitud para transferir el registro' después de completar el proceso de registro, así como información sobre cómo acceder a tu cuenta FTP. NO necesitas transferir tu registro. Puede permanecer en tu compañía de registro actual por ahora. Siempre puedes transferirlo en una fecha posterior.

Migrar los archivos del sitio

Descargar tus archivos de GoDaddy

  1. Inicia sesión en tu cuenta GoDaddy.
  2. Una vez inicies sesión, selecciona el enlace para la página My Products.
  3. Bajo la sección Web Hosting, encuentra tu dominio y haz clic en el botón Manage junto a el.
  4. En el panel de GoDaddy, en la barra de menú superior, selecciona File Manager.

    Se inicia el administrador de archivos cPanel, que muestra una lista de directorios a la izquierda.

  5. Haz clic en la carpeta  'public_html'. Todos los archivos de tu sitio estarán visibles a la derecha.
  6. Selecciona la casilla 'Select All' para seleccionar los archivos de tu sitio y luego haz clic en el botón Compress en la barra de menú superior izquierda.
  7. Selecciona la opción Archivo Zip y haz clic en Compress Now.
  8. Una vez que se completa la compresión y se crea el archivo zip, haz clic en él y selecciona el botón Download.
 

Paso 3 — Subir tu información a DreamHost

  1. Después de descargar el archivo zip, navega a la página Administrar Sitios Web del panel DreamHost.
  2. Debajo de tu dominio, haz clic en el botón My Files. Esto inicia sesión automáticamente en el servidor del administrador de archivos.
  3. Haz doble clic en el directorio de tu sitio de la lista de archivos y carpetas para ingresarlo.
  4. Una vez que estés en el directorio del sitio, selecciona el icono de carga. En el menú que aparece, selecciona Upload Zip. Selecciona el archivo zip de tu carpeta public_html que descargaste anteriormente.

    Aparece una ventana preguntándote qué quieres hacer con el archivo.

  5. Haz clic en el botón Upload and Unzip en ese menú para cargar los datos del sitio y completar el proceso.

Migrar bases de datos del sitio

Exportar la base de datos de tu sitio en GoDaddy

  1. Inicia sesión en tu cuenta de GoDaddy.
  2. Una vez inicies sesión, selecciona el enlace de la página My Products.
  3. En la sección de Web Hosting, busca tu dominio y haz clic en el botón Manage al lado.
  4. En el panel de GoDaddy, en la barra de menú superior, selecciona phpMyAdmin.
  5. Cuando se inicie phpMyAdmin, busca tu(s) base(s) de datos en el menú del lado izquierdo. Selecciona la base de datos de tu sitio.
  6. E al menú superior y haz clic en el botón Export.
  7. Selecciona Quick como tu método de exportación y SQL como tu formato. Presiona el botón Go y la base de datos se descargará a tu computador.

Subir tu base de datos en DreamHost

  1. Inicia sesión en tu cuenta DreamHost y navega a la página Bases de Datos MySQL.
  2. Inicia sesión en phpMyAdmin.
  3. Selecciona la base de datos de destino en el panel izquierdo.
  4. Haz clic en la pestaña Import en el centro superior del panel.
  5. En la sección File to import, haz clic en Browse y busca el archivo con la extensión .sql que deseas importar.
  6. Selecciona o deselecciona las casillas para 'Partial import' y 'Other options'.
  7. En el menú desplegable Format elige 'SQL'.
  8. Haz clic en el botón Go en la parte inferior para importar la base de datos.

Todas las tablas se mostrarán al seleccionar la base de datos en el panel izquierdo de phpMyAdmin.

Una vez que se carga la base de datos, es probable que debas actualizar el archivo de configuración de tu sitio para conectarte a la nueva base de datos. Revisa el siguiente artículo para obtener una lista de software popular y tus archivos de configuración:

  • Credenciales incorrectas de base de datos en tu archivo config

Correo Electrónico

Es posible que estés acostumbrado a ver referencias a "secureserver" en GoDaddy, que hacen referencia a tus servidores de correo. Si estás utilizando los servicios de correo DreamHost, puedes encontrar la información de tu nuevo servidor de correo aquí, y las credenciales de inicio de sesión de webmail están aquí.

Puedes importar tus mensajes de correo desde tus direcciones de correo alojadas en GoDaddy a tus nuevos servicios de correo DreamHost. Consulta el siguiente artículo para obtener instrucciones sobre este proceso.

  • Importar el correo otro alojamiento a DreamHost

Las direcciones alojadas en Google Workspace No es necesario migrar entre alojamientos. Solo debes actualizar los registros MX proporcionados por DreamHost para asegurarte de que tu dominio apunte a Google Workspace después de la migración. Consulta el siguiente artículo para obtener más detalles sobre este proceso.

  • DNS de Google Workspace
 

Paso 4 — Ver tu sitio en DreamHost antes de apuntar los DNS

Ahora que tu sitio web se ha migrado a los servidores de DreamHost, es hora de revisar el sitio para confirmar que se ve y funciona como se esperaba. Consulta el siguiente artículo para obtener información sobre cómo crear un dominio Mirror (Espejo). Esto te permite ver tu sitio tal como aparece en los servidores de DreamHost antes de actualizar tu DNS en tu alojamiento actual:

  • Previsualizar tu sitio con un subdominio dreamhosters.com antes de cambiar tu DNS

Si no deseas crear un dominio Mirror (Espejo), también puedes ver tu sitio utilizando el archivo hosts en el computador de tu hogar.

  • Ver tu sitio de DreamHost usando un archivo hosts
 

Paso 5 — Apuntar tu DNS a DreamHost

En este punto, tu sitio web y todos tus archivos están en los servidores de DreamHost, pero tu sitio todavía está 'alojado' en GoDaddy. Ahora debes actualizar tu DNS para apuntar a DreamHost. Consulta el siguiente artículo para obtener instrucciones sobre cómo hacer esto:

  • ¿A que registros DNS de DreamHost apuntar mi sitio?

Una vez que el DNS se resuelve en DreamHost, al visitar tu sitio se muestra el sitio tal como aparece en tu servidor DreamHost.
