Generalidades

Instalar una aplicación no es suficiente: para mantener la estabilidad y la seguridad del sitio, es importante mantener tus aplicaciones web actualizadas.

Este artículo detalla cómo actualizar las aplicaciones web instaladas a través del instalador de un clic en tu panel.

Algunas instalaciones de un clic en el panel no son tan actuales como la versión en el sitio web de los desarrolladores. En estos casos, la única opción para actualizar sería visitar el sitio web de los desarrolladores y seguir sus instrucciones sobre cómo actualizar manualmente.

Acciones de actualización automática

De manera predeterminada, las instalaciones con un clic están configuradas para actualizarse automáticamente con cada actualización importante. Recibirás una notificación por correo electrónico una vez que se haya completado una actualización.

Para cambiar la acción de actualización predeterminada para tu dominio, sigue estas instrucciones:

  1. Ve a la página One-Click Installs en el panel.
    Se abre la página de Instalaciones con un clic.
  2. Haz clic en el botón Manage Installed Applications para ver tus aplicaciones instaladas.
    Aparece una página que enumera tus dominios con aplicaciones instaladas.
  3. Debajo de la columna Upgrade Action, haz clic en el enlace que dice 'edit'.
    Verás las siguientes opciones:
  4. Haz una selección de las siguientes opciones:
    • No action: DreamHost no realiza actualizaciones de un clic para este dominio.
    • Notify me: Solo te avisa por correo electrónico cuando haya una actualización de revisión importante disponible. No se realiza ninguna actualización.
    • Upgrade automatically: Actualiza el sitio y te envía correos electrónicos automáticamente.
    • Upgrade without notification: Actualiza el sitio pero no te envía un correo electrónico.

Actualización manual de tus instalaciones con un clics

Si hay una actualización disponible para tu sitio y aún no se ha realizado, verás un "1" en la columna Updates Needed:

  1. Navega a la página One-Click Installs.
  2. Haz clic en el botón Manage Installed Applications.
  3. Hacia la derecha de tu dominio, revisa si ves un "1" al lado de 'Updates Needed'.
  4. Bajo la columna Actions, puedes hacer clic en el botón Upgrade to X.X para actualizar la versión de la aplicación que usa tu sitio.
    • En este ejemplo, la acción de actualización dice Upgrade to 4.9.7.
  5. Una ventana de confirmación aparece. Haz clic en el botón Ok para continuar.
  6. Un cuadro que dice Success! Aparece confirmando que la actualización está en proceso.

¿Qué pasa si moví el directorio en el que estaba instalada la aplicación de un clic?

Si has movido la ubicación del directorio de tu instalación de un clic, el panel no lo actualizará automáticamente, debes comunicarte con el área de soporte. Informa al soporte técnico de la Instalación con un clic y el directorio al que se movió. Soporte puede ajustar la ruta del panel por ti.
