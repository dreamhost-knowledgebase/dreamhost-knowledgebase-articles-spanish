Generalidades

Todos los planes de alojamiento en DreamHost admite medios de transmisión HTTP (también conocido como Descarga progresiva). Puedes usar tu servidor DreamHost para transmitir audio y video siempre que no infrinjan las leyes de derechos de autor.

Si tienes un acto musical o un negocio, la transmisión de medios es una herramienta mucho más poderosa que permite descargas de MP3 desde tus sitios web. Cuando transmites un archivo de audio o video, el usuario de ese archivo puede escuchar o ver ese archivo casi de inmediato porque se reproduce mientras se descarga. Cuando descargues un MP3, debes esperar hasta que se cargue todo el archivo antes de poder usarlo.

Streaming de audio HTML5

Puedes transmitir audio fácilmente con HTML5. Visita el siguiente enlace para más detalles:

  • Audio en HTML5

El archivo de audio que desea transmitir debe estar en un formato que el navegador pueda leer. Para archivos de música, mp3 es el estándar.

Aquí hay un ejemplo muy simple que crea una caja de control de audio:

  • Coloca el siguiente código en el <body> de tu documento HTML
  • Asegúrate de que el archivo .mp3 está en el mismo directorio que el archivo. De lo contrario, debes ajustar elsrc= attribute
<audio controls>
<source src="Für_Elise.mp3" type="audio/mpeg">
</audio>

Las etiquetas <audio> y <source> también tienen varios atributos que puedes ajustar:

  • Atributos de etiqueta de audio
  • Atributos de la etiqueta origen

Transmisión de video HTML5

Puedes transmitir video usando HTML5 de la misma manera. Visita el siguiente enlace para más detalles:

  • Vídeo en HTML5

El archivo de video que deseas transmitir debe estar en un formato que el navegador pueda leer. Para archivos de video, mp4 es el estándar.

Aquí hay un ejemplo muy simple que crea un cuadro de control de video:

  • Coloca el siguiente código en el <body> de tu documento HTML.
  • Asegúrate de que el archivo .mp4 esté en el mismo directorio que el archivo. De lo contrario, debes ajustar el src= attribute
<video width="320" height="240" controls>
<source src="movie.mp4" type="video/mp4">
</video>

La etiquetas <video> y <source> también tiene varios atributos que puedes ajustar:

  • Atributos de etiqueta de video
  • Atributos de la etiqueta de origen
