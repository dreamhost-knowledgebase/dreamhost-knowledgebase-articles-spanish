Generalidades

Si ya tienes un sitio de WordPress existente en una Instalación de Un-Clic, puedes importar tu contenido a tu sitio DreamPress.

Importar a DreamPress como entorno de prueba

Las siguientes instrucciones te ayudan a crear una copia segura de tu sitio DreamPress para fines de prueba, sin afectar el existente. Por ejemplo, puedes configurar una nueva instancia de DreamPress en un dominio diferente y luego importar tu sitio WordPress existente. Una vez que haya terminado de probar, elimínalo para evitar costos adicionales.

Puedes importar una instalación actual de WordPress en un sitio temporal de DreamPress. Esto te permite usar el dominio temporal como entorno de prueba antes de migrar a la URL real de tu sitio. Mira los siguientes artículos para obtener detalles sobre cómo crear un dominio temporal de DreamPress:

  • Crear un dominio temporal de DreamPress
  • Cambiar el nombre de un dominio DreamPress

Si no necesitas el entorno de prueba, puedes actualizar tu instalación de WordPress directamente a DreamPresss. De lo contrario, sigue los pasos a continuación para importar a un sitio existente de DreamPress.

Importar a DreamPress

  1. Navega hasta la página WordPress Administrado.
  2. Si deseas cambiar el nombre de su instalación de WordPress a otro dominio durante la importación, crea una nueva instalación de DreamPress.
    Si deseas probar tu importación en un dominio temporal, crea un sitio temporal de DreamPress.
  3. Una vez creado, haga clic en el botón Administrar.
  4. Da clic a la pestaña Dominio en la parte superior.
  5. En la parte inferior derecha, haz clic en el título 'Importa un sitio web existente'.
    Aparece una ventana que le permite seleccionar un sitio desde el cual importar su contenido:
  6. Elige el sitio desde el que deseas importar datos en el menú desplegable.
  7. Da clic al botón Empezar importación para iniciar el proceso de importación.

Esto crea un clon/copia del sitio de WordPress y cambia el nombre de dominio. Esto es ideal para probar DreamPress, probar nuevos plugins o temas, etc., ya que deja el sitio original intacto. También es bueno para dividir una sola instalación de WordPress de un dominio que tiene múltiples aplicaciones. Por ejemplo, puede convertir example.com/blog en blog.example.com.

Si estás importando tu instalación de WordPress desde un nombre de dominio diferente, la herramienta de importación intentará reemplazar el dominio/URL. Si no lo haces, el correo que sigue a la importación lo indicará. En cualquier caso, siempre debes verificar si estos datos son correctos después de que se haya completado la importación.

Si deseas migrar el contenido de WordPress de un dominio diferente a un dominio que desees en DreamPress, debes agregar el dominio primero a DreamPress y luego usar la función Importar.
