El siguiente artículo describe cómo instalar WordPress desde tu panel de DreamHost. También detalla las características adicionales que recibes si eliges la instalación gratuita Deluxe.

Este artículo explica cómo instalar un nuevo sitio de WordPress en DreamHost. Miar el siguiente artículo si necesitas migrar tu sitio a DreamHost.

  • Migrar un sitio WordPress desde un alojamiento diferente

Antes de instalar

Para usar el instalador de Un-Clic, tu dominio primero debe estar completamente alojado.

Esto significa que no se puede estar Redirigido, Reflejado, Parqueado, o configurado como Solo DNS.

Antes de instalar WordPress, verifica qué archivos ya están en el directorio de tu sitio web.

Si hay archivos de WordPress existentes

Si hay archivos de WordPress existentes, se les cambiará el nombre con una cadena de números al final. Por ejemplo:

  • index.php.1353108851

Consulta el siguiente artículo para obtener instrucciones sobre cómo eliminar estos archivos.

  • Eliminar archivos duplicados de instalaciones de Un-Clic 

Si hay archivos que no son de WordPress en tu directorio

Si hay archivos o directorios que no sean de WordPress, la instalación no los tocará. Los nuevos archivos de WordPress se instalarán junto con tus archivos existentes.

Instalar WordPress

En el panel de Instalaciones de Un-Clic

  1. Para instalar WordPress en DreamHost, inicia sesión en tu panel y dirígete a la página de Instalaciones de Un-Clic.
  2. Haz clic en el botón Learn More & Install.
  3. Introduce lo siguiente:
    • Install to: Selecciona el dominio en el que deseas instalarlo. El segundo cuadro de texto es si deseas instalarlo en un subdirectorio (si no deseas que esté almacenado en el directorio principal). Por ejemplo, si deseas que se instale en example.com/blog, primero selecciona example.com en el primer menú desplegable. En el segundo cuadro de texto escribe blog.
    • Select Database: Probablemente desees 'Crear automáticamente base de datos', ya que esto crea una nueva base de datos única para tu sitio. También puedes elegir una base de datos existente si lo deseas.
    • Deluxe Install: Se recomienda marcar esta casilla si deseas las funciones adicionales que se mencionan a continuación. Esta es una adición gratuita a tu instalación.
    • Install WP Website Builder: Esto agrega la herramienta WP Website Builder a tu panel para ayudarte a crear tu sitio.
  4. Haz clic en el botón Install it for me now!. En los próximos 15 minutos, se te enviará un correo electrónico al correo electrónico principal que figura en tu cuenta de DreamHost. Este correo electrónico detalla los pasos a seguir para completar la instalación.

En el panel de Sitios Web

  1. Inicia sesión en tu panel y ve a la página de Administrar Sitios Web.
  2. Encuentra tu sitio web y presiona el botón Administrar.
  3. Desplázate hacia abajo hasta la sección de WordPress y selecciona el botón Instalar.
  4. Confirma la instalación presionando el botón Instalar WordPress. En 15 minutos, se te envía un correo electrónico al correo electrónico principal que figura en tu cuenta de DreamHost. Este correo electrónico detalla los pasos a seguir para completar la instalación.

Cómo iniciar sesión en tu sitio de WordPress

Después de instalar WordPress, recibirás un correo a la dirección de correo principal en tu cuenta de DreamHost. Este correo enumera los pasos a seguir para completar la instalación.

Cuando WordPress finalmente está instalado en tu sitio, puedes iniciar sesión en él en:

  • example.com/wp-login.php

La instalación Deluxe

La instalación Deluxe gratuita incluye varias personalizaciones importantes para facilitar el uso de usuario principiantes. Incluye temas gratuitos y de código abierto de Automattic, una selección estándar de plugins y una mejora de la seguridad.

Temas

Las instalaciones Deluxe incluyen la siguiente selección de temas gratuitos y de código abierto de Automattic:

  • argent
  • button
  • dyad
  • illustratr
  • kelly
  • libre
  • orvis
  • resonar
  • scrawl
  • sketch
  • twentyfifteen (También presente en instalaciones normales)
  • twentyseventeen (También presente en instalaciones normales)
  • twentysixteen (También presente en instalaciones normales)

Plugins

La instalación de Un- Clic de DreamHost también incluye los siguientes plugins populares:

  • Akismet (También presente en instalaciones normales).
  • Hello Dolly
  • Jetpack
  • WP Super Cache (habilitado automáticamente por defecto).
  • WPForms Lite (habilitado automáticamente por defecto).

Aunque estos plugins se instalan de manera predeterminada para las instalaciones Deluxe, es posible que necesites activarlos en tu panel de control de WordPress visitando 'Plugins '> 'Plugins instalados'.

Mejora de seguridad

Además, la instalación de WordPress agrega protección contra la ejecución de scripts dentro de tu directorio de cargas.

En términos más técnicos, esto significa que un archivo .htaccess con los siguientes contenidos se agrega automáticamente a tu directorio /wp-content/uploads/:

### Generated by DreamHost. DO NOT modify!!! ###
SetHandler no-handler
################################################

Si tu sitio se ve comprometido por la colocación de un script malicioso en tu directorio de subidas, el script se devuelve como una descarga cuando se accede. Esto evita que el script se ejecute y ayuda a evitar daños mayores.

DreamHost continuará agregando mejoras de seguridad adicionales a medida que estén disponibles.

Problemas de instalación

Si la instalación de Un-Clic no se completa en 20 minutos o falla con un error, comunícate con Soprte.

Además, si la instalación se completa pero no recibes el correo con las instrucciones sobre cómo acceder a tu nueva instalación de WordPress, siempre puedes consultar tus Mensajes recientes ("Recent Messages") en la página Contactar a Soporte, y así obtener una copia archivada de ese correo. 

El enlace de instalación no funciona

Un error 404 o "Sitio No Encontrado" puede aparecer en las nuevas configuraciones de WordPress cuando intentas ver el sitio después de cambiarlo desde otra opción de alojamiento (como reflejo/espejo o redirección). En estos casos, el sitio aún se resuelve con los registros antiguos Mirror o Redirection A, en lugar de los nuevos registros A 'Completamente Alojado'. Afortunadamente, esto se resolverá una vez que se complete la propagación de DNS, suponiendo que el sitio esté usando los nameservers de DreamHost.
