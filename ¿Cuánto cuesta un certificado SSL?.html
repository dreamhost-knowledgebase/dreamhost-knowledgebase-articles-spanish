Generalidades

Puedes agregar dos tipos diferentes de certificados en tu panel DreamHost:

  • Certificado 'Let's Encrypt' gratuito.
  • Certificado de Sectigo pagado.

Las diferencias entre estos se describen a continuación.

Certificados SSL Let's Encrypt SSL gratuitos

Let's Encrypt es un Autoridad Certificadora (CA) que ofrece certificados SSL GRATUITOS. Este proyecto fue pionero en hacer que las conexiones encriptadas fueran estándar de forma predeterminada a través de Internet.

Aunque estos certificados son tan seguros como los certificados pagos actuales, solo deben ser usados por sitios más pequeños que no intercambian información confidencial. Esto incluiría:

  • Blogs
  • Foros
  • Sitios personales

Esto se debe a que estos certificados carecen de la información de la organización, lo que significa que los visitantes no pueden validar el certificado al verlo. Además, estos son potencialmente vulnerables a los ataques de phishing. Por ejemplo, un usuario malintencionado podría crear un sitio similar con un certificado para crear una copia falsificada de tu tienda en línea. Por estos motivos, no se recomiendan para sitios de eCommerce que procesan información de pago.

Certificado firmado profesionalmente pagado (Sectigo - $15USD/año)

Cualquier certificado de Comodo emitido después de enero de 2019 hace referencia a Sectigo como la autoridad de certificación, mientras que cualquier certificado emitido antes de esta fecha hace referencia a Comodo. Ambos nombres se refieren a la misma compañía.

Estos tipos de certificados validan todo lo que hace un certificado de 'Let's Encrypt' al mismo tiempo que validan información adicional de la organización sobre quién está comprando el certificado, como su Nombre, ciudad, estado y País. El nombre de la organización también se incluye en el certificado, lo que agrega una capa adicional de confianza a los visitantes para que puedan asegurarse de que el sitio web y la empresa sean confiables.

Sitios eCommerce

Si tienes un sitio orientado al cliente o, lo que es más importante, un sitio que maneja información confidencial, como contraseñas y datos de tarjetas de crédito (como un sitio de eCommerce), se recomienda encarecidamente un certificado Sectigo pagado. ¿Qué tipo de certificado debería usar para mi sitio eCommerce?

Puedes usar un certificado 'Let's Encrypt' o 'Sectigo' firmado profesionalmente para tu sitio web de eCommerce, ya que ambos usan el mismo nivel de cifrado.

La única diferencia es que un certificado 'Sectigo' pagado incluye la información de la organización. Esta validación adicional puede parecer más confiable para los clientes, por lo que se recomienda este tipo para los sitios eCommerce.

IP Unica

Una IP Unica NO se requiere dirección para usar un certificado SSL. Sin embargo, si planeas usar una aplicación de eCommerce, te recomendamos que agregues una dirección IP única para proporcionar la máxima compatibilidad con los navegadores de Internet más antiguos.

Para obtener más información sobre las direcciones IP únicas (incluidas las instrucciones para agregar una a su dominio), revisa el artículo de IP única.
