Generalidades

Puedes agregar dos tipos diferentes de certificados en tu panel DreamHost:

  • Certificado 'Let's Encrypt' gratuito
  • Certificado de Sectigo pagado

Las diferencias entre estos se describen a continuación.

Certificados SSL Let's Encrypt SSL gratuitos

Let's Encrypt es una Autoridad de Certificación (CA) que ofrece certificados SSL GRATUITOS que son tan seguros como los certificados pagos actuales, aunque se recomienda un certificado pago para sitios de comercio electrónico ya que los certificados 'Let's Encrypt' no ofrecen 'Validación Organizacional'. Este proyecto fue pionero en hacer de las conexiones cifradas el estándar predeterminado en Internet.

Certificado firmado profesionalmente pagado ($15/años)

El 1 de noviembre de 2018, Comodo anunció que la compañía está cambiando su nombre a Sectigo. Para obtener más información al respecto, consulte el siguiente anuncio en el sitio web de Sectigo:

  • Comodo CA es ahora Sectigo

Cualquier certificado de Comodo emitido después de enero de 2019 hará referencia a Sectigo como la autoridad de certificación, mientras que cualquier certificado emitido antes de eso seguirá haciendo referencia a Comodo. Estas siguen siendo la misma compañía, y la única diferencia es el nombre.

Como se mencionó anteriormente, un certificado autofirmado arroja una advertencia de navegador cuando visita el sitio. Si tienes un sitio orientado al cliente, o lo que es más importante, un sitio que maneja información confidencial como contraseñas y datos de tarjetas de crédito, se recomienda un certificado Sectigo pagado.

Los certificados pagados no muestran advertencias del navegador y ofrecen información adicional en el certificado para garantizar su autenticidad. Por ejemplo, estos certificados incluyen Validación Organizacional (OV) que brinda a tus usuarios más tranquilidad para garantizar que el certificado se haya emitido correctamente a una organización legítima.

IP Unica

Una IP Unica NO se requiere dirección para usar un certificado SSL. Sin embargo, si planeas usar una aplicación de comercio electrónico, te recomendamos que agregues una dirección IP única para proporcionar la máxima compatibilidad con los navegadores de Internet más antiguos.

Para obtener más información sobre las direcciones IP únicas (incluidas las instrucciones para agregar una a su dominio), consulte el artículo de IP única.
