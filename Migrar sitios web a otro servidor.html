Generalidades

DreamHost ofrece varios planes de alojamiento que puede usar para poner tu sitio en línea. Dependiendo de tus necesidades de tiempo extra, es posible que desees cambiar el plan de alojamiento que estás usando. Este artículo detalla los pasos necesarios para migrar tu sitio a otro plan de alojamiento.

Migrar un solo sitio a un servidor diferente

Puedes mover un solo dominio creando un nuevo nombre de usuario en el nuevo servidor y luego cambiando tu sitio para usar este nuevo nombre de usuario.

Cuando selecciones este nuevo nombre de usuario, marque la casilla 'Move files to the new user?' Cuando guardes los cambios, todos los archivos se migrarán al nuevo servidor. Mira el siguiente artículo para más detalles:

  • Migrar un solo sitio web a un servidor diferente

¿Cuánto tardan en migrar los datos del usuario?

Esto depende de la cantidad de datos que hay que copiar. Cuando guardas la configuración de tu dominio con el nuevo nombre de usuario, los datos de su sitio web comienzan a copiarse inmediatamente en el nuevo servidor. Sin embargo, dependiendo de la cantidad de datos que tengas, esto podría tomar 30 minutos o varias horas.

¿Tendré que cambiar el hostname de mi cliente FTP cuando me conecte?

Posiblemente. Hay dos nombres de host que puedes usar para conectarte a tu sitio web:

  • Usando el nombre del sitio web
  • Usando los nameservers

Si estás usando el nombre del sitio web para conectarte, iniciará sesión en el servidor que DreamHost ha asignado a tu sitio.

Si estás usando  nameservers, este debe actualizarse a tus nuevos nameservers. Revisa el siguiente artículo para más detalles:

  • Buscar tu hostname de FTP

Propagación de DNS después de migrar

Si estás migrando entre dos servidores DreamHost, no hay tiempo de inactividad durante el movimiento ya que los datos se copian mientras el servidor anterior continúa sirviendo al sitio web. Sin embargo, no debes trabajar en tu sitio durante este período de 4 a 6 horas. Esto se debe a que es posible que tus cambios se realicen en el sitio anterior, no en el nuevo. Debes esperar hasta que el DNS se haya actualizado completamente para continuar trabajando en tu sitio web.

Si tu dominio no está utilizando los nameservers de DreamHost, deberás actualizar sus registros DNS manualmente en la compañía donde están alojados tus nameservers.

Puede ver todos los registros DNS configurados para cualquier sitio que aloje en DreamHost en tu panel. Ten en cuenta que después de cualquier cambio de DNS, los registros deben propagarse en línea.
