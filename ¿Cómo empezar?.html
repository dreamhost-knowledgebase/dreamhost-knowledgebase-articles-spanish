Configuración Básica

Crear una cuenta en DreamHost

Si deseas utilizar algún servicio de DreamHost, debes crear una cuenta. Revisa el siguiente artículo para obtener instrucciones sobre cómo abrir una cuenta.

  • Cómo abrir una cuenta en DreamHost

Entrar al panel

Una vez terminado el proceso de registro, recibirás un mensaje para notificarte que tu cuenta está configurada. Desde allí, puedes iniciar sesión en el panel en:

  • https://panel.dreamhost.com/

Para iniciar sesión, usa las credenciales del correo electrónico que se te enviaron.

Registrar un dominio

Para cargar cualquier página web, el propietario debe primero ser propietario del registro del nombre de dominio. Si no posee el registro, no tiene derecho a utilizar el nombre de dominio.

Para comprar un dominio, dirígete a la página de Registros de Dominios.

En la página de Registrar Dominios, puedes registrar cualquier dominio que desees que aún esté disponible:

DreamHost permite el registro de muchos dominios de nivel superior diferentes (TLD). Puedes ver una lista completa de estos en la página de Registro de Dominios haciendo clic en el enlace Ver precio completo.

Para más información, por favor revisa el artículo Registro de Dominios.

Agregar un plan de alojamiento

Después de comprar un dominio, debes agregar un plan de alojamiento para que el sitio resuelva en línea. Hay diferentes opciones de alojamiento que puedes usar para tu dominio.

  • Completamente alojado — tu sitio se encuentra alojado en un servidor y es visible en línea.
  • Remixer — tu sitio se encuentra alojado en el plan Remixer (nuestra herramienta para construir sitios web).
  • WordPress — tu sitio se encuentra alojado en un servidor (completamente alojado) pero está ejecutando el programa WordPress.
  • DreamPress — tu sitio tiene completamente alojado corriendo WordPress pero se encuentra en servidor de DreamPress.

Cómo agregar un plan de alojamiento

Para agregar un plan de alojamiento, dirígete a la página Administrar Cuenta.

En la página Administrar Cuenta, verás una sección titulada Planes de servicio:

Si ya has agregado un plan, lo verás aquí. En este ejemplo, se agregó un plan de alojamiento compartido que se paga por adelantado.

Agregar un plan de alojamiento a un dominio

Revisa el siguiente artículo para obtener instrucciones sobre cómo agregar alojamiento a tu dominio para que esté completamente alojado.

Incluso si configuras un sitio para que esté COMPLETAMENTE ALOJADO en tu panel, no quiere decir que el sitio esté alojado en DreamHost. Los registros DNS también deben estar apuntados a tu servidor de DreamHost. La compañía que aloja tus Nameservers, controla tus DNS. Puedes revisar adonde apuntas tus servidores de dominio al hacer una búsqueda pública de tu información WHOIS, aquí:

  • dreamhost.com/es/dominios/

Si los nombres de servidor ya están apuntando a DreamHost, entonces to lo que necesitas es configurar el alojamiento. Revisa los siguientes artículos para más información:

  • Cómo agregar Dominios y Subdominios
  • DNS
  • Nameservers

Alojamiento en Remixer

DreamHost también ofrece un creador de sitios web personalizado llamado 'Remixer' que no requiere experiencia en codificación. Remixer es un servicio de alojamiento que distinto a plan de completamente alojado como el Compartido, VPS, Dedicado, o DreamPress. Con Remixer, puedes construir tu sitio web usando la interfaz en línea con sólo hacer click. Opciones de alojamiento en WordPress

Agregar WordPress

WordPress es el sistema de administración de contenido (CMS) más popular del mundo, y DreamHost es uno de los pocos proveedores recomendados por WordPress.org. Si eres nuevo en el alojamiento de un sitio web, querrás darle un vistazo a WordPress para ayudarte a construir tu sitio web. En cuestión de minutos, puedes crear un sitio que incorpore todas las características modernas que ahora incluyen los sitios web, con una capacidad casi ilimitada para agregar nuevas funciones a medida que las necesites.

Para ayudarte a empezar a construir tu sitio WordPress en DreamHost, consulta los siguientes artículos:

  • Planes de Alojamiento de WordPress en DreamHost

Alojamiento DreamPress

DreamPress es el Alojamiento de WordPress administrado de DreamHost, que proporciona funciones aún más potentes, como planes de memoria y almacenamiento más grandes, velocidades más rápidas y almacenamiento en caché mejorado.

Agregar un certificado SSL

Agregar un certificado SSL a un dominio

Después de configurar el alojamiento para tu dominio, te recomendamos encarecidamente que también le agregues un certificado SSL al dominio (ya sea un certificado gratuito 'Let's Encrypt' o uno firmado profesionalmente). Agregar un certificado es un proceso simple que no solo asegura tu dominio para realizar transacciones monetarias y de eCommerce, sino que también logra que tu sitio sea calificado como seguro. De hecho, Google dará una calificación más favorable ya qe tener un sitio seguro es un factor importante para los motores de búsqueda.

Para ayudarte a iniciar el proceso de agregar un certificado, por favor dale un vistazo a los siguientes artículos:

  • Agregar un certificado Let's Encrypt gratis
  • ¿Cómo comprar un certificado SSL firmado profesionalmente?
  • ¿Cómo agregar un certificado SSL autofirmado?

Configuraciones adicionales

Los siguientes son servicios o productos adicionales que puedes agregar a tu sitio, pero no son obligatorios para que tu sitio funcione.

Crear una dirección de correo

Una vez que agregues el alojamiento a un dominio, puedes crear una dirección de correo.

Para crear una dirección de correo:

  1. Dirígete a la página Administrar Correo.
  2. Haz clic en el botón Crear nueva dirección de correo en la parte superior.
    Luego podrás ingresar un nuevo correo y una contraseña.

Una vez que guardes los cambios, los registros DNS necesarios para el correo se agregarán al DNS de tu sitio.

Puedes encontrar más información sobre la creación y configuración de cuentas de correo con DreamHost en la página Configuración de correo

También es posible alojar tu correo electrónico fuera de DreamHost. Para alojar en otro lugar, puede configurar registros MX personalizados para que apunten a tu servidor de correo externo. Visita el artículo Registros MX para más información.

Agregar estadísticas del sitio

Todas las cuentas de DreamHost tienen la capacidad de agregar estadísticas del sitio a cualquier sitio web totalmente alojado; las estadísticas las puedes habilitar en el panel. Allí, puede realizar un seguimiento de la información, como el número de solicitudes, el tráfico por dominio, el tráfico por archivo, las solicitudes por tipo de navegador / SO / URL, y más.

Para obtener más información sobre cómo agregar estadísticas del sitio, consulta los siguientes artículos:
