Generalidades

wp-cli es una herramienta de línea de comandos, integrada en todos los servidores DreamHost, que puedes usar para administrar tu sitio de WordPress. Si no deseas usar el instalador de un solo clic en el panel, puedes usar esta herramienta para descargar una nueva copia de tus archivos de WordPress.

Instalar WordPress

  1. Inicia sesión en tu servidor a través de SSH.
  2. Navega al directorio de tu sitio web.
  3. Ejecuta el siguiente comando para descargar todos los archivos de WordPress: 
    [server]$ wp core download
  4. Visita un sitio en tu navegador para completar el proceso de instalación.
