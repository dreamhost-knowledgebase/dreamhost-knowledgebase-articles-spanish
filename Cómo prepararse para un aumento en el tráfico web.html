Generalidades

Si esperas un aumento significativo en el tráfico web, probablemente tengas algunas preguntas sobre cómo optimizar tu sitio web para lograr un rendimiento óptimo. A continuación se proporcionan respuestas a algunas de estas preguntas.

También es posible que desees enviar un mensaje a soporte y hacerles saber que esperas un mayor tráfico web para que puedan vigilarlo.

Asegúrate de habilitar las estadísticas del sitio para permitirte ver y controlar los cambios en el tráfico de tu sitio. Revisa el siguiente artículo para más detalles:

  • Generalidades sobre estadísticas
 

Mantener tu software actualizado

Asegúrate que tus aplicaciones están actualizadas. Por ejemplo, si estás ejecutando un sitio de WordPress, asegúrate de que lo siguiente esté actualizado:

  • Versión de WordPress.
  • Versión de Plugins.
  • Versión de Tema.

Si estás usando una instalación de WordPress de DreamHost puedes actualizar la versión del software en el panel. Visita el siguiente artículo para más detalles:

  • Actualizaciones automáticas para las instalaciones de WordPress de DreamHost
 

Habilitar el almacenamiento en caché

Si tu sitio proporciona contenido dinámico o está basado en una base de datos (i.e., WordPress, Joomla, ZenCart, MediaWiki, phpBB, WebCalendar, etc.), puedes habilitar alguna forma de plugin de "almacenamiento en caché". Consulta la página de soporte de esa aplicación en particular para obtener más detalles y sugerencias. Si estás ejecutando un sitio de WordPress, revisa el siguiente artículo:

  • Almacenamiento en caché de WordPress

Mira el siguiente artículo para más opciones de almacenamiento en caché:

  • Opciones de Almacenamiento en caché
 

Optimización de base de datos

Para limpiar y optimizar tus bases de datos, visita el siguiente artículo para obtener más detalles:

  • Optimizar una base de datos
 

Deshabilitar software no usado

Deshabilita plugins innecesarios, poco confiables o que requieran muchos recursos, si es posible. Si estás ejecutando un sitio de WordPress, desactiva los temas no utilizados, los plugins o las instalaciones adicionales de WordPress. Visita el siguiente artículo para más detalles:

  • Remover software viejo
 

Prevenir spam

Si tu sitio permite que los visitantes dejen comentarios, asegúrate de utilizar algún tipo de plugin para evitar comentarios de spam. Por ejemplo, si estás usando WordPress, Akismet es una buena opción.

 

Asegurar los formularios de contacto

Asegúrate de que tus scripts de formulario "contáctenos" no sean vulnerables al abuso por parte de spammers.

 

Optimizar archivos multimedia

Es posible que desees reemplazar temporalmente grandes gráficos, archivos, películas, etc. con versiones de menor resolución (más pequeñas).

  • Optimizando Multimedia

Convierte contenido dinámico en contenido estático

Si puedes convertir páginas dinámicas en páginas estáticas, eso sería de gran ayuda (pero puede ser difícil de hacer). Algunas herramientas de almacenamiento en caché, como wp-super-cache, incluyen la capacidad de hacer esto usando el plugin.

Acelerar o deshabilitar tu sitio

Si por alguna razón tu incremento de tráfico causa problemas al servidor, puede ser necesario que el soporte "acelere" tu sitio (restringiendo el número de visitantes simultáneos o el rendimiento máximo, o similares) o posiblemente incluso lo deshabilite. Esto se hace para ser justos con los otros clientes en el servidor.

Recuerda que tienes un plan de Shared Hosting y que un cliente no puede poner en peligro el funcionamiento del servidor para todos los demás clientes. El soporte de DreamHost ofrece sugerencias de optimización para ayudar a que tu sitio permanezca en funcionamiento, sin embargo, en última instancia, es tu responsabilidad asegurarte de que tu sitio no afecte negativamente a otros usuarios.

Si por algún motivo es necesario limitar tu sitio, el soporte se comunicará con el propietario de la cuenta con los detalles. Si hay un problema con tu sitio que puedes resolver, asegúrate de notificar al soporte con detalles de lo que hiciste para solucionar el problema. Luego, el soporte puede quitar el acelerador en lugar.

 

Opciones de actualización

Si ya no es posible optimizar tu sitio, puede ser necesario actualizar tu plan de alojamiento a un VPS. Este es un plan de alojamiento independiente en un VPS que te brinda tu propia 'máquina virtual'. El plan VPS te permite asignar la cantidad de RAM necesaria para ejecutar tu sitio. Incluso en el caso de un aumento en el tráfico, tu sitio no afectará a otros usuarios en el servidor.
