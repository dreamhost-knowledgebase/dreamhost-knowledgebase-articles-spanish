Generalidades

Este artículo explica cómo usar la Llave de Migración que se encuentra en el plugin Plugin de Migración Automatizada de DreamHost (DreamHost Automated Migration Plugin) para migrar tu sitio WordPress a DreamHost.

Directorio de WordPress

Este plugin solo funcionará si tu instalación de WordPress se encuentra en el directorio principal del sitio web.

  • example.com

No funcionará si WordPress está instalado en un subdirectorio:

  • example.com/blog

Migrar tu sitio a DreamHost

 

Paso 1 — Ubicar las credenciales de tu sitio web

Necesitarás las siguientes credenciales del sitio web de DreamHost al que estás migrando:

  • Usuario/Contraseña del Sitio Web: Revisa el artículo Generalidades sobre FTP y sus credenciales para ver instrucciones sobre cómo ubicar esta información. Anota el tipo de usuario asignado a tu dominio.
  • Hostname: Revisa el artículo Generalidades sobre FTP y sus credenciales para ver instrucciones sobre cómo ubicar esta información.
  • IP del Servidor FTP de destino: Revisa el artículo Ver tus registros DNS de DreamHost para ver los pasos sobre cómo ver la dirección IP de tu sitio web.

Paso 2 — Instalar el plugin en el alojamiento anterior

  1. Inicia sesión en tu sitio WordPress en vivo en la compañía de alojamiento antigua.
  2. Selecciona Plugins > Añadir Nuevo en el menú de la derecha.
  3. Escribe DreamHost Automated Migration Plugin en el campo de la casilla ubicada en la parte superior derecha.
  4. Haz clic en el botón Instalar Ahora en la casilla cuando esta aparezca.
  5. Da clic al botón Activar.

Paso 3 — Configurar el plugin

  1. Haz clic en DreamHost en el menú de la izquierda.
    Se abre la página del plugin.
  2. Ingresa la dirección de correo electrónico donde quieres recibir las notificaciones sobre el estado de la migración.
  3. Revisa y acepta los Términos & Condiciones y Política de Privacidad.
  4. Haz clic en el botón Migrar para abrir la página de configuraciones:
  5. Haz clic en el enlace Ingresar los detalles del Alojamiento Manualmente en la parte superior derecha.
  6. Ingresa lo siguiente:
    • URL del Sitio de Destino — El nombre de dominio al que estás migrando. (Ejemplo: https://example.com).

      Suele ser el mismo dominio que tu sitio en tu alojamiento actual. Pero también podría ser un nuevo dominio en DreamHost. Asegúrate de agregar https:// antes del dominio.

    • IP del Servidor FTP de destino — La dirección IP del servidor. Revisa arriba para ver más detalles.
    • Tipo de FTP — Selecciona SFTP para usuarios de SFTP/SSH.
    • Usuario FTP — El nombre de usuario. Revisa arriba para ver más detalles.
    • Contraseña de FTP — La contraseña. Revisa arriba para ver más detalles.
    • Sitio Web de Origen Protegido con Contraseña: Si el sitio web de origen está protegido con contraseña, ingresa tus credenciales.
    • Sitio Web de Destino Protegido con Contraseña: Si el sitio web de destino está protegido con contraseña, ingresa tus credenciales.
    • Opciones Avanzadas: Deja estos campos en blanco.
    • Agregar Carpetas: Esta opción te permite seleccionar archivos principales no específicos de de WordPress en caso de que tengas una configuración personalizada.
    • Agregar Tablas: Tu base de datos de WordPress está seleccionada de forma predeterminada. También puedes seleccionar tablas de bases de datos adicionales si es necesario.
  7. Haz clic en el botón Migrar.
    Aparece informe de progreso. Cuando se complete la migración, aparecerá un mensaje exitoso y recibirás un correo de confirmación.

Revisar el sitio migrado en DreamHost

Si el nombre de dominio de DreamHost es el mismo

example.com => example.com

Si el dominio de tu sitio en el alojamiento anterior y el de DreamHost son el mismo, al visitar el sitio se seguirá mostrando en tu alojamiento anterior.

Para ver cómo aparece en tu servidor DreamHost, debes crear un archivo hosts en tu computador. Esto te permite ver el sitio en DreamHost mientras todavía está alojado en tu antigua compañía de alojamiento. Revisa el siguiente artículo para más detalles:

  • Ver tu sitio DreamHost usando un archivo hosts

Apuntar el DNS a DreamHost

Una vez que hayas confirmado que el sitio se migró correctamente, puedes apuntar el DNS de tu dominio a DreamHost. Para obtener más información sobre cómo cambiar los nameservers en tu alojamiento actual, revisa el siguiente artículo:

  • Cambiar los nameservers en tu alojamiento actual

Si el dominio de DreamHost es diferente

example.com => dreamhostexample.com

Si migraste a un dominio diferente, y los registros DNS de tu sitio web están alojados en DreamHost, visitar el sitio mostrará el nuevo contenido migrado.
