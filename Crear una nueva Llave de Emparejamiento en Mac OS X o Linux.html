Crear una Llave de Emparejamiento

Para generar una nueva Llave de Emparejamiento, corre los siguientes comandos en el computador de tu hogar.

  1. Abre tu terminal y corre el siguiente comando con tu nombre de usuario. 
    [local]$ ssh-keygen -t rsa
    Esto crea un par de claves pública/privada del tipo (-t) rsa
    Generating a public/private rsa key pair.
Enter the file in which you wish to save they key (i.e., /home/username/.ssh/id_rsa):
  2. Presiona Enter en tu teclado para continuar con el nombre y el directorio predeterminado.

    La línea de comando por defecto guarda la nueva Llave par en el directorio /home/username/.ssh/ y lo llamará id_rsa. Sin embargo, tu puedes cambiar la ubicación o el nombre del archivo aquí si deseas escribiendo la ruta completa.

    Enter a passphrase (leave empty for no passphrase).
  3. Presiona Enter en tu teclado para continuar.

    No se recomienda que uses una frase de acceso.

    Enter same passphrase again:
  4. Presiona la tecla Enter en tu teclado para continuar.
    El siguiente mensaje aparece:
    Your identification has been saved in /home/username/.ssh/id_rsa
Your public key has been saved in /home/username/.ssh/id_rsa.pub

The key fingerprint is:
ar:bc:d3:9e:g3:1f:63:6f:6b:32:2e:97:ee:42:e1:be username@server.dreamhost.com

The key’s randomart image is:

+--[ RSA 2048]----+
| ..+**B.o++o     |
|  . o+==o. o     |
|    . .oo.=      |
|      . +E+ .    |
|        S .      |
|                 |
|                 |
|                 |
|                 |
+-----------------+
  5. Corre el siguiente comando para ver la nueva Llave pública y privada: 
    [local]$ ls -1 ~/.ssh/
id_rsa
id_rsa.pub
  6. Importa la llave publica a tu panel de DreamCompute.
