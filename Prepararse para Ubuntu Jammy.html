Generalidades

Este artículo proporciona información para ayudarte a preparar tu sitio para la actualización a Ubuntu 22.04.2 LTS (también conocida como Jammy).

¿Qué esperar?

DreamHost te contactará por correo electrónico antes de la actualización para asegurar que haya varios ingenieros altamente capacitados disponibles en caso de que surjan problemas.

  • La actualización generalmente toma menos de una hora en completarse.
  • Cualquier servicio que dependa de una base de datos (como WordPress o cualquier otra aplicación basada en bases de datos) no estará disponible hasta que se complete la actualización. Esto NO afectará a los servicios de DreamObjects ni a los servicios de correo.

Cambios en la versión del servidor web

Revisa el artículo Ver las versiones del software de tu servidor para obtener instrucciones sobre cómo encontrar las versiones actuales del software instalado en tu servidor.

Solución de problemas

A continuación se proporciona más información sobre cómo solucionar algunos problemas conocidos.

ssh-rsa

El cifrado de clave ssh ssh-rsa ya no está permitido. Deberás regenerar tu llave de ssh si estás utilizando ssh-rsa. Revisa las siguientes guías para obtener instrucciones sobre cómo crear nuevas llaves de emparejamiento:

  • Cómo configurar el inicio de sesión sin contraseña en Mac OS y Linux
  • Configurar el inicio de sesión sin contraseña en PuTTY

Entorno virtual de Python

Si tu sitio web está utilizando un ambiente virtual de Python y notas algún error después de la actualización, reinstala tu entorno virtual y todos los módulos. Esto ayuda a asegurar que todo sea compatible con las nuevas bibliotecas en el servidor.

¿Dónde puedo encontrar soporte?

Si tienes alguna pregunta o inquietud, por favor contacta a soporte.
