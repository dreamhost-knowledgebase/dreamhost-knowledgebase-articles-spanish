Generalidades

Todos los sistemas operativos vienen con un cliente FTP hecho que corre de un intento de comando usando el comando ftp.

Quizá quieras ver los siguiente artículos de Wikipedia para más información:

  • Protocolo de transferencia de archivos
  • Comparación de los clientes de Software de FTP

Soporte de FTP en el buscador Web

Si estás usando el buscador web de Firefox, este incluye un plugin de FTP excelente llamado "FireFTP".

Adicionalmente, todos los buscadores modernos (Internet Explorer, Firefox, Opera, Chrome, y así) incluyen soporte de FTP hecho también que hace un poco de lo mismo. Sin embargo, solo podrás descargar contenido a través del buscador.

Soporte de WebFTP de DreamHost

También puedes conectar tu sitio vía WebFTP (construido en Pydio). Éste no está intencionado para que funcione como un cliente FTP, pero alcanzará cuando no tienes uno a la mano. Las cargas son limitadas a 7MB, pero puedes encontrarlo muy útil para características como mover archivos fácilmente y editar páginas en vivo.
