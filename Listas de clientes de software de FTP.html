Generalidades

Todos los sistemas operativos vienen con un cliente FTP hecho que corre de un intento de comando usando el comando ftp.

Quizá quieras ver los siguiente artículos de Wikipedia para más información:

  • Protocolo de transferencia de archivos
  • Comparación de los clientes de Software de FTP

También hay otros clientes completos de FTP disponibles para varios sistemas operativos que soportan algunas opciones más.

Si has seleccionado un programa para usar, mira el siguiente artículo para ayudarte a ubicar tus credenciales de inicio de sesión:

  • Generalidades y credenciales de FTP

Abajo verás una lista de algunos de estos clientes terceros.

Administrador de archivos DreamHost

También puedes conectarte a tu sitio utilizando el administrador de archivos del panel. Éste no está intencionado para que funcione como un cliente FTP, pero alcanzará cuando no tienes uno a la mano.
