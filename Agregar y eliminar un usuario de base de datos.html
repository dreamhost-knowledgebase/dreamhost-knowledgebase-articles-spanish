Generalidades

Cuando creas una base de datos en el panel de DreamHost, se crea un único usuario que puede usar para iniciar sesión en la base de datos.

Si necesitas dar acceso a otra persona a tu base de datos (como un desarrollador), puedes crear un nuevo usuario en el panel.

Asegúrese de eliminar al usuario cuando ya no requiera acceso a la base de datos.

¡Solo dale acceso a la base de datos a alguien en quien confías!

Agregar un nuevo usuario de base de datos

Los usuarios de la base de datos deben ser únicos en todas las cuentas de DreamHost. Debes elegir un nombre diferente si el nombre del usuario nuevo ya está en uso dentro de DreamHost.

  1. Navega a la página Bases de Datos MySQL.
  2. A la derecha de tu base de datos, da clic al botón Agregar usuario.
  3. En la siguiente página, escoge Crear un nuevo usuario ahora ... del menú desplegable.
  4. Ingresa un nuevo usuario y contraseña.
  5. Da clic a ¡Añadir nuevo usuario ahora!.
Ese nuevo usuario ahora tiene acceso a la base de datos.

Eliminar un usuario de base de datos

  1. Navega a la página Bases de Datos MySQL.
  2. A la derecha de la base de datos, debajo de la columna Acceso de Usuarios da clic al usuario.
    La siguiente página muestra las bases de datos a las que el usuario tiene acceso.
  3. Da clic al botón Eliminar a la derecha de la base de datos para eliminar el acceso a este usuario.
El usuario no va a tener más acceso a la base de datos.
