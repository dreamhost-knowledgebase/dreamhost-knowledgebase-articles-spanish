El módulo de comandos de domain de API administra los dominios y los registros de dominio.

Commandos

domain-list_domains

Vuelca una lista de todos los dominios alojados (sin incluir los registros) en todas las cuentas a las que tienes acceso.

domain-registration_available

Comprueba si un nombre de dominio está disponible para registrarse. Debe ser un dominio .com, .net .org o .info (y no un subdominio).

Comando domain-registration_available
Valores

domain
Resultad

success
available
yes

or

success
available
no
Posibles errores

invalid_domain : El dominio no es válido o es un subdominio
invalid_tld : El dominio de nivel superior no es .com, .net, .org, o .info
