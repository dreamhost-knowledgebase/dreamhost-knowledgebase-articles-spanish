Generalidades

La opción de respaldo del panel es una manera simple de crear una copia de seguridad toda tu cuenta, incluyendo todos los usuarios, bases de datos, y buzones de correo electrónico.

Crear una copia de seguridad de tu cuenta

  1. Visita la página Administrar Cuenta.
  2. Baja hasta la sección Hacer Un Respaldo De Tu Cuenta
  3. Da clic en el botón Crear una Copia de Seguridad para abrir la página de respaldo del panel.
  4. Da clic en el botón ¡Cópiame!.

El proceso de copia de seguridad comienza y tarda un tiempo en completarse en función de la cantidad de contenido que hay para copiar. Se te notificará por correo una vez que esto se complete.

Confirmación de tu copia de seguridad

Dicha notificación vía correo te redirige a la página Copias de Seguridad de Tu Cuenta donde verás los siguientes detalles:

Ver tu Copia de Seguridad

Cuando el respaldo se complete, vas a ver todos tus archivos en una lista en el panel con extensiones como .tar.gz o .sql.gz; vas a ver algo como esto:

Verás enlaces como estos:

Los usuarios de correo electrónico podrán aparecer así:

mail/usuario@example.com.tar.gz

Las bases de datos podrán aparecer así:

mysql/basededatos.sql.gz

Los usuarios podrán aparecer así:

usuario/servidor_usuario.tar.gz

La columna Size muestra el peso de cada uno de los archivos. Todas las copias de seguridad están en la lista como archivos .tar.gz o .sql.gz para que los puedas descargar de manera local.

Cómo descargar tu copia de seguridad via Línea de Comando

Mac y Linux

Puedes descargar estos archivos directamente a tu servidor o computador Mac OSX/Linux usando ya sea curl o wget.

Windows

Si estás corriendo Windows, vas a necesitar instalar curl. Puedes descargar el archivo comprimido zip desde la sección Win64 - Generic.

curl

Aquí encontrarás un ejemplo de cómo descargar un solo archivo usando el comando curl.

  1. Haz clic derecho en el enlace del archivo en tu panel. En este ejemplo es user/servername_username.tar.gz.
  2. Desde el menú emergente, escoge Copy Link Location/Address. Pega esto en un editor simple de texto para guardarlo por ahora. En general es una URL muy larga.
  3. Abre una terminal (si estás usando Mac o Linux) o comando (si estás usando Windows).
  4. Corre el comando curl o wget para descargar el archivo.

curl ejemplo

Corre el comando curl con la siguiente sintaxis:

curl --insecure --output filename.tar.gz "COPIED_LINK_LOCATION"

El archivo filename.tar.gz es uno de los nombres de archivos en tu panel en la página Copia de Seguridad de Tu Cuenta. Asegúrate de NO incluir la primera parte del nombre del archivo antes de la barra inclinada. Si el enlace en el panel es user/servername_username.tar.gz, el nombre del archivo debe tomarse solamente como servidor_usuario.tar.gz.

La parte que dice COPIED_LINK_LOCATIAON es la información que copiaste cuando hiciste clic derecho en el nombre del archivo y que pegaste en tu editor de texto.

Aquí está el ejemplo completo de curl:

Asegúrate de poner la URL completa entre comillas dobles. Si no lo haces, el comando no descargará el archivo.

[server]$ curl --insecure --output servername_username.tar.gz "https://objects-us-east-1.dream.io/accountbackup/1358572/2016-12-06-SLRG8WEQ57K6CQTE/user/server_username.tar.gz?AWSAccessKeyId=2XoecBOZcL8WQXo9DT9E&Signature=9HWxQE28LaYxxGXX6NsvcEELAwE%3D&Expires=1451659279"

wget

Corre el comando wget con la siguiente sintaxis:

wget --no-check-certificate --output-document=filename.tar.gz "COPIED_LINK_LOCATION"

El archivo nombre.tar.gz es uno de los nombres de archivos en tu panel en la página Copia de Seguridad de Tu Cuenta. Asegúrate de NO incluir la primera parte del nombre del archivo antes de la barra inclinada. Si el enlace en el panel es user/servername_username.tar.gz, el nombre del archivo debe tomarse solamente como servidor_usuario.tar.gz.

La parte que dice COPIED_LINK_LOCATION es la información que copiaste cuando hiciste clic derecho en el nombre del archivo y que pegaste en tu editor de texto.

Aquí está el ejemplo completo de wget:

Asegúrate de poner la URL completa entre comillas dobles. Si no lo haces, el comando no descargará el archivo.

[server]$ wget --no-check-certificate --output-document=servername_username.tar.gz "https://objects-us-east-1.dream.io/accountbackup/1358572/2016-12-06-SLRG8WEQ57K6CQTE/user/server_username.tar.gz?AWSAccessKeyId=2XoecBOZcL8WQXo9DT9E&Signature=9HWxQE28LaYxxGXX6NsvcEELAwE%3D&Expires=1451659279"

Hacer clic en el archivo para descargarlo

También puedes hacer clic en cada uno de los archivos directamente para descargarlos uno por uno a tu computador. Desafortunadamente, no hay una opción en el momento para descargar todos los archivos al mismo tiempo.

 

Descomprimir los archivos descargados

Una vez esté descargado en tu computador, se comprime en un archivo comprimido de formato .tar.gz o .sql.gz.

  • Si estás usando Windows, necesitarás un programa para descomprimir como 7-Zip.
  • Si estás usando Mac o Linux, puedes correr el siguiente comando para descomprimir archivos .tar.gz:
    macbook$ tar -zxvf user\@example.com.tar.gz
    Si el archivo tiene una extensión .sql.gz ,corre el siguiente comando:
    macbook$ gunzip databasename.sql.gz

Solución de Problemas

Si crees que la copia de seguridad realizada no se ejecutó por completo y no incluye realmente toda la información de tu cuenta, por favor contacta a soporte para obtener asistencia.
