Generalidades

Si tienes varios dominios y usuarios configurados en tu panel, puedes filtrarlos fácilmente para ubicar a un usuario específico del sitio web. El panel te permite filtrar los usuarios de tu sitio web por lo siguiente:

  • Dominios
  • Servidores
  • Tipo
 

Filtrar usuarios por dominio

Esta opción es útil si tienes muchos dominios y necesitas encontrar rápidamente el usuario asignado a uno de ellos. Para ubicar un usuario asignado a un dominio:

  1. Navega a la página Usuarios & Archivos SFTP.
  2. En el menú de la parte superior, haz clic en el botón Filtros.
    Aparece una ventana emergente.
  3. Selecciona el filtro Dominios.
  4. Selecciona un dominio de la lista.
    Aparece el usuario asignado al dominio.
 

Filtrar usuarios por servidor

Esta opción es útil si tienes más de un tipo de servidor en tu cuenta. Por ejemplo, un servidor compartido y un VPS. Para ubicar un usuario asignado a un servidor:

  1. Navega a la página Usuarios & Archivos SFTP.
  2. En el menú de la parte superior, haz clic en el botón Filtros.
    Aparece una ventana emergente.
  3. Selecciona el filtro Servidores.
  4. Selecciona un servidor de la lista.
    Aparecen los usuarios asignados a este servidor.
 

Filtrar usuarios por el tipo de usuario

Esta opción es útil si tienes varios usuarios creados y necesitas identificar rápidamente de qué tipo son.

Antes de asignar un dominio a un usuario, debes considerar qué tipo de uso puede necesitar. Puede ser un usuario de SFTP o Shell. El siguiente artículo explica más sobre las diferencias entre estos dos tipos de usuarios:

  • Diferencia entre los usuarios de SFTP y Shell

Para ver una lista entre los usuarios SFTP o Shell:

  1. Navega a la página Usuarios & Archivos SFTP.
  2. En el menú de la parte superior, haz clic en el botón Filtros.
    Aparece una ventana emergente.
  3. Selecciona el filtro Tipo.
  4. Selecciona entre SFTP o SSH para mostrar una lista de este tipo de usuario.
