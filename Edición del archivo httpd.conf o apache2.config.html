Generalidades

Para editar el archivo httpd.conf o apache2.config, debes tener un servidor Dedicado con un usuario admin.

El acceso de root/sudo no está disponible en las maquinas VPS.

Ver la versión del sistema operativo

La ubicación del archivo de configuración del servidor es diferente dependiente de cual sistema operativo está corriendo. Para ver la versión, inicia sesión en tu servidor vía SSH y corre el siguiente comando.

[server]$ lsb_release -a
No LSB modules are available.
Distributor ID: Ubuntu
Description:    Ubuntu 18.04.4 LTS
Release:        18.04
Codename:       bionic

This output shows Ubuntu 18 is currently running.

Ubicación del archivo httpd.conf (Solo máquinas Ubuntu 14 (Trusty))

Si tu servidor Dedicado ejecuta Ubuntu 14 (Trusty), puedes encontrar este archivo en la siguiente ubicación:

/dh/apache2/httpd-INSTANCE/etc/httpd.conf

Ubicación del archivo apache2.conf (Solo máquinas Ubuntu 18 (Bionic))

Si tu servidor Dedicado ejecuta Ubuntu 18 (Bionic), puedes encontrar este archivo en la siguiente ubicación:

/etc/apache2-httpd-INSTANCE/apache2.conf
