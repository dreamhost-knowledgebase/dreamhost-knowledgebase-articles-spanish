Generalidades

Hay varias formas de crear una Llave de Emparejamiento para usar tu instancia de DreamCompute. La forma más fácil es hacer que el panel de DreamCompute generar una por ti.

No se puede agregar una nueva Llave de Emparejamiento a una instancia de DreamCompute una vez creada. Si no asignaste una Llave de Emparejamiento durante la creación de la instancia o perdiste el acceso a tu instancia, sigue las instrucciones de los artículos a continuación para recuperar el acceso:

  • Recuperar el acceso a una instancia efímera cuando se pierden las llaves de SSH
  • Recuperar el acceso a una instancia basada en volumen cuando se pierden las llaves de SSH

Crear la Llave de Emparejamiento

  1. Navega a la página DreamCompute.
  2. Haz clic en el botón View Dashboard.
  3. En el menú de la izquierda, da clic en Compute.
  4. Del menú desplegable haz clic en Key Pairs.
  5. En la parte superior derecha haz clic en el botón Create Key Pair.
  6. Escribe un nombre para la llave y haz clic en Create Key Pair.

    Tu navegador comenzará a descargar la clave privada. De forma predeterminada, tendrás el mismo nombre que ingresaste en el campo Key Pair Name y tendrás una extensión de archivo .pem, pero puedes cambiar el nombre del archivo por el que desees.
