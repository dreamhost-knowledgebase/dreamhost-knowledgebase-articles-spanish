Paso 1: Configuración Básica

Crear una cuenta en DreamHost

Si deseas utilizar alguno de los servicios de DreamHost, debes crear una cuenta. Revisa este artículo para obtener instrucciones sobre cómo abrir una cuenta.

Iniciar sesión en el panel

Después de registrarte, recibirás un mensaje para notificarte que tu cuenta está activa. Este correo también contiene las credenciales para que inicies sesión en el panel en:

  • panel.dreamhost.com

Registrar un dominio

Para cargar cualquier página web, el propietario debe primero ser propietario del registro del nombre de dominio. Para registrar un dominio, navega a la página Administrar Registros de Dominio.

En la página de Administrar Registros de Dominio, puedes registrar cualquier dominio que esté disponible.

DreamHost ofrece muchos dominios de nivel superior (TLD) diferentes para que los registres. Puedes ver una lista completa en la página de Administrar Registro de Dominio haciendo clic en el enlace Ver precios completos. Revisa este artículo para obtener más información.

Paso 2: Agregar un plan de alojamiento

Después de comprar un registro, debes agregar un plan de alojamiento para que el dominio pueda mostrarse en línea. Puedes elegir entre varias opciones de alojamiento diferentes.

  • Completamente alojado — tu sitio se encuentra alojado en un servidor y es visible en línea.
  • WordPress — tu sitio se encuentra alojado en un servidor (completamente alojado) pero está ejecutando el programa WordPress.
  • DreamPress — tu sitio tiene completamente alojado corriendo WordPress pero se encuentra en servidor de DreamPress.

Agregar un plan de alojamiento

Para agregar un plan de alojamiento, navega a la página Administrar Cuenta.

En la página Administrar Cuenta, verás una sección titulada Planes Activos.

Si ya has agregado un plan, lo verás aquí. Si no ves un plan, agrega uno dando clic al enlace Agregar un Plan de Alojamiento.

Agregar un plan de alojamiento a un dominio

Revisa el artículo de Agregar un sitio web y alojamiento para obtener instrucciones sobre cómo agregar alojamiento a tu dominio para que esté totalmente alojado.

Apuntar el DNS

Aunque tu sitio esté COMPLETAMENTE ALOJADO en tu panel, eso no significa que esté alojado en DreamHost. Tus registros de DNS controlan dónde está alojado tu dominio, por lo que deben apuntar a tu servidor de DreamHost.

Revisa el artículo ¿Dónde está alojado mi sitio web? para confirmar dónde está alojado actualmente y cómo dirigir los registros DNS a DreamHost.

Paso 3: Subir y construir tu sitio web

Una vez que tu sitio web esté alojado en un servidor web, puedes cargar el contenido de tu sitio o usar WordPress para construirlo.

 

Subir tu sitio

Archivos del sitio web

Si ya tienes tu sitio web construido, puedes cargar todos sus archivos a tu servidor web mediante FTP. Revisa este artículo para obtener instrucciones sobre cómo encontrar tus credenciales FTP y cargar tus archivos.

Bases de datos

Si tu sitio web utiliza una base de datos, primero debes crearla en el panel. Luego, puedes importar tu base de datos existente a través de phpMyAdmin.

Opciones de WordPress

Si aún no tienes un sitio web creado, puedes utilizar WordPress para construirlo. A continuación se describen las opciones de WordPress disponibles en DreamHost.

Agregar WordPress

WordPress es el sistema de gestión de contenido (CMS) más popular del mundo, y DreamHost es uno de los pocos proveedores recomendados por WordPress.org. Si eres nuevo en el alojamiento de sitios web, querrás considerar WordPress para ayudarte a construir tu sitio. En pocos minutos, puedes crear un sitio que incorpore todas las características modernas que incluyen los sitios web actuales, con una capacidad casi ilimitada para agregar nuevas funciones según las necesites.

Para ayudarte a empezar a construir tu sitio WordPress en DreamHost, consulta los siguientes artículos:

  • Instalar WordPress usando el instalador de WordPress de DreamHost
  • Planes de Alojamiento de WordPress en DreamHost
  • Guía de DreamHost para WordPress

Alojamiento DreamPress

DreamPress es el producto de WordPress administrado de DreamHost, que ofrece funciones aún más potentes, como planes de almacenamiento y memoria más grandes, velocidades más rápidas, y almacenamiento en caché mejorado. Revisa el artículo DreamPress para obtener más información.

Opciones de migración de WordPress

Revisa este artículo para obtener una lista de las diferentes opciones disponibles para migrar WordPress.

WordPress Migration Service

Si tienes problemas para migrar tu sitio web desde otro host, o simplemente no tienes tiempo, DreamHost actualmente ofrece un Servicio de Migración de WordPress que se encarga de eso por tí. Puedes encontrar más información en el artículo de los servicios de migración de sitios web DreamHost.

Si tu sitio de WordPress ya está alojado en DreamHost y necesitas migrarlo a otra cuenta de DreamHost, revisa el siguiente artículo.

Paso 4: Agregar un certificado SSL

Agregar un certificado SSL a un dominio

Después de configurar el alojamiento para tu dominio, te recomendamos encarecidamente que también le agregues un certificado SSL al dominio (ya sea un certificado gratuito 'Let's Encrypt' o uno firmado profesionalmente). Agregar un certificado es un proceso simple que no solo asegura tu dominio para realizar transacciones monetarias y de eCommerce, pero también garantiza que tu sitio sea visto como seguro y seguro para visitar.

Para ayudarte a iniciar el proceso de agregar un certificado, por favor dale un vistazo a los siguientes artículos:

  • Agregar un certificado Let's Encrypt gratis
  • Comprar un certificado SSL firmado profesionalmente
  • Agregar un certificado SSL autofirmado gratuito

Paso 5: Configuraciones adicionales

Los siguientes son servicios o productos adicionales que puedes agregar a tu sitio, pero no son obligatorios para que tu sitio funcione.

Crear una dirección de correo

Una vez que tu dominio esté alojado, puede proceder a agregar correos electrónico.

También es posible alojar tu correo fuera de DreamHost al configurar registros MX personalizados para que apunten a tu alojamiento de correo externo.

Agregar estadísticas del sitio

Todas las cuentas de DreamHost tienen la capacidad de agregar estadísticas del sitio a cualquier sitio web totalmente alojado que no sean DreamPress; las estadísticas las puedes habilitar en la página Estadísticas del Sitio Allí, puede realizar un seguimiento de la información, como el número de solicitudes, el tráfico por dominio, el tráfico por archivo, las solicitudes por tipo de navegador/SO/URL, y más.

Para obtener más información sobre cómo agregar estadísticas del sitio, consulta los siguientes artículos:

  • Habilitar estadísticas
  • Hacer estadísticas accesibles con un archivo .htaccess
  • Generalidades sobre estadísticas análogas

Agregar el servicio de seguridad web de DreamShield

DreamHost también ofrece un servicio de seguridad de pago en el panel. Cuando está habilitado, este servicio te permite escanear tu sitio web en busca de malware y te notifica si encuentra algún problema.

Para obtener más información sobre cómo habilitar el servicio de seguridad del sitio web, revisa el artículo Habilitar y administrar DreamShield.

Contactar a soporte

Si hay algo en lo que el equipo de soporte te pueda ayudar con la configuración y carga de tu sitio web, no dudes en enviar un caso de soporte en la página Contactar a Soporte desde tu panel de DreamHost.
