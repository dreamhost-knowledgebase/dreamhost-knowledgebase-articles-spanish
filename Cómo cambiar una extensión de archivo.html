Generalidades

A veces es necesario cambiar la extensión de un archivo para asegurarse de que funcione según lo previsto. Por ejemplo, si creas un archivo de texto en tu computadora, se guarda con la extensión .txt.

Si el archivo necesita ser un archivo phpinfo.php, .htaccess o phprc, la extensión debe cambiarse o eliminarse. Este artículo explica cómo ajustar la extensión del archivo.

Este artículo asume que ya ha creado o subido el archivo a tu sitio web en DreamHost. Ve los siguientes artículos para obtener instrucciones sobre cómo cargar tu archivo o crearlo en el servidor:

  • Descripción general y credenciales de FTP
  • ¿Cómo subo mi sitio a DreamHost?
  • ¿Dónde subo mis archivos una vez conectado a través de FTP?
  • Crear y editar un archivo a través de FTP

Editando la extensión del archivo en Filezilla

Estas instrucciones asumen que ha subido el archivo a DreamHost utilizando los enlaces anteriores. Una vez que el archivo esté en el servidor, inicia sesión con un cliente FTP como Filezilla.

Haz clic derecho en el archivo y elige 'Cambiar nombre'. Edita el archivo para eliminar la extensión .txt. Haga clic en 'Entrar' y el archivo se guardará inmediatamente con la nueva extensión.

Editar la extensión del archivo a través de SSH

Estas instrucciones asumen que ha subido el archivo a DreamHost utilizando los enlaces anteriores. Una vez que el archivo esté en el servidor, inicia sesión con tu usuario de Shell:

  • Cómo crear un usuario con acceso Shell (SSH)
  • Generalidades de SSH
  • Crear y editar un archivo via SSH

Una vez que hayas iniciado sesión en tu servidor a través de SSH, navega hasta la ubicación del archivo. A continuación, puedes cambiar el nombre del archivo para eliminar la extensión. Por ejemplo, el siguiente comando cambia un archivo titulado phpinfo.php.txt a phpinfo.php:

[server]$ mv phpinfo.php.txt phpinfo.php

Este ejemplo muestra cómo eliminar la extensión y también cambiarle el nombre:

[server]$ mv php.ini phprc

Luego puedes ejecutar ls -la para confirmar que se le cambió el nombre:

[server]$ ls -la
