Generalidades

Este artículo explica cómo instalar un nuevo sitio WordPress en tu panel usando el instalador de WordPress de DreamHost.

Revisa el siguiente artículo si en vez necesitas migrar tu sitio existente de WordPress a DreamHost.

  • Migrar un sitio WordPress desde un alojamiento diferente

¿Qué es una instalación de WordPress de DreamHost?

El instalador de WordPress de DreamHost te permite configurar rápidamente nuevas instalaciones de WordPress en el panel DreamHost con una interacción mínima. El panel DreamHost proporciona varias funciones (como el cambio de nombre del sitio web, actualizaciones automáticas de la versión de WordPress y actualizaciones sencillas de DreamPress) para instalaciones de WordPress de DreamHost.

Si deseas usar las funciones del instalador de WordPress de DreamHost con una instalación existente de WordPress, por favor contacta a soporte para hacer que conviertan tu instalación.

 
 

Paso 1 — Configurar el alojamiento

Para usar el instalador de WordPress de DreamHost, tu dominio primero debe estar Completamente Alojado. Esto significa que no puede estar Redirigido, Reflejado, Parqueado, o configurado como Solo DNS.

 

Paso 2 — Revisar el directorio del sitio web antes de instalar

Antes de instalar WordPress, verifica qué archivos ya están en el directorio del sitio. Puede que existan las siguientes tres configuraciones:

  • WordPress ya se instaló usando el instalador de WordPress de DreamHost
  • WordPress ya se instaló manualmente
  • Hay archivos que no son de WordPress en el directorio

Si WordPress ya fue instalado usando el instalador de WordPress de DreamHost

Si instalaste WordPress previamente usando el instalador de WordPress de DreamHost, la instalación parará mostrando el siguiente mensaje:

'WordPress ya existe en este dominio y solo se puede instalar en un subdirectorio. Especifica una ruta.'

Debes ingresar:

  • Eliminar la instalación de WordPress de DreamHost en el panel, o
  • Instalar WordPress en un subdirectorio Install de ese sitio web

Si instalaste WordPress previamente de forma manual

Si subiste un sitio de WordPress de forma manual a tu servidor, no se mostrará en el panel. Sin embargo, dado que los archivos de WordPress existen en el directorio de tu sitio web, el instalador de WordPress de DreamHost agregará una marca de tiempo a los archivos antiguos. Por ejemplo:

  • index.php.1555975213
  • wp-login.php.1555975213
  • wp-settings.php.1555975213

Los nuevos archivos del instalador de WordPress de DreamHost se usarán lugar de los archivos antiguos. A continuación, puedes eliminar los archivos antiguos siguiendo las instrucciones del siguiente artículo:

  • Eliminar archivos duplicados en la instalación de WordPress de DreamHost

Si hay archivos en tu directorio que no son de WordPress

Si hay archivos o directorios que no sean de WordPress, la instalación no los tocará. Los nuevos archivos de WordPress se instalarán junto con tus archivos existentes.

 

Paso 3 — Instalar WordPress

  1. Navega a la página Instalar WordPress. page.
  2. Selecciona tu dominio del menú desplegable.

    Instalar en un subdirectorio

    Si necesitas instalar WordPress en un subdirectorio como example.com/blog.

    1. Da clic en Avanzado.
    2. Debajo de Especificar ruta, ingresa el subdirectorio en el que deseas instalar.
  3. Se recomienda que dejes el menú desplegable Seleccionar base de datos como Crear Base de Datos Automáticamente.

    Si no deseas que se cree una nueva base de datos, puedes usar una base de datos existente que ya hayas creado en su panel. Haz clic en Avanzado y selecciona tu base de datos del menú desplegable Seleccionar Base de Datos.

    Si eliges una base de datos existente del menú desplegable que contiene una instalación anterior de WordPress, las tablas de la base de datos actual permanecerán y las nuevas tablas de WordPress se instalarán junto con las tablas existentes.

  4. Da clic en Instalar Wordpress.
  5. La siguiente página te permite seleccionar el propósito de un sitio para una instalación guiada de WordPress. Elegir una opción sugerirá plugins específicos para la instalación. Selecciona una opción (o Ninguna) y haz clic en Siguiente.
  6. La siguiente página muestra los plugins sugeridos específicos para el propósito del sitio que elegiste. Si no deseas que se instale un plugin, desmarca la casilla correspondiente.
  7. Da clic en Instalar.

Dentro de 15 minutos, se te enviará un correo a la dirección principal que figura en tu cuenta de DreamHost. Este correo detalla los pasos a seguir para completar la instalación.

Inconvenientes de instalación

Si la instalación se completa pero no recibes el correo con las instrucciones sobre cómo acceder a tu nueva instalación de WordPress, siempre puedes consultar tus "Mensajes Recientes" en la página Contactar a Soporte, y así obtener una copia archivada de ese correo. 

Si la instalación de WordPress de DreamHost no se completa en 20 minutos o falla con un error, por favor contacta a Soporte.

El enlace de instalación no funciona

Un error 404 o "Site Not Found" puede aparecer en las nuevas configuraciones de WordPress cuando intentas ver el sitio después de cambiarlo desde otra opción de alojamiento (como espejo o redirección). En estos casos, el sitio aún se resuelve con los registros A antiguos de reflejo o redirección, en lugar de los nuevos registros A 'Completamente Alojado'. Afortunadamente, esto se resolverá una vez que se complete la propagación de DNS, suponiendo que el sitio esté usando los nameservers de DreamHost. Consulta el siguiente artículo para obtener más información sobre la propagación de DNS:

  • Generalidades sobre propagación de DNS
