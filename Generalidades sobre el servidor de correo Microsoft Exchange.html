Generalidades

Microsoft Exchange server es un servidor de correo creado por Microsoft. Es posible alojar tu sitio web en DreamHost pero seguir utilizando Microsoft para el correo. Esto requiere que configures algunos registros DNS. Este artículo explica lo que debe ajustarse.

Encontrar tu configuración de DNS

Lo siguiente asume que tus Nameservers son controlados en DreamHost. Si no lo están, debes hacer estos cambios en la compañía donde apuntan Nameservers.

Cuando creas tu cuenta de Exchange, se te proporcionan registros DNS específicos. Estos se pueden agregar en tu panel DreamHost. El siguiente enlace es una descripción general de los registros que Microsoft puede brindarle:

  • Registros DNS externos para Office 365

Una vez que tengas la lista de registros DNS de Microsoft, ingrésalos en tu panel DreamHost. El siguiente artículo te explica cómo crear diferentes tipos de registros DNS personalizados

  • Agregar registros de DNS personalizados

Revisa el siguiente artículo para obtener información sobre cómo cambiar tus registros MX:

  • Configurar registros MX personalizados

Propagación de DNS

Una vez que hayas agregado tus registros DNS personalizados, deben actualizarse en línea. Esto puede tardar varias horas. Mira el siguiente artículo propagación DNS para más detalles.

  • whatsmydns.net — Un sitio popular que mira los registros DNS al rededor del mundo.
  • Ver tus registros DNS de DreamHost — Usar el verificador de propagación de DNS en el panel.
