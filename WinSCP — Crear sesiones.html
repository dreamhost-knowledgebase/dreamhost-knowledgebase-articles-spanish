Generalidades

Las sesiones son configuraciones de conexión guardadas. Crear una sesión te permite conectarte a un servidor muy rápidamente en lugar de tener que ingresar todos tus datos cada vez que deseas conectarte.

Crear una sesión

  1. Haz clic en la pestaña Session en el menú superior.
  2. Elige New Session del menú desplegable.
    Se muestra el siguiente cuadro de diálogo:
  3. Al igual que lo hiciste cuando inicialmente hiciste tu primera conexión, ingresa las credenciales de conexión para esta nueva sesión.
  4. Haz clic en el botón Save para guardar estas credenciales en una nueva sesión.
    A continuación, se muestra el siguiente cuadro de diálogo:
    • Esta ventana permite nombrar la sesión que fue programada.
    • El nombre predeterminado probablemente sea perfecto para que puedas usarlo (es una combinación del usuario@nombredehost, que es una buena descripción para la sesión); pero puedes cambiarle el nombre como quieras.
    • Es mejor usar lo que tenga más sentido para ti, ya que serás tú quien lo use.

    Si deseas guardar la contraseña para esta sesión, asegúrate de marcar la casilla de verificación junto a 'Save password'. Si esto no está marcado, deberás ingresar la contraseña la próxima vez que intentes conectarte con esta sesión.

  5. Haz clic en el botón OK para guardar.

Después de guardar la sesión, verás el nombre de la sesión guardada en el lado izquierdo del cuadro de diálogo de WinSCP.

Conectarte usando una sesión guardada

En el lado izquierdo de la caja de diálogo de WinSCP verás tu sesión guardada. Haz doble clic en ella para conectarte.
