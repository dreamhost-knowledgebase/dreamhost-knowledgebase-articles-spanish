Generalidades

Hay dos tipos de planes Remixer para elegir — Remixer Standard y Remixer All Access:

Ambos planes 'independientes' asumen que aún no has comprado un plan de Shared Hosting.

Si tuvieras un plan de Shared Hosting, se enumeraría en la página Administrar Cuenta.

Remixer Standard

El plan Remixer Standard incluye soporte para múltiples dominios dentro de Remixer. Esto significa que cualquier dominio que desees alojar debe estar alojado en la plataforma Remixer. No hay acceso a ningún otro tipo de plan de alojamiento.

Los planes anuales y plurianuales incluyen un registro de dominio gratuito de un año que se puede utilizar para varios dominios de nivel superior:

Remixer All Access

El plan Remixer All Access incluye todos los beneficios de Remixer Standard y un plan de Shared Hosting (incluye cuentas de correo y acceso a un servidor web compartido). Esto significa que también se incluye en el descuento de organizaciones sin ánimo de lucro.

¿Remixer está incluido en mi plan de DreamHost?

Si ya has comprado un plan de Shared Hosting, se incluye Remixer 'All Access' sin cargos extra.

Remixer no está incluido en planes de Alojamiento VPS, DreamPress, ni Dedicados. Para usar Remixer con tu plan de Alojamiento VPS o Dedicado, puedes comprar un Remixer Standard, Remixer All Access o un plan de Shared Hosting. Remixer no funciona con DreamPress y no se incluye con los registros de dominio independientes.

Cómo confirmar qué plan Remixer has comprado

Puedes ver en qué plan de Remixer estás en tu panel DreamHost. Navega a la página Administrar Cuenta. Aquí verás una línea de ítem de Remixer.
