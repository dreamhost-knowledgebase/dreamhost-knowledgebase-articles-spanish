Generalidades

A continuación, se explica cómo mover manualmente tu dominio y sus datos entre dos cuentas DreamHost. Este proceso implica los siguientes pasos:

  • Paso 1 — Respaldar la información de tu dominio
  • Paso 2 — Mover las Listas de Discusión/Anuncios
  • Paso 3 — Mover el alojamiento del dominio
  • Paso 4 — Subir la información del sitio web

Nuevos propietarios de cuentas

Si el dominio que deseas mover a tu cuenta está alojado actualmente en otra cuenta de DreamHost, debes crear una nueva cuenta utilizando un subdominio dreamhosters.com.

Luego, puedes continuar con los pasos a continuación para mover el dominio y sus datos a tu nueva cuenta.

¿Existen otras opciones?

Las siguientes opciones alternativas están disponibles para mover los datos de tu dominio entre cuentas.

Servicio de migración — DreamHost ofrece un servicio que se encarga de esto por ti. Puedes encontrar más información en el artículo Servicios de migración de sitios web.

Opciones de migración de WordPress — Revisa el artículo Generalidades de las migraciones de WordPress para obtener una lista de diferentes opciones de WordPress.

 

Restricciones

Las siguientes restricciones se aplican a los movimientos internos de dominios entre cuentas:

  • Promociones activas — La cuenta Receptora no puede utilizar una promoción activa. Esto significa que si la cuenta aún se encuentra dentro del primer año de alojamiento y el propietario de la cuenta se registró utilizando un descuento promocional, no es posible agregar el dominio.
  • Referidos — La cuenta Receptora no puede tener un referido (esto también aplica únicamente si la cuenta está dentro del año inicial de servicio).

Contacta a soporte si tienes cualquier preguntas sobre las restricciones mencionadas anteriormente.

Mover la información de un dominio

Los siguientes pasos mueven los datos de tu dominio a una nueva cuenta DreamHost.

 
 

Respaldar la información de tu dominio

Este paso hace una copia de seguridad de los datos de tu dominio en tu computadora de hogar.

NO elimines el dominio del panel hasta que hayas respaldado todos los datos, ya que esto puede eliminar permanentemente tus datos.

Los siguientes artículos brindan información sobre cómo respaldar los archivos, la base de datos, y el correo de tu sitio web.

Archivos del sitio web — Usa un cliente FTP para iniciar sesión en tu servidor de DreamHost y descarga los archivos de tu sitio web.

Bases de datos — Revisa este artículo para conocer varias opciones de respaldo de bases de datos.

Correo — Revisa este artículo para obtener instrucciones sobre cómo hacer una copia de seguridad de tu correo en tu computador usando un cliente de correo con una conexión POP.

Respaldo de toda la cuenta

También puedes correr una copia de seguridad completa de la cuenta en el panel, que incluye archivos del sitio web, bases de datos, y correo.

 

Mover las Listas de Discusión/Anuncios

Las listas de Discusión y de Anuncios requieren atención especial para moverlas correctamente. Comunícate con soporte en la página Contactar a Soporte antes de comenzar el proceso para que se puedan hacer los arreglos necesarios para ayudarte a moverlas. Revisa este artículo para obtener más información.

 

Mover el alojamiento del dominio

Ahora puedes mover tus información a la nueva cuenta.

Habrá tiempos de inactividad inevitables a medida que migras el dominio entre cuentas.

  1. Sigue las instrucciones del artículo Eliminar un dominio para eliminar tu dominio del panel.
  2. Inicie sesión en la segunda cuenta de DreamHost en panel.dreamhost.com.
  3. Sigue las instrucciones del artículo Agregar un dominio para agregar este dominio a la nueva cuenta.
    Una vez que el dominio esté completamente alojado en la nueva cuenta, puedes proceder a cargar todo el contenido que hayas respaldado.
 

Subir la información del sitio web

Este paso carga los datos que respaldaste previamente en la nueva cuenta DreamHost.

Archivos del sitio web — Usa un cliente de FTP para iniciar sesión en tu servidor de DreamHost y carga los archivos de tu sitio web. Revisa este artículo para obtener instrucciones sobre dónde cargar los archivos en el servidor.

Bases de datos — Revisa este artículo para obtener instrucciones sobre cómo importar tu base de datos.

Correo — Revisa este artículo para obtener instrucciones sobre cómo importar el correo a tu nueva cuenta de DreamHost.

Una vez que los datos se hayan cargado a la nueva cuenta y el DNS se haya propagado, el sitio entonces se resuelve desde la nueva cuenta de DreamHost.

Transferir el registro de dominio

Mover un registro de dominio entre cuentas es lo mismo que transferirlo a otro registrador. Para obtener instrucciones sobre cómo transferir el registro únicamente, revisa los siguientes artículos:

  • Transferir el registro de un dominio a DreamHost
  • Transferir el registro de un dominio fuera de DreamHost

No es necesario mover el registro del dominio a la nueva cuenta para habilitar el alojamiento del sitio web. Sin embargo, tener el alojamiento y el registro en la misma cuenta hace que sea más sencillo realizar un seguimiento de ambos.

Además, algunos dominios internacionales no pueden ser transferidos de una cuenta a otra por el cliente. En estos casos, el panel del cliente receptor solicita que contacte a soporte para obtener más ayuda.
