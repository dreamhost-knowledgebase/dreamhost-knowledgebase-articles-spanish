mail-add_filter

Agrega un nuevo filtro de correo a una dirección de correo que tengas con DreamHost.

Comando mail-add_filter
Valores

address : El correo completo en el cual quieres agregar el filtro.
filter_on : Asunto, de, para, cc, cuerpo, reply-to, encabezados.
filter : Que filtrar (sensible a mayúsculas y minúsculas).
action : Move, forward, delete, add_subject,forward_shell, and, or
action_value : El parámetro para la acción (Nota: Opcional si la acción es delete, and, or).
contains : yes o no (opcional, por deyecto es yes).
stop : yes o no (opcional, por defecto es yes. Nota: Debe ser yes si la acción es delete).
rank : El rango del el filtro, indexes desde 0. Los menores se ejecutan primero (opcional, por defecto es el número de filtros para la dirección) Para crear un filtro complejo de varias partes, simplemente agrega un guión bajo a cada uno de los anteriores, seguido de un número (comenzando en 1) para el cual el paso es.

mail-remove_filter

Elimina un filtro de correo de una dirección de correo que tengas con DreamHost.

Comando mail-remove_filter
Values

address : El correo completo del cual quieres eliminar el filtro.
filter_on : subject, from, to, cc, body, reply-to, headers.
filter: Que filtrar (sensible a mayúsculas y minúsculas).
action : move,forward,delete,add_subject,forward_shell.
action_value : El parámetro para la acción (Note: Opcional si la acción es delete).
contains : yes o no.
stop : yes o no.
rank : El rango del filtro. El menor se ejecuta primero.
