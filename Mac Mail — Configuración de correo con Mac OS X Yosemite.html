Generalidades

Este artículo te guía a través de los pasos para configurar Apple Mail 8.0 en Mac OS X 10.10 (Yosemite).

Una vez que completes la configuración, tendrás una conexión IMAP entrante con SSL (sin advertencias) y una conexión SMTP con SSL (sin advertencias).

 

Paso 1 — Verificar que el usuario de correo existe en tu panel

Asegúrate de que tu correo ya esté configurado en el panel DreamHost yendo a Administrar Correo.

Verás tu usuario de correo en la página:

Paso 2 — Encontrar tu servidor de correo

En el pasado, era posible usar mail.example.com para conectarte (donde example.com es tu nombre de dominio real). Debido a cómo los clientes de correo modernos manejan la seguridad, esto NO se recomienda ya que puede generar errores de conexión. Asegúrate de utilizar solo el nombre de tu servidor de correo al conectarte.

Será uno de los siguientes, dependiendo de si estás utilizando una conexión IMAP o POP:

Hostname IMAP: imap.dreamhost.com

Hostname POP: pop.dreamhost.com

Cuando configuras la primera dirección de correo en un dominio (que está configurado para ser alojado con DreamHost), el sistema DreamHost crea automáticamente los nombres de host y los registros MX necesarios para procesar el correo para ese dominio. Ten en cuenta también que si esta es tu primera dirección en un dominio, estos correos pueden tardar hasta 6 horas en funcionar.

Paso 3 — Configurar una nueva cuenta en Mail

Para configurar una nueva cuenta en Mail, sigue estas instrucciones:

  1. Abre la aplicación Apple Mail haciendo clic en Mail en la barra de menú superior.
  2. Selecciona la opción Add Account.
    La página 'Choose a mail account to add...' se abre:
  3. Selecciona la opción Add Other Mail Account… y haz clic en el botón Continue.
    Este ejemplo usa la dirección de correo electrónico "user@dreamhost.com":
  4. Ingresa lo siguiente:
    • Full Name: Así es como te gustaría que se muestre tu nombre al enviar un correo.
    • Email Address: Este campo debe contener la dirección de correo completa.
    • Password: Esta es la contraseña que configuraste para el correo al crearla en tu panel.
  5. Haz clic en el botón Create cuando hayas terminado.
    Deberías recibir una notificación de que la cuenta debe configurarse manualmente:
  6. Haz clic en el botón Next para continuar a configurar manualmente tu dirección de correo.
    Luego de hacer click en Next para continuar, la página Incoming Mail Server Info se abre:

    En este ejemplo, IMAP se utiliza como conexión, ya que este es el protocolo preferido. Pero, si lo deseas, también puedes usar POP para conectarte.

  7. Ingresa la siguiente configuración en la página 'Incoming Mail Server Info':
    • Account Type: IMAP
    • Mail Server: Usa tu nombre de servidor de correo. Esto es imap.dreamhost.com o pop.dreamhost.com.
    • User Name: Tu dirección de correo completa.
    • Password: Esta es la contraseña que configuraste para el correo al crearla en tu panel.
  8. Haz clic en el botón Next para continuar.
  9. Si te pide aceptar el certificado, haz clic en el botón Connect para aceptar.
    Una vez conectado, la página Outgoing Mail Server Info se abre:
    Mac Mail — Ajustar las configuraciones de salida y entrada
  10. Ingresa la siguiente configuración en la página Outgoing Mail Server Info:
    • SMTP Server: Este es smtp.dreamhost.com.
    • User Name: Tu dirección de correo completa
    • Password: Esta es la contraseña que configuraste para el correo al crearla en tu panel.
  11. Haz clic en el botón Create para crear la dirección de correo.

Tu nueva dirección de correo ya está configurada y lista para usar. Puedes enviar un mensaje de prueba para confirmar que funciona correctamente.
