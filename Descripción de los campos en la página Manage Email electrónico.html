Generalidades

Visita la página Manage Emails en tu panel:

Create New Email Address: Se debe hacer clic en este botón para crear una nueva dirección de correo electrónico como dice.

Bulk-Edit Forwarding-Only Addresses: Este botón es para la creación de una dirección de correo electrónico de "solo reenvío" (forward-only). Esto te permite agregar múltiples reenvíos muy rápidamente.

Show these domains: Show these domains: Esta sección enumera todos tus dominios alojados con DreamHost. Puedes ver los usuarios de correo electrónico en ese dominio resaltando el nombre del dominio y luego haciendo clic en el botón Show Selected Domains. (También puedes seleccionar varios dominios para ver manteniendo presionado el botón "Control" y haciendo clic en los dominios que deseas ver).

Las siguientes son las tres columnas que enumeran información para cada dirección de correo electrónico:

Mailbox: Los detalles enumerados en esta columna incluyen lo siguiente:

  • El nombre del buzón.
  • Enlace al interfaz de mailbox manager para ese usuario.
  • Detalles del tamaño del buzón para esa dirección junto con un porcentaje de cuán lleno está actualmente el buzón.

El panel se actualiza cada 24 horas. Si recientemente eliminó una gran cantidad de correo electrónico, notarás que el tamaño no ha cambiado; El nuevo tamaño del buzón puede tardar hasta 24 horas en mostrarse correctamente.

Also forwards to: Este campo muestra los alias de reenvío que configuraste para la dirección de correo electrónico que estás viendo.

En el pasado, era posible crear un correo electrónico totalmente alojado que también se reenviaba a otra bandeja de entrada. Esta característica ya no es compatible, y las nuevas direcciones de correo electrónico creadas no tendrán esta opción. Las direcciones de correo electrónico existentes que usan esta función no se verán afectadas hasta que se editen, momento en el que deberás seleccionar entre reenviar solo o totalmente alojado.

Si no hay alias de reenvío configurados para la dirección de correo electrónico, mostrará este mensaje:

“There are no additional forwards; messages are only delivered to this mailbox.”

Actions: Esta columna consta de tres botones para cada dirección de correo electrónico: Edit, Disable, y Delete.

  • Edit: Este botón Editar te lleva a otra página donde puede realizar cambios en tu dirección de correo electrónico, como aumentar el tamaño del buzón.
  • Disable: Este botón deshabilita la dirección de correo electrónico dentro de una hora. Al hacer clic en este botón, cambia lo que ves para ese usuario en la página Manage Emails marcando una línea con el nombre y el tamaño del buzón del usuario, y configurando un nuevo mensaje en la columna Also forwards to. El nuevo mensaje leerá:  
    Deleted with bounce.

    (Lo que esto significa es que cualquier correo electrónico enviado a este usuario deshabilitado será eliminado y rechazado de nuevo al remitente con un error de devolución).

  • Delete: Al hacer clic en este botón, se elimina el usuario, pero aparece una ventana emergente que te solicita que verifique esta solicitud primero con este mensaje: 
    REALLY DELETE user@example.com AND ALL EMAIL IN ITS MAILBOX PERMANENTLY?!!

    Al hacer clic en "OK" se elimina el usuario y el buzón de ese usuario. Esto requiere unos minutos, pero una vez hecho esto, ya no verás a ese usuario en la lista de la página Manage Emails para ese dominio. Esto es diferente a cuando deshabilita al usuario, que aún mantendrá el nombre de usuario en la página Manage Emails en caso de que desees configurarlo como una dirección de correo electrónico alojada nuevamente en el futuro.

Para encontrar rápidamente tu configuración de correo electrónico en el panel, abre la página Manage Email y busca el enlace en la esquina superior derecha. Haz clic en el texto para abrir las instrucciones de acceso rápido para tu dirección de correo electrónico:

Entrante
  • imap.dreamhost.com
  • pop.dreamhost.com
Saliente
  • smtp.dreamhost.com
