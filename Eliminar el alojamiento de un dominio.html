Generalidades

Cuando agregas un dominio a tu plan de alojamiento, el sitio web está disponible en línea para que cualquiera pueda verlo y se considera que está "completamente alojado".

Si ya no deseas mantener el sitio web en línea, puedes eliminar el alojamiento del mismo. Los pasos que debes seguir dependen de si deseas eliminar sólo un dominio o todos los dominios.

Elimina el alojamiento de un dominio si es el único dominio en tu cuenta

Esto supone que tienes un único plan de alojamiento con un único sitio web alojado. Si ya no deseas que se aloje el sitio y no tiene intención de agregar otro sitio, puedes continuar con las siguientes instrucciones para cerrar tu plan de alojamiento. Elige una de las siguientes opciones según el plan de alojamiento que estés utilizando.

  • Cancelar solo tu plan de VPS o Dedicado

Elimina el alojamiento de un dominio si tienes varios dominios alojados

Esto supone que tienes un único plan de alojamiento, pero varios sitios web están alojados en el plan. Como tienes otros sitios web en el plan, solo debes eliminar el alojamiento para un solo dominio.

Puedes hacerlo editando el dominio y cambiando su alojamiento a cualquiera de las siguientes opciones:

  • Redireccionar
  • Reflejar
  • Parquear
  • Solo DNS

Elimina el alojamiento si tienes varios tipos de planes de alojamiento

Si tienes varios planes de alojamiento, como Compartido, VPS o Dedicado, deberás determinar si solo necesitas eliminar el alojamiento para un solo dominio o cerrar todo el plan de alojamiento en el que se encuentra ese dominio. Revisa los siguientes artículos para más detalles:

  • Generalidades sobre la cancelación de una cuenta
