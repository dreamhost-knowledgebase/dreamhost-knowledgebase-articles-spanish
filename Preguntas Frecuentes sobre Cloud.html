General

¿Qué servicios en la nube ofrece DreamHost?

DreamHost ofrece dos productos en la nube:

  • DreamObjects — Opción de almacenamiento en la nube
  • DreamCompute — Opción de servidor en la nube

¿DreamHost ofrece un servicio CDN?

No, sin embargo, DreamHost está asociado a Cloudflare. Al habilitar sus servicios en tu panel de DreamHost, puedes aprovechar el CDN de Cloudflare.

DreamObjects

¿Puede DreamObjects hacer una copia de seguridad de mi sitio web?

DreamObjects como tal, es solo una opción de almacenamiento. Debes configurar tu sitio para hacer una copia de seguridad de los datos en DreamObjects. Por ejemplo, podrías usar un complemento de WordPress para esto:

  • UpdraftPlus
  • Usar DreamObjects con DreamPress

DreamCompute

¿Qué sistemas operativos puedo instalar en DreamCompute?

DreamHost ofrece varias distribuciones de Linux para que instales la que quieras. Estos incluyen, CentOS, Debian, Fedora y Ubuntu. Revisa el siguiente artículo para obtener una lista actual:

  • Cómo iniciar una instancia con el panel de DreamCompute

¿DreamCompute incluye el servicio de correo electrónico?

El servicio de correo electrónico está incluido sin costo adicional en cualquier plan de Alojamiento Compartido (Shared), VPS, DreamPress, o Servidor Dedicado. Dado que los planes solo de DreamCompute / DreamObjects no incluyen servicio de correo electrónico, puedes comprar un plan de complemento de correo electrónico a bajo costo.

¿Puedo instalar Windows en DreamCompute?

Solo las distribuciones de Linux están disponibles al crear una instancia de DreamCompute.

¿Tengo acceso root / sudo en DreamCompute?

Sí. DreamCompute es una opción totalmente no administrada para que puedas personalizarla a tu gusto.
