General

¿Qué servicios en la nube ofrece DreamHost?

DreamHost ofrece dos productos en la nube:

  • DreamObjects — Opción de almacenamiento en la nube
  • DreamCompute — Opción de servidor en la nube

DreamObjects

¿Puede DreamObjects hacer una copia de seguridad de mi sitio web?

No. DreamObjects en sí solo almacena datos. Sin embargo, puedes configurar tu sitio para hacer copias de seguridad de datos en DreamObjects. Por ejemplo, podrías usar un plugin de WordPress como UpdraftPlus para esto.

DreamCompute

¿Qué sistemas operativos puedo instalar en DreamCompute?

DreamHost ofrece varias distribuciones de Linux para que instales. Revisa el artículo Cómo lanzar una instancia usando el panel de DreamCompute para obtener una lista actualizada.

¿DreamCompute incluye el servicio de correo?

Los planes exclusivamente de DreamCompute no incluyen servicio de correo, pero puedes comprar un plan adicional de correo para alojar tu correo en DreamHost.

El alojamiento de correo está incluido sin costo en los planes de alojamiento Shared, VPS, DreamPress, y Dedicados solamente.

¿Puedo instalar Windows en DreamCompute?

Solo las distribuciones de Linux están disponibles al crear una instancia de DreamCompute.

¿Tengo acceso root/sudo en DreamCompute?

Sí. DreamCompute es una opción totalmente no administrada para que puedas personalizarla a tu gusto.

¿Puedo alojar sitios web en DreamCompute?

Sí. Revisa el siguiente artículo Agregar un sitio web a DreamCompute para más detalles.
