Generalidades

Los siguientes pasos de migración aplican a todos los tipos de sitios WordPress.

A continuación se describen varias opciones a tener en cuenta al migrar tu sitio de WordPress desde otro host a DreamPress.

Opción 1 — Plugin de Migración Automatizada de DreamHostn

Las siguientes instrucciones no se pueden utilizar al migrar a un nombre de dominio temporal como .dreamhosters.com o .dream.press. Revisa la sección de la Opción 4 a continuación para obtener más información.

Si estás migrando un sitio de WordPress de otra empresa de alojamiento a un plan de DreamPress, DreamHost se ha asociado con los creadores de BlogVault para proporcionar el plugin de migración automatizada DreamHost, un plugin gratuito de WordPress que te permite migrar rápidamente tu sitio de WordPress a DreamPress, en sí mismo.

Revisa el siguiente artículo para más información:

  • Uso del plugin de WordPress de Migración Automatizada de DreamHost

Opción 2 — Servicios Profesionales de migración

Si estás migrando un sitio de WordPress de otra empresa de alojamiento a un plan de DreamPress, el equipo de Servicios Profesionales de DreamHost transferirá tu sitio de forma gratuita. Si deseas que DreamHost avance con la migración de tu sitio a DreamPress, comunícate con el soporte con el nombre de dominio que deseas migrar:

  • Si ya eres cliente de DreamHost, haz clic aquí para ponerte en contacto con el servicio de asistencia.
  • Si deseas migrar tu sitio a DreamHost, pero aún no te has registrado, haz clic aquí para comunicarte con el equipo de soporte de DreamHost.

Un miembro del equipo de Servicios Profesionales de DreamHost responderá para ayudarte a comenzar.

Opcipon 3 — Migración manual

Si prefieres migrar tu sitio de WordPress manualmente a DreamPress, puedes hacerlo. Mira el siguiente artículo para más instrucciones:

  • Copias de seguridad de MySQL

Opción 4 — Migrar a un dominio dream.press

Esta opción es solo para migrar a un dominio temporal dream.press. Revisa los siguientes enlaces para obtener más información:

  • Migrar un sitio de WordPress a un dominio temporal
  • Migrating to a DreamPress plan with the DreamHost Automated Migration Plugin

Solución de problemas de la nueva importación

Si tu sitio no se muestra como debería, es posible que debas limpiar el caché de los objetos y/o el caché de DreamPress. Revisa los siguientes artículos para más detalles:

  • Limpiar el caché de DreamPress
  • Limpiar el caché de los objetos usando 'wp cache flush'

A continuación hay ejemplos sobre cómo limpiar ambos vía SSH:

Limpiar el caché DreamPress para todos tus archivos de tema

[server]$ curl -X PURGE -D - https://example.com/wp-content/themes/*

(Solamente para DreamPress Pro) Limpiar el caché de los objetos usando WordPress wp-cli

[server]$ wp cache flush
