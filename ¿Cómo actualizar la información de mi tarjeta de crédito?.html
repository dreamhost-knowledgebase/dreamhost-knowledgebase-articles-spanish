Generalidades

Este artículo proporciona información general sobre cómo actualizar la información de pago de tu tarjeta de crédito.

¿Cuándo es necesario?

Si utiliza el sistema de AutoPago, es posible que necesite actualizar los datos de tu tarjeta de crédito con el tiempo para reemplazar la información obsoleta o vencida.

Actualizar tu tarjeta de crédito

Puedes hacer este cambio de forma segura desde tu panel de DreamHost. Revisa este artículo para obtener instrucciones sobre cómo actualizar la información de tu tarjeta de crédito.

Dominios de Renovación Automática

Cuando actualizas una tarjeta que se utilizó anteriormente para pagar un dominio configurado en Renovación Automática, el dominio se configura en Preguntar. Luego, debes regresar y configurar el dominio nuevamente en Renovación Automática después de actualizar tu tarjeta de crédito.

Solución de problemas de tarjetas expiradas y errores de AutoPago

Si recibes un error debido a una tarjeta vencida, utiliza las instrucciones anteriores para agregar tu nueva tarjeta al panel como la nueva tarjeta de AutoPago registrada.

Mientras los datos de tu tarjeta de crédito guardados en AutoPago sean precisos, el sistema de facturación intentará volver a cargar tu tarjeta dentro de una semana hasta que se resuelva tu saldo adeudado.
