Generalidades

Una imagen es un archivo único que contiene un sistema operativo de arranque. Las imágenes se utilizan para crear instancias dentro de DreamCompute. Para obtener información sobre la creación de archivos de imagen, mira la guía de Imágenes de máquinas virtuales de OpenStack. Luego puedes usar el panel de DreamCompute para cargar y administrar imágenes en tu proyecto.

También puedes usar el cliente de comando de línea de OpenStack para subir y administrar imágenes. Para ejemplos, ve estos artículos:

  • Hoja de trucos de la interfaz de línea de comandos de DreamCompute
  • Lanzar una instancia usando un archivo de imagen ISO con la CLI de OpenStack

Subir una imagen

Para subir una imagen a un proyecto:

  1. Navega a la página DreamCompute.
  2. Haz clic en el botón View Dashboard para iniciar sesión en tu panel de DreamCompute.
  3. En el panel izquierdo, haz clic en la pestaña Project para expandir el menú.
  4. En este menú haz clic en la pestaña Compute para expandir el menú.
  5. De la lista, haz clic en Images.
  6. Haz clic en el botón Create Image. La caja de diálogo Create Image aparece.

  7. En la caja de diálogo Create Image, ingresa los siguientes valores:

    Image Name Ingresa un nombre para la imagen.
    Image Description Ingresa una breve descripción de la imagen.
    Source Type DreamCompute solo soporta cargas de "Archivo" de tu computador local.
    File Busca el archivo de imagen en tu computador local y agrégalo.
    Format DreamCompute solo soporta imágenes RAW e ISO.
    Kernel No soportado; dejar en blanco.
    Ramdisk No soportado; dejar en blanco.
    Architecture No soportado; dejar en blanco.
    Minimum Disk (GB) Si deseas que los volúmenes basados en esta imagen sean siempre mayores que un tamaño específico, ingresa ese tamaño aquí.
    Minimum RAM (MB) Si deseas que las instancias que usan esta imagen usen al menos una cantidad particular de memoria RAM, ingresa esa cantidad aquí.
    Protected Selecciona si deseas o no que esta imagen sea borrable (Nota: esta configuración puede ser ajustada más tarde por cualquier persona con acceso a tu proyecto).

  8. Haz clic en el botón Create Image.

    El proceso de carga de imagen comienza. Cuando está completo, la caja de diálogo Create Image dialog se cerrará automáticamente.

    Si cierras el cuadro de diálogo manualmente, la carga se cancelará.

Actualizar una imagen

Para actualizar las propiedades de una imagen existente:

  1. Navega a la página DreamCompute.
  2. Haz clic en el botón View Dashboard para iniciar sesión en tu panel de DreamCompute.
  3. En el panel izquierdo, haz clic en la pestaña Project para expandir el menú.
  4. En este menú haz clic en la pestaña Compute para expandir el menú.
  5. De la lista, haz clic en Images.
  6. Junto a la imagen que deseas editar, haz clic en el botón de menú debajo de la columna Actions.
  7. De la lista, selecciona Edit Image. La caja de diálogo Edit Image aparece.
  8. En la caja de diálogo Edit Image, puedes realizar varias acciones. Por ejemplo:
    • Cambia el nombre de la imagen.
    • Provee una descripción para la imagen.
    • Establece requisitos mínimos de disco o RAM.
    • Especifica si deseas o no que cualquier persona con acceso a tu proyecto pueda eliminar la instancia.
  9. Cuando hayas finalizado de hacer cambios, haz clic en Update Image para guardar tus cambios.

Eliminar una imagen

Solo puedes eliminar una imagen si seleccionaste 'No' al elegir una opción 'Protected'. La eliminación de imágenes es permanente y no se puede revertir.

  1. Navega a la página DreamCompute.
  2. Haz clic en el botón View Dashboard para iniciar sesión en tu panel de DreamCompute.
  3. En el panel izquierdo, haz clic en la pestaña Project para expandir el menú.
  4. En este menú haz clic en la pestaña Compute para expandir el menú.
  5. De la lista, haz clic en Images.
  6. Junto a la imagen que deseas eliminar, haz clic en el botón de menú debajo de la columna Actions.
  7. De la lista, selecciona Delete Image.
  8. En el cuadro de diálogo Confirm Delete Image, haz clic en Delete Images para confirmar la eliminación.

Si deseas eliminar varias imágenes a la vez, selecciona las imágenes usando la casilla de verificación junto a ellas. Luego haz clic en el botón Delete Images.
