Generalidades

A continuación se describe cómo crear una base de datos MySQL en el panel DreamHost.

Crear una base de datos MySQL

Para crear una base de datos MySQL:

  1. Navega a la página Bases de Datos MySQL.
  2. Desplázate hasta la parte inferior de la página Bases de Datos MySQL:
  3. Ingresa lo siguiente:
    • Nombre de la base de datos — Ingresa un nombre único para la base de datos. Los nombres de las bases de datos deben ser únicos en todo el sistema de DreamHost y solo pueden contener números, las letras a-z, y guiones bajos _. Se recomienda que elijas un nombre que sea largo y descriptivo. Para mayor claridad, DreamHost sugiere colocar tu dominio en el nombre de la base de datos, como blog_example_com, en lugar de un ejemplo menos claro, como blog12345.

      No es posible cambiar el nombre de una base de datos después de ser creada. La única opción es crear una nueva base de datos e importar la antigua en ella.

    • Usar Hostname / Nuevo Hostname — Crea un hostname o selecciona uno existente en el menú desplegable. Se recomienda crear un hostname para cada dominio, ya que facilita el seguimiento de con qué dominio se usa.

      Los nuevos hostnames pueden tardar hasta 4 horas en funcionar debido a la propagación de DNS.

    • Usuario Nuevo — Crea un nuevo usuario de MySQL. Por razones de seguridad, se recomienda tener un usuario MySQL para cada base de datos, de modo que ningún sitio o credenciales comprometidos puedan usarse para acceder a tus otras bases de datos.
    • Nueva Contraseña — Si estás creando un nuevo usuario, ingresa aquí una contraseña para ese usuario. Si seleccionas un usuario existente, este campo estará deshabilitado.
    • Nueva Contraseña de Nuevo — Si estás creando un nuevo usuario, vuelve a ingresar la misma contraseña que se indicó anteriormente. Si seleccionas un usuario existente, este campo estará deshabilitado.
    • Comentario de Base de Datos — Ingresa aquí una descripción opcional de tu base de datos para uso organizacional.
  4. Da clic en el botón ¡Agregar nueva base de datos ahora! para crear la base de datos.

Preguntas frecuentes

¿Es posible cargar una base de datos existente?

No. No es posible cargar una base de datos existente al panel de DreamHost.

Primero debes crear una base de datos en blanco como se explicó en la sección anterior. Una vez creada, puedes usar phpMyAdmin para cargar tu archivo de base de datos.

¿Mi base de datos tiene privilegios completos?

No todos los privilegios de base de datos están disponibles en un entorno MySQL compartido. Revisa los siguientes artículos para obtener una lista de privilegios y su disponibilidad:

  • Limitaciones de MySQL debido a un Shared Hosting
  • Administrar usuarios de MySQL, hosts permitidos, y privilegios

¿En qué servidor MySQL se crea la base de datos si tengo un VPS MySQL?

Debes usar un único servidor de base de datos para todas tus bases de datos. Cuando compras un VPS MySQL, es posible que algunas de tus bases de datos se dividan entre el antiguo servidor MySQL compartido y el nuevo VPS MySQL. Para solucionar esto, debes fusionar las bases de datos en el nuevo servidor.
