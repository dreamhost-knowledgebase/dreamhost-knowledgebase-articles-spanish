Generalidades

Este artículo describe las diversas formas en que puedes aumentar tu plan desde un plan de alojamiento compartido Shared Starter.

Actualizar a un Shared Hosting

Consulta el siguiente artículo para obtener instrucciones sobre cómo actualizar a un Shared Hosting.

  • Actualizar un plan Shared Starter a Shared Unlimited

Actualizar el plan a un VPS

Consulta el siguiente artículo para obtener instrucciones sobre cómo agregar un plan de alojamiento de servidor privado.

  • Agregar un servidor privado (VPS)

Al momento de la compra del servidor privado, tendrás la oportunidad de migrar TODOS tus usuarios y dominios a este nuevo servidor. También puedes optar por no migrar ningún dato.

Si solo deseas pagar por un servidor privado y eliminar el Shared Hosting, marca el botón de opción para migrar todos los usuarios y dominios durante este proceso.

Actualizar a un Servidor Dedicado

Consulta el siguiente artículo para obtener instrucciones sobre cómo agregar un nuevo servidor dedicado.

  • ¿Cómo agregar un Servidor Dedicado?

Mientras creas el Servidor Dedicado, puedes elegir migrar todos tus usuarios compartidos a la nueva máquina.
