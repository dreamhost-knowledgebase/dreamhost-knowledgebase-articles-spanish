Generalidades

Puede agregar una dirección IP única (Unique IP) en su panel.

Este artículo explica cómo agregar una dirección IPv4. Si quieres en vez agregar una dirección IPv6, vé el siguiente artículo:

  • Agregar una dirección IPv6

Agregar una dirección IPv4

  1. Abre la página Manage Domains.
  2. Da click en el enlace UniqueIP debajo del dominio deseado.
    La página Adding unique IP address se abrirá:
  3. Selecciona una opción de pago del menú desplegable, y luego da click en el boton Add a Unique IP to DOMAIN now!.
  4. Regresa a la página Manage Domains para confirmar que la dirección IP ha sido agregada.
    • Una vez la dirección IP se haya agregado, tomará varias horas en resolver en línea.
    • Ésta dirección IP Única se renovará automáticamente al final de su término (mensual o anual). Debes eliminarla automáticamente para parar cualquier cargo futuro.

Eliminar una dirección IP Única de un dominio

Para eliminar una dirección IP Única:

  1. Abre la página Manage Domains.
  2. Da click en el ícono rojo X al lado de la IP Única que deseas eliminar.
    La página Destroy IP address se abrirá:
  3. Confirma la eliminación marcando la caja y dando click en el botón Remove ###.###.###.### from DOMAIN now!.
