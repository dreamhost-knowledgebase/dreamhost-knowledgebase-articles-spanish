Generalidades

WordPress es un sistema de gestión de contenidos (CMS) y una herramienta de blogs. Se puede utilizar para crear sitios web y blogs personales.

Para obtener más información sobre qué es WordPress y qué puedes hacer con él, visita el sitio web de WordPress en:

  • wordpress.org

Versión de WordPress en DreamHost

DreamHost trabaja en estrecha colaboración con WordPress para ofrecerle las funciones más actualizadas. Como tal, la versión más nueva de WordPress siempre está disponible en tu panel DreamHost.

Lo que necesitas antes de instalar

Si encuentras Remixer o WordPress regular demasiado limitado para tus necesidades, el soporte de DreamHost puede proporcionarte acceso gratuito al nuevo Creador de sitios web de WordPress (BoldGrid).

Para obtener más información sobre cómo instalar BoldGrid en tu sitio actual WordPress, comunícate con el equipo de soporte de DreamHost a través del LiveChat o enviando un correo a support@dreamhost.com. Un miembro del equipo de soporte de DreamHost puede ayudarte aún más con la configuración del nuevo Creador de sitios web de WP para tu sitio.

Consulta los siguientes artículos para obtener más información sobre BoldGrid:

  • Generalidades sobre BoldGrid
  • Centro de Soporte de BoldGrid

Antes de instalar WordPress, primero debes tener un plan de alojamiento activo como un plan de Shared Hosting, VPS o Dedicado.

A continuación, deberás haber comprado un registro de dominio. Si ya has comprado un registro de dominio en otra compañía, deberás apuntar tus nameservers o registros A a DreamHost. Si compraste el registro en DreamHost, no necesitas ajustar los DNS. Usar las actualizaciones automáticas de WordPress

A partir de la versión 3.7, WordPress ahora presenta actualizaciones automáticas. Estas son solo para actualizaciones menores de revisión (por ejemplo, 4.1 a 4.1.1).

DreamHost recomienda encarecidamente que permitas que estas actualizaciones se realicen automáticamente, ya que generalmente incluyen actualizaciones de seguridad importantes o correcciones de errores.

Puedes leer más sobre esta función y cómo deshabilitarla a través del siguiente enlace:

  • codex.wordpress.org/es:Configurando_actualizaciones_automaticas

Importar

Después de instalar y configurar WordPress, puedes importar contenido externo. Puedes hacerlo iniciando sesión en tu sitio de WordPress y yendo a 'Herramientas' > 'Importar'.

Importar contenido puede requerir plugins adicionales dependiendo de la plataforma que estés importando.
