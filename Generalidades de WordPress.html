Generalidades

Este artículo ofrece una descripción general de WordPress y las formas en que se implementa en DreamHost.

¿Qué es WordPress?

WordPress es un Sistema de Gestión de Contenido (CMS) y una herramienta de blogs para crear sitios web y blogs personales. Visita wordpress.org para obtener más información sobre qué es WordPress y qué puedes crear con este.

DreamHost trabaja en estrecha colaboración con WordPress para brindarte las funciones más actualizadas. Por eso, la versión más reciente de WordPress siempre está disponible en tu panel de DreamHost.

Diferencias entre WordPress.com y WordPress.org

WordPress ofrece dos versiones de producto: WordPress.org y WordPress.com.

WordPress.org

Este sitio ofrece software gratuito de código abierto que puedes descargar para crear tu sitio. Sin embargo, luego debes elegir una compañía de alojamiento para alojar el sitio y también serás responsable de administrar personalmente el contenido del mismo (actualizaciones, seguridad, temas, plugins, base de datos, etc.).

Los sitios WordPress de DreamHost utilizan una versión personalizada del código de WordPress.org y brindan todo lo que necesitas para comenzar.

WordPress.com

Este es el sitio oficial de alojamiento de WordPress, que se encarga de casi todo por ti. Sin embargo, sus funciones son mucho más limitadas que las del código gratuito que puedes descargar en WordPress.org. Por ejemplo, estos planes no permiten el acceso directo para modificar el código del sitio web y el correo no está disponible, por lo que deberás comprar un plan de alojamiento de correo en otra compañía

¿Cuál debería escoger?

Si planeas hacer crecer tu sitio con el tiempo, personalizarlo, o ganar dinero con él, WordPress.org (instalado en DreamHost) es la mejor opción.

Si tienes un sitio web muy simple que no planeas monetizar y no te importa la marca y los anuncios de WordPress, WordPress.com es una buena opción.

WordPress en DreamHost

WordPress está disponible en DreamHost en los siguientes planes de alojamiento.

Planes Shared, VPS, y Dedicados

La primera opción es comprar un plan de alojamiento Shared, VPS o Dedicado. Si compras uno de estos planes, puedes instalar WordPress usando el instalador de WordPress de DreamHost.

DreamPress

La segunda opción es comprar un plan de DreamPress, el plan de alojamiento de WordPress personalizado y administrado por DreamHost. Se trata de una mejora del alojamiento compartido, diseñado con beneficios adicionales como solidez del servidor, almacenamiento en caché y administración de memoria. Si compras este plan, WordPress se instala automáticamente.
