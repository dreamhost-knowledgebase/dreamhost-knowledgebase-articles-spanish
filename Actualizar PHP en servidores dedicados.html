Usuarios de Apache dedicados sin privilegios de administrador

  • Los usuarios de Apache sin privilegios de administrador se actualizaron automáticamente a PHP 5.6 el 14 de marzo de 2017.
  • Consulta las instrucciones sobre cómo actualizar la versión de PHP que estás utilizando.

DreamHost no recomienda cambiar a un usuario administrador para que puedas seleccionar una versión anterior de PHP; esto podría hacer que tu sitio sea vulnerable a cualquier amenaza que pueda estar en la naturaleza de dicho script y versión de PHP. Si tu sitio web usa un script que no está actualizado, se corre el riesgo de que ese script se vea comprometido y se puedan realizar acciones maliciosas en tu servidor.

Usuarios dedicados sin privilegios de administrador en Nginx

  • Los usuarios que no son administradores de Nginx todavía usan PHP 5.6.

DreamHost no recomienda cambiar a un usuario administrador para que puedas seleccionar una versión anterior de PHP; esto podría hacer que su sitio sea vulnerable a cualquier amenaza que pueda estar en la naturaleza de dicho script y versión de PHP. Si tu sitio web usa un script que no está actualizado, corre el riesgo de que ese script se vea comprometido y se puedan realizar acciones maliciosas en tu servidor.

Usuarios administradores (Nginx y Apache)

  • Los usuarios administradores (Nginx y Apache) pueden seleccionar la versión de PHP que desean usar.
  • DreamHost recomienda actualizar lo antes posible a PHP 7 ya que ejecutar versiones anteriores de PHP expone tu sitio web a ataques maliciosos.
  • Consulta las instrucciones sobre cómo actualizar la versión de PHP que estás usando.
