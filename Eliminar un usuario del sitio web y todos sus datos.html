Generalidades

Un 'usuario' del sitio web también se conoce como el nombre de usuario asignado a un dominio. Este es el usuario que utiliza para iniciar sesión en el servidor a través de FTP o SSH para actualizar tu sitio. Cada sitio web tiene un usuario asignado, y puedes cambiar el usuario en cualquier momento.

Si has cambiado el usuario de tu sitio o has eliminado tu sitio web en el panel, también puedes eliminar al usuario del servidor para asegurarte de que también se eliminen todos los datos del usuario junto con los datos del sitio web anterior.

Antes de eliminar un usuario

Se recomienda que primero hagas una copia de seguridad de todos los datos del usuario y luego asegúrate de que no haya sitios web asignados a un usuario que desees eliminar.

Copia de seguridad de todos los datos

Para hacer una copia de seguridad de los datos del usuario, inicia sesión en tu servidor a través de FTP:

  • Generalidades de FTP

Después de iniciar sesión, descarga enel computador de tu hogar todos los archivos y carpetas que desees guardar.

Asegurate de que no hay sitios web asignados al usuario

No se pueden asignar sitios web a un usuario que deseas eliminar. Puedes confirmar esto en la página Dominios Alojados. El nombre de usuario se muestra a la derecha del dominio en la columna Alojamiento Web:

No deberías ver el nombre de usuario que estás a punto de eliminar en ninguna parte de esta página. Si lo haces primero debes cambiar el usuario asignado a los dominios restantes. Revisa el siguiente artículo para obtener más detalles sobre cómo hacer un cambio a un usuario:

  • Mover un dominio a un usuario nuevo

Eliminar un usuario

Después de hacer una copia de seguridad de todos los datos del sitio web y asegurarte de que no se asignen nombres de dominio al usuario, ahora puedes proceder a eliminar el usuario del servidor.

Los siguientes pasos eliminan permanentemente los datos del usuario del servidor. No es posible recuperar ninguno de estos datos después de eliminarlos del panel. Asegúrate de haber realizado una copia de seguridad de todos los datos del usuario antes de continuar.

  1. Navega a la página Archivos y Usuarios FTP.
  2. Da clic en el botón de los 3 puntos verticales a la derecha de tu usuario.
  3. Del menú emergente, da clic en el enlace Borrar Usuario.

    Es posible que veas una advertencia al eliminar tu usuario. Esto significa que un dominio aún está asignado al usuario. Regresa a la página Dominios Alojados y cambia el nombre de usuario asignado a cualquier dominio con este usuario. Revisa el siguiente artículo para más información:

    • Mover un dominio a un usuario nuevo
    Si no hay dominios asignados al usuario, aparece mensaje confirmando la eliminación del usuario.
  4. Haz clic en Confirmar Eliminación de Usuario para eliminarlo permanentemente del servidor.
    Una vez el usuario haya sido eliminado, aparece un mensaje de éxito.
