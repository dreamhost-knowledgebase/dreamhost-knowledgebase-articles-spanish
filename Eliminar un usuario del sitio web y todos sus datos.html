Generalidades

Un 'usuario' del sitio web también se conoce como el nombre de usuario asignado a un dominio. Este es el nombre de usuario que utiliza para iniciar sesión en el servidor a través de FTP o SSH y para actualizar tu sitio. Cada sitio web tiene un usuario asignado, y puedes cambiar el usuario en cualquier momento.

Si has cambiado el usuario de tu sitio o has eliminado tu sitio web en el panel, también puedes eliminar al usuario del servidor para asegurarte de que también se eliminen todos los datos del usuario junto con los datos del sitio web anterior.

Antes de eliminar un usuario

Se recomienda que primero hagas una copia de seguridad de todos los datos del usuario y luego asegúrate de que no haya sitios web asignados a un usuario que desees eliminar.

Copia de seguridad de todos los datos

Para hacer una copia de seguridad de los datos del usuario, inicia sesión en tu servidor a través de FTP:

  • Generalidades de FTP

Después de iniciar sesión, descarga en la computadora de tu hogar todos los archivos y carpetas que desees guardar.

Asegurate de que no hay sitios web asignados al usuario

No se pueden asignar sitios web a un usuario que deseas eliminar. Puedes confirmar esto en la página Administrar Dominios. El nombre de usuario se muestra a la derecha del dominio en la columna Web Hosting:

No deberías ver el nombre de usuario que estás a punto de eliminar en ninguna parte de esta página. Si lo haces primero debes cambiar el usuario asignado a los dominios restantes. Consulta el siguiente artículo para obtener más detalles sobre cómo hacer un cambio a un usuario:

  • Cambiar el usuario asignado a un dominio

Eliminar un usuario

Después de hacer una copia de seguridad de todos los datos del sitio web y asegurarte de que no se asignen nombres de dominio al usuario, ahora puedes proceder a eliminar el usuario del servidor.

Los siguientes pasos eliminan permanentemente los datos del usuario del servidor. No es posible recuperar ninguno de estos datos después de eliminarlos del panel. Asegúrate de haber realizado una copia de seguridad de todos los datos del usuario antes de continuar.

  1. Navega a la página Archivos.

  2. Haz clic en la flecha hacia abajo 'Show info' junto a tu nombre de host para abrir el cuadro de configuración para el usuario.

  3. Haz clic en el enlace Delete This User.
