Generalidades

Puedes crear tres tipos diferentes de usuarios del sitio web.

  • FTP
  • SFTP
  • Shell

Estos usuarios se usan para iniciar sesión en el servidor para acceder a los archivos de tu sitio web. Esto es diferente al nombre de usuario que usa para iniciar sesión en un sitio CMS que ejecuta software como WordPress. Si necesitas restablecer la contraseña de ese usuario, revisa el siguiente artículo.

  • Cambiar contraseña (Aplicaciones de terceros)

Puedes cambiar la contraseña para todos los tipos en la página Archivos.

Cambiar la contraseña de usuario de tu sitio web

Para restablecer la contraseña de tu usuario en el panel:

  1. Navega a la página Archivos.
  2. Haz clic en la flecha hacia abajo 'Mostrar info' junto a tu nombre de alojamiento para abrir el cuadro de configuración para el usuario.
  3. Haz clic en el enlace Reestablecer Contraseña.
    Aparece el siguiente cuadro de diálogo que te solicita que ingreses una contraseña:
    Si deseas ver la contraseña, haz clic en el icono del ojo:
  4. Ingresa una contraseña y haz clic en el botón Restablecer Contraseña para restablecer la contraseña.

    Si la contraseña que ingresaste se encuentra en la base de datos haveibeenpwned, aparece el siguiente mensaje:

    Debes ingresar una nueva contraseña antes de continuar. Consulta haveibeenpwned.com para obtener más información.
