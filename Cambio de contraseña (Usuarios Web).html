Generalidades

Puedes crear tres tipos diferentes de usuarios del sitio web.

  • FTP
  • SFTP
  • Shell

Puedes cambiar la contraseña para todos los tipos en la página Manage Users.

Cambiar la contraseña de su usuario FTP

Para restablecer la contraseña de tu usuario en el panel:

  1. Navega a la página de Manage Users.
  2. Haz clic en la flecha hacia abajo 'Show Info' junto a tu nombre de host para abrir el cuadro de configuración para el usuario.
  3. Haz clic en el enlace 'Reset Password'.
    Se te solicitará que confirmes con el siguiente mensaje:
  4. Haz clic en el botón Yes, Generate New Password para restablecer la contraseña.

El panel genera una contraseña para ti. Para garantizar la seguridad del usuario y tus datos, no es posible agregar una contraseña personalizada.

Después de cambiar tu contraseña, espera hasta 5 minutos para que se actualice en el sistema DreamHost.
