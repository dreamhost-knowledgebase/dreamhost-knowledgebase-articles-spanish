Generalidades

A continuación se describe cómo editar tu usuario de WordPress en el panel de administración de WordPress para cambiar tu nombre para mostrar, dirección de correo electrónico o preferencias personales.

Editar tu usuario en el panel WordPress

  1. Inicia sesión en tu cuenta administrativa de WordPress.
  2. En el panel izquierdo, haz clic Users > All users.
  3. De la lista de todos los nombres de usuario actuales que aparecen, haz clic en el usuario que deseas editar.
  4. Realice cambios en su usuario de WordPress. Consulte las Secciones de Perfil a continuación para obtener una lista de opciones de usuario.
  5. Haz clic en Update Profile para guardar tus cambios.

Secciones de Perfil

Las opciones de usuario dentro de la página de perfil se dividen en las siguientes secciones:

Personal options

  • Visual Editor: Marca esta casilla para deshabilitar el editor visual.
  • Syntax Highlighting: Marca esta casilla para deshabilitar el resaltado de sintaxis.
  • Admin Color Scheme: Esto cambia el esquema de color de la interfaz de usuario del panel de administración.
  • Keyboard Shortcuts: Marca esta casilla para habilitar abreviaciones de teclado para moderación de comentarios.
  • Toolbar: Marca esta casilla para habilitar la barra de herramientas de administración cuando inicies sesión y veas el sitio.

Name

  • Username: Esto enumera el nombre de usuario interno de WordPress del usuario, que no se puede cambiar una vez creado el usuario.
  • First Name y Last Name: Estos campos son opcionales, pero se pueden completar para asociar al usuario con una persona específica.
  • Nickname (required): Edita este campo para especificar un nombre personalizado para el usuario. Si no se completa, se utilizará por defecto el nombre de usuario del usuario.
  • Display name publicly as: Haz clic en el menú desplegable para elegir qué opción de nombre desead que WordPress muestre a los visitantes para el usuario.

Contact Info

  • Email (required): Edita este campo para cambiar la dirección de correo electrónico asociada con el usuario. Si se cambia la dirección de correo electrónico del usuario, a la nueva dirección se te enviará un correo electrónico de confirmación antes de que pueda activarse.
  • Website: Edita este campo para agregar un sitio web personal al perfil del usuario.

About Yourself

  • Biographical Info: Este campo es opcional, pero se puede completar para agregar información biográfica al perfil del usuario.
  • Profile Picture: El avatar del usuario no se puede personalizar directamente en esta página, pero se puede editar a través del servicio Gravatar.

Account Management

  • New Password: Haz clic en el botón Generate Password para generar una nueva contraseña para el usuario. Consulta el siguiente artículo para obtener más información:
    • Cambiar tu contraseña en el panel de WordPress
  • Sessions: Haz clic en el botón Log Out Everywhere Else botón para finalizar todas las sesiones del usuario actual, excluyendo la actual.
