Generalidades

BoldGrid es un creador de sitios de WordPress fácil de usar. Utiliza un editor de arrastrar y soltar, que te permite crear y personalizar rápidamente un sitio web profesional sin necesidad de codificación.

BoldGrid viene preconfigurado con una lista de temas optimizados para dispositivos móviles para que puedas elegir según las necesidades de tu sitio. También hay disponibles herramientas adicionales como copias de seguridad para garantizar que tus datos estén seguros.

Registrarse en BoldGrid al abrir una cuenta DreamHost

Ve el siguiente artículo para abrir una nueva cuenta DreamHost.

  • Cómo registrarse para obtener una cuenta DreamHost 

Asegúrate de marcar la casilla titulada 'WP Website Builder'. Esto instala BoldGrid dentro de tu instalación de WordPress.

¿Cómo puedo habilitar BoldGrid en una instalación de WordPress?

Si originalmente no te inscribiste en BoldGrid en tu plan DreamHost WordPress, aún puedes habilitarlo de dos maneras diferentes a través de tu panel web.

Instalaciones de WordPress existentes

Si tienes una instalación de WordPress existente:

  1. Navega a la página One-Click Installs.
    El botón Manage Installed Applications aparece cerca de la parte superior de la página de Instalaciones de un clic en tu panel:
  2. Haz clic en el botón Manage Installed Applications.
  3. Expande el dominio haciendo clic en el ícono de flecha a la izquierda.
    La instalación de un clic se expande para ese dominio, lo que te muestra opciones adicionales:
  4. Haz clic en el botón Install WP Website Builder a la derecha.
    Se abre un cuadro emergente de confirmación que solicita confirmar la instalación:
  5. Haz clic en el botón Install para confirmar. WP Website Builder estará disponible en tu instalación de WordPress en unos minutos.

Nuevas instalaciones de WordPress

Si aún no tienes una instalación de WordPress:

  1. Navega a la página One-Click Installs.
  2. Selecciona WordPress de la lista que aparece.
    El siguiente cuadro de configuración aparece cuando selecciona WordPress:
  3. Realiza tus selecciones en los cuadros desplegables y marca la casilla de verificación 'Install WP Website Builder'.
  4. Haz clic en el botón Install it for me now! cuando hayas tenrminado.

Desactivar Boldgrid

Es altamente recomendado que hagas un respaldo de tu sitio de WordPress antes de desactivar un plugin. Revisa el siguiente artículo para más información:

  • Herramientas de copia de seguridad en WordPress

Usa estas instrucciones si necesitas desactivar el plugin del constructor de sitios web BoldGrid Inspirations:

  1. Abre la página de wp-admin de tu sitio de WordPress.
  2. En la barra de navegación izquierda, selecciona Plugins > Installed Plugins (menú de Plugins Instalados).
  3. Haz clic en el enlace para Deactivate debajo del plugin BoldGrid Inspirations.

Cosas por considerar al desactivar Boldgrid

El plugin de BoldGrid Inspirations controla muchas de las funcionalidades de diseño en WordPress, y tu sitio puede verse algo diferente cuando se desactive.

Cuando desactivas el plugin de BoldGrid Inspirations, tu tema va a cambiar al tema estándar de WordPress, Twenty Twenty. Todas tus entradas, páginas e imágenes (si hay) permanecerán intactas. También puedes reactivar BoldGrid, borrarlo completamente, o seleccionar un nuevo tema de ser necesario.

En adición a BoldGrid Inspirations, hay muchos más plugins que puede que quieras desactivar, cómo BoldGrid Gallery y BoldGrid Post and Page Builder (por nombrar algunos). Puedes usar las mismas instrucciones descritas en la sección previa para desactivar estos plugins igualmente.
