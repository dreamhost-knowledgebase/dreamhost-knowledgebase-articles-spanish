Generalidades

Este artículo explica qué es el período de Redención de dominio y cada paso de su proceso.

¿Qué es el periodo de redención?

El período de redención es el período de 30 días después de que ha expirado el registro de un dominio. Durante este tiempo, el registro retoma el control sobre el registro del dominio, y debes pagar una tarifa adicional de procesamiento para recuperarlo. Los siguientes pasos desglosan cada etapa de este proceso.

  • Retención de 30 días:: Para la mayoría de las extensiones/TLDs, DreamHost mantiene el registro del dominio durante 30 días para permitirte renovarlo en caso de que se haya pasado la fecha de renovación.
  • Periodo de expiración: Después de que pase este período de 30 días, DreamHost libera el dominio de vuelta al servicio de registro nacional, que inicia el verdadero proceso de expiración a su nivel.
  • Periodo de redención: El Registro luego colocará el dominio en un período de 'redención' durante otros 30 días y solo lo devolverá a DreamHost a cambio de una tarifa que cobra (además de un cargo por la renovación de un año del dominio). Para ver la tabla de tarifas de redención, visita la página de Términos de Registro de Dominio para obtener más detalles.
  • Periodo de eliminación: El dominio luego entra en un período de "Eliminacion Pendiente", durante el cual el registro está eliminando la propiedad y preparándose para liberar el registro del dominio al público para su compra. Durante este período, el registro ya no puede ser redimido.
  • Liberación al público: Una vez que el período de redención termina, el servicio de registro programa el dominio para su liberación de nuevo al público.

¿Puedo recuperar mi dominio durante este período?

Sí. Durante el período de redención, tienes una última oportunidad para recuperar el nombre de dominio pagando una tarifa de procesamiento de redención además del costo estándar de renovación anual del registro.

Visita la página Términos de Registro de Dominios para más detalles.

¿Cómo puedo redimir mi dominio?

Para redimir un nombre de dominio, navega a la página Administrar Registros de Dominio y haz clic en el botón ¡Redimir este dominio!. Esto te pedirá que pagues el costo de redención antes de poder completar el proceso.

Por favor ten en cuenta que este proceso puede tardar hasta 30 minutos. Si el dominio no se recupera y renueva en ese tiempo, Contactar a Soporte para recibir asistencia.
