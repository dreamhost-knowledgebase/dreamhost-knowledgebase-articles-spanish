Overview

SQLite es una aplicación de base de datos ligera que no utiliza una base de datos SQL cliente/servidor. Esto generalmente funciona muy bien con la mayoría de los sitios web de tráfico bajo a medio y debería administrarse bien en dominios con un promedio de 100.000 visitas por día.

SQLite en servidores DreamHost

SQLite está preinstalado en servidores Shared, VPS y Dedicados.

Versiones de Ubuntu 14 (Trusty)

Los servidores que ejecutan Ubuntu 14 ejecutan las siguientes versiones:

  • 2.8.17 para sqlite
  • 3.7.9 para sqlite3

Versiones de Ubuntu 18 (Bionic)

Los servidores que ejecutan Ubuntu 18 corren las siguientes versiones:

  • 2.8.17 para sqlite
  • 3.22.0 para sqlite3

Los enlaces para C, PHP, Perl, Python y Ruby ya están incluidos con el servicio SQLite.

El rendimiento es aceptable y puede mejorarse aún más estableciendo los parámetros adecuados: por ejemplo, el nuevo Registro de escritura anticipada puede ser habilitado usando PRAGMA journal_mode=WAL; (SQLite >= 3.7.0).
