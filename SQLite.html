Generalidades

SQLite es una aplicación de base de datos ligera que no utiliza una base de datos SQL cliente/servidor. Esto generalmente funciona muy bien con la mayoría de los sitios web de tráfico bajo a medio y debería administrarse bien en dominios con un promedio de 100.000 visitas por día.

SQLite en servidores DreamHost

SQLite está preinstalado en Servidores Shared, VPS y Dedicados. Revisa el siguiente artículo para confirmar la versión del sistema operativo de tu servidor y la de SQLite.

  • Ver las versiones del software de tu servidor

Los enlaces para C, PHP, Perl, Python y Ruby ya están incluidos con el servicio SQLite.
