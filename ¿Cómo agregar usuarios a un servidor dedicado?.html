Agregar un usuario de Shell

Visita el artículo acceso Crear un usuario con Shell (SSH) acceso para obtener instrucciones sobre cómo agregar un usuario Shell normal a tu servidor dedicado.

El usuario de Shell puede ejecutar comandos básicos de Shell después de iniciar sesión en el servidor a través de SSH.

Agregar un usuario administrador

El usuario normal de Shell no puede ejecutar comandos sudo/root en el servidor. Si necesitas ejecutar comandos sudo/root, debes crear un usuario administrador.

Para agregar un usuario administrador:

  1. Ve a la página de Admin Users en tu panel.
  2. Haz clic en el botón + Add A User la esquina superior derecha.
    Aparece la página de 'Add User Details:
  3. Ingresa lo siguiente:
    • Username — Un nombre de usuario para el nuevo usuario.
    • Nickname — Un apodo para el nuevo usuario. Este es un nombre de referencia rápida que solo se muestra en tu panel.
    • What server will this user have access to? — Haz clic en la flecha hacia abajo y selecciona el servidor dedicado al que tendrá acceso el usuario.
    • Select the Shell Type — Selecciona el tipo de Shell que deseas utilizar. Puedes seleccionar bash, tcsh, ksh o zsh de la lista desplegable. Visita el artículo de tipos de Shell para obtener más información.
    • I have read and agreed to the terms of use — Marca la casilla de verificación. Debes marcar la casilla de verificación 'términos de uso' para agregar un nuevo usuario administrador.
  4. Cuando termines, haz clic en el botón Create User.

Nota importante sobre los cambios personalizados del usuario administrador

Con un usuario administrador, puedes realizar cambios de root/sudo en tu servidor, pero ten en cuenta que DreamHost todavía administra tu servidor dedicado. Esto significa que si realizas cambios personalizados en Apache, se sobrescribirán cada vez que se ejecute una configuración de Apache. Esto puede suceder al agregar alojamiento a un dominio o al agregar un certificado SSL. 

Si realizas cambios personalizados, asegúrate de anotar claramente cuáles fueron para poder corregir rápidamente cualquier cosa que se haya sobrescrito.
