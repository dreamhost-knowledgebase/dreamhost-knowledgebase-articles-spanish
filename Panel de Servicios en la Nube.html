Generalidades

DreamHost ofrece dos servicios que están disponibles en el panel Cloud Services:

  • DreamObjects – DreamObjects
  • DreamCompute – DreamCompute

DreamObjects

DreamObjects es un servicio rentable de almacenamiento en la nube, que puedes usar para alojar datos estáticos para tus sitios web, almacenar copias de seguridad o desarrollar el próximo gran proyecto. Puedes acceder a DreamObjects en tu panel usando la interfaz integrada, mediante programación a través de API estándar, o con una creciente biblioteca de aplicaciones. Incluso es compatible con la API de Amazon S3.

Mira el siguiente artículo para obtener una lista de aplicaciones compatibles:

  • Aplicaciones Compatibles con DreamObjects

Puedes encontrar información adicional sobre DreamObjects aquí:

  • DreamObjects

DreamCompute

DreamCompute es un servicio de computación en la nube altamente escalable creado para impulsar todo, desde:

  • Aplicaciones web y móviles.
  • Medios digitales y sitios web de comercio electrónico.
  • Análisis de big data.
  • Entornos de prueba y desarrollo.

Todas las instancias se pueden administrar a través del panel de DreamCompute construido a partir del Openstack de Horizon. Para comenzar DreamCompute, puedes iniciar el servicio en tu panel en la página DreamCompute.
