Generalidades

DreamObjects es un sistema de almacenamiento de objects donde los archivos se almacenan como objects en buckets. Revisa el siguiente artículo de Wikipedia para obtener más detalles sobre el almacenamiento de objects:

  • wikipedia.org/wiki/Object_storage

¿Qué tipos de archivos se pueden almacenar?

DreamObjects se puede usar para almacenar archivos como fotos, canciones, documentos, copias de seguridad y más sin preocuparte por llenar el espacio en disco. Es capaz de almacenar casi cualquier tipo de archivo, siempre que cumpla con las condiciones de la política de uso aceptable.

¿Qué tan segura está mi información?

Aunque DreamObjects no admite el cifrado del lado del servidor, es seguro, ya que la única forma de acceder a tus datos es a través de una llave secreta y de acceso (comparable a un nombre de usuario y contraseña FTP/shell, pero generada al azar por DreamHost). Además, puedes emplear el cifrado del lado del cliente cifrando los datos en su ubicación y luego utilizando SSL para cargar los datos en DreamObjects.

DreamObjects está diseñado para una durabilidad del 99.99999% durante un año determinado. Usando lo último en tecnología de almacenamiento en la nube, tus datos se almacenan con bits de paridad adicionales en todo el centro de datos de DreamObjects, lo que garantiza una mayor seguridad de respaldo.

¿Qué necesito para usar DreamObjects?

La siguiente lista es la configuración básica y las herramientas que necesitarás para usar DreamObjects:

  1. Crear una cuenta DreamObjects en tu panel en la página DreamObjects.
  2. Crear un usuario.
  3. Crear un bucket.
  4. Agregar una llave secreta y de acceso a tu bucket.
  5. Subir archivos usando lo siguiente:
    • Subir/Descargar en el panel de DreamHost
    • Subir/Descargar usando un cliente de terceros
