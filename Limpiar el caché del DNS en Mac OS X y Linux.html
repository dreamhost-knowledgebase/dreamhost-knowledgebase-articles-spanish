Generalidades

Lo siguiente describe cómo limpiar la caché DNS en un sistema operativo macOS o Linux.

Antes de continuar, asegúrate de haber abierto una terminal en tu computador. Aquí es donde escribirás el comando para limpiar el caché DNS.

  • Abrir la Terminal en la Mac
  • Open la terminal en Ubuntu Linux

Limpiar el DNS en macOS

Corre el siguiente comando para limpiar el caché DNS en macOS (Sonoma, Ventura, Monterey):

macbook$ sudo dscacheutil -flushcache;sudo killall -HUP mDNSResponder

Limpiar el DNS en Linux

La mayoría de las distribuciones de Linux no utilizan una caché de resolución DNS como Windows y macOS. En su lugar, deberás consultar el sitio web o los foros de tu distribución de Linux para confirmar qué demonio particular está instalado.

Sin embargo, una aplicación de caché DNS comúnmente utilizada es el Name Service Caching Daemon (nscd). Es probable que no esté instalada por defecto, por lo que no es necesario limpiar la caché. Pero, si ya la has instalado, puedes limpiar la caché ejecutando el siguiente comando en una terminal:

[local]$ sudo service nscd restart

Alternativamente, puedes tratar estos comandos:

[local]$ /etc/rc.d/init.d/nscd stop
[local]$ /etc/rc.d/init.d/nscd start
