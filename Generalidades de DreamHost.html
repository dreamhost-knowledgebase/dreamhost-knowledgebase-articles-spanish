Generalidades

DreamHost se fundó originalmente en 1997 como New Dream Network, mientras que sus fundadores todavía eran estudiantes universitarios en Harvey Mudd College en Claremont, California. Esta es una foto de Josh, Sage, Michael, y Dallas el día de la graduación.

Desde entonces, el nombre de la compañía ha cambiado a DreamHost y actualmente aloja más de un millón de sitios web de todo el mundo.

DreamHost actualmente ofrece una variedad de servicios para alojar tu sitio web, que incluyen:

  • Alojamiento de páginas web.
  • Alojamiento dedicado de WordPress.
  • Alojamiento de correo electrónico.
  • Registro de Dominios.
  • Servicios en la nube.

Este artículo es una descripción general rápida de estos servicios e incluye enlaces adicionales para ayudarte a obtener una comprensión rápida de lo que DreamHost tiene para ofrecer. Si deseas omitir esta descripción general y comenzar de inmediato, visita el siguiente artículo:

  • Comenzar

Alojamiento de páginas web

DreamHost ofrece cuatro tipos de servidores web que puedes utilizar para alojar tu sitio web.

  • Servidores compartidos – Shared Hosting
  • Servidores privados VPS
  • Servidores Dedicados – Dedicados
  • DreamPress – Alojamiento WordPress Dedicado

Puedes encontrar una explicación detallada de cada uno en las siguientes páginas:

  • https://www.dreamhost.com/es/hosting/
  • Alojamiento

WordPress administrado (DreamPress)

DreamHost también ofrece planes de alojamiento mejorados y administrados para sitios de WordPress llamados DreamPress. Este paquete premium de WordPress incluye las siguientes características:

  • Cada instancia de DreamPress se configura en el servicio de computación en la nube de DreamHost llamado DreamCompute.
  • La memoria se escala automáticamente.
  • Utiliza un caché incorporado en el servidor.
  • Opciones de precios simples.

Visita el artículo de DreamPress para más detalles.

Alojamiento de correo electrónico

El alojamiento de correo se incluye de forma gratuita con la mayoría de los planes de alojamiento.

También está disponible como complemento a bajo costo para un plan de dominio de solo registro o un plan Shared Started.

También puedes crear una cuenta de solo correo sin un plan de alojamiento de sitios web.

El plan complementario de correo electrónico incluye gratis el servicio de Webmail también

Registros de dominio

DreamHost ofrece la posibilidad de comprar un registro de dominio desde el panel. La compra de un registro de dominio no requiere que también tengas un plan de alojamiento con DreamHost — una vez que se registra el dominio, puedes alojarlo en cualquier lugar que desees. Sin embargo, es más fácil mantener todo en el mismo lugar.

Adicionalmente, si compras un plan de Shared Hosting, obtendrás un registro gratuito durante el primer año de alojamiento. Visita el siguiente artículo para obtener una lista de dominios a los que esto aplica:

  • Registros de dominio gratis

Para calificar para un crédito de registro de dominio gratuito durante el primer año completo, debes adquirir un plan de 1 año o 3 años en un Shared Starter, Shared Unlimited o cualquier plan anual de DreamPress (mensual de Shared Unlimited o los planes mensuales de DreamPress no aplican). Este crédito gratuito NO está disponible en ningún otro plan de alojamiento. Visita las siguientes páginas para ver más detalles:

  • Términos de registro de dominio
  • Suscríbete a Shared Hosting

Para ver una lista de los nuevos TLD que DreamHost recientemente comenzó a ofrecer junto con una herramienta para ver si el registro está disponible, visita la siguiente página:

  • https://www.dreamhost.com/es/dominios/

DreamHost también ofrece un servicio gratuito de privacidad de WHOIS:

  • Privacidad de WHOIS

La mayoría de los registradores te obligan a pagar una tarifa adicional para mantener oculta tu información personal, pero DreamHost incluye este servicio de forma gratuita con tu registro de dominio. Sin embargo, ten en cuenta que no todos los registros admiten la privacidad. Visita el siguiente artículo para obtener una lista de dominios que no ofrecen privacidad en DreamHost:

  • Privacidad de registro de dominio

Servicios de Cloud

DreamHost actualmente ofrece dos servicios en la nube:

  • DreamObjects
  • DreamCompute

DreamObjects

DreamObjects es un servicio de almacenamiento en la nube, que puede utilizar para hacer una copia de seguridad de los datos que desee. Visite las siguientes páginas para más información:

  • https://www.dreamhost.com/cloud/storage/
  • Generalidades de DreamObjects

DreamCompute

DreamCompute te permite controlar tu propio servidor personal en la nube con recursos escalables, lo que es especialmente útil para desarrolladores y nuevas empresas. Visita las siguientes páginas para obtener más información sobre este servicio:

  • https://www.dreamhost.com/cloud/computing/
  • Generalidades de DreamCompute
