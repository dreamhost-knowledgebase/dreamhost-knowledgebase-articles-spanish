Generalidades

DreamHost fue fundada originalmente en 1997 como New Dream Network, y sus fundadores (Josh, Sage, Michael y Dallas) aún eran estudiantes de pregrado en Harvey Mudd College en Claremont, California. Desde entonces, el nombre de la empresa ha cambiado a DreamHost y actualmente aloja más de un millón de sitios web en todo el mundo.

DreamHost actualmente ofrece una variedad de servicios para alojar tu sitio web, incluyendo:

  • Alojamiento Web
  • Alojamiento para WordPress
  • Alojamiento de Correo
  • Registros de Dominios
  • Servicios de Nube

Este artículo ofrece una visión general rápida de estos servicios e incluye enlaces adicionales para ayudarte a entender nuestras ofertas.

Alojamiento Web

Existen cuatro tipos de servidores para alojar tu sitio web en DreamHost:

  • Servidores Shared
  • Servidores Virtuales Privados (VPS)
  • Servidores Dedicados
  • Alojamiento Administrado de WordPress – DreamPress

Alojamiento Administrado de WordPress (DreamPress)

DreamHost ofrece un plan de alojamiento WordPress mejorado y administrado llamado DreamPress. Este paquete premium de WordPress incluye las siguientes características:

  • Cada instancia de DreamPress se configura en el servicio de computación en la Nube de DreamHost llamado DreamCompute.
  • La memoria se escala automáticamente
  • Utiliza una caché incorporada en el servidor
  • Opciones de precios sencillos

Visita el artículo de DreamPress para más detalles.

Alojamiento de Correo

El correo electrónico está incluido de forma gratuita con la mayoría de los planes de alojamiento. También está disponible como un complemento de bajo costo para un plan de solo registro de dominio o Shared Starter. Este plan de correo incluye el servicio Webmail gratuito. También puedes crear una cuenta de solo correo sin un plan de alojamiento web.

Servicios de Listas de Correo

Hay dos tipos de servicios de listas en DreamHost: Listas de Anuncios y Listas de Discusión (estos servicios no requieren un plan de alojamiento).

Registros de Dominios

DreamHost ofrece la posibilidad de comprar un registro de dominio desde el panel. Comprar un registro de dominio no requiere que también tengas un plan de alojamiento con nosotros; una vez registrado el dominio, puedes alojarlo donde quieras.

Adicionalmente, si compras un plan de Alojamiento Shared, obtienes un registro gratuito para el primer año de alojamiento. Revisa el artículo de registros de dominios gratuitos para ver una lista de los dominios a los que esto aplica.

Para calificar para un crédito de registro de dominio gratuito durante el primer año completo, debes inscribirte en un plan anual de 1 año o 3 años en un plan Shared Starter, Shared Unlimited, o cualquier plan anual de DreamPress (los planes mensuales Shared o mensuales DreamPress no aplican). Este crédito gratuito NO está disponible bajo ningún otro plan de alojamiento. Por favor, visita las páginas Términos de Registro de Dominios y Regístrate para Alojamiento Shared para más detalles.

Visita nuestra página web Dominios para ver una lista de los nuevos TLDs que DreamHost ha comenzado a ofrecer recientemente y una herramienta para verificar si el registro está disponible.

La mayoría de los registradores te obligan a pagar una tarifa adicional para mantener tu información personal oculta, pero DreamHost incluye este servicio de privacidad WHOIS de forma gratuita con tu registro de dominio. Sin embargo, ten en cuenta, por favor, que no todos los registros admiten privacidad. Revisa el artículo de privacidad de registro de dominio para ver una lista de los dominios que no ofrecen privacidad en DreamHost.

Servicios de Nube

DreamHost actualmente ofrece dos servicios en la nube:

  • DreamObjects: un servicio de almacenamiento en la nube que puedes usar para respaldar cualquier dato que desees.

  • DreamCompute: te permite controlar tu propio servidor en la nube personal con recursos escalables.
