Generalidades

Este artículo explica cómo ver el plan actual de DreamPress que estás utilizando.

Ver tu plan

Una vez que tengas al menos un dominio en DreamPress, ve a la página de Managed WordPress.

Haz clic en el botón Manage. Aparece un panel que te muestra una descripción general de tu servicio DreamPress. A continuación se muestra una descripción de cada campo:

Sección superior

  • Manage WordPress — Haz clic aquí para iniciar sesión directamente en tu sitio DreamPress.
  • Jetpack Professional — Muestra si has habilitado Jetpack Professional para tu sitio DreamPress.

Sección central

  • Plan — Se muestra si tienes un plan anual o mensual y tu fecha de renovación.
  • Storage Used — Muestra la cantidad de su plan de almacenamiento actual que estás utilizando. También puede hacer clic en 'Resize storage' para cambiar la cantidad de almacenamiento que usa tu plan.

Sección inferior (pestaña Details)

  • Administrar archivos — haz clic en el botón 'Show Credentials' para ver las credenciales de inicio de sesión de FTP/SSH de su usuario. Luego puedes usar un cliente FTP o SSH para iniciar sesión en su servidor.
  • Administrar base de datos — haz clic en el botón 'Show Credentials' para ver las credenciales de inicio de sesión de la base de datos de tu sitio web. Luego puedes usar phpMyAdmin para iniciar sesión en tu base de datos.

Sección inferior (pestaña Domain)

Cuando haces clic en la pestaña 'Domain', tienes la opción de cambiar tu nombre de dominio de DreamPress. A la derecha también está la opción de importar un sitio de WordPress existente.
