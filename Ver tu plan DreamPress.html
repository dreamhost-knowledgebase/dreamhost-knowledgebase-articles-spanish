Generalidades

Este artículo explica cómo ver el plan actual de DreamPress que estás utilizando.

Ver tu plan

Una vez que tengas al menos un dominio en DreamPress, ve a la página de Managed WordPress.

Haz clic en el botón Manage. Aparece un panel que te muestra una descripción general de tu servicio DreamPress. A continuación se muestra una descripción de cada campo:

Sección superior

  • Open WP-Admin — Haz clic aquí para iniciar sesión directamente en tu sitio DreamPress.
  • Jetpack Professional — Muestra si has habilitado Jetpack Professional para tu sitio DreamPress.

Sección central

  • Plan — Se muestra si tienes un plan anual o mensual y tu fecha de renovación.
  • Storage — Muestra la cantidad de su plan de almacenamiento actual que estás utilizando. También puede hacer clic en 'Resize storage' para cambiar la cantidad de almacenamiento que usa tu plan.

Sección inferior (pestaña Details)

  • sFTP User — Este es el usuario web que utilizas para iniciar sesión en tu servidor utilizando SFTP o SSH. El usuario usa SFTP de manera predeterminada, pero puedes cambiarlo haciendo clic en el enlace 'Manage user'. Ten en cuenta que este NO es tu usuario de WordPress, este es solo el usuario web que usas para iniciar sesión en tu servidor DreamPress si alguna vez es necesario.
  • Server — El nombre de tu servidor web DreamPress.
  • mySQL User — Tu usuario de la base de datos MySQL de DreamPress. Ten en cuenta que no tienes acceso para iniciar sesión en el servidor MySQL.
  • mySQL Server — El nombre de tu servidor de base de datos MySQL.

Sección inferior (pestaña Domain)

Cuando haces clic en la pestaña 'Domain', tienes la opción de cambiar tu nombre de dominio de DreamPress. A la derecha también está la opción de importar un sitio de WordPress existente.
