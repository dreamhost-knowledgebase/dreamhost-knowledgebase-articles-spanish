Generalidades

Este artículo describe las diversas formas en que puede actualizar o degradar desde un plan de alojamiento compartido ilimitado.

Cambiar tu plan a Starter Shared

Si desea cambiar de un plan de alojamiento compartido ilimitado a un plan de alojamiento compartido básico, asegúrese de cumplir con las limitaciones mencionadas en el siguiente artículo:

  • Agregar un plan de alojamiento Starter Shared

Si quieres continuar, Contact Support para solicitar esto.

Mejorar tu plan a VPS

Consulta el siguiente artículo para obtener instrucciones sobre cómo agregar un nuevo plan de alojamiento de servidor privado:

  • Agregar un servidor privado (VPS)

Al comprar, tienes la oportunidad de migrar TODOS tus usuarios y dominios al nuevo servidor privado. También puedes optar por no migrar ningún dato.

Si solo deseas pagar por un servidor privado y eliminar el alojamiento compartido, marca el botón de opción para migrar todos los usuarios y dominios durante este proceso.

Mejorar tu plan a servidor Dedicado

Consulta el siguiente artículo para obtener instrucciones sobre cómo agregar un nuevo servidor dedicado:

  • ¿Cómo agregar un servidor Dedicado?

Mientras creas el servidor dedicado, puedes elegir migrar todos tus usuarios compartidos a la nueva máquina.

Mejorar tu plan a DreamPress

Si ya tienes un sitio de WordPress en alojamiento compartido, puedes usar el siguiente artículo para actualizarlo a DreamPress:

  • ¿Cómo mejorar una página de WordPress a DreamPress?

También puedes agregar un nuevo dominio a DreamPress usando el siguiente artículo:

  • ¿Cómo agrego DreamPress a un nuevo dominio?
