Generalidades

Este artículo describe cómo agregar usuarios al servidor y a tu sitio web.

Agregar un usuario a un servidor

Para agregar un nuevo usuario web:

  1. Navega a la página Archivos y Usuarios SFTP en tu panel.
  2. En la ventana emergente que aparece, ingresa lo siguiente:
    • Nombre de Usuario— Un nombre de usuario para el nuevo usuario.
    • Servidor — Da clic en la flecha hacia abajo y selecciona el servidor al cual el usuario tendrá acceso.
    • Contraseña (Generar contraseña segura) — Ingresa una contraseña o da clic en el botón para que el panel cree una contraseña fuerte por tí.
    • Acceso Seguro Shell (SSH) — Si no está seleccionado, un usuario SFTP sin acceso Shell es creado. Activa esto en para habilitar el acceso a SSH
  3. Cuando termines, haz clic en el botón Crear Usuario.

¿Qué sucede si tengo más de un sitio web en DreamHost?

Cada sitio web alojado con DreamHost es "propiedad" de exactamente un usuario. Sin embargo, un solo usuario puede ser propietario de más de un sitio web. En ese caso, cada sitio tendría su propio directorio bajo el directorio raíz del usuario.

Puedes configurar la cuenta para que un usuario pueda acceder a un solo subdirectorio en un sitio web de DreamHost.

¿Por qué dice que mi nombre ya está tomado?

El "namespace" para los usuarios de DreamHost se comparte en un grupo bastante grande de máquinas. Como resultado, todos los nombres de usuario creados para usuarios SFTP o SHELL deben ser únicos en todo el grupo. Por lo general, los nombres de usuario comunes como "admin" ya se han tomado, por lo que los usuarios pueden necesitar agregar caracteres adicionales a sus usuarios para asegurarse de que sean únicos.
