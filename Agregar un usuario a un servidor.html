Generalidades

Este artículo es una descripción general sobre cómo agregar usuarios a tu servidor, y luego tu sitio web.

Agregar un usuario a un servidor

Para agregar un nuevo usuario web:

  1. Navega a la página Usuarios & Archivos FTP en tu panel.
  2. Haz clic en el botón + Agregar un Usuario en la esquina superior derecha.
    Aparece la página 'Agregar Detalles de Usuario':
  3. Ingresa lo siguiente:
    • Nombre de Usuario — Un nombre de usuario para el nuevo usuario.
    • Apodo  — Un apodo para el nuevo usuario.
    • ¿A qué servidor tendrá acceso este usuario? — Haz clic en la flecha hacia abajo y selecciona el servidor en el que el usuario tendrá acceso.
    • Acceso seguro a Shell — Usa el botón de alternar para habilitar Secure Shell/SSH (derecha) o deshabilitar (izquierda). Si el 'Acceso seguro a shell' está deshabilitado, se usa SFTP.
    • Selecciona el tipo de Shell — Si el acceso seguro de shell está habilitado, aparece el cuadro desplegable 'Selecciona el tipo de Shell'. Puedes seleccionar bash, tcsh, ksh o zsh. Visita la sección Usuarios Shell a continuación para más información.
  4. Cuando termines, haz clic en el botón Crear Usuario.

¿Qué sucede si tengo más de un sitio web en DreamHost?

Cada sitio web alojado con DreamHost es "propiedad" de exactamente un usuario. Sin embargo, un solo usuario puede ser propietario de más de un sitio web. En ese caso, cada sitio tendría su propio directorio bajo el directorio raíz del usuario.

Puedes configurar la cuenta para que un usuario solo tenga acceso a un subdirectorio en un sitio web de DreamHost.

¿Por qué dice que mi nombre ya está tomado?

El "namespace" para los usuarios de DreamHost se comparte en un grupo bastante grande de máquinas. Como resultado, todos los nombres de usuario creados para usuarios FTP/SFTP/SHELL deben ser únicos en todo el grupo. Por lo general, los nombres de usuario comunes como "admin" ya se han tomado, por lo que los usuarios pueden necesitar agregar caracteres adicionales a sus usuarios para asegurarse de que sean únicos.
