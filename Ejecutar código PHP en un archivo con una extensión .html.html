Generalidades

Normalmente, cuando PHP y HTML se combinan en el mismo archivo, el archivo usa una extensión .php y muestra datos en el navegador. Por ejemplo, este archivo se llama hello.php.

<html
<body>

<?php
  echo "hello";
?>

</body>
</html>

Sin embargo, si necesitas agregar código PHP en un archivo con una extensión .html, debes agregar un Handler a tu archivo .htaccess para que el código PHP funcione.

Extensiones de archivo personalizadoas

Puedes usar los ejemplos de este artículo para cualquier extensión de archivo que necesites. Por ejemplo, la extensión del archivo .test no existe, pero si creas un archivo llamado myfile.test, puedes forzarlo para correr como PHP.

Solamente cambia el código en el archivo .htaccess para usar la extensión .test en vez de .html.

Código para agregar a tu archivo .htaccess

El archivo que necesitas agregar al código se llama .htaccess. Si no existe en el directorio home del sitio web, revisa los siguientes artículos para ver instrucciones sobre cómo crear un archivo usando un cliente de FTP o SSH.

  • Crear y editar archivos a través de FTP
  • Crear y editar archivos a través de SSH

FCGI

Agrega la siguiente línea al archivo .htaccess para que use FastCGI para PHP:

AddHandler fcgid-script .html
FcgidWrapper "/dh/cgi-system/php82.cgi" .html

Si cargas el archivo en un buscador, cargará como un archivo de PHP normal.

Este código también cambia la versión del PHP que tienes configurada en el panel a la versión en la segunda línea.
