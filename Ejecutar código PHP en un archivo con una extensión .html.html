Generalidades

Normalmente, cuando PHP y HTML se combinan en el mismo archivo, el archivo usa una extensión .php y muestra datos en el navegador. Por ejemplo, este archivo se llama hello.php.

<html
<body>

<?php
  echo "hello";
?>

</body>
</html>

Sin embargo, si necesitas agregar código PHP en un archivo con una extensión .html, debes agregar un Handler a tu archivo .htaccess. Por ejemplo, el mismo código de arriba puede ser agregado en un archivo llamado hello.html.

Código para agregar a tu archivo .htaccess

Consulta los siguientes artículos para obtener instrucciones sobre cómo editar archivos con un cliente FTP o SSH.

  • Crear y editar archivos vía FTP
  • Crear y editar archivos vía SSH

El archivo al que necesitas agregar código se llama .htaccess. Si este no existe en el directorio principal de tu sitio web, usa los artículos anteriores para crearlo. Agrega la siguiente línea al archivo .htaccess.

AddHandler php5-cgi .html

Esta línea de código permite que los archivos con extensiones .html ejecuten código PHP en tu navegador.
