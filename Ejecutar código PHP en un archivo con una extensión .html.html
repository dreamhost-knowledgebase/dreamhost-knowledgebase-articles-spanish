Generalidades

Normalmente, cuando PHP y HTML se combinan en el mismo archivo, el archivo usa una extensión .php y muestra datos en el navegador.

Sin embargo, si necesitas colocar código PHP en un archivo con una extensión .html, debes agregar un controlador a tu archivo .htaccess.

Código para agregar a tu archivo .htaccess

Consulta los siguientes artículos para obtener instrucciones sobre cómo editar archivos con un cliente FTP o SSH.

  • Crear y editar archivos via FTP
  • Crear y editar archivos via SSH

El archivo al que necesitas agregar código se llama .htaccess. Si este no existe en el directorio principal  de tu sitio web, usa los artículos anteriores para crearlo. Agrega la siguiente línea al archivo .htaccess, según la versión de Linux en la que se esté ejecutando tu sitio.

Si tu sitio está en un servidor que ejecuta Ubuntu 18

Puedes agregar cualquiera de los siguientes.

AddHandler php-cgi .html
AddHandler php5-cgi .html

Si tu sitio está en un servidor que ejecuta Ubuntu 14

Solo lo siguiente funcionará

AddHandler php5-cgi .html

Esta línea de código permite que los archivos con extensiones .html ejecuten código PHP en tu navegador.
