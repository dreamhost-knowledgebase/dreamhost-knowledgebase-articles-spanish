Generalidades

Es posible que aunque hayas agregado el archivo phprc, correctamente, puede que te des cuentas de los cambios no se vean aplicados en línea. Es posible que debas eliminar todos los procesos PHP en ejecución para garantizar que la configuración nueva surta efecto.

A continuación se describen los pasos para impulsar los cambios que realices en el archivo phprc.

Estas instrucciones se aplican a sitios que no son de DreamPress. Si estás intentando actualizar la configuración de PHP en un sitio de DreamPress, consulta el siguiente artículo:

  • Personalizar la configuración PHP en DreamPress

Actualizar tu archivo phprc file por SSH

    1. Conectate a tu servidor usando SSH.
    2. Una vez conectado, ejecuta el siguiente comando teniendo en cuenta la versión PHP de tu sitio, para eliminar los procesos PHP que se estén ejecutando en el momento:

En los ejemplos a continuación, reemplaza 'usuarioshell'” por tu usuario SSH todo en minúsculas.

For PHP 5.6:

[server]$ killall -9 php56.cgi -u usuarioshell

For PHP 7.0:

[server]$ killall -9 php70.cgi -u usuarioshell

For PHP 7.1:

[server]$ killall -9 php71.cgi -u usuarioshell

For PHP 7.2:

[server]$ killall -9 php72.cgi -u usuarioshell

For PHP 7.3:

[server]$ killall -9 php73.cgi -u usuarioshell

Eliminar los procesos PHP de esta manera actualiza tu archivo phprc.

Si después de ejecutar el comando killall ves php56.cgi: no process found (no se encontró ningún proceso), esto significa que no hay procesos PHP en ejecución. Realiza una actualización completa en su navegador para ver tu página phpinfo.php actualizada. Ahora deberías ver que tu configuración se ha actualizado.

Actualizando el archivo phprc en el panel

Para usar esta opción, tu dominio debe estar Completamente Alojado.

También puedes actualizar tu archivo phprc guardando la configuración de tu dominio en el panel.

  1. Navega a la página Manage Domains.
  2. Haz click en el botón Edit a la derecha de tu dominio en la columna 'Web Hosting'.
    • Se abre la página de configuración 'Manage Domains':
  3. En la primera sección, haz click en el botón Change settings.
    • Esto actualizará el archivo phprc en 10 minutos.

Actualización del archivo phprc en un servidor Nginx

Si tu servidor ejecuta Nginx, puedes actualizar los procesos PHP al guardar tu configuración en el panel como se menciona anteriormente

Si estás en un Servidor Dedicado con un usuario administrador, también puedes volver a cargar y reiniciar Nginx. Consulta el siguiente artículo para más instrucciones:

  • Cómo volver a cargar y reiniciar Nginx (solo para servidores Dedicados)

Sitios WordPress

Si has hecho algún ajuste a la configuración de PHP y tu sitio de WordPress no muestra el nuevo valor, verifica el archivo wp-config.php. Puedes ver una línea como la siguiente:

define( 'WP_MEMORY_LIMIT', '128M' );

Asegúrate de que cualquier configuración coincida con lo que has agregado al archivo phprc.
