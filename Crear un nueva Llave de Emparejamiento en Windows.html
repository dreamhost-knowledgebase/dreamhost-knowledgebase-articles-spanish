  • Configurar PuTTY

Crear la Llave de Emparejamiento

  1. Navega hasta el directorio donde guardaste el archivo puttygen.exe y ábrelo.
    Aparece la ventana PuTTY Key Generator
  2. En la derecha inferior, haz clic en el botón Generate.
  3. Sigue las instrucciones y mueve el mouse en el área en blanco para generar datos aleatorios.
  4. Haz clic en el botón Save public key para guardar una versión modificada de la clave pública.

    Ten en cuenta que este archivo de llave pública no se puede importar al panel de DreamCompute porque no está formateado correctamente.

  5. Haz clic en el botón Save private key para guardar el archivo .ppk (archivo de clave privada PuTTY) para usarlo con PuTTY más tarde.

    Una ventana emergente se muestra con el siguiente texto:

    Are you sure you want to save this key without a passphrase to protect it?

    Da clic en el botón Yes para guardar sin una contraseña.

  6. Copia el texto en el cuadro titulado Public key for pasting into OpenSSH authorized_keys file:
  7. Se recomienda que lo pegues en un archivo de bloc de notas en blanco en tu computador para asegurarte de tener una copia guardada. De lo contrario, sigue las instrucciones a continuación para pegarlo/importarlo directamente en tu panel de DreamCompute.
