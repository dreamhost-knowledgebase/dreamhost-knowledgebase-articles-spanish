Generalidades

Una dirección de correo de solo reenvío es una dirección de correo que no tiene un buzón activo. Esto se debe a que la función principal de este usuario es reenviar todos los correos enviado a otro usuario de correo. El mensaje solo se almacenará en el usuario configurado para recibir el correo reenviado.

NO establezcas una dirección de correo activa en una dirección de solo reenvío. Esto eliminará todos los correos guardado anteriormente con esta dirección.

Prerrequisitos

Debes tener un plan de alojamiento activo, o un Plan de correo adicional pago para usar correo.

Todos los planes de alojamiento, excepto Shared Starter ofrecen correo sin costo.

Mientras los planes de Shared Starter y los registros de dominio solo no incluyen con correo, puedes agregar un plan de correo pagado en tu cuenta que incluye hasta 5 direcciones de correo. Revisa el siguiente artículo para más información:

  • Plan de correo pago adicional

Crear una redirección de correo

  1. Visita la página Administrar Correo en tu panel.
  2. Haz clic en el botón Crear una Nueva Dirección de Correo.
  3. Desplázate hacia abajo a la sección Solo Redirección.
    Dirección de Correo Electrónico
    Ingresa tu alias deseado a la izquierda del símbolo "@". Selecciona el dominio o subdominio bajo el cual se creará el usuario a la derecha del símbolo "@".
    Lista todas las direcciones de correo a reenviar, una por línea
    Ingresa la dirección o direcciones de correo a las que deseas reenviar a este usuario. Puedes crear hasta 10 redirecciones por correo. Asegúrate de tener cada dirección de correo para reenviar y configurar en tu propia línea dentro del campo vacío.

  4. Haz clic en el botón ¡Reenviar a Estas Direcciones! para crear el reenvío.
    Las columnas en la captura de pantalla anterior son para direcciones de correo de solo reenvío, que no tienen un buzón activo. Debido a esto, estos usuarios se enumeran por separado de los usuarios que tienen buzones activos.

    Las direcciones de correo de solo reenvío son direcciones de correo que no tienen un buzón activo porque estos usuarios están destinados a reenviar todos los correos que se le envían a otra dirección de correo.

Descripción de los campos en la página de correo de solo reenvío

Direcciones de Solo Reenvío: En esta columna, solo verás la dirección de correo del usuario reenviador porque no hay un buzón activo para mostrar detalles para.

Reenvía a: Esta columna mostrará la dirección de correo del usuario que recibirá los mensajes enviados a este usuario reenviador. Puedes agregar más de uno si lo deseas.

Acciones: Esta columna tiene dos opciones para cada usuario enumerado: Editar y Eliminar.

Editar: El botón editar te lleva a otra página donde puedes realizar cambios en tu dirección de reenvío.
Eliminar: Al hacer clic en este botón se elimina al usuario, pero aparece una ventana emergente que te solicita que verifiques esta solicitud primero con el mensaje:
¿Eliminar realmente a usuario@example? (No hay correo almacenado con él).
Al hacer clic en “OK” se eliminará el usuario y el buzón de ese usuario. Esto requerirá unos minutos, pero una vez hecho esto, ya no verás a ese usuario en la lista en la página Administrar Correo para ese dominio.

Eliminar el reenvío de correo

Hay dos opciones para eliminar el reenvío de correo:

  • Cambiar de reenvío a una dirección completamente alojada o correo basura.
  • Elimina la dirección de reenvío.

Para eliminar el reenvío de un correo:

  1. Visita la página Administrar Correo en tu panel.
  2. A la derecha de la dirección de reenvío hay dos botones, Editar y Eliminar.
  3. Haz clic en cualquiera de esos botones según lo que te gustaría hacer con la dirección.
