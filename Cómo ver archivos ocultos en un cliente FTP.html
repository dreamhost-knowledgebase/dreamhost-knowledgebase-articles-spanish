Deberías usar SFTP para conectarte (en lugar de FTP) ya que SFTP garantiza una conexión segura a tu servidor.

Si ya se estás conectando usando SFTP, es posible que ya puedas ver archivos ocultos. Esto depende del cliente que estés usando.
