Revisar las configuraciones predeterminadas

Cada versión de PHP tiene varias configuraciones predeterminadas que puede ver creando un archivo phpinfo.php: Este es un archivo de PHP simple que puedes agregar a tu sitio web. Cuando ves la página, este muestra las configuraciones predeterminadas de PHP que están configuradas en tu dominio. Revisa el siguiente artículo para ver los pasos de como crear este archivo:

  • Ver tu versión PHP y configuraciones

Ajustar las configuraciones predeterminadas

Puedes ajustar las configuraciones predeterminadas de PHP creando un archivo phprc personalizado. Este es un archivo de configuración personalizada que te permite ajustar valores específicos. Revisa los siguientes artículos para ver pasos de como crear este archivo vía FTP o SSH:

  • Crear un archivo phprc vía FTP
  • Crear un archivo phprc vía SSH
