Generalidades

DreamHost ofrece la capacidad de crear muchos tipos diferentes de usuarios, cada uno de los cuales es necesario para diferentes tareas dentro del panel, así como para interactuar con tu sitio.

Cada usuario tiene su propia contraseña única que es independiente de otros usuarios. Este artículo detalla cómo cambiar las contraseñas para cada tipo de usuario en tu cuenta DreamHost.

Compartir contraseñas con el Equipo de soporte de DreamHost

A veces es necesario iniciar sesión en un sitio para solucionar un problema. Esto podría suceder con sitios CMS como WordPress o una instancia de DreamCompute (a la que el equipo de soporte no tiene acceso).

Ten en cuenta que enviar contraseñas por correo electrónico no es seguro y podría comprometer la seguridad de tu contraseña.

Si es absolutamente necesario compartir las credenciales de inicio de sesión para solucionar un problema, asegúrate de seguir estos dos pasos.

  1. Si es posible, crea un usuario de prueba y una contraseña para compartir con el Equipo de soporte de DreamHost.
  2. Una vez que se resuelva tu problema, cambia inmediatamente tu contraseña.

Pautas sobre contraseñas

Hay algunas cosas que siempre debes tener en cuenta al crear una contraseña. Esto ayudará a garantizar la seguridad de tu usuario, así como a proteger a otros usuarios en caso de una intrusión:

  1. Nunca uses tu nombre de usuario como contraseña, ni siquiera una parte de dicho usuario.
  2. Nunca uses información personal como tu nombre, número de teléfono, dirección, aniversario, cumpleaños, nombre de la mascota, fecha de nacimiento, matrícula, etc.
  3. Cuanto más larga sea la contraseña, mejor. Se recomienda que tenga al menos 8 caracteres.
  4. Nunca uses la misma contraseña dos veces. Al cambiarla, usa siempre algo nuevo.
  5. Combina letras, números y caracteres especiales para que tu contraseña sea más difícil de descifrar.
  6. Para aumentar la seguridad, mezcla letras mayúsculas y minúsculas.
  7. No uses palabras comunes que puedan adivinarse como "invitado" o "contraseña" o cualquier variación de esas palabras.
  8. Cuando sea posible, habilita la Autenticación Multifactor.
  • También puedes usar un sitio web para ayudar a crear una contraseña. Un ejemplo sería passwordgenerator.

Lo más importante, no compartas tu contraseña con nadie. Si alguien necesita acceso a tu cuenta, intenta opciones alternas como crear un nuevo usuario para ellos o darles acceso al panel a través de Privilegios de Cuenta.

Limitaciones

La longitud máxima de la contraseña es de 31 caracteres.

Puedes agregar caracteres especiales. Los únicos caracteres que no están permitidos son comillas simples y dobles.

Solución de problemas

Es posible que después de restablecer una contraseña dentro del panel, encuentres que no funciona o que la contraseña anterior todavía funciona. Esto se debe comúnmente a dos problemas:

  1. No ha pasado suficiente tiempo desde que cambiaste la contraseña. Espera 30 minutos para que tu contraseña se actualice en línea.
  2. Tu contraseña anterior está almacenada en el caché. Es posible que necesites borrar la memoria caché de tu navegador web si te das cuenta de que la contraseña anterior de tu usuario todavía se está almacenando en el caché al iniciar sesión. Si la nueva contraseña no funciona después de 30 minutos de haberla cambiado, intenta borrar la memoria caché de tu navegador web y luego intenta nuevamente:
    • Limpiar caché
    • Limpiar DNS
