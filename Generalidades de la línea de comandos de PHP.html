Generalidades

Lo siguiente explica cómo utilizar la línea de comandos para encontrar información relacionada con la versión de PHP que utiliza tu servidor.

Requisitos previos

Para correr los comandos de este artículo, debes configurar un usuario de Shell e iniciar sesión en tu servidor a través de SSH.

Encontrar las ubicaciones PHP

Las rutas para los binarios instalados por defecto de DreamHost PHP-CLI son las siguientes:

  • PHP 8.1 — /usr/local/php81/bin/php
  • PHP 8.2 — /usr/local/php82/bin/php
  • PHP 8.3 — /usr/local/php83/bin/php

DreamHost mantiene accesos directos convenientes para todas las versiones de PHP en el directorio /usr/local/bin/:

lrwxrwxrwx  1 root root    16 Apr 12 13:11 php-8.1 -> ../php81/bin/php
lrwxrwxrwx  1 root root    16 Apr 13 11:11 php-8.2 -> ../php82/bin/php
lrwxrwxrwx  1 root root    16 Apr 12 11:33 php-8.3 -> ../php80/bin/php

El directorio /usr/local/bin está incluido en la ruta del shell para los usuarios de manera predeterminada, por lo que puedes usar los accesos directos en lugar de la ruta completa a los binarios.

Cómo obtener la información de la versión de PHP

Esta sección explica cómo ver las versiones específicas de PHP y los módulos instalados con este.

Revisar la versión de PHP

Corre este comando para ver la versión predeterminada de PHP-CLI que se ejecuta en tu servidor:

[server]$ php -v

Puedes ver el número de versión exacto ejecutando un binario específico. Por ejemplo:

[server]$ php-8.3 -v

Revisar los módulos instalados

Puedes determinar qué módulos PHP están instalados para una versión específica de PHP corriendo el siguiente comando:

[server]$ /usr/local/php83/bin/php -m
