Generalidades

Django es un marco de desarrollo web para Python de la misma manera que Rails es un marco para Ruby. Es utilizado por varios sitios web importantes, incluido Google (es decir, para Google Application Engine), y puede facilitar mucho más el desarrollo de aplicaciones web enriquecidas.

Django no es una aplicación en sí misma. Necesitará dominio en la programación de Python para escribir una aplicación usando Django.

Habilitar Passenger en tu dominio

El uso de Passenger con Django permite una implementación, mantenimiento y rendimiento más fáciles. Mira el siguiente artículo para activar Passenger en tu dominio:

  • ¿Cómo habilito Passenger en mi dominio?

Podrías usar FCGI con Django, pero la configuración es mucho más complicada y no tan eficiente.

Crear una base de datos

Navega a MySQL Databases y crea una nueva base de datos en blanco que usarás solo para tu aplicación Django.

Visita el artículo de phpMyAdmin para obtener detalles sobre tu hostname, usuario, y contraseña.
