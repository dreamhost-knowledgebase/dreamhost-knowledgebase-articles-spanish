Generalidades

Este artículo describe la información básica que necesitarás para configurar una dirección de correo de DreamHost en cualquier cliente de correo que prefieras. Revisa Clientes de Correo para ver una lista de clientes populares que puedes utilizar.

Como configurar un correo de DreamHost

 
 

Agregar un correo en el panel

Puedes crear una nueva dirección de correo en la página de Administrar Correo. Revisa el artículo Agregar un correo electrónico para obtener instrucciones.

Ubicar el usuario y la contraseña

Una vez que crees un correo, tendrás tu Nombre de usuario y Contraseña.

Usuario: Esta es la parte antes de tu dominio. Por ejemplo, nombredeusuario@example.com. Tu cliente de correo puede requerir la dirección completa para conectarse, pero a menudo solo se necesita la parte del nombre de usuario.

Contraseña: La contraseña del usuario es la que elegiste al crear el correo. La contraseña se puede cambiar en cualquier momento en la página de Administrar Correo. Revisa el artículo Cambiar contraseña para obtener instrucciones y el artículo sobre Contraseñas para obtener ayuda sobre cómo elegir una contraseña segura.

 

Escoger POP o IMAP para conectarse

POP e IMAP son diferentes protocolos de conexión que los clientes de correo utilizan para recuperar los correos de un servidor. La siguiente tabla ofrece una visión general de las diferencias entre estos protocolos.

Herramienta POP IMAP
Donde los correos se almacenan

POP descarga todos los correos del servidor y los almacena en tu computador. De esta manera, los correos están disponibles cuando no estás conectado a Internet.

También tienes la opción (en tu cliente de correo) de mantener los correos en el servidor. Sin embargo, si esta configuración no está habilitada, los correos se eliminan del servidor y solo se almacenan localmente en tu computador.

IMAP mantiene todos los correos en el servidor mientras los sincroniza con tu cliente de correo. Dado que los correos permanecen en el servidor, puedes ver todos los correos desde cualquier dispositivo.

El Webmail de DreamHost usa IMAP.
¿Cuántos dispositivos pueden acceder al correo? Un dispositivo, a menos que hayas habilitado la opción en tu cliente de correo para mantener los correos en el servidor. Múltiples dispositivos (computador, tablet, celular, etc.)
Limitaciones El correo solo se descargará de tu carpeta de Bandeja de entrada. Cualquier otro correo en carpetas o subcarpetas (como Papelera, Borradores y Enviados) debe ser movido a la Bandeja de entrada para poder ser visto o descargado a través de POP. N/A
 

¿Cuál debería escoger?

Se recomienda IMAP porque el correo está disponible desde cualquier dispositivo con el que elijas conectarte y está almacenado de forma segura en el servidor.

No se recomienda POP porque descarga los correos en tu computador local, lo que significa que podrían perderse si algo le sucede a tu dispositivo.

Respalda el correo si cambias de POP a IMAP.

Procede con los siguientes pasos si configuraste tu correo en tu cliente para usar POP y quieres cambiar a IMAP:

  1. En tu cliente de correo, actualiza el nombre de host y el número de puerto a los valores de IMAP.
  2. Sal y reinicia el cliente de correo.
  3. Verifica tus configuraciones de nombre de host y puerto para confirmar que ahora se están utilizando los valores de IMAP.
    • Si tu cliente actualizó el nombre de host y el puerto, has cambiado con éxito a IMAP y ahora puedes ver el correo desde cualquier dispositivo.
    • Si tu cliente no actualizó el nombre de host y el puerto a IMAP, necesitarás proceder con los siguientes pasos.
  4. Haz una copia de seguridad de todos los correos que tu cliente POP haya descargado en tu computador local. Revisa la documentación de tu cliente de correo para obtener los pasos sobre este proceso.

    Esto es importante ya que, dependiendo del cliente de correo, es posible que cualquier correo descargado previamente en tu computador a través de POP se elimine después de eliminar la dirección.

  5. Una vez que hayas hecho el respaldo del correo, elimina la dirección POP actual de tu cliente.
  6. Agrega nuevamente la dirección usando el nombre de host y el puerto de IMAP.
  7. Utiliza las instrucciones de este artículo para cargar los correos respaldados en tu computador a tu cuenta IMAP. Esto es necesario si tu antigua configuración POP no habilitó la opción para guardar los correos en el servidor.

Has cambiado a IMAP con éxito y ahora puedes ver el correo desde cualquier dispositivo.

Errores del cliente de correo

Si no usas uno de los nombres de host mencionados arriba, tu cliente no podrá conectarse. Podrías ver los siguientes errores:

  • Discordancia de certificado
  • Esta identidad no puede ser verificada

Solo debes usar los nombres de host mencionados arriba para evitar estos errores. Revisa este artículo para más detalles.

 

Configurar el nombre de host y el puerto de salida

Tanto POP como IMAP utilizan el mismo nombre de host de salida y puertos.

¿Cuál puerto debería usar?

Se recomienda el puerto 465 con SSL, sin embargo, algunos clientes de correo no pueden usar este puerto. Si no puedes usar el puerto 465, la siguiente mejor opción es el puerto 587 usando STARTTLS.

Servidor/Puerto POP / IMAP
Nombre de host de salida smtp.dreamhost.com
Puerto 465 Transporte Seguro (SSL habilitado)
Puerto 587

Transporte inseguro (Actualizado a una conexión segura usando STARTTLS)
Puerto 25 Obsoleto y no recomendado. La autenticación de nombre de usuario/contraseña DEBE estar habilitada si se usa este puerto.

¿Qué es el tipo de conexión STARTTLS?

El método STARTTLS se conecta al puerto SMTP/IMAP/POP regular y luego mejora la conexión a TLS enviando una solicitud STARTTLS.

Proveedores de Servicios de Internet

Algunos Proveedores de Servicios de Internet (ISP) requieren que utilices su servidor SMTP para enviar correos (por ejemplo: smtp.tuISP.com). Si notas problemas al enviar correos, verifica con tu Proveedor de Servicios de Internet para confirmar esto.

Bloqueos temporales

Múltiples intentos fallidos de inicio de sesión en un correo pueden resultar en que el servidor bloquee temporalmente tu IP. Este bloqueo se levantará automáticamente con el tiempo, pero si necesitas que se elimine antes, por favor Contactar a Soporte.
