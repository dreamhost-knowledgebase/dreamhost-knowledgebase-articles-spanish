Este artículo describe que es un plan de alojamiento Starter Shared y cómo crear una cuenta con este tipo de plan.

¿Qué es un plan de alojamiento Starter Shared?

El plan Starter Shared, es el plan más básico de alojamiento ofrecido por DreamHost.

Este plan, es una excelente opción para poner en funcionamiento un solo sitio web a un costo muy bajo. Ya que es un plan de inicio hay algunas limitaciones a tener en cuenta que se mencionan a continuación.

¿DreamHost ofrece otros planes de alojamiento?

Además del plan Starter Shared, también están disponibles los siguientes planes de alojamiento:

  • Shared — Tu sitio web es alojado en un servidor compartido
  • VPS — Tu sitio web es alojado en un servidor privado
  • Dedicated — Tu sitio web es alojado en un servidor privado
  • Remixer — Tu sitio web es alojado en el plan de Remixer creador de sitios web
  • DreamPress — Tu sitio web es alojado en un servidor privado corriendo una versión de WordPress administrada po DreamHost

¿Hay alguna limitación?

Ya que el Starter Shared es un plan introductorio, hay ciertas limitaciones que debes tener en cuenta:

  • Solo un sitio web puede ser alojado
  • No incluye correo electrónico
  • 6 Bases de datos MySQL
  • 5 subdominios
  • 50GB de espacio de almacenamiento
  • 6 usuarios web server

¿Cómo agregar un plan Starter Shared?

Para crear un plan Starter Shared, visita la siguiente página:

  • https://www.dreamhost.com/hosting/shared/

Debajo de 'Starter Shared' haz clic en el botón Sign Up Now, y sigue las indicaciones para inscribirte a una nueva cuenta DreamHost. Una vez completado, tendrás acceso a tu panel desde:

  • panel.dreamhost.com

Agregar tu sitio web al plan

Después de que el plan ha sido agregado, inicia sesión en tu panel. Puedes ver más información sobre cómo iniciar sesión en tu panel de control de DreamHost, visitando el siguiente artículo:

  • Generalidades del panel de control de DreamHost

En la página de Manage Account, tu plan Starter Shared aparecerá:

Para agregar tu página al nuevo plan:

  1. Navega a la página Manage Users. Usa el siguiente artículo para agregar un nuevo usuario a tu servidor.
    • Agregar usuarios a un servidor
  2. Una vez agregado, navega a la página Manage Domains. Usa el siguiente artículo para agregar un nuevo dominio a tu servidor.
    • ¿Cómo agrego un dominio?
  3. Después de agregar el dominio, espera unas horas mientras es propagado en línea. Una vez sea completado, podrás visitar tu sitio web.

Cambiar cual dominio está alojado con un plan de Hosting Shared Starter

Puedes usar los siguientes pasos si quieres que el plan de tu dominio en Shared Started a 'Fully Hosted' a otro que tengas.

Solo un dominio puede ser alojado al tiempo con un plan Starter Shared. Sin embargo, no necesitas contactar a Soporte para cambiar el dominio activo. Para cambiar el sitio web a 'Fully Hosted' en el plan:

  1. Navega a la página Manage Domains Ve el siguiente artículo para eliminar el alojamiento para el dominio activo actual:
    • Configura tu dominio como 'DNS Only'
  2. Después que el dominio esté como 'DNS Only', ve el siguiente artículo para agregar tu nuevo dominio como 'Fully Hosted' en tu servidor.
    • ¿Cómo agregar un dominio?
