Generalidades

El equipo de desarrollo de PHP tiene un ciclo regular de introducción de nuevas versiones y también finalización del soporte activo y las correcciones de seguridad para versiones antiguas de PHP. DreamHost recomienda que corras una versión actualizada de PHP, pero también reconoce que no siempre es factible. Como resultado, DreamHost mantiene estas versiones anteriores ofreciendo el Soporte Extendido de PHP, lo que te da más tiempo.

¿Qué es el Soporte Extendido de PHP?

El Soporte Extendido de PHP es para clientes que necesitan correr versiones no soportadas de PHP durante un tiempo después del período admitido definido por el equipo de desarrollo de PHP. Con el Soporte Extendido de PHP, DreamHost crea y configura paquetes de tu versión descontinuada de PHP en una infraestructura moderna para mantener el sitio en línea.

¿Por qué se ha agregado el servicio de Soporte Extendido de PHP a mi cuenta?

Los sitios web que usan versiones no soportadas de PHP pueden seguir activos con el Soporte Extendido de PHP. Si los escaneos muestran que uno o más dominios en tu plan de alojamiento incluyen una versión no compatible de PHP, el Soporte Extendido de PHP se agregará automáticamente a tu cuenta.

¿Cuándo se agregará el servicio de Soporte Extendido de PHP a mi cuenta?

Si agregas o renuevas un servicio y todavía estás ejecutando versiones no soportadas de PHP, el Soporte Extendido de PHP se agregará automáticamente a tu cuenta 60 días después, el primer día del siguiente mes.

¿Qué versiones de PHP son compatibles con el Soporte Extendido de PHP?

Las versiones de PHP que actualmente requieren Soporte Extendido son PHP 8.0 y anteriores. Ten en cuenta que esto cambia cada año a medida que se lanzan nuevas versiones y PHP ya no mantiene las versiones antiguas.

¿DreamHost mantiene el código de mi sitio por mi?

No. Tu eres el responsable de mantener tu propio código y asegurarte de que es compatible con los lanzamientos actuales de PHP.

¿Cuánto cuesta el Soporte Extendido de PHP?

El Soporte Extendido de PHP cuesta actualmente $5USD por mes, por cuenta DreamHost. No se te cobrará por dominio.

¿Cómo agrego el Soporte Extendido de PHP a mi cuenta?

El soporte extendido de PHP solo se agrega durante las actualizaciones de PHP si tu sitio no es compatible con la versión compatible más antigua. Por favor contacta a soporte si crees que tu sitio falla debido a una actualización y necesitas devolverlo a una versión anterior de PHP.

¿Qué opciones tengo para eliminar el Soporte Extendido de PHP?

Puedes considerar una de las siguientes opciones para eliminar el Soporte Extendido de PHP:

  • Actualizar a una versión moderna de PHP.
  • Deshabilitar tu sitio web configurándolo como Parqueado.

Además, no es posible cancelar el plan mientras tu sitio esté ejecutando una versión no compatible de PHP.

¿Necesito eliminar el Soporte Extendido de PHP de mi cuenta después de actualizar PHP?

No. Una vez que DreamHost detecte que tu sitio ya no ejecuta una versión no soportada de PHP, el Soporte Extendido de PHP se eliminará automáticamente de tu cuenta y ya no se te cobrará por él dentro del siguiente ciclo de facturación.
