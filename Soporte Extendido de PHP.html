Si necesitas ayuda para actualizar tu sitio para que admita versiones modernas de PHP, los Servicios Profesionales de DreamHost pueden ayudarlo a actualizar tu código.

Generalidades

El equipo de desarrollo de PHP tiene un ciclo regular de introducción de nuevas versiones y también finalización del soporte activo y las correcciones de seguridad para versiones antiguas de PHP. DreamHost recomienda que corras una versión actualizada de PHP, pero también reconoce que no siempre es factible. Como resultado, DreamHost mantiene estas versiones anteriores ofreciendo el Soporte Extendido de PHP, lo que te da más tiempo.

¿Qué es el Soporte Extendido de PHP?

El Soporte Extendido de PHP es para clientes que necesitan correr versiones no soportadas de PHP durante un tiempo después del período admitido definido por el equipo de desarrollo de PHP. Con el Soporte Extendido de PHP, DreamHost crea y configura paquetes de tu versión descontinuada de PHP en una infraestructura moderna para mantener el sitio en línea.

¿Por qué se agregó el servicio de Soporte Extendido de PHP a mi cuenta?

Después del 1 de marzo de 2021, los sitios web que usan versiones no soportadas de PHP pueden seguir activos con el Soporte Extendido de PHP. Si los escaneos muestran que uno o más dominios en tu plan de alojamiento incluyen una versión no compatible de PHP, el Soporte Extendido de PHP se agregará automáticamente a tu cuenta.

¿Cuándo se agregará el servicio de Soporte Extendido de PHP a mi cuenta?

Se agregará el Soporte Extendido de PHP a tu cuenta:

  • Si agregas o renuevas un servicio entre el 4 de diciembre de 2020 y el 25 de enero de 2021, y todavía estás usando versiones no soportadas de PHP, el Soporte Extendido de PHP se agregará automáticamente a tu cuenta el 1 de marzo de 2021.
  • Si agregas o renueva un servicio el 26 de enero de 2021 o después de esa fecha, y todavía estás corriendo versiones no compatibles de PHP, el Soporte Extendido de PHP se agregará automáticamente a tu cuenta 60 días después del primer día del mes siguiente.

¿Qué versiones de PHP son compatibles con el Soporte Extendido de PHP?

Las versiones de PHP que actualmente requieren Soporte Extendido son PHP 7.2 y anteriores. Ten en cuenta que esto cambia cada año a medida que se lanzan nuevas versiones y PHP ya no mantiene las versiones antiguas.

¿Cuánto cuesta este soporte?

El plan de soporte extendido de PHP cuesta actualmente $5USD por mes, por cuenta DreamHost. No se te cobrará por dominio.

¿Qué opciones tengo para eliminar el Soporte Extendido de PHP?

Puedes considerar las siguientes opciones para eliminar el Soporte Extendido de PHP:

  • Actualizar a una versión moderna de PHP.
  • Deshabilitar tu sitio web configurandolo como Parqueado.

¿Necesito eliminar el servicio de Soporte Extendido de PHP en mi cuenta después de actualizar PHP?

No. Una vez que DreamHost detecte que tu sitio ya no ejecuta una versión no soportada de PHP, el servicio se eliminará automáticamente de tu cuenta y ya no se te cobrará por él.
