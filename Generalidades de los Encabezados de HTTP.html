¿Qué son los encabezados HTTP?

Los encabezados de HTTP son información que se transmite entre un navegador web y un sitio web cuando se visita. Estos encabezados contienen se pueden usar para agregar o confirmar si un servicio está activo en tu sitio web, como CORS, encabezados de Seguridad, o Cloudflare.

Lista de encabezados de HTTP

Los encabezados están agrupados dentro de las siguientes cuatro categorías:

  • General headers
  • Request headers
  • Response headers
  • Entity headers

Dentro de cada uno de estos grupos hay muchos tipos diferentes de encabezados que puede configurar. Consulta el siguiente enlace para obtener más información.

  • https://developer.mozilla.org/en-US/docs/Web/HTTP/Headers
