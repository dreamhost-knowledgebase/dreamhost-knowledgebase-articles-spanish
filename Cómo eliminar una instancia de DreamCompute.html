Generalidades

Cuando ya no necesites una instancia, puedes eliminarla usando el cliente de linea de comandos de OpenStack o dentro del panel de DreamCompute.

Eliminar desde el panel

  1. Navega a la página DreamCompute e inicia sesión en tu panel de DreamCompute.
  2. En la izquierda superior, haz clic en la pestaña Compute.
  3. En la lista expandida, haz clic en Instances.
  4. Para eliminar tu instancia, haz clic en el menú desplegable debajo de la columna Action junto a tu instancia. En el menú, selecciona Delete Instance.

eliminar usando la línea de comando

Debes instalar el cliente de línea de comandos de OpenStack para ejecutar los siguientes comandos.

Ver todas las instancias

Lista todas las instancias con el comando list.

[user@localhost]$ openstack server list
+-------------+----------------------+--------+------------+-------------+------------------+
| ID          | Name                 | Status | Task State | Power State | Networks         |
+-------------+----------------------+--------+------------+-------------+------------------+
| 8a99547e... | myInstanceFromVolume | ACTIVE | None       | Running     | private=10.0.0.4 |
| d7efd3e4... | newInstance          | ERROR  | None       | NOSTATE     |                  |
+-------------+----------------------+--------+------------+-------------+------------------+

Eliminar la instancia

Ejecuta el comando delete para eliminar la instancia. El siguiente ejemplo elimina la instancia de newInstance que está en estado ERROR:

[user@localhost]$ openstack server delete newInstance

El comando no te notifica que tu instancia fue eliminada, pero puedes agregar el parámetro --wait para evitar que se realicen más acciones en la instancia mientras se elimina.

Verifica que fue eliminada

Para verificar que la instancia fue eliminada, ejecuta el comando list como se muestra arriba, o actualiza la página Compute > Instances del panel de DreamCompute. La instancia ya no debería aparecer en la lista.
