Generalidades

En un Servidor dedicado, todas las bases de datos MySQL se almacenan localmente en la misma máquina. Esto te brinda acceso completo como root para modificar tus bases de datos MySQL según lo necesites.

Esto es diferente de los planes de alojamiento Shared, VPS, o DreamPress, donde las bases de datos MySQL se almacenan en un servidor MySQL separado.

Acceder a la base de datos

Los usuarios de Servidores Dedicados pueden administrar sus bases de datos en el panel de DreamHost, al igual que con cualquier otro plan de alojamiento.

Acceso a SSH

Si eres un usuario avanzado con conocimientos de la Shell de UNIX, puedes iniciar sesión en tu servidor a través de SSH para realizar personalizaciones más allá de lo que ofrece el panel. Por favor ten en cuenta que el soporte de DreamHost no puede ayudar con ningún tipo de modificaciones personalizadas en el servidor.

Cómo actualizar o degradar tu plan

Las siguientes secciones describen cómo actualizar o degradar un plan de servidor dedicado.

Actualizar a un servidor Dedicado

Cuando actualizas a un plan de alojamiento Dedicado, DreamHost automáticamente mueve todas tus bases de datos al nuevo servidor dedicado.

El servicio MySQL se mueve incluso si la opción del panel no mover datos está seleccionada.

Degradar desde un servidor Dedicado

Si cancelas el plan de alojamiento Dedicado, tus bases de datos MySQL se moverán a un servidor MySQL compartido.
