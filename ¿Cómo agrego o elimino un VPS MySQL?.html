Agregar un VPS MySQL

  1. Navega hasta la página VPS.
  2. Haz clic en el botón Add MySQL VPS.
    • Una vez agregado, puede tomar algunas horas aprovisionar el nuevo servidor.
  • Cuando se registra para un VPS MySQL, todas sus bases de datos se trasladan a la medianoche, hora del Pacífico. (El panel lo muestra como una fusión).
  • El acceso SSH no está disponible.

Una vez aprovisionado, puede administrarlo en la página del panel de VPS.

Eliminar un VPS MySQL

Si ya no necesitas tu servidor de base de datos privado, puedes eliminarlo en su panel.

  1. Navega hasta la página Dashboard.
  2. Verás tu servidor listado como psmysqlXXXXX.
  3. A la derecha, haz clic en el botón Delete.
  4. Se te pide que confirmes que deses que los datos de +tu base de datos se migren a un servidor MySQL compartido.
  5. Haz clic en el botón Move everything on psmysqlXXXXX back to shared and then destroy psmysqlXXXXX.
  6. Tus datos se migrarán al servidor compartido dentro de las 24 horas.
