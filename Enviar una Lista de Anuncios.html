Generalidades

Este artículo describe cómo enviar una Lista de Anuncios dentro de tu panel DreamHost.

Enviar una lista

  1. Navega hasta la página Lista de Anuncios.
  2. A la derecha de tu lista, haz clic en el botón Correo.
  3. Ingresa lo siguiente:
    • De: La dirección de correo que envías el mensaje no se puede cambiar, pero puede cambiar el nombre De si es necesario.
    • Asunto: Ingresa la línea de asunto de la publicación que estás enviando. Este campo solo acepta texto sin formato.
    • Tu mensaje: Ingresa el cuerpo del mensaje aquí. Este campo aceptará otros caracteres no estándar, a diferencia del campo "Asunto" anterior.
    • ¿Es esto HTML? Si ingresas HTML en el campo Tu mensaje, marca la casilla de verificación junto a Preview as HTML in a new window. Para obtener una vista previa del mensaje antes de enviarlo, haz clic en el enlace titulado Mostar vista anticipada como HTML en una nueva ventana.
    • Instrucciones para Dar de Baja en pie de página: Este es el mensaje incluido automáticamente en la parte inferior de las publicaciones para permitir a los usuarios darse de baja de la lista si así lo desean. El mensaje a la derecha de este campo es el mensaje de cancelación de suscripción que has configurado actualmente para las publicaciones de tu lista. Para editar ese mensaje, haz clic en el pequeño botón editar debajo de ‘Unsubscribe Directions Footer:’.
    • Enviar Anuncio a: Ingresa la fecha y la hora en que deseas enviar la publicación a los usuarios suscritos. El campo utiliza un reloj de 24 horas en lugar del reloj de 12 horas y está configurado en Hora del Pacífico (EE. UU.).
  4. Haz clic en el botón Enviar Anuncio Ahora una vez hayas configurado tu mensaje para enviar tu Lista de Anuncios.

Preguntas frecuentes

¿Puedo enviar mensajes codificados en HTML a mi lista?

Cualquier código <html> en 'Pie de página de instrucciones de cancelación' no se muestra si se añaden..

Sí. Marca la casilla titulada ¿Es este HTML?, lo que permite que las etiquetas HTML en tu mensaje se interpreten como HTML para las personas cuyos clientes de correo tengan capacidad para HTML. Debes pegar tu código HTML directamente en el área de texto que dice "Tu mensaje:".

Además, incluso si algunas personas en tu lista no pueden leer correos con formato HTML, tu mensaje se enviará como un mensaje de múltiples partes: esos usuarios con clientes de correo que no pueden leer HTML seguirán recibiendo una versión en texto plano.

¿Cómo publican los miembros en la lista?

Los miembros de una Lista de Anuncios no pueden publicar en la lista. Esta es una lista de correo solo para anuncios, no una lista de discusión. Revisa el artículo sobre la Lista de Discusión para obtener más información.
